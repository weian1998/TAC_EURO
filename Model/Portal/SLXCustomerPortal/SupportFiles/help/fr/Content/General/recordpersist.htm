Rappels de données non-enregistrées

Lorsque vous mettez à jour des informations d'enregistrements, un rappel de « données non-enregistrées » apparaît dans la barre de titre. Si vous avez des données non-enregistrées et que vous souhaitez accéder à une autre page, vous serez invité à enregistrer vos modifications. Cliquez sur OK pour enregistrer vos modifications ou sur Annuler pour rester sur la page en cours.

Procédure

Enregistrer les modifications sur une page
  • Cliquez sur Enregistrer.
Annuler les modifications sur une page
  • Cliquez sur Réinitialiser.
