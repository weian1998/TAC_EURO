Affichage des détails du ticket avant impression

Vous pouvez afficher une version imprimable de la vue Détails du ticket. La version imprimable affiche des informations générales relatives au ticket, son objet, la description du problème, la résolution du ticket (si elle est disponible), tous les commentaires internes et toutes les informations contenues dans les onglets associés du ticket. Utilisez la barre de défilement pour afficher toutes les informations imprimables. Cette vue est en lecture seule.

Pour afficher les détails du ticket avant impression

  1. Dans la vue Détails du Ticket, cliquez sur le bouton Vue avant impressionde la barre d'outils.

La version en lecture seule de la vue Détails du ticket s'ouvre et vous pouvez alors consulter un aperçu.

  1. Utilisez le bouton Imprimer de votre navigateur pour imprimer.
  2. Pour fermer la version imprimable, cliquez sur le bouton Fermer la vue avant impression.
