Ajout d'un ticket dans le portail client

La boîte de dialogue Créer un ticket permet d'ajouter un nouveau ticket pour votre problème. Le système insère automatiquement votre nom, le nom de votre société, votre numéro de téléphone professionnel et vos coordonnées e-mail.

Pour ajouter un ticket

  1. Dans la barre d'outils, cliquez sur Ajouter un ticket.
  2. Cliquez sur le bouton Rechercher en regard des cases Zone, Catégorie ou Problème, sélectionnez les éléments appropriés dans les listes puis cliquez sur OK.

Les éléments Catégorie affichés dépendent de l'option de Zone sélectionnée et les éléments Problème disponibles dépendent de l'option de Catégorie sélectionnée.

  1. Dans la zone Priorité, cliquez sur la flèche du menu déroulant, puis sélectionnez une option dans la liste.
  2. Dans la zone Description, saisissez un descriptif complet du problème.
  3. Cliquez sur Enregistrer.

    4. La vue Détails du ticket s'affiche.

  4. Vous pouvez cliquer sur l'onglet Commentaires pour ajouter des commentaires à un ticket. Les commentaires fournissent davantage d'informations sur le problème au représentant du service client chargé de vérifier les tickets.
  5. Vous pouvez cliquer sur l'onglet Pièces jointes pour ajouter des pièces jointes à un ticket. L'association de pièces jointes à un ticket permet au représentant du service client de mieux cerner le problème.
  6. Une fois toutes les informations remplies, déconnectez-vous ou cliquez sur Ajouter un ticket pour ajouter un autre ticket.
