Dépannage des problèmes courants

Vous trouverez ci-après une liste des problèmes courants :

Mes requêtes renvoient aux mêmes données et j'obtiens toujours les mêmes pages.

Votre navigateur ne traite pas correctement certaines balises et récupère les pages dans le cache (dossier où sont stockées les pages Web) au lieu de les récupérer sur le serveur, lors de votre utilisation du client Web. Vous devez paramétrer les options d'affichage de votre navigateur de telle sorte qu'il récupère les données sur le serveur et non pas dans le cache.

Pour corriger ce problème dans Microsoft Internet Explorer
  1. Dans Microsoft Internet Explorer version 9.0 et supérieure : dans le menu Outils, cliquez sur Options Internet, puis sur l'onglet Général.
  2. Dans la section Fichiers Internet temporaires, cliquez sur Paramètres.
  3. Dans la section Vérifier s'il existe une version plus récente des pages enregistrées, cliquez sur À chaque visite de la page.
  4. Cliquez deux fois sur OK pour enregistrer les modifications.
Pour corriger ce problème dans Firefox
  1. Dans le menu Outils de Firefox, cliquez sur Options.
  2. Cliquez sur Confidentialité.
  3. Dans la section Données privées, cliquez sur Paramètres.
  4. Sélectionnez Cache et tout autre type de données privées à supprimer lorsque vous effacez les données privées.
  5. Cliquez sur OK.
  6. (Facultatif) Pour effacer votre cache chaque fois que vous fermez votre navigateur, sélectionnez l'option Toujours effacer mes données privées lorsque je ferme Firefox.
  7. Cliquez sur le bouton Effacer maintenant.
  8. Cliquez sur OK.

Le fonctionnement de l'aide de Sage SalesLogix est différent dans Firefox

Si vous utilisez Firefox en tant que navigateur, l'aide de Sage SalesLogix s'ouvre dans un nouvel onglet dans la même fenêtre du navigateur et redimensionne la fenêtre. Toutefois, vous pouvez définir des options qui vont empêcher la fenêtre de se redimensionner ou ouvrir de nouvelles pages dans une fenêtre distincte.

La modification de ces options va avoir des conséquences sur le comportement des onglets et des fenêtres pour tous les sites Web et pas seulement pour les pages concernant Sage SalesLogix.
Pour empêcher la fenêtre de se redimensionner
  1. Dans le menu Outils de Firefox, cliquez sur Options.
  2. Cliquez sur Contenu.
  3. En regard de Activer Javascript cliquez sur Avancé.
  4. Désélectionnez l'option permettant de Déplacer ou redimensionner les fenêtres existantes, puis cliquez sur OK.
  5. Cliquez sur OK.
Pour ouvrir toutes les nouvelles pages dans une fenêtre distincte
  1. Dans le menu Outils de Firefox, cliquez sur Options.
  2. Cliquez sur Onglets.
  3. Sous la section Les nouvelles pages doivent être ouvertes dans sélectionnez l'option une nouvelle fenêtre.
  4. Cliquez sur OK.
