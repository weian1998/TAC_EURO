Предпросмотр результатов быстрого поиска

После осуществления поиска и выбора элемента, вы можете использовать предпросмотр для получения детальной информации об элементе.

Для быстрого поиска соответствующей информации

  1. Запустите быстрый поиск и нажмите кнопку Поиск.
  2. При необходимости, используйте кнопки Первая страница, Предыдущая страница, Следующая страница и Последняя страница для просмотра выданных записей и поиска записи для предпросмотра.
  3. Нажмите Предпросмотр(красная ссыдка под записью)

  4. Выполните одно из следующих действий:

    • Нажмите Перейти к первому найденному, чтобы перейти к первому вхождению ключевого слова.
    • Используйте кнопки Предыдущее ключевое слово и следующее ключевое слово, которые располагаются с обеих сторон от ключевого слова для перехода к предыдущему или следующему найденному вхождению ключевого слова.
    • Используйте внутреннюю полосу прокрутки для просмотра записи или файла.

Если область предпросмотра отображает сообщение "Доступ запрещен", значит у вас недостаточно прав доступа для просмотра результатов поиска. Обратитесь к вашему системному администратору для получения доступа.

Чтобы пометить результат как использованный

Если вы считаете какой-либо элемент полезным, но не используете вставку, сделайте следующее:

  • Выберите элемент и нажмите Пометить этот элемент как использованный. Это поможет системе отслеживать, какие элементы используются часто.
