Использование фильтрации по параметрам в Быстром поиске

Фильтрация по параметрам становится доступна по нажатию ссылки Расширенный в клиенте Быстрого поиска. Эти фильтры позволяют ограничить записи по заданым условиям.

Доступные фильтры задаются вашим системным администратором и могут подойти не для всех элементов области поиска. Например, фильтр Категория применим только к заявкам и дефектам.Если вы дополнительно выберете Документ в области поиска, то получите в результате поиска все документы, поскольку фильтр Категория не применим к Документам.

Часто используемые фильтры отображаются при любом поиске. Если вы выставили флаг напротив часто используемого фильтра, в результаты поиска попадут только те записи, которые отвечают условиям заданным вашим системным администратором для выбранного фильтра.

Для получения более подробной информации об установке параметров для часто используемых фильтров обратитесь к разделу "Настройка основных параметров Быстрого поиска" в руководстве Администратора.

Следующая таблица содержит список всех доступных фильтров, на основе стандартных индексов Sage SalesLogix.

Некоторые фильтры могут быть недоступны из клиентского Web портала.
