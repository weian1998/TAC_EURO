Устранение общих неисправностей

Ниже перечислены часто встречающиеся проблемы:

Все запросы возвращают одни и те же данные и загружаются одни и те же страницы.

Ваш браузер некорректно обрабатывает некоторые тэги и формирует страницы на основе данных из кэша (папка для хранения веб-страниц) вместо того, чтобы получать страницы с сервера. Вам необходимо изменить настройки в браузере таким образом, чтобы он загружал данные с сервера, а не из кэша.

Решение проблемы в Microsoft Internet Explorer
  1. В Microsoft Internet Explorerверсии 9.0 и выше в меню Сервис выберите Параметры интернета , а затем откройте закладку Общие.
  2. В разделе Временные интернет файлы нажмите Настройки.
  3. В разделе Проверка обновлений сохраненных страниц нажмите При каждом переходе на страницу.
  4. Дважды нажмите OK для сохранения изменений.
Решение проблемы в Firefox
  1. В Firefox, в меню Сервис выберите Параметры.
  2. Нажмите Конфиденциальные.
  3. В разделе Личные данные sнажмите Настройка.
  4. Выберите Кеш и любые другие типы данных, которые будут удалены при очистке личных данных.
  5. Нажмите OK.
  6. Необязательно) Чтобы очистить кэш каждый раз, при закрытии браузера, выделите опцию Всегда удалять мои личные данные, когда я закрываю Firefox.
  7. Нажмите кнопку Очистить сейчас.
  8. Нажмите OK.

Sage SalesLogix Web Help отображается иначе в Firefox

Если вы используете Firefox в качестве браузера, Sage SalesLogix Web Help открывается в новой вкладке в том же окне браузера и меняется размер окна. Можно настроить свойства, предотвращающие изменение размера окна и открывающие новые страницы в отдельном окне.

Изменение этих свойств отразится на поведении всех закладок и окон для всех Веб Сайтов, не только Sage SalesLogix.
Чтобы предотвратить изменение размера окон
  1. В Firefox, в меню Сервис выберите Параметры.
  2. Нажмите Содержание.
  3. Рядом с Включить Javascriptнажмите Дополнительно.
  4. Уберите опцию Перемещать или изменять размер сущестующих окони нажмите OK.
  5. Нажмите OK.
Чтобы открывать все новые страницы в отдельном окне
  1. В Firefox, в меню Сервис выберите Параметры.
  2. Нажмите Закладки.
  3. В разделе Новая страница должна быть открыта в выберите новом окне.
  4. Нажмите OK.
