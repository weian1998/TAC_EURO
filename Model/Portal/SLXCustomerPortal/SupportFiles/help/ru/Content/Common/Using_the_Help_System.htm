Использование справочной системы

Доступная в Sage SalesLogix Справка содержит объяснения свойств и функциональности с ссылками на связанные разделы.

  • Нажатие кнопки Помощь открывает нужный раздел Справки.
  • Нажатие голубых ссылок отображает дополнительную информацию, открывая другой раздел.
  • Нажатие голубой ссылки отображает дополнительную информацию.

дополнительную информацию

  • Возможные действия и Связанные Разделы, которые появляются в многих разделах, предлагают добавочную информацию.
  • Содержание позволяет находить сгруппированную по функциям информацию.
  • Поиск позволяет найти информацию, вводя термин для поиска.
  • Используйте кнопку Назад в верхней части окна Справки, чтобы вернуться к предыдущему разделу.
  • Используйте кнопку Избранное чтобы управлять разделами Справки, которые вы используете чаще всего.

Вы можете посмотреть соглашения, используемые в этой системе Помощи.
