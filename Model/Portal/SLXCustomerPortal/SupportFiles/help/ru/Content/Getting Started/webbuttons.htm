Предназначение доступных кнопок

Многие функции Web клиента Sage SalesLogix доступны через кнопки. Ниже приведен список доступных кнопок и описание действий, которые можно выполнить через эти кнопки. При навеведении курсора мыши на кнопку Web клиенте Sage SalesLogix, появляется всплывающая подсказка, объясняющая функции кнопки.

Кнопка Подсказка или описание Действие
Добавить, связать Нажмите на эту кнопку для того чтобы добавить или связать элементы.
Добавить Условие Нажмите на эту кнопку для того чтобы добавить новое условие поиска в списочном представлении или на лицевой карточке Субъекта, Контакта, Сделки и Заявки.

Календарь

 Нажмите эту кнопку для выбора даты и времени.
Закрыть просмотр печати Для того чтобы закрыть Просмотр Печати нажмите на эту кнопку.
Свернуть Нажмите эту кнопку для скрытия дополнительной информации в строке списка.
Удалить или разъединить Для того чтобы удалить или разъединить элементы нажмите на эту кнопку.
Редактировать Для того чтобы изменить информацию нажмите на эту кнопку.
Раскрыть Для того чтобы просмотреть дополнительную информацию по записи из списка нажмите на эту кнопку.
Поиск Для поиска записи нажмите на эту кнопку.
Перейти к первому Для того чтобы просмотреть первую запись в группе нажмите на эту кнопку.
Помощь Для того чтобы открыть Справку по Web Client нажмите на эту кнопку.
Перейти к последнему Для того чтобы просмотреть последнюю запись в группе нажмите на эту кнопку.
Списочное представление Для того чтобы переключиться в режим просмотра списка записей нажмите на эту кнопку.
Развернуть окно Для того чтобы развернуть окно до максимального размера нажмите на эту кнопку.
Свернуть окно Для того чтобы свернуть окно нажмите на эту кнопку.
Перейти к след. Для того чтобы просмотреть следующую запись нажмите на эту кнопку.
Перейти к пред. Для того чтобы просмотреть предыдущую запись нажмите эту кнопку.
Предыдущее Ключевое слово и Следующее Ключевое слово (Расширенный поиск) При использовании модуля Расширенный поиск, для того чтобы просмотреть предыдущую или следующую найденную запись, используйте эти кнопки.
Просмотр Печати Просмотр Печати
Обновить Нажмите эту кнопку, чтобы обновить список записей в списке.
Удалить Условие Нажмите на эту кнопку для того чтобы добавить новое условие поиска в списочном представлении или на лицевой карточке Субъекта, Контакта, Сделки и Заявки.
Сохранить Для того чтобы сохранить внесенные изменения нажмите на эту кнопку.
Быстрый Поиск Для того чтобы открыть модуль SpeedSearch нажмите на эту кнопку.
