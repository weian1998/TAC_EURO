Modificare le Informazioni di un Allegato

Utilizzare questa vista per modificare la descrizione di un allegato e caricare una sua nuova versione. Gli allegati vengono sincronizzati.

Per impostazione predefinita non è possibile allegare file maggiori di 4MB. A meno che l'amministratore non abbia personalizzato le impostazioni, verrà segnalato un errore.

Per modificare le informazioni dell'allegato

  1. Aprire la scheda Allegati.
  2. Selezionare l'allegato da modificare e poi premere su Modifica.
  3. Se necessario, aggiornare la Descrizione.
  4. Effettuare una delle seguenti operazioni:
    • Per caricare un file differente, fare clic sul pulsante Seleziona File Differente, sfogliare le cartelle in cerca del nuovo file e fare clic su Apri.

      Se vengono effettuate modifiche ad un allegato è necessario caricare nuovamente il file per rendere disponibili le modifiche in SalesLogix. Le modifiche vengono fatte ad una copia locale del file, non a quella in allegato.

    • Per collegare a un URL diverso digitare il nuovo URL nel campo URL.

  5. Fare clic su OK.
