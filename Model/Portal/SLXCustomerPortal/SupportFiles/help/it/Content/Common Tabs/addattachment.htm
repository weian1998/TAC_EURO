Aggiungere un Allegato

E' possibile aggiungere un allegato a un record. Tutti i file sono archivviati nella directory condivisa degli allegati e sincronizzati ai database remoti.

Per impostazione predefinita non è possibile allegare file maggiori di 4MB. Se un file (o un gruppo di file) maggiore di queste dimensioni è allegato, può essere visualizzato un errore o un avviso. Consultare il proprio amministratore per maggiori informazioni.

Come fare a?

Allegare un file
  1. Aprire la scheda Allegati.
  2. Fare clic sul pulsante Aggiungi File.
  3. Se si apre la finestra di dialogo Allega File, fare clic su Seleziona File.
  4. Localizzare il file da aggiungere e poi premere Apri.
  5. Nel campo Descrizione inserire una descrizione per l'allegato se il nome del file non è sufficiente a descriverlo.
  6. Fare clic su OK.
Allegare un documento di Google
  1. Aprire la scheda Allegati.
  2. Fare clic sul pulsante Aggiungi Documento di Google.
  3. In base alle proprie impostazioni di sicurezza può risultare necessario:
    • Consentire l'apertura di pop-up nella propria sessione del browser.
    • Connettersi al proprio account di Google e consentire l'accesso.
  4. Nell'elenco, fare doppio clic sul documento da aggiungere.
  5. Una volta terminato, chiudere la finestra di dialogo.
Allegare un URL
  1. Aprire la scheda Allegati.
  2. Fare clic sul pulsante Aggiungi URL.
  3. Nel campo URL, digitare l'indirizzo URL completo.
  4. Nella casella Descrizione, digitare una descrizione della tabella.
  5. Fare clic su OK.
    Sia una descrizione che l'URL sono obbligatorie.
