Beheben gängiger Fehler

Im Folgenden werden häufig auftretende Probleme beschrieben:

Problem:

Alle Abfragen liefern die gleichen Ergebnisse, und es werden immer die gleichen Seiten angezeigt.

Ich muss den Cache löschen

Lösung:

Während Sie mit Web-Client arbeiten, werden bestimmte Tags von Ihrem Browser eventuell nicht richtig verarbeitet, und es werden Seiten aus Ihrem Cache (Ordner, der Web-Seiten speichert) anstatt vom Server aufgerufen. Sie müssen die Anzeigeoptionen Ihres Browsers so festlegen, dass Daten vom Server und nicht aus dem Cache abgerufen werden.

Detaillierte Informationen dazu finden Sie in der Browserhilfe.

Problem:

In Firefox verhält sich die Sage SalesLogix-Webhilfe anders

Wenn Sie Firefox als Browser verwenden, wird die Sage SalesLogix-Webhilfe im gleichen Browserfenster in einem neuen Tab geöffnet und das Fenster verkleinert. Sie können jedoch entsprechende Optionen festlegen, um zu verhindern, dass die Fenstergröße geändert wird oder neue Seiten in einem separaten Fenster geöffnet werden.

Wenn Sie diese Optionen ändern, wirkt sich dies nicht nur auf das Verhalten von Sage SalesLogix-Seiten, sondern auf das Verhalten der Tabs und Fenster aller Websites aus.

Lösungen:

So öffnen Sie alle neuen Seiten in einem separaten Fenster

  1. Klicken Sie in Firefox im Menü Extras auf Einstellungen.
  2. Klicken Sie auf Tabs.
  3. Wählen Sie Statt neuem Fenster neuen Tab öffnen aus.
  4. Klicken Sie auf OK.

So verhindern Sie, dass die Fenstergröße geändert wird

  1. Klicken Sie in Firefox im Menü Extras auf Einstellungen.
  2. Klicken Sie auf Inhalt.
  3. Klicken Sie neben JavaScript aktivieren auf Erweitert.
  4. Deaktivieren Sie die Option Existierende Fenster verschieben oder deren Größe ändern und klicken Sie auf OK.
  5. Klicken Sie auf OK.

Problem:

Ich habe meine Browser- oder Ländereinstellungen geändert, doch manche Informationen werden trotzdem in der falschen Sprache angezeigt.

Lösung:

  • Ihre Länder- und Browsereinstellungen müssen identisch sein. Stellen Sie sicher, dass sowohl bei den Länder- als auch den Browsereinstellungen dieselbe Sprache festgelegt ist.
