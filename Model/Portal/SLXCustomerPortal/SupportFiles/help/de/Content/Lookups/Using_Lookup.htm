Verwenden der Suchfunktion

Im Rahmen der Suchfunktion können Sie Kriterien aus einer Auswahlliste auswählen oder einen Text eingeben. Zum Anzeigen von Ergebnissen müssen Sie für die Abfrage mindestens einen Suchparameter angeben. Sie können Ihre Suche einschränken, indem Sie weitere Bedingungen hinzufügen.

Sage SalesLogix Kundenportal-Benutzer können Suchen nicht als Gruppe speichern.

So führen Sie eine Suche durch

  1. Führen Sie einen der folgenden Schritte durch:
    • Öffnen Sie die Detail- oder Listenansicht für den gesuchten Datensatztyp, und klicken Sie auf die Registerkarte Suchen.
  2. Wählen Sie im Feld Suchen nacheine Suchbedingung aus der Dropdown-Liste aus.
  3. Wählen Sie aus der Dropdown-Liste einen Suchbedingungsoperator aus (z. B. Enthält oder Beginnt mit).
  4. Geben Sie im nächsten Feld Ihre Suchkriterien ein, die der ausgewählten Suchbedingung entsprechen müssen.

  5. Um ein weiteres Suchkriterium hinzuzufügen, klicken Sie auf die Schaltfläche Bedingung hinzufügen und wiederholen zu jedem hinzugefügten Suchkriterium die Schritte 2 bis 4.
  6. Wenn Sie all Ihre Suchanforderungen eingegeben haben, klicken Sie auf Suchen.
    Die Datensätze, die Ihren Kriterien entsprechen, werden in der Registerkarte Suchergebnisse angezeigt.

Alle Suchvorgänge werden für die Datensätze durchgeführt, auf die Sie zugreifen können. Abhängig von den vom Administrator festgelegten Optionen können Sie auch Datensätze auf der Grundlage der Gruppe suchen, der sie angehören.
