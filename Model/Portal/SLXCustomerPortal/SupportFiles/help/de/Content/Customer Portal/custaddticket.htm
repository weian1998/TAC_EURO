Hinzufügen eines Tickets im Kundenportal

Im Dialog Ticket erstellen können Sie ein neues Ticket für ein Problem erstellen. Das System gibt automatisch Ihren Namen, Firmennamen, Ihre Diensttelefonnummer und E-Mail-Informationen ein.

So fügen Sie ein Ticket hinzu

  1. Klicken Sie in der Symbolleiste auf Ticket Hinzufügen.
  2. Klicken Sie auf die Schaltfläche Suchen neben dem Feld Bereich, Kategorie oder Problem, wählen Sie die entsprechenden Einträge aus der Liste aus und klicken Sie auf OK.
    Welcher Kategorie die aufgeführten Einträge angehören, ist von der Option abhängig, die Sie als Bereich ausgewählt haben, und welche Probleme verfügbar sind, ist von der Option abhängig, die Sie als Kategorie ausgewählt haben.
  3. Klicken Sie im Feld Dringlichkeit auf die den Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus.
  4. Geben Sie in das Feld Betreff eine Kurzzusammenfassung des Problems ein.
  5. Geben Sie alle Details zu dem Problem in das Feld Beschreibung ein.
  6. Klicken Sie auf die Schaltfläche Speichern.
    Die Detailansicht des Tickets wird angezeigt.
  7. Durch Klicken auf die Registerkarte Kommentare können Sie dem Ticket Kommentare hinzufügen. Durch Kommentare erhält der für die Bearbeitung der Tickets zuständige Kundendienstmitarbeiter detailliertere Informationen zu dem jeweiligen Problem.
  8. Durch Klicken auf die Registerkarte Anlagen können Sie dem Ticket Anlagen hinzufügen. Anlagen können es dem Kundendienstmitarbeiter erleichtern, das Problem zu verstehen.
  9. Wenn alle Informationen eingegeben sind, melden Sie sich ab oder klicken auf Ticket hinzufügen, um ein weiteres Ticket hinzuzufügen.
