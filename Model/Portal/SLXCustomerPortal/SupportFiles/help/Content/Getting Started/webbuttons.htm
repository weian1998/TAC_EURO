Understanding the Sage SalesLogix Customer Portal Workspace

The workspace is the area where various views appear in the main Sage SalesLogix Customer Portal window.

Each view in your Sage SalesLogix Customer Portal contains many of the following elements.

Element Description
Menu Bar

The menu bar contains the Sage SalesLogix commands. These include Recently Viewed, Add Ticket, Help, and the SpeedSearch button.

Click a menu to expand it.
Navigation Bar Located on the left side of the Sage SalesLogix Customer Portal workspace, the Navigation Bar contains the button that opens the Ticket List view.
Title Bar

The title identifies which record is currently being displayed in the active view.
Tabs Tabs organize information on many of the detail views and dialog boxes.
Status Bar The status bar displays the user name.
Task Pane The Task Pane displays to the right of the main window pane. The Task Pane consists of common tasks and features designed to help you complete your work. The available options depend on the record type and type of view where you are working.
