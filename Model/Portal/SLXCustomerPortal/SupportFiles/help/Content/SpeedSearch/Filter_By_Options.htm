Using SpeedSearch Filter By Options

The Filter By options are made available by clicking Advanced in the SpeedSearch Client. These filters allow you to limit the results to records that meet specific criteria.

The available filters are determined by your system administrator and may not apply to all items in the Search In area. For example, Category only applies to tickets and defects. If you also select Document in the Search In area, all of the documents are included in the search since Category does not apply.

The Frequently Used filter appears for all searches. If you select the check box next to Frequently Used, only records that meet the frequently used criteria defined by your system administrator are included in the search.

See "Configuring SpeedSearch General Settings" in the Administrator Help for more information on setting the parameters for the Frequently Used filter.

The following grid lists all available filters, based on standard Sage SalesLogix indexes.

Some filters may not be available from the Web Customer Portal.
Filter Description Applies To
Date Created Within Used to locate records created within X number of days. All Searches
Last Updated Within Used to locate records updated within X number of days. All Searches
AccountID Like Used to locate records containing an Account ID that is like the number entered. Activity, History
ContactID Like Used to locate records containing a Contact ID that is like the number entered. Activity, History
OpportunityID Like Used to locate records containing an Opportunity ID that is like the number entered. Activity, History
TicketID Like Used to locate records containing a Ticket ID that is like the number entered. Activity, History
File Name Like Used to locate records in which the file names are like the text entered. Attachment, Library, Document
File Last Modified Within Used to locate files last modified within X number of days. Attachment, Library, Document
File Date Created Within Used to locate files created within X number of days. Attachment, Library, Document
Status Like Used to locate records in which the Status field of the record type selected contains the text entered. Defect, Defect Internal, Ticket, Ticket Internal
Area Like Used to locate records in which the Area field of the record type selected contains the text entered. Defect, Defect Internal, Ticket, Ticket Internal
Category Like Used to locate records in which the Category field of the record type selected contains the text entered. Defect, Defect Internal, Ticket, Ticket Internal
Project Like Used to locate records in which the Project field of the record type selected contains the text entered. Defect, Defect Internal
Fixed In Like Used to locate records in which the Fixed In field of the record type selected contains the text entered. Defect, Defect Internal
