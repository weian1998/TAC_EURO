Visualizzare la Versione SQL di una Query

Quando si crea una query, essa viene convertita in un'istruzione SQL (Structured Query Language). Tale istruzione è applicata al database. E' possibile visualizzare la versione SQL di una query; comunque, non è possibile modificare l'istruzione SQL.

Visualizzare l'istruzione SQL per una query facilita la comprensione del funzionamento della query. Quest'opzione può inoltre essere utile quando si prova a scrivere SQL per altre parti del programma e si desidera avere un esempio di codice SQL funzionante.

Per visualizzare la versione SQL di una query

  1. Aprire il Query Builder.
  2. Fare clic su Visualizza SQL.
  3. Dopo aver visualizzato l'espressione SQL, chiudere la finestra di dialogo.
