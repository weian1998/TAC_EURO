Stringa Formato

Utilizzare la stringa formato per definire il modo con cui i dati sono visualizzati nella query. Per prima cosa, selezionare il Tipo Formato. Quindi,digitare una stringa formato per modificare il tipo di formato selezionato. Le opzioni valide dipendono dal tipo di formato selezionato. Un formato stringa deve cominciare con il segno % e terminare con un carattere.

[%] [tipo]

  • uno specificatore indice argomento
  • un Indicatore di Giustificazione Sinistra
  • uno specificatore di larghezza
  • uno specificatore di precisione

[%] [indice:] [-] [larghezza] [.precisione] [tipo]

Per specificare quanti decimali utilizzare in una stringa formato, impostare la stringa al formato normale che si utilizzerebbe. Quindi, aggiungere lo specificatore di precisione - un punto decimale seguito dal numero di cifre - prima del carattere di formato stringa, ma dopo il % iniziale. Ad esempio:

  • Tipo Campo Float
  • Tipo Formato: ftCurrency.
  • Stringa Formato: %.4f
  • Dati (Non formattati): 3.1415926
  • Risultato: 3.1416

E' possibile utilizzare uno qualsiasi dei seguenti tipi di stringhe formato:

  • Stringhe Formato per campi Booleani
  • Stringhe Formato Valuta e telefono
  • Stringhe Formato Campi Ora/Data
  • Stringhe Formato per Campi Fissi
  • Stringhe Formato per Campi Interi
