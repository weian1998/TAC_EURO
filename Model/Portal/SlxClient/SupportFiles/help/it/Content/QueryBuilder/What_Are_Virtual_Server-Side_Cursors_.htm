Cosa sono i Virtual Server-Side Cursor (VSSC)?

I Virtual Server-Side Cursor (VSSC) sono strutturati per aumentare le prestazioni e l'uso della memoria per gruppi numerosi nelle viste gruppi e Elenco del Client SalesLogix.

Per determinare se un gruppo sta usando VSSC

  • Selezionare la scheda gruppo dall'appropriata vista Elenco o Dettaglio e cercare una delle icone seguenti nell'angolo superiore destro della vista, vicino al pulsante Opzioni Gruppo.
Icona Cosa significa?
Il gruppo è in Modalità Server Virtuale - il gruppo è stato elaborato usando i Virtual Server-Side Cursor.
Il gruppo è in Modalità Lato Client - il gruppo è stato elaborato sul client. Non sta usando i Virtual Server-Side Cursor.

 

Per maggiori informazioni circa i Virtual Server-Side Cursor, vedere Riferimento Sviluppatori LAN.
