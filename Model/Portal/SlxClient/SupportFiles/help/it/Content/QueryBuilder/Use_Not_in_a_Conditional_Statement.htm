Utilizzo di "Non" in un'Espressione Condizionale

E' possibile inserire NON prima di un'espressione o gruppo di espressioni condizionali. Se NON viene inserito prima di un gruppo, esso si applicherà a tutte le condizioni nel gruppo.

Per inserire NON prima di un'espressione condizionale o gruppo

  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Condizioni ed aggiungere tutte le espressioni condizionali.
  3. Se necessario, raggruppare le espressioni condizionali.
  4. Spuntare il campo Non di un'espressione condizionale, oppure la prima espressione in un gruppo.
  5. Fare clic su OK per eseguire la query.

Esempio

Il Query Builder selezionerà tutti i contatti che vivono in Italia, ma non quelli che vivono a Roma o Milano.

La griglia della scheda Condizioni presenta questo aspetto:

Non ( Campo Operatore Valore Maiusc/Minusc ) E/o
NON ( Contatto.Indirizzo.Citta Uguale a Milano Vero   OR
    Contatto.Indirizzo.Citta Uguale a Roma Vero ) AND
    Contatto.Indirizzo.Nazione Uguale a Italia Vero   END

Una query può contenere sottogruppi all'interno di un gruppo. Se NON è utilizzato in precedenza di un gruppo, esso si applicherà a tutti i sottogruppi contenuti nel gruppo.
