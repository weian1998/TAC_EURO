Campi di Informazioni dell'Elenco di Selezione

La tabella seguente descrive i campi informazione predefiniti. Per saperne di più sulle opzioni degli elenchi di selezione...

Prima di configurare le opzioni dell'elenco assicurarsi di aver compreso a pieno le proprietà del controllo Elenco di Selezione e il modo in cui i dati dell'elenco vengono memorizzati nel database. La comprensione del modo in cui i dati sono collegati tra elenco e database può aiutare a stabilire se un'opzione è supportata o meno. Se ad esempio si modifica l'elenco di selezione Preferenze con "Consenti selezioni multiple", l'elenco non funzionerà correttamente dato che il campo del database è troppo piccolo per poter memorizzare grandi quantità di testo.

Campo Descrizione/Azione Si applica al Web Si applica alla Rete
Nome Il nome dell'elenco di selezione.
Valore Richiesto Forza gli utenti a inserire il valore prima di continuare con il prossimo campo. Una volta spuntata, questa opzione si applica automaticamente agli elenchi di Rete, ed anche agli elenchi Web nel momento in cui viene selezionato Applica opzioni al Web. *
Consenti Selezioni Multiple Permette agli utenti di selezionare più di una voce dall'elenco. Una volta spuntata, questa opzione si applica automaticamente agli elenchi di Rete, ed anche agli elenchi Web nel momento in cui viene selezionato Applica opzioni al Web. *
Il testo deve coincidere con una voce dell'elenco Assicurarsi che il valore inserito coincida con uno degli elementi dell'elenco. Una volta spuntata, questa opzione si applica automaticamente agli elenchi di Rete, ed anche agli elenchi Web nel momento in cui viene selezionato Applica opzioni al Web. *
Ordinato alfabeticamente Visualizza le voci dell'elenco di selezione in ordine alfabetico. Se questa opzione è selezionata, i valori di Ordine saranno ignorati. Una volta spuntata, questa opzione si applica automaticamente agli elenchi di Rete, ed anche agli elenchi Web nel momento in cui viene selezionato Applica opzioni al Web. *
Gli utenti possono modificare gli elementi Consente agli utenti di aggiungere elementi all'elenco di selezione o modificarli. Questa opzione si applica soltanto agli elenchi di Rete. Gli utenti non possono modificare elementi nel Client Web.
