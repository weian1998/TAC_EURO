Risoluzione dei Problemi di Desktop Integration - Invio a SLX e Registrazione nella Cronologia

E' possibile riscontrare i seguenti problemi con Desktop Integration e Invia a SLX.

Problema:

L'icona Desktop Manager di SalesLogix non appare nella mia barra di sistema.

Cause e Soluzioni possibili:

  • Il modulo Desktop Integration non è installato.
  • E' necessario installare il modulo Desktop Integration.
  • Il modulo Desktop Integration è stato installato prima del browser.
  • Se si è installato il browser corrente dopo il modulo Desktop Integration, è necessario reinstallare il modulo Desktop integration utilizzando il browser corrente.

    		•

    Questo può accadere se si è installato Firefox dopo aver installato il modulo Desktop Integration utilizzando Internet Explorer.
  • Sfogliare le cartelle in cerca di SLXDesktopManager.exe e aprirlo manualmente. Il percorso di questo file varia in base al proprio sistema operativo:

Di norma è:

\\Documenti\[nome utente]\Impostazioni Locali\Dati Applicazioni (XP/2000/2003) oppure

\\Utenti\<utente.dominio>\appdata\local (Vista/7/2008).

Problema:

Il mio pulsante Invia a SLX non è presente in Microsoft Outlook

Cause e Soluzioni possibili:

  • Il modulo Desktop Integration non è installato.
  • E' necessario installare il modulo Desktop Integration.
  • Il modulo Desktop Integration è stato installato prima del browser.
  • Se si è installato il browser corrente dopo il modulo Desktop Integration, è necessario reinstallare il modulo Desktop integration utilizzando il browser corrente.

    		•

    Questo può accadere se si è installato Firefox dopo aver installato il modulo Desktop Integration utilizzando Internet Explorer.

Problema:

La mia e-mail non viene registrata nella cronologia

Cause e Soluzioni possibili:

  • Le informazioni sul database Desktop Manager di SalesLogix non sono configurate.
    • E' necessario configurare le opzioni di Desktop Integration.
  • E' possibile disattivare l'opzione di visualizzazione del messaggio di sistema che avvisa in caso di contatti o nominativi non risolti. Se un contatto o un nominativo non possono essere trovati e non viene richiesto dal sistema di risolvere questi record, allora l'e-mail non sarà registrata nella cronologia di SalesLogix.
    • Controllare le proprie Opzioni Cronologia in Desktop Manager.
  • E' possibile disattivare l'opzione di visualizzazione del messaggio di sistema che avvisa in caso di contatti o nominativi duplicati. Se più di un contatto o nominativo hanno lo stesso indirizzo e-mail e non si riceve alcun messaggio di sistema che richiede la risoluzione di questi record, allora l'e-mail sarà registrata nella cronologia corrispondente al record che ha il maggior numero di elementi di cronologia.
    • Controllare le proprie Opzioni Cronologia in Desktop Manager.

Problema:

Il sistema mi ha avvisato che il mio messaggio Invia a SLX non può essere processato

Cause e Soluzioni possibili:

  • Desktop Manager non è in grado di comunicare con il Server SalesLogix.
    • Desktop Manager tenterà di inviare nuovamente l'e-mail ad intervalli specifici, fino a 24 ore dopo il primo tentativo. Si riceverà una notifica ogni volta che l'invio del messaggio non ha avuto successo.
    • Gli intervalli sono da 5 minuti, 10 minuti, 20 minuti, 45 minuti, 90 minuti, 3 ore, 6 ore, 12 ore e 24 ore.
    • Dopo le 24 ore Desktop Manager non tenterà ulteriormente di inviare il messaggio.

Problema:

Non posso trascinare un'e-mail nel Client Web di SalesLogix

Cause e Soluzioni possibili:

  • Il modulo Desktop Integration non è installato.
    • E' necessario installare il modulo Desktop Integration.
  • E' possibile trascinare le e-mail nelle viste Dettaglio Contatto, Azienda, Nominativo, Opportunità e Ticket.
