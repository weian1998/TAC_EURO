Risoluzione di Problemi del Modulo Desktop Integration

E' possibile riscontrare i seguenti problemi durante l'utilizzo del modulo Desktop Integration.

Problema:

Come faccio a sapere se ho installato la versione corretta del modulo Desktop Integration?

Soluzione:

  • Visualizzare la schermata di accesso del Client Web SalesLogix oppure fare clic sulla scheda Generale nelle Opzioni.

Se il testo vicino al pulsante Migliora SalesLogix riporta che "I miglioramenti sono stati installati.", allora indica che la corretta versione è installata. Se il testo riporta "Clicca qui per scaricare i miglioramenti del prodotto.", allora è necessario installare l'ultima versione del modulo Desktop Integration.

Problema:

L'icona Desktop Manager di SalesLogix non appare nella mia barra di sistema.

Cause e Soluzioni possibili:

  • Il modulo Desktop Integration non è installato.
  • E' necessario installare il modulo Desktop Integration.
  • Il modulo Desktop Integration è stato installato prima del browser.
  • Se si è installato il browser corrente dopo il modulo Desktop Integration, è necessario reinstallare il modulo Desktop integration utilizzando il browser corrente.

    		•

    Questo può accadere se si è installato Firefox dopo aver installato il modulo Desktop Integration utilizzando Internet Explorer.
  • Sfogliare le cartelle in cerca di SLXDesktopManager.exe e aprirlo manualmente. Il percorso di questo file varia in base al proprio sistema operativo:

Di norma è:

\\Documenti\[nome utente]\Impostazioni Locali\Dati Applicazioni (XP/2000/2003) oppure

\\Utenti\<utente.dominio>\appdata\local (Vista/7/2008).

Problema:

Come disattivo Desktop Manager?

Soluzione:

  • Nella barra di sistema della Barra applicazioni di Windows fare doppio clic sull'icona Desktop Manager per SalesLogix e fare clic su Esci.

E' necessario uscire da Desktop Manager prima di poter disinstallare il modulo Desktop Integration.

Problema:

Il sistema richiede di scaricare una nuova versione del Modulo Desktop Integration.

Soluzione:

  • Scaricare la nuova versione del Modulo Desktop Integration.

    • Se si utilizza Firefox come browser, è necessario prima disinstallare il modulo aggiuntivo Desktop Integration da Firefox.

    Per disinstallare il modulo aggiuntivo Desktop Integration, aprire Firefox. Sul menù Strumenti, fare clic su Componenti Aggiuntivi, selezionare Sage SalesLogix Desktop Integration Module e poi fare clic su Disinstalla. Fare clic su Disinstalla per confermare. Fare clic su Riavvia Firefox. Fare clic su Riavvia.

Problema:

Come disinstallo il Modulo Desktop Integration?

Soluzione:

  1. Aprire Aggiungi/Rimuovi Programmi di Windows.
  2. Selezionare Modulo Sage SalesLogix Desktop Integration nell'elenco e fare clic su Rimuovi.
  3. Fare clic su Sì.

    		4.

    Se si utilizza Firefox come browser, è necessario prima disinstallare il modulo aggiuntivo Desktop Integration da Firefox.

    Per disinstallare il modulo aggiuntivo Desktop Integration, aprire Firefox. Sul menù Strumenti, fare clic su Componenti Aggiuntivi, selezionare Sage SalesLogix Desktop Integration Module e poi fare clic su Disinstalla. Fare clic su Disinstalla per confermare. Fare clic su Riavvia Firefox. Fare clic su Riavvia.
