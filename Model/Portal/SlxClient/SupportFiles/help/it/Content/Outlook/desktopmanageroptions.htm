Configurare le Opzioni di Cronologia E-mail di Desktop Manager

Utilizzare il SalesLogix Desktop Manager per inserire informazioni sulla connessione e impostare le opzioni di cronologia per l'email.

Per configurare le opzioni

  1. Nella barra di sistema della Barra applicazioni di Windows fare doppio clic sull'icona Desktop Manager per SalesLogix.
  2. Effettuare una delle seguenti operazioni:
  • Utenti del Client Web - Nel campo URL Portale digitare l'indirizzo a cui connettersi. Utilizzare il seguente formato:

http://nomeserver:porta

  • Utenti del Client Web Offline - Nel campo URL Portale digitare l'indirizzo a cui connettersi. Utilizzare il seguente formato:

http://localhost

Si si utilizza un nome completo, è necessario aggiungere nome dominio.com dopo il nome server.

  1. Effettuare una delle seguenti operazioni:
  • Se si utilizza l'autenticazione di Windows per connettersi a SalesLogix, selezionare l'opzione Usa l'Autenticazione di Windows.
  • Inserire il proprio nome utente e password.
  1. Nel campo Utente inserire il nome utente utilizzato per connettersi al Client Web SalesLogix.
  2. Nella casella Password digitare la password SalesLogix per l'utente inserito sopra.
  1. Fare clic su Prova.
  2. Sotto Opzioni Cronologia, selezionare quando si desidera essere avvisati durante la registrazione di un'email Outlook all'interno della cronologia SalesLogix. Selezionare una delle seguenti opzioni:
  • Avvisa in caso di Contatti Duplicati - Avvisa in caso di contatti o nominativi duplicati.
  • Avvisa per i Contatti - Avvisa per selezionare il contatto o il nominativo a cui dovrebbe essere associata la cronologia.
  • Avvisa in caso di Contatti Non Risolti - Avvisa quando non bviene trovato un contatto o nominativo nel database.
  • Visualizza i Dettagli della Cronologia prima di salvare - Avvisa con la finestra di dialogo Completa Email prima di registrare un messaggio email in cronologia.
  1. Spuntare il campo Visualizza le Notifiche se si desidera vedere un messaggio pop-up che descrive le attività in esecuzione del Desktop Manager.

Questa opzione non si applica al trascinamento delle e-mail. Utilizzare Registra in Cronologia in Opzioni per attivare il trascinamento delle e-mail.

Per maggiori informazioni sul Desktop Manager, fare riferimento alla guida in linea.
