Cosa sono i Report?

La vista Report fornisce l'accesso a un insieme di query di database predefinite che producono report commerciali comunemente utilizzati. I report specifici disponibili sono stabiliti dal proprio amministratore di sistema e sono di norma personalizzati in base alle esigenze della propria attività.

Il Cruscotto è uno strumento configurato in modo da consentire la visualizzazione e l'interazione con una serie di informazioni di analisi e di prestazioni elaborate sotto forma di cruscotto. E' possibile utilizzare i dati forniti per avere una panoramica dettagliata delle prestazioni individuali e della propria organizzazione, scoprire le cause fondamentali di problemi di rendimento e intraprendere le azioni per risolverli oppure prendere decisioni strategiche.

SalesLogix Web Reporting permette di:

  • Condividere i filtri del report.
  • Filtrare report per limitare i dati inclusi.
  • Esportare e stampare report.

Per aprire la vista Report

  • Sulla barra di navigazione fare clic su Report.

Per eseguire i report Web in stato disconnesso, il Client Web Offline deve essere eseguito su Window 2003 Server con IIS.Devono essere installati e configurati anche il Server del Web Reporting SalesLogix e il Crystal Reports Server. Se non vengono soddisfatti questi requisiti, gli utenti del Client Web disconnesso possono eseguire Web Reporting soltanto se dispongono di una connessione a Host Web Reporting Server. Questi utenti vedranno i report che rappresentano i dati del database host e non del loro database remoto.
