Visualizzare le Formule di Condizione

Gli amministratori che sono familiari con Extended Markup Language (XML) o SQL possono visualizzare le formule di condizioni del report per assisterli nella risoluzione dei problemi. Questa opzione potrebbe essere utile per risolvere problemi.

Per visualizzare le Formule Condizione

  1. Nella vista Report selezionare un filtro report e poi fare clic su Modifica.
  2. In Condizioni effettuare una delle seguenti operazioni:
  • Fare clic su Visualizza XML per visualizzare il codice XML del report.
  • Fare clic suVisualizza SQL per visualizzare il codice SQL del report.
  1. Fare clic su OK.
