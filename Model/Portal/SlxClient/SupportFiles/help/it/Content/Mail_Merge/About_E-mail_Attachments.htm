Informazioni sugli Allegati E-mail

E' possibile allegare un file quando si invia un messaggio email a un contatto oppure ad un nominativo. Se le opzioni corrette sono state selezionate, il file sarà registrato nella scheda Allegati di Contatti e Aziende, così come nella cartella Documenti. Per utenti di SalesLogix Web, il file viene registrato nella cartella Documenti del Web Server. Nella scheda Allegati, il nome file è l'esatto nome del file o del documento. Nella cartella Documenti, il nome del file include un codice di riferimento interno assegnato da SalesLogix e il nome del documento.

E' possibile allegare ad un modello file di Windows o della Libreria di SalesLogix. Quando il modello viene utilizzato in un'unione, i file allegati vengono automaticamente inviati al contatto o nominativo. Durante la copia di un modello vengono copiati anche i suoi allegati. E' possibile rimuovere un allegato da un modello.

Per allegare un file ad un modello
  1. Aprire l'Editor dei Modelli.
  2. Effettuare una delle seguenti operazioni:
  • Per allegare un file Windows, fare clic sul pulsante Inserisci File, sfogliare fino alla posizione del file e poi fare clic su Inserisci File.
  • Per allegare un file della Libreria SalesLogix, fare clic sul pulsante Inserisci File Libreria SalesLogix. Nella finestra di dialogo Libreria SalesLogix, fare clic sui file e fare clic su Inserisci.
  1. E' possibile allegare altri file, oppure fare clic su Salva e Chiudi.
Per rimuovere un file da un modello
  1. Aprire l'Editor dei Modelli.
  2. Nella casella Allegati, selezionare l'allegato da eliminare.
  3. Effettuare una delle seguenti operazioni:
  • Clic destro e poi fare clic su Rimuovi.
  • Premere Elimina.
  1. E' possibile rimuovere altri file, oppure fare clic su Salva e Chiudi.
