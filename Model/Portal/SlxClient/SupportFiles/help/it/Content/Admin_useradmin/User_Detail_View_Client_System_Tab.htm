Scheda Sistema Client

La scheda Sistema Client della vista Dettaglio Utente è utilizzata per includere i modelli di base di un utente e per configurare il proprietario predefinito delle aziende create da questo utente. Gli utenti possono modificare tali impostazioni tramite la scheda Generale delle Opzioni.

Per configurare le opzioni

  1. Dalla vista Dettaglio Utente, fare clic sulla scheda Sistema Client.
  2. Utilizzare la sezione Proprietario Predefinito per configurare un proprietario predefinito per le nuove aziende.
    1. Nel campo Aziende selezionare il proprietario predefinito delle nuove aziende create dall'utente.
    2. Deselezionare la casella Consenti all'utente di modificare queste impostazioni per impedire all'utente di cambiare la configurazione del proprietario predefinito. Per saperne di più...

    3. Se si consente ad un utente di modificare queste impostazioni, la proprietà di un'azienda può essere assegnata ad un proprietario che potrebbe non aderire alle regole aziendali stabilite. Si corre così il rischio di non includere queste aziende nei report o nelle interrogazioni stabilite.

  3. Selezionare i Modelli Base Stampa Unione che si desidera utilizzare quando viene creato un nuovo modello email contatto, fax o lettera.
    1. Fare clic sulla freccia Tipo e selezionare Contatto.
    2. Fare clic sul pulsante Trova Modello Base E-mail.

    3. Selezionare un modello e fare clic su OK.

    4. Ripetere i passi b-c per Modello Base Lettera e Modello Base Fax.

  4. Selezionare i Modelli Stampa Base Unione che si desidera utilizzare quando viene creato un nuovo modello email nominativo, fax o lettera.

    Quando si selezionano i modelli per i nominativi selezionare solamente quelli che terminano con "'(Nominativo)"'.

    1. Fare clic sulla freccia Tipo e selezionare Nominativo.
    2. Fare clic sul pulsante Trova Modello Base E-mail.

    3. Selezionare un modello e fare clic su OK.

    4. Ripetere i passi b-c per Modello Base Lettera e Modello Base Fax.
  5. (Opzionale) Deselezionare la casella Consenti all'utente di modificare queste impostazioni per impedire all'utente di cambiare i modelli predefiniti.
  6. Fare clic su Salva..

Quando si configurano gli utenti WebViewer è disponibile solo la sezione Proprietario Predefinito.
