Aggiungere una Join Locale

Una join locale è temporanea ed esiste nel Query Builder per la durata di una sessione. L'oggetto che utilizza la query risultante manterrà le proprietà della join locale finchè in uso.

Per aggiungere

  1. Aprire il Query Builder
  2. Selezionare una tabella.
  3. Selezionare un campo.
  4. Fare clic sul collegamento Crea Join Locale.

Nella finestra di dialogo Join Locale, la tabella e il campo selezionati nel Query Builder appaiono nell'area Tabella Principale e Campo Join.

  1. Nell'area Tabella Secondaria e Campo di Join, selezionare la tabella del database e il campo per cui si sta creando la join locale.
  2. Nell'area Tipo Join, selezionare il tipo di join.

Il tipo della join predefinito è Interno.

  1. Fare clic su OK.
