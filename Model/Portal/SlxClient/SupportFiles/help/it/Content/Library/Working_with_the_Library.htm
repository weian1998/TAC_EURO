Utilizzo della Libreria

La Libreria è un archivio centralizzato di tutte le informazioni della propria società (sotto forma di documenti, URL, etc.) che facilita la distribuzione di informazioni agli utenti remoti . I file sono organizzati in cartelle e ordinati in base al nome. Fare clic sull'intestazione di colonna per ordinare l'elenco dei file della libreria.

Usando il Client Web SalesLogix, è possibile richiedere e scaricare un file della libreria quando necessario. La libreria visualizza il nome file, dimensione, creazione, date revisione e scadenza, e una breve descrizione del file.

La libreria della propria organizzazione può includere informazioni su prodotti, normative aziendali e procedure, presentazioni, indirizzi Web, etc.

Soltanto l'amministratore di sistema e un utente amministrativo può aggiungere documenti e file URL alla Libreria.

Come fare a?

Aprire la Libreria
  • Sulla barra di navigazione fare clic su Libreria.
Scaricare un file dalla Libreria
  1. Nella vista ad albero fare clic sul simbolo "+" per espandere la cartella contenente il file da aprire.
  2. Nell'elenco fare clic sul nome del File da scaricare.

  3. Effettuare una delle seguenti operazioni:

    • Aprire il file
    • Salvare il file

Se il proprio browser apre il file invece di salvarlo in una posizione del computer, cambiare le impostazioni predefinite del browser per impostare il salvataggio automatico dei file anzichè aprirli. In alternativa, tenere premuto [Shift] e poi fare clic sul collegamento per scaricare senza aprire tutti i file ad eccezione per quelli con estensione HTML. I file HTML vengono aperti in un'altra finestra.

Possono accedere a queste funzioni l'amministratore di sistema e gli utenti con il ruolo appropriato.Possono accedere a queste funzioni l'amministratore di sistema e gli utenti con il ruolo da amministratori.
Aggiungere un documento alla Libreria
  1. Selezionare la cartella in cui si vuole aggiungere l'elemento.
  2. Fare clic su Aggiungi File.
  3. Selezionare i file da aggiungere.
  4. Fare clic su Apri.
Aggiungere un documento alla Libreria con il trascinamento

  • E' possibile che non si abbiano i diritti d'accesso per questa funzionalità. Per maggiori informazioni contattare l'amministratore di sistema.
  • Se si utilizza Internet Explorer, questa funzionalità richiede l'installazione del Modulo di Integrazione Desktop.
  1. Cercare fra le cartelle il file da aggiungere.
  2. Sulla barra di navigazione fare clic su Libreria.
  3. Nella vista ad albero fare clic sul simbolo "+" per espandere la cartella in cui si vuole aggiungere il file.
  4. Trascinare il file e rilasciarlo nella vista Libreria.
  5. Nel campo Descrizione inserire una descrizione per il file se la descrizione predefinita non è chiara.
  6. Fare clic su OK.

Non è possibile trascinare nella Libreria una cartella e il suo contenuto. Si possono trascinare soltanto i singoli file.
