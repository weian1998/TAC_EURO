Diagnostica Intellisync

Intellisync include un'opzione per la risoluzione di problemi, Diagnostica Intellisync. E' possibile trovare soluzioni per la conessione al portale e controllare la versione degli assembly. In molti casi il controllo delle versioni è effettuato solo su richiesta del Supporto Tecnico Sage.

E' possibile accedere alla Diagnostica Intellisync da IIS Manager, dal portale Intellisync e dal Server Web personale di Sage SalesLogix.

Per accedere a IIS Manager

  1. Fare clic su Start/Programmi/Strumenti d'Amministrazione.
  2. Fare clic su Internet Information Services (IIS) Manager.
  3. Fare clic sul nome della macchina per espanderlo, poi su Siti Web e quindi su SalesLogix.
  4. Fare doppio clic su SLXIntellisync.
  5. Effettuare una delle seguenti operazioni:
  • Windows 2003: Nell'elenco di file nel pannello a destra, fare clic destro su Default.aspx e poi su Sfoglia.
  • Windows 2008: Nell'elenco di file nel pannello a destra, fare clic destro su Documento Predefinito, puntare su Gestisci Applicazione e quindi cliccare su Sfoglia. Nel browser sarà visualizzato il portale Intellisync. Se non viene visualizzato, assicurarsi che le impostazioni siano configurate correttamente e che il portale sia stato rilasciato.
  1. Nel browser fare clic su Diagnostica ed effettuare una delle seguenti operazioni:
  • Per effettuare un test sulle credenziali dell'utente, digitare Nome Utente e Password e fare clic su Prova Connessione. Una richiesta viene sottoposta al server per verificare che le credenziali siano valide per il portale. Un messaggio indicherà se il test ha avuto esito positivo.

    • Questo test determina soltanto se le credenziali siano valide per la connessione al portale. Non verifica se c'è effettivamente una connessione con un database funzionante.

  • Per visualizzare gli assembly e il loro numero di versione fare clic sui pulsanti successivo e precedente in fondo all'elenco.

Per accedere dal portale

  1. Aprire il portale Intellisync.

Ad esempio digitare l'URL del portale nel browser utilizzando il formato: http://server:port/portal name/default.aspx

  1. Fare clic su Diagnostica.
  1. Effettuare una delle seguenti operazioni:
  • Per visualizzare gli assembly e il loro numero di versione fare clic sui pulsanti successivo e precedente in fondo all'elenco.
  • Per effettuare un test sulle credenziali dell'utente, digitare Nome Utente e Password e fare clic su Prova Connessione.

Una richiesta viene sottoposta al server per verificare se l'utente può accedere al portale Intellisync. Un messaggio indicherà se il test ha avuto esito positivo.

Per accedere dal Server Web personale

  • Per verificare le credenziali di un utente
  1. Fare clic destro sull'icona Server Web personale di Sage SalesLogix nella barra di sistema, poi su Proprietà.
  2. Nella finestra di dialogo Server Web personale di Sage SalesLogix fare clic su Gestisci.
  3. Inserire nome utente e password dell'utente da verificare.
  4. Fare clic su Test Connessione.

Una richiesta viene sottoposta al server per verificare se l'utente può accedere al portale Intellisync. Un messaggio indicherà se il test ha avuto esito positivo.

  • Per visualizzare gli assembly e il loro numero di versione
  1. Fare clic destro sull'icona Server Web personale di Sage SalesLogix nella barra di sistema, poi su Proprietà.
  2. Nella finestra di dialogo Server Web personale di Sage SalesLogix selezionare il portale Intellisync nel campo Sito Web.
  3. Fare clic su Dettagli.

Saranno visualizzati gli assembly installati per il portale Intellisync e il loro numero di versione.

  • Per confrontare le versioni del portale
  1. Fare clic destro sull'icona Server Web personale di Sage SalesLogix nella barra di sistema, poi su Versioni Fianco a Fianco.
  2. Sarà visualizzata la versione del file sage.saleslogix.dll per ciascun portale installato.
