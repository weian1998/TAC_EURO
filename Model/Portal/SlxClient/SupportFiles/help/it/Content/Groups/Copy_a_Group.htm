Copiare un Gruppo

E' possibile selezionare un gruppo esistente e crearne una copia. E' possibile mantenere le condizioni esistenti del gruppo o modificarle per soddisfare le proprie necessità. Non è consentito copiare i gruppi ad hoc.

Per copiare un gruppo esistente

  1. Nella vista Elenco fare clic destro sulla scheda di un gruppo.
  2. Fare clic su Copia.
    Si apre il Query Builder.
  3. Per modificare il gruppo, utilizzare le schede del Query Builder.
    • Scheda Proprietà - Consente di definire il nome e la descrizione del gruppo.
    • Scheda Condizioni - Consente di impostare le istruzioni condizionali utilizzate per selezionare i dati. Questa scheda non è disponibile per i gruppi ad hoc.
    • Scheda Layout - Consente di definire quali dati sono mostrati nella vista Elenco.

      Il layout predefinito è determinato dal gruppo selezionato come Gruppo Predefinito nelle opzioni di Gruppo.

    • Scheda Ordinamento - Consente di definire la modalità di ordinamento dei dati.
  4. Al termine delle modifiche, fare clic su OK.
