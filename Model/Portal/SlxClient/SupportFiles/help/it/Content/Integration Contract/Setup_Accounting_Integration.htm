Configurare l'Integrazione Contabilità

La finestra Configurazione Integrazione Contabilità per definire i dettagli dei feed di contabilità.

Campo Descrizione/Azione
Feed Contabilità

Digitare l'URL SData del sistema di contabilità. Utilizzare il seguente formato: http://server:port/sdatavirtualdir/ applicazione/contratto/compagniaoperante /

  • Inserire al posto di applicazione il nome della propria applicazione di contabilità. Ad esempio Sage1000.
  • Al posto di contratto inserire gcrm.
  • Al posto di compagniaoperante inserire il nome della compagnia operante nella propria applicazione di contabilità con cui si sta eseguendo l'integrazione.
Nome Utente Digitare il nome utilizzato per connettersi al sistema di contabilità.
Password Digitare la password corrispondente al nome utente.
Esegui Test di Feed Fare clic per eseguire il test e visualizzare lo stato di connessione.
Stato Visualizza i risultati del test del feed di contabilità.
Nome Visualizzato Digitare il nome che sarà visualizzato per il sistema di contabilità. Questo nome sarà mostrato nel Client Web nel momento in cui gli utenti collegano i record tra Sage SalesLogix e il sistema di contabilità.
Abilita Sincronizzazione, Collegamento e Transazione Spuntare o deselezionare per attivare o disattivare questo feed. Quando il feed è disattivato, il sistema di contabilità non è disponibile per il collegamento o la sincronizzazione da/a SalesLogix.
Filtra per Risorsa

Selezionare un tipo di risorsa e fare clic sul pulsante Esegui per visualizzare le differenze di schema nella griglia.

Questa funzionalità confronta lo schema della risorsa selezionata nei due sistemi integrati e restituisce qualsiasi differenza delle lunghezze di stringa. Se i valori non corrispondono, utilizzare Database Manager o qualsiasi altra applicazione che consenta la modifica dellos chema in uno dei sistemi.

Se questo non è il primo feed di sistema di contabilità da essere collegato, è necessario ripristinare IIS sul server Web in modo da poter includere, durante la sincronizzazione, qualsiasi numero di nuovi feed.
