Configurare il Servizio Sincronizzazione Integrazione Contabilità

Il Servizio Sincronizzazione Integrazione Contabilità (SalesLogix System Service) consente il trasferimento di dati tra Sage SalesLogix e il proprio sistema di contabilità. Il servizio deve essere avviato e configurato in modo da essere eseguito automaticamente.

Per configurare

  1. Assicurarsi di aver configurato e abilitato i punti finali prima di avviare il servizio.
  2. Controllare il file Sage.SalesLogix.SyncEngineService.exe.config per assicurarsi che la connessione sia definita correttamente al termine del file. Le informazioni di connessione dovrebbero corrispondere a quelle di connection.config rilasciate per il proprio sito Web.

Per impostazione predefinita il file di configurazione è in ...\Program Files\SalesLogix\Sage.SalesLogix.SyncEngineService.exe.config e il file di connessione è rilasciato in ...\inetpub\wwwroot\slxclient

  1. Nella Workstation di Amministrazione aprire Servizi.
  2. Fare clic destro su SalesLogix System Service, quindi su Proprietà.
  3. Nella casella Tipo di avvio selezionare Automatico.
  4. Impostare il servizio per connettersi ed eseguire come utente WebDLL.
  5. Assicurarsi di aver abilitato Autenticazione Windows per l'utente WebDLL.
  6. Fare clic su Avvia.
  7. Fare clic su OK.
