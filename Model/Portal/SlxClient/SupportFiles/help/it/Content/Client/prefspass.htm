Modificare la Password

La password è modificabile attraverso le Opzioni nel Sage SalesLogix Web Client.

Per cambiare la propria password

  1. Nel menu Strumenti fare clic su Opzioni.
  2. Fare clic sulla scheda Password e poi inserire la nuova password.
  3. Reinserire la nuova password nel campo Conferma Password.
  4. Fare clic su Salva..
