Modificare la Password (Amministratore)

E' possibile modificare la password di un utente nella vista Utenti e tramite le Opzioni Utente.

Per saperne di più...

Gli utenti appena creati hanno una password predefinita, password. L'amministratore di sistema, l'utente Amministrativo o l'utente Sage SalesLogix possono modificarle in qualsiasi momento.
  • Le password devono iniziare e terminare con un carattere numerico (0-9) oppure alfabetico (A-Z oppure a-z).
  • Un punto e virgola non è un carattere valido per una password.

Per modificare una password in Opzioni Utente

  1. Nel menu Strumenti fare clic su Opzioni.
  2. Fare clic sulla scheda Password e selezionare un utente dall'elenco fornito.
  3. Digitare la nuova password.
  4. Reinserire la nuova password nel campo Conferma Password.
  5. Fare clic su Salva.. Compare un messaggio che avvisa che la password è stata cambiata con successo.

    		6.

    E' necessario disconnettersi dal Client Web di Sage SalesLogix e poi riconnettersi per rendere effettive le modifiche.

Per modificare una password nella vista Utenti

  1. Sulla barra di navigazione fare clic su Amministrazione.
  2. Fare clic su Utenti poi sul nome utente appropriato.
