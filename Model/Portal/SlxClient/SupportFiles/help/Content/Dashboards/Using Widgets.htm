Using Widgets

Sage SalesLogix provides several out-of-the-box widgets. You can add, edit, and remove widgets from the dashboard, and you can rearrange widgets on the page with drag-and-drop.

Using pre-defined dimensions, which are a type of filter, you can display metrics for your data across ranges or from distinct values and display this data in one of the charting widgets. For example, you can configure a column chart widget to display all the opportunities closed and won, by owner and the total amount of the sale, or a pie chart depicting sales won by a team.

How Do I?

Add a widget
  1. Right-click the My Dashboard tab, and click Add Content.
  2. On the Add Content dialog box, click the Add button for the widget you want to add.
    For example, to add a pie chart widget, click Add under Pie Chart.
    The blank widget appears on the dashboard and the widget editor opens.
  3. Configure the widget settings, and click OK.
Edit an existing widget
  1. On the widget toolbar, click the Settings button.
  2. Make the changes necessary.
  3. Click OK.
    The widget displays the changes.

      • Y-Axis - The label used to describe the data on the X axis of the chart.

      • Truncate Labels - Select Yes or No to truncate labels so that they fit on the chart.

Configure the Group List widget
  1. On the Group List Settings dialog box, type a title for the widget.
  2. Select an Entity and a Group from the drop-down lists. For example, Opportunities, and Closed Won.
  3. Type the Number of Records you want in the list. For example, 10, 25, or 50.
  4. Type the number of records you want visible in the Visible Records box; 10, 25 or 50.
    For example, if you typed 50 records in step 2, but only want to see ten at a time in the list, type 10. The widget resizes to accommodate the number of visible records.

    You can use any number between 1 and the maximum number of records configured to display. The default maximum is 50.

  5. Click OK.
    The widget displays the list.

    To show or hide columns in a group list, right-click any column header and select the columns you want to add and clear the columns you want to remove.

Configure the Links List widget
  1. On the Link Settings dialog box, type a title for the widget.
  2. Click Add Link.
  3. Double-click the Icon URL box, and type the path to the icon you want to display for the link. For example, /google.jpg.

    		4.

    This URL must be a relative path to an image on the Web server.
  4. Double-click the Link URL box, and type the URL for the link you want to display. For example, http://localhost/SlxClient/Account.aspx.

    5. This path can be a relative path within the SLXClient portal, or an absolute link to an external site. The URL for an external site must include the http:// designator. For example, to access www.google.com the URL must read http://www.google.com. Otherwise the browser will search for http://<SLXClient site>/www.google.com.
  5. Double-click the Link Text box, and type the text you want to display as the link. For example, Google Search Page.
  6. Double-click the Target box, and type an identifier for the target. For example, if you want the link to open in a new browser window, type New Window. When you click the link, a new browser window/tab opens.

    		7.

    If you leave this field blank, the link opens in the active browser window/tab when clicked. If several links have the same target designation, each link replaces the other as you click them.
  7. Click Add Link to add another link, or click OK to close the editor.
    When you click OK, the widget refreshes to display the links.
Minimize a widget
  • On the widget toolbar, click the Minimize  button.
    The widget content is minimized in the dashboard. Click the button again to maximize the widget.
Remove a widget
  • On the widget toolbar, click the Close  button.
    The widget is permanently removed from the dashboard.
