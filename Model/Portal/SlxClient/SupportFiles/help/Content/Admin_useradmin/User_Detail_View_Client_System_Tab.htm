Client System Tab

Use the Client System tab on the User Detail view to define a user's base templates and to set the default owner of accounts created by this user. Users can change these settings on the General/Search tab in Options.

To set options

  1. In the User Detail view, click the Client System tab.
  2. Use the Default Owner section to set a default owner for new accounts.
  3. In the Accounts box, select the default owner of new accounts created by this user.
  4. Clear the Allow user to change these settings check box to restrict the user from changing the default owner setting. Tell me more...

If you allow a user to change this setting, the ownership of an account can default to an owner that does not adhere to established business rules. This may leave accounts out of established reports or queries.

  1. Use the Base Templates section to set the default e-mail, letter, and fax templates.
  2. E-mail - sets the default e-mail template to be used when writing e-mail messages.
  3. Letter - sets the default template to be used when writing a quick letter.
  4. Fax - sets the default template to be used when sending a fax.
  5. Clear the Allow user to change these settings check box to restrict the user from changing the default templates.
  6. Click Save.

When configuring WebViewer users, only the Default Owner section is available.
