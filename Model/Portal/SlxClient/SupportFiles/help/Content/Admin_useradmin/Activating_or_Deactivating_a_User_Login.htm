Activating or Deactivating a User Login

You can set the login status of a user as active or inactive. New users are active by default. When a user login is inactive, they cannot access the Sage SalesLogix database. However, an inactive user’s profile is not deleted and a license remains assigned to that user.

Before you deactivate a user, reassign the user’s accounts using territory realignment in the Administrator. To reassign accounts, the user must be active. You can leave the user deactivated indefinitely.

If you need to release the license assigned to the user, retire the user.

To activate or deactivate

  1. Open the User Detail view for the user you want to activate or deactivate.
  2. Select or clear the Login Enabled check box.
  3. Click Save.
