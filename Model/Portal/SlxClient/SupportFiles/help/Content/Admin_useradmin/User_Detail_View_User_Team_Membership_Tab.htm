User Team Membership Tab

User teams allow users to access accounts owned by another user. For example, if you want Lee to have access to all of Dan’s accounts, you can add Lee to Dan’s user team.

The user whose profile you are editing will have access to the accounts owned by users listed on this tab.

How Do I?

Open the User Team Membership tab
  • In the User Detail view, click the User Team Membership tab. If the tab is not visible, click the More Tabs tab.
Add the user to a team
  1. To add a user, click Add this user to another user team.
  2. Use the lookup to find and select the user you want to add.

  3. Click OK.

Edit the security profile for the main user on the selected user's team
  1. Click the Security Profile link and set permissions in the dialog box.
  2. Click OK.
    This security profile only applies when the main user (the user whose profile you are editing) accesses accounts owned by the selected user. It does not apply when the selected user accesses accounts for which they are the owner.
Remove the user from a team
  1. Select the record you want to remove and click the Remove button.
  2. Click OK.
