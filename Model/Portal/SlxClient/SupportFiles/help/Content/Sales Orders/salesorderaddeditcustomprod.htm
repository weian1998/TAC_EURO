Adding or Editing a Custom Product

You can add or edit a custom product from the Sales Order Product tab.

You cannot add a custom product from the Insert Sales Order view. Use the Sales Order Detail view to add custom products.

To add or edit

  1. Open the Sales Order Products tab.
  2. Do one of the following:
    • Click the Add Custom Product button.
    • Click the Edit button button for the custom product you want to edit.
  3. In the Custom Product dialog box, set the following:
    • Product - Type a name for the custom product.
    • Description - Type a product description.
    • SKU - Type the product SKU.
    • Family - Select the family the product will be associated with.
    • Price - Type the price of the product.
    • Discount - Type the discount percentage for the product.
    • Adjusted Price - Displays the base price of the product, minus the discount.
    • Quantity - Type the number of products for this sales order.
    • Extended - Displays the calculated total of the product price multiplied by the quantity, less any discount. This calculation is based on the default currency as defined by your administrator.
    • Price Locked - Select to lock the product's price. You must have the appropriate role assigned to lock prices. (Accounting Integration Only)

  4. Click OK.
