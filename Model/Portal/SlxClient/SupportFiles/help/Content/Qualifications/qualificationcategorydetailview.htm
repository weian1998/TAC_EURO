Using the Qualification Detail View

Only the system administrator and users with an administrator role can access these features.

Use the Detail view to see, add, or edit information for a selected record.

How Do I?

Open the Qualification Detail view
  • On the Navigation Bar, click Administration, and then click Qualifications to open the Qualifications List view. If necessary, use the Lookup to locate a qualification, and then click on the qualification name.

Use the Lookup to search for records

Switch to the List view
  • On the toolbar click the List View button.

Use the Task Pane group list or common tasks

View a group

Information Boxes

The Description box provides a brief description of what is being qualified.

Detail View Tabs

The Items tab shows all of the items associated with a qualification. Click a link to see more information about each tab.

  • Items

user defined section

You can drag and drop tabs into the user defined middle section. The tab will remain there until it is moved back to the lower section.
