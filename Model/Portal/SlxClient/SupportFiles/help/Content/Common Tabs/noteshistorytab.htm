Notes/History Tab

The Notes/History Tab displays on the Account, Contact, Opportunity and Lead Detail views. The tab contains filtering options that allow you to quickly access important information. You can also use this tab to e-mail a history record or create a Microsoft Word document.

The default Notes/History tab displays all interactions such as meetings, phone calls, to-dos, mail merge items, e-mail messages logged to history, and literature requests. It also displays notes associated with each interaction.

How Do I?

Open the Notes /History tab
  • From the Contact, Account, Opportunity or Lead Detail view, click the Notes/History tab. If the tab is not visible, click the More Tabs tab.
Apply filters
  1. Click Filter to display the filter options.

You may select more than one filter option. For example, you can check the My History and Date Range From options to view all your history items within a certain date range.

  1. Select one or more of the following filters:
  • To filter the list by record type, in the Type list, select the type of records you want to display.
  • Database changes are hidden by default. To view database changes, select the Show Database Changes option.
  • To filter the list using a date range, click in the Date boxes and use the calendars to select beginning and end dates.
  • To filter the list by user, type the user's first or last name in the User box.
  • To filter the list by description, type the first few letters in the Description box.
  1. Click Apply.
  2. To clear filters, click Reset.
Preview note or history details
  • When you select a record in the list, the details of that record display in the preview pane to the right of the list.
Open a note or history record
  • Click the Type link of the item you want to view.
E-mail history records
  1. Select the record(s) that you want to e-mail.
  2. Click Send via E-mail.

An e-mail message box opens and includes the contact information, subject, and history record information. If you select multiple records, the information for each record is included.

  1. Click Send.
Send history items to Microsoft Word
  1. Select the record(s) you want to copy to a Word document.
  2. Click Send to Word.

Microsoft Word opens and displays a document with the note or history record information.

  1. You can print the document or save the file.
Add a note
  • On the Notes/History tab, click Add Note and complete the steps to add a note.

Complete an activity to add as a history item
