Timeline

A Timeline shows the occurrence and duration of events, such as activities, history, tickets and opportunities over a period of time (past, present, and future).

The Timeline shows events in two scales, called bands. The lower band shows events in the year in which they occurred. The upper band shows a particular timeframe in greater detail, providing additional information about each event.

Use your mouse to scroll forward or backward through the Timeline. The timeframe that displays in the upper part of the view is highlighted in the lower part of the view with a lighter background color.

Timeline events display differently, depending on the type of event and how the event is associated with the record. The following table describes how each event type displays.

Timeline Event Type Icons Description
Activities   - Phone Call

- Meeting

- To-Do

- Personal Activity

- Ticket Activity
  • Icon identifies the activity type. For example, phone call.
  • Activities associated with an opportunity display with the opportunity description in green text.
  • Activities associated with a ticket display with the ticket ID in red text.
  • Activities not associated with a ticket or opportunity display in blue text.
  • Open activities display in darker text.
History   - Letter

- Fax

- E-mail
  • Icon identifies the history item type. For example, e-mail.
  • History items display in purple text.
Opportunities
  • Icon displays.
  • Opportunity description displays in green text.
  • Open opportunities display in darker green text.
  • A green line represents the length of time that the opportunity was or has been open.
Tickets
  • Icon displays.
  • Ticket ID displays in red text.
  • Open tickets display in darker red text.
  • A red line represents the length of time that the ticket was or has been open.
Notes
  • Icon displays.
  • Notes associated with a ticket display with the ticket ID in red text.
  • Notes not associated with a ticket display in purple text.

How Do I?

Open a Timeline view
  • On the Ticket Detail view, click the Ticket Timeline tab or on the Account Detail view , click Account Timeline.If the tab is not visible, click the More Tabs tab.
View Timeline event details
  • Click an event on the Timeline.

Detailed information about the event displays in a separate window. To open an event, click the description information at the top of the window.
