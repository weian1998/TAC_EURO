Additional Help for Application Setup

Additional information about configuring the following settings is available in the help topics available through the Intellisync for SalesLogix Advanced Setting Wizard.

  • Creating Filters
  • Mapping Fields
  • Providing Login Information
  • Receiving confirmations
  • Selecting a desktop application
  • Specifying filter conditions and rules
  • Specifying what to sync
  • Specifying how to sync
  • Specifying when to sync
  • Specifying Outlook (or Lotus Notes) data exchange options
  • Specifying Outlook (or Lotus Notes) task options
  • Specifying Outlook (or Lotus Notes) calendar options

The Intellisync setup interface Help buttons call topics created by Intellisync's creator, Nokia. Some of the terminology in these help topics may differ from the terms used in a Sage SalesLogix implementation.
