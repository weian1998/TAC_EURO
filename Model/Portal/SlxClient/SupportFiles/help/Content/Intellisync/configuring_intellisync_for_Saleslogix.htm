Configuring Intellisync for SalesLogix

Synchronization preferences must be configured on each user’s computer. To sync, an ad hoc group must exist in Sage SalesLogix, and be configured as the sync group. To configure Intellisync, you will need the URL for the Intellisync for SalesLogix portal.  Contact your system administrator for more information.

You can configure different settings for different types of records. For example, you can enable one-way sync for contacts and two-way sync for calendar items, and set them to sync at different times. The following steps outline the recommended synchronization settings between Microsoft Outlook or Lotus Notes and Intellisync for SalesLogix.

The Intellisync setup interface Help buttons call topics created by Intellisync's creator, Nokia. Some of the terminology in these help topics may differ from the terms used in a Sage SalesLogix implementation.  See the topic Additional Help for Application Setup for more information.

To configure

  1. Start Intellisync for SalesLogix.
    • Click Start, point to Programs, point to Sage SalesLogix, point to Intellisync for SalesLogix, and then click Intellisync for SalesLogix.
  1. Click Setup and do the following.
    1. In the User ID box, type your Sage SalesLogix user name.
    2. In the Password box, type your password.
    3. In the Connection URL box, type the path to your Intellisync for SalesLogix portal.
      Contact your system administrator for the URL to the portal location.
  2. Click Application Setup.
  3. On the Setup screen, you can select or clear the check boxes under Active for each of the types of data. Active means the data is synced whenever a sync occurs.

    In order to limit the contacts to be synchronized to Sage SalesLogix, Sage SalesLogix recommends creating a SyncSalesLogix filter.

  4. Click the Setup Wizard button.
  5. On the Desktop Application screen, select Microsoft Outlook or Lotus Notes, and click Next.
  6. On the What to Sync screen, select All Intellisync for SalesLogix applications that are supported by Microsoft Outlook or Lotus Notes, and click Next.
  7. On the How to Sync screen, select Two way sync, and click Next.
  8. On the When To Sync screen, select when you want synchronization to occur, and click Next.
    • Every 10 minutes – select an increment from the drop-down list.
      The user starts Intellisync manually, PushSync starts, and syncs data at the interval specified.
    • Every day at a specified time - select an increment from the drop-down list.
      The user starts Intellisync manually, PushSync starts, and syncs data at the interval specified.
    • When I log onto my computer – When the user logs in, Intellisync starts and runs in the systray.
    • When a change in Lotus Notes/Outlook is detected - Intellisync starts and runs in the systray
    • When I connect to Intellisync for SalesLogix Server.
  9. On the Intellisync for SalesLogix Connection Information screen, select your user profile , and click Next.
  10. Complete additional options/screens as appropriate for the type of information you are synchronizing for either Outlook or Lotus Notes.
    1. On the Options for ToDo’s screen for either Outlook or Lotus Notes, select Transfer all items and click Next.
      This options synchronizes all active, non-completed, and completed ToDo’s.
    2. On the Options for Appointments screen, select Transfer only future items, and click Next.
    3. If you are configuring Intellisync for Lotus Notes, on the Options for Contacts screen, browse to the Lotus Notes database and click Next.
  11. On the Intellisync for SalesLogix Connection Information screen, verify your Sage SalesLogix user ID, password, and the connection URL, and then click Next.
  12. Click Finish.
  13. Click Close.
  14. Click OK.
  15. If you have not already done so, you must select the SyncSalesLogix group.
