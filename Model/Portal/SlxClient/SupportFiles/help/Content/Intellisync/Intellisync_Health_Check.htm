Intellisync Health Check

Intellisync for SalesLogix includes a troubleshooting option called Intellisync Health Check. You can troubleshoot logging into the portal, and check the version of the assemblies. In most cases, checking the assembly versions is only done at the request of Sage Technical Support.

You can access the Intellisync Health Check from the IIS Manager, the Intellisync portal and the Sage SalesLogix Personal Web Server.

To access from IIS Manager

  1. Click Start, point to Programs, then point to Administrative Tools.
  2. Click Internet Information Services (IIS) Manager.
  3. Expand the machine name, expand Web Sites, then expand SalesLogix.
  4. Double click SLXIntellisync.
  5. Do one of the following:
  • Windows 2003: In the list of files in the right pane, right-click Default.aspx and click Browse.
  • Windows 2008: In the list of files in the right pane, right-click Default Document, point to Manage Application, and then click Browse. The browser displays the Intellisync portal. If it does not display, ensure the settings are configured correctly and that the portal has been deployed.
  1. In the browser, click Diagnostics, and do one of the following:
  • To test user credentials, type the User Name and Password for the user you want to test and then click Try to Connect. A request is submitted to the server to test whether the credentials are valid for the portal. A message indicates whether the test is successful.

    • This test only determines whether the credentials are successful in logging into the portal. This does not test whether there is a working database connection.

  • To view the assemblies and their version number, click the forward and back buttons at the bottom of the list.

To access from the portal

  1. Open the Intellisync portal.

For example, type the portal URL into the browser using the format: http://server:port/portal name/default.aspx

  1. Click Diagnostics.
  1. Do one of the following:
  • To view assemblies and their version number, click the forward and back buttons at the bottom of the list.
  • To test user credentials, enter the User Name and Password for the user you want to test and then click Try to Connect.

A request is submitted to the server to test if the user can log on to the Intellisync portal. A message indicates if the test was successful.

To access from the Personal Web Server

  • To test user credentials
  1. Right-click the Sage SalesLogix Personal Web Server icon in the system tray, and click Properties.
  2. In the Sage SalesLogix Personal Web Server dialog box, click Manage.
  3. Enter the User Name and Password for the user you want to test.
  4. Click Test Connection.

A request is submitted to the server to test if the user can log on to the Intellisync portal. A message indicates whether the test is successful.

  • To view assemblies and their version number
  1. Right-click the Sage SalesLogix Personal Web Server icon in the system tray, and click Properties.
  2. In the Sage SalesLogix Personal Web Server dialog box, select the Intellisync portal in the Web Site box.
  3. Click Details.

View the installed assemblies for the Intellisync portal and their version number.

  • To compare portal versions
  1. Right-click the Sage SalesLogix Personal Web Server icon in the system tray, and click Side by Side Versions.
  2. View the version of the sage.saleslogix.dll file for each installed portal.
