Using the Returns Tab

The Returns tab on the Account and Ticket Detail views lists all returns for the selected record.

To open

  1. Open the Account or Ticket Detail view.
  2. Click the Returns tab.If the tab is not visible, click the More Tabs tab.

To add

  1. Click Create Return.
  2. Complete the steps for adding a return.

To see more information about a return

  • Click the Return ID to open the Return Detail view.
