Editing Ticket Activity Parts

Use this view to manage parts in relationship to the ticket activity.

To edit

  1. Do one of the following:
    • Open the Ticket Activity Parts Tab and click Edit Part next to the part you want to edit.
    • Open the Ticket Activity view and in the Ticket Activity Parts area, click Edit next to the part you want to edit.
      • The Edit Ticket Activity Part dialog box opens.
  2. In the Serial Number box, type the serial number that identifies the part.
  3. In the Price box, type the cost of the part.
  4. In the Description box, type a brief description of the part.
  5. Click OK.
