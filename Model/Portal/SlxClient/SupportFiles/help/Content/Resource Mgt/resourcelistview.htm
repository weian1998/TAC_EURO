Using the Resource List View

Only the system administrator and users with an administrator role can access these features.

The Resources List view displays all of the resources that you have access to. You can use groups and lookups to narrow the list of records that display.

How Do I?

Open the Lead Sources List View
  • On the Navigation Bar, click Administration, and then click Resources.

Use the Lookup to search for records

Open the Resource Detail View
  • Click the resource name to open the Resource Detail view.

View a group

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Save Records as a Group
  • Export to File
