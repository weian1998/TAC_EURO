Using the Defect Tasks Tab

Use this tab to see all activities associated with the defect, to add or change time for working on a defect, or edit an activity. Activities can be any action involved in processing or resolving the defect. For example, if you need to research for the defect, add this activity here.

The Defect Tasks tab displays:

  • Activity Type
  • User
  • Received Date
  • Completed Date
  • Rate Type
  • Time Units
  • Elapsed Hours
  • Total Fee

To open

  1. Open the Defect Detail view.
  2. Click the Defect Tasks tab. If the tab is not visible, click the More Tabs tab.

To add an activity

  1. Click Add Task.
  2. Complete the information in the Insert Defect Task information boxes.
  3. Click OK.

To edit an activity

  1. Click Edit next to the activity you want to edit.
  2. Edit the information boxes.
  3. Click OK.

To delete an activity

  1. Select the item you want to remove, and then click Delete Activity.
  2. Click Yes.
