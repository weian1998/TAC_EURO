Using the Defect List View

The Defect List view displays all of the defects that you have access to. You can use groups and lookups to narrow the list of records that display.

How Do I?

Open the Defect List view
  • On the Navigation Bar, click Defects.

Use the Summary View

Use the Lookup to search for records

Open the Defect Detail view
  • Click the defect's ID to open the Defect Detail view.
Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Save Records as a Group
  • Export to File
  • Promote to Dashboard
