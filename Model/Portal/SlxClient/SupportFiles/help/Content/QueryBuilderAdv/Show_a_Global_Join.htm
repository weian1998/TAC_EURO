Showing a Global Join

In Sage SalesLogix, a number of global joins are set up for you. However, not all of them are visible when the Query Builder is opened due to space constraints. If one of these joins already exists when you are using the Query Builder to select data, you should use it rather than creating a new join.

To add a local join

  1. Open the Query Builder.
  2. If necessary, use the plus sign to expand the list of tables on the upper left side of the Query Builder. Select a table.
  3. Click Global Join.

The Global Join Manager dialog appears, displaying existing global joins.

If a global join is not available, you can add a local join.
