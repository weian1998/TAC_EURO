Editing a Global Join

 System Administrator Only

A global join is permanent and exists from session to session in the Query Builder data tree structure. With predefined global joins (system joins), you can only edit the Join Type, Visibility, and Cascade Type options.

To edit

  1. Open the Query Builder, and then click the Global Joins button.

This button only appears if you are logged on as Admin.

  1. In the Global Join Manager dialog box, select the join to edit and click Edit.
  2. Change the Join Type, Visibility, or Cascade Type.
  3. Click OK..
