Deleting a Global Join

 System Administrator Only

A global join is permanent and exists from session to session in the Query Builder data tree structure. You cannot delete predefined global joins (system joins).

To delete a global join

  1. Open the Query Builder, and then click the Global Joins button.

This button only appears if you are logged on as Admin.

  1. Select the join you want to delete.
  2. Click Delete.
  3. Click OK.
