Troubleshooting Desktop Integration Module

You may encounter the following issues with the Desktop Integration module.

Problem:

How do I know if I have the correct version of the Desktop Integration module installed?

Solution:

  • View the Sage SalesLogix Web Client login screen or in Options, click the General/Search tab.

If the text near the Enhance SalesLogix button says "Enhancements have been installed." then the correct version is installed. If the text says "Click to download product enhancements." then you must install the latest version of the Desktop Integration module.

Problem:

The Sage SalesLogix Desktop Manager icon does not appear in my SysTray.

Possible Causes and Solutions:

  • The Desktop Integration Module is not installed.
  • You must install the Desktop Integration Module.
  • The Desktop Integration Module was installed before your browser was installed.
  • If you installed your current browser after you installed the Desktop Integration Module, you must reinstall the Desktop Integration Module using your current browser.

    		•

    This can occur if you installed Firefox after installing the Desktop Integration Module using Internet Explorer.
  • Browse to the SLXDesktopManager.exe and open it manually. The location of this file depends on your operating system:

The default location is:

\\Documents and Settings\[username]\Local Settings\Application Data\ (XP/2000/2003) or

\\Users\<user.domain>\appdata\local (Vista/7/2008).

Problem:

How do I turn off the Desktop Manager?

Solution:

  • In the System Tray section of the Windows Taskbar, right-click the Sage SalesLogix Desktop Manager icon and click Exit.

You must first exit Sage SalesLogix Desktop Manager before you can uninstall the Sage SalesLogix Desktop integration Module.

Problem:

I was prompted to download a new version of the Desktop Integration Module.

Solution:

  • Download the new version of the Desktop Integration Module.

    • If you use Firefox as your browser, you must first uninstall the Sage SalesLogix Desktop Integration Module Add-on in Firefox.

    To uninstall the Desktop Integration Module add-on, open Firefox. On the Tools menu, click Add-ons, select Sage SalesLogix Desktop Integration Module and click Uninstall. Click Uninstall to confirm. Click Restart Firefox. Click Restart.

Problem:

How do I uninstall the Desktop Integration Module?

Solution:

  1. Open Windows Add or Remove Programs.
  2. Select Sage SalesLogix Desktop Integration Module from the list and click Remove.
  3. Click Yes.

    		4.

    If you use Firefox as your browser, you must first uninstall the Sage SalesLogix Desktop Integration Module Add-on in Firefox.

    To uninstall the Desktop Integration Module add-on, open Firefox. On the Tools menu, click Add-ons, select Sage SalesLogix Desktop Integration Module and click Uninstall. Click Uninstall to confirm. Click Restart Firefox. Click Restart.
