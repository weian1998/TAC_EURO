Viewing and Printing Detail View Reports

Some detail views have a detail report that you can view or print. You can view and print detail reports from the Contact, Account, Opportunity, Ticket, and Defect detail views.
  • This feature requires Sage SalesLogix Web Reporting to be installed and configured on your Web Host.
  • Detail reports are not be available for all detail views.

To view a report

  1. Open the detail view of the record you want to view or print.
  2. In the Task Pane, under Common Tasks, click Detail Report.
  3. If necessary, set the report prompts. Click OK.

Setting the paging prompt will not display multiple detail reports. You can only view the detail report of the selected record.

To print a detail report

  1. In the SalesLogix Crystal Report Viewer, click the Print button on the toolbar.
  2. Select one of the Page Range options for the number of pages you want to print. Select All to print the entire report, or select the Pages option and enter the first page number in the From box, and then enter the last page number in the To box.
  3. Click OK. Acrobat Reader opens and displays the report you selected.
  4. Click Print. If the report does not print correctly, you may need to reformat it for your printer. The report orientation settings will override the orientation in the printer settings.
