Using the Group Shortcut Menu

The groups shortcut menu enable you to create and manage groups.

Not all menu options are available on all list views.

To open the groups shortcut menu

  1. On a list or detail view, right-click a group tab.
  2. Choose from the following options:
    • Copy
    • Edit
    • Hide
    • Delete
    • Share
    • Export to Excel
    • Export to File
    • Set as Default
    • Revert Layout

    Undoes any customizations made to the group and returns the group appearance to layout determined by the Query Builder Layout tab.

    • Add Group

To open the Lookup Results shortcut menu

  1. After performing a Lookup, right-click the Lookup Results tab.
  2. Choose from the following options:
    • Save Lookup as Group
    • Export to Excel
    • Export to File
    • Revert Layout

    Undoes any customizations made to the group and returns the group appearance to layout determined by the Query Builder Layout tab.

    • Add Group

To open the group list view shortcut menu,

  1. Select one or more records.
  2. Right-click in the group list view.
  3. Choose from the following options:
    • Save Records as Group
    • Remove from Group
    • Add to Existing Group

    • Update Leads (Lead List view only)

    Allows you to assign an owner to the selected records.

    • Delete Leads (Lead List view only)
    • Update Opportunities (Opportunity List view)
    • Opportunity Statistics (Opportunity List view)
    • Merge Records (Contact and Account List views only)
    • E-mail (Contact and Lead List views only)
