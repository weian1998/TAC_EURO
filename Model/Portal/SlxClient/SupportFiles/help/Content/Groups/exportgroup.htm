Exporting a Group to Excel

You can export groups to Excel. The group column headings and formatting will also be exported.

If the Export to Excel option is unavailable, you must download Sage SalesLogix Desktop Integration. You can install Sage SalesLogix Desktop Integration from the Web Client login screen. 

To export a group to a file

  1. View the group or lookup results that contain the records you want to export.
  2. Right-click the group tab and click Export to Excel.
    Microsoft Excel opens with the exported group. If the group contains currency information, Excel also creates a Pivot table. Excel also adds a total row to every column that contains currency information, so that you can view the group total.
  • Exporting a group to Excel will export the entire group regardless of any filters you may have applied.
  • You may receive an error if you try to export a group that includes a data field with a large amount of data.
