Using the Resolution Detail View

Only the system administrator and users with an administrator role can access these features.

Use the Detail view to see, add, or edit information for a selected record.

How Do I?

Open the Resolution Detail view
  • On the Navigation Bar, click Service, and then click Standard Resolutions to open the Resolutions List view. If necessary, use the Lookup to locate a resolution, and then click on the resolution ID.

Use the Lookup to search for records

Switch to the List view
  • On the toolbar click the List View button.

Use the Task Pane group list or common tasks

View a group

Information Boxes

The Description box provides a brief description of the resolution and the Notes box is for additional information.

Detail View Tabs

The Items tab shows all of the items associated with a resolution. Click a link to see more information about each tab.
