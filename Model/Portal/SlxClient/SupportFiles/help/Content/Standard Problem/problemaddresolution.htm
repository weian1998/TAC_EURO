Adding or Editing a Standard Problem Resolution

Only the system administrator and users with an administrator role can access these features.

Standard problem resolutions provide information for resolving a commonly occurring problem.

How Do I?

Add a resolution
  1. On the Standard Problem Detail view, open the Resolutions tab and click Add Resolution.
  2. Add the following information:
  • Description - Type a description for the resolution.
  • Notes - Type any notes to describe how to resolve the problem.

  1. Click OK.

Edit a resolution
  1. On the Standard Problem Detail view, open the Resolutions tab and click Edit next to the resolution you want to edit.
  2. Edit any of the following information:
  • Description - Type a description for the resolution.
  • Notes - Type any notes to describe how to resolve the problem.

  1. Click OK.
