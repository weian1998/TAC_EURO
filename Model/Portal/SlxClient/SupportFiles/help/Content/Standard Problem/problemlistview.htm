Using the Standard Problem List View

The system administrator and users with the appropriate role can access these features.

The Standard Problems List view displays all of the standard problems that you have access to. You can use groups and lookups to narrow the list of records that display.

How Do I?

Open the Standard Problems List View
  • On the Navigation Bar, click Service, and then click Standard Problems.
Refresh list information
  • Click the Refresh button.

Work with records in a grid

Filter my list

Use the Lookup to search for records

Open the Standard Problem Detail View
  • Click the standard problem ID to open the Standard Problem Detail view.

View a group

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Save Records as a Group
  • Export to File
