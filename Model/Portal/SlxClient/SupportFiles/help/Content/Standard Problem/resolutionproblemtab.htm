Standard Resolution Standard Problems Tab

Only the system administrator and users with an administrator role can access these features.

Use this view to view or associate standard problems with a resolution.

How Do I?

Open this view
  • On the Standard Resolution Detail view, click the Standard Problems tab.
Associate a resolution
  1. Click the Associate problem button.
  2. Use the lookup to find a standard problem.
Delete a resolution
  1. Click Remove next to the problem you want to delete.
  2. Click OK.

This view contains the following columns:

  • Description - Indicates the description of the problem.
  • Area - Indicates where the problem occurs.
  • Category - Indicates the type of problem.
  • Issue - Indicates the problem.
  • Notes - Indicates brief notes about the problem.
