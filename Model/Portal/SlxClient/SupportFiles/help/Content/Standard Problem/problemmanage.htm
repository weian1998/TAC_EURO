Managing Standard Problems

Only the system administrator and users with an administrator role can access these features.

Use this view to enter information that you want to be available to all users when adding tickets.

How Do I?

Add a Standard Problem
  1. On the New menu, click Standard Problem.
  2. Complete the standard problem information boxes.
  1. Click Save.
Edit a Standard Problem
  1. On the Navigation Bar, click Service, and then click Standard Problems to open the Standard Problems List view. If necessary, use the Lookup to locate a problem, and then click the standard problem ID.
  2. If necessary, change the standard problem information boxes.
  3. Click Save.
Delete a Standard Problem
  1. On the Navigation Bar, click Service, and then click Standard Problems to open the Standard Problem List view. If necessary, use the Lookup to locate a problem, and then click on the standard problem ID.
  2. On the Standard Problem Detail view, click the Delete button.
  3. Click OK.
