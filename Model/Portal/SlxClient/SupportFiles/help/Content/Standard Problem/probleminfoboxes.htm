Standard Problem Information Boxes

The standard problem boxes contain information about the selected problem. You can edit or delete the information as needed.

 

Box

Description/Action

Description

A brief description of the resource.

Area - Category - Issue

These three fields briefly describe the nature of the problem being reported. Click one of the Find buttons next to each box, select the appropriate items from the lists, and then click OK.

The Category items listed depend on the option you selected for Area and the Issue items available depend on the option you selected for Category.

Notes

 Type information that will help other users understand this resolution.
