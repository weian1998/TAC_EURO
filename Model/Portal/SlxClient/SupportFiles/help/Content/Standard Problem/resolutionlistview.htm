Using the Standard Resolutions List View

Only the system administrator and users with an administrator role can access these features.

The Standard Problems List view displays all of the standard problems that you have access to. You can use groups and lookups to narrow the list of records that display.

How Do I?

Open the Standard Resolutions List View
  • On the Navigation Bar, click Service, and then click Standard Resolutions.

Use the Lookup to search for records

Open the Standard Resolution Detail View
  • Click the standard resolution ID to open the Standard Resolution Detail view.
