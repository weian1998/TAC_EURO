Using the Standard Problem Detail View

Only the system administrator and users with an administrator role can access these features.

Use the Detail view to see, add or edit information for a selected record.

How Do I?

Open the Standard Problem Detail view
  • On the Navigation Bar, click Service, and then click Standard Problems to open the Standard Problems List view. If necessary, use the Lookup to locate a problem, and then click the standard problem ID.

Use the Lookup to search for records

Switch to the List view
  • On the toolbar click the List View button.

Use the Task Pane group list or common tasks

View a group

Information Boxes

The standard problem information boxes contain the primary information about the standard problem. Users with appropriate permissions can edit these boxes.  

Detail View Tabs

Each tab shows specific information related to the role. Click a link to see more information about each tab.

  • Standard Resolutions

user defined section

You can drag and drop tabs into the user defined middle section. The tab will remain there until it is moved back to the lower section.
