Related Documentation

In addition to this Help system, the following printed and online documentation is available:

  • Sage SalesLogix Quick Reference card provides commonly used buttons, e-mail configuration options, and tips for working with groups and Outlook Integration. The Sage SalesLogix Quick Reference card is accessible from the Help menu.
  • The Sage SalesLogix Implementation Guide provides step-by-step instructions for installing Sage SalesLogix. The Sage SalesLogix Implementation Guide is available on the Sage SalesLogix DVD.
  • The Administrator Help provides detailed instructions for managing user, teams, offices, Network bundles, and remotes. Some user management features are available in the Web Client, while full features and functionality are available in the Administrator.
  • The Sage SalesLogix Mail Merge Help provides detailed information on the following: writing an e-mail message to a contact, writing an e-mail message, letter, or fax using a template, and using the Mail Merge engine to send correspondence to a group of contacts, accounts, etc. This help system also explains how to create and print address labels or envelopes, as well as, how to create, edit, and delete templates.

Help is available in each Sage SalesLogix application. Additional resources and documentation are available to Partners and Customers with a valid technical support contract at the Support Portal Web sites:

  • Sage SalesLogix Partner Support Portal Web site: partners.sagenorthamerica.com.
  • Sage SalesLogix Customer Support Portal Web site: customers.sagenorthamerica.com.
