What's New In This Release

Sage SalesLogix Web Client contains the following updates:

Version 8.0

  • List view groups remember your sort and column width changes.
  • Lookup improvements include the ability to perform a Lookup from the left Navigation bar in order to look up any type of record and a new option to set a default lookup condition.
  • Alerts tell you when there is an activity alarm or when you have a new unconfirmed activity notification. You can set Alerts options to customize your alerts.
  • Improved Activities:
    • The Availability tab allows you to view other user's calendars and determine availability for other user's and resources.
    • A Location box allows you to enter a location for a phone call or meeting or automatically add the location if one is selected from the Availability tab.
    • Activity confirmations now provide more information to help you determine whether to accept or decline an invitation.
    • Updated Activity List view has new tabs, improved filters and Common Tasks. Use the My Activities tab to view activities where you are the leader or a member.
    • You can view contact and account or lead information, like a phone number, by hovering over the name in the activity detail view or Alerts view.
  • New and updated activity and calendar user options:
    • Options are now shared between the Sage SalesLogix Client and Sage SalesLogix Web Client. Some users may need to be reset their options.
    • New activity user options allow you to set defaults per activity type.
    • New calendar user options allow you to determine what activity information to show on the calendar, the days of your work week, and your first day of the week. You can also set an option to remember the users whose calendars you selected to view.
  • Calendar has been updated and includes new features such as:
    • Ability to customize which days comprise your work week
    • Ability to see up to four users on each calendar view
  • Move Contact and Duplicate Contact are now handled separately. The new methods allow you to determine what information is moved or reassigned.
  • New Mail Merge templates for leads.
  • Ability to share Mail Merge templates with other users.
  • The Opportunity menu has been removed. Update Multiple Opportunities and Opportunity Statistics are now available on the Opportunity List view right-click menu.
  • Accounting Integration now includes X3 integration.
  • Features for Administrators and those with appropriate role access:
    • The Security Manager allows you to manage field-level access within security profiles.
    • The Standard Role now grants view access to products and packages, instead of add, edit, or delete access.
    • Resource management allows you to identify if a resource is a location.

    • The Form Designer enables you to make simple changes to a view. For example, you can change captions, move things around, or add and hide information. For more information, refer to the help topic “Working with Forms”.
