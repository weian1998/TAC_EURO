Using the Secured Actions List View

Only the system administrator and users with an administrator role can access these features.

The Secured Actions List view displays secured actions in a grid. The grid provides information about each secured action, such as the secured action name and description.

  • To sort the grid by a particular field, click one of the column headings.
  • To display all secured actions of a particular type, click a tab in the grid.
  • To only display secured actions that meet specific criteria, configure filters.
