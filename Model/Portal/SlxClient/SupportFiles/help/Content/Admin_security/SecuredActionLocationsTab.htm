Locations Tab

Only the system administrator can access this feature.

The Locations tab in the Secured Action Detail view displays all elements that use the secured action. This information is read-only.

When you delete or rename a secured action, the secured action is de-coupled from the element in the Application Architect. Before modifying the secured action, Sage recommends you save the information found on this tab as a history record. You can copy and paste the information to a text file.

If necessary, you can use the information on this tab to recreate a secured action or to remove the action from elements in the Application Architect.
