Adding an Import Group Action Campaign Target

Use this view to add each imported lead as a campaign target. Each lead will appear on the selected Campaign Detail view Targets tab.

To add

  1. When importing leads, create an import action and click Define next to Add Campaign Target.
  2. Click the Campaign Find button to find a campaign.
  3. Click Save.
