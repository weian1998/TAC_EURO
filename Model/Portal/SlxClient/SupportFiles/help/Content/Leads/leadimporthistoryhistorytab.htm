Lead Import History - History Tab

Use this tab to view a list of records that encountered problems during the import. Records display in the tab if any of the following conditions occurred:

  • Error - The record encountered an error and was not imported.
  • Duplicate - The record was marked as a duplicate that must be resolved before it can be imported.
  • Merged -The record was imported and merged with another record.

To open

  • On the Lead Import History Detail view, click the History tab. If the tab is not visible, click the More Tabs tab.

This view contains the following columns:

  • Timestamp - Date and time the problem occurred.
  • Type - The problem that occurred.
  • Description - Description of the record, the problem encountered and any action that was taken.
  • Resolved - Indicates if the duplicate resolved.
  • Resolved Date - The date the duplicate was resolved.
  • Resolved Description - Description of how the duplicate was resolved. For example, Merged with.
