Saving an Import Template

You can save import information, such as matching fields, default lead owner and default lead source, and use it for future imports.

To save an import template

  1. After defining import map fields, click Save As Import Template.
    The Import Lead Template dialog box opens
  2. In the Description box, type a brief description of the import template, and then click Save As Import Template.
    The new import template description is added to the Import Template list on the Import Leads dialog box.
