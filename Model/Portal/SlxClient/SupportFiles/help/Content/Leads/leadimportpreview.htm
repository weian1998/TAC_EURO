Defining a Delimiter

Use this view to select the appropriate delimiter value for your list and preview records before you import them to Sage SalesLogix. A delimiter parses fields in a list. Sage SalesLogix analyzes the list to determine which delimiter is currently used, but you can select a different delimiter if necessary.

To define a delimiter and preview

  1. When importing leads, after selecting a file, click Next.
  2. If you need to change the delimiter, click one of the following options:
  • Tab
  • Semicolon
  • Comma
  • Space
  • Other
  1. If you selected Other, you must specify the delimiter used to separate fields in the list. Type the delimiter character in the Other box.
  2. From the Text Qualifier list, click the drop-down arrow and select a qualifier from the list.

A text qualifier identifies text that should be imported exactly as it appears.

  1. If the list contains field names, select the First Row Contains Field Names check box to display the field names in the Preview area.

If there are extended characters in the imported file that do not correctly display in the Preview area, you must resave the import file and explicitly define the encoding type as UTF-8, Unicode, or ANSI. After you resave the file, click the Back button to return to the Select a File step and reselect the updated file before continuing the import.
  1. Click Next to go to the next step.
