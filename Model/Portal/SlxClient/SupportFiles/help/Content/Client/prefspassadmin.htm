Changing a Password (Administrator)

You can change a user's password in the Users view and through User Options. Once you have made the changes you want, you must log off, and then log back in for changes to take effect.

Tell me more...

New users are created with a default password of password. The system administrator, Administrative user, or Sage SalesLogix user can change this password at any time.
  • Passwords must begin and end with a numeric character (0-9) or an alphabetical character (A-Z or a-z).
  • A semi-colon is not a valid password character.

To change a password in User Options

  1. Click Options on the menu bar.
  2. Click the Password tab, and select the user you want in the list provided.
  3. Enter the new password.
  4. Reenter the new password in the Confirm Password field.
  5. Click Save. A message appears stating the password successfully changed.
