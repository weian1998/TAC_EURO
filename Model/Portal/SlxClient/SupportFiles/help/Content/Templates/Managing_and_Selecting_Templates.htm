Managing and Selecting Templates

You can use the Manage Templates or Select a Template dialog box to access the Public and Private templates that are available to you.

How Do I?

Open the Manage Templates or Select a Template dialog box.

  • On the Write menu, select Templates.

Create a new template

Edit a template

Set or update the Properties for a template

Copy a template

Delete a template

Release a private template to another user

Change the template type of a private template
  1. In the Private Templates list, click the Letter, Fax, or E-mail folder to expand the list of templates.
  2. Right-click the template that you want to change. From the shortcut menu, point to Template Type, and then click one of the types.
    The default types are: Letter, Fax, E-mail.

Preview a template

  1. Select a template from the list, and then click Show Preview.

A secondary window opens displaying the Subject text, Attachment information, and body of the template.

  1. To preview the template with merged data, select the Show Template with Merged Data check box.
  2. To hide the preview, click Hide Preview.

Right-click on a template name to access the shortcut menu (Collapse All, Expand All, New, Edit, Properties, Copy, Delete, and Template Type).
