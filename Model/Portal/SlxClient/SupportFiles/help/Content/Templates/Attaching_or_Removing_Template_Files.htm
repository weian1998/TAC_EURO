Attaching or Removing Template Files

You can attach Windows files or Library files to a template. When the template is used in a mail merge, the attached files are automatically sent to the contact. If the template containing the attachment is copied, the attachment is copied also. You can also remove an attachment from a template.

To attach a file to a template

  1. Open the Template Editor.
  2. Depending on the type of file you want to attach, do one of the following:
  • Windows file: Click Insert File. Browse to the location of the file, and then click Insert File.
  • Library file: Click Insert SalesLogix Library File. In the SalesLogix Library dialog box, find the file(s), and then click Insert.

The file name appears in the Attachment(s) box.

  1. You can attach another file, or click Save and Close.

To remove an attachment from a template

  1. Open the Template Editor.
  2. In the Attachment(s) box, select the attachment you want to remove.
  3. Do one of the following to remove the attachment:
  • Right-click, and then select Remove.
  • Press Delete.
  1. You can remove another file, or click Save and Close to close the Template Editor.
