Sharing Templates with Other Users

You can share your templates with other Sage SalesLogix users, teams, or departments.

This feature is only available in the Sage SalesLogix Web Client.

How Do I?

Share a template
  1. On the Write menu, select Templates.
  2. Open the Manage Templates or Select a Template dialog box.
    • Sage SalesLogix Network and Web users: On the Write menu, select Templates.
    • Architect users: On the Manage menu, select Templates.
  3. In the Manage Templates dialog box, select the template you want to share from the list.
  4. Click Share.
  5. Click the Add button.

  6. Do one of the following:

    • Click the Users tab. To limit the users listed, select a type of user from the User Type drop-down list (for example, Remote).
    • Click the Departments tab.
    • Click the Teams tab.To limit the teams listed, select the My Teams check box.

    • If you are logged in as Administrator, the My Teams check box will be unavailable.
  7. Select from the list and then click OK.
Remove access to a shared template
  1. On the Write menu, select Templates.
  2. In the Manage Templates dialog box, select the template you want to remove access to from the list.
  3. Click Share.
  4. Select the user, department, or team you no longer want to share your template with.
  5. Click the Remove button, and then click OK.
Update a shared template

If you make changes to a template and want to share the changes, you must share the template again.

  1. On the Write menu, select Templates.
  2. In the Manage Templates dialog box, select the template you want to share from the list.
  3. Click Share.
  4. In the Share Template dialog box, click to OK release the changes to the listed users.
