Using the Fax Editor

With the Fax Editor window you can create a new Fax template or edit an existing template. Like the E-Mail and Letter Editors, the Fax Editor contains a Sage SalesLogix toolbar which give you access to the special template functions as well as many Microsoft Word features.

The Fax Editor view displays the following options when editing or creating a template:

Box Description/Action
Insert SLX Field Click to display a menu with commonly used Sage SalesLogix merge fields.
Insert File This option is only available for an E-mail template.
Insert Library File This option is only available for an E-mail template.
Save and Close Click to save the changes made to the template and close the template editor.
Save Click to save the changes made to the template but leave the template editor window open.
Show This option is only available for an E-mail template.
Message Format This option is only available for an E-mail template.
Preview Click to view the template as it would be merged with the current record.
SLX Mail Merge Click to launch a Mail Merge.
Fax Subject Click to enter a subject for the fax template. When performing a mail merge to this template and selecting Output to Fax, the subject line of the Mail Merge will auto-populate with text from the template's subject line.
Show Word Menu Click to display the Microsoft Word Menu.
Edit Fields Click to open the SalesLogix Mail Merge Field Editor window.
Insert Special Field Click to insert a new opportunity product table, SQL statement, or custom mail merge field into the template.
E-mail Options This option is only available for an E-mail template.
