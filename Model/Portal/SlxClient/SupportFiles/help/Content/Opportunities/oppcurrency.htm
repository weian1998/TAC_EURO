Updating the Opportunity Currency

This functionality is only available if Multi-Currency is configured.  

You can use the Opportunity Currency dialog box to change an opportunity's current type and the rate of a specified currency.  

To change an opportunity's rate and type of currency

  1. In the Change Rate to box, click the Find button and find the type of currency.

The From and To fields allow you to view the potential when it is converted from the current currency rate and type to the new currency rate and type. If this is a new opportunity, the From and To fields will have the same value.

  1. To lock the conversion rate, select the Lock in the rate for this Opportunity checkbox.

When this option is selected, any global changes to the conversion rate will not affect this opportunity.

  1. Click OK.
