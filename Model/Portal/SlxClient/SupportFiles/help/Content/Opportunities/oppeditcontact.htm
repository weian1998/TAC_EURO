Editing an Opportunity Contact

After you have added a contact to an opportunity, you can edit the contact by adding information where needed. You cannot, however, change the contact or the contact's title.

To edit

  1. Open the Edit Opportunity Contact dialog box.
  • From the Opportunity Detail view Contacts tab, click the Edit hyperlink for the contact you want to edit.
  • From the Insert Opportunity dialog box, click the Edit hyperlink for the contact you want to edit.
  1. Select the Primary Contact box if this contact is the primary contact for the opportunity.

If more than one contact is marked as primary, the contact last selected will become the only primary contact.
  1. In the Role box, click the ellipsis button and select an item from the list.
  2. Next to the Competitor Pref box, click the Find button, select the competitor you want to add, and then click OK.
  3. In the Standing box, click the ellipsis button and select an item from the list.
  4. In the Strategy box type a brief description of the competitor's strategy (for example, value or simplicity).
  5. In the Personal Benefits, type any notes specific to this contact.
  6. In the Issues area, type any other information you need that is specific to this contact.

  7. Click OK.
