Using the Opportunity List View

The List view displays all of the contacts that you have access to. You can use groups and lookups to narrow the list of records that display.

Your security access determines what functions are available. Contact your administrator for any access rights changes. WebViewer users may not access full functionality.

How do I?

Open the Opportunity List view
  • On the Navigation Bar, click Opportunities.
Refresh list information
  • Click the Refresh button.

Use the Summary View

Work with records in a grid

Filter my list

Use the Lookup to search for records

Open the Opportunity Detail view
  • Click the opportunity description to open the Opportunity Detail view.

View a group

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Add to Group
  • Remove from Group
  • Save Records as a Group
  • Export to File
  • Promote to Dashboard
