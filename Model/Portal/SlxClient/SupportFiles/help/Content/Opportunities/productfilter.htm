Product Filter Options

Use Filter options to limit the products that display in the list of available products. In the Add Opportunity dialog box, the filters retain your last selection as the default values.

The available filters and selection method differs if you are adding products from the Products tab or the Opportunity Options view. You can filter by Name, Family, Price Level, Price, Status, and SKU.

Depending on the selected filter, you may not see package information in the tree view. Some filters only return product information without displaying the associated package.

To filter products accessed from the Products tab

  1. In the Lookup by box, select a filter.

Packages display in the list within a product listing. Expand a package name to view the associated product(s).

  1. Enter or select the appropriate conditional information.
  2. If necessary, add additional filters.
  3. Click Search.

To filter products accessed from the Opportunity Options view

  1. Select the check boxes of the filters you want to apply.

Selecting the Packages option displays product suites and their contents. When selected, this filter disables the SKU, Family, and Status fields, and changes the Name label to Package Name.

  1. Enter or select the appropriate conditional information.
  2. To clear filter options, click Reset.
  3. Click Search.
