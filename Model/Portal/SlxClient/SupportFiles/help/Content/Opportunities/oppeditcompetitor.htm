Editing an Opportunity Competitor

After you have added a competitor to an Opportunity, you can edit the competitor information from the Edit Competitor dialog box.

To edit

  1. Open the Edit Opportunity Competitor dialog box
  • From the Opportunity Detail view Competitors tab, click the Edit hyperlink for the competitor you want to edit.
  • From the Insert Opportunity dialog box, click the Edit hyperlink for the competitor you want to edit.
  1. If you want to modify this competitor's rating, type a number in the Rating box.

This only changes the competitor's rating for this opportunity, not the competitor's overall rating.

  1. Select Incumbent to identify the competitor as the incumbent.
  2. Type the competitor's Strengths and Weaknesses in the boxes provided.
  3. In the Strategy area, type any notes about the competitor's strategies.
  4. In the Notes area, type any additional information about the competitor.
  5. Click OK.
