What is the Query Builder?

Query Builder enables you to select data from the database based on one or more conditions. For example, you may need to see a list of primary contacts in the banking industry. Query Builder is connected to various areas throughout the Sage SalesLogix clients. To use Query Builder effectively, you must understand the Sage SalesLogix database structure and relational database concepts, such as data relationships and joins.

Query Builder is available in:

Sage SalesLogix Client

Use Query Builder to:

Create lookups and groups – You can quickly find accounts, contacts, opportunities, tickets, contracts, returns, and defects and save them as groups.
