Adding/Editing/Completing Campaign Stages

Campaign stages are groups of tasks that can help Sage SalesLogix Marketing users manage their campaigns. You can add and edit stage information, or complete a stage once it is finished.

How Do I?

Add a stage
  1. From the Campaign Detail view, select the Stages/Tasks tab, and then click Add Stage.
  2. Type a Description for the stage.
  3. In the Status box, click the drop-down arrow, and select an item from the list.
  4. In the Lead Source box, click the Find button to select a lead source.
    Your selection is added to the dialog box.
  5. If necessary, change the Code. Click OK at the dialog box.

    Changing the campaign code means that the campaign code for this stage will no longer match the campaign or any other campaign stages.

  6. In the Comments box, type any additional information.
  7. If necessary, click the Start Date Calendar button to select the date.  
  8. Click the End Date Calendar button to select the date.  
  9. Add tasks to the Tasks grid.
  10. Click OK.
Edit a campaign stage
  1. From the Campaign Detail view, select the Stages/Tasks tab, and then click Edit Stage.
    • From the Insert Campaign or Copy Campaign dialog boxes, select the stage you want to edit, and then click Edit.
    • From the Campaign Detail view, select the Stages/Tasks tab, right-click the stage you want to edit, and then point to Edit Stage.
  2. Make changes if necessary, and then click OK
Complete a campaign stage
  1. From the Campaign Detail view, select the Stages/Tasks tab, select the stage to complete, and then click Complete Stage.
    The Complete Stage dialog box opens with Status set to "Completed".
  2. Make changes if necessary, and then click OK.
