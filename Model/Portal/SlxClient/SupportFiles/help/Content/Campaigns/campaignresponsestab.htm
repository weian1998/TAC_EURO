Campaign Responses Tab

The Campaign Responses tab lists all of the target responses for a campaign. You can add, edit, and delete campaign responses, as well as filter the list of responses.

How Do I?

Open the Campaign Responses Tab
  • On the Campaign Detail view, click the Responses tab.

Add a campaign response

Edit a campaign response

This view contains the following columns:

  • Lead Source - Displays the lead source selected for the campaign response.
  • Name - Displays the name of the contact or lead.
  • Type - Displays the target type: Contact or Lead.
  • Response Date - Displays the date the response was added to the campaign.
  • Response Method - Displays the response method.
  • Stage - Displays the campaign response stage.
  • Comments - Displays any comments entered when adding or editing the target response.
