Adding/Editing/Completing Campaign Stages

Campaign stages are groups of tasks that can help Sage SalesLogix Marketing users manage their campaigns. You can add and edit stage information, or complete a stage once it is finished.

To add

  1. Open the Add Stage dialog box.
  • From the Campaign Detail view, select the Stages/Tasks tab, and then click Add Stage.
  1. Type a Description for the stage.
  2. In the Status box, ellipsis button and select an item from the list.
  3. In the Lead Source box, click Find to select a lead source. Your selection is added to the dialog box.
  4. If necessary, change the Campaign Code. Click OK at the dialog box.

Changing the campaign code means that the campaign code for this stage will no longer match the campaign or any other campaign stages.
  1. In the Comment box, type any additional information.
  2. If necessary, click the Start Date Calendar/Time button to select a date, and then click OK.
  3. Click the End Date Calendar/Time button to select a date, and then click OK..
  4. Click OK.

To edit a campaign stage

  1. Open the Edit Stage dialog box.
  • From the Campaign Detail view, select the Stages/Tasks tab, and then click Edit Stage.
  1. Make changes if necessary, and then click OK.

To complete a campaign stage

  1. From the Campaign Detail view, select the Stages/Tasks tab, select the stage to complete, and then click Complete Stage.

The Complete Stage dialog box opens with Status set to "Complete".

  1. Make changes if necessary, and then click OK.
