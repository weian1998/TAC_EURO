Updating Target Information

Update multiple target contacts or leads at once using the Update Target dialog box. Use the boxes at the bottom of the dialog box to update multiple targets with the same information.

To open the Update Targets dialog box

  • On the Campaign Detail view, click the Targets tab, and then click Update selected Targets .

To update multiple opportunities with the same information

  1. Click the Select check boxes of the targets you want to update, or click the Select All button to update all of the targets, and then click Update Targets.
  2. From the drop-down box near the bottom of the dialog box, select the information you would like to update. Select from the following options:
  • Stage
    1. Select Stage from the drop-down list near the bottom of the dialog box.
    2. Click the To drop-down arrow, and then select from the list.
  • Status
    1. Select Status from the drop-down list.
    2. Click the To drop-down arrow, and then select a new status from the list.
  • Initial Target
    1. Select Initial Target from the drop-down.
    2. Select the Yes option if the target was part of the campaign when it was launched, or select No if the target was added after the campaign was launched.
  • Add Response
    1. Select Add Response.
    2. Complete the Response information.
  1. When you are finished updating target information, click Update.
