Adding/Editing/Completing Campaign Tasks

Campaign tasks can help Sage SalesLogix Marketing users manage their campaigns. You can add and edit task information, or complete a task once it is finished.

How Do I?

Add campaign tasks
  1. From the Campaign Detail view, click the Stages/Tasks tab, find the stage you want to add a task to, and then click Add Task.
  2. Type a Description for the task.
  3. In the Status box, click the drop-down arrow and select an item from the list.
  4. In the Priority box, click the drop-down arrow and select an item from the list.
  5. In the % Complete box, type a percentage between 0-100.
  6. Click the Needed Date Calendar button to select the date.  
  7. If the task has already been completed, click the Completed Date Calendar button to select the date.  
  8. In the Comments box, type any additional information.
  9. In the Estimated Cost box, type the estimated cost of the task.
  10. If you know the actual cost of the task, type the amount in the Actual Cost box.
  11. In the Estimated Hours box, type the estimated hours it will take to complete the task.
  12. If you know the actual hours to complete the task, type the amount in the Actual Hours box.
  13. If Multi-Currency is configured, select Display using "My Currency" to view Task Costs using the currency type you selected in your user option.
  14. Select the appropriate Assign to option.
  15. In the Assign To box, do one of the following:
    • If you selected Existing User/Team, click the Find button and then select the user or team that you want to assign the task to.
    • If you selected Existing Department, click the drop-down arrow and select from the list.
    • Existing Contact, click the Find button, and then select the person you want to assign the task to.
    • If you selected Other Individual, type the name of the person you want to assign the task to.
    • If you selected None, go to the next step.
  16. Click the Assigned Date Calendar button to select the date.  
  1. Click OK.
Edit campaign tasks
  1. From the Campaign Detail view, select the Stages/Tasks tab, and click Edit.
    • From the Insert Campaign or Copy Campaign dialog boxes,select the task you want to edit, and then click Edit.
    • From the Add, Edit or Complete Stage dialog box, click Edit Task.
    • From the Campaign Detail view, click the Stages/Tasks tab, right-click in the grid, and then point to Edit Task.
  2. Make changes if necessary, and click OK.

If you change the Needed Date, you will be prompted to adjust the Needed Date of all other uncompleted tasks by the same number of days. Click Yes to adjust the Needed Date of the other tasks.
Complete campaign tasks
  1. From the Campaign Detail view, select the Stages/Tasks tab, click Complete.
    The Complete Stage dialog box opens with Status set to "Completed", % Complete set to 100 and the Completed Date displays the current date.
  2. Make changes if necessary, and click OK.

If you change the Needed Date, you will be prompted to adjust the Needed Date of all other uncompleted tasks by the same number of days. Click Yes to adjust the Needed Date of the other tasks.
