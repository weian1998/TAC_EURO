Campaign Targets Tab

The Campaign Targets tab lists all of the targets for a campaign. You can perform campaign related tasks such as, managing the target list, updating target information, exporting targets to Excel or creating a group. These features are not available if you use an external list of targets, but you can include information about the external list.

How Do I?

Open the Campaign Targets tab
  • On the Campaign Detail view, click the Targets tab. If no records appear, click Refresh.
Use targets from an external list
  1. Select the Use external list check box.

If you use an external list, all other associated targets will be removed from the campaign.

  1. Click OK to continue.
  2. Use the following boxes to include information about the list:
  • Description - type a description or the name of the list of targets you are using for the campaign.
  • Location - type the location where the list is stored. For example, C:\My Documents\Campaign Target List
  • Number of Targets - type the number of targets
  1. Click Save.
View records
  • Select the Display results when opened check box.

Remove a target
  1. On the Campaign Detail view Targets tab, select the check box(es) of the record(s) you want to remove from the campaign.
  2. Click Remove selected Targets.
  3. Click OK.
