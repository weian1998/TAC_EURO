Using the Library

The Library is a central repository for company information (in the form of documents, URLs, and so on) that provides an easy way to distribute information to remote users. Using the Sage SalesLogix Web Client, you can request and download a Library file when you need it. The Library displays the file name, size, creation, revision and expiration dates, and a brief description of the file. Files are sorted by name. Click the column headings to sort the library files list.

Only the system administrator and users with an administrator role can access these features.

Your company’s library might include:

  • Product information
  • Presentations
  • Policies and procedures
  • Sales literature
  • Spreadsheets
  • News bulletins
  • Price sheets
  • Annual reports
  • Manuals
  • Hyperlinks to intranet/internet locations of files

How Do I?

Open the Library
  • On the Navigation Bar, click Library.
Add a document to the Library
  1. Select the folder under which you want to add the item.
  2. Click Add File.
  3. Select the file(s) you want to add.
  4. Click Open.
Add a document to the Library using drag and drop

  • You may not have access rights to this feature. For more information, please contact your administrator.

  • This feature requires the installation of the Desktop Integration Module.
  1. Browse to the file you want to add.
  2. On the Navigation Bar, click Library.
  3. In the tree view, click the "+" signs to expand the folder to the location where you want to add the file.
  4. Drag the file and drop it on the Library view.

If prompted to install Sage SalesLogix Desktop Integration and you want to use this feature, click Yes and follow the steps to install Sage SalesLogix Desktop Integration. In order to complete the install you will be prompted to close all browser windows and Microsoft Outlook.

  1. Click OK.

You cannot drag and drop a folder and its contents to the Library. You can only drag and drop individual files.

Edit Library File Properties

Add, rename or delete a Library folder

Download a Library file
  1. On the Navigation Bar, click Library.
  2. In the tree view, click the "+" signs to expand the appropriate folder.
  3. In the list, click the File name of the file you want to download.

If your browser opens the attachment instead of saving it to a location on your computer, change your default browser settings to automatically save files instead of opening them. Alternatively, hold down [Shift] and then click the link to download without opening all files except for those with the extension HTML. HTML files are opened in another window.
