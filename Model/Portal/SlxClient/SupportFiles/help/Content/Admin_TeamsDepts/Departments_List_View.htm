Using the Departments List View

Only the system administrator and users with an administrator role can access these features.

The Departments List view displays departments in a grid. The grid provides information about each department.

  • To sort the grid by a particular field, click one of the column headings.
  • To display all departments of a particular type, click a tab in the grid.
  • To only display departments that meet specific criteria, configure filters.

How Do I?

Create a Department

Open the Departments List View
  • On the Navigation Bar, click Administration, and then click Departments.

Open the Department Detail View
  • Click the department name to open the Department Detail view.

Use the Lookup to search for department records

View a Group

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Add to Group
  • Remove from Group
  • Save Records as a Group
  • Export to File
  • Add to Team
  • Remove from All Teams
