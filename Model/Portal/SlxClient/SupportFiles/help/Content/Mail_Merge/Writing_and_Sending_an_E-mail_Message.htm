Writing and Sending an E-mail Message

You can write an e-mail to a single contact or lead using either the Write E-mail or the Write E-mail Using Template option. If you need to send an e-mail message to more than one contact or lead, use Mail Merge.

You must have Microsoft Outlook installed on your computer to use Write E-mail features.

If the Write menu is unavailable, you must download Sage SalesLogix Desktop Integration. You can install Sage SalesLogix Desktop Integration from the Web Client login screen. 
To write an e-mail without using a template
  1. Do one of the following:
    • On the Write menu, point to E-mail.
    • Click the E-mail button next to the contact's or lead's e-mail address.
  2. If e-mail is enabled, but you have not logged on, at the message box, click Yes to log on.

  3. Your next step depends on your focus within the Sage SalesLogix Web Client.

    • If your focus is on a particular contact or lead, the merge will continue.
    • If you do not have focus on a particular contact or lead, use the Select Contact or Select Lead dialog box to select a contact or lead.
    • If your focus is on an account or an opportunity, the Select Contact dialog box will prompt you to select a contact.
    • If your focus is on a ticket the document will merge with the associated contact.

  4. Complete the Subject box and message body of the new e-mail window.
  5. (Optional) If the e-mail message requires an attachment, click Insert, and then click one of the menu options. If attaching a file, refer to About E-mail Attachments for more details.
  6. Send your e-mail by doing one of the following:
    • Click Send SLX. If you attached a file, and want to save a copy, click Yes at the prompt. The e-mail item is recorded to the contact's history and the file appears on the Attachments tab.
    • Click Send. The message will not be automatically recorded to the contact's history; however you can drag and drop the message from Outlook into Sage SalesLogix.

      		•

      Graphic files smaller than the minimum size set by your administrator will not be saved as separate attachments.

  7. Use the Complete an Activity dialog box to edit the history information, type notes, or schedule a follow-up activity, and then click Complete.

  8. If you selected to schedule a follow-up activity, complete the dialog box for the type of follow-up activity.
To write an e-mail using a template
  1. On the Write menu, point to E-mail Using Template, and click a template name or click More Templates to select another template. If prompted to add the template to the most recently used templates list, click Yes to add the template to the list; otherwise click No.
  2. If e-mail is enabled, but you have not logged on, at the message box, click Yes to log on.

  3. Your next step depends on your focus within the Sage SalesLogix Web Client.

    • If your focus is on a particular contact or lead, the merge will continue.
    • If you do not have focus on a particular contact or lead, use the Select Contact or Select Lead dialog box to select a contact or lead.
    • If your focus is on an account or an opportunity, the Select Contact dialog box will prompt you to select a contact.
    • If your focus is on a ticket the document will merge with the associated contact.

  4. If you chose More Templates in Step 1, in the Managing Templates dialog box, select a template and click OK. If prompted, choose an address from the Select Type of Address dialog box.
    The e-mail address of the contact is displayed in the To box, the Subject box contains the information that was stored in the template, and attachments to the template are identified.
  5. (Optional) In the text box, you can type additional information or change the default signature.
  6. (Optional) If the e-mail message requires an attachment, click Insert, and then click one of the menu options. If attaching a file, refer to About E-mail Attachments for more details.
  7. Send your e-mail by doing one of the following:
    • Click Send SLX. If you attached a file, and want to save a copy, click Yes at the prompt. The e-mail item is recorded to the contact's history and the file appears on the Attachments tab.
    • Click Send. The message will not be automatically recorded to the contact's history; however you can drag and drop the message from Outlook into Sage SalesLogix.

      		•

      Graphic files smaller than the minimum size set by your administrator will not be saved as separate attachments.

  8. Use the Complete an Activity dialog box to edit the history information, type notes, or schedule a follow-up activity, and then click Complete.

  9. If you selected to schedule a follow-up activity, complete the dialog box for the type of follow-up activity.
