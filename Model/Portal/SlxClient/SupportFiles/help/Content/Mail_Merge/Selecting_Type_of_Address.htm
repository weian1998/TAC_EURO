Selecting the Type of Address

The Select Type of Address dialog box displays when performing a mail merge to a contact with more than one address or a group of contacts and using a template which contains any Sage SalesLogix merge fields from the address table.

To select an address type

  1. In the Select Type of Address dialog box, select one of the following options:
    • Contact’s primary address – This option will use the address set as “primary” for all contacts processed in the mail merge.
    • Contact’s shipping address – This option will use the address set as “shipping” for all contacts processed in the mail merge.
    • Other address – Use the drop-down arrow to select from the list of other types of addresses assigned to contacts. Any contacts processed in the mail merge that do not have the type of address selected will default to the primary address.
  2. Click OK.

If you wish to prevent the Select Type of Address dialog box from displaying but need address information in the template, choose an address field that is joined to either the Contact or Userinfo tables. Regardless of the merge type, using address fields joined to Contact or Unserinfo tables will force the merge to the primary address for the contact(s) and prevent the Select Type of Address dialog box from displaying.
