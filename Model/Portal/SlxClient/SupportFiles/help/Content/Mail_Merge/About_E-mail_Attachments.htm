About E-mail Attachments

You can attach a file when sending an e-mail message to a contact or lead. If the correct options are selected, the file is stored on the Attachments tab for the Contact and Account, as well as in the default Documents folder. For Sage SalesLogix Web users, the file is stored in the Documents folder of the Web Server. On the Attachments tab, the file name is the exact name of the file or document. In the Documents folder, the file name includes an internal reference code assigned by Sage SalesLogix and the document name.

You can attach Windows files or Sage SalesLogix Library files to a template. When you e-mail the template during a mail merge, the attached files are sent to the contact(s) or lead(s). If you copy a template, any attachments are also copied to the new template. You can remove an attachment from a template.

To attach a file to a template
  1. Open the Template Editor.
  2. Do one of the following:
  • To attach a Windows file, click the Insert File button, browse to the location of the file, and then click Insert File.
  • To attach a Sage SalesLogix Library file, click the Insert SalesLogix Library File button. In the SalesLogix Library dialog box, click the file(s), and then click Insert.
  1. You can attach another file or click the Save and Close button.
To remove a file from a template
  1. Open the Template Editor.
  2. In the Attachment(s) box, select the attachment you want to remove.
  3. Do one of the following:
  • Right-click, and then click Remove.
  • Press Delete.
  1. You can remove another file or click the Save and Close button.
