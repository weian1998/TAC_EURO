Selecting a Contact

When you want to write an e-mail message, or use a template to write a letter, fax, or e-mail message to a single contact and the focus is not currently on a single contact, then you must use the Select Names dialog box to select a contact to receive the correspondence.

If you want to send correspondence to all the contacts associated with the account or opportunity, you must use Mail Merge.

To select a contact

  1. In the Select Names dialog box, a list of contact names associated with the account or opportunity is displayed.
  2. Depending upon the type of correspondence you are writing, do one of the following:
  • If you are using a template to write an e-mail message, letter, or fax, select a contact name, and then click Merge.
  • If you are not using a template to write an e-mail message, select a contact name, then click the To button. The contact name will appear in the box. Click OK.
