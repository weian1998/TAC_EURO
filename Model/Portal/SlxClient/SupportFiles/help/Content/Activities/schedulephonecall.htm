Activity Information Boxes

Box Description/Action
Regarding In the Regarding box, click the ellipsis button and select an item from the list.
Priority In the Priority box, click the ellipsis button and select an item from the list.
Category

In the Category box, click the ellipsis button and select an item from the list.

Start Time To modify the date, click the Calendar/Time button, select a date, and the click OK.
Timeless If the activity does not need to occur at a specific time, select the Timeless check box.
Duration Type a number and select Minute(s), Hour(s), or Day(s) from the drop-down list to determine the length of time the activity requires.
Reminder If you want to be reminded of the activity before it occurs, select the Reminder check box, type a number in the box, and select Minute(s), Hour(s), or Day(s) from the drop-down list to determine the length of time the reminder should occur before the activity start time.
Auto-Rollover to Next Day

This option is only available if Timeless is selected. Select the Auto Rollover to Next Day check box to automatically roll over the activity to the next day if it is not completed on the scheduled date.

Note: A recurring activity cannot be rolled over.
Leader Select a Leader from the drop-down list.
Contact or Lead Determines which entity types are available to associate with the activity.
Lead or Contact, Account, Opportunity, and Ticket

Determines which entity types are associated with the activity. Click the Find button and edit using the lookup.

Note: To remove a Lead, Contact, Account, Opportunity or Ticket, click the Delete button next to the appropriate box.
Notes In the Notes area, type any additional information for the activity.
