Completing an Activity

When you complete an activity, details about it are recorded in the History tab for the related account or contact.

If you do not want to complete the activity, click Cancel to return to the previous view.

To complete

  1. Do one of the following:

  • Open the activity you want to complete, from the General tab, click Complete.
  • From the Activity Reminders view, select an activity, and then click the Complete button.
  • From the Activities List view or any of the Activities tabs, click Complete.
  • On the Schedule menu, select Complete an Activity. From the Complete an Activity dialog box, you can select a scheduled activity or create a new unscheduled activity to complete.

    If you clicked the Complete button from the Edit view, any changes you made will be automatically saved and visible in the Complete Activity view.
  1. If necessary, modify the Completed date. To modify the date, click the Calendar/Time button to select a date, and then click OK.
  2. In the Duration boxes, type a number and select Minute(s), Hour(s), or Day(s) from the drop-down list to determine the length of time the activity required.
  1. In the Result box,click the ellipsis button and select an item from the list.
  2. If necessary, specify a follow-up activity in the Follow-Up box. Select the Carry Over Notes or Carry Over Attachments check box to copy notes or attachments from this activity to the follow-up activity.
  3. If necessary, use the Find or Delete buttons to change or remove an associated lead, contact, account, opportunity or ticket.
  4. In the Regarding, Category, and Priority boxes, ellipsis button and select an item from the list.
  5. In the Notes box, type any additional information about this activity.
  6. If another user completed the activity, click the Leader Find button, select the user completed the activity, and then click OK.
  7. Use the Follow-Up area to schedule follow-up activities.
  8. Click the Attachments tab to add a document or URL to the activity.
  9. Click As Scheduled to complete the activity using the date and time you entered in Steps 2 and 3. Or, click Now to complete the activity using the current date and time.

If you scheduled a follow-up activity, the Schedule an Activity dialog box appears for you to complete.
