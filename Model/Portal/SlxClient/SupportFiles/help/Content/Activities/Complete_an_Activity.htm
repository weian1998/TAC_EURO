Completing an Activity

When you complete an activity, or write an e-mail, letter or fax, details about it are recorded in the Notes/History tab for the related account, contact, lead, ticket or opportunity.

You can complete a scheduled activity, or an unscheduled activity, such as an unplanned phone call.

If you do not want to complete the activity, click Cancel to return to the previous view.

Your calendar security determines whose activities you can schedule, edit, complete, accept/decline, or delete. You can also view activities associated with contacts, leads, accounts, opportunities, or tickets that you can access.

To complete

  1. Do one of the following:

    • Open the activity you want to complete, and click Complete.

      If you clicked the Complete button from the Edit view, any changes you made will be automatically saved and visible in the Complete Activity view.

    • From any of the Activities tabs, click Complete.
    • From the Activities List view, right-click an activity and select Complete Activity.
    • On the Schedule menu, select Complete an Activity. From the Complete an Activity dialog box, you can select a scheduled activity or create a new unscheduled activity to complete.
    • On the Alerts, Alarms tab, select the activity you want to complete, then click Complete Selected Activities.
  2. If you selected a recurring activity and are prompted, select Complete this Occurrence or Complete all Occurrences, and then click Continue.
  3. In the Regarding box, click the drop-down arrow and select an item from the list.
  4. Use the Location field to type a location.
  5. If necessary, modify the Completed date. To modify the date, click the Calendar button to select the date.  
  6. If necessary, modify the Scheduled date. To modify the date, click the Calendar button to select the date.  

  7. Do one of the following:

    • Select the Timeless check box to create an activity that occurs on a specific date, but at no specific time and without a specific duration.
    • Click the Duration drop-down arrow and select from the list.
  8. In the Result box,click the drop-down arrow and select an item from the list.
  9. If necessary, use the Find or Remove buttons to change or remove an associated lead, contact, account, opportunity or ticket.
  10. In the Priority box, select the importance of the activity.
  11. In the Category box, select the category of the activity.
  12. In the Leader box, click the Find button and use the lookup to select a leader if someone other than you completed the activity. Not all users are available. You can only choose a user to be an activity leader if you have access to their calendar.
  13. In the Notes box, type any additional information about this activity.
  14. If necessary, specify a follow-up activity in the Follow-Up box. Select the Carry Over Notes or Carry Over Attachments check box to copy notes or attachments from this activity to the follow-up activity.
  15. Click the Availability tab to select and view members and resources who participated in the activity.
  16. Click the Attachments tab to add a document or URL to the activity.
  17. Click As Scheduled to complete the activity using the Scheduled date and time. Or, click Now to complete the activity using the current date and time.
    If you scheduled a follow-up activity, the Schedule an Activity dialog box appears for you to complete.
