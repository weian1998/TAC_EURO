Setting Alarms

Alarms can alert you to an activity that is due to be completed. You can set an alarm for an existing activity or when you schedule an activity. You must have Alerts set up to include alarms in order to be notified of your alarms.

To set the alarm

  1. Do one of the following:
  • Schedule a new activity.
  • Find the activity for which you want to set an alarm, and open the Activity Detail view.
  1. Select the Alarm check box, select the amount of time at which you want to be reminded of this activity.
  2. When finished, click OK.
