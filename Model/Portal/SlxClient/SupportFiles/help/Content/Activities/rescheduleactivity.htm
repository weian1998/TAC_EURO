Rescheduling an Activity

You can reschedule an activity from the Activity Reminders view.

To reschedule an activity

  1. From the Activity Reminders view, select the check box for each activity you want to reschedule, and then click Reschedule Selected Activities.

The Reschedule Activity dialog box displays, with the current record highlighted in blue in the Activity Queue.

  1. In the Start Date box, click the Calendar/Time button, select the day and time that you want to reschedule the activity, and then click OK.
  2. If the activity does not need to occur at a particular time, select Timeless, and then click Save.
  3. If you selected multiple activities in step 1, repeat steps 2-4 for each of the activities listed in the Activity Queue.
