Scheduling or Editing an Activity

Use the Activity detail view to add or edit an activity or to see in-depth information about an activity, including date, time, duration, and notes. You can also complete or delete an activity after you schedule it.

Your calendar security determines whose activities you can schedule, edit, complete, accept/decline, or delete. You can also view activities associated with contacts, leads, accounts, opportunities, or tickets that you can access.

How Do I?

Schedule an Activity

  1. Do one of the following:

    • On the Schedule menu, click the type of activity you want to schedule (Phone Call, Meeting, To-Do, or Personal Activity).
    • On the Contact, Account, Opportunity, or Ticket Detail view, click the Activities tab, and then click the Schedule Phone Call, Schedule Meeting, or Schedule To-Do button. If the tab is not visible, click the More Tabs tab.
  2. In the Regarding box, click the drop-down arrow and select an item from the list.
  3. Use the Location field to type a location, or use the Availability tab to select a location resource.
  4. In the Start Time box, click the Calendar button to select the date and time when the activity will occur, and then click OK.

  5. Do one of the following:

    • Select the Timeless check box to create an activity that occurs on a specific date, but at no specific time and without a specific duration.
      Timeless activities are displayed on the Calendar day view.
    • In the Duration box, type or click the drop-down arrow and select from the list.
  6. Select the Alarm check box to set an alarm to notify you before the activity begins. Type or select a length of time.
  7. Select the Auto rollover to next day check box to automatically roll over the activity to the next day if it is not completed on the scheduled date. This option is only available if Timeless is selected.

    A recurring activity cannot be rolled over. If you select this option and then set Recurring options, the Auto rollover to next day option is cleared.

  8. If necessary, select Contact or Lead.
  9. Click the Find button next to the appropriate box (Name, Company, Account, Opportunity, Ticket) to open the Lookup. Find the record you want to associate with the activity, and then click OK.(If you are scheduling a personal activity, proceed to the next step.)
    After selecting a contact or lead, you can hover over the record name to see limited detailed information.

    To remove a Lead, Contact, Account, Opportunity or Ticket from the activity, click the Remove button next to the appropriate box.

  10. In the Priority box, select the importance of the activity.
  11. In the Category box, select the category of the activity.
  12. In the Leader box, click the Find button and use the lookup to select a leader if someone other than you will be leading the activity. Not all users are available. You can only choose a user to be an activity leader if you have access to their calendar.
  13. If necessary, in the Notes area, type notes about the activity.
  14. Click the Availability tab to select and check the availability of the members and resources participating in the activity.

    You can only add members or resources to phone calls and meetings.

  15. Click the Recurring tab to select how often the activity occurs (Once, Daily, Weekly, Monthly, Yearly).
  16. Click the Attachments tab to add or view a document or URL.

    If an activity member has calendar access to the activity leader, then the member will be able to add or edit attachments. If the member does not have calendar access, then the member can only view, copy or save attachments.

  17. Click OK.
Edit an Activity
  1. Do one of the following:
    • Open the Activities List view, find the activity you want to edit, and then click the activity Type hyperlink.
    • From What's New, click the Activities tab, and then click the Type icon of the activity you want to edit.
  2. If you are updating a recurring activity and are prompted, select Edit this Occurrence of the Activity to make the change to only the selected occurrence, or Edit all Instances of this Activity to make the change to all occurrences of the activity, and then click Continue.

  3. View or the activity information.
    You can hover over a contact, account, or lead to see limited detailed information.
  4. Click OK.
