Completing Multiple Activities

You can complete multiple activities from the Activities view. You can either complete multiple activities at the same time, or complete each activity separately. If you choose to complete each activity separately, the Complete Activity view will open allowing you to edit activity information before completing it. Once you complete an activity, the Complete Activity view will open for the next selected activity until each of the selected activities have been completed.

Your calendar security determines whose activities you can schedule, edit, complete, accept/decline, or delete. You can also view activities associated with contacts, leads, accounts, opportunities, or tickets that you can access.

To complete multiple activities at once

  1. On the Activities view, select the activities you want to complete, and in the Task Pane Common Tasks area click Complete Activity.
  2. Click the Result drop-down arrow and select an item from the list.
  3. In the Note field, add any information that you want to be added to all of the activities you are completing.
  4. Click As Scheduled to complete the activities using the date and time you entered when you scheduled the activities. Or, click Now to complete the activities using the current date and time.

    If any of the selected activities are recurring, then only the selected occurrence or occurrences of the recurring activity will be completed.

The activities complete automatically and are marked with the date specified.

To complete multiple activities individually

  1. On the Activities view, select the activities you want to complete, and in the Task Pane Common Tasks area click Complete Activity.
  2. Click the Result drop-down arrow and select an item from the list.
  3. In the Note field, add any information that you want to be added to all of the activities you are completing.
  4. Click Individually.
  5. If any of the selected activities are recurring, and you are prompted, select Complete this Occurrence or Complete all Occurrences, and then click Continue.
  6. Follow the remaining steps for completing each activity until each activity has been completed.
