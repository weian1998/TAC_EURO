Scheduling or Editing an Activity

Use the Activity Detail view to add or edit an activity or to see in-depth information about an activity, including date, time, duration, and notes. You can also complete or delete an activity after you schedule it.

How Do I?

Schedule an Activity
  1. Do one of the following:
    • On the Schedule menu, click the type of activity you want to schedule (Phone Call, Meeting, To-Do, or Personal Activity).
    • On the Contact, Account, Opportunity, or Ticket Detail view, click the Activities tab, and then click the Schedule Phone Call, Schedule Meeting, or Schedule To-Do button. If the tab is not visible, click the More Tabs tab
  1. Use the General tab to enter remaining activity information.
  2. Click the Members tab to select the members participating in the activity. Use CTRL and click to select multiple options.

This tab is only available for phone calls and meetings.
  1. Click the Resources tab to select the internal resources necessary for the activity. Use CTRL and click to select multiple options.

This tab is only available for phone calls and meetings.
  1. Click the Recurring tab to select how often the activity occurs (Once, Daily, Weekly, Monthly, Yearly).
  2. Click Time Zone Calculator to determine the date and time that an activity will occur in a different time zone.
  3. Click the Attachments tab to add or view a document or URL.

If an activity member has calendar access to the activity leader, then the member will be able to add or edit attachments. If the member does not have calendar access, then the member can only view, copy or save attachments.
  1. Click OK.
Edit an Activity
  1. Do one of the following:
    • Open the Activities List view, find the activity you want to edit, and then click the activity Type hyperlink.
    • From What's New, click the Activities tab, and then click the Type icon of the activity you want to edit.
  2. If you are updating a recurring activity and are prompted, select Edit this Occurrence of the Activity to make the change to only the selected occurrence, or Edit all Instances of this Activity to make the change to all occurrences of the activity, and then click Continue.

    The Recurring tab is not available if you select the Edit this Occurrence of the Activity option.
  3. Edit the activity information, and then click OK.

    You cannot change the user of a personal activity.

If you click the Complete button from the Edit view, any changes you made will be automatically saved and visible in the Complete Activity view.
