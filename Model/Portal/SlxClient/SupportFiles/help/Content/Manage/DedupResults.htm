Check for Duplicates History Detail View

The system administrator and users with the appropriate role can access these features.

The Check for Duplicates History Detail view displays the details of a job run in the Check for Duplicates Wizard. You can view the job information and resolve any potential duplicates.

How Do I?

Open the Check for Duplicates History Detail view
  1. On the Tools menu, click Duplicate Job History.
  2. Click a job number in the list to open the detail view.

Use the Lookup to search for records

Switch to the List view
  • On the toolbar click the List View button, or click the current group tab.

Use the Task Pane group list or common tasks

View a group

Information Boxes

The following job information is read-only.

Box Description
Job Number The Check for Duplicates wizard job number.
Job Description The entity the job was processed on.
Entity Type The entity the job was processed on.
Source Description The group the job was processed on.
Source Count The number of records in the source group.
Processed Count The number of records processed in the job.
Duplicate Count The number of potential duplicates.
Started On The date and time the job started.
Completed On The date and time the job completed.
Started By The user that submitted the job.
Status The job status.

Detail View Tabs

Use the Job Results tab to view and resolve duplicates.
