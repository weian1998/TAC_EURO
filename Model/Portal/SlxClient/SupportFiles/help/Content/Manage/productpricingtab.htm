Product Price Levels Tab

Only the system administrator and users with an administrator role can access these features.

You can use the Prices tab on the Product Detail to view or manage product pricing information.

If the Prices tab does not appear in the Detail view, click More Tabs to access the list of available tabs. Select Prices and drag it into the tab area or into the user-defined area of the Detail view.

Delete a Product Price
  • Click Delete next to the price you want to edit.

The Prices tab displays the following information:

  • Description - Displays the price description.
  • Price - Displays the price for the product. This column is based upon the default currency type as defined by your administrator.
  • Default - Displays the default price for the product.
