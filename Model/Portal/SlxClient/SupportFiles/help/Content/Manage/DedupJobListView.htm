Check for Duplicates History List View

Only the system administrator and users with the appropriate role can access these features.

The Check for Duplicates History List view displays all scheduled and completed duplication jobs. To view job results, click the job number in the list.

This view contains the following columns:

  • Job Number- The job number.
  • Status - The job status.
  • Description - The job description.
  • Entity Name - The entity that contains the records for the job.
  • Source Description - The group the job was processed on.
  • Started - The date and time the job started or is scheduled to start.
  • User - The user who scheduled the job.
  • Completed - The date and time the job completed.
  • Record Count - The number of records within the job.
  • Processed Count - The number of records within the job that have been processed.
  • Duplicate Count - The number of duplicate records.

How Do I?

Open the Check for Duplicates History List View
  • On the Tools menu, click Duplicate Job History.

Use the Lookup to search for records

Open the Check for Duplicates History Detail View
  • From the List view, click the job number to open the Check for Duplicates History Detail view.
