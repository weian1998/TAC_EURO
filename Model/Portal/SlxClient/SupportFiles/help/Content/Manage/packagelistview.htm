Using the Packages List View

The Packages List view displays all of the packages that you have access to. You can use groups and lookups to narrow the list of records that display.

How Do I?

Open the Packages List View
  • On the Navigation Bar, click Sales, Support, or Administration, and then click Packages.
Refresh list information
  • Click the Refresh button.

Work with records in a grid

Filter my list

Use the Lookup to search for records

Open the Packages Detail View
  • Click the package name to open the Package Detail view.

Use the Task Pane group list

View a group

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Add to Group
  • Remove from Group
  • Save Records as a Group
  • Export to File
