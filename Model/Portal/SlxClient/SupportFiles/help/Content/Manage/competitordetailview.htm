Using the Competitor Detail view

Only the system administrator and users with an administrator role can access these features.

Use the Detail view to see, add or edit information for a selected record.

Your workspace may be different depending on your access rights or if your installation uses a customized Sage SalesLogix Web Client.

How Do I?

Open the Competitor Detail view
  • On the Navigation Bar, click Administration, and then click Competitors to open the Competitors List view. If necessary, use the Lookup to locate a competitor, and then click on the competitor's name.

Use the Lookup to search for records

Switch to the List view
  • On the toolbar click the List View button.

Use the Task Pane group list or common tasks

View a group

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Add to Group
  • Remove from Group

Information Boxes

  • Competitor - Displays the competitor's name.
  • Account - Click the Find button and select an account using the lookup.
  • Contact -Click the Find button
  • Phone - Enter or edit the competitor's phone number.
  • Web Address - Enter or edit the competitor's Web address.
  • Rating - Enter or edit the competitor's rating.
  • Weaknesses - Enter or edit the competitor's weaknesses.
  • Strengths - Enter or edit the competitor's strengths.
  • Strategy - Enter or edit the sales strategy that the competitor uses.
  • Notes - Enter or edit any notes about the competitor.
