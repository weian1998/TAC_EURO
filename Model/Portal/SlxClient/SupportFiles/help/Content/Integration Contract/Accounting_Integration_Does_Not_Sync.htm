Troubleshooting Accounting Integration - Information Does Not Sync

You may encounter the following issue when synchronizing information between Sage SalesLogix and supported accounting applications.

Problem:

Accounting integration information does not synchronize

Solutions:

  • Verify that the SYNCDIGEST table TICK field is incrementing. Tell me more...
    1. In the database, open the SYNCDIGEST table.
    2. Find the APPID field for the appropriate resource and note the TICK field value.
    3. After the next synchronization, verify that the TICK field incremented.
  • Enable additional logging in the ContractSyncTrigger. Tell me more...
    1. Open the registry editor and ensure the HKEY_CURRENT_USER\Software\SalesLogix\Logging key exists. If the Logging key does not exist, then create a new key named Logging.
    2. Add a registry string value named Debug.
  •  View log errors, warnings or debug messages. Tell me more...
    • On the Sync Job Detail View, open the Log Items tab, and filter by Log Type to view errors, warnings, or debug messages.
  • Use the Sync Results Detail View to see detailed information about a synchronization job item.
