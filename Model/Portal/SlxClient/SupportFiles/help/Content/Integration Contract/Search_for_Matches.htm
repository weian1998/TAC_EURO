Search for Matches

The Search for Matches screen allows you to select the criteria to use when searching the accounting system for customers that are potential matches for the account you are linking from Sage SalesLogix.

To select

  1. Open the Linking wizard and proceed through the previous screens.
  1. Complete the Search for Matches screen, and click Next to go to the next step.
Box Description/Action
Property Select a field to use when searching for matching data. The available fields are fields mapped between Sage SalesLogix and the account system.
Operator Select the matching criteria used by the system for the selected property. Available operators are Equal To, Starts With, and Contains.
Search Value Type the value you want to search on.
