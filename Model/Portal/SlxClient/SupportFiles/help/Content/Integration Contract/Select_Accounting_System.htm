Select an Accounting System

The Select an Accounting System screen in the Linking Wizard displays the accounting systems available to link with the selected account.

To select

  1. Open the Linking wizard.
  1. Complete the Select an Accounting System screen, and click Next to go to the next step.
Box Description/Action
System Select the accounting system with which you want to link the account.
