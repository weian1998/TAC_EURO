Merge Contacts

The Merge Contacts screen displays contacts for the Sage SalesLogix account you are linking to the customer you selected in the accounting system. Use this screen to create links and merge your Sage SalesLogix and accounting system contact records.

To select

  1. Open the Linking wizard and proceed through the previous screens.
  1. Complete the Merge Contacts screen.
    BoxDescription/Action
    Unlinked Contacts from Sage SalesLogixDisplays the contacts in Sage SalesLogix that are available to link to the accounting system contact. Click Select to select a contact.
    Unlinked Contacts from [your accounting system endpoint]Displays the contacts in the accounting system that are available to link to a Sage SalesLogix contact. Click Select to select a contact.
    Link button

    Select an unlinked contact in the Unlinked Contacts from ERP and Unlinked Contacts from SLX grids, and then click Link to create a connection.

    To select a contact, click the Select link.

    Linked Contacts

    Displays the system matches that have been identified as potential links using your defined matching criteria.

    • Click Unlink to clear the proposed link and create a new link.
    • Click Select to display the contact data fields in the source and target systems.
  2. Click Next to go to the final step, and then click Close.
