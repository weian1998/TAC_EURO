Accounting Integration Detail View

Use the Accounting Integration Detail view to configure the connections between Sage SalesLogix and your accounting system(s).

Accounting Integration is enabled when the Sage SalesLogix endpoint and at least one target endpoint are configured and enabled. If no feeds are enabled on the Accounting Systems tab, then integration is disabled.

When you enable or disable Accounting Integration, changes do not take effect until you log off and log back on to Sage SalesLogix.

Set your feed details for Sage SalesLogix, then configure the feed for each integrated accounting system.

To set

  1. On the Administration menu, click Integration Setup
  2. In the SalesLogix Feed box, type the SData URL that Sage SalesLogix uses to share information between systems.
    1.  Do one of the following:
      • If your implementation does not use the default port (port 80), use format http://server:port/sdatavirtualdir/application/contract/operatingcompany/-/
      • If your implementation uses port 80, omit the port number from the URL. The Sync Service automatically strips the default port from the URL which causes matching issues and errors. Avoid this by using format http://server/sdatavirutaldir/application/contract/operatingcompany/-/
    2. Set application to slx.
    3. Set contract to gcrm.

    4. Set operatingcompany to - to the operating company in your accounting application that you are integrating with.(Sage SalesLogix does not have an operating company).

  3. In the User Name box, type the user name for the Sage SalesLogix database that the accounting system connects to.
  4. In the Password box, type the password for the user name.
  5. In the Display Name box, type the display name for the feed.
  6. If necessary, select the Restrict account promotion to single accounting system option to limit your integration to Sage SalesLogix and one accounting system.
  7. If necessary, select the Allow accounting system to manage Sales Orders option to remove the ability to create and edit sales orders in Sage SalesLogix. When you select this option, your accounting system must be used to manage sales orders.

    8. This option is required for certain integrations, for example with X3.
  8. Click Save.
  9. Use the Accounting Systems tab to configure the feed details for your accounting system(s).
  10. Use the Synchronization tab to configure synchronization intervals.
  11. Use the Matching tab to set global default matching criteria for users creating links.
