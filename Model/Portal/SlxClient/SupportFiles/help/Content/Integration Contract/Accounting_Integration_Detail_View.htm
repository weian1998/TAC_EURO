Accounting Integration Detail View

Use the Accounting Integration Detail view to configure the connections between Sage SalesLogix and your accounting system(s).

Accounting Integration is enabled when the Sage SalesLogix endpoint and at least one target endpoint are configured and enabled. If no feeds are enabled on the Accounting Systems tab, then integration is disabled.

When you enable or disable Accounting Integration, changes do not take effect until you log off and log back on to Sage SalesLogix.

Set your feed details for Sage SalesLogix, then configure the feed for each integrated accounting system.

To set

  1. On the Administration menu, click Integration Setup.
  2. In the SalesLogix Feed box, type the SData URL to the GCRM feed.

Use the format: http://server:port/sdatavirtualdir/slx/gcrm/-/ 

  • Set sdatavirtualdir to the directory to which you deployed your SData portal.

The default directory is "SData".

For example, http://localhost:3333/sdaata/slx/gcrm/-/

  1. In the User Name box, type the user name for the Sage SalesLogix database that the accounting system connects to.
  2. In the Password box, type the password for the user name.
  3. In the Display Name box, type the display name for the feed.
  4. Click Save.
  5. If necessary, select the Restrict account promotion to single accounting system option to limit your integration to Sage SalesLogix and one accounting system.
  1. Use the Accounting Systems tab to configure the feed details for your accounting system(s).
  2. Use the Synchronization tab to configure synchronization intervals.
  3. Use the Matching tab to set global default matching criteria for users creating links.
