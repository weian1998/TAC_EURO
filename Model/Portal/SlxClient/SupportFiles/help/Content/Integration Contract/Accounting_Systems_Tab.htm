Accounting Systems Tab

Use the Accounting Systems tab to manage the connections (feeds) to your accounting systems.

How Do I?

Add or Edit a Feed

  1. Open the Accounting Integration Detail view.
  2. Click the Accounting Systems tab.
  3. Do one of the following:
  • To add a feed, click Setup Accounting Integration.
  • To edit an existing feed, click the Edit link for the feed you want to change.
  1. Complete the Setup Accounting Integration dialog box.
  2. Click OK.

If this is not the first accounting system feed to be linked, you must reset IIS on the Web server so that any new feeds will be included during synchronization.

Synchronize All Accounting Systems

  1. Open the Accounting Integration Detail view.
  2. Click the Accounting Systems tab.
  3. Click the Sync all accounting systems button.
