Configuring Accounting Integration

Complete the following tasks to configure Accounting Integration.

  • If multi-currency is enabled, ensure Sage SalesLogix and your accounting system have the same base currency.
  • Accounting Integration is enabled when the Sage SalesLogix endpoint and at least one target endpoint is configured and enabled. Use the Accounting Integration Detail view to enable endpoints.

When you enable or disable Accounting Integration, changes do not take affect until you log off and log back on to Sage SalesLogix.
  • Compare the database schema in Sage SalesLogix with your accounting application schema to ensure data integrity. Use the Accounting Systems tab to determine schema differences.

If you have customized Sage SalesLogix you must review the Sage CRM ERP Contract and verify information will synchronize between systems. For example, if you created custom entities that do not match Sage SalesLogix, synchronization may not work.
  • Use the Matching tab to set global default matching criteria for users creating links.
  • Configure the Accounting Integration Synchronization Service.
  • If necessary), export and merge records from your accounting system.

Use the export tool provided by your accounting system to export the appropriate data into Sage SalesLogix

  • Configure user roles.
