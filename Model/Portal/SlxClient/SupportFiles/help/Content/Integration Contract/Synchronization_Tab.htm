Synchronization Tab

Use the Synchronization tab to configure how often linked records synchronize between Sage SalesLogix and your accounting system.

Synchronization runs for each accounting system in the order the linked accounting systems are listed on the Accounting Systems tab. Synchronization is scheduled from the start of the synchronization cycle. Tell me more... For example, if you set the interval for 30 minutes and the sync cycle runs for 45 minutes, the next synchronization will start immediately after the first finishes.

To set

  1. Open the Accounting Integration Detail view.
  2. Click the Synchronization tab.
  3. Type a value in the Specify synchronization interval in minutes box to set the synchronization interval.
  4. Click Save.
