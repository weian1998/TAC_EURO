Working with the Navigation Bar Shortcut Menus

The Navigation Bar shortcut menus enable you to open a list view, create a new record, or when applicable, create a new group. The shortcut menu options available vary depending on the record type.

The Navigation Bar shortcut menu is not available for What's New, Library, Reports, or Processes.

To open the navigation bar shortcut menu

  • On the Navigation Bar, right-click Contacts, Accounts, Opportunities, Leads, Contracts, Returns, Defects or Tickets.
Menu Item Description
Lookup... Opens the Lookup.
New... Opens the New Contact/Account, New Opportunity, New Lead, Insert New Contract, Insert New Defect, Insert Return, or New Ticket view.
Save as Group Only available in a List view after a Lookup is performed. Creates a new group using the lookup search criteria.
Create New Group Opens the New Group dialog box that allows you to define group criteria.

To open the shortcut menu for calendar or activities

  • On the Navigation Bar, right-click Calendar or Activities.
    • Menu Item Description
    Schedule Phone Call Opens the Schedule Phone Call view.
    Schedule Meeting Opens the Schedule Meeting view.
    Schedule To Do Opens the Schedule To Do view.
    Schedule Personal Activity Opens the Schedule Personal Activity view.
    Schedule Event Opens the Schedule Event view.
