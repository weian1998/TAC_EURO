Editing Your Personal Workspace Options

Use this view to change how your Personal Workspace appears. You can change the name of the tab and the template for how the information on the tab displays.

To edit

  1. Click the Personal Workspace tab you want to edit.
  2. On the Personal Workspace tab menu, click Edit Options.
  3. If necessary, in the Name box, type the name you want to display as the tab name.
  4. To make this tab the default tab, select Make this my default page.
  5. Under Choose the template you want to use, select the option that matches the layout you would like to use for this tab. Your choices are:
  • Two columns split
  • Three columns split
  • Two column larger left
  • Two column larger right
  • One column
  1. Click OK.
