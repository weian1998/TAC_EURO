Task Pane Common Tasks

Use the Task Pane common tasks to complete your work. The available options depend on the record type and type of view where you are working.

Not all options are available from all views.

 

Sales

The following table explains which options are available from the Task Pane on each list view.

If you do not select any records before selecting a common task, you will be asked if all records in the group should be used. If you click OK, all group records will be selected, regardless of any filters.
