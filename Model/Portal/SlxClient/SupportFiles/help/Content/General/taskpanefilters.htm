Using Task Pane Filters

Use the Task Pane Filters to narrow the list of records that display in a list view group. The available filters are based on the columns in the selected list view and the information contained in those columns. Filters are saved on your computer until your browser cache is cleared.

Next to each filter item is the number of records that meet the criteria of any applied filters and the total number of records in the group that match the filter item. Example If (12/27) displays next to an item, then there are a total of 27 records within the group that match the filter item, but only 12 of those records also meet the criteria of any applied filters.

If the corresponding column for a selected filter is removed from the list view group, the filter will not be applied to the list of records.

Each filter consists of the criteria you select. When you apply a filter, only the records that meet all of the criteria display. For example:

How Do I?

Apply a filter
  1. From a list view, in the Task Pane, under Filters, click a filter name. For example, click Owner.
  2. Select the filter items that you want to apply to the list of records. The available filter items can be customized.
Clear a filter
  • Click the filter you want to clear and click Clear.
Clear a selected filter item
  1. Click the filter that includes the filter item you want to clear.
  2. Click the filter item to clear the check box.
Clear all filters
  • From a list view, in the Task Pane, click Clear All.
    Clearing all filters will clear any filters and display all records within the group.
