Using Task Pane Filters

Use the Task Pane filters to narrow the list of records that display in a list view. You can add or remove filters from the list and create a "short list" of the criteria available for each filter. The available filters are based on the columns in the selected list view and the information contained in those columns.

If the corresponding column for a selected filter is removed from the list view group, the filter will not be applied to the list of records.

Each filter consists of the criteria you select. When you apply a filter, only the records that meet all of the criteria display. For example:

How Do I?

Add or remove filters that display in the Task Pane
  1. From a list view, in the Task Pane, click Edit.
  2. Select the check boxes of the filters you want to display in the Filters list. Clear the check boxes of the filters you want to hide.

  3. Close the Edit Filters dialog box.
Create a "short list" of filter criteria
  1. From a list view, in the Task Pane, click a filter heading. For example, click Owner.

  2. In the list of filter criteria options, clear the check boxes for any criteria that you do not want to be available under the filter. To hide all of the filter criteria, clear the All check box. If you want all of the filter criteria to be available, select the All check box.

  3. Close the list of filter criteria.

The "short list" is the filter criteria available when you view a filter.

Apply a filter
  1. From a list view, in the Task Pane, expand the filter that you want to apply. For example, click the "[+]" next to Owner.

  2. Select the filter criteria check boxes that you want to apply to the list of records. The available criteria is based on the "short list" of criteria that you specified.

The list view only displays records that meet the filter criteria.

Clear all filters
  • From a list view, in the Task Pane, click Clear All.

Clearing all filters will display all records within the group.
