Using the Task Pane

The Task Pane consists of common tasks and features designed to help you complete your work. The available options depend on the record type and type of view where you are working.

From a list view, you can:

  • Filter Records
  • Perform Common Tasks

From a detail view, you can:

  • Use the Group List
  • Perform Common Tasks
  • Perform Process Tasks

    Process Tasks are only available if your administrator has configured process orchestration.

From the Web Form Designer, you can:

  • Add fields

    The system administrator and users with the appropriate role can access these features.

How Do I?

Find the Task Pane
  • The Task Pane displays to the right of the main window pane.
Expand or collapse the Task Pane
  • Move your cursor over the inner border of the pane you want to resize and drag the pane to the desired size.
Expand or collapse sections within the Task Pane
  • Click the column heading of the section you want to collapse. To expand the section, click the heading again.

Filter Records

Perform Common Tasks

The available options depend on the record type and type of view where you are working. To find out which options are available for a specific view, please see the help topic for that view. For example, for the Account Detail View, see the topic "Using the Account Detail View."

If you do not select any records before selecting a common task, you will be asked if all records in the group should be used. If you click OK, all group records will be selected, regardless of any filters.
Perform Process Tasks
  • Under Process Tasks, click Click to Complete next to the item to open and complete.
  • Click Cancel to cancel a step.
