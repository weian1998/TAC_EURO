Installing and Using Sage SalesLogix Desktop Integration

The Sage SalesLogix Desktop Integration Module is a collection of features to enhance your Sage SalesLogix Web experience.

These features include:

  • Drag and Drop Attachments
  • Drag and Drop E-mail from Microsoft Outlook
  • Outlook Integration - Send SLX and Record to History
  • Mail Merge
  • Export to Excel
  • Drag and Drop Library files (Administrator only)

How Do I?

Install Desktop Integration Module if Internet Explorer is my browser
  1. Do one of the following:
    • On the Sage SalesLogix Web Client login screen, click Install Enhancements.
    • On the Tools menu, click Options, click the General tab, and then click Install Enhancements.
    • If prompted to install Sage SalesLogix Desktop Integration and you want to use this feature, click Yes.

      After installing the Desktop Integration Module, the button changes to Enhancements Installed. You can click the button to reinstall at any time, but reinstalling is only necessary when you are instructed to do so by your administrator.

  2. Click Run.
  3. Close Microsoft Outlook and any open browser windows.
  4. To install the Desktop Integration Module, click Yes.
  5. If prompted that Microsoft Outlook or Internet Explorer is in use, close each program, and click Retry.
  6. On the Welcome screen, click Next.
  7. On the License Agreement screen, read the information, accept the agreement, and then click Next.
  8. Do one of the following:
    • Select Complete to install all of the Desktop Integration features to the default install location.
    • Select Custom to select which Desktop Integration features you want to install or change the install location.
  9. Click Next.
  10. If you selected the Custom install, do any of the following, and then click Next:
    • If you do not want to install Office Integration (Mail Merge, Export to Excel, drag and drop), click the drop-down arrow next to Office Integration and select This feature will not be available.
    • If you do not want to install Microsoft Outlook Integration (Send SLX/Record to History and Desktop Manager), click the drop-down arrow next to Outlook Integration and select This feature will not be available.
    • To change the install location, click Change, browse to or type the desired install location, and then click OK.
  11. Click Install.
  12. Click Finish.

The Desktop Manager opens, which allows you to set your Desktop Integration options for Send SLX and Record to History.

If you are not using Outlook Integration, you do not need to configure any options in the Desktop Manager.

    To open the Sage SalesLogix Desktop Manager manually, in the System Tray section of the Windows Taskbar, double-click the Sage SalesLogix Desktop Manager icon.
Install Desktop Integration Module if Firefox is my browser

To install using Firefox

  1. Do one of the following:
    • On the Sage SalesLogix Web Client login screen, click Install Enhancements.
    • In Options, click the General tab and click Install Enhancements.
    • If prompted to install Sage SalesLogix Desktop Integration and you want to use this feature, click Yes.

      After installing the Desktop Integration Module, the button changes to Enhancements Installed. You can click the button to reinstall at any time, but reinstalling is only necessary when you are instructed to do so by your administrator.

  2. Click Save File.
  3. In the Downloads dialog box, double-click SLXDesktopIntegrationSetup.exe.
  4. Click Yes.
  5. Close Microsoft Outlook and any open browser windows.
  6. If prompted that Microsoft Outlook or Firefox is in use, close each program, and click Retry.
  7. On the Welcome screen, click Next.
  8. On the License Agreement screen, read the information, accept the agreement, and then click Next.
  9. Do one of the following:
    • Select Complete to install all of the Desktop Integration features to the default install location.
    • Select Custom to select which Desktop Integration features you want to install or change the install location.
  10. Click Next.
  11. If you selected the Custom install, do any of the following, and then click Next:
    • If you do not want to install Office Integration (Mail Merge, Export to Excel, drag and drop), click the drop-down arrow next to Office Integration and select This feature will not be available.
    • If you do not want to install Microsoft Outlook Integration (Send SLX/Record to History and Desktop Manager), click the drop-down arrow next to Outlook Integration and select This feature will not be available.
    • To change the install location, click Change, browse to or type the desired install location, and then click OK.
  12. Click Install.
  13. Click Finish.
    The Desktop Manager opens, which allows you to set your Desktop Integration options for Send SLX and Record to History.
    If you are not using Outlook Integration, you do not need to configure any options in the Desktop Manager.
