Was sind Gruppen?

Gruppen sind eine Zusammenstellung von Datensätzen. Gruppen helfen Ihnen bei der Verwaltung Ihrer Arbeitsabläufe. Mithilfe von Gruppen können Sie mit einem Teil Ihrer Datensätze arbeiten, Daten für den Ausdruck in Berichten auswählen und Mailings, E-Mails und Faxe mithilfe der Seriendruckfunktionalität senden. Ein Datensatz kann mehreren Gruppen angehören. Sie können Gruppendatensätze nach Excel exportieren oder ihre Gruppen für andere Benutzer freigeben.

Um weitere Gruppenregisterkarten anzuzeigen, klicken Sie auf die Pfeile.

Es sind drei Arten von Gruppen verfügbar:

  • Ad-hoc-Gruppen - Dies sind anhand ausgewählter Datensätze erstellte Gruppen.
  • Suchgruppen - Dies sind auf einer Suche basierende Gruppen.
  • Mit erweiterten Abfragen erstellte Gruppen - Dies sind Gruppen, die auf Bedingungen des Abfrage-Assistenten basieren.

So öffnen Sie eine Gruppe

  • Führen Sie einen der folgenden Schritte durch:
    • Wählen Sie im Menü Gruppen die Gruppe aus, die Sie anzeigen möchten.
    • Klicken Sie auf die Registerkarte der Gruppe, die Sie öffnen möchten. Zum Beispiel Alle Firmen.

So zeigen Sie Gruppenbedingungen an

  1. Klicken Sie mit der rechten Maustaste auf eine Gruppenregisterkarte.
  2. Klicken Sie auf Bearbeiten.
    Der Abfrage-Assistent von SalesLogix wird geöffnet.
  3. Auf der Registerkarte Bedingungen des Abfrage-Assistenten werden die Bedingungen angezeigt, die Datensätze in der Gruppe erfüllen müssen. Diese Registerkarte ist für Ad-hoc-Gruppen nicht verfügbar.
