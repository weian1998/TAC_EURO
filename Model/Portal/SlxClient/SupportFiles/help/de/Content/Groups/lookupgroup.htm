Speichern einer Suche als Gruppe

Nachdem Sie eine Suche erstellt haben, um Kontakte, Firmen, Verkaufschancen, Interessenten oder Tickets zu suchen, können Sie die Suche als Gruppe speichern, damit Sie später wieder auf sie zugreifen können.

So erstellen Sie eine Gruppe mithilfe einer Suchabfrage

  1. Führen Sie in einer Listenansicht eine Suchedurch.
  2. Klicken Sie mit der rechten Maustaste auf die Registerkarte Suchergebnisse, und wählen Sie Suche als Gruppe speichern aus.
  3. Geben Sie im Dialogfeld Suchergebnisse als neue Gruppe speichern einen beschreibenden Namen in das Feld Gruppenname ein.
  4. Klicken Sie auf OK.
