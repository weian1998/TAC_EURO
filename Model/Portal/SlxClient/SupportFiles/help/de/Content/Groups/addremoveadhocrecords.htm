Hinzufügen und Entfernen von Ad-hoc-Gruppen-Datensätzen

Sie können Ihre Ad-hoc-Gruppen durch Hinzufügen oder Entfernen von Datensätzen verwalten.

So fügen Sie Datensätze zu einer vorhandenen Ad-hoc-Gruppe hinzu

  • Führen Sie einen der folgenden Schritte durch:
  • Öffnen Sie eine Listenansicht. Öffnen Sie eine Gruppe, oder führen Sie eine Suche durch, um die Datensätze zu suchen, die zu der Gruppe hinzugefügt werden sollen.Wählen Sie einen oder mehrere Datensätze aus, indem Sie beim Klicken auf die einzelnen Einträge die STRG- oder UMSCHALTTASTE drücken. Klicken Sie mit der rechten Maustaste in das Raster, und wählen Sie Zu vorhandener Gruppe hinzufügen aus, oder wählen Sie im Aufgabenbereich die Option Zu Gruppe hinzufügen aus.

Wenn Sie keine Datensätze auswählen, wird angefragt, ob alle Datensätze in der Gruppe verwendet werden sollen. Wenn Sie auf OK klicken, werden alle Gruppendatensätze ausgewählt, und zwar unabhängig von Filtern, die eventuell im Aufgabenbereich festgelegt sind.
  • Öffnen Sie die Detailansicht für den Datensatz, den Sie hinzufügen möchten. Wählen Sie im AufgabenbereichZu Gruppe hinzufügen aus.

So gehen Sie zum Entfernen vor

  1. Öffnen Sie die Ad-hoc-Gruppe, die Sie aktualisieren möchten.
  2. Führen Sie einen der folgenden Schritte durch:
    • Wählen Sie in der Listenansicht einen oder mehrere Datensätze aus, indem Sie beim Klicken auf die einzelnen Einträge die STRG- oder UMSCHALTTASTE drücken. Klicken Sie mit der rechten Maustaste in das Raster der Listenansicht und wählen Sie Aus Gruppe entfernen aus oder wählen Sie im Aufgabenbereich den Befehl Aus Gruppe entfernen aus.
    • Öffnen Sie die Detailansicht für den Datensatz, den Sie entfernen möchten. Wählen Sie im Aufgabenbereich den Befehl Aus Gruppe entfernen aus.
