Hinzufügen einer Ad-hoc-Gruppe

Sie können eine Ad-hoc-Gruppe erstellen, um eine Liste mit Datensätzen zu speichern, die Sie in einer der Listenansichten ausgewählt haben.

So fügen Sie eine Ad-hoc-Gruppe hinzu

  1. Öffnen Sie eine Listenansicht.
  2. Öffnen Sie eine Gruppe oder führen Sie eine Suche durch, um die Datensätze zu suchen, die zu der Gruppe hinzugefügt werden sollen.
  3. Wählen Sie einen oder mehrere Datensätze aus, indem Sie beim Klicken auf die einzelnen Einträge die STRG- oder UMSCHALT-Taste gedrückt halten, und führen Sie dann einen der folgenden Schritte aus:
    • Wählen Sie im Aufgabenbereich den Befehl Datensätze als Gruppe speichern aus.
    • Klicken Sie mit der rechten Maustaste in das Raster der Listenansicht und wählen Sie Datensätze als Gruppe speichern aus.

    		4.

    Wenn Sie keine Datensätze auswählen, wird angefragt, ob alle Datensätze in der Gruppe verwendet werden sollen. Wenn Sie auf OK klicken, werden alle Gruppendatensätze ausgewählt.
  4. Geben Sie im Dialogfeld Datensätze zu neuer Gruppe hinzufügen einen beschreibenden Namen in das Feld Gruppenname ein.

  5. Klicken Sie auf OK.
