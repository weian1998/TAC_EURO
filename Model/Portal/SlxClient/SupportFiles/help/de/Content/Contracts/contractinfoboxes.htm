Vertragsinformationsfelder

Durch Ihren Sicherheitszugriff wird bestimmt, welche Funktionen verfügbar sind. Wenden Sie sich an den Administrator, wenn Sie Ihre Zugriffsrechte ändern möchten. Für WebViewer-Benutzer kann die Funktionalität eingeschränkt sein.

 

Feld Beschreibung/Aktion
Referenznr. Standardmäßig weist das System eine eindeutige Referenznummer für interne Überwachungszwecke zu, Sie können jedoch eine andere Nummer eingeben. Ist eine Referenznummer bereits vorhanden, wird das Dialogfeld Vertragsreferenznr. - Duplikat geöffnet.
Aktiv Sie können dieses Kontrollkästchen aktivieren, um den Vertrag zu aktivieren, oder das Kontrollkästchen deaktivieren, um den Vertrag zu deaktivieren.
Firma Hier wird der Name der Firma angezeigt, die mit dem Vertrag verknüpft ist. Zum Ändern der Firmaklicken Sie auf die Schaltfläche Suchen. Nach dem Erstellen des Vertrags kann dies nicht mehr geändert werden.
Kontakt Hier wird der Name des Hauptkontakts angezeigt, der mit dem Vertrag verknüpft ist. Zum Ändern des primären Kontaktsklicken Sie auf die Schaltfläche Suchen.
Kommentare In dieses Feld können weitere Informationen zum Vertrag eingegeben werden.
Service Der Service-Typ, den der Vertrag abdeckt. Zum Ändern des Service klicken Sie auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
Vertragstyp Zum Ändern des Service klicken Sie auf die Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus. Die Auswahl des Vertragstyps wirkt sich auf die Optionen in den anderen Vertragsinformationsfeldern aus. Wenn der Vertragstyp beispielsweise Tage lautet, wird das Feld Frist (Tage) verfügbar. Nach dem Erstellen des Vertrags kann das Feld Vertragstyp nicht mehr bearbeitet werden.
Anfangsdatum Das Anfangsdatum des Vertrags. Für die Bearbeitung klicken Sie auf die Kalender-Schaltfläche, um das Datum auszuwählen.  
Mit dem ersten Ticket beginnen Wenn der Vertrag nach dem Erstellen des ersten Tickets in Kraft treten soll, aktivieren Sie dieses Kontrollkästchen. Diese Option ist nur verfügbar, wenn für den Vertragstyp die Option Tage gewählt wurde. Wenn diese Option aktiviert ist, sind die Felder Anfangsdatum und Enddatum leer, bis das erste Ticket geöffnet wird. Nach dem Erstellen des Vertrags kann dies nicht mehr geändert werden.
Ablaufdatum Das Ablaufdatum des Vertrags. Für die Bearbeitung klicken Sie auf die Kalender-Schaltfläche, um das Datum auszuwählen.  
Frist (Tage) Geben Sie Anzahl von Tagen nach dem Ablaufdatum ein, für die der Service oder Support dem Vertragskunden angeboten werden soll. Gilt für alle Vertragstypen.
Menge

Die Anzahl der Einheiten des im Feld Vertragstyp angezeigten Typs. Die Anzahl wird je nach den Informationen in den Feldern Vertragstyp, Frist und Mit dem ersten Ticket beginnen möglicherweise automatisch angezeigt oder kann vom Benutzer eingegeben werden.

Wenn Vertragstyp auf Tage festgelegt und eine Menge angegeben ist, wird das Ablaufdatum basierend auf dem Anfangsdatum berechnet.
Rest Zeigt die noch geltende Dauer des aktuellen Vertrags an. Das System berechnet die Restlaufzeit dieses Vertrags auf Basis der Einträge für Vertragstyp, Frist (Tage), Kaufdatum, Anfangsdatum und Enddatum. Wenn Tickets und Ticket-Aktivitäten erstellt und abgeschlossen werden, wird der verbleibende Wert reduziert. Das Feld wird angezeigt, wenn ein vorhandener Vertrag angezeigt wird, und kann nicht bearbeitet werden.
Bestellnummer Falls möglich, kann der Benutzer eine Bestellnummer für den Vertrag eingeben.
Kaufdatum Das Kaufdatum. Für die Bearbeitung klicken Sie auf die Kalender-Schaltfläche, um das Datum auszuwählen.  
Listenpreis Geben Sie einen Wert für den Vertrag ein. Wenn kein Wert eingegeben wird, verwendet das System standardmäßig den Wert von Vertragsbetrag abzüglich des Werts für Rabatt.
Rabatt Falls möglich, kann der Benutzer einen Rabattwert eingeben.
Vertragsbetrag Geben Sie den Wert für den Vertrag ein. Wenn kein Betrag eingegeben wird, verwendet das System standardmäßig den Wert von Listenpreis abzüglich des Werts für Rabatt.
Aktualisiert von Zeigt den Benutzer an, der die Vertragsinformationen zuletzt aktualisiert hat. Das Feld wird angezeigt, wenn ein vorhandener Vertrag angezeigt wird, und kann nicht bearbeitet werden.
Letzte Aktualisierung Zeigt das Datum und die Uhrzeit der letzten Aktualisierung der Vertragsinformationen an. Das Feld wird angezeigt, wenn ein vorhandener Vertrag angezeigt wird, und kann nicht bearbeitet werden.
