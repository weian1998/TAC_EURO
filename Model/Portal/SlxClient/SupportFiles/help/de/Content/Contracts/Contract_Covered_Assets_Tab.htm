Registerkarte "Abgedeckte verkaufte Artikel"

Die Registerkarte Abgedeckte verkaufte Artikel zeigt Informationen zu jedem verkauften Artikel an, der von dem Vertrag abgedeckt wird. Auf dieser Registerkarte können Sie vom Vertrag abgedeckte verkaufte Artikel verknüpfen oder eine Verknüpfung aufheben.

So gehen Sie zum Öffnen vor

  1. Öffnen Sie die Vertragsdetailansicht.
  2. Klicken Sie auf die Registerkarte Abgedeckte verkaufte Artikel. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.

So gehen Sie zum Verknüpfen vor

  1. Klicken Sie auf Verkauften Artikel verknüpfen.
  2. Über die Suche können Sie den verkauften Artikel finden und auswählen.
  3. Klicken Sie auf Auswahl hinzufügen.

So gehen Sie zum Aufheben einer Verknüpfung vor

  1. Wählen Sie den Datensatz aus, den Sie entfernen möchten, und klicken Sie auf die Schaltfläche Entfernen.
  2. Klicken Sie auf Ja.
