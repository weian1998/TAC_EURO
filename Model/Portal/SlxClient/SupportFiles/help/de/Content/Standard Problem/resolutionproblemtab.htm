Standardlösung - Registerkarte "Häufig auftretende Probleme"

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

In dieser Ansicht können Sie häufig auftretende Probleme anzeigen oder diesen eine Lösung zuordnen.

Vorgehensweise

Öffnen dieser Ansicht
  • Klicken Sie in der Detailansicht von Häufig auftretende Probleme auf die Registerkarte Häufig auftretende Probleme.
Zuordnen einer Lösung
  1. Klicken Sie auf die Schaltfläche Problem zuordnen.
  2. Suchen Sie mithilfe der Suchfunktion nach einem häufig auftretenden Problem.
Löschen einer Lösung
  1. Wählen Sie den Datensatz aus, den Sie entfernen möchten, und klicken Sie auf die Schaltfläche Entfernen.
  2. Klicken Sie auf OK.

Diese Ansicht enthält folgende Spalten:

  • Beschreibung - Eine Beschreibung des Problems.
  • Bereich - Gibt an, wo das Problem auftritt.
  • Kategorie - Gibt den Typ des Problems an.
  • Problem - Gibt das Problem an.
  • Notizen - Enthält Hinweise zum Problem.
