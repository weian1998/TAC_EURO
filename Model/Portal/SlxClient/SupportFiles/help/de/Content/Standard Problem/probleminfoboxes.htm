Häufig auftretendes Problem - Informationsfelder

Die Informationsfelder zu häufig auftretenden Problemen enthalten Informationen zum ausgewählten Problem. Sie können die Informationen ggf. bearbeiten und löschen.

 

Feld Beschreibung/Aktion
Beschreibung Geben Sie eine für andere Benutzer sinnvolle Beschreibung der Lösung ein.

Bereich

Kategorie

Problem

In diesen drei Feldern wird die Art des gemeldeten Problems kurz beschrieben. Klicken Sie auf die Schaltfläche Suchen neben den Feldern, wählen Sie die entsprechenden Einträge aus den Listen aus und klicken Sie auf OK.

Welcher Kategorie die aufgeführten Einträge angehören, ist von der Option abhängig, die Sie als Bereich ausgewählt haben, und welche Probleme verfügbar sind, ist von der Option abhängig, die Sie als Kategorie ausgewählt haben.
Notizen Geben Sie Informationen ein, die anderen Benutzern helfen, die Lösung zu verstehen.
