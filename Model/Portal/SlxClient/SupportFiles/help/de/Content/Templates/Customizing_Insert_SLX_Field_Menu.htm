Anpassen des Menüs "SLX-Feld einfügen"

In Sage SalesLogix können Sie die Felder im Menü SLX-Feld einfügen anpassen. Sie können den Formattyp und eine Zeichenfolge für das Feld festlegen, ein Trennzeichen einfügen, Felder entfernen oder Felder in der Liste nach oben oder unten verschieben.

So passen Sie das Menü SLX-Feld einfügen an:

  1. Öffnen Sie den Vorlagen-Editor.
  2. Klicken Sie auf der Symbolleiste "Vorlagen-Editor" auf SLX-Feld einfügen, und wählen Sie danach Anpassen aus.
    Das Dialogfeld Menü anpassen wird angezeigt.
  3. Führen Sie einen oder mehrere der folgenden Schritte aus, um das Menü anzupassen:
    • Zum Ändern des Formats eines Feldes wählen Sie das Feld aus und klicken dann auf Feld formatieren.
    • Wenn Sie dem Menü "SLX-Feld einfügen" ein Feld hinzufügen möchten, klicken Sie auf Feld einfügen.
    • Um ein Trennzeichen einzufügen, positionieren Sie den Cursor an der gewünschten Stelle im Menü und klicken dann auf Trennzeichen einfügen.
    • Wenn Sie ein Feld aus dem Menü entfernen möchten, wählen Sie den Feldnamen aus und klicken dann auf Entfernen.
    • Die Position eines Feldes im Menü können Sie ändern, indem Sie den Feldnamen auswählen und dann auf Nach oben oder auf Nach unten klicken.
    • Wenn Sie das standardmäßige Menü mit Sage SalesLogix-Seriendruckfeldern wiederherstellen möchten, klicken Sie auf Standardeinstellungen wiederherstellen. Klicken Sie im Meldungsfeld Standardeinstellungen wiederherstellen bestätigen auf Ja, um sämtliche Anpassungen zurückzusetzen, oder klicken Sie auf Nein, um die Anpassungen beizubehalten.
