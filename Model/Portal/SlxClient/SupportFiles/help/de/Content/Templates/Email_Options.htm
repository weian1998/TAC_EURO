E-Mail-Optionen

Beim Erstellen einer E-Mail-Vorlage können Sie Optionen für die E-Mail-Nachricht(en) festlegen, die Sie senden möchten. Es kann beispielsweise vorkommen, dass Sie als Administrator im Namen Ihres Vertriebsteams von einer echten Firmenadresse (z. B. salesteam@company.com) aus eine Massendrucksache als Seriendruck senden müssen. Beim Erstellen des Seriendrucks können Sie das Feld Antwort an der abgehenden E-Mail-Nachricht dynamisch ausfüllen lassen, damit die Antworten der Empfänger Ihrer E-Mail automatisch an den für sie zuständigen Account Manager adressiert werden.

Die Schaltfläche E-Mail-Optionen ist nur im E-Mail-Editor verfügbar.

So stellen Sie E-Mail-Optionen ein:

  1. Wählen Sie eine E-Mail-Vorlage aus und öffnen Sie danach den Vorlagen-Editor.
  2. Klicken Sie in der Symbolleiste auf E-Mail-Optionen.
  3. Wählen Sie im Feld Antwort an aus, wer im Feld An der Antwort auf Ihre E-Mail angezeigt werden soll. Klicken Sie auf die Schaltfläche Adressbuch, um einen bestimmten Namen aus dem Adressbuch von Outlook auszuwählen, oder wählen Sie aus dem Dropdown-Menü des Felds eine der folgenden Optionen:
  • <<Aktueller Benutzer>>: Mit dieser Option wird die Zeichenfolge <<Aktueller Benutzer>> in das Feld Antwort an eingefügt und der aktuelle Benutzer zum Seriendruckbenutzer bestimmt. Dies ist die Standardoption.
  • <<Account Manager>>: Mit dieser Option wird die Zeichenfolge <<Account Manager>> in das Feld Antwort an eingefügt und der Benutzer zum Account Manager des betreffenden Kontakts bestimmt. Diese Option ist nur für Outlook verfügbar.
  • Bestimmter Benutzer: Mit dieser Option wird das Adressbuch von Outlook geöffnet und der ausgewählte Outlook-Anzeigename im Feld Antwort an angezeigt, wodurch der Benutzer zum ausgewählten Benutzer aus dem Adressbuch bestimmt wird. Diese Option ist nur für Outlook verfügbar.
  1. Wählen Sie aus der Liste Wichtigkeit entweder Niedrig, Normal (Standard) oder Hoch aus.
  2. Wenn Sie die Vertraulichkeit der E-Mail-Nachricht mitteilen möchten, wählen Sie Normal (Standard), Persönlich, Privat oder Vertraulich aus der Liste Vertraulichkeit aus.
  3. Wenn Sie eine Benachrichtigung darüber erhalten möchten, dass die E-Mail-Nachricht gesendet wurde, wählen Sie Übermittlungsbestätigung anfordern aus.
  4. Wenn Sie eine Benachrichtigung darüber erhalten möchten, dass die E-Mail-Nachricht gelesen wurde, wählen Sie Lesebestätigung anfordern aus.
  5. Klicken Sie auf OK.
