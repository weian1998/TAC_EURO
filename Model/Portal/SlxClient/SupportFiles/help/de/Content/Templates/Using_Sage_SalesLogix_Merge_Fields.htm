Verwenden der Sage SalesLogix-Seriendruckfelder

Das Menü SLX-Feld einfügen enthält die am häufigsten verwendeten Sage SalesLogix-Seriendruckfelder. Wenn Sie jedoch eine Vorlage erstellen oder bearbeiten und ein Sage SalesLogix-Feld benötigen, das sich nicht im Menü befindet, dann können Sie Weitere SLX-Felder auswählen, um auf sämtliche Sage SalesLogix-Seriendruckfelder zuzugreifen. Darüber hinaus können Sie dem Menü Felder hinzufügen und das Menü anpassen.
  • Die Sage SalesLogix-Seriendruckfelder liegen im Format XML vor, daher muss die Groß-/Kleinschreibung berücksichtigt werden. Wenn die Groß-/Kleinschreibung eines Seriendruckfelds geändert wird, funktioniert das Zusammenführen der Daten nicht mehr.
  • Beim Erstellen oder Bearbeiten einer Vorlage dürfen keine Sage SalesLogix-Seriendruckfelder in einem Textfeld platziert werden, da diese Funktion von Microsoft Word nicht unterstützt wird.

Das Dialogfeld SalesLogix-Seriendruckfeld einfügen setzt sich aus zwei Teilfenstern zusammen:

  • Das linke Teilfenster enthält die Registerkarten mit den Kontakt- oder Interessenten-, Adressen- und Benutzerinfo-Tabellen. Klicken Sie auf das Pluszeichen, um die Verzeichnisstruktur mit den Tabellen zu erweitern. Wählen Sie Alle Joins anzeigen aus, um alle verbundenden Tabellen anzuzeigen.
  • Im rechten Teilfenster werden alle Felder der ausgewählten Tabelle angezeigt. Sie können mit der rechten Maustaste auf ein Feld klicken, um einen Local Join (lokale Verknüpfung) zu erstellen.
    • Registerkarte Feld
    Kontakt oder Interessent Enthält Informationen über den Empfänger des Briefs, einschließlich Name und Firma. Wenn in den Eigenschaften der Vorlage als HaupttabelleKontakt ausgewählt wurde, werden die Felder aus der Tabelle Kontakt angezeigt. Wenn als Haupttabelle Interessent ausgewählt wurde, werden die Felder aus der Tabelle Interessent angezeigt.
    Adresse Enthält Informationen zur Adresse des Empfängers. Verwenden Sie diese Registerkarte, wenn Sie die Adressfelder Kontakt oder Interessent auswählen.
    Benutzerinfo Enthält Felder über den Absender, einschließlich der Adresseninformationen.

So greifen Sie auf weitere Seriendruckfelder zu

  1. Klicken Sie im Menü SLX-Feld einfügen auf Weitere SLX-Felder.
  2. Wählen Sie eine der Registerkarten aus.
  3. Wählen Sie im linken Teilfenster einen Tabellennamen aus.
  4. Wählen Sie im rechten Teilfenster ein Feld aus und klicken Sie auf Einfügen.
  5. Fügen Sie der Vorlage nach Bedarf weitere Felder hinzu und klicken Sie anschließend auf Fertig.

So fügen Sie Felder zum Menü SLX-Feld einfügen hinzu

  1. Klicken Sie im Menü SLX-Feld einfügen auf Anpassen.
  2. Klicken Sie auf Feld einfügen.
  3. Klicken Sie auf die Registerkarte, die die Tabellen und Felder enthält, die Sie einfügen möchten.
  4. Wählen Sie im rechten Teilfenster ein Feld aus.
  5. Wenn Sie den Formattyp des Feldes ändern müssen, können Sie aus der Liste Formattypeinen Formattyp auswählen.
  6. Geben Sie in das Feld Formatzeichenfolge die Formatzeichenfolge ein.
  7. Das Feld wird im Menü wie dargestellt angezeigt; Sie können jedoch im Feld Anzeigen als einen Anzeigenamen für das Feld eingeben.
  8. Klicken Sie auf Zu Menü hinzufügen.
  9. Fügen Sie weitere Felder hinzu, oder klicken Sie auf Fertig.
