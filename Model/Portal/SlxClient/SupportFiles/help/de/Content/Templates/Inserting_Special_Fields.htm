Einfügen von Spezialfeldern

Die Option zum Einfügen von Spezialfeldern steht nur erfahrenen Benutzern zur Verfügung.

Sie können auf die Schaltfläche Spezialfeld einfügen des E-Mail-, Fax- oder Briefvorlagen-Editors klicken, um eine neue Verkaufschancen-Produkttabelle, eine SQL-Anweisung oder ein benutzerdefiniertes Seriendruckfeld einzufügen.

So fügen Sie ein Spezialfeld ein:

  1. Öffnen Sie den Vorlagen-Editor.
  2. Klicken Sie in der Symbolleiste auf Spezialfeld einfügen.
  3. Wählen Sie aus dem Menü den Feldtyp aus, den Sie zur Vorlage hinzufügen möchten.
    • Verkaufschancen-Produkttabelle

    Damit wird eine Tabelle mit Informationen zu Verkaufschancen-Produkten und -Preisen hinzugefügt.

    • SQL

      • Sie können die folgenden SQL-Felder einfügen:

      • Text
      • Bild
      • Tabelle
      • Aufzählung
      • Nummerierte Liste
    • Benutzerdefiniert

      • Wenn ein benutzerdefiniertes Feld zur Vorlage hinzugefügt wird und die Skripts nicht ordnungsgemäß geändert wurden, wird der Seriendruck erst ausgeführt, nachdem die Skripts korrigiert wurden oder das benutzerdefinierte Feld entfernt wurde.

    Bevor Sie ein benutzerdefiniertes Feld einfügen, muss das benutzerdefinierte Feld bearbeitet werden, indem im Skript "OnCustomFieldName" für SLX-Seriendrucke (LAN) oder in der Klassenfunktion "Sage.MailMergeService.prototype.HandleOnCustomFieldName" unter "jscript\sage-mailmerge\sage-mailmerge-service.js") Code geschrieben wird.

    Das Dialogfeld des Feld-Editors für die SalesLogix-Seriendruckfunktion wird geöffnet.

  4. Nehmen Sie bei Bedarf mithilfe der Funktion Felder bearbeiten Ihre Änderungen vor und klicken Sie danach auf Anwenden.
  5. Klicken Sie auf OK.
