Verwenden des Vorlagen-Editors

Im Vorlagen-Editor können Sie eine neue Vorlage erstellen oder eine vorhandene Vorlage bearbeiten. Der Editor enthält eine Sage SalesLogix-Symbolleiste, über die Sie Zugriff auf die speziellen Vorlagenfunktionen sowie zahlreiche Microsoft Word-Funktionen haben. Welche Funktionen verfügbar sind, hängt davon ab, ob es sich bei der Vorlage um eine Vorlage vom Typ "Brief", "E-Mail" oder "Fax" handelt.

Damit Sie sich schnell mit den Symbolleistenschaltflächen vertraut machen können, bietet Sage SalesLogix QuickInfos zur Identifizierung jeder Schaltfläche. Zum Anzeigen einer QuickInfo bewegen Sie den Cursor über die Symbolleistenschaltfläche.

Da der Editor in Microsoft Word integriert ist, sind auch Word-Symbolleisten sichtbar. Klicken Sie auf die Schaltfläche Word-Menü anzeigen, um zusätzliche Microsoft-Word-Symbolleisten anzuzeigen, die zurzeit nicht sichtbar sind.

So öffnen Sie den Vorlagen-Editor:

  1. Klicken Sie im Menü Verfassen auf Vorlagen.

  2. Führen Sie einen der folgenden Schritte durch:

    • Wählen Sie eine private Vorlage aus und klicken Sie auf die Schaltfläche Bearbeiten.
    • Klicken Sie auf die Schaltfläche Neu, legen sie die Vorlageneigenschaften fest und klicken Sie anschließend auf OK.

So fügen Sie einer Vorlage SLX-Seriendruckfelder hinzu

  1. Setzen Sie den Cursor in der Vorlage an die Position, an der das Seriendruckfeld platziert werden soll.
  2. Klicken Sie auf SLX-Feld einfügen.
    Im Menü werden dann die am häufigsten verwendeten Sage SalesLogix-Seriendruckfelder angezeigt.
  3. Wählen Sie einen Feldnamen aus.
  4. Klicken Sie auf Weitere SLX-Felder, um auf zusätzliche Sage SalesLogix-Seriendruckfelder zuzugreifen.
  5. Um eine Vorschau der Seriendruckvorlage anzuzeigen, klicken Sie in der Symbolleiste auf Vorschau.
    Dadurch werden die Daten des aktuellen Kontakts mit der Vorlage zusammengeführt.
    Sie können eine Vorlage im Vorschaufenster weder bearbeiten noch speichern. Schließen Sie das Vorschaufenster, wenn Sie die Anzeige beenden möchten.
  6. Klicken Sie in der Symbolleiste auf Speichern oder Speichern und schließen.
    Die Vorlage wird in der Sage SalesLogix-Datenbank gespeichert. Wenn Sie Speichern und schließen ausgewählt haben, wird der Editor geschlossen. Das Dialogfeld Seriendruck wird mit dem Namen der aktuellen Vorlage im Feld Vorlage geöffnet.
  • Sage SalesLogix-Seriendruckfelder können nicht zum Feld Betreff einer E-Mail hinzugefügt werden.
  • Die Sage SalesLogix-Seriendruckfelder liegen im Format XML vor, daher muss die Groß-/Kleinschreibung berücksichtigt werden. Wenn die Groß-/Kleinschreibung eines Seriendruckfelds geändert wird, funktioniert das Zusammenführen der Daten nicht mehr.
  • Beim Erstellen oder Bearbeiten einer Vorlage dürfen keine Sage SalesLogix-Seriendruckfelder in einem Textfeld platziert werden.
  • Die Vorlage muss ein Seriendruckfeld aus der Adresstabelle enthalten, um während eines Seriendrucks das Dialogfeld Adresstyp auswählen anzeigen zu können. Weitere Informationen erhalten Sie unter "Auswählen des Adresstyps".
