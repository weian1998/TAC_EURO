Bearbeiten von SQL-Textfeldern

Mit dem Seriendruckfeld-Editor können Sie neue und vorhandene SQL-Textseriendruckfelder bearbeiten. Wenn Sie in der Listenansicht ein SQL-Textfeld auswählen, werden die Bearbeitungsoptionen im Bearbeitungsteilfenster angezeigt.

  • Feld-ID: Ermöglicht Ihnen, den Text zu ändern, der in der Vorlage angezeigt wird.
  • SQL: Ermöglicht Ihnen, eine SELECT-Anweisung einzugeben, um Daten vom OLE-DB-Provider abzurufen. Nur der erste Datensatz der ersten Spalte wird angezeigt.
    Klicken Sie auf die Schaltfläche "Bearbeiten", um das Dialogfeld SQL bearbeiten zu öffnen.
  • Formatieren als: Zeigt den Seriendruckfeldtyp an.
  • Formattyp: Ermöglicht Ihnen, die Art der Formatierung zu ändern, die auf das ausgewählte Seriendruckfeld
  • angewendet wird.
  • Formatzeichenfolge: Ermöglicht Ihnen, den im Seriendruckfeld angezeigten Wert zu formatieren. Sie müssen den Formattyp festlegen, bevor Sie eine Formatzeichenfolge hinzufügen. Welche Zeichenfolgenformate gültig sind, hängt vom ausgewählten Formattyp ab.
  • Text vor: Ermöglicht Ihnen, Text einzugeben, der vor dem ausgewählten Seriendruckfeld angezeigt wird.
  • Text nach: Ermöglicht Ihnen, Text einzugeben, der nach dem ausgewählten Seriendruckfeld angezeigt wird.
  • Spezielles Format: Ermöglicht Ihnen, eine spezielle Zeichenformatierung auf das Seriendruckfeld anzuwenden (dies umfasst Text, der vor und nach einem Seriendruckfeld angezeigt wird).
  • Hyperlink-Adresse: Ermöglicht Ihnen, einen URL einzugeben, der vom Seriendruckfeld geöffnet werden soll.
    Beispiele für Hyperlink-Adressen
    • http://www.saleslogix.com/
    • mailto:jabbott@abbott.demo?Subject=Sonderangebot&Body=Ja, schicken Sie mir bitte Informationen zu diesem Sonderangebot zu!
    • mailto::SLXUserName <:SLXUserEmail>?Subject=Sonderangebot&Body=Ja, schicken Sie mir bitte Informationen zu diesem Sonderangebot zu!
    • ftp://ftp.winzip.com/winzip81.exe
    • news://msnews.microsoft.com/microsoft.public.word.mailmerge.fields
  • Hyperlink-Ziel: Ermöglicht Ihnen auszuwählen, wie der URL geöffnet werden soll. Wählen Sie in der Dropdown-Liste ein Ziel aus.
    • Zieloption Beschreibung
    Seitenstandard (ohne) Öffnet den Link im selben Fenster.
    Gleicher Frame

    Öffnet den Link im selben Frame.

    Dies entspricht dem HTML-Ziel "_self".
    Ganze Seite

    Öffnet den Link im gesamten Hauptteil des aktuellen Fensters.

    Dies entspricht dem HTML-Ziel "_top".
    Neues Fenster

    Öffnet den Link in einem neuen Fenster ohne Titel.

    Dies entspricht dem HTML-Ziel "_blank".
    Übergeordneter Frame

    Öffnet den Link in dem Fenster, das dem Dokument mit dem Link unmittelbar übergeordnet ist.

    Dies entspricht dem HTML-Ziel "_parent".
  • Hyperlink-Bildschirmtipp: Ermöglicht die Eingabe von Text, der angezeigt wird, wenn der Mauszeiger über dem Hyperlink platziert wird.
    Parameter können in Hyperlink-Adresse und Hyperlink-Bildschirmtipp eingefügt werden und beim Erstellen eines Hyperlinks vom Typ mailto: hilfreich sein.
    • Sage SalesLogix-Parameter Beschreibung
    SLXAccountManagerEmail Der Parameter ":SLXAccountManagerEmail" wird durch die E-Mail-Adresse des Account Managers für den Kontaktdatensatz ersetzt, für den der Seriendruck erstellt wird.
    SLXAccountManagerName Der Parameter ":SLXAccountManagerName" wird durch den Namen des Account Managers für den Kontaktdatensatz ersetzt, für den der Seriendruck erstellt wird.
    SLXAccountManagerTitle Der Parameter ":SLXAccountManagerTitle" wird durch die Anrede des Account Managers für den Kontaktdatensatz ersetzt, für den der Seriendruck erstellt wird.
    SLXAccountManagerPhone Der Parameter ":SLXAccountManagerPhone" wird durch die Telefonnummer des Account Managers für den Kontaktdatensatz ersetzt, für den der Seriendruck erstellt wird.
    SLXAccountManagerFax Der Parameter ":SLXAccountManagerFax" wird durch die Faxnummer des Account Managers für den Kontaktdatensatz ersetzt, für den der Seriendruck erstellt wird.
    SLXUserEmail Der Parameter ":SLXUserEmail" wird durch die E-Mail-Adresse des Seriendruckbenutzers ersetzt.
    SLXUserName Der Parameter ":SLXUserName" wird durch den Namen des Seriendruckbenutzers ersetzt.
    SLXUserTitle Der Parameter ":SLXUserTitle" wird durch die Anrede des Seriendruckbenutzers ersetzt.
    SLXUserPhone Der Parameter ":SLXUserPhone" wird durch die Telefonnummer des Seriendruckbenutzers ersetzt.
    SLXUserFax Der Parameter ":SLXUserFax" wird durch die Faxnummer des Seriendruckbenutzers ersetzt.
