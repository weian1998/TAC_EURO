Auswählen eines Sage SalesLogix-Benutzers

Das Dialogfeld Benutzer auswählen wird in folgenden Fällen verwendet:

  • Wenn Sie eine Vorlage erstellen oder kopieren und einen Zugriffsberechtigten für die Vorlage auswählen müssen
  • Wenn Sie Historienoptionen für den Seriendruck festlegen und einen Organisator für eine Folgeaktivität auswählen müssen

Vorgehensweise

Auswählen eines Zugriffsberechtigten für eine Vorlage
  1. Erstellen oder kopieren Sie eine Vorlage.
  2. Klicken Sie im Dialogfeld Vorlageneigenschaften im Feld Zugriffsberechtigter auf Suchen.
  3. Führen Sie im Dialogfeld Benutzer auswählen einen der folgenden Schritte aus, um einen Zugriffsberechtigten auszuwählen:
    • Wählen Sie einen Benutzer aus der Liste aus und klicken Sie dann auf OK.
    • Doppelklicken Sie auf den Namen eines Benutzers.
Auswählen eines Organisators für eine Folgeaktivität
  1. Klicken Sie im Menü Verfassen auf Seriendruck.
  2. Wählen Sie eine Vorlage für den Seriendruck aus.
  3. Geben Sie auf der Registerkarte Historienoptionen im Bereich Folgeaktivität planen die erforderlichen Daten zum Planen der Folgeaktivität ein.
  4. Klicken Sie im Feld Organisator auf Suchen.
  5. Führen Sie im Dialogfeld Benutzer auswählen einen der folgenden Schritte aus, um einen Zugriffsberechtigten auszuwählen:
    • Wählen Sie einen Benutzer aus der Liste aus und klicken Sie dann auf OK.
    • Doppelklicken Sie auf den Namen eines Benutzers.
