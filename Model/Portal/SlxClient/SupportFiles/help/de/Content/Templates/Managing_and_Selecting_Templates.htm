Verwalten und Auswählen von Vorlagen

Über die Dialogfelder Vorlagen verwalten oder Vorlage auswählen können Sie auf die für Sie verfügbaren öffentlichen und privaten Vorlagen zugreifen.

Wählen Sie beim Auswählen von Vorlagen für Interessenten nur Vorlagen aus, die mit "(Interessent)" enden.

Vorgehensweise

Öffnen Sie das Dialogfeld "Vorlagen verwalten" oder "Vorlage auswählen".
  • Klicken Sie im Menü Verfassen auf Vorlagen.

Ändern des Vorlagentyps einer privaten Vorlage
  1. Klicken Sie in der Liste Private Vorlagen auf einen der Ordner Brief, Fax oder E-Mail, um die Liste der Vorlagen zu erweitern.
  2. Klicken Sie mit der rechten Maustaste auf die Vorlage, die Sie ändern möchten. Zeigen Sie im Kontextmenü mit der Maus auf Vorlagentyp und klicken Sie danach auf einen der Typen.
    Die Standardtypen sind: Brief, Fax, E-Mail.

Anzeigen einer Vorlage in der Vorschau

  1. Wählen Sie eine Vorlage in der Liste aus und klicken Sie auf Vorschau anzeigen.
    Nun wird ein zweites Fenster geöffnet, in dem der Text des Betreffs, Informationen zur Anlage sowie der Textkörper der Vorlage angezeigt werden.
  2. Wenn eine Vorschau der Vorlage mit zusammengeführten Daten angezeigt werden soll, wählen Sie das Kontrollkästchen Vorlage mit zusammengeführten Daten anzeigen aus.
  3. Klicken Sie zum Ausblenden der Vorschau auf Vorschau ausblenden.

Klicken Sie mit der rechten Maustaste auf den Namen einer Vorlage, um Zugriff auf die Kontextmenüoptionen Alles ausblenden, Alles erweitern, Neu, Bearbeiten, Eigenschaften, Kopieren, Löschen und Vorlagentyp zu erhalten.

Hinzufügen eines neuen Vorlagenordners
  • Der Systemadministrator kann einen benutzerdefinierte Vorlagenordner hinzufügen, indem er einen neuen Eintrag zur Auswahlliste Vorlagentypen hinzufügt. Weitere Informationen hierzu erhalten Sie von Ihrem Systemadministrator.
  • Systemadministratoren finden eine detaillierte Beschreibung zum Hinzufügen von Auswahllisteneinträgen entweder unter dem Thema "Hinzufügen eines Eintrags zu einer Auswahlliste" in der Administrator-Hilfe oder im Sage SalesLogix Web Client unter dem Thema "Registerkarte "Auftragseinträge"".
