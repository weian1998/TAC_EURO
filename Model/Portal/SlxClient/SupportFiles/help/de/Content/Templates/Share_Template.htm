Gemeinsames Verwenden von Vorlagen mit anderen Benutzern

Sie können Ihre Vorlagen für andere Sage SalesLogix-Benutzer, -Teams oder -Abteilungen freigeben.

Diese Funktion steht nur im Sage SalesLogix Web-Client zur Verfügung.

Vorgehensweise

Freigeben einer Vorlage
  1. Klicken Sie im Menü Verfassen auf Vorlagen.
  2. Öffnen Sie das Dialogfeld Vorlagen verwalten oder Vorlage auswählen.
    • Sage SalesLogix Network- und SalesLogix Web-Benutzer: Klicken Sie im Menü Verfassen auf Vorlagen.
    • Architekt-Benutzer: Wählen Sie im Menü Verwalten den Eintrag Vorlagen aus.
  3. Wählen Sie im Dialogfeld Vorlagen verwalten die freizugebende Vorlage in der Liste aus.
  4. Klicken Sie auf Freigeben.
  5. Klicken Sie auf die Schaltfläche Hinzufügen.

  6. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie auf die Registerkarte Benutzer. Wenn Sie die aufgelisteten Benutzer begrenzen möchten, wählen Sie in der Dropdown-Liste Benutzertyp einen Benutzertyp aus (z. B. Remote).
    • Klicken Sie auf die Registerkarte Abteilungen.
    • Klicken Sie auf die Registerkarte Teams. Sie können die angezeigten Teams eingrenzen, indem Sie das Kontrollkästchen Eigene Teams aktivieren.

    • Wenn Sie als Administrator angemeldet sind, ist das Kontrollkästchen Eigene Teams nicht verfügbar.
  7. Treffen Sie eine Auswahl in der Liste und klicken Sie auf OK.
Entfernen des Zugriffs auf eine freigegebene Vorlage
  1. Klicken Sie im Menü Verfassen auf Vorlagen.
  2. Wählen Sie im Dialogfeld Vorlagen verwalten die Vorlage, auf die der Zugriff verwehrt werden soll, in der Liste aus.
  3. Klicken Sie auf Freigeben.
  4. Wählen Sie Benutzer, Abteilungen oder Teams aus, für die Ihre Vorlage nicht mehr freigegeben sein soll.
  5. Klicken Sie auf die Schaltfläche Entfernen und danach auf OK.
Aktualisieren einer freigegebenen Vorlage

Wenn Sie Änderungen an einer Vorlage vornehmen und diese Änderungen freigeben möchten, müssen Sie die Vorlage erneut freigeben.

  1. Klicken Sie im Menü Verfassen auf Vorlagen.
  2. Wählen Sie im Dialogfeld Vorlagen verwalten die freizugebende Vorlage in der Liste aus.
  3. Klicken Sie auf Freigeben.
  4. Klicken Sie im Dialogfeld Vorlage freigeben auf OK, um die Änderungen für die aufgelisteten Benutzer freizugeben.
