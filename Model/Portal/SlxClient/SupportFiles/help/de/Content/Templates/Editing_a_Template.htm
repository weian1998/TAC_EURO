Bearbeiten einer Vorlage

Sie können private oder öffentliche Vorlagen bearbeiten; wenn Sie eine öffentliche Vorlage bearbeiten möchten, müssen Sie die Vorlage erst kopieren und sie als private Vorlage speichern.

Wenn Sie die Vorlage während des Seriendrucks bearbeiten, müssen Sie im Anschluss daran auf Speichern und schließen klicken, um den Editor zu schließen, damit der Seriendruck fortgesetzt werden kann.

So bearbeiten Sie eine Vorlage:

  1. Klicken Sie im Menü Verfassen auf Vorlagen.
  2. Wählen Sie im Dialogfeld Vorlagen verwalten eine Vorlage aus der Liste Private Vorlagen oder der Liste Öffentliche Vorlagen aus und klicken Sie dann auf Bearbeiten.
  3. Wenn Sie eine öffentliche Vorlage bearbeiten möchten, klicken Sie in der Warnmeldung auf Ja, um die Vorlage zu kopieren.
  4. Führen Sie im Dialogfeld Eigenschaften einen der folgenden Schritte aus:
    • Legen Sie die Eigenschaften der Vorlage fest und klicken Sie dann auf OK.
    • Lassen Sie die Eigenschaften wie angegeben und klicken Sie danach auf Schließen.
      Im Editor wird die Vorlage mit Standardtexten und Seriendruckfeldern angezeigt.
  5. Um Änderungen an der Vorlage vorzunehmen, können Sie nun weiteren Text in die Vorlage eingeben oder die Sage SalesLogix-Symbolleiste verwenden, um Folgendes zu tun:
    • Sage SalesLogix-Seriendruckfelder in die Vorlage einfügen
    • Vorlage in der Vorschau betrachten
    • Dateien an die Vorlage anhängen, wenn es sich um eine Vorlage vom Typ "E-Mail" handelt.
  6. Nach Abschluss der Bearbeitung klicken Sie in der Sage SalesLogix-Symbolleiste auf Speichern und schließen.
