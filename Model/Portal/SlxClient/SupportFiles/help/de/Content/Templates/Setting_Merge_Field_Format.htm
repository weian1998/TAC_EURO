Festlegen des Formats Seriendruckfeldern

Im Dialogfeld SalesLogix-Feld formatieren können Sie den Formattyp und die Formatzeichenfolge eines Sage SalesLogix-Seriendruckfelds festlegen.

Das Standard-Feldformat für die meisten Textfelder ist Kein. Wenn das Feld jedoch numerische Informationen enthält (Telefonnummer, Währungsangabe, Wahr-/Falschwert oder andere Daten), ändern Sie das Feldformat entsprechend. Aus der Liste Feldformat können Sie ggf. eine geeignete Option für die verwendeten Daten auswählen.

Beispiele anzeigen
  • Wenn Sie "Datum" als Formattyp wählen, dann würden Sie die Formatzeichenfolge als "mm/tt/jjjj" eingeben.
  • Mit dem Formattyp "Benutzer" wird eine Benutzer-ID standardmäßig als "Nachname, Vorname" formatiert, und wenn Sie die Formatzeichenfolge "FL" verwenden, wird die Benutzer-ID als "Vorname Nachname" formatiert.

So legen Sie das Format eines Seriendruckfelds fest:

  1. Wählen Sie im Dialogfeld SalesLogix-Feld formatieren einen Typ aus der Liste Formattyp aus.
  2. Geben Sie im Feld Formatzeichenfolge eine für den Formattyp geeignete Zeichenfolge ein.
  3. Klicken Sie auf OK.
