Auswählen von Importgruppenaktionen

In dieser Ansicht legen Sie Aktionen fest, die für die Interessenten durchgeführt werden sollen, die Sie in Sage SalesLogix importieren. Sie können eine Notiz hinzufügen, den Datensatz zu einer Kampagne hinzufügen, eine Antwort erstellen oder eine Aktivität planen.

So gehen Sie beim Auswählen vor

  1. Beim Importieren von Interessenten klicken Sie nach dem Verwalten doppelter Datensätze auf Weiter.
  2. Um eine Notiz hinzuzufügen, klicken Sie neben Notiz hinzufügen auf Definieren und füllen die Notizenfelder zu importierten Interessenten aus. Aktivieren Sie das Kontrollkästchen Aktiv.
  3. Um eine Antwort hinzuzufügen, die nicht mit einer Kampagne verknüpft ist, klicken Sie neben Antwort hinzufügen auf Definieren und füllen die Importantwortfelder aus. Aktivieren Sie das Kontrollkästchen Aktiv.
  4. Um eine Antwort für ein Kampagnenziel hinzuzufügen, klicken Sie neben Kampagnenziel hinzufügen auf Definieren und wählen eine Kampagne aus. Aktivieren Sie das Kontrollkästchen Aktiv.
  5. Um einen Anruf zu planen, klicken Sie auf Anruf planen und füllen die Felder für die Interessentenimportaktivität aus. Aktivieren Sie das Kontrollkästchen Aktiv.
  6. Um ein Meeting zu planen, klicken Sie auf Meeting planen und füllen die Felder für die Interessentenimportaktivität aus. Aktivieren Sie das Kontrollkästchen Aktiv.
  7. Um eine Aufgabe zu planen, klicken Sie auf Aufgabe planen und füllen die Felder für die Interessentenimportaktivität aus. Aktivieren Sie das Kontrollkästchen Aktiv.
  8. Klicken Sie auf Weiter, um zum nächsten Schritt zu gelangen.
