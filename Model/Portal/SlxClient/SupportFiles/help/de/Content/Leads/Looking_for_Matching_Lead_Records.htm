Suchen nach übereinstimmenden Interessentendatensätzen

Wenn Sie einen Interessenten hinzufügen, können Sie den Interessenten mit anderen Interessenten, Kontakten oder Firmen in der Datenbank vergleichen. Wenn in der Datenbank ein Datensatz mit identischen Informationen enthalten ist, können Sie dadurch doppelte Datensätze vermeiden.

Sage SalesLogix kann nur nach doppelten Datensätzen suchen, zu deren Anzeige Sie berechtigt sind. Es können dann trotzdem noch doppelte Datensätze existieren, auf die Sie nicht zugreifen dürfen.

So suchen Sie nach übereinstimmenden Datensätzen

  1. Geben Sie im Dialogfeld Interessenten einfügen Interessenteninformationen ein, und klicken Sie danach auf die Schaltfläche Nach übereinstimmenden Datensätzen suchen.
    Das Dialogfeld Duplikate suchen wird geöffnet.
  2. Wählen Sie neben Typ die zu suchenden Datensatztypen aus.
  3. Aktivieren Sie unter Mit Filtern abgleichen die Kontrollkästchen der Filter, die Sie anwenden möchten.
  4. Wählen Sie eine der folgenden Optionen aus:
    • Mit allen ausgewählten Filtern abgleichen: Ein Datensatz ist nur ein Duplikat, wenn er allen ausgewählten Filtern entspricht.
    • Mit ausgewählten Filtern abgleichen: Ein Datensatz ist ein Duplikat, wenn er einem der ausgewählten Filter entspricht.
  5. Um eine erweiterte Suche durchzuführen, klicken Sie auf Erweiterte Optionen.
  6. Klicken Sie auf Potenzielle Übereinstimmungen aktualisieren.
  7. Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie neben einem potenziell übereinstimmenden Datensatz auf Öffnen, um zur Detailansicht für diesen Datensatz zu wechseln.
    • Klicken Sie im Raster Überstimmende Firmendatensätze auf Kontakt hinzufügen, um den Interessenten in einen Kontakt zu konvertieren und diesen zu einer vorhandenen Firma hinzuzufügen.
    • Klicken Sie auf Abbrechen, um zur Ansicht Interessent einfügen zurückzukehren und das Hinzufügen des Interessenten fortzusetzen.
