Bearbeiten eines Interessenten

In der Ansicht Interessentendetail können Sie Interessenteninformationen bearbeiten.

So bearbeiten Sie Informationen

  1. Klicken Sie auf der Navigationsleiste auf Interessenten.
  2. Suchen Sie den Interessenten mithilfe der Suchfunktion.
  3. Bearbeiten Sie die Interessenteninformationen.
  4. Klicken Sie auf Speichern.
