Verwenden der Interessentenlistenansicht

In der Interessentenlistenansicht werden alle Interessenten angezeigt, auf die Sie Zugriff haben. Mithilfe von Gruppen und Suchen können Sie die Liste der angezeigten Datensätze begrenzen.

Durch Ihren Sicherheitszugriff wird bestimmt, welche Funktionen verfügbar sind. Wenden Sie sich an den Administrator, wenn Sie Ihre Zugriffsrechte ändern möchten. Für WebViewer-Benutzer kann die Funktionalität eingeschränkt sein.

Vorgehensweise

Öffnen der Interessentenlistenansicht
  • Klicken Sie auf der Navigationsleiste auf Interessenten.
Aktualisieren von Listeninformationen
  • Klicken Sie auf die Schaltfläche Aktualisieren.
