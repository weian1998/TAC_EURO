Interessentenimporthistorie – Registerkarte "Duplikate"

Auf dieser Registerkarte können Sie eine Liste mit Datensätzen anzeigen, die als Duplikate gekennzeichnet wurden, die vor dem Import aufgelöst werden müssen.

So gehen Sie zum Öffnen vor

  • Klicken Sie in der Detailansicht Interessentenimporthistorie auf die Registerkarte Duplikate. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.

So lösen Sie ein Duplikat auf

  • Klicken Sie neben dem aufzulösenden Datensatz auf Auflösen.

Das Dialogfeld Doppelten Importinteressenten auflösen wird geöffnet.
