Speichern einer Importvorlage

Sie können Importdaten, beispielsweise übereinstimmende Felder, den standardmäßigen Zugriffsberechtigten für einen Interessenten und die standardmäßige Interessentenquelle, speichern und für künftige Importe verwenden.

So speichern Sie eine Importvorlage

  1. Klicken Sie nach dem Definieren der Importfeldzuordnung auf Als Importvorlage speichern.
    Der Dialog Interessent importieren - Vorlage wird angezeigt.
  2. Geben Sie im Feld Beschreibung eine kurze Beschreibung der Importvorlage ein, und klicken Sie auf Als Importvorlage speichern.
    Die Beschreibung der neuen Importvorlage wird in die Liste Importvorlage im Dialog Interessenten importieren eingefügt.
