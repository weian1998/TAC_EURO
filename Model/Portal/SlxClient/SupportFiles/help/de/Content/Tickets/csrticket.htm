Hinzufügen eines neuen Tickets

Über den Dialog Neues Ticket hinzufügen können Sie ein neues Ticket für eine neue Anfrage bzw. ein neues Problem einer Firma hinzufügen. Damit Sie das Ticket hinzufügen können, müssen die Firma und der Kontakt in Sage SalesLogix bereits vorhanden sein.

Durch Ihren Sicherheitszugriff wird bestimmt, welche Funktionen verfügbar sind. Wenden Sie sich an den Administrator, wenn Sie Ihre Zugriffsrechte ändern möchten. Für WebViewer-Benutzer kann die Funktionalität eingeschränkt sein.

So fügen Sie ein Ticket hinzu

  1. Öffnen Sie den Dialog Neues Ticket hinzufügen.
  • Klicken Sie im Menü Neu auf Ticket.
  • Klicken Sie in der Ansicht Kontaktdetail oder der Ansicht Firmendetail auf die Registerkarte Tickets und klicken Sie dann auf die Schaltfläche Ticket erstellen.
  1. Wenn das Dialogfeld Firmenserviceinformationen automatisch angezeigt wird oder wenn Sie auf die Schaltfläche Firmenserviceinformationen geklickt haben, gehen Sie wie folgt vor:
  2. Wählen Sie im Bereich Kontakte den Kontakt aus, der das Problem übermittelt hat.

Wenn der Kontakt zum Anfordern des Service berechtigt ist, wird in der Spalte Autorisiert "Ja" angezeigt.

  1. Wählen Sie im Bereich Verträge den Vertrag aus, mit dem das Ticket verrechnet wird.

Wenn kein Vertrag angezeigt wird, können Sie dennoch ein Ticket hinzufügen. Mit SalesLogix-Client oder SalesLogix Support können Sie einen Vertrag für die Firma hinzufügen.

  1. Wenn Sie die auf eine Aktivität verwendete Zeit aufzeichnen möchten, klicken Sie in der Symbolleiste auf die Schaltfläche Zeitmessung für Ticket aktivieren, um die Zeitaufzeichnung für Rechnungszwecke zu starten.

Wenn Sie die Kundendienstoption für die automatische Aktivierung der Zeitmessung festgelegt haben, müssen Sie nicht auf die Schaltfläche Zeitmessung für Ticket aktivieren klicken.
  1. Füllen Sie die erforderlichen Ticket-Informationsfelder aus.
  2. Schließen Sie die Detailinformationen ab.
  • Geben Sie in das Feld Betreff eine Kurzzusammenfassung des Problems ein.
  • Klicken Sie auf die Schaltfläche Benutzerdatumsstempel neben dem Feld Beschreibung, um Ihren Namen sowie das Datum und die Uhrzeit einzugeben, und geben Sie anschließend eine vollständige Erläuterung des Problems ein. Klicken Sie zum Suchen nach einem häufig auftretenden Problem auf die Schaltfläche Suchen, und verwenden Sie anschließend die Suchfunktion.

Sie können auch auf die Schaltfläche SpeedSearch klicken, um zu prüfen, ob das Problem bereits gemeldet wurde.

  • Wenn das Ticket gelöst wurde, klicken Sie auf die Schaltfläche Benutzerdatumsstempel neben dem Feld Lösung, um Ihren Namen sowie das Datum und die Uhrzeit einzugeben, und geben Sie anschließend eine vollständige Erläuterung dazu ein, wie das Problems gelöst wurde. Klicken Sie zum Suchen nach einem häufig auftretenden Problem auf die Schaltfläche Suchen, und verwenden Sie anschließend die Suchfunktion.

Sie können auch auf die Schaltfläche SpeedSearch klicken, um zu prüfen, ob das Problem bereits gemeldet wurde.

  1. Klicken Sie auf die Schaltfläche Benutzerdatumsstempel neben dem Feld Intern, um Ihren Namen sowie das Datum und die Uhrzeit (jeweilige Ländereinstellungen) ihrer Kommentare einzugeben, und geben Sie Ihre Kommentare ein.
  2. Wenn alle erforderlichen Informationen ausgefüllt sind, klicken Sie auf die Schaltfläche Speichern.
