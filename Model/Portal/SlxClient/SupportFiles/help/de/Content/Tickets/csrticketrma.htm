Verwenden der Registerkarte "Retouren"

Auf der Registerkarte Retouren in der Firmen- und Ticket-Detailansicht finden Sie alle Retouren für den ausgewählten Datensatz.

So gehen Sie zum Öffnen vor

  1. Öffnen Sie die Ansicht Firmendetail oder Ticket-Detail.
  2. Klicken Sie auf die Registerkarte Retouren. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.

So fügen Sie eine Retoure hinzu (nur Ticketdetailansicht)

  1. Klicken Sie auf Neue Retoure einfügen.
  2. Folgen Sie den Schritten zum Hinzufügen einer Retoure.

So zeigen Sie weitere Informationen zu einer Retoure an

  • Klicken Sie auf die Retouren-ID, um die Ansicht Retourendetail zu öffnen.
