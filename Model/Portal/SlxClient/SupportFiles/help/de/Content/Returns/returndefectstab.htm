Verwenden der Registerkarte "Fehler"

Auf der Registerkarte Fehler in der Ticket- und Retourendetailansicht können Sie alle zugehörigen Fehler anzeigen. Außerdem können Sie in dieser Registerkarte eine Verknüpfung mit einem Fehler hinzufügen oder entfernen.

Die Registerkarte enthält schreibgeschützte Informationen zu Fehler-ID, Gemeldet am, Priorität, Schweregrad, Status, Lösung und Beschreibung.

So gehen Sie zum Öffnen vor

  1. Öffnen Sie die Detailansicht Ticket oder Retoure.
  2. Klicken Sie auf die Registerkarte Fehler. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.

So verknüpfen Sie einen Fehler

  1. Klicken Sie auf Fehler verknüpfen.
  2. Suchen Sie den Eintrag über die Suche und wählen Sie ihn aus.
  3. Klicken Sie auf OK.

So heben Sie die Verknüpfung eines Fehlers mit einer Retoure auf

  1. Wählen Sie den Datensatz aus, den Sie entfernen möchten, und klicken Sie auf die Schaltfläche Entfernen.

  2. Klicken Sie auf Ja.

So fügen Sie einen neuen Fehler hinzu

  1. Klicken Sie nur in der Detailansicht Ticket auf Fehler anlegen.
  2. Folgen Sie den Schritten zum Hinzufügen eines Fehlers.
