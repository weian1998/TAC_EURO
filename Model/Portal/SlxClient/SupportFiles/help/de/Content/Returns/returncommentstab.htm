Verwenden der Registerkarte "Kommentare" für Retouren

Verwenden Sie diese Registerkarte für alle internen Notizen zu einer Retoure.

So fügen Sie Kommentare hinzu

  1. Öffnen Sie die Ansicht Retourendetail.
  2. Klicken Sie auf die Registerkarte Kommentare. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  3. Geben Sie die Informationen in das Feld Kommentare ein.
