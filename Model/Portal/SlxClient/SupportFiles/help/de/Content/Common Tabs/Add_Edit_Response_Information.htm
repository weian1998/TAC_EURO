Hinzufügen oder Bearbeiten von Antwortinformationen

In diesem Dialogfeld können Sie Antwortinformationen aktualisieren. Mithilfe von Antworten werden Interaktionen mit Kontakten oder Interessenten nachverfolgt. Die Antwort wird auf der Registerkarte Antworten der Detailansichten angezeigt.

Vorgehensweise

Hinzufügen einer Antwort
  1. Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie in der Detailansicht Kontakt, Interessent oder Firma auf die Registerkarte Antworten oder Marketing. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
    • Klicken Sie in der Ansicht Kampagnendetail auf die Registerkarte Antworten. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  2. Klicken Sie auf die Schaltfläche Antwort hinzufügen oder Kampagnenantwort hinzufügen.
  3. Wählen Sie die Option Kontakt oder Interessent aus.
  4. Abhängig von Ihrer Auswahl im vorherigen Schritt, klicken Sie auf die Schaltfläche Suchen neben dem Kontakt oder Interessent und suchen den entsprechenden Datensatz mithilfe der Suchfunktion.
  5. Klicken Sie bei Bedarf auf die Schaltfläche Kampagne und Schaltfläche Suchen, um nach einer Kampagne zu suchen. Ihre Auswahl wird zum Dialogfeld hinzugefügt.
  6. Wenn Sie eine Kampagne angegeben haben, klicken Sie auf zu Phase gehörenden den Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus.
  7. Klicken Sie unter Interessentenquelle auf die Schaltfläche Suchen, um eine Interessentenquelle zu suchen. Ihre Auswahl wird zum Dialogfeld hinzugefügt.
  8. Klicken Sie neben Status auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  9. Klicken Sie neben Antwortdatum auf Kalender-Schaltfläche, um das Datum auszuwählen.  .
  10. Klicken Sie neben Antwortmethode auf die Dropdown-Pfeil. Wählen Sie einen Eintrag aus der Liste aus.
  11. Klicken Sie neben Interesse auf die Dropdown-Pfeil, um eine Beschreibung für die Art der Kontakt- oder Interessentenantwort auszuwählen.
  12. Klicken Sie neben Grad des Interesses auf die Dropdown-Pfeil, um anzugeben, wie groß das Interesse des Kontakts oder Interessenten ist.
  13. Geben Sie in das Feld KommentareAnmerkungen ein.
  14. Auf der Registerkarte Produkte können Sie mit der Antwort verknüpfte Produkte anzeigen. Sie können auch auch auf Produkt hinzufügen klicken, um ein Produkt zu suchen und hinzuzufügen.
  15. Klicken Sie auf OK.
Bearbeiten von Antworten
  1. Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie in der Detailansicht Kontakt, Interessent oder Firma auf die Registerkarte Antworten oder Marketing. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
    • Klicken Sie in der Ansicht Kampagnendetail auf die Registerkarte Antworten. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  2. Klicken Sie neben dem zu bearbeitenden Datensatz auf den Hyperlink Bearbeiten.
  3. Aktualisieren Sie die Antwortinformationen.
  4. Im Bereich Produkte können Sie mit der Antwort verknüpfte Produkte anzeigen. Sie können auch auch auf Produkt hinzufügen klicken, um ein Produkt zu suchen und hinzuzufügen.
  5. Klicken Sie auf OK.
