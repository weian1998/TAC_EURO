Hinzufügen/Bearbeiten einer alternativen Adresse

Sie können das Dialogfeld Adresse verwenden, um Adressinformationen zu ändern. Sie können für einen Kontakt oder eine Firma beliebig viele Kundenadressen eingeben. Sie können jedoch nur eine einzige Hauptrechnungs- und eine einzige Hauptlieferadresse auswählen.

So öffnen Sie das Dialogfeld Adresse

  • Führen Sie einen der folgenden Schritte durch:
  • Klicken Sie in der Detailansicht Firma, Kontakt oder Interessent auf Adressen.Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten und wählen die Registerkarte in der Liste aus. Klicken Sie auf den Hyperlink Bearbeiten der Adresse, die Sie ändern möchten, oder klicken Sie auf die Schaltfläche Adresse hinzufügen, um eine neue Adresse hinzuzufügen.
  • Klicken Sie auf die Schaltfläche Adressebearbeiten in der Ansicht Kontakt-/Firmeninformationen oder Interessent einfügen.
  • Klicken Sie auf die Schaltfläche Adressebearbeiten in der Detailansicht Kontakt, Firma oder Interessent.
  • Klicken Sie auf die Schaltfläche Adressebearbeiten auf der Registerkarte Rechnungs-/Lieferadresse in der Detailansicht Verkaufsauftrag.

So gehen Sie zum Hinzufügen oder Bearbeiten vor

  1. Klicken Sie im Feld Beschreibung auf die Dropdown-Pfeil, wählen Sie eine Beschreibung für die Adresse aus und klicken Sie danach auf OK.
  2. Aktivieren bzw. deaktivieren Sie die Kontrollkästchen Haupt,Hauptrechnungsadresse oder Hauptlieferadresse wie erforderlich.
  3. Geben Sie die Adressinformationen in die Adressfelder ein bzw. bearbeiten Sie die Angaben.
  4. Klicken Sie in den Feldern Stadt, Bundesland/Kanton oder Land auf die Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus. Klicken Sie anschließend auf OK.
  5. Klicken Sie in das Feld Postleitzahl, und geben Sie etwaige Änderungen ein.
  6. Klicken Sie in das Feld Bundesland/Kanton, und geben Sie die Änderungen ein.
  7. Klicken Sie in das Feld Zu Händen von, und geben Sie die Grußformel ein, die Sie für schriftliche Korrespondenz verwenden möchten.
  8. Klicken Sie auf OK.
