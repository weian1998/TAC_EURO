Registerkarte "Notizen/Historie"

Die Registerkarte Notizen/Historie wird in den Detailansichten Firma, Kontakt, Verkaufschance und Interessent angezeigt. Die Registerkarte enthält Filteroptionen, über die Sie rasch auf wichtige Informationen zugreifen können. Über diese Registerkarte können Sie auch einen Historiendatensatz per E-Mail senden, ein Microsoft Word-Dokument erstellen, eine Anlage anzeigen oder Notizen bearbeiten.

Die Registerkarte Notizen/Historie enthält sämtliche Interaktionen einschließlich aller Textnotizen und stellt Filteroptionen zum Anzeigen, zum Drucken und zum Senden von Historieneinträgen und Notizen per E-Mail bereit.

Vorgehensweise

Öffnen der Registerkarte "Notizen/Historie"
  • Klicken Sie in der Detailansicht Kontakt, Firma, Verkaufschance oder Interessent auf die Registerkarte Notizen/Historie. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
Anwenden von Filtern
  1. Klicken Sie auf Filtern, um die Filteroptionen anzuzeigen.

Sie können mehr als eine Filteroption auswählen. Beispielsweise können Sie die Optionen Eigene Historie und Datumsbereich aktivieren, um alle von Ihnen angelegten Historieneinträge innerhalb eines bestimmten Datumsbereichs anzuzeigen.

  1. Wählen Sie einen oder mehrere der folgenden Filter aus:
    • Wählen Sie zum Filtern der Liste anhand des Datensatztyps in der Liste Typ den Datensatztyp aus, den Sie anzeigen möchten.
    • Datenbankänderungen sind standardmäßig ausgeblendet. Wählen Sie zum Anzeigen von Datenbankänderungen die Option Datenbankänderungen anzeigen aus.
    • Wählen Sie mit Hilfe der Felder zu Datumsbereich einen Datumsbereich aus, in dem die Anlage zuletzt aktualisiert wurde.
    • Klicken Sie auf den Benutzerden Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus.
    • Geben Sie in das Feld Betreff eines oder mehrere Zeichen ein, die im Betreff enthalten sind. Je mehr Zeichen Sie eingeben, desto kürzer ist die Ergebnisliste.
  2. Klicken Sie auf Übernehmen.
  3. Klicken Sie zum Entfernen von Filtern auf Zurücksetzen.
    Wenn Sie die Filter ausblenden möchten, klicken Sie erneut auf die Schaltfläche Filtern.
Vorschau auf Notizen oder Historiendetails
  • Wenn Sie einen Datensatz in der Liste auswählen, werden die Details dieses Datensatzes rechts neben der Liste im Vorschaubereich angezeigt.
Öffnen eines Notizen- oder Historiendatensatzes
  • Klicken Sie für den Eintrag, den Sie anzeigen möchten, auf den Hyperlink Typ.
E-Mail-Historiendatensätze
  1. Wählen Sie die Datensätze aus, die Sie per E-Mail senden möchten.
  2. Klicken Sie auf Per E-Mail senden.
    Es wird eine E-Mail-Nachricht mit den Kontaktinformationen, dem Betreff und den Informationen zu den Historiendatensätzen geöffnet. Wenn Sie mehrere Datensätze auswählen, werden die Angaben aus jedem Datensatz aufgeführt.
  3. Klicken Sie auf Senden.
Senden von Historieneinträgen an Microsoft Word
  1. Wählen Sie die Datensätze aus, die Sie in ein Word-Dokument kopieren möchten.
  2. Klicken Sie auf An Word senden.
    Microsoft Word wird geöffnet und zeigt ein Dokument mit den Notiz- oder Historiendatensatzinformationen an.
  3. Sie können das Dokument drucken oder die Datei speichern.
Hinzufügen von Notizen
  • Klicken Sie auf der Registerkarte Notizen/Historie auf die Schaltfläche Notiz hinzufügen und führen Sie die Schritte zum Hinzufügen einer Notiz aus.
