Hinzufügen von Anlagen

Sie können Anlagen zu einem Datensatz hinzufügen. Alle Dateien werden im gemeinsamen Anlagenverzeichnis gespeichert und mit Remote-Datenbanken synchronisiert.

Standardmäßig dürfen als Anlagen hinzugefügte Dateien maximal 4 MB groß sein. Wenn Sie eine Datei oder eine Gruppe von Dateien mit einer Größe von mehr als 4 MB anfügen, wird ggf. eine Fehler- oder Warnmeldung angezeigt. Weitere Informationen hierzu erhalten Sie von Ihrem Systemadministrator.

Vorgehensweise

Anhängen einer Datei
  1. Öffnen Sie die Registerkarte Anlagen.
  2. Klicken Sie auf die Schaltfläche Hinzufügen.
  3. Wenn das Dialogfeld Datei anhängen angezeigt wird, klicken Sie auf Durchsuchen.
  4. Wechseln Sie zu der Datei, die Sie hinzufügen möchten, und klicken Sie auf Öffnen.
  5. Geben Sie im Feld Beschreibung eine Beschreibung der Anlage ein, wenn der Dateiname nicht aussagekräftig ist.
  6. Klicken Sie auf OK.
Anhängen einer Datei durch Ziehen und Ablegen

Für diese Funktion müssen Sie ggf. das Desktopintegrationsmodul installieren.
  1. Wechseln Sie zur Datei, die Sie hinzufügen möchten.
  2. Ziehen Sie die Datei, und legen Sie sie auf der gewünschten Datensatz-Detailansicht ab.

Der Dialog Anlagen hinzufügen wird geöffnet.

    Falls Sie aufgefordert werden, die Sage SalesLogix-Desktopintegration zu installieren, und Sie dies wünschen, klicken Sie auf Ja, und befolgen Sie die Anweisungen zum Installieren der Sage SalesLogix-Desktopintegration. Zum Abschließen der Installation werden Sie aufgefordert, alle Browserfenster und Microsoft Outlook zu schließen.
  1. Geben Sie im Feld Beschreibung eine Beschreibung für die Datei ein, wenn die Standardbeschreibung nicht verständlich ist.

  2. Klicken Sie auf OK.

Anhängen eines URL
  1. Öffnen Sie die Registerkarte Anlagen.
  2. Klicken Sie auf die Schaltfläche URL hinzufügen.
  3. Geben Sie im Feld URL die vollständige URL-Adresse ein.
  4. Geben Sie in das Feld Beschreibung eine Beschreibung ein.
  5. Klicken Sie auf OK.
    Beschreibung und URL sind erforderlich.
