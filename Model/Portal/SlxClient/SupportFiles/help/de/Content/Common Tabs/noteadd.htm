Hinzufügen einer Notiz

Sie verwenden Notizen, um das Ergebnis einer Aktivität zu dokumentieren und um Interaktionen mit einer Firma, einem Kontakt, einem Interessenten oder einer Verkaufschance festzuhalten. Die von Ihnen hinzugefügten Notizen werden auf den Registerkarten Notizen/Historie der Firmen-, Kontakt-, Interessenten- und Verkaufschancen-Detailansichten angezeigt.

Wenn Sie eine Notiz zu einer Verkaufschance erstellen und dann die Verknüpfung zwischen der Notiz und dem Kontakt und der Firma aufheben, wird die Notiz nicht an Remote-Benutzer gesendet.

So fügen Sie eine Notiz hinzu

  1. Klicken Sie im Menü Neu auf Notiz.
  2. Klicken Sie im Feld Betreff auf die Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  3. Geben Sie, falls nötig, im Feld Standort einen Standort ein.
  4. Ändern Sie bei Bedarf die Datumsangaben unter Abgeschlossen und Geplant. Zum Ändern des Datums klicken Sie auf Kalender-Schaltfläche, um das Datum auszuwählen.  
  5. Führen Sie einen der folgenden Schritte durch:
    • Aktivieren Sie Ohne Zeitangabe, wenn Sie für die Notiz keine Zeitdauer eingeben möchten.
    • Klicken Sie neben Dauer auf den den Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus.
  6. Wählen Sie Kontakt oder Interessent aus. Abhängig von Ihrer Auswahl, klicken Sie auf die Schaltfläche Suchen neben den entsprechenden Feldern (Name, Firma, Verkaufschance oder Ticket) und suchen mithilfe des Dialogfelds Suchenden entsprechenden Datensatz.

    Klicken Sie auf die Schaltfläche Entfernen neben dem entsprechenden Feld, um einen Interessenten, einen Kontakt, eine Firma, eine Verkaufschance oder ein Ticket zu entfernen.
  7. Klicken Sie im Feld Priorität auf die Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  8. Wählen Sie ggf. einen Organisatoraus, indem Sie auf die Schaltfläche Suchen klicken und die Suchfunktion verwenden.
  9. Geben Sie in das Feld Notizendie entsprechenden Informationen ein.
  10. Klicken Sie, falls nötig, auf die Registerkarte Anlagen, um ein Dokument oder einen URL hinzuzufügen.
  11. (Optional) Klicken Sie neben Folgeaktivität auf die Dropdown-Pfeil und wählen Sie den Typ der Folgeaktivität aus. Planen Sie eine Folgeaktivität.
  1. Verwenden Sie das Kontrollkästchen Notizen übertragen oder Anlagen übertragen, um Notizen oder Anlagen von dieser Notiz in die Folgeaktivität zu kopieren.
  2. Klicken Sie auf OK.
