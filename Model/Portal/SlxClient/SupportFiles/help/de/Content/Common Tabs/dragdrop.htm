Ziehen und Ablegen mit Drag & Drop

Mit der Drag & Drop-Funktion können Sie Teile der Benutzeroberfläche von Sage SalesLogix neu anordnen. Wenn Sie die Desktopintegration installiert haben, können Sie auch Anlagen oder Microsoft Outlook-E-Mails zu Sage SalesLogix hinzufügen.

Vorgehensweise

Neuanordnen von Elementen innerhalb der Benutzeroberfläche
  1. Bewegen Sie die Maus auf das Objekt, das Sie verschieben möchten. Wenn die Registerkarte verschoben werden kann, ändert sich die Form des Mauszeigers.
  2. Drücken Sie die Maustaste auf dem Objekt, halten Sie die Maustaste gedrückt und ziehen Sie das Objekt an die gewünschte Position.

Dies kann beispielsweise eine andere Position in einem Raster, auf der Registerkartenleiste, im Bereich Weitere Registerkarten oder im benutzerdefinierten mittleren Bereich sein.

  1. Lassen Sie die Maustaste los.
Anhängen einer Datei durch Ziehen und Ablegen

  • Wenn Sie Internet Explorer verwenden, müssen Sie das Desktopintegrationsmodul installieren, um diese Funktion nutzen zu können.
  • Das Ziehen und Ablegen ist auf der Registerkarte Anlagen für die Historie oder Aktivitäten nicht verfügbar.
  1. Wechseln Sie zur Datei, die Sie hinzufügen möchten.
  2. Ziehen Sie die Datei, und legen Sie sie auf der gewünschten Datensatz-Detailansicht ab.

Der Dialog Anlagen hinzufügen wird geöffnet.

    Falls Sie aufgefordert werden, die Sage SalesLogix-Desktopintegration zu installieren, und Sie dies wünschen, klicken Sie auf Ja, und befolgen Sie die Anweisungen zum Installieren der Sage SalesLogix-Desktopintegration. Zum Abschließen der Installation werden Sie aufgefordert, alle Browserfenster und Microsoft Outlook zu schließen.
  1. Geben Sie im Feld Beschreibung eine Beschreibung für die Datei ein, wenn die Standardbeschreibung nicht verständlich ist.

  2. Klicken Sie auf OK.

Aufzeichnen einer E-Mail-Nachricht in der Historie
  1. Öffnen Sie die Detailansicht für den entsprechenden Kontakt, die Firma, die Verkaufschance, den Interessenten oder das Ticket.
  2. Wählen Sie in Microsoft Outlook die E-Mail-Nachricht aus, die Sie zu Sage SalesLogix hinzufügen möchten.
  3. Klicken Sie auf die Nachricht, halten Sie die Maustaste gedrückt und ziehen Sie die Nachricht in die geöffnete Detailansicht in Sage SalesLogix.
    Abhängig von Ihren Optionseinstellungen werden Sie aufgefordert, Historiendaten einzugeben.
