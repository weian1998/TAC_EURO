Neues in dieser Version

Sage SalesLogix Web Client enthält folgende Neuerungen:

Version 8.0

  • Bei Listenansichtsgruppen werden Ihre Änderungen an der Sortierung und Spaltenbreite gespeichert.
  • Die Verbesserungen der Suchfunktion umfassen die Möglichkeit, Suchen von der linken Navigationsleiste durchzuführen, um beliebige Datensatztypen zu finden, sowie eine neue Option zur Festlegung einer Standardsuchbedingung.
  • Warnungen informieren Sie, wenn ein Aktivitätsalarm oder eine neue Benachrichtigung zu einer nicht bestätigten Aktivität vorliegt. Sie können Ihre Warnungen durch Festlegen von Warnungsoptionen anpassen.
  • Neuerungen bei Aktivitäten:
    • Auf der Registerkarte Verfügbarkeit können Sie die Kalender anderer Benutzer anzeigen und so die Verfügbarkeit von Benutzern und Ressourcen feststellen.
    • In das Feld Standort können Sie einen Standort für einen Anruf oder ein Meeting eingeben oder den Standort automatisch hinzufügen, sofern er auf der Registerkarte Verfügbarkeit ausgewählt wurde.
    • Bestätigungen zu Aktivitäten enthalten jetzt mehr Informationen, damit Sie besser entscheiden können, ob Sie eine Einladung annehmen oder ablehnen.
    • Die Listenansicht Aktivitäten wurde aktualisiert und enthält jetzt neue Registerkarten, verbesserte Filter und allgemeine Aufgaben. Auf der Registerkarte Eigene Aktivitäten können Sie alle Aktivitäten anzeigen, bei denen Sie Organisator oder Mitglied sind.
    • Sie können Informationen zu Kontakten, Firmen oder Interessenten anzeigen (z. B. eine Telefonnummer), indem Sie in der Aktivitätsdetailansicht oder der Ansicht Warnungen den Mauszeiger über den zugehörigen Namen bewegen.
  • Neue und aktualisierte Aktivitäts- und Kalenderbenutzeroptionen:
    • Optionen werden jetzt bei Sage SalesLogix Client und Sage SalesLogix Web Client gemeinsam verwendet. Eventuell müssen manche Benutzer ihre Optionen neu einstellen.
    • Die neuen Aktivitätsbenutzeroptionen ermöglichen, Standardwerte pro Aktivitätstyp festzulegen.
    • Neue Kalenderbenutzeroptionen ermöglichen Ihnen festzulegen, welche Aktivitätsinformationen auf dem Kalender angezeigt werden, sowie die Tage Ihrer Arbeitswoche und den ersten Tag Ihrer Woche einzustellen. Über eine Option können Sie zudem festlegen, dass die Benutzer, deren Kalender Sie für die Anzeige ausgewählt hatten, gespeichert werden.
  • Der Kalender wurde aktualisiert und bietet neue Funktionen wie z. B.:
    • Möglichkeit anzupassen, welche Tage die Arbeitswoche bilden
    • Möglichkeit, bis zu vier Benutzer in jeder Kalenderansicht anzuzeigen
  • Kontakt verschieben und Kontakt duplizieren werden jetzt getrennt durchgeführt. Bei den neuen Methoden können Sie festlegen, welche Informationen verschoben oder neu zugeordnet werden.
  • Es gibt neue Seriendruckvorlagen für Interessenten.
  • Möglichkeit zur Freigabe von Seriendruckvorlagen für andere Benutzer
  • Das Menü Verkaufschance wurde entfernt. Die Befehle Mehrere Verkaufschancen aktualisieren und Verkaufschancenstatistik befinden sich jetzt im Kontextmenü der Ansicht Verkaufschancenliste.
  • Die Buchhaltungsintegration beinhaltet jetzt eine X3-Integration.
  • Funktionen für Administratoren und Benutzer mit den entsprechenden Rollenberechtigungen:
    • Mit dem Sicherheits-Manager können Sie den Zugriff auf Feldebene in Sicherheitsprofilen verwalten.
    • Die Standardrolle berechtigt jetzt zum Anzeigen von Produkten und Paketen (nicht zum Hinzufügen, Bearbeiten oder Löschen).
    • In der Ressourcenverwaltung lässt sich feststellen, ob es sich bei einer Ressource um einen Standort handelt.

    • Mit dem Web-Formular-Designer können Sie einfache Änderungen an einer Ansicht vornehmen. Sie können beispielsweise Beschriftungen ändern, Objekte neu positionieren sowie Informationen hinzufügen oder ausblenden. Weitere Informationen finden Sie im Hilfethema Arbeiten mit Formularen".
