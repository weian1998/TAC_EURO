Verwenden des Prozess-Managers

Mithilfe des Kontaktprozess-Managers im Web können Sie die Ihren Kontakten zugewiesenen Prozesse anzeigen oder neue Prozesse hinzufügen. Alle weiteren Änderungen an Prozessen müssen im Sage SalesLogix Network Client oder SalesLogix Remote Client vorgenommen werden. Verwenden Sie die Registerkarte Prozesse in der Kontaktdetailansicht, um die mit einem einzelnen Kontakt verknüpften Prozesse anzuzeigen.

Der Standardstatus wird durch die Benutzereinstellungen festgelegt. Sie können jedoch andere Statusangaben anzeigen, indem Sie im Prozess-Manager auf die entsprechende Registerkarte klicken. Es gibt folgende Prozessstatus: Alle Prozesse, Starten, In Bearbeitung, Warten, Angehalten, Abgeschlossen und Abgebrochen. Sie können nur die Prozesse anzeigen, die mit Kontakten verknüpft sind, auf die Sie Zugriff haben.

Durch Ihren Sicherheitszugriff wird bestimmt, welche Funktionen verfügbar sind. Wenden Sie sich an den Administrator, wenn Sie Ihre Zugriffsrechte ändern möchten. Für WebViewer-Benutzer kann die Funktionalität eingeschränkt sein.

So zeigen Sie einen Prozess an

  1. Klicken Sie auf der Navigationsleiste auf Prozesse.
  2. Klicken Sie in der Ansicht Prozess-Manager auf eine Registerkarte, um alle Prozesse mit einem bestimmten Status anzuzeigen. Folgende Optionen stehen zur Wahl: Starten, In Bearbeitung, Warten, Angehalten, Abgeschlossen, Abgebrochen und Alle Prozesse. Der Standardstatus wird durch die Benutzereinstellungen festgelegt. Sie können jedoch andere Statusangaben anzeigen, indem Sie im Prozess-Manager auf die entsprechende Registerkarte klicken.

So zeigen Sie Detailinformationen an

  • Klicken Sie in der Liste auf den Namen des Kontakts, den Sie anzeigen möchten.
