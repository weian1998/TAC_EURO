Verwenden von Analyseansichten

Analyseansichten zeigen ähnlich wie Inhaltsansichten Informationen an, bieten aber mehr Möglichkeiten für eine Filterung der Daten. Analyseansichten werden durch Klicken auf Verknüpfungen in Inhaltsansichten geöffnet.

In jeder Analyseansicht werden bestimmte Daten aus der Sage SalesLogix-Datenbank angezeigt. In einer Analyseansicht können Sie

  • Daten aus einem bestimmten Zeitraum anzeigen;
  • Die Parameter unter Auflisten nach und Zeitraum ändern, die Sie zuvor in der Inhaltsansicht ausgewählt haben.
  • Daten filtern, um weitere Details anzuzeigen.
  • Zur Verkaufschancen-Detailanalyseansicht gehen.
  • Wie bei Inhaltsansichten stellt das Raster den Schlüssel für eine Analyseansicht dar. Die Diagramm stellt die Daten aus dem Raster dar. Sie können die Daten im Raster mit Hilfe von Filtern steuern. Unten im Raster wird die Gesamtsummenzeile angezeigt, in der die Ergebnisse berechnet werden.
  • Sie können aus einer Vielzahl von Filtergruppen einzelne Filterwerte auswählen, um Ihre Suchergebnisse gezielt auszurichten.
  • Die Größe der Diagramme und des Rasters lässt sich durch Verschieben ihrer Ränder ändern.
  • Die Ergebnisse aus Rastern von Analyseansichten können mit einem einzigen Mausklick in Microsoft Excel exportiert werden.

So zeigen Sie Daten in der Analyseansicht an:

  1. Öffnen Sie im Cockpit eine Inhaltsansicht und ändern Sie gegebenenfalls die Parameter.
  2. Klicken Sie in dem Raster auf den Hyperlink zu der Analyseansicht, die Sie anzeigen möchten.
    Wenn Sie beispielsweise Details zu den Forecasts eines bestimmten Produkts anzeigen möchten, klicken Sie in der Inhaltsansicht Produkt-Forecast in dem Raster auf den Namen eines Produkts.

Die Analyseansicht wird angezeigt. Je nachdem, welche Analyseansicht Sie ausgewählt haben, enthält das Raster einen oder mehrere Einträge. In dem Diagramm werden jetzt nicht mehr die Daten der Inhaltsansicht dargestellt, sondern nur noch die Daten zu dem von Ihnen ausgewählten Eintrag.

  1. (Optional) Geben Sie in das Feld Einblenden über dem Raster die Anzahl der anzuzeigenden Zeilen ein, oder wählen Sie sie aus, und klicken Sie anschließend auf Aktualisieren (Schaltfläche über dem Diagramm).
  2. Um den Zeitraum zu ändern, wählen Sie eine Option aus der Liste Zeitraum aus.
  3. Um die Parameter unter Auflisten nach für die gesamte Analyseansicht zu ändern, wählen Sie eine Option aus dem Feld Auflisten nach aus.
    oder
    Um die Parameter für einzelne Einträge in dem Analyseansicht-Raster zu ändern, gehen Sie wie folgt vor:
  1. Klicken Sie auf die Schaltfläche Drilldownn neben dem Eintrag, zu dem Sie die Parameter ändern möchten.
  2. Zeigen Sie auf Detailsuche/Pivot und wählen Sie einen Parameter aus der Liste aus.

Die Analyseansicht zur Verkaufschancen-Pipeline ist die einzige Analyseansicht, zu der Sie die Parameter für Auflisten nach nicht ändern können.

  1. Um weitere Informationen zu einem Eintrag in dem Raster zu suchen und anzuzeigen, klicken Sie auf den Hyperlink des betreffenden Eintrags.

Häufig sind mehrere Analyseansichtfenster gleichzeitig geöffnet und haben auch oft ähnliche Daten. Stellen Sie sicher, dass Sie im richtigen Fenster sind, indem Sie den Namen der Analyseansicht in der Titelleiste überprüfen.

Auswählen eines Datumsbereichs

Ein Datumsbereich ist ein benutzerdefinierter Zeitraum. Sie können sich beispielsweise die Daten vom 1. Januar bis zum 31. März eines beliebigen Jahres anzeigen lassen.

  1. Wählen Sie aus der Liste Zeitraum den Eintrag Datumsbereich aus.
  2. Geben Sie im Dialogfeld Datumsbereich festlegen in die Felder Anfangsdatum und Enddatum die entsprechenden Daten ein.
    -oder-
    Klicken Sie auf die Kalenderschaltfläche und wählen Sie ein Datum aus.
    • Verwenden Sie die Pfeilschaltflächen, um einen Monat auszuwählen, oder klicken Sie auf den Namen des Monats, um eine Liste der Monate anzuzeigen, aus denen Sie auswählen können.
    • Klicken Sie auf das angezeigte Jahr, um den Jahresselektor zu verwenden.
  1. Klicken Sie auf OK.

Sortieren der Spalten im Raster

  • Klicken Sie in dem Raster auf eine Spaltenüberschrift.

Das Raster wird in aufsteigender oder absteigender Reihenfolge sortiert.

Schließen eines Analyseansichtsfensters

  • Klicken Sie auf die Schließen-Schaltfläche () oben rechts im Cockpit.
    Wenn Sie mehrere Fenster gleichzeitig geöffnet haben, gelangen Sie zum zuvor geöffneten Fenster, wenn Sie ein Fenster schließen.
