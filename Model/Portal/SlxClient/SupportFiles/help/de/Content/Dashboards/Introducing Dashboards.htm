Einführung in Cockpits

Cockpits ermöglichen auf der Sage SalesLogix Web Client-Begrüßungsseite das Anzeigen von und die Interaktion mit einer Vielzahl von Leistungs- und Analysefunktionen. Sie können eine oder mehrere individuell angepasste Cockpitregisterkarten verwenden und bei entsprechenden Berechtigungen Cockpits anpassen, die von Sage bereitgestellt oder Ihrem Administrator angepasst wurden.

Sie können die Daten im Cockpit mithilfe von Ein dynamisches, wieder verwendbares Element auf der Benutzeroberfläche, das die angegebenen Daten anzeigt.Widgets formatieren, um Einblick in die Leistung des Unternehmens und einzelner Mitarbeiter zu gewinnen. Die Daten können in einem Kreis-, Balken- oder Säulendiagramm, einer Pipeline oder einem Liniendiagramm visualisiert werden. Sie können die detaillierten Daten auch in einer Liste anzeigen.
