Einführung in Widgets

Widgets sind Plug-Ins, die Sie einem Cockpit hinzufügen können. Es gibt verschiedene Typen von Widgets, mit deren Hilfe Sie Ihre Daten anzeigen können. Bewegen Sie den Mauszeiger über dem Diagramm in einem Widget, um die für dieses Widget ausgewählte Dimension und Kennzahl anzuzeigen.

Dimensionen sind die einer Entität, die Sie anzeigen möchten, zugeordneten Daten.

Für die Entität Firma können Sie beispielsweise die folgenden Dimensionen anzeigen:

  • Account Manager
  • Ort
  • Branche
  • Zugriffsberechtigter
  • BDI
  • Bundesland/Kanton
  • Status
  • Untertyp
  • Typ

Kennzahlen bieten eine Möglichkeit der Berechnung der Daten, die Sie anzeigen möchten.

Für die Entität Firma können Sie beispielsweise die folgenden Kennzahlen verwenden:

  • Durchschnittliche Anzahl Tage seit letzter Aktivität
  • Durchschnittlicher Umsatz
  • Durchschnittliche Zeit als Kunde
  • Firmenanzahl ermitteln
  • Gesamtumsatzerlös

Klicken Sie unten in jedem Widget auf den Hyperlink, um detaillierte Informationen zu diesen Daten anzuzeigen. Wenn Sie beispielsweise das Widget Gruppenliste so konfiguriert haben, dass alle offenen Verkaufschancen angezeigt werden, wird über den Hyperlink Gruppe anzeigen die Verkaufschancen-Listenansicht geöffnet. Klicken Sie auf der Navigationsleiste auf Willkommen, um zum Cockpit zurückzukehren.

Es gibt mehrere Typen von Widgets:

Diagramm-Widgets

  • Balken - Zeigt die Daten als horizontale Balken an. BeispielWenn Sie die Daten für Durchschnittliche Auftragsgröße nach Zugriffsberechtigtem auswählen, werden die Teams auf der Y-Achse und die Anzahl der Aufträge auf der X-Achse angezeigt. Jeder Balken erstreckt sich horizontal, um den Wert auf der X-Achse zu erreichen, der am nächsten an das Gesamtergebnis kommt.
  • Säule - Zeigt die Daten als vertikale Säulen an. BeispielWenn Sie die Daten für Durchschnittliche Auftragsgröße nach Zugriffsberechtigtem auswählen, werden die Teams auf der X-Achse und die Anzahl der Aufträge auf der Y-Achse angezeigt. Jede Säule erstreckt sich vertikal, um den Wert auf der Y-Achse zu erreichen, der am nächsten an das Gesamtergebnis kommt. Sie können der Konfiguration ein Ziel hinzufügen, das als horizontale Linie über dem Diagramm angezeigt wird und veranschaulicht, wo sich die Säulen im Verhältnis zum Ziel befinden.
  • Trichter - Zeigt die Daten in einem farblich codierten Trichter in progressiven Proportionen an. Verwenden Sie Trichterdiagramme, um Informationen in einem mehrstufigen Prozess darzustellen, wie etwa dem potenziellen Umsatz oder der Anzahl der Verkaufschancen in Ihrer Verkaufschancen-Pipeline. Beispiele

Offene Verkaufschancen nach Verkaufspotenzial

Verwenden Sie ein Trichterdiagramm zum Aufzeigen des potenziellen Umsatzes von Verkaufschancen in Ihrer Pipeline für jede Umsatzebene.

  1. Klicken Sie mit der rechten Maustaste auf eine Cockpit-Registerkarte, und klicken Sie auf Inhalt hinzufügen.
  2. Klicken Sie unter Trichterdiagramm auf die Schaltfläche Hinzufügen.
  3. Geben Sie in Titel einen Namen für das Trichterdiagramm ein, z. B. Offene Verkaufschancen nach Verkaufspotenzial.
  4. Wählen Sie in der Liste Entität die Option Verkaufschance aus.
  5. Wählen Sie in der Liste Gruppe die Option Alle offenen aus.
  6. Wählen Sie in der Liste Dimension die Option Phase aus.
  7. Wählen Sie in der Liste Metrik die Option Verkaufspotenzial gesamt aus.
  8. Legen Sie über Beschriftungen anzeigen, Beschriftungen abschneiden und Beschriftungslänge die gewünschten Einstellungen fest.
  9. Klicken Sie auf OK.
    Ein Bereich von Verkaufspotenzialbeträgen wird angezeigt, wobei jeder Verkaufspotenzialbereich als eine andersfarbige Ebene dargestellt wird. Die Größe jeder Ebene stellt die Anzahl der Verkaufschancen in einem bestimmten Verkaufspotenzialbereich dar.

Offne Verkaufschancen nach Phase

Veranschaulichen Sie mit einem Trichterdiagramm, wie viele Verkaufschancen sich in den einzelnen Phasen der Verkaufschancen-Pipeline befinden.

  1. Klicken Sie mit der rechten Maustaste auf eine Cockpit-Registerkarte, und klicken Sie auf Inhalt hinzufügen.
  2. Klicken Sie unter Trichterdiagramm auf die Schaltfläche Hinzufügen.
  3. Geben Sie in Titel einen Namen für das Trichterdiagramm ein, z. B. Meine Verkaufschancen nach Phase.
  4. Wählen Sie in der Liste Entität die Option Verkaufschance aus.
  5. Wählen Sie in der Liste Gruppe die Option Alle Verkaufschancen aus.
  6. Wählen Sie in der Liste Dimension die Option Phase aus.
  7. Wählen Sie in der Liste Metrik die Option Verkaufschancenanzahl ermitteln aus.
  8. Legen Sie über Beschriftungen anzeigen, Beschriftungen abschneiden und Beschriftungslänge die gewünschten Einstellungen fest.
  9. Klicken Sie auf OK.
    Jede Phase wird in der Ablauffolge des Prozesses als andersfarbige Ebene dargestellt. Die Größe jeder Ebene stellt die Anzahl der Verkaufschancen in einer bestimmten Phase dar.
  • Linie - Zeigt die Daten als Folge von Datenpunkte an, die durch eine Linie verbunden sind. Beispiel Wenn Sie die Daten für Aktive Marketingkampagnen nach Region auswählen, werden die Teams auf der X-Achse, die Anzahl der Kampagnen auf der Y-Achse und die Anzahl der Datenpunkte im Diagramm angezeigt.
  • Kreis - Zeigt die Daten in einem Kreis nach Farben getrennt und mit einer Legende mit begleitenden Daten an. Beispiel Wenn Sie Firmen nach Region mit fünf Segmenten anzeigen, zeigt das Diagramm einen fünffarbigen Kreis mit einer Legende, die die Teams sowie die Gesamtanzahl der Firmen pro Team enthält. Sie können das Kreisdiagramm so konfigurieren, dass fünf oder zehn Segmente angezeigt werden, und bestimmten, ob das Segment Sonstige angezeigt werden soll, das alle nicht gezeigten Daten enthält.

Listen-Widgets

  • Gruppe- Zeigt die angegebenen Gruppeninformationen an. Beispiel Wenn Sie Eigene abgeschlossene Verkaufschancen auswählen, können Sie die Spalten in der Liste sortieren und über den Hyperlink Gruppe anzeigen zu der Firma oder Entität navigieren, um detaillierte Informationen anzuzeigen.

    		•
    • Wenn Sie eine Registerkarte Cockpit freigeben, die eine Gruppe enthält, muss die Gruppe ebenfalls für diese Benutzer freigegeben werden, damit die Gruppe auf der Registerkarte Cockpit angezeigt wird.
    • Der Hyperlink Gruppe anzeigen steht für die Historie nicht zur Verfügung.

  • Hyperlinks- Zeigt die angegebenen Hyperlinks an. Beispiel Sie können Hyperlinks hinzufügen, die Sie in Sage SalesLogix oder im Internet häufig nutzen.
  • Zuletzt angezeigt - Zeigt die zuletzt angezeigten Details als Hyperlinks an. Wenn Sie die Details einer Firma oder eines Kontakts angezeigt haben, enthält die Liste Hyperlinks zu diesen Detailansichten.

Andere Widgets

  • Aktivitäten von heute - Zeigt eine Liste aller Ihrer für den heutigen Tag geplanten Aktivitäten an. Wenn Sie weitere Informationen einblenden möchten, klicken Sie auf den Hyperlink, um die Detailansicht der Aktivität aufzurufen.
  • Willkommen - Zeigt eine Einführung in den Sage SalesLogix Web-Client mit einem Link zur Hilfe zu Sage SalesLogix Web-Client an. Eine kurze Beschreibung aller Funktionen, die Sage SalesLogix Ihnen zum Verwalten Ihrer Arbeit bereitstellt.
