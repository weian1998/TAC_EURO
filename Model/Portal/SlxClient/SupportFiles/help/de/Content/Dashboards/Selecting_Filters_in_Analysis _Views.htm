Auswählen von Filtern in der Analyseansicht

Ändern der Filter von Analyseansichten

Indem Sie filtern, können Sie den Schwerpunkt auf bestimmte Daten legen. Wenn Sie beispielsweise aus dem Feld Zeitraum den Eintrag Letzte 30 Tage auswählen, gibt die Datenbank nur die Ergebnisse der letzten 30 Tage aus. Sie können Ihre Ergebnisse weiter eingrenzen, indem Sie nach weiteren Informationen suchen.

Unser Szenario:
Sie möchten sich offene Verkaufschancen in Ihrem Unternehmen genauer ansehen.

So öffnen Sie eine Analyseansicht

  1. Klicken Sie in der Navigationsleiste auf Cockpits und anschließend auf Vertriebs-Cockpit.
  2. Klicken Sie im Abschnitt Pipeline-Status auf die Registerkarte Offene Verkaufschancen.
  3. Klicken Sie auf den ersten Hyperlink in der Spalte Account Manager.
    Hierdurch wird das Fenster Offene Verkaufschancen - Analyseansicht geöffnet, wobei der Filter "Account Manager" auf den Parameter eingestellt ist, den Sie in der Inhaltsansicht Offene Verkaufschancen ausgewählt hatten.

    Analyseansicht

  4. Klicken Sie auf Filteroptionen.
    Das Filterfenster wird mit einer Liste von Filtereinträgen geöffnet. Der Filter "Account Manager" zeigt den Account Manager, den Sie im vorherigen Schritt ausgewählt haben.
  5. Klicken Sie im Bereich der Filteroptionen links von der Analyseansicht auf Filter löschen.
    Nach dem Entfernen der Filtereinschränkung werden das Raster und die Diagramme aktualisiert und zeigen nun alle verfügbaren Daten für offene Verkaufschancen an. Die Auswahl Auflisten nach ist immer noch auf "Account Manager" eingestellt.
  6. Klicken Sie in der Spalte Wert neben dem Filter "Firmenzugriffsberechtigter" auf ALLE.
  7. Klicken Sie auf die Auslassungspunkte rechts in der Spalte Wert.
    Daraufhin wird das Dialogfeld Filterwerte hinzufügen/bearbeiten angezeigt.

    Dialog Filterwerte hinzufügen/bearbeiten

  8. Wählen Sie drei Werte für Firmenzugriffsberechtigte aus. Sie können die Suche mit Hilfe der Bildlaufleiste durchführen oder das Feld Suchen verwenden, um die Auswahl zu verfeinern.
  9. Wenn Sie fertig sind, klicken Sie auf OK.
    Im Raster und in den Diagrammen werden die offenen Verkaufschancen für die ausgewählten Firmenzugriffsberechtigten angezeigt.

    Sie können durch Klicken auf Filteroptionen den Filterbereich ausblenden und so den sichtbaren Bereich für Raster und Diagramme in der Analyseansicht vergrößern.

  10. Zum Drucken der Analyseansicht klicken Sie im Menü Datei auf Bildschirm drucken.
  11. Um die Rasterdaten in Microsoft Excel zu exportieren, klicken Sie auf In Excel exportieren.
    Eine Excel-Arbeitsmappe wird mit allen Rasterspalten und Daten geöffnet. (Hierfür ist Microsoft Excel erforderlich.)

    Es wird nur die im Raster angezeigte Anzahl von Zeilen exportiert. Verwenden Sie bei Bedarf die Auswahl für die Zeilenanzeige, um zusätzliche Zeilen anzuzeigen.

Weitere Informationen finden Sie unter dem Thema "Planen oder Bearbeiten einer Aktivität " in der Cockpit-Hilfe zu .
