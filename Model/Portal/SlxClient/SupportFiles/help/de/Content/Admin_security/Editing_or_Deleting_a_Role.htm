Bearbeiten oder Löschen einer Rolle

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Wenn Sie eine Rolle löschen, werden alle Beziehungen mit dieser Rolle entfernt. Falls der gelöschten Rolle Benutzer zugewiesen waren, haben sie keinen Zugriff mehr auf die in der Rolle definierten Aktionen. Die Benutzer behalten den Zugriff auf Aktionen, die ihnen in anderen Rollen zugewiesen sind.

Änderungen werden wirksam, nachdem sich der Benutzer vom Web Client abgemeldet hat.

Vorgehensweise

Bearbeiten einer Rolle

  1. Öffnen Sie die Rollendetailansicht der Rolle, die Sie bearbeiten möchten.
  2. Auf der Registerkarte Aktionen können Sie die Rollenaktionen bearbeiten.
  3. Auf der Registerkarte Benutzer können Sie Benutzer der Rolle hinzufügen oder aus ihr entfernen.
  4. Klicken Sie auf Speichern.

Löschen einer Rolle

  1. Öffnen Sie die Rollendetailansicht der Rolle, die Sie löschen möchten.
  2. Klicken Sie auf Löschen.
  3. Klicken Sie auf OK, um den Vorgang zu bestätigen.
