Verwenden der Rollenlistenansicht

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

In der Rollenlistenansicht werden Rollen in einem Raster angezeigt. Das Raster enthält Informationen zu jeder Rolle, z. B. den Namen und eine Beschreibung der Rolle.

  • Um das Raster nach einem bestimmten Feld zu sortieren, klicken Sie auf eine der Spaltenüberschriften.
  • Um alle Rollen eines bestimmten Typs anzuzeigen, klicken Sie im Raster auf eine Registerkarte.
  • Konfigurieren Sie Filter, um nur Rollen anzuzeigen, die bestimmte Kriterien erfüllen.
