Registerkarte "Rollen"

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.Der Systemadministrator und Benutzer mit einer Administratorrolle können auf diese Funktionen zugreifen.

In der Detailansicht Gesicherte Aktion können Sie auf der Registerkarte Rollen die einer Rolle zugewiesenen gesicherten Aktionen verwalten. Eine Rolle bestimmt, auf welche Funktionen ein Benutzer im Web Client zugreifen darf. Durch Hinzufügen gesicherter Aktionen zu einer Rolle wird bestimmt, auf welche Funktionen die Rolle Zugriff hat. Durch Hinzufügen von Benutzern zur Rolle wird bestimmt, wer Zugriff hat.

Die Beziehung zwischen Rollen und gesicherten Aktionen wird auf dieser Registerkarte und auf der Registerkarte Aktionen in der Rollendetailansicht angezeigt.

  • Auf der Registerkarte Rolle werden die zur Rolle gehörigen gesicherten Aktionen angezeigt.
  • Auf der Registerkarte Aktion werden die Rollen angezeigt, die die gesicherte Aktion enthalten.

Wenn Sie einer Rolle eine gesicherte Aktion hinzufügen oder aus dieser entfernen, wird die Änderung auf beiden Registerkarten wiedergegeben.

Vorgehensweise

Hinzufügen einer gesicherten Aktion zu einer Rolle

  1. Öffnen Sie die Detailansicht Gesicherte Aktion.
  2. Klicken Sie auf die Registerkarte Rollen.
  3. Klicken Sie auf Rolle hinzufügen.
  4. Wählen Sie über die Suchfunktion die Rolle aus, der Sie die gesicherte Aktion hinzufügen möchten.
  5. Wiederholen Sie die Schritte 3-4 für jede weitere Rolle.
  6. Klicken Sie auf OK.

Entfernen einer gesicherten Aktion aus einer Rolle

  1. Öffnen Sie die Detailansicht Gesicherte Aktion.
  2. Klicken Sie auf die Registerkarte Rollen.
  3. Klicken Sie für die Rolle, aus der Sie die gesicherte Aktion entfernen möchten, auf den Hyperlink Entfernen.

  4. Klicken Sie auf OK.
