Erstellen gesicherter Aktionen

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.Der Systemadministrator und Benutzer mit einer Administratorrolle können auf diese Funktionen zugreifen.

Zum Sichern von Elementen im Web Client können Sie neue gesicherte Aktionen erstellen.

Stellen Sie beim Erstellen einer neuen gesicherten Aktion sicher, dass die Aktion einen eindeutigen bislang nicht verwendeten Namen hat (es sei denn, Sie stellen eine versehentlich gelöschte gesicherte Aktion wieder her). Wenn Sie eine gesicherte Aktion gelöscht haben, ohne die Aktion manuell aus den Elementen in Application Architect zu entfernen, bietet die neue Aktion ggf. einen unerwünschten Zugriff. Weitere Informationen...

Angenommen, Sie haben die gesicherte Aktion Drucken/Berichte erstellt und mehreren Schaltflächen vom Typ Drucken hinzugefügt (mithilfe der Eigenschaft Angewandte Sicherheit in Application Architect). Anschließend haben Sie die gesicherte Aktion Drucken/Berichte gelöscht, ohne die Eigenschaft Angewandte Sicherheit für die Schaltflächen vom Typ Drucken zu ändern. Wenn Sie eine neue gesicherte Aktion mit dem Namen Drucken/Berichte erstellen, werden alle Schaltflächen vom Typ Drucken, die zuvor der Aktion Drucken/Berichte zugeordnet waren, mit der neuen Aktion Drucken/Berichte verknüpft.

Sage empfiehlt das Verwenden eines beschreibenden Namens für Ihre gesicherten Aktionen, der zum Nachverfolgen des beabsichtigten Zwecks der Aktion dienen kann. Die gesicherte Aktion Entität/Firma/Löschen dient beispielsweise zum Steuern von Elementen, die Benutzern das Löschen von Firmen ermöglichen. Diese Benennungsstruktur wird empfohlen, aber nicht erzwungen. Der Name der gesicherten Aktion ist ein Zeichenfolgenwert mit beliebigen Benennungsregeln.

So gehen Sie zum Erstellen vor

  1. Klicken Sie im Menü Administration auf Neue gesicherte Aktion.
  2. Geben Sie in das Feld Gesicherte Aktion den Namen der gesicherten Aktion ein.
  3. Geben Sie in das Feld Beschreibung beschreibende Informationen zur gesicherten Aktion ein.
  4. Klicken Sie auf Speichern, um die gesicherte Aktion zu speichern, oder auf Speichern und neu, um die gesicherte Aktion zu speichern und eine weitere zu erstellen.
  5. Auf der Registerkarte Rollen können Sie die gesicherte Aktion einer Rolle hinzufügen.
  6. Auf der Registerkarte Standorte können Sie prüfen, ob die gesicherte Aktion verwendet wird.
