Zuweisen von Benutzern zur Rolle "Standardbenutzer"

Benutzern, die im Web Client erstellt werden, wird standardmäßig die Rolle Standardbenutzer zugewiesen. Diese Rolle ermöglicht das Hinzufügen, Bearbeiten und Löschen von Berechtigungen für nicht administrative Entitäten wie Firmen, Kontakte und Verkaufschancen.

In Administrator erstellten Benutzern wird keine Rolle zugewiesen, und in Administrator kann die Rollensicherheit nicht festgelegt werden. Wenn Benutzer vorhanden sind oder Sie neue Benutzer im Administrator erstellen, müssen Sie die Benutzer der Rolle Standardbenutzer zuweisen.

So nehmen Sie Zuweisungen vor

  1. Öffnen Sie die Benutzerlistenansicht.
  2. Wählen Sie die Benutzer aus, die Hinzufügungs-, Bearbeitungs- und Löschberechtigungen für nicht administrative Entitäten im Web Client haben sollen.
  3. Klicken Sie im Bereich Benutzeraufgaben auf Zu Rolle hinzufügen.
  4. Wählen Sie im Dialogfeld Rolle auswählen die Rolle Standardbenutzer aus, und klicken Sie auf OK.

  5. Klicken Sie auf OK.
