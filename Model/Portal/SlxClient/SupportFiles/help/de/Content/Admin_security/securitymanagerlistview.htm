Verwenden der Sicherheitsprofil-Manager-Ansicht

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

In der Sicherheitsprofil-Manager-Listenansicht können Sie den Zugriff auf Feldebene in Sicherheitsprofilen verwalten.

Vorgehensweise

Öffnen der Sicherheitsprofil-Manager-Listenansicht
  • Klicken Sie auf der Navigationsleiste auf Administration., und klicken Sie dann auf Sicherheitsprofil-Manager.
Aktualisieren von Zugriffsrechten für Felder in einem Sicherheitsprofil
  1. Wählen Sie in der Sicherheitsprofil-Manager-Ansicht die Registerkarte des Sicherheitsprofils aus, das Sie ändern möchten.
  2. Wählen Sie für jedes Feld eine der folgenden Zugriffsebenen aus:
  1. Lesen/Schreiben: Benutzer können die Daten anzeigen und bearbeiten. Dies ist die Standardoption, die allen Tabellen und Feldern zugewiesen wird.
  2. Schreibgeschützt - Benutzer können die Daten nur anzeigen.
  3. Kein Zugriff - Benutzer können die Daten nicht anzeigen oder bearbeiten.

Zum Zuordnen derselben Zugriffsebene zu allen Feldern im Sicherheitsprofil aktivieren Sie das Kontrollkästchen neben der entsprechenden Spaltenüberschrift. Aktivieren Sie beispielsweise das Kontrollkästchen neben der Spalte Lesen/Schreiben, um Benutzern im Sicherheitsprofil das Anzeigen und Bearbeiten von Daten aller Felder zu erlauben.

  1. Sobald Sie mit dem Ändern der Zugriffsrechte für Tabellen und Felder fertig sind, klicken Sie auf Speichern..
    Falls Sie Ihre Änderungen vor dem Speichern rückgängig machen möchten, klicken Sie auf Zurücksetzen.
Aktualisieren von Listeninformationen
  • Klicken Sie auf die Schaltfläche Aktualisieren.
Allgemeine Aufgaben

  • Bearbeiten eines Profils
