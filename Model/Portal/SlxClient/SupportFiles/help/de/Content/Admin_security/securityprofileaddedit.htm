Hinzufügen oder Bearbeiten eines Sicherheitsprofils

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

In diesem Dialogfeld können Sie Informationen für ein neues Sicherheitsprofil hinzufügen oder die Informationen eines vorhandenen Sicherheitsprofils bearbeiten.

So wird ein Sicherheitsprofil hinzugefügt oder bearbeitet

  1. Öffnen Sie die Sicherheitsprofil-Manager-Ansicht.
  2. Klicken Sie im Aufgabenbereich unter Allgemeine Aufgaben auf Profil hinzufügen oder Profil bearbeiten.
  3. Geben Sie in das Feld Name einen kurzen, aussagekräftigen Namen für das Sicherheitsprofil ein.
  4. Wählen Sie in der Liste Profiltyp die Option Benutzer oder System aus.
    Diese beiden Typen werden lediglich zu organisatorischen Zwecken bereitgestellt; sie besitzen keine unterschiedlichen Funktionen. Systemprofile sind dafür vorgesehen, für Teams verwendet zu werden, und Benutzerprofile sollten für einzelne Benutzer verwendet.

  5. Klicken Sie auf OK.

Anweisungen zum Aktualisieren von Zugriffsrechten in einem Profil finden Sie unter "Aktualisieren von Zugriffsrechten für Felder in einem Sicherheitsprofil" im Hilfethema Verwenden der Sicherheitsprofil-Manager-Ansicht.
