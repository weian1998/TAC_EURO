Fehlerbehebung für das Desktopintegrationsmodul

Beim Desktopintegrationsmodul können die folgenden Probleme auftreten.

Problem:

Das Sage SalesLogix Desktop-Manager-Symbol wird nicht auf der Taskleiste angezeigt.

Mögliche Ursachen und Lösungen:

  • Das Desktopintegrationsmodul ist nicht installiert.
  • Sie müssen das Desktopintegrationsmodul installieren.
  • Das Desktopintegrationsmodul wurde installiert, bevor Ihr Browser installiert wurde.
  • Sie haben Ihren aktuellen Browser nach dem Desktopintegrationsmodul installiert. Sie müssen in Ihrem aktuellen Browser das Desktopintegrationsmodul neu installieren.

    		•

    Dies kann passieren, wenn Sie Firefox installiert haben, nachdem Sie das Desktopintegrationsmodul mithilfe von Internet Explorer installiert haben.
  • Navigieren Sie zu SLXDesktopManager.exe, und öffnen Sie die Datei manuell. Der Speicherort dieser Datei hängt von Ihrem Betriebssystem ab:

Der Standardspeicherort lautet:

\\Dokumente und Einstellungen\[Benutzername]\Lokale Einstellungen\Anwendungsdaten\ (Windows XP/2000/2003) oder

\\Users\<Benutzer.Domäne>\appdata\local (Windows Vista/7/2008).

Problem:

Wie wird der Desktop-Manager deaktiviert?

Lösung:

  • Klicken Sie auf der Windows-Taskleiste mit der rechten Maustaste auf das Symbol Sage SalesLogix Desktop-Manager, und klicken Sie auf Beenden.

Sie müssen den Sage SalesLogix Desktop-Manager zuerst beenden, bevor Sie das Sage SalesLogix-Desktopintegrationsmodul deinstallieren können.

Problem:

Wie wird der Desktop-Manager deinstalliert?

Lösung:

  1. Öffnen Sie die Windows-Systemsteuerungsoption Software.
  2. Wählen Sie in der Liste Sage SalesLogix-Desktopintegrationsmodul aus, und klicken Sie auf Entfernen.
  3. Klicken Sie auf Ja.

    		4.

    Wenn Sie Firefox als Browser nutzen, müssen Sie zuerst in Firefox das Add-on für das Sage SalesLogix-Desktopintegrationsmodul deinstallieren.

    Öffnen Sie zum Deinstallieren des Desktopintegrationsmoduls Firefox. Klicken Sie im Menü Extras auf Sage SalesLogix-Desktopintegrationsmodul, und klicken Sie auf Deinstallieren. Klicken Sie auf Deinstallieren, um den Vorgang zu bestätigen. Klicken Sie auf Firefox neu starten. Klicken Sie auf Neu starten.
