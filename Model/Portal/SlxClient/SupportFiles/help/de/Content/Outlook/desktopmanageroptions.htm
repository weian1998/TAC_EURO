Festlegen von E-Mail-Historienoptionen in Desktop Manager

Im Sage SalesLogix Desktop-Manager können Sie Verbindungsinformationen eingeben und Historienoptionen für E-Mail festlegen.

So legen Sie Optionen fest

  1. Doppelklicken Sie auf der Windows-Taskleiste auf das Symbol Sage SalesLogix Desktop-Manager.
  2. Führen Sie einen der folgenden Schritte durch:
  • Web Client-Benutzer - Geben Sie in das Feld Portal-URL den URL ein, mit dem eine Verbindung hergestellt werden soll. Wählen Sie folgendes Format:

http://Servername:Port

  • Offline Web-Client-Benutzer - Geben Sie in das Feld Portal-URL den URL ein, mit dem eine Verbindung hergestellt werden soll. Wählen Sie folgendes Format:

http://localhost

Wenn Sie einen voll qualifizierten Namen verwenden, müssen Sie hinter dem Servernamen ggf. Domänenname.com hinzufügen.

  1. Führen Sie einen der folgenden Schritte durch:
  • Wenn Sie sich mithilfe Ihrer Windows-Kontoinformatinen bei Sage SalesLogix anmelden, wählen Sie die Option Windows-Authentifizierung verwenden.
  • Geben Sie Ihren Benutzernamen und Ihr Passwort ein.
  1. Geben Sie in das Feld Benutzer den Benutzernamen ein, mit dem Sie sich am Sage SalesLogix Web Client anmelden.
  2. Geben Sie in das Feld Passwort das Sage SalesLogix-Passwort für den zuvor eingegebenen Benutzernamen ein.
  1. Klicken Sie auf Testen.
  2. Wählen Sie unter Historienoptionen aus, wann Sie beim Protokollieren einer Outlook-E-Mail-Nachricht als Sage SalesLogix-Historiendatensatz zu einer Aktion aufgefordert werden möchten. Folgende Optionen stehen zur Auswahl:
  • Bei doppelten Kontakten nachfragen - Fragt nach, wenn ein doppelter Kontakt oder Interessent gefunden wird.
  • Bei Kontakt nachfragen - Fordert Sie zur Auswahl auf, mit welchen Kontakten oder Interessenten der Historieneintrag verknüpft werden soll.
  • Bei nicht gefundenem Kontakt nachfragen - Fragt nach, wenn ein Kontakt oder Interessent nicht in der Datenbank gefunden wird.
  • Historiendetails vor dem Speichern anzeigen - Blendet das Dialogfeld E-Mail abschließen ein, bevor eine E-Mail-Nachricht in der Historie aufgezeichnet wird.
  1. Aktivieren Sie das Kontrollkästchen Benachrichtigungen auf Taskleiste anzeigen, wenn Sie Einblendmeldungen wünschen, in denen die jeweiligen Desktop Manager-Aktivitäten beschrieben werden.

Diese Optionen gelten nicht für das Ziehen und Ablegen von E-Mails. Verwenden Sie die Option In Historie aufzeichnen in Optionen zum Ziehen und Ablegen von E-Mails.

Weitere Informationen zum Desktop-Manager finden Sie im Online-Hilfesystem zum Desktop-Manager.
