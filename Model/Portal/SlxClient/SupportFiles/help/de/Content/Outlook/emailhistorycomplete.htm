Abschließen eines E-Mail-Historiendatensatzes

Im Dialogfeld E-Mail abschließen können Sie Informationen zu einer abgeschlossenen E-Mail aufzeichnen.

So schließen Sie mehrere E-Mail-Historiendatensätze ab

  1. Führen Sie einen der folgenden Schritte durch:
    • Wählen Sie Outlook eine oder mehrere E-Mail-Nachrichten aus, und klicken Sie auf In Historie aufzeichnen.
    • Klicken Sie in Outlook nach Verfassen einer E-Mail-Nachricht auf Mit SLX senden.
    • Ziehen Sie eine oder mehrere E-Mail-Nachrichten aus Outlook, und legen Sie sie auf einer Sage SalesLogix-Firmen-, Interessenten-, Verkaufschancen- oder Ticket-Detailansicht ab.

      • Wenn Sie aufgefordert werden, bei doppelten Datensätzen eine Auswahl zu treffen oder zu bestimmen, für welche Datensätze eine Historie aufgezeichnet werden soll, müssen Sie zuerst den ordnungsgemäßen Datensatz auswählen, bevor die E-Mail in der Historie aufgezeichnet werden kann.
      • Wenn gemeldet wird, dass der Kontakt oder Interessent nicht gefunden wurde, können Sie die E-Mail nicht in der Historie aufzeichnen.
  2. Führen Sie im Dialogfeld Schnell abschließen einen der folgenden Schritte aus:
    • Klicken Sie auf Einzeln, um alle E-Mail-Historiendatensätze einzelnen aufzuzeichnen. Fahren Sie mit Schritt 3 fort.
    • Im Dialogfeld Schnell abschließen können Sie Historiendaten für mehrere E-Mail-Nachrichten eingeben.

      • Dieses Dialogfeld wird nur angezeigt, wenn Sie eine E-Mail gezogen und abgelegt haben und die Option In Historie aufzeichnen in Optionen auf Immer nachfragen festgelegt ist. Wenn Sie ausgewählt haben, dass Sie nicht zu Historieneinträgen aufgefordert werden möchten, wird die E-Mail-Historie automatisch aufgezeichnet, weshalb Sie die restlichen Schritte überspringen können.

  3. Wenn sich das Dialogfeld E-Mail abschließen öffnet, können Sie hier Historiendaten protokollieren.
    1. Klicken Sie im Feld Betreff auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
    2. Geben Sie mithilfe des Felds Standort einen Standort ein, sofern zutreffend.
    3. Ändern Sie bei Bedarf das Abschlussdatum. Zum Ändern des Datums klicken Sie auf Kalender-Schaltfläche, um das Datum auszuwählen.  
    4. Ändern Sie bei Bedarf den geplanten Termin. Zum Ändern des Datums klicken Sie auf Kalender-Schaltfläche, um das Datum auszuwählen.  
    5. Führen Sie einen der folgenden Schritte durch:
      • Klicken Sie auf Dauer (den Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus).
      • Aktivieren Sie das Kontrollkästchen Ohne Zeitangabe, wenn die Aktivität an einem bestimmten Termin erfolgt ist, aber keine spezifische Uhrzeit und Dauer aufwies.
    6. Klicken Sie im Feld Ergebnis auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
    7. Wählen Sie bei Bedarf Kontakt oder Interessent aus, und klicken Sie auf die Schaltfläche Suchen oder Löschen, um verknüpfte Interessenten, Kontakte, Firmen, Verkaufschancen oder Tickets zu ändern oder zu entfernen.
    8. Wählen Sie im Feld Priorität die Wichtigkeit der E-Mail aus.
    9. Wählen Sie im Feld Kategorie die Kategorie der E-Mail aus.
    10. Wählen Sie im Feld Organisator ggf. einen Organisator aus, wenn Sie nicht die Aktivität abgeschlossen haben. Nicht alle Benutzer werden in der Auswahlliste Organisator aufgeführt. Sie können einen Benutzer nur als Organisator der Aktivität auswählen, wenn Sie Zugriff auf seinen Kalender haben.
    11. Geben Sie im Feld Notizen alle weiteren Informationen zu dieser Aktivität ein.
    12. Planen Sie ggf. eine Folgeaktivität im Feld Folgeaktivität. Aktivieren Sie das Kontrollkästchen Notizen übertragen oder Anlagen übertragen, um Notizen oder Anlagen von dieser Aktivität in die Folgeaktivität zu kopieren.
    13. Klicken Sie auf E-Mail öffnen , um die E-Mail-Nachricht anzuzeigen.
    14. Klicken Sie auf die Registerkarte Anlagen, um ein Dokument oder einen URL hinzuzufügen oder ein Dokument oder einen URL anzuzeigen.

      15. Bei Verwendung der Schaltfläche Mit SLX senden in Outlook werden E-Mail-Anlagen mit einer der folgenden Methoden gespeichert:

      • E-Mail-Anlagen werden als separate Dateien gespeichert.
      • Die Speicherung erfolgt als einzelne E-Mail-Anlage, die die E-Mail und alle Anlagen enthält.

      Die verwendete Methode wird vom Administrator festgelegt.

    15. Klicken Sie auf OK.

      Wenn Sie eine Folgeaktivität geplant haben, wird das Dialogfeld Aktivität planen angezeigt.
