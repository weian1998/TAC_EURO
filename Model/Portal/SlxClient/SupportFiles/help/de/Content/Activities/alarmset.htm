Festlegen von Alarmen

Alarme können Sie auf eine Aktivität hinweisen, die abgeschlossen werden muss. Sie können einen Alarm zu einer bestehenden Aktivität einstellen oder wenn Sie eine Aktivität planen. Sie müssen Alarme innerhalb der Warnungen konfiguriert haben, damit Sie die Alarmbenachrichtigung verwenden können.

So legen Sie einen Alarm fest

  1. Führen Sie einen der folgenden Schritte durch:
  • Planen Sie eine neue Aktivität.
  • Suchen Sie nach der Aktivität, zu der Sie einen Alarm einstellen möchten, und öffnen Sie die Detailansicht zu der Aktivität.
  1. Aktivieren Sie das Kontrollkästchen Alarm, und wählen Sie die Zeitdauer aus, nach der Sie an diese Aktivität erinnert werden möchten.
  2. Klicken Sie auf OK, wenn Sie fertig sind.
