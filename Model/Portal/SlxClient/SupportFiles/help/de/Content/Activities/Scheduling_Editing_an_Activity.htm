Planen oder Bearbeiten einer Aktivität

Verwenden Sie die Aktivitätsdetailansicht, um eine Aktivität hinzuzufügen oder zu bearbeiten oder um ausführliche Informationen zu einer Aktivität anzuzeigen, wie Datum, Zeit, Dauer und Notizen. Sie können eine Aktivität nach dem Planen auch abschließen oder löschen.

Die Kalendersicherheitseinstellungen bestimmen, wessen Aktivitäten Sie planen, bearbeiten, abschließen oder löschen können. Sie können auch Aktivitäten anzeigen, die mit Kontakten, Interessenten, Firmen, Verkaufschancen oder Tickets verknüpft sind, auf die Sie Zugriff haben.

Vorgehensweise

Planen einer Aktivität

  1. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie im Menü Planen auf den zu planenden Aktivitätstyp (Anruf, Meeting, Aufgabe oder Persönliche Aktivität).
    • Klicken Sie in der Detailansicht Kontakt, Firma, Verkaufschance oder Ticket auf die Registerkarte Aktivitäten und anschließend auf die Schaltfläche Anruf planen, Meeting planen oder Aufgabe planen. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  2. Klicken Sie im Feld Betreff auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  3. Geben Sie in das Feld Standort einen Standort ein oder verwenden Sie die Registerkarte Availability, um eine Standortressource auszuwählen.
  4. Klicken Sie im Feld Startzeit auf die Kalender-Schaltfläche, um das Datum und die Uhrzeit der Aktivität auszuwählen, und klicken Sie dann auf OK.

  5. Führen Sie einen der folgenden Schritte durch:

    • Aktivieren Sie das Kontrollkästchen Ohne Zeitangabe, um eine Aktivität zu erstellen, die an einem bestimmten Termin stattfindet, aber keine spezifische Uhrzeit und Dauer aufweist.
      Aktivitäten ohne Zeitangabe werden in der Tagesansicht des Kalenders angezeigt.
    • Geben Sie in das Feld Dauer einen Wert ein, oder klicken Sie auf den den Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus.
  6. Aktivieren Sie das Kontrollkästchen Alarm, um einen Benachrichtigungsalarm vor Beginn der Aktivität einzurichten. Geben Sie eine Zeitdauer ein bzw. wählen Sie sie aus.
  7. Aktivieren Sie das Kontrollkästchen Automatisch auf nächsten Tag ausdehnen, um die Aktivität automatisch auf den nächsten Tag auszudehnen, wenn sie nicht am geplanten Termin abgeschlossen wird. Diese Option ist nur verfügbar, wenn Ohne Zeitangabe ausgewählt ist.

    Sich wiederholende Aktivitäten können nicht ausgedehnt werden. Wenn Sie diese Option auswählen und dann Wiederholungsoptionen festlegen, wird die automatische Ausdehnung auf den nächsten Tag aufgehoben.

  8. Wählen Sie ggf. Kontakt oder Interessent aus.
  9. Klicken Sie auf die Schaltfläche Suchen neben dem entsprechenden Feld (Name, Firma, Verkaufschance, Ticket oder Unternehmen), um die Suchfunktion aufzurufen. Suchen Sie den Datensatz, den Sie mit der Aktivität verknüpfen möchten, und klicken Sie auf OK. (Wenn Sie eine persönliche Aktivität planen, fahren Sie mit dem nächsten Schritt fort.)
    Wenn Sie einen Kontakt oder Interessenten ausgewählt haben, können Sie den Mauszeiger über den Datensatznamen bewegen, um eingeschränkte Detailinformationen anzuzeigen.

    Klicken Sie auf die Schaltfläche Entfernen neben dem entsprechenden Feld, um einen Interessenten, einen Kontakt, eine Firma, eine Verkaufschance oder ein Ticket aus der Aktivität zu entfernen.

  10. Wählen Sie im Feld Priorität die Wichtigkeit der Aktivität aus.
  11. Wählen Sie im Feld Kategorie die Kategorie der Aktivität aus.
  12. Wählen Sie im Feld Organisator einen Organisator aus, wenn Sie nicht selbst der Organisator der Aktivität sind. Klicken Sie dazu auf die Schaltfläche Suchen und verwenden Sie die Suchfunktion. Nicht alle Benutzer stehen zur Verfügung. Sie können einen Benutzer nur als Organisator der Aktivität auswählen, wenn Sie Zugriff auf seinen Kalender haben.
  13. Geben Sie im Bereich Notizen ggf. Notizen zur Aktivität ein.
  14. Klicken Sie auf die Registerkarte Verfügbarkeit, um die Verfügbarkeit der an der Aktivität beteiligten Mitglieder und Ressourcen festzustellen.

    Sie können nur Mitglieder oder Ressourcen zu Anrufen und Meetings hinzufügen.
  15. Klicken Sie auf die Registerkarte Wiederkehrend, um auszuwählen, wie oft die Aktivität stattfinden soll (Einmal, Täglich, Wöchentlich, Monatlich oder Jährlich).
  16. Klicken Sie auf die Registerkarte Anlagen, um ein Dokument oder einen URL hinzuzufügen oder anzuzeigen.

    Wenn ein Aktivitätsmitglied Zugriffsrechte für den Kalender des Organisators der Aktivität hat, kann das Mitglied Anlagen hinzufügen oder bearbeiten. Ist dies nicht der Fall, kann das Mitglied Anlagen nur anzeigen, kopieren und speichern.

  17. Klicken Sie auf OK.
Bearbeiten einer Aktivität
  1. Führen Sie einen der folgenden Schritte durch:
    • Öffnen Sie die Listenansicht Aktivitäten, suchen Sie die zu bearbeitende Aktivität und klicken Sie dann auf den Hyperlink Typ der Aktivität.
    • Klicken Sie in der Ansicht Neuigkeiten auf die Registerkarte Aktivitäten und anschließend auf das Symbol des Typs der zu bearbeitenden Aktivität.
  2. Wenn Sie eine periodische Aktivität aktualisieren, wählen Sie bei entsprechender Aufforderung Bearbeiten Sie dieses Vorkommen der Aktivität aus, um nur die ausgewählte Wiederholung zu ändern, oder Bearbeiten Sie alle Vorkommen dieser Aktivität, um die Änderung auf alle Wiederholungen der Aktivität anzuwenden, und klicken anschließend auf Fortfahren.

    		3.

    Die Registerkarte Sich wiederholend steht nicht zur Verfügung, wenn Sie die Option Bearbeiten Sie dieses Vorkommen der Aktivität auswählen.
  3. Zeigen Sie die Informationen zur Aktivität an.
    Sie können den Mauszeiger über einen Kontakt, eine Firma oder einen Interessenten bewegen, um eingeschränkte Detailinformationen anzuzeigen.
  4. Klicken Sie auf OK.

    		5.

    Sie können den Benutzer einer persönlichen Aktivität nicht ändern.
