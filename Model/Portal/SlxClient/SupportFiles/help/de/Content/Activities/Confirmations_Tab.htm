Registerkarte "Bestätigungen"

Auf der Registerkarte "Bestätigungen" in der Ansicht Aktivitäten werden Meetings oder Anrufe angezeigt, zu denen Sie von anderen Benutzern eingeladen wurden. Sie können diese Aktivitäten bestätigen oder ablehnen. Wenn der Organisator für die Aktivität das Datum, die Uhrzeit oder die Dauer der Aktivität ändert, müssen Sie die geänderte Aktivität bestätigen oder ablehnen.

Ansonsten werden (wenn Sie z. B. von einem Benutzer eine Zusage für eine Aktivität erhalten oder eine Literaturanfrage abgeschlossen wurde) Einträge auf der Registerkarte Bestätigungen angezeigt, die Sie anzeigen und entfernen können.

Vorgehensweise

Anzeigen eigener Bestätigungen
  • Klicken Sie in der Ansicht Aktivitäten auf Bestätigungen.
    Grenzen Sie die Liste ggf. mithilfe der Filter ein.
Ein- und Ausblenden des Detailfensters
  1. Klicken Sie auf Details.
  2. Wählen Sie einen Datensatz aus.
    Das Detailfenster wird im unteren Teil der Listenansicht angezeigt und enthält detaillierte Informationen zum ausgewählten Datensatz.
  1. Zum Ausblenden des Detailfensters klicken Sie auf Details ausblenden.
So zeigen Sie die Zusammenfassungsansicht an
  1. Klicken Sie auf Zusammenfassung.
  2. Um zur Listenansicht zurückzukehren, klicken Sie auf Liste. Anzeigen der ZusammenfassungsansichtKlicken Sie auf Zusammenfassung.Um zur Listenansicht zurückzukehren, klicken Sie auf Liste.
Akzeptieren einer Bestätigung ohne Überprüfung der Aktivität
  • Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie in der Ansicht Zusammenfassung oder im Detailfenster auf die entsprechende Schaltfläche Akzeptieren.
    • Wählen Sie die Bestätigung aus, die Sie akzeptieren möchten, und klicken Sie im Aufgabenbereich unter Allgemeine Aufgaben auf Bestätigung akzeptieren.
    • Klicken Sie mit der rechten Maustaste auf die Bestätigung, die Sie akzeptieren möchten, und wählen Sie Bestätigung akzeptieren.
Akzeptieren einer Bestätigung nach Überprüfung der Aktivität
  1. Klicken Sie auf den Link Neu neben der zu akzeptierenden Bestätigung.
  2. Führen Sie die Schritte zum Akzeptieren einer Aktivität aus.
Ablehnen einer Bestätigung ohne Überprüfung der Aktivität
  • Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie in der Ansicht Zusammenfassung oder im Detailfenster auf die entsprechende Schaltfläche Ablehnen.
    • Wählen Sie die Bestätigung aus, die Sie ablehnen möchten, und klicken Sie im Aufgabenbereich unter Allgemeine Aufgaben auf Bestätigung ablehnen.
    • Klicken Sie mit der rechten Maustaste auf die Bestätigung, die Sie ablehnen möchten, und wählen Sie Bestätigung ablehnen.
Ablehnen einer Bestätigung nach Überprüfung der Aktivität
  1. Klicken Sie auf den Link Neu neben der zu akzeptierenden Bestätigung.
  2. Führen Sie die Schritte zum Ablehnen einer Aktivität aus.
Löschen einer Bestätigung
  1. Wählen Sie die zu löschende Aktivität aus, und klicken Sie im Aufgabenbereich unter Allgemeine Aufgaben auf Bestätigung löschen.
  2. Klicken Sie auf OK.
Löschen mehrerer Bestätigungen
  1. Wählen Sie die Bestätigungen aus, die Sie löschen möchten.
  2. Klicken Sie im Aufgabenbereich unter Allgemeine Aufgaben auf Bestätigung löschen.
  3. Klicken Sie auf OK.
Planen eines Anrufs

  1. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie im Aufgabenbereich Allgemeine Aufgaben unter Planen auf Anruf.
    • Klicken Sie im Raster mit der rechten Maustaste Planen, und klicken Sie dann auf Anruf.
  2. Befolgen Sie die Anweisungen zum Planen eines Anrufs.
Planen eines Meetings

  1. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie im Aufgabenbereich Allgemeine Aufgaben unter Planen auf Meeting.
    • Klicken Sie im Raster mit der rechten Maustaste auf Planen, und klicken Sie dann auf Meeting.
  2. Befolgen Sie die Anweisungen zum Planen eines Meetings.
Planen einer Aufgabe

  1. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie im Aufgabenbereich Allgemeine Aufgaben unter Planen auf Aufgabe.
    • Klicken Sie im Raster mit der rechten Maustaste auf Planen, und klicken Sie dann auf Aufgabe.
  2. Befolgen Sie die Anweisungen zum Planen einer Aufgabe.
Planen einer persönlichen Aktivität

  1. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie im Aufgabenbereich Allgemeine Aufgaben unter Planen auf Persönliche Aktivität.
    • Klicken Sie im Raster mit der rechten Maustaste auf Planen, und klicken Sie dann auf Persönliche Aktivität.
  2. Befolgen Sie die Anweisungen zum Planen einer persönlichen Aktivität.
