Abschließen einer Aktivität

Wenn Sie eine Aktivität abschließen oder einen Brief, ein Fax oder eine E-Mail schreiben, werden Details hierzu auf der Registerkarte Notizen/Historie der entsprechenden Firma, des Kontakts, des Interessenten, des Tickets oder der Verkaufschance angezeigt.

Sie können geplante Aktivitäten oder ungeplante Aktivitäten, z. B. einen nicht geplanten Anruf, abschließen.

Wenn Sie die Aktivität nicht abschließen möchten, klicken Sie auf die Schaltfläche Abbrechen, um zur vorhergehenden Ansicht zurückzukehren.

Die Kalendersicherheitseinstellungen bestimmen, wessen Aktivitäten Sie planen, bearbeiten, abschließen oder löschen können. Sie können auch Aktivitäten anzeigen, die mit Kontakten, Interessenten, Firmen, Verkaufschancen oder Tickets verknüpft sind, auf die Sie Zugriff haben.

So schließen Sie eine Aufgabe ab

  1. Führen Sie einen der folgenden Schritte durch:

    • Öffnen Sie die abzuschließende Aktivität, und klicken Sie dann auf Abschließen.

      Wenn Sie in der Ansicht Bearbeiten auf Abschließen klicken, werden alle Änderungen automatisch gespeichert und in der Ansicht Aktivität abschließen angezeigt.

    • Klicken Sie in einer der Registerkarten zu Aktivitäten auf Abschließen.
    • Klicken Sie in der Listenansicht Aktivitäten mit der rechten Maustaste auf eine Aktivität, und wählen Sie Aktivität abschließen aus.
    • Klicken Sie im Menü Planen auf Aktivitätabschließen. Im Dialogfeld Aktivität abschließen können Sie eine geplante Aktivität auswählen oder eine neue, ungeplante Aktivität erstellen und diese abschließen.
    • Wählen Sie auf der Registerkarte Alarme zu Warnungen die Aktivität aus, die Sie abschließen möchten, und klicken Sie dann auf Ausgewählte Aktivitäten abschließen.
  2. Wenn Sie eine sich wiederholende Aktivität ausgewählt haben und eine entsprechende Aufforderung erhalten, wählen Sie Dieses Vorkommen abschließen oder Alle Vorkommen abschließen aus und klicken dann auf Fortfahren.
  3. Klicken Sie im Feld Betreff auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  4. Geben Sie in das Feld Standort einen Standort ein.
  5. Ändern Sie bei Bedarf das Abschlussdatum. Zum Ändern des Datums klicken Sie auf Kalender-Schaltfläche, um das Datum auszuwählen.  
  6. Ändern Sie bei Bedarf den geplanten Termin. Zum Ändern des Datums klicken Sie auf Kalender-Schaltfläche, um das Datum auszuwählen.  

  7. Führen Sie einen der folgenden Schritte durch:

    • Aktivieren Sie das Kontrollkästchen Ohne Zeitangabe, um eine Aktivität zu erstellen, die an einem bestimmten Termin stattfindet, aber keine spezifische Uhrzeit und Dauer aufweist.
    • Klicken Sie auf Dauerden Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus.
  8. Klicken Sie im Feld Ergebnis auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  9. Verwenden Sie bei Bedarf die Schaltflächen Suchen oder Entfernen, um verknüpfte Interessenten, Kontakte, Firmen, Verkaufschancen oder Tickets zu ändern oder zu entfernen.
  10. Wählen Sie im Feld Priorität die Wichtigkeit der Aktivität aus.
  11. Wählen Sie im Feld Kategorie die Kategorie der Aktivität aus.
  12. Wählen Sie im Feld Organisator einen Organisator aus, wenn nicht Sie selbst die Aktivität abgeschlossen haben. Klicken Sie dazu auf Schaltfläche Suchen, und verwenden Sie die Suchfunktion. Nicht alle Benutzer stehen zur Verfügung. Sie können einen Benutzer nur als Organisator der Aktivität auswählen, wenn Sie Zugriff auf seinen Kalender haben.
  13. Geben Sie im Feld Notizen alle weiteren Informationen zu dieser Aktivität ein.
  14. Planen Sie ggf. eine Folgeaktivität im Feld Folgeaktivität. Aktivieren Sie das Kontrollkästchen Notizen übertragen oder Anlagen übertragen, um Notizen oder Anlagen von dieser Aktivität in die Folgeaktivität zu kopieren.
  15. Klicken Sie auf die Registerkarte Verfügbarkeit, um Mitglieder und Ressourcen anzuzeigen, die an der Aktivität beteiligt waren.
  16. Klicken Sie auf die Registerkarte Anlagen, um ein Dokument oder einen URL zu der Aktivität hinzuzufügen.
  17. Klicken Sie auf Wie geplant, um die Aktivität mit dem unter Geplant festgelegten Datum und der Uhrzeit abzuschließen. Klicken Sie alternativ auf Jetzt, um die Aktivität am aktuellem Datum und zur aktuellen Zeit abzuschließen.
    Wenn Sie eine Folgeaktivität geplant haben, wird das Dialogfeld Aktivität planen angezeigt.
