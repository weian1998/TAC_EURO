Sich wiederholende Aktivitäten, monatlich

Sie können festlegen, ob eine Aktivität Einmal, Täglich, Wöchentlich, Monatlich oder Jährlich stattfinden soll.

So planen Sie eine monatliche periodische Aktivität

  1. Planen oder bearbeiten Sie eine Aktivität.
  2. Klicken Sie auf die Registerkarte Sich wiederholende Aktivität.
  3. Wählen Sie unter Diese Aktivität findet statt... die Option Monatlich aus.
  4. Legen Sie fest, wie oft die Aktivität wiederholt werden soll.
    • Wenn die Aktivität an einem bestimmten Tag im Monat wiederholt werden soll, wählen Sie die Option Alle X Monate am aus und geben die Anzahl der Monate zwischen den Wiederholungen ein. Klicken Sie anschließend auf den Dropdown-Pfeil, um den Tag des Monats auszuwählen, an dem die Aktivität wiederholt werden soll.
      Eine Aktivität soll z. B. alle zwei Monate am 15. Tag des Monats wiederholt werden.
    • Wenn die Aktivität jeden Monat an einem bestimmten Wochentag wiederholt werden soll, wählen Sie die Option Alle X Monate am aus, geben an, wie viele Monate zwischen den Wiederholungen liegen sollen, und klicken auf den Dropdown-Pfeil, um die Woche des Monats auszuwählen. Klicken Sie anschließend auf den Dropdown-Pfeil, um den Wochentag auszuwählen, an dem die Aktivität wiederholt werden soll.
      Eine Aktivität soll z. B. jeden dritten Dienstag im Monat wiederholt werden.
    • Wenn die Aktivität nur nach Abschluss der vorherigen Aktivitätsvorkommen wiederholt werden soll, wählen Sie die Option Alle X Monate nach Abschluss aus und geben die Anzahl der Monate zwischen den Wiederholungen ein.
      Eine Aktivität soll z. B. einen Monat nach ihrem Abschluss wiederholt werden.
  5. Klicken Sie im Feld Sich wiederholende Aktivität starten auf die Kalender-Schaltfläche, wählen Sie ein Datum aus, und klicken Sie dann auf OK.

  6. Führen Sie einen der folgenden Schritte durch:

    • Geben Sie in das Feld Ende nach ein, wie oft die Aktivität sich wiederholen soll.
    • Klicken Sie unter Wiederholung beenden auf die Kalender-Schaltfläche, wählen Sie ein anderes Datum aus, und klicken Sie dann auf OK.
  7. Wenn Sie alle Aktivitätsänderungen vorgenommen haben, klicken Sie auf OK.
