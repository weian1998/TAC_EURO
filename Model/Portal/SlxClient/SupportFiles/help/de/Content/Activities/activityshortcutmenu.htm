Arbeiten mit dem Kontextmenü der Ansicht "Aktivitätenliste"

Über das Kontextmenü der Ansicht Aktivitätenliste können Sie die Aktivität anzeigen, abschließen und löschen oder den zugehörigen Kontakt bzw. Interessenten öffnen.

So öffnen Sie das Kontextmenü

  1. Öffnen Sie die Ansicht Aktivitätenliste.
  2. Klicken Sie auf die Registerkarte Eigene Aktivitäten, Alle offenen Aktivitäten, Überfällig oder Alarme, und klicken Sie dann mit der rechten Maustaste auf eine Aktivität in der Liste.

Das Kontextmenü Aktivitätenliste enthält die folgenden Optionen:

Menüpunkt Beschreibung
Aktivität öffnen Hiermit können Sie Aktivitätsdetails anzeigen oder bearbeiten.
Aktivität abschließen Dient dazu, die Ansicht Aktivität abschließen zu öffnen.
Aktivität löschen Hiermit können Sie eine Aktivität löschen.
Gehe zu Hiermit können Sie die Detailansicht für einen mit der Aktivität verknüpften Datensatz öffnen.
Planen Ermöglicht Ihnen, einen Anruf, ein Meeting, eine Aufgabe oder eine persönliche Aktivität zu planen.
