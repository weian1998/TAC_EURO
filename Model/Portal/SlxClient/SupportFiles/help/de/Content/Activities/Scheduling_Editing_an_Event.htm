Planen oder Bearbeiten eines Ereignisses

Ereignisse können geplant, bearbeitet, gelöscht und im Web-Client angezeigt werden.

Wenn Sie Intellisync for SalesLogix für die Synchronisierung von Aktivitäten und Ereignissen mit Microsoft Outlook oder Lotus Notes verwenden, werden unter folgenden Umständen Duplikate erzeugt:

  • Sie erstellen das Ereignis in Sage SalesLogix und
  • synchronisieren das Ereignis mit Outlook oder Lotus Notes und
  • ändern das Ereignis anschließend in Outlook oder Lotus Notes in ein sich wiederholendes Ereignis.

Wird das Ereignis wieder mit SalesLogix synchronisiert, wird es sich wiederholendes Ereignis ohne Zeitangabe angezeigt, doch das ursprüngliche Ereignis bleibt ebenfalls im Kalender. Um Verwirrung zu vermeiden, können Sie das ursprüngliche Ereignis manuell löschen. Weitere Informationen finden Sie im Thema Löschen eines Ereignisses.

So planen Sie ein Ereignis

  1. Klicken Sie im Menü Planen auf Ereignis.
  2. Klicken Sie im Feld Startdatum auf die Kalender-Schaltfläche, um das Datum auszuwählen, an dem das Ereignis stattfinden wird, und klicken Sie dann auf OK.

Wenn Sie das standardmäßige Startdatum nicht verwenden, wird das Enddatum automatisch an das Startdatum angepasst, sobald Sie ein neues Startdatum eingegeben haben.

  1. Klicken Sie im Feld Enddatum auf die Kalender-Schaltfläche, um das Datum auszuwählen, an dem das Ereignis stattfinden wird, und klicken Sie dann auf OK.
  2. Im Feld Ereignistypklicken Sie auf die Dropdown-Pfeil und wählen den Ereignistyp aus.
  3. Geben Sie in das Feld Standort einen Standort für das Ereignis ein.
  4. Geben Sie im Feld Beschreibung Informationen zum Ereignis ein.
  5. Im Feld Benutzer wählen Sie den Benutzer aus, für den das Ereignis geplant wird. Klicken Sie dazu auf die Schaltfläche Suchen, und verwenden Sie die Suchfunktion,

  6. Klicken Sie auf OK.

So zeigen Sie ein Ereignis in der Ansicht Aktivitäten an oder bearbeiten es

  1. Öffnen Sie die Listenansicht Aktivitäten und klicken Sie auf Ereignisse.
  2. Grenzen Sie die Liste ggf. mithilfe der Filter ein.
  3. Klicken Sie auf das Startdatum des Ereignisses, um ein Ereignis zu öffnen.
  4. Ändern Sie bei Bedarf die Ereignisinformationen.

  5. Klicken Sie auf OK.
