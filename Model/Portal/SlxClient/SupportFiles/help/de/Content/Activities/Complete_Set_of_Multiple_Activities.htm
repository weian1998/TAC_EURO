Abschließen mehrerer Aktivitäten

In der Ansicht Aktivitäten haben Sie die Möglichkeit, mehrere Aktivitäten abzuschließen. Dabei können Sie entweder mehrere Aktivitäten gleichzeitig oder jede Aktivität einzeln abschließen. Wenn Sie jede Aktivität einzeln abschließen, wird die Ansicht Aktivität abschließen geöffnet, in der Sie vor dem Abschließen der Aktivität Aktivitätsinformationen bearbeiten können. Wenn Sie eine Aktivität abgeschlossen haben, wird die Ansicht Aktivität abschließen für die nächste ausgewählte Aktivität geöffnet. Dies wird wiederholt, bis alle ausgewählten Aktivitäten abgeschlossen wurden.

Die Kalendersicherheitseinstellungen bestimmen, wessen Aktivitäten Sie planen, bearbeiten, abschließen oder löschen können. Sie können auch Aktivitäten anzeigen, die mit Kontakten, Interessenten, Firmen, Verkaufschancen oder Tickets verknüpft sind, auf die Sie Zugriff haben.

So schließen Sie mehrere Aktivitäten auf ein Mal ab

  1. Wählen Sie in der Ansicht Aktivitäten die Aktivitäten aus, die Sie abschließen möchten, und klicken Sie im Aufgabenbereich im Bereich Allgemeine Aufgaben auf Aktivität abschließen.
  2. Klicken Sie neben Ergebnis auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  3. Geben Sie in das Feld Notiz Informationen ein, die zu allen von Ihnen abgeschlossenen Aktivitäten hinzugefügt werden sollen.
  4. Klicken Sie auf Wie geplant, um die Aktivitäten zu dem Datum und der Uhrzeit abzuschließen, die Sie beim Planen der Aktivitäten eingegeben haben. Klicken Sie alternativ auf Jetzt, um die Aktivitäten am aktuellem Datum und zur aktuellen Zeit abzuschließen.

    Wenn es sich bei ausgewählten Aktivitäten um sich wiederholende Aktivitäten handelt, wird nur das bzw. die ausgewählten Vorkommen dieser sich wiederholenden Aktivität abgeschlossen.

Die Aktivitäten werden automatisch abgeschlossen und mit dem angegebenen Datum gekennzeichnet.

So schließen Sie mehrere Aktivitäten einzeln ab

  1. Wählen Sie in der Ansicht Aktivitäten die Aktivitäten aus, die Sie abschließen möchten, und klicken Sie im Aufgabenbereich im Bereich Allgemeine Aufgaben auf Aktivität abschließen.
  2. Klicken Sie neben Ergebnis auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  3. Geben Sie in das Feld Notiz Informationen ein, die zu allen von Ihnen abgeschlossenen Aktivitäten hinzugefügt werden sollen.
  4. Klicken Sie auf Einzeln.
  5. Wenn es sich bei ausgewählten Aktivitäten um sich wiederholende Aktivitäten handelt und Sie eine entsprechende Aufforderung erhalten, wählen Sie Dieses Vorkommen abschließen oder Alle Vorkommen abschließen aus und klicken dann auf Fortfahren.
  6. Führen Sie die restlichen Schritte zum Abschließen jeder Aktivität aus, bis alle Aktivitäten abgeschlossen wurden.
