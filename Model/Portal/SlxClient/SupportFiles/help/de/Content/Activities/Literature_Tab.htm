Registerkarte "Literatur"

Auf der Registerkarte "Literatur" in der Ansicht Aktivitäten werden Literaturanfragen aufgeführt, die von ausgewählten Benutzern geplant wurden. Zur Literatur zählen Kataloge, Broschüren, Zeitungsanzeigen usw.

Vorgehensweise

Anzeigen von Literaturanfragen
  1. Öffnen Sie die Ansicht Aktivitäten.
  2. Klicken Sie auf die Registerkarte Literatur.
    Daraufhin wird eine Liste aller Literaturanfragen angezeigt, auf die Sie zugreifen können.
  3. Grenzen Sie die Liste ggf. mithilfe der Filter ein.
  4. Klicken Sie auf Anzeigen, um eine Literaturanfrage zu öffnen, oder klicken Sie mit der rechten Maustaste auf eine Anfrage, und klicken Sie dann auf Literaturanfrage öffnen.
Ein- und Ausblenden des Detailfensters
  1. Klicken Sie auf Details.
  2. Wählen Sie einen Datensatz aus.
    Das Detailfenster wird im unteren Teil der Listenansicht angezeigt und enthält detaillierte Informationen zum ausgewählten Datensatz.
  1. Zum Ausblenden des Detailfensters klicken Sie auf Details ausblenden.
Anzeigen der Zusammenfassungsansicht
  1. Klicken Sie auf Zusammenfassung.
  2. Um zur Listenansicht zurückzukehren, klicken Sie auf Liste.
Planen einer Literaturanfrage

  1. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie im Aufgabenbereich im Bereich Allgemeine Aufgaben unter Planen auf Anforderung.
    • Klicken Sie mit der rechten Maustaste in das Raster, und klicken Sie anschließend auf Anforderung planen.
  2. Führen Sie die Schritte zum Planen einer Literaturanfrage aus.
