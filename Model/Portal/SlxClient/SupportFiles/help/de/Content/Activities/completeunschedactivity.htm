Abschließen einer Aktivität über das Menü

Sie können eine regelmäßig geplante oder eine ungeplante Aktivität erstellen und dann abschließen. Eine ungeplante Aktivität ist eine Aktivität, die vor ihrem Abschluss nicht geplant wurde.

Wenn Sie eine Aktivität abschließen, werden Details hierzu auf der Registerkarte Notizen/Historie der entsprechenden Firma, des Kontakts, der Verkaufschance oder des Interessenten verzeichnet.

So schließen Sie eine geplante Aktivität ab

  1. Klicken Sie im Menü Planen auf Aktivität abschließen.
  2. Wählen Sie die Option Kontakt oder Interessent aus.
  3. Abhängig von Ihrer Auswahl in Schritt 2, klicken Sie auf die Schaltfläche Suchen neben dem entsprechenden Feld (Interessent, Kontakt, Firma, Verkaufschance oder Ticket) und suchen den entsprechenden Datensatz mithilfe des Dialogfelds Suchen.

    Klicken Sie auf die Schaltfläche Entfernen neben dem entsprechenden Feld, um einen Interessenten, einen Kontakt, eine Firma, eine Verkaufschance oder ein Ticket aus der Aktivität zu entfernen.

  4. Wählen Sie die Option Geplante Aktivität abschließen aus , und klicken Sie neben der Aktivität, die Sie abschließen möchten, auf die Schaltfläche Abschließen.
    Sortieren Sie die Liste, indem Sie auf eine Spaltenüberschrift klicken. Um die Sortierrichtung zu ändern, klicken Sie erneut auf die Spaltenüberschrift.

So schließen Sie eine ungeplante Aktivität ab

  1. Klicken Sie im Menü Planen auf Aktivität abschließen.
  2. Wählen Sie die Option Kontakt oder Interessent aus. Abhängig von Ihrer Auswahl klicken Sie auf die Schaltfläche Suchen neben dem entsprechenden Feld (Interessent, Kontakt, Firma, Verkaufschance oder Ticket) und suchen den gewünschten Datensatz mithilfe des Dialogfelds Suchen.

    Klicken Sie auf die Schaltfläche Löschen neben dem entsprechenden Feld, um einen Interessenten, einen Kontakt, eine Firma, eine Verkaufschance oder ein Ticket aus der Aktivität zu entfernen.

  3. Wählen Sie die Option Neue ungeplante Aktivität zum Abschließen erstellen aus.
  4. Wählen Sie Anruf, Meeting oder Aufgabe aus, und klicken Sie dann auf Fortfahren.
  5. Klicken Sie im Feld Betreff auf die Dropdown-Pfeil und wählen Sie einen Eintrag aus der Liste aus.
