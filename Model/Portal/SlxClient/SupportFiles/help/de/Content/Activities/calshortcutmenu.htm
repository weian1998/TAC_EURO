Arbeiten mit Kontextmenüs für Kalender

Mit Kontextmenüs für Kalender können Sie eine neue Aktivität planen oder mit geplanten Aktivitäten arbeiten. Die im Kontextmenü verfügbaren Optionen variieren abhängig davon, wo Sie auf das Menü zugreifen.

So planen Sie eine Aktivität

  • Klicken Sie in einer beliebigen Kalenderansicht mit der rechten Maustaste (aber nicht auf eine Aktivität).
Menüpunkt Beschreibung
Anruf planen Öffnet das Dialogfeld Anruf planen.
Meeting planen Öffnet das Dialogfeld Meeting planen.
Aufgabe planen Öffnet das Dialogfeld Aufgabe planen.
Persönliche Aktivität planen Öffnet das Dialogfeld Persönliche Aktivität planen.
Ereignis planen Öffnet das Dialogfeld Ereignis planen.

So arbeiten Sie mit einer geplanten Aktivität

  • Klicken Sie in einer beliebigen Kalenderansicht mit der rechten Maustaste auf eine Aktivität.
Menüpunkt Beschreibung
Aktivität öffnen Öffnet das Dialogfeld Aktivität bearbeiten.
Aktivität abschließen Öffnet das Dialogfeld Aktivität abschließen.
Aktivität löschen Löscht die Aktivität.
Gehe zu Hiermit können Sie die Detailansicht für einen mit der Aktivität verknüpften Datensatz öffnen.

So arbeiten Sie mit einer abgeschlossenen Aktivität

  • Klicken Sie in einer beliebigen Kalenderansicht mit der rechten Maustaste auf eine abgeschlossene Aktivität.
Menüpunkt Beschreibung
Historie bearbeiten Öffnet das Dialogfeld Historiendetails.

So arbeiten Sie mit einem Ereignis

Menüpunkt Beschreibung
Ereignis öffnen Öffnet das Dialogfeld Aktivität bearbeiten.
Ereignis löschen Löscht das Ereignis
