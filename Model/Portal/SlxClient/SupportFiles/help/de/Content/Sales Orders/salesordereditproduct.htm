Bearbeiten von Produktinformationen

Im Dialogfeld Verkaufsauftragselement bearbeiten können Sie Produktinformationen eines Verkaufsauftrags bearbeiten.

So gehen Sie zum Bearbeiten vor

  1. Öffnen Sie die Verkaufsauftrag-Detailansicht und klicken Sie auf die Registerkarte Produkte.
  2. Klicken Sie für das Produkt, das Sie ändern möchten, auf den Hyperlink Bearbeiten.

Wählen Sie zum Bearbeiten des Produkts im Raster das gewünschte Produkt aus, und wählen Sie Informationen in der entsprechenden Spalte aus oder geben Sie sie ein.

  1. Bearbeiten Sie im Dialogfeld Bearbeiten die entsprechenden Informationen.
  • Geben Sie in das Feld Linie die gewünschten Produktlinieninformationen ein.
  • Geben Sie in das Feld Beschreibung eine Beschreibung des Produkts ein.
  • Wählen Sie im Feld Familie eine Produktfamilie aus.
  • Klicken Sie auf den Dropdown-Pfeil des Felds Preisstufe, und wählen Sie die gewünschte Preisstufe aus.
  • Geben Sie im Feld Rabatt einen prozentualen Rabatt ein. Die Zahl wird automatisch in eine Prozentangabe geändert. Beispiel: "50" wird zu "50 %". Wenn Sie ",5" eingeben, wird dies zu ",5 %".

Der angepasste Preis wird automatisch berechnet.

  • Geben Sie in das Feld Menge die Menge des Produkts ein.
  1. Klicken Sie auf OK.
