Hinzufügen oder Bearbeiten eines benutzerdefinierten Produkts

Benutzerdefinierte Produkte können auf der Registerkarte Produkte für Verkaufsaufträge hinzugefügt, bearbeitet und gelöscht werden.

Benutzerdefinierte Produkte können nicht in der Ansicht Verkaufsauftrag hinzufügen hinzugefügt werden. Fügen Sie benutzerdefinierte Produkte in der Verkaufsauftrag-Detailansicht hinzu.

So gehen Sie zum Hinzufügen oder Bearbeiten vor

  1. Klicken Sie auf die Registerkarte Produkte für Verkaufsaufträge.
  2. Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie auf die Schaltfläche Benutzerdefiniertes Produkt hinzufügen.
    • Klicken Sie auf die Schaltfläche Bearbeiten für das benutzerdefinierte Produkt, das Sie bearbeiten möchten.
  3. Legen Sie im Dialog Benutzerdefiniertes Produkt folgende Angaben fest:
    • Produkt - Geben Sie einen Namen für das benutzerdefinierte Produkt ein.
    • Beschreibung - Geben Sie eine Beschreibung des Produkts ein.
    • SKU - Geben Sie die Produkt-SKU ein.
    • Familie - Wählen Sie die Produktfamilie aus, der das Produkt zugeordnet wird.
    • Preis - Geben Sie den Preis des Produkts ein.
    • Rabatt - Geben Sie den Rabattprozentsatz für das Produkts ein.
    • Angepasster Preis - Zeigt den Basispreis des Produkts abzüglich des Rabatts an.
    • Menge - Zeigt die Anzahl der Produkte für diesen Verkaufsauftrag an.
    • Erweitert - Zeigt den errechneten Gesamtbetrag des Produktpreises multipliziert mit der Menge abzüglich aller Rabatte an. Diese Berechnung basiert auf der vom Administrator festgelegten Standardwährung.
    • Preis gesperrt - Aktivieren Sie das Kontrollkästchen, um den Produktpreis zu sperren. Zum Sperren von Preisen müssen Sie der entsprechenden Rolle zugewiesen sein. (Nur Buchhaltungsintegration)

  4. Klicken Sie auf OK.
