Erste Schritte mit dem Sage SalesLogix Web Client

Mit dem Sage SalesLogix Web Client können Sie alle Aspekte des Vertriebszyklus über das Web verwalten. Dieses Thema enthält Links zu häufig durchgeführten Aufgaben sowie Übersichtsthemen, um Ihnen den Einstieg in Sage SalesLogix Web Client zu erleichtern.

Je nach Ihren Zugriffsrechten und abhängig davon, ob Ihre Installation speziell angepasst wurde, ist Ihr Arbeitsbereich möglicherweise anders beschaffen.
