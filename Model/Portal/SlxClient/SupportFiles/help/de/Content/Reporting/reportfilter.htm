Filtern von Daten in Berichten

Wenn Sie Berichte ausführen, können Sie mithilfe von Filtern die einzubeziehenden Daten eingrenzen. Filter setzen sich aus einem oder mehreren Bedingungen zusammen. Sie können beispielsweise Bedingungen festlegen, um nur bestimmte Firmen oder Gruppen in einem Bericht zu erfassen. Filter lassen sich speichern, was eine schnelle Anwendung der Filter bei jeder Berichterstellung ermöglicht. Zur Fehlerbehebung können Administratoren bei Bedarf die Bedingungsformeln für Berichte anzeigen.

So erstellen Sie einen Filter

  1. Wählen Sie den Bericht aus, den Sie ausführen möchten, und klicken Sie dann auf Bearbeiten.

Die Filterbedingungen werden auf der rechten Bildschirmseite angezeigt.

  1. Wählen Sie in der Liste Abgleichen eine der folgenden Optionen aus:
    • Alle, um alle angegebenen Bedingungen abzugleichen.
    • Beliebige, um eine oder mehrere angegebene Bedingungen abzugleichen.
  1. Wählen Sie in der Liste Abgleichen anhand von eine Bedingung aus (wie unten beschrieben), und geben Sie alle noch benötigten Informationen für die Bedingung an. Klicken Sie auf die Plusschaltfläche (+), um dem Filter die Bedingung hinzuzufügen. Sie können so viele Bedingungen wie nötig hinzufügen.

    		2.

    Der Administrator wählt die Felder aus, auf denen die Berechnung der Daten- und Benutzerfilterbedingungen basiert. Wenn Sie diese Filter verwenden und nicht die gewünschten Informationen erhalten, können die zur Berechnung verwendeten Felder die Ursache dafür sein. Weitere Informationen hierzu erhalten Sie von Ihrem Systemadministrator.
    Abgleichen anhand von Kommentare
    Gruppe

    Filtert Datensätze nach einer vordefinierten Gruppe. In manchen Berichten ist das Filtern nach Gruppentyp nicht möglich. Beispielsweise wird in der Liste die Option Gruppe nicht angezeigt, wenn Sie den Berichtstyp Historie auswählen.

    Um diesen Filter zu verwenden, wählen Sie aus der Liste Gruppenname eine der Gruppen aus.
    Datumsbereich

    Filtert den Bericht nach dem ausgewählten Zeitraum bzw. nach dem Datumsbereich.

    Um diesen Filter zu verwenden, gehen Sie in der Liste Bereich wie folgt vor:

    • Wählen Sie einen Zeitraum aus.
    • Klicken Sie auf die Kalenderschaltfläche, um einen Datumsbereich auszuwählen.
    Benutzer Filtert Datensätze nach dem ausgewählten Benutzertyp oder dem Benutzernamen.
    Abfrage Filtert Datensätze auf Basis von Tabellen, Feldern, Operatoren oder Werten, die ausgewählt oder eingegeben wurden.
  1. (Optional) Zum Speichern des Filters geben Sie einen Namen in das Feld Bedingung speichern als ein und klicken dann auf Speichern.

So entfernen Sie Bedingungen aus einem Filter

  • Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie neben der zu entfernenden Bedingung auf Bedingung entfernen (rotes X).
    • Klicken Sie auf Alle löschen, um sämtliche Bedingungen zu entfernen.
