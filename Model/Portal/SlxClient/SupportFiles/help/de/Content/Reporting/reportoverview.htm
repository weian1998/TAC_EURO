Was sind Berichte?

Die Ansicht Berichte bietet Zugriff auf eine Reihe vordefinierter Datenbankabfragen, aus denen häufig verwendete Berichte für Geschäftszwecke erstellt werden können. Die speziell verfügbaren Berichte werden von Ihrem Systemadministrator festgelegt und sind in der Regel auf die Belange Ihres Geschäfts abgestimmt.

Das Cockpit bietet eine Reihe von Hilfsmitteln, mit denen Sie eine Vielzahl von Leistungs- und Analysedaten in einem Cockpit-Format anzeigen und interaktiv bearbeiten können. Die gelieferten Daten können Sie verwenden, um Einblick in die Leistungen des Unternehmens oder Einzelner zu gewinnen, die Ursachen von Leistungsproblemen zu erkennen und Korrekturmaßnahmen zu ergreifen oder strategische Entscheidungen zu treffen.

Sage SalesLogix Web Reporting bietet folgende Funktionen:

  • Berichtsfilter freigeben.
  • Berichte filtern, um die enthaltenen Daten einzugrenzen.
  • Berichte exportieren und drucken.

So öffnen Sie die Ansicht "Berichte"

  • Klicken Sie auf der Navigationsleiste auf Berichte.

Zum Ausführen von Web Reporting im nicht verbundenen Zustand gilt: der Offline Web-Client muss unter Window Server 2003 mit IIS ausgeführt werden.. Außerdem müssen der SalesLogix Web Reporting-Server und Crystal Reports Server installiert und konfiguriert sein. Sind diese Voraussetzungen nicht erfüllt, können Benutzer nicht verbundener Web-Clients das Web Reporting nur ausführen, wenn sie mit dem Web Reporting-Serverhost verbunden sind. Die Berichte, die diese Benutzer angezeigt bekommen, stellen Daten aus der Hostdatenbank und nicht aus ihrer Remote-Datenbank dar.
