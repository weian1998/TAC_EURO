Verfassen und Senden einer Faxnachricht

Im Folgenden wird erläutert, wie eine Faxnachricht an einen einzelnen Kontakt oder Interessenten verfasst und gesendet wird. Wenn Sie eine Faxnachricht an mehrere Kontakte oder Interessenten senden möchten, verwenden Sie die Funktion Seriendruck.

Wenn das Menü Verfassen nicht zur Verfügung steht, müssen Sie die Sage SalesLogix-Desktopintegration herunterladen. Sie können die Installation der Sage SalesLogix-Desktopintegration vom Web Client-Anmeldebildschirm aus durchführen. 

So verfassen und senden Sie eine Faxnachricht:

  1. Zeigen Sie im Menü Verfassen auf Fax aus Vorlage und wählen Sie eine Vorlage aus der Liste mit den zuletzt verwendeten Vorlagen aus, oder klicken Sie auf Weitere Vorlagen, um eine andere Vorlage auszuwählen. Wenn Sie aufgefordert werden, die Vorlage der Liste mit den zuletzt verwendeten Vorlagen hinzuzufügen, klicken Sie auf Ja, um die Vorlage hinzuzufügen, andernfalls auf Nein.

    Wenn auf dem Rechner kein kompatibler Fax-Dienst eingerichtet wurde, ist die Option Weitere Vorlagen deaktiviert.

  2. Der nächste Schritt hängt davon ab, wo sich der Fokus in Sage SalesLogix Web Client befindet.

    • Wenn der Fokus auf einem bestimmten Kontakt oder Interessenten liegt, wird der Seriendruck fortgesetzt.
    • Ist zurzeit kein bestimmter Kontakt oder Interessent aktiv, verwenden Sie das Dialogfeld Kontakt auswählen bzw. Interessent auswählen, um einen Kontakt oder Interessenten auszuwählen.
    • Wenn Sie den Fokus auf eine Firma oder eine Verkaufschance gesetzt haben, werden Sie im Dialogfeld Kontakt auswählen aufgefordert, einen Kontakt auszuwählen.
    • Liegt sich der Fokus auf einem Ticket, wird das Dokument mit dem verbundenen Kontakt zusammengeführt.

  3. Im Dialogfeld Adresstyp auswählen wird angezeigt, ob der Kontakt über mehr als nur eine Adresse verfügt und ob die Vorlage Seriendruckfelder aus der Adresstabelle enthält.
  4. Bearbeiten Sie die Vorlage bei Bedarf im Dialogfeld Fax-Editor.
  5. Schließen Sie den Fax-Editor.
  6. Geben Sie einen Betreff in das Feld Betreff im Dialogfeld Faxoptionen ein.
  7. Führen Sie im Feld Deckblatt einen der folgenden Schritte aus:
    • Behalten Sie das nun angezeigte Standarddeckblatt bei.
    • Klicken Sie auf den Pfeil und anschließend auf ein anderes Deckblatt.
    • Klicken Sie auf den Pfeil und anschließend auf Kein Deckblatt.
  1. Um eine Nachricht auf dem Deckblatt einzufügen, führen Sie eine der folgenden Aktionen durch:
    • Kurze Nachrichten können in das Feld Nachrichten eingegeben werden.
    • Klicken Sie bei einer längeren Mitteilung auf Nachricht. Geben Sie im Dialog Nachricht bearbeiten die Mitteilung ein, und klicken Sie dann auf OK.
  1. (Optional) Um den standardmäßigen Versandtermin der Faxnachricht zu ändern, klicken Sie auf Übermittlungsoptionen. Legen Sie die gewünschten Übermittlungsoptionen fest und klicken Sie auf OK.
  2. Klicken Sie auf Senden.

  3. Im Dialogfeld Aktivität abschließen können Sie die Historieninformationen bearbeiten, Notizen schreiben oder eine Folgeaktivität planen. Klicken Sie anschließend auf Abschließen.

  4. Wenn Sie eine Folgeaktivität geplant haben, füllen Sie das Dialogfeld Aktivität planen aus und klicken Sie auf Speichern..
