Drucken von Adressetiketten oder Briefumschlägen

Sie können mit der Seriendruckfunktion Adressetiketten und Umschläge erstellen und drucken. Sie können Adressetiketten oder Briefumschläge für einen oder für mehrere Kontakte oder Interessenten erstellen.

Wenn das Menü Verfassen nicht zur Verfügung steht, müssen Sie die Sage SalesLogix-Desktopintegration herunterladen. Sie können die Installation der Sage SalesLogix-Desktopintegration vom Web Client-Anmeldebildschirm aus durchführen. 
So erstellen und drucken Sie Adressetiketten oder Briefumschläge
  1. Zeigen Sie im Menü Verfassen auf Adressaufkleber.
  2. Wählen Sie im Bereich Zusammenführen mit eine der optionalen Quellen aus.
      • Aktueller Datensatz - Wählen Sie diese Option, um den aktuell aktiven Datensatz auszuwählen. Die Quelldaten können einen Kontakt für den aktuell aktiven Firmen, Kontakt- oder Verkaufschancen-Datensatz oder einen Interessenten enthalten.
      • Aktuelle Gruppe - Wählen Sie diese Option, um die aktuell aktive Kontakt-, Firmen-, Verkaufschancen- oder Interessenten-Gruppe auszuwählen. Quelldaten enthalten Kontaktnamen, Firmennamen, Namen von Interessenten und Adressen.
      • Bestimmte Gruppe - Wählen Sie eine Gruppe in der Liste aus. Quelldaten sind Kontaktnamen, Firmennamen, Namen von Interessenten und Adressen der Gruppe.
      • Spezifisch - Wählen Sie in der Liste eine Firma oder Verkaufschance aus, und suchen Sie anschließend einen Datensatz über die Suchfunktion. Firma - Quelldaten sind Kontaktnamen, Firmenname und Adressen. Verkaufschance - Quelldaten sind Kontaktnamen, Firmenname und Adressen im Zusammenhang mit der Verkaufschance.
  3. Wählen Sie gegebenenfalls NICHT an mit "Keine Informationen erwünscht" markierte Kontakte senden aus.
  4. Zur Schätzung der Anzahl der benötigten Adressetiketten klicken Sie auf Anzahl, um statistische Informationen zum Seriendruck anzuzeigen.
  5. Wählen Sie im Bereich Etikettentyp ein Etikett oder einen Briefumschlag aus.
  6. Klicken Sie im Report Viewer auf die Schaltfläche Drucken.
  7. Konfigurieren Sie Ihre Druckereinstellungen, legen Sie Etiketten oder Briefumschläge in den ausgewählten Drucker ein und klicken Sie anschließend auf OK.
So drucken Sie ein Adressetikett oder einen Umschlag für einen Brief
  1. Gehen Sie wie unter Verfassen eines Briefs beschrieben vor.
  2. Klicken Sie im Dialogfeld Adressetiketten erstellen auf Ja.
  3. Klicken Sie auf den Dropdown-Pfeil des Felds Etikettentyp und wählen Sie ein Etikett oder einen Briefumschlag aus der Liste aus.
  4. Klicken Sie im Report Viewer auf die Schaltfläche Drucken.
  5. Konfigurieren Sie Ihre Druckereinstellungen, legen Sie Etiketten oder Briefumschläge in den ausgewählten Drucker ein und klicken Sie anschließend auf OK.
So drucken Sie Adressetiketten oder Umschläge für den Seriendruck
  1. Gehen Sie wie unter Verwenden der Seriendruckfunktion beschrieben vor.
  2. Wählen Sie auf der Registerkarte Seriendruckoptionen im Bereich Ausgabe an die Option Drucker.
  3. Wählen Sie Adressetiketten erstellen und klicken Sie auf Etiketten konfigurieren.
  4. Wählen Sie im Bereich Etikettentyp ein Etikett oder einen Briefumschlag aus.
  5. Klicken Sie auf OK.
  6. Wenn Sie einen anderen Drucker auswählen möchten, klicken Sie auf Druckereigenschaften, wählen Sie einen Drucker aus und klicken Sie auf OK.
  7. Klicken Sie nach dem Festlegen der gewünschten Seriendruck- und Historienoptionen auf Seriendruck.
  8. Wenn die Vorlage eine Adressauswahl erforderlich macht oder der Kontakt über mehr als eine Adresse verfügt, wählen Sie im Dialogfeld Adresstyp auswählen eine Adresse aus und klicken Sie auf OK.
  9. Wenn Sie dazu aufgefordert werden, legen Sie Etiketten oder Briefumschläge in den Drucker ein und klicken anschließend auf OK.
