Bearbeiten der Seriendruckausgabe vor dem Versand

Sie können das Seriendruckdokument für einzelne Empfänger vor dem Versand bearbeiten. Beispielsweise können Sie einem einzelnen Brief persönliche Grüße hinzufügen. Diese Option finden Sie auf der Registerkarte Seriendruckoptionen des Seriendrucks.

So bearbeiten Sie die Seriendruckausgabe vor dem Versand:

  1. Klicken Sie im Menü Verfassen auf Seriendruck.
  2. Füllen Sie auf der Registerkarte Seriendruckoptionen die Felder Vorlage, Zusammenführen mit und Ausgabe an aus.
  3. Wählen Sie im Bereich Seriendruckoptionen die erste Option mit der Bezeichnung Einzelnes <Dokument> vor dem Senden bearbeiten aus.
  4. Füllen Sie je nach Bedarf die anderen Informationsfelder aus und klicken Sie dann auf Seriendruck.

Die Seriendruck-Engine zeigt den Fortschritt an und Sie müssen ggf. eine Kontaktadresse auswählen.

  1. Wählen Sie im Dialogfeld Seriendruckdokumente bearbeiten das zu bearbeitende Dokument aus und klicken Sie dann auf Öffnen.
  2. Nehmen Sie im Editor die gewünschten Änderungen vor, oder fügen Sie dem Dokument einen persönlichen Gruß hinzu.
  3. Klicken Sie auf der Symbolleiste auf Speichern und schließen.
  4. (Optional) Wählen Sie ein weiteres Dokument aus, klicken Sie auf Öffnen und wiederholen Sie die Schritte 6 und 7 für jedes Dokument.
  5. Wenn alle Dokumente bearbeitet wurden, klicken Sie auf Fertig.

Die zusammengeführten Dokumente werden dann gemäß der unter Ausgabe an ausgewählten Option verarbeitet.
