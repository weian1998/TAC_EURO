Arbeiten mit dem Menü "Verfassen"

Über das Menü Verfassen von Sage SalesLogix können Sie einen Seriendruck ausführen oder eine E-Mail-Nachricht senden. Sie können an einen einzelnen Kontakt oder Interessenten oder an eine Gruppe von Kontakten oder Interessenten schreiben.

Wenn das Menü Verfassen nicht zur Verfügung steht, müssen Sie die Sage SalesLogix-Desktopintegration herunterladen. Sie können die Installation der Sage SalesLogix-Desktopintegration vom Web Client-Anmeldebildschirm aus durchführen. 

Verwenden Sie das Menü Verfassen, um Folgendes durchzuführen:

  • Schreiben einer E-Mail-Nachricht an einen Kontakt oder Interessenten
  • Schreiben einer E-Mail, eines Briefs oder eines Fax mithilfe einer Vorlage
  • Verwenden der Seriendruck-Engine zum Senden von Korrespondenz an eine Gruppe von Kontakten, Interessenten, Firmen usw.
  • Erstellen und Drucken von Adressetiketten oder Briefumschlägen
  • Erstellen, Bearbeiten und Löschen von Vorlagen

Die Menüoptionen zum Verfassen von vorlagenbasierten E-Mails, Briefen und Faxen enthalten die zuletzt verwendeten Vorlagen. Die Anzahl der anzuzeigenden Einträge wird von Ihrem Systemadministrator bei der Konfiguration der SalesLogix-Optionen eingestellt.

Wenn eine Liste von Vorlagen abgeblendet angezeigt wird, können Sie auf diese Menüoption evtl. nicht zugreifen. Wenden Sie sich an Ihren Systemadministrator, wenn Sie eine Zugriffsberechtigung benötigen.

Überlegungen zum Seriendruck

  • Microsoft Word ist bereits geöffnet: Wenn bereits eine Instanz von Word geöffnet ist, werden Sie aufgefordert, Word zu schließen, bevor der Seriendruck fortgesetzt werden kann.
  • Ausgabe von E-Mails im HTML-Format: Beim Seriendruck wird die HTML-Formatierung von Microsoft Internet Explorer für E-Mail-Nachrichten verwendet. Für Empfänger, die keine HTML-Nachrichten lesen können, sollten Nachrichten als Multi-Part MIME-Nachrichten versendet werden. Darüber hinaus zeigt Word auf HTML-Seiten auch keinen Nachrichtenkopf oder -fuß und auch keine Grafiken an, sondern speichert die Daten.
  • Seriendruck mit Inhaltsverzeichnis: Der Seriendruck mit einer Vorlage mit Inhaltsverzeichnis wird nicht unterstützt. Es empfiehlt sich, das Inhaltsverzeichnis nach Abschluss des Seriendrucks zu erstellen. Verwenden Sie die Option Seriendruckausgabe vor dem Senden bearbeiten, um das Inhaltsverzeichnis nachträglich hinzuzufügen.
  • Positionieren von Seriendruckfeldern in Textfeldern: Sie können Sage SalesLogix-Seriendruckfelder nicht in Textfeldern positionieren. Der Seriendruck wird nicht ordnungsgemäß verarbeitet.
