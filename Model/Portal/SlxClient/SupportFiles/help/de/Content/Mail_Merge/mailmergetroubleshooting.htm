Fehlerbehebung bei der Desktopintegration - Seriendruck

Bei der Desktopintegration und Seriendruckfunktion können die folgenden Probleme auftreten.

Problem:

"Der Seriendruckdienst konnte nicht geladen werden. Dieser Fehler kann auftreten, wenn die Sage SalesLogix-Desktopintegration nicht bzw. falsch installiert wurde.

Mögliche Ursachen und Lösungen:

  • Das Desktopintegrationsmodul ist nicht installiert.
  • Sie müssen das Desktopintegrationsmodul installieren.
  • Das Desktopintegrationsmodul wurde installiert, bevor Ihr Browser installiert wurde.
  • Sie haben Ihren aktuellen Browser nach dem Desktopintegrationsmodul installiert. Sie müssen in Ihrem aktuellen Browser das Desktopintegrationsmodul neu installieren.

    		•

    Dies kann passieren, wenn Sie Firefox installiert haben, nachdem Sie das Desktopintegrationsmodul mithilfe von Internet Explorer installiert haben.

Problem:

"Fehler bei GetMailMergeService(). Dieser Fehler kann auftreten, wenn Sage SalesLogix-Desktopintegration nicht bzw. falsch installiert wurde. Der angegebene Klassenname kann keinem COM-Objekt zugeordnet werden. Details: Name=Fehler; Zeile=257; Quelle=sage-mailmerge-loader-debug.js.”

Mögliche Ursachen und Lösungen:

  • Eine der DDL-Dateien der Seriendruckfunktion wurde nicht ordnungsgemäß installiert.
  • Sie müssen das Desktopintegrationsmodul erneut installieren.

Problem:

"Fehler bei Sage.MailMergeService.PopulateWriteMenu(). Dieser Fehler kann auftreten, wenn Sage SalesLogix-Desktopintegration nicht bzw. falsch installiert wurde. Der angegebene Klassenname kann keinem COM-Objekt zugeordnet werden. Details: Name=SyntaxError; Nummer=-2146827287."

Mögliche Ursachen und Lösungen:

  • Eine der DDL-Dateien der Seriendruckfunktion wurde nicht ordnungsgemäß installiert.
  • Sie müssen das Desktopintegrationsmodul erneut installieren.

Problem:

"Fehler bei ExecuteWriteAction(). Das Objekt unterstützt diese Eigenschaft oder Methode nicht ."

Mögliche Ursachen und Lösungen:

  • Ihre Sage SalesLogix Web Client-Website ist keine vertrauenswürdige Website.
  • Fügen Sie die Sage SalesLogix Web Client-Website der Liste der vertrauenswürdigen Websites Ihres Internetbrowsers hinzu. Weitere Informationen dazu finden Sie Hilfesystem des Internetbrowsers.
  • Ihr Browser wird im geschützten Modus ausgeführt.
  • Bei Verwenden von Internet 8.0 müssen Sie den geschützten Modus deaktivieren. Weitere Informationen dazu finden Sie Hilfesystem des Internetbrowsers.

Problem:

"In dem Skript auf dieser Seite ist ein Fehler aufgetreten. Fehler: 'Ext.ux.data.Buffered.Json.Reader' ist NULL oder kein Objekt."

Mögliche Ursachen und Lösungen:

  • Ihre Sage SalesLogix Web Client-Website ist keine vertrauenswürdige Website.
  • Fügen Sie die Sage SalesLogix Web Client-Website der Liste der vertrauenswürdigen Websites Ihres Internetbrowsers hinzu. Weitere Informationen dazu finden Sie Hilfesystem des Internetbrowsers.
  • Ihr Browser wird im geschützten Modus ausgeführt.
  • Bei Verwenden von Internet 8.0 müssen Sie den geschützten Modus deaktivieren. Weitere Informationen dazu finden Sie Hilfesystem des Internetbrowsers.
