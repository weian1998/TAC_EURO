Verfassen und Senden von Briefen und Memos

Im Folgenden wird erläutert, wie ein Brief an einen einzelnen Kontakt oder Interessenten verfasst wird. Wenn Sie einen Brief oder ein Memo an mehrere Kontakte oder Interessenten senden möchten, verwenden Sie die Funktion Seriendruck.

Wenn das Menü Verfassen nicht zur Verfügung steht, müssen Sie die Sage SalesLogix-Desktopintegration herunterladen. Sie können die Installation der Sage SalesLogix-Desktopintegration vom Web Client-Anmeldebildschirm aus durchführen. 

So verfassen Sie einen Brief:

  1. Führen Sie im Menü Verfassen einen der folgenden Schritte aus:
    • Zeigen Sie auf Brief mit Vorlage und wählen Sie dann eine Vorlage aus der Liste der zuletzt verwendeten Vorlagen aus.
    • Klicken Sie auf Weitere Vorlagen, wenn Sie eine andere Vorlage auswählen möchten. Wenn Sie eine bestimmte Vorlage zum ersten Mal auswählen, wird im Dialogfeld Vorlage zu Menü hinzufügen eine Eingabeaufforderung angezeigt. (Optional) Wählen Sie Ja aus, um die Vorlage zu Ihrem vorhandenen Menü Brief basierend auf Vorlage... hinzuzufügen.

  2. Der nächste Schritt hängt davon ab, wo sich der Fokus in Sage SalesLogix Web Client befindet.

    • Wenn der Fokus auf einem bestimmten Kontakt oder Interessenten liegt, wird der Seriendruck fortgesetzt.
    • Ist zurzeit kein bestimmter Kontakt oder Interessent aktiv, verwenden Sie das Dialogfeld Kontakt auswählen bzw. Interessent auswählen, um einen Kontakt oder Interessenten auszuwählen.
    • Wenn Sie den Fokus auf eine Firma oder eine Verkaufschance gesetzt haben, werden Sie im Dialogfeld Kontakt auswählen aufgefordert, einen Kontakt auszuwählen.
    • Liegt sich der Fokus auf einem Ticket, wird das Dokument mit dem verbundenen Kontakt zusammengeführt.

  3. Im Dialogfeld Adresstyp auswählen wird angezeigt, ob der Kontakt über mehr als nur eine Adresse verfügt und ob die Vorlage Seriendruckfelder aus der Adresstabelle enthält.
  4. (Optional) Im Dialogfeld Brief-Editor können Sie zusätzliche Informationen eingeben oder Änderungen an der Vorlage vornehmen.
  5. Es stehen folgende Möglichkeiten zur Auswahl:
    • einen Brief oder ein Memo in einer Datei speichern. Wählen Sie aus dem Menü Datei eine der Optionen zum Speichern aus.
    • einen Brief oder ein Memo drucken. Klicken Sie auf die Schaltfläche Drucken.
  6. Klicken Sie auf Speichern und schließen, um den Brief-Editor zu schließen.
  7. Klicken Sie in der Eingabeaufforderung Möchten Sie einen Adressaufkleber erstellen? auf Ja, wenn Sie zu dem Brief ein Adressetikett oder einen Umschlag drucken möchten, oder klicken Sie auf Nein.

  8. Im Dialogfeld Aktivität abschließen können Sie die Historieninformationen bearbeiten, Notizen schreiben oder eine Folgeaktivität planen. Klicken Sie anschließend auf Abschließen.

  9. Wenn Sie eine Folgeaktivität geplant haben, füllen Sie das Dialogfeld Aktivität planen aus und klicken Sie auf Speichern..
