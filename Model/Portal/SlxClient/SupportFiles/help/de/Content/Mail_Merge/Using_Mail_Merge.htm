Die Seriendruckfunktion

Sie können die Seriendruckfunktion verwenden, wenn Sie mit einer Gruppe von Kontakten, Firmen, Verkaufschancen oder Interessenten kommunizieren möchten. Mit der Seriendruckfunktion lässt sich der Vorgang vereinfachen, vielen Personen dieselben Informationen zu senden, indem Sie die Informationen mit einer Vorlage zusammenführen. Sie können einen Seriendruck in eine Datei ausgeben, als E-Mail oder als Fax senden oder das Seriendruckdokument drucken.

  • Wenn das Menü Verfassen nicht zur Verfügung steht, müssen Sie die Sage SalesLogix-Desktopintegration herunterladen. Sie können die Installation der Sage SalesLogix-Desktopintegration vom Web Client-Anmeldebildschirm aus durchführen. 

  • Wenn die von Ihnen gewählte Vorlage für den Seriendruck über ein Inhaltsverzeichnis verfügt, das Word-Seriendruckfelder enthält, funktioniert der Seriendruck nicht ordnungsgemäß. Es wird empfohlen, die Funktion Einzelnes <Dokument> vor dem Senden bearbeiten zu wählen und das Inhaltsverzeichnis nach Abschluss des Seriendrucks hinzuzufügen.
  • Wird auf der Registerkarte Seriendruckoptionen eine Ausgabeoption gewählt, werden nur gültige Optionen zur Auswahl gestellt. Wurde beispielsweise kein E-Mail-System konfiguriert, steht E-Mail als Ausgabeoption nicht zur Verfügung.

So verwenden Sie die Seriendruckfunktion:

  1.  Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie im Menü Verfassen auf Seriendruck.
    • Klicken Sie im Aufgabenbereich in bestimmten Detailansichten auf Seriendruck.
  2. Wählen Sie im Dialogfeld Seriendruck eine Vorlage aus.
  3. Klicken Sie auf die Registerkarte Seriendruckoptionen, um die Seriendruckoptionen einzustellen.
  4. Klicken Sie auf die Registerkarte Historienoptionen, um die Historienoptionen einzustellen.
  5. Wenn Sie auf der Registerkarte Seriendruckoptionen als Ausgabeoption Drucker gewählt haben und Adressetiketten drucken möchten, vergewissern Sie sich, dass sich Papier und Etiketten im Drucker befinden.
  6. Klicken Sie auf Zusammenführen.
  7. Wenn Sie auf der Registerkarte Seriendruckoptionen die Option Vorlage für alle Kontakte bearbeiten gewählt haben, nehmen Sie im Editor-Dialogfeld die Änderungen an der Vorlage vor und klicken dann auf Speichern und schließen.
  8. Wenn die Vorlage eine Adressauswahl erforderlich macht oder der Datensatz über mehr als eine Adresse verfügt, wählen Sie im Dialogfeld Adresstyp auswählen eine Adresse aus.
  9. Wenn Sie unter Seriendruckoptionen die Option Einzelnes <Dokument> vor dem Senden bearbeiten gewählt haben, verwenden Sie das Dialogfeld Seriendruckdokumente bearbeiten, um einen Eintrag auszuwählen und zu bearbeiten. Wenn Sie mit Ihren Änderungen fertig sind, klicken Sie auf Fertig.
  10. Wenn Sie die Meldung erhalten, dass eine gewisse Anzahl an Zusammenführungen fehlschlagen wird, können Sie den Seriendruck fortsetzen oder abbrechen. Wenn Sie den Vorgang fortsetzen, wird bei Beendigung des Prozesses eine Fehlermeldung angezeigt, die Ihnen ermöglicht, die Gruppe Seriendruckprobleme einzusehen. Wenn Sie auf Ja klicken, wird die Gruppe Seriendruckprobleme geöffnet. Wenn Sie auf Nein klicken, wird den Dialog geschlossen, und Sie kehren zu der Ansicht zurück, von der aus Sie den Seriendruck initiiert haben.
