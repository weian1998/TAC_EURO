Verfassen und Senden einer E-Mail-Nachricht

E-Mail-Nachrichten an einzelne Kontakte oder Interessenten können Sie mit den Optionen E-Mail schreiben oder E-Mail aus Vorlage verfassen. Wenn Sie eine E-Mail-Nachricht an mehr als einen Kontakt oder Interessenten senden möchten, verwenden Sie die Funktion Seriendruck.

Um Die Funktionen zum Verfassen von E-Mail verwenden zu können, müssen Sie MS Outlook auf Ihrem Rechner installiert haben.

Wenn das Menü Verfassen nicht zur Verfügung steht, müssen Sie die Sage SalesLogix-Desktopintegration herunterladen. Sie können die Installation der Sage SalesLogix-Desktopintegration vom Web Client-Anmeldebildschirm aus durchführen. 
So schreiben Sie eine E-Mail ohne Vorlage
  1. Führen Sie einen der folgenden Schritte durch:
    • Zeigen Sie im Menü Verfassen auf E-Mail.
    • Klicken Sie auf die Schaltfläche E-Mail neben der E-Mail-Adresse des Kontakts oder Interessenten.
  2. Wenn die E-Mail-Funktion aktiviert ist, Sie sich aber noch nicht angemeldet haben, klicken Sie im Meldungsfeld auf Ja, um sich anzumelden.

  3. Der nächste Schritt hängt davon ab, wo sich der Fokus in Sage SalesLogix Web Client befindet.

    • Wenn der Fokus auf einem bestimmten Kontakt oder Interessenten liegt, wird der Seriendruck fortgesetzt.
    • Ist zurzeit kein bestimmter Kontakt oder Interessent aktiv, verwenden Sie das Dialogfeld Kontakt auswählen bzw. Interessent auswählen, um einen Kontakt oder Interessenten auszuwählen.
    • Wenn Sie den Fokus auf eine Firma oder eine Verkaufschance gesetzt haben, werden Sie im Dialogfeld Kontakt auswählen aufgefordert, einen Kontakt auszuwählen.
    • Liegt sich der Fokus auf einem Ticket, wird das Dokument mit dem verbundenen Kontakt zusammengeführt.

  4. Füllen Sie das Feld Betreff und das Nachrichtenfeld des Fensters für die neue E-Mail aus.
  5. (Optional) Soll gemeinsam mit der E-Mail-Nachricht eine Anlage versendet werden, klicken Sie auf Einfügen und dann auf eine der Menüoptionen. Soll eine Datei angehängt werden, finden Sie weitere Einzelheiten unter Info über E-Mail-Anlagen.
  6. Zum Versenden Ihrer E-Mail führen Sie einen der folgenden Schritte durch:
    • Klicken Sie auf Mit SLX senden. Wenn Sie eine Datei angehängt haben und eine Kopie speichern möchten, klicken Sie im Bestätigungsfeld auf Ja. Der E-Mail-Eintrag wird in die Historie des Kontakts übernommen, und die Datei wird auf der Registerkarte "Anlagen" aufgeführt.
    • Klicken Sie auf Senden. Die Nachricht wird nicht automatisch in die Historie des Kontakts übernommen; Sie können die Nachricht jedoch mit der Maus von Outlook nach Sage SalesLogix ziehen und dort ablegen.

      		•

      Grafikdateien, die kleiner als die vom Administrator festgelegte Mindestgröße sind, werden nicht als separate Anlagen gespeichert.

  7. Im Dialogfeld Aktivität abschließen können Sie die Historieninformationen bearbeiten, Notizen schreiben oder eine Folgeaktivität planen. Klicken Sie anschließend auf Abschließen.

  8. Wenn Sie eine Folgeaktivität planen möchten, füllen Sie das Dialogfeld für den gewünschten Typ der Folgeaktivität aus.
So schreiben Sie eine E-Mail mithilfe einer Vorlage
  1. Zeigen Sie im Menü Verfassen auf E-Mail mit Vorlage und klicken Sie dann auf den Namen einer Vorlage oder klicken Sie auf Weitere Vorlagen, um eine andere Vorlage auszuwählen. Wenn Sie aufgefordert werden, die Vorlage der Liste mit den zuletzt verwendeten Vorlagen hinzuzufügen, klicken Sie auf Ja, um die Vorlage hinzuzufügen, andernfalls auf Nein.
  2. Wenn E-Mail aktiviert ist, Sie sich aber noch nicht angemeldet haben, klicken Sie im Meldungsfeld auf Ja, um sich anzumelden.

  3. Der nächste Schritt hängt davon ab, wo sich der Fokus in Sage SalesLogix Web Client befindet.

    • Wenn der Fokus auf einem bestimmten Kontakt oder Interessenten liegt, wird der Seriendruck fortgesetzt.
    • Ist zurzeit kein bestimmter Kontakt oder Interessent aktiv, verwenden Sie das Dialogfeld Kontakt auswählen bzw. Interessent auswählen, um einen Kontakt oder Interessenten auszuwählen.
    • Wenn Sie den Fokus auf eine Firma oder eine Verkaufschance gesetzt haben, werden Sie im Dialogfeld Kontakt auswählen aufgefordert, einen Kontakt auszuwählen.
    • Liegt sich der Fokus auf einem Ticket, wird das Dokument mit dem verbundenen Kontakt zusammengeführt.

  4. Wenn Sie in Schritt 1 die Option Weitere Vorlagen ausgewählt haben, wählen Sie nun im Dialogfeld Vorlagen verwalten eine Vorlage aus und klicken auf OK. Wählen Sie bei Eingabeaufforderung im Dialogfeld Adresstyp auswählen eine Adresse aus.
    Die E-Mail-Adresse des Kontakts wird im Feld An angezeigt. Das Feld Betreff enthält die Informationen, die in der Vorlage gespeichert wurden, und es werden die Anlagen der Vorlage aufgeführt.
  5. (Optional) Im Textfeld können Sie zusätzliche Informationen eingeben oder die Standardunterschrift ändern.
  6. (Optional) Soll gemeinsam mit der E-Mail-Nachricht eine Anlage versendet werden, klicken Sie auf Einfügen und dann auf eine der Menüoptionen. Soll eine Datei angehängt werden, finden Sie weitere Einzelheiten unter Info über E-Mail-Anlagen.
  7. Zum Versenden Ihrer E-Mail führen Sie einen der folgenden Schritte durch:
    • Klicken Sie auf Mit SLX senden. Wenn Sie eine Datei angehängt haben und eine Kopie hiervon gespeichert werden soll, klicken Sie im Bestätigungsfeld auf Ja. Der E-Mail-Eintrag wird in die Historie des Kontakts übernommen, und die Datei wird auf der Registerkarte "Anlagen" aufgeführt.
    • Klicken Sie auf Senden. Die Nachricht wird nicht automatisch in die Historie des Kontakts übernommen; Sie können die Nachricht jedoch mit der Maus von Outlook nach Sage SalesLogix ziehen und dort ablegen.

      		•

      Grafikdateien, die kleiner als die vom Administrator festgelegte Mindestgröße sind, werden nicht als separate Anlagen gespeichert.

  8. Im Dialogfeld Aktivität abschließen können Sie die Historieninformationen bearbeiten, Notizen schreiben oder eine Folgeaktivität planen. Klicken Sie anschließend auf Abschließen.

  9. Wenn Sie eine Folgeaktivität planen möchten, füllen Sie das Dialogfeld für den gewünschten Typ der Folgeaktivität aus.
