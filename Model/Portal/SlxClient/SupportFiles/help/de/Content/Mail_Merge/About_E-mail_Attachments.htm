Informationen zu E-Mail-Anlagen

Beim Senden einer E-Mail-Nachricht an einen Kontakt oder Interessenten können Sie an die Nachricht eine Datei anhängen. Wenn Sie die richtigen Optionen ausgewählt haben, wird diese Datei auf der Registerkarte Anlagen des Kontakts und der Firma bzw. des Interessenten sowie im Standardordner Dokumente gespeichert. Bei Sage SalesLogix Web-Benutzern wird die Datei im Ordner Dokumente des Webservers abgelegt. Der Dateiname auf der Registerkarte Anlagen entspricht exakt dem Namen der Datei oder des Dokuments. Im Ordner Dokumente beinhaltet der Dateiname eine interne von Sage SalesLogix zugewiesene Referenznummer und den Namen des Dokuments.

Sie können Windows-Dateien oder Sage SalesLogix Library-Dateien an eine Vorlage anhängen. Wenn Sie die Vorlage mit der Seriendruckfunktion per E-Mail versenden, werden die angehängten Dateien an den/die Kontakt(e) oder Interessenten gesendet. Wenn Sie eine Vorlage kopieren, werden auch alle Anlagen in die neue Vorlage kopiert. Sie können eine Anlage aus einer Vorlage entfernen.

So hängen Sie eine Datei an eine Vorlage an
  1. Öffnen Sie den Vorlagen-Editor.
  2. Führen Sie einen der folgenden Schritte durch:
  • Wenn Sie eine Windows-Datei anhängen möchten, klicken Sie auf die Schaltfläche Datei einfügen, wechseln Sie zum Speicherort der Datei, und klicken Sie auf Datei einfügen.
  • Wenn Sie eine Sage SalesLogix Library-Datei anhängen möchten, klicken Sie auf die Schaltfläche SalesLogix Library-Datei hinzufügen. Klicken Sie im Dialogfeld SalesLogix-Bibliothek auf die Dateien und klicken Sie auf Einfügen.
  1. Sie können eine weitere Datei anhängen oder auf die Schaltfläche Speichern und schließen klicken.
So entfernen Sie eine Datei aus einer Vorlage
  1. Öffnen Sie den Vorlagen-Editor.
  2. Wählen Sie die Anlage, die Sie entfernen möchten, im Feld Anlagen aus.
  3. Führen Sie einen der folgenden Schritte durch:
  • Klicken Sie mit der rechten Maustaste, und klicken Sie auf Entfernen.
  • Drücken Sie die Taste ENTF.
  1. Sie können eine weitere Datei entfernen oder auf die Schaltfläche Speichern und schließen klicken.
