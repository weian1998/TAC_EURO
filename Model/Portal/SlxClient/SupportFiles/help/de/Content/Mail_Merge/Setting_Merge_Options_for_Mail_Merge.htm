Festlegen von Optionen für den Seriendruck

Auf der Registerkarte Seriendruckoptionen können Sie die Optionen für den Seriendruck festlegen.

So legen Sie Seriendruckoptionen fest

  1. Klicken Sie auf der Registerkarte Seriendruckoptionen im Feld Vorlage auf den Ordner, um eine Vorlage auszuwählen.
    Wählen Sie beim Auswählen von Vorlagen für Interessenten nur Vorlagen aus, die mit "(Interessent)" enden.
  2. Wählen Sie im Bereich Verbinden mit eine der SalesLogix-Quellenoptionen aus.
      • Aktueller Datensatz - Wählen Sie diese Option, um den aktuell aktiven Datensatz auszuwählen. Die Quelldaten können einen Kontakt für den aktuell aktiven Firmen, Kontakt- oder Verkaufschancen-Datensatz oder einen Interessenten enthalten.
      • Aktuelle Gruppe - Wählen Sie diese Option, um die aktuell aktive Kontakt-, Firmen-, Verkaufschancen- oder Interessenten-Gruppe auszuwählen. Quelldaten enthalten Kontaktnamen, Firmennamen, Namen von Interessenten und Adressen.
      • Bestimmte Gruppe - Wählen Sie eine Gruppe in der Liste aus. Quelldaten sind Kontaktnamen, Firmennamen, Namen von Interessenten und Adressen der Gruppe.
      • Spezifisch - Wählen Sie in der Liste eine Firma oder Verkaufschance aus, und suchen Sie anschließend einen Datensatz über die Suchfunktion. Firma - Quelldaten sind Kontaktnamen, Firmenname und Adressen. Verkaufschance - Quelldaten sind Kontaktnamen, Firmenname und Adressen im Zusammenhang mit der Verkaufschance.
  3. Wählen Sie in der Liste Ausgabe an eine der Optionen, und füllen Sie die Informationsfelder der ausgewählten Option aus. Mit der Option Ausgabe an Drucker können Sie Adressetiketten und Briefumschläge drucken.

    • E-Mail

      1. Wählen Sie in der Liste Ausgabe an die Option E-Mail aus.
        Im Feld Von wird die E-Mail-Adresse des angemeldeten Benutzers angezeigt.
      2. (Optional) Klicken Sie auf das Adressbuch, um die E-Mail-Adresse eines anderen Benutzers als Absender auszuwählen.
      3. (Optional) Klicken Sie in den Feldern Cc und Bcc auf das Adressbuch, um andere Empfänger auszuwählen.
      4. Geben Sie in das Feld Betreff die Informationen ein, die in der Betreffzeile der E-Mail angezeigt werden sollen.
      5. (Optional) Klicken Sie im Feld Anhängen auf den Pfeil neben dem Ordner, und wählen Sie den Typ der Anlage aus, der mit dem Seriendruckdokument versendet werden soll.
      6. Im Feld Format wird das Standardformat der gewählten Vorlage angezeigt; Sie können jedoch auch ein anderes Format auswählen.
      7. Die Option Kopie in 'Gesendete Objekte' speichern ist standardmäßig aktiviert, kann jedoch deaktiviert werden.
        Diese Option sorgt dafür, dass eine Kopie aller Seriendruckdokumente im Ordner Gesendete Objekte von Outlook oder des Mail-Clients gespeichert wird.

    • Fax

      1. Wählen Sie in der Liste Ausgabe an den Eintrag Fax aus.
      2. Geben Sie im Feld Betreff einen Betreff ein.
      3. Führen Sie im Feld Deckblatt einen der folgenden Schritte aus:
        • Behalten Sie das nun angezeigte Standarddeckblatt bei.
        • Klicken Sie auf den Pfeil, und wählen Sie ein anderes Deckblatt aus.
        • Klicken Sie auf den Pfeil, und wählen Sie Kein Deckblatt aus.
      4. Wenn Sie bereits auf dem Deckblatt eine Mitteilung schreiben möchten, führen Sie einen der folgenden Schritte aus:
        • Geben Sie eine kurze Mitteilung direkt in das Feld ein.
        • Klicken Sie bei einer längeren Mitteilung auf Nachricht. Geben Sie im Dialog Nachricht bearbeiten die Mitteilung ein, und klicken Sie dann auf OK.

    • Drucker

      1. Wählen Sie in der Liste Ausgabe an den Eintrag Drucker aus. Der Name des Standarddruckers wird im Feld angezeigt; Sie können jedoch auch einen anderen Drucker suchen und auswählen.
      2. (Optional) Zum Konfigurieren des Druckers klicken Sie auf Druckereigenschaften.
      3. Zum Drucken von Adressetiketten oder Briefumschlägen klicken Sie auf Adressetiketten erstellen.
      4. Zum Formatieren von Adressetiketten oder Briefumschlägen klicken Sie auf Etiketten konfigurieren.

    • Datei

      1. Wählen Sie aus der Liste Ausgabe an den Eintrag Datei aus.
      2. Klicken Sie im Feld Verzeichnis auf den Ordner, um einen Speicherort für die Datei auszuwählen.
        Die Seriendruckdatei wird wie folgt benannt: Vorlagenname (Name der ENTITÄT)DATUMUHRZEIT.DOC. Bei Kontakten oder Interessenten mit demselben Namen fügt Sage SalesLogix dem Namen eine Zahl hinzu, wenn bereits eine Datei mit dem Namen vorhanden ist.
        Beispiel:
        Standardbrief (Johannes Aberle) 07-13-2005 14.07.17PM.doc
        Standardbrief (JohannesAberle2) 09-14-2005 12.11.43PM.doc
  4. Für eine Schätzung, wie viel Papier oder wie viele Etiketten in den Drucker eingelegt werden müssen, klicken Sie auf Anzahl, um statistische Informationen zum Seriendruck anzuzeigen.
    Anhand der statistischen Informationen können Sie vor dem Seriendruck zudem erkennen, ob E-Mail-Adressen oder Faxnummern fehlen.
  5. Wählen Sie im Bereich Seriendruckoptionen eine oder mehrere der Optionen aus. Welche Optionen verfügbar sind, ist vom gewählten Ausgabetyp abhängig.
    • Einzelne <E-Mails/Briefe/Faxnachrichten> vor dem Senden bearbeiten
      • Möglicherweise möchten Sie einzelne Seriendruckdokumente vor dem Versand bearbeiten. Beispielsweise können Sie einem Brief persönliche Grüße hinzufügen. Wenn diese Option gewählt wird, wird das Dialogfeld Seriendruckdokumente bearbeiten während des Seriendrucks angezeigt, damit Sie Dokumente auswählen und deren Inhalt bearbeiten können.
    • Vorlage für alle Kontakte bearbeiten
      • Sie möchten die Vorlage dieses Seriendruckdokuments bearbeiten. Einmal angenommen, Sie haben die Basisvorlage eines Briefs ausgewählt, möchten dieser Vorlage jedoch nun weitere Seriendruckfelder hinzufügen. Wird diese Option gewählt, wird der Vorlagen-Editor während des Seriendrucks geöffnet, damit die Vorlage bearbeitet werden kann.
    • NICHT an mit "Keine Informationen erwünscht" markierte Kontakte senden
      • Diese Option ist standardmäßig aktiviert, da Sie sicher keine Seriendruckdokumente an Kontakte senden möchten, die speziell darum gebeten haben, beim Versand nicht berücksichtigt zu werden. Das Kontrollkästchen Nicht anfragen wird in der Detailansicht des Kontakts und Interessenten angezeigt.
      • Sie können diese Option deaktivieren und das Seriendruckdokument an alle Kontakte senden.
  6. Klicken Sie nach dem Festlegen der gewünschten Seriendruck- und Historienoptionen auf Seriendruck.
