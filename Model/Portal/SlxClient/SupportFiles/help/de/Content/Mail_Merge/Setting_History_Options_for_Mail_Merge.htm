Festlegen von Historienoptionen für den Seriendruck

Auf der Registerkarte Historienoptionen können Sie einen Historieneintrag für den Seriendruck erstellen und eine Folgeaktivität planen.

So legen Sie Historienoptionen fest

  1. Gehen Sie auf der Registerkarte Historienoptionen im Bereich Historie wie folgt vor:
    1. Wenn Sie das Schreiben zu der Registerkarte Anlagen jedes Datensatzes hinzufügen möchten, wählen Sie <E-Mail/Fax/Brief> an jeden <Kontakt/Interessenten> anhängen aus.
      Die Seriendruckdatei wird abhängig von der Option Zusammenführen mit, die Sie in den Seriendruckoptionen ausgewählt haben, an alle Interessenten, Kontakten, Firmen und Verkaufschancen angehängt.
      Bei Sage SalesLogix Web-Benutzern bewirkt diese Option, dass beim Seriendruck eine Warnmeldung angezeigt wird.
    2. Wenn Sie der Historie des Datensatzes einen Eintrag hinzufügen möchten, wählen Sie Historiendatensatz für jeden <Kontakt/Interessenten> hinzufügen aus.
    3. Klicken Sie im Feld Ergebnis auf die Schaltfläche mit den Auslassungspunkten, um einen Ergebnistyp in der Liste verfügbarer Aufgabenergebnis-Codes auszuwählen.
    4. Klicken Sie im Feld Betreff: auf die Schaltfläche mit den Auslassungspunkten, um einen Betrefftyp aus der Liste auszuwählen.
    5. Klicken Sie im Feld Kategorie auf die Auslassungspunkte, um einen Kategorietyp aus der Liste auszuwählen.
    6. Geben Sie im Feld Notizen die Notizen oder wichtigen Angaben ein, die Sie in die Historie aufnehmen möchten.
  2. Wählen Sie zum Planen einer Folgeaktivität Folgeaktivität planen aus, und führen Sie die folgenden Schritte aus:
    1. Wenn eine Aktivitätsbestätigungsmeldung angezeigt wird, klicken Sie auf OK.
    2. Wählen Sie in der Liste den Typ der Folgeaktivität aus.
    3. Aktivieren Sie das Kontrollkästchen Notizen übertragen, um Notizen von dieser Aktivität in die Folgeaktivität zu kopieren.
    4. Klicken Sie im Feld Startzeit auf die Kalenderschaltfläche, und wählen Sie das Datum für die Aktivität.
    5. Führen Sie einen der folgenden Schritte durch:
      • Aktivieren Sie das Kontrollkästchen Ohne Zeitangabe, um eine Aktivität zu erstellen, die an einem bestimmten Termin stattfindet, aber keine spezifische Uhrzeit und Dauer aufweist.
      • Klicken Sie auf Dauer (den Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus).
    6. Wählen Sie zum Festlegen eines Aktivitätsalarms Erinnerung aus. Wählen Sie in den Dropdown-Feldern die Anzahl der Minuten, Stunden oder Tage für die Vorlaufzeit aus. Geben Sie einen Wert von 1 bis 240 ein.
    7. Klicken Sie im Feld Betreff: auf die Schaltfläche mit den Auslassungspunkten, um einen Betrefftyp auszuwählen.
    8. Führen Sie im Bereich Notizen einen der folgenden Schritte aus:
      • Geben Sie Notizen in das Feld ein.
      • Klicken Sie auf Notizen, um ein größeres Eingabefeld für Notizen zu öffnen, geben Sie Ihre Notizen ein, und klicken Sie dann auf OK.
    9. Wählen Sie im Feld Priorität die Wichtigkeit der Aktivität aus.
    10. Klicken Sie im Feld Kategorie auf die Auslassungspunkte, um einen Kategorietyp auszuwählen.
    11. Führen Sie im Feld Organisator eine Suche durch, um den Organisator der Aktivität auszuwählen.
  3. Klicken Sie nach dem Festlegen der gewünschten Seriendruck- und Historienoptionen auf Seriendruck.
