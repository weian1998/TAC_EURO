Zuletzt verwendete Vorlagen

Mithilfe der Liste der zuletzt verwendeten Vorlagen können Sie eine der zuletzt verwendeten Vorlagen auswählen, ohne das Dialogfeld Vorlage auswählen öffnen zu müssen. Die Anzahl der zuletzt verwendeten Vorlagen, die in der Liste angezeigt werden, wird von Ihnen oder vom Systemadministrator bei der Konfiguration der Sage SalesLogix-Optionen festgelegt.

  • Wenn das Menü Verfassen nicht zur Verfügung steht, müssen Sie die Sage SalesLogix-Desktopintegration herunterladen. Sie können die Installation der Sage SalesLogix-Desktopintegration vom Web Client-Anmeldebildschirm aus durchführen. 

  • Die Liste der zuletzt verwendeten Vorlagen wird nicht automatisch aktualisiert, wenn eine Vorlage nicht mehr existiert. Wenn Sie eine bereits gelöschte Vorlage oder eine Vorlage in der Liste wählen, auf die Sie keinen Zugriff mehr haben, werden Sie durch eine Meldung informiert, dass die Vorlage aus dem Menü entfernt wird. Wenden Sie sich an Ihren Systemadministrator, wenn Sie Zugriff auf eine nicht verfügbare Vorlage benötigen.

So wählen Sie eine Vorlage aus der Liste der zuletzt verwendeten Vorlagen aus:

  1. Wählen Sie im Menü Verfassen aus, dass Sie mithilfe einer Vorlage eine E-Mail, einen Brief oder ein Fax verfassen möchten.

Um Ihnen die Arbeit zu erleichtern, wird eine Liste der zuletzt verwendeten Vorlagen angezeigt.

  1. Klicken Sie auf die Vorlage, die Sie verwenden möchten.
