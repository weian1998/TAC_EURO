Ereignisauslösende Aktionen in Sage SalesLogix

Der Systemadministrator verwaltet die Aktionen, die Ereignisse auslösen.

Zum Verwenden aller Vertriebsprozessfunktionen müssen Sie die Sage SalesLogix-Desktopintegration herunterladen. Sie können die Installation der Sage SalesLogix-Desktopintegration vom Web Client-Anmeldebildschirm aus durchführen. 
Aktion Ereignis
Keine Dieser Schritt enthält keine Aktion.
Serienbrief Startet die Bildschirme, mit denen Sie ein Dokument aus einer Vorlage erstellen können. Mit dieser Funktion können Sie einen Brief, ein Fax oder eine E-Mail-Nachricht erstellen. Die zur Verfügung stehenden Optionen hängen davon ob, welche Optionen die Person gewählt hat, die den Verkaufsprozess erstellt hat.
Basic-Skript Führt das ausgewählte Skript aus der Verkaufsprozessfamilie aus.
Formular Öffnet ein Sage SalesLogix-Formular.
Anruf Öffnet den Dialog Anruf planen. Die zur Verfügung stehenden Optionen hängen davon ob, welche Optionen die Person gewählt hat, die den Verkaufsprozess erstellt hat.
Aufgabe Öffnet den Dialog Aufgabe planen. Die zur Verfügung stehenden Optionen hängen davon ob, welche Optionen die Person gewählt hat, die den Verkaufsprozess erstellt hat.
Besprechung Öffnet den Dialog Meeting planen. Die zur Verfügung stehenden Optionen hängen davon ob, welche Optionen die Person gewählt hat, die den Verkaufsprozess erstellt hat.
Literaturanfrage Öffnet den Dialog Literaturanfrage, in dem Sie Literatur anfordern können.
Kontaktprozess Öffnet den Dialog Prozess planen.

Der Systemadministrator kann die Auswahlliste Aktivitätstyp ändern, um Aktionen hinzuzufügen oder zu entfernen, die SalesLogix-Ereignisse auslösen.
