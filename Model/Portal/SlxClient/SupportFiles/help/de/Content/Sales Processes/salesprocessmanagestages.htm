Aktualisieren von Verkaufsprozessphasen

Sie können die Phasen in einem Verkaufsprozess aktualisieren. Sie können eine Phase abschließen, auch wenn Sie noch nicht alle Schritte der Phase abgeschlossen haben. Sie können auch die aktuelle Phase ändern.

Wenn Sie eine Phase überspringen, wird die Wahrscheinlichkeit der Verkaufschance nicht aktualisiert. Damit die Wahrscheinlichkeit der Verkaufschance automatisch aktualisiert wird, müssen Sie die Phasen der Reihe nach abschließen.

So ändern Sie die aktuelle Phase

  1. Klicken Sie in der Verkaufschancen-Detailansicht auf die Registerkarte Verkaufsprozess. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  2. Klicken Sie auf den Dropdown-Pfeil neben dem Feld Phasen und wählen Sie eine Phase aus der Liste aus. Klicken Sie bei entsprechender Aufforderung auf OK.

So schließen Sie eine Phase ab

  1. Klicken Sie in der Verkaufschancen-Detailansicht auf die Registerkarte Verkaufsprozess. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  2. Klicken Sie auf den Hyperlink Aktuelle Phase neben dem Dropdown-Feld.

Das Dialogfeld Phasen verwalten wird geöffnet.

  1. Aktivieren Sie das Kontrollkästchen in der Spalte Abgeschlossen, um die Phase zu identifizieren, die Sie abschließen möchten, und schließen Sie das Dialogfeld anschließend.
