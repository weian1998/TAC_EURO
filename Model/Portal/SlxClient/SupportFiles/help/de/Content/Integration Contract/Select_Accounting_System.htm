Auswählen eines Buchhaltungssystems

Auf dem Bildschirm Buchhaltungssystem auswählen des Assistent "Verknüpfung zur Buchhaltung" können Sie die Buchhaltungssysteme auswählen, die zum Herstellen einer Verknüpfung mit der ausgewählten Firma zur Verfügung stehen.

So gehen Sie beim Auswählen vor

  1. Öffnen Sie Assistenten "Verknüpfung zur Buchhaltung".
  1. Füllen Sie den Bildschirm Buchhaltungssystem auswählen aus.
    FeldBeschreibung/Aktion
    SystemWählen Sie das Buchhaltungssystem aus, mit dem Sie die Firma verknüpfen möchten.
  2. Klicken Sie auf Weiter, um zum nächsten Schritt zu gelangen.
