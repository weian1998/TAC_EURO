Konfigurieren der Buchhaltungsintegration

Führen Sie zum Konfigurieren der Buchhaltungsintegration die folgenden Aufgaben aus.

  • Stellen Sie sicher, wenn mehrere Währungen aktiviert sind, dass Sage SalesLogix und Ihr Buchhaltungssystem dieselbe Basiswährung verwenden.
  • Führen Sie das Telefon- und E-Mail-Aktualisierungsprogramm aus.

  • Für den Benutzer mit Administratorrechten muss vor der Ausführung des Telefon- und E-Mail-Aktualisierungsprogramms die Windows-Authentifizierung konfiguriert werden. Einzelheiten dazu finden Sie unter dem Thema "Verwenden der Windows-Authentifizierung" in der Administrator-Hilfe.
    • Die Buchhaltungsintegration ist aktiviert, sobald der Sage SalesLogix-Endpunkt und mindestens ein Zielendpunkt konfiguriert und aktiviert sind. Öffnen Sie die Buchhaltungsintegration-Detailansicht zum Aktivieren von Endpunkten.

    Wenn Sie die Buchhaltungsintegration aktivieren bzw. deaktivieren, wird die Änderung erst wirksam, nachdem Sie sich von Sage SalesLogix abgemeldet und wieder angemeldet haben.
    • Vergleichen Sie das Datenbankschema in Sage SalesLogix mit dem Ihrer Buchhaltungsanwendung, um Datenintegrität zu gewährleisten. Auf der Registerkarte Buchhaltungssysteme können Sie Schemaunterschiede bestimmen.

    Wenn Sie Sage SalesLogix angepasst haben, müssen Sie den Sage CRM ERP Contract auf Informationen überprüfen, die zwischen Systemen synchronisiert werden. Wenn Sie z. B. benutzerdefinierte Entitäten erstellt haben, die keine Sage SalesLogix-Entsprechung haben, funktioniert die Synchronisierung ggf. nicht.
    • Auf der Registerkarte Abgleich können Sie globale standardmäßige Übereinstimmungskriterien für Benutzer festlegen, die Verknüpfungen erstellen.
    • Konfigurieren Sie den Synchronisierungsdiensts für die Buchhaltungsintegration.
    • Bei Bedarf können Sie Datensätze aus dem Buchhaltungssystem exportieren und zusammenführen.

    Exportieren Sie die entsprechenden Daten in dem vom Buchhaltungssystem bereitgestellten Export-Tool in Sage SalesLogix.

    • Konfigurieren Sie Benutzerrollen.

      Die Rolle Integration Contract - Preis sperren berechtigt zum Sperren von Produktpreisen auf einem Verkaufsauftrag. Die Aktion Firma hochstufen gibt Benutzern die Berechtigung, Firmen hoch zu stufen, und gehört standardmäßig zur Rolle Standardbenutzer.
