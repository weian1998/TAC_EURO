Buchhaltungsintegration-Detailansicht

In der Buchhaltungsintegration-Detailansicht können Sie die Verbindungen zwischen Sage SalesLogix und Ihren Buchhaltungssystemen konfigurieren.

Die Buchhaltungsintegration ist aktiviert, sobald der Sage SalesLogix-Endpunkt und mindestens ein Zielendpunkt konfiguriert und aktiviert sind. Wenn auf der Registerkarte Buchhaltungssysteme keine Feeds aktiviert sind, ist die Integration deaktiviert.

Wenn Sie die Buchhaltungsintegration aktivieren bzw. deaktivieren, wird die Änderung erst wirksam, nachdem Sie sich von Sage SalesLogix abgemeldet und wieder angemeldet haben.

Legen Sie die Feed-Details für Sage SalesLogix fest, und konfigurieren Sie anschließend den Feed für jedes integrierte Buchhaltungssystem.

So nehmen Sie die Festlegung vor

  1. Klicken Sie im Menü Administration auf Integrations-Setup.
  2. Geben Sie in das Feld SalesLogix-Feed den SData-URL zum GCRM-Feed ein.
    Verwenden Sie folgendes Format: http://Server:Port/Virtuelles_SDATA-Verzeichnis/slx/gcrm/-/ 
    • Geben Sie für Virtuelles_SDATA-Verzeichnis das Verzeichnis ein, in dem Sie Ihr SData-Portal bereitgestellt haben.
      Das Standardverzeichnis lautet "SData".
      Beispiel: http://localhost:3333/sdata/slx/gcrm/-/
  3. Geben Sie in das Feld Benutzername den Benutzernamen für die Sage SalesLogix-Datenbank ein, mit der das Buchhaltungssystem eine Verbindung herstellen soll.
  4. Geben Sie in das Feld Passwort das Passwort für den Benutzernamen ein.
  5. Geben Sie in das Feld Anzeigename einen Anzeigenamen für den Feed ein.
  6. Aktivieren Sie bei Bedarf die Option Hochstufen von Firmen auf einzelnes Buchhaltungssystem beschränken, um die Integration auf Sage SalesLogix und ein Buchhaltungssystem zu beschränken.
  7. Aktivieren Sie bei Bedarf die Option Verwaltung von Verkaufsaufträgen durch Buchhaltungssystem zulassen, damit in Sage SalesLogix keine Verkaufsaufträge erstellt oder bearbeitet werden können. Bei Aktivierung dieser Option muss das Buchhaltungssystem zum Verwalten von Verkaufsaufträgen verwendet werden.

    8. Diese Option ist bei bestimmten Integrationslösungen, etwa bei der Integration mit X3, erforderlich.
  8. Klicken Sie auf Speichern.
  9. Auf der Registerkarte Buchhaltungssysteme können Sie die Feed-Details für die Buchhaltungssysteme konfigurieren.
  10. Auf der Registerkarte Synchronisierung können Sie Synchronisierungsintervalle festlegen.
  11. Auf der Registerkarte Abgleich können Sie globale standardmäßige Übereinstimmungskriterien für Benutzer festlegen, die Verknüpfungen erstellen.
