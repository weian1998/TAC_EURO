Konfigurieren des Synchronisierungsdiensts für die Buchhaltungsintegration

Der Synchronisierungsdiensts für die Buchhaltungsintegration (SalesLogix-Systemdienst) ermöglicht Datenübertragungen zwischen Sage SalesLogix und Ihrem Buchhaltungssystem. Der Dienst muss gestartet und auf eine automatische Ausführung festgelegt werden.

So gehen Sie zum Konfigurieren vor

  1. Sie müssen vor dem Start des Diensts Endpunkte konfigurieren und aktivieren.
  2. Überprüfen Sie die Datei Sage.SalesLogix.SyncEngineService.exe.config, um sicherzustellen, dass die Verbindung am Ende der Datei ordnungsgemäß festgelegt ist. Die Verbindungsinformationen müssen der für Ihre Website bereitgestellten Datei connection.config entsprechen.
    Standardmäßig wird die Konfigurationsdatei unter ...\Programme\SalesLogix\Sage.SalesLogix.SyncEngineService.exe.config und die Verbindungsdatei unter ...\inetpub\wwwroot\slxclient bereitgestellt.
  3. Öffnen Sie auf dem Verwaltungscomputer Dienste.
  4. Klicken Sie mit der rechten Maustaste auf SalesLogix-Systemdienst und klicken Sie dann auf Eigenschaften.
  5. Wählen Sie im Feld Starttyp die Option Automatisch aus.
  6. Legen Sie den Dienst so fest, dass Anmeldung und Ausführung durch den Benutzer WebDLL erfolgen.
  7. Vergewissern Sie sich, dass für den Benutzer WebDLL die Windows-Authentifizierung aktiviert ist.
  8. Klicken Sie auf Starten.
  9. Klicken Sie auf OK.
