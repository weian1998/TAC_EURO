Registerkarte "Abgleich"

Auf der Registerkarte Abgleich können Sie die Optionen für das Bestimmen von Übereinstimmungen zwischen Sage SalesLogix und integrierten Buchhaltungssystemen konfigurieren. Die hier ausgewählten Optionen bestimmen, wie die Daten, die von Benutzern in Sage SalesLogix eingegeben werden, mit im Buchhaltungssystem vorhandenen Systemen abgeglichen werden. Weitere Informationen...

Diese Einstellungen dienen zum Erstellen der standardmäßigen Abgleichkriterien für alle Benutzer, die Buchhaltungsdaten verknüpfen. Durch Festlegen zu eingrenzender Suchkriterien werden den Benutzern ggf. nicht genügend Ergebnisse zurückgegeben,wodurch sie ggf. Duplikate erstellen.

So nehmen Sie die Konfiguration vor

  1. Öffnen Sie die Buchhaltungsintegration-Detailansicht.
  2. Klicken Sie auf die Registerkarte Abgleich.
  3. Klicken Sie auf den Hyperlink Bearbeiten für die Ressource, für die Sie Übereinstimmungskriterien festlegen oder ändern möchten.
  4. Vervollständigen Sie im Dialog Übereinstimmungskriterien bearbeiten folgende Angaben:
    1. Wählen Sie im Feld Eigenschaft eine Eigenschaft aus, für die Werte verglichen werden sollen.
      Ihre Buchhaltungssysteme müssen das Filtern für diese Eigenschaft unterstützen. Andernfalls bekommen Benutzer, die versuchen Firmen hochzustufen, erst Übereinstimmungen angezeigt, wenn sie ihre Suche ändern. Weitere Informationen...
      • Geben Sie in Ihrem Browser den Buchhaltungs-Feed-URL, den Sie beim Einrichten der Buchhaltungsintegration festgelegt haben, gefolgt von /$schema ein, und stellen Sie sicher, dass das Attribut Filtern möglich der Eigenschaft auf Wahr eingestellt ist.

        URL-Beispiel:

        http://Server:Port/Virtuelles_SDATA-Verzeichnis/Anwendung/gcrm/aktives_Unternehmen/$schema

    2. Wählen Sie im Feld Operator einen Operator aus, um die Übereinstimmungsbedingung zu vervollständigen.
      • Gleich - Der Eigenschaftswert in beiden Systemen muss für eine Übereinstimmung identisch sein. Aberle ist beispielsweise keine Übereinstimmung mit Aberle GmbH.
      • Beginnt mit - Der Eigenschaftswert in beiden Systemen muss mit demselben Wert beginnen.
      • Enthält - Der Eigenschaftswert in beiden Systemen muss denselben Wert enthalten. Aberle ist dann beispielsweise eine Übereinstimmung mit "Aberle GmbH" und "Aberle weltweit".
    3. Klicken Sie zum Hinzufügen zusätzlicher Übereinstimmungskriterien auf .
    4. Klicken Sie zum Entfernen von Übereinstimmungskriterien auf .
    5. Klicken Sie auf OK.
