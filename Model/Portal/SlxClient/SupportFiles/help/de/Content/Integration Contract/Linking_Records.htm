Verknüpfen von Datensätzen mit einem Buchhaltungssystem

Diese Funktion steht nur zur Verfügung, wenn Ihr Administrator die Buchhaltungsintegration konfiguriert hat.

Beim Hochstufen einer Firma verknüpfen Sie eine Sage SalesLogix-Firma mit Ihrem Buchhaltungssystem.

So gehen Sie zum Verknüpfen vor

  1. Öffnen Sie den Firmendatensatz, den Sie in der Firmendetailansicht verknüpfen möchten, oder wählen Sie in der Firmenlistenansicht eine oder mehrere Firmen aus.
  2. Führen Sie im Bereich Buchhaltungsaufgaben einen der folgenden Schritte aus:
    • Klicken Sie auf Firma hochstufen.
    • Klicken Sie auf In anderes Buchhaltungssystem übertragen.
      Diese Option ist nicht verfügbar, wenn Ihr Unternehmen nur die Integration mit einem Buchhaltungssystem zulässt.
  3. Vervollständigen Sie im Assistenten Mit Buchhaltung verknüpfen die einzelnen Bildschirme, und klicken Sie auf Weiter. Je nach Ihren Systemen und Verknüpfungskriterien werden nicht alle der folgenden Bildschirme angezeigt.
    • Auswählen eines Buchhaltungssystems
    • Auswählen oder Erstellen einer Firma
    • Suchen nach Übereinstimmungen
    • Zusammenführen von Daten
    • Zusammenführen von Kontakten
  4. Klicken Sie auf Schließen.
