Senden von Verkaufsaufträgen

Verkaufsaufträge müssen zur Verarbeitung an das Buchhaltungssystem übermittelt werden. Nachdem ein Verkaufsauftrag an das Buchhaltungssystem übermittelt wurde, kann er in Sage SalesLogix nicht mehr geändert werden. Änderungen von Verkaufsaufträgen müssen im Buchhaltungssystem verwaltet werden.

  • Sie können ein Angebot erst drucken oder per E-Mail senden, nachdem der Verkaufsauftrag vom Buchhaltungssystem genehmigt wurde.
  • Verkaufsaufträge, die vor Aktivierung der Buchhaltungsintegration aktiv waren, können nicht an das Buchhaltungssystem übermittelt werden. Verkaufsaufträge müssen Produkte enthalten, die mit dem Buchhaltungssystem synchronisiert und ihm zugeordnet sind.

So gehen Sie zum Senden vor

  1. Öffnen Sie die Ansicht Verkaufsauftragsdetails für den Verkaufsauftrag, den Sie senden möchten.
  2. Klicken Sie im Bereich Buchhaltungsaufgaben auf An Buchhaltungssystem senden.
  3. Klicken Sie im Dialogfeld An Buchhaltungssystem senden auf Ja.
  4. Überprüfen Sie im Dialog Preisaktualisierungen auf Preisaktualisierungen aus dem Buchhaltungssystem.
  • Klicken Sie auf Änderungen übernehmen, um die Preise zu akzeptieren und den Verkaufsauftrag an das Buchhaltungssystem zu übermitteln.
  • Klicken Sie auf Abbrechen, um die vorhandenen Preise beizubehalten und zum Verkaufsauftrag zurückzukehren.
  1. Nach dem Senden wird der Verkaufsauftrag mit Informationen aus dem Buchhaltungssystem aktualisiert.
