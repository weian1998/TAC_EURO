Verwenden der Registerkarte "Aufgaben für Fehler"

Auf dieser Registerkarte können Sie alle Aktivitäten anzeigen, die mit dem Fehler verknüpft sind, Zeit für die Bearbeitung eines Fehlers hinzufügen oder verrechnen oder eine Aktivität bearbeiten. Aktivitäten sind beliebige Aktionen bei der Bearbeitung bzw. Lösung des Fehlers. Wenn Sie beispielsweise Recherchen für den Fehler durchführen müssen, fügen Sie diese Aktivität hier hinzu.

Diese Ansicht enthält folgende Spalten:

  • Aktivitätstyp
  • Benutzer
  • Empfangsdatum
  • Abschlussdatum
  • Gebührentyp
  • Zeiteinheiten
  • Verstrichene Stunden
  • Gesamtgebühr

So gehen Sie zum Öffnen vor

  1. Öffnen Sie die Fehlerdetailansicht.
  2. Klicken Sie auf die Registerkarte Aufgaben für Fehler. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.

So fügen Sie eine Aufgabe hinzu

  1. Klicken Sie auf Aufgabe hinzufügen.
  2. Geben Sie die Informationen in die Informationsfelder zum Hinzufügen einer neuen Aufgabe für einen Fehler ein.
  3. Klicken Sie auf OK.

So bearbeiten Sie eine Aufgabe

  1. Klicken Sie neben der zu bearbeitenden Aufgabe auf Bearbeiten.
  2. Bearbeiten Sie die Informationsfelder.
  3. Klicken Sie auf OK.

So löschen Sie eine Aufgabe

  1. Wählen Sie den Datensatz aus, den Sie entfernen möchten, und klicken Sie auf die Schaltfläche Löschen..
  2. Klicken Sie auf Ja.
