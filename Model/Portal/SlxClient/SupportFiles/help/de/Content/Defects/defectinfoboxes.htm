Fehlerinformationsfelder

In der nachfolgenden Tabelle werden die Standardinformationsfelder beschrieben.

 

Feld Beschreibung/Aktion

Fehler-ID

(Nur Ansicht Verkaufsauftragsdetails)

 Zeigt beim Anlegen des Fehlers automatisch die nächste verfügbare ID an.
Projekt Das Projekt, mit dem der Fehler verknüpft ist. Klicken Sie auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
Gefundene Version Wählen Sie die Version des Produkts oder Projekts aus, in dem der Fehler gefunden wurde. Klicken Sie auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus
Zielversion Wählen Sie die Version aus, in der der Fehler erwartungsgemäß behoben sein soll. Klicken Sie auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
Aktueller Status Wählen Sie eine Statusoption für den Fehler aus, z. B. Ausstehend und Nicht gefunden. Klicken Sie auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
Korrigiert in Version Wählen Sie die entsprechende Version aus, in der der Fehler behoben wurde.
Bereich/Kategorie/Problem

In diesen drei Feldern wird die Art des gemeldeten Problems kurz beschrieben. Klicken Sie auf die Schaltfläche Suchen neben den Feldern, wählen Sie die entsprechenden Einträge aus den Listen aus und klicken Sie auf OK.

Welcher Kategorie die aufgeführten Einträge angehören, ist von der Option abhängig, die Sie als Bereich ausgewählt haben, und welche Probleme verfügbar sind, ist von der Option abhängig, die Sie als Kategorie ausgewählt haben.

Betreff

(Nur bei Fehler einfügen)

 Geben Sie eine kurze Beschreibung des Fehlers ein.

Beschreibung

(Nur bei Fehler einfügen)

 Geben Sie eine detaillierte Beschreibung des Problems ein.

Lösung

(Nur bei Fehler einfügen)

 Geben Sie die zur Lösung des Fehlers ausgeführten Schritte ein.
Status Standardmäßig lautet der Status Offen. Klicken Sie zum Bearbeiten auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
Quelle Die Empfangsquelle für die Fehlerinformationen (beispielsweise Ticket oder Rückruf). Klicken Sie zum Bearbeiten auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
Typ Wählen Sie den Fehlertyp aus. Klicken Sie auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
Priorität Die Priorität bestimmt die Reihenfolge, in der Fehler behandelt werden, und die Ressourcen, die für die Lösung von Problemen zugewiesen werden. Fehler mit einer hohen Priorität werden in der Regel zuerst behandelt. Klicken Sie auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
Schweregrad Weisen Sie dem Fehler einen Schweregrad zu, indem Sie in der Liste einen Eintrag auswählen. Klicken Sie auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
Häufigkeit Wählen Sie in der angezeigten Auswahlliste die Häufigkeit aus.
SpeedSearch durchführen

Aktivieren Sie dieses Kontrollkästchen, wenn Sie Fehlerinformationen über die Suchfunktion auch für andere Benutzer verfügbar machen möchten.

Je nachdem, wie SpeedSearch vom Administrator konfiguriert wurde, stehen die Informationen durch Auswählen dieser Option entweder automatisch für die Suche zur Verfügung oder müssen von einem anderen Benutzer mit der Option Für SpeedSearch genehmigt freigegeben werden.
Für SpeedSearch genehmigt Aktivieren Sie dieses Kontrollkästchen, um einen Fehler zu genehmigen, der an SpeedSearch übergeben wurde. Diese Option ist verfügbar, wenn Ihr Administrator Sie als Person ausgewiesen hat, die Fehler für SpeedSearch genehmigen darf.
Öffentlicher Zugriff

Dieses Feld wird angezeigt, wenn der Administrator Ihnen die Berechtigung zum Genehmigen von SpeedSearch-Einsendungen erteilt hat. Wählen Sie eine der folgenden Optionen aus:

  • 5-Intern - Fehler werden in SpeedSearch-Indizes aufgenommen, auf die nur interne Benutzer zugreifen dürfen.
  • 1-Kunde - Fehler werden in SpeedSearch-Indizes aufgenommen, auf die interne Benutzer und Kunden, die Web Customer Portal verwenden, zugreifen dürfen.
Zugewiesen zu Das System fügt den Benutzernamen auf der Grundlage der automatischen Zuweisungseinstellungen für das ausgewählte Bereichselement ein. Um einen anderen Benutzer zuzuweisen, klicken Sie auf die Schaltfläche Suchen, suchen den entsprechenden Benutzer und klicken anschließend auf OK.
Telefon Dieses Feld wird mit der Telefonnummer des zugewiesenen Benutzers ausgefüllt.
Datum der Zuweisung Das Datum und die Uhrzeit, zu denen der Fehler dem Benutzer zugewiesen wurde, dem es aktuell zugewiesen ist. Um das Datum der Zuweisung zu ändern, klicken Sie auf Kalender-Schaltfläche, wählen das Datum aus und klicken auf OK.
Zugriffsberechtigter Benutzer oder Benutzerteams mit Zugriffsrechten für den Fehler. Verwenden Sie die Suchfunktion, um einen Benutzer oder ein Team auszuwählen.
