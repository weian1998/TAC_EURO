Öffnen des Abfrage-Assistenten

Mit dem Abfrage-Assistenten können Sie Gruppen hinzufügen oder ändern.

So öffnen Sie den Abfrage-Assistenten

  • Führen Sie einen der folgenden Schritte durch:
  • Wählen Sie in der entsprechenden Listenansicht im Menü Gruppe den Eintrag Gruppe hinzufügen aus, um eine Gruppe hinzuzufügen.
  • Klicken Sie mit der rechten Maustaste auf die Gruppenregisterkarte, die Sie bearbeiten möchten, und klicken Sie dann auf Bearbeiten.
