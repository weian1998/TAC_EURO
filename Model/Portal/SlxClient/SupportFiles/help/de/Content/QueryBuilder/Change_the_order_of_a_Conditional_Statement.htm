Ändern der Reihenfolge einer Bedingungsanweisung

Wenn eine Abfrage mehrere Bedingungsanweisungen enthält, können Sie jede beliebige Anweisung nach oben oder nach unten versetzen und damit die Reihenfolge verändern, mit der die Bedingungen in der Abfrage angewendet werden.

So ordnen Sie eine Bedingungsanweisung neu an:

  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Bedingungen.
  3. Wählen Sie eine Anweisung auf dem Raster aus.
  4. Klicken Sie auf Nach oben oder auf Nach unten.
