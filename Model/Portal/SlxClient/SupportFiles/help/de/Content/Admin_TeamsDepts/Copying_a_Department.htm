Kopieren von Abteilungen

Wenn Sie eine vorhandene Abteilung kopieren, werden alle Mitglieder der neuen Abteilung hinzugefügt.

So gehen Sie zum Kopieren vor

  1. Öffnen Sie die Abteilungsdetailansicht der Abteilung, die Sie kopieren möchten.
  2. Klicken Sie im Bereich Allgemeine Aufgaben auf Abteilung kopieren.
  3. Klicken Sie auf OK, um den Vorgang zu bestätigen.
  4. Geben Sie in das Feld Name einen Namen für die neue Abteilung ein.
  5. Ändern Sie bei Bedarf die Teammitglieder und Teams, zu denen die Abteilung gehört.
  6. Klicken Sie auf Speichern.
