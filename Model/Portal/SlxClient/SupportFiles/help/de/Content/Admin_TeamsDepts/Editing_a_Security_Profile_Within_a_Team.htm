Bearbeiten eines Sicherheitsprofils innerhalb eines Teams

Nach dem Hinzufügen eines Teammitglieds können Sie die Sicherheitsrechte ändern. Die ursprünglichen Sicherheitsrechte des Benutzers sind so festgelegt, dass sie dem Standardsicherheitsprofil des Benutzers entsprechen. WebViewer-Sicherheitsprofile können nicht bearbeitet werden. Weitere Informationen zur Sicherheit...

So bearbeiten Sie Informationen

  1. Klicken Sie in der Teamdetailansicht auf die Registerkarte Mitglieder.
  2. Klicken Sie für das Mitglied, das Sie bearbeiten möchten, auf den Hyperlink Sicherheitsprofil.
  3. Wählen Sie ein neues Sicherheitsprofil aus.

Die Standardprofile heißen wie folgt:

  • Schreibgeschützt - Standard: Gewährt einen schreibgeschützten Zugriff auf alle Tabellen und Felder.
  • Lesen/Schreiben - Standard: Gewährt einen Lese-/Schreibzugriff auf alle Tabellen und Felder mit Ausnahme des Felds Seccodeid in der Tabelle Firma. Auf diese Weise kann der Zugriffsberechtigte der Firma vom Benutzer nicht geändert werden.
  • Profil des Teamzugriffsberechtigten: Gewährt einen Lese-/Schreibzugriff auf alle Tabellen und Felder. Mit diesem Profil kann der Zugriffsberechtigte der Firma vom Benutzer geändert werden. WebViewer-Benutzern kann das Profil des Teamzugriffsberechtigten nicht zugewiesen werden.
  1. Klicken Sie auf OK.
  2. Klicken Sie auf Speichern.
