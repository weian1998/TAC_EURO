Arbeiten mit der Bibliothek

In der Bibliothek können Unternehmensinformationen zentral abgelegt werden. Die Ablage erfolgt in Form von Dokumenten, URLs usw., die dann schnell und einfach an Remote-Benutzer verteilt werden können. Dateien werden in Ordnern organisiert und nach Namen sortiert. Klicken Sie auf die Spaltenüberschriften, um die Liste der Bibliotheksdateien zu sortieren.

Mithilfe von Sage SalesLogix Web Client können Sie bei Bedarf eine Bibliotheksdatei anfordern und herunterladen. Die Bibliothek zeigt den Dateinamen, die Größe, das Erstellungs-, Änderungs- und Ablaufdatum sowie eine kurze Beschreibung der Datei an.

Die Bibliothek Ihrer Organisation kann Produktinformationen, Administratorrechte, Präsentationen, Web-Adressen und so weiter enthalten.

Dateien können nur vom Administrator oder einem Administratorbenutzer zur Bibliothek hinzugefügt werden.

Vorgehensweise

Öffnen der Bibliothek
  • Klicken Sie auf der Navigationsleiste auf Bibliothek.
Herunterladen einer Bibliotheksdatei
  1. Klicken Sie in der Strukturansicht auf die Pluszeichen, um den Ordner einzublenden, der die Datei enthält, die Sie öffnen möchten.
  2. Klicken Sie in der Liste Datei auf den Namen der Datei, die Sie herunterladen möchten.

  3. Führen Sie einen der folgenden Schritte durch:

    • Öffnen der Datei
    • Speichern der Datei

Wenn Ihr Browser die Datei öffnet, anstatt sie in einem lokalen Verzeichnis auf Ihrem Computer zu speichern, ändern Sie die Browser-Einstellungen so, dass Dateien automatisch gespeichert und nicht geöffnet werden. Alternativ dazu können Sie die UMSCHALT-Taste gedrückt halten und auf den Hyperlink klicken, um alle Dateien, mit Ausnahme der Dateien mit der Erweiterung .html herunterzuladen ohne sie zu öffnen. HTML-Dateien werden in einem anderen Fenster geöffnet.

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.Der Systemadministrator und Benutzer mit einer Administratorrolle können auf diese Funktionen zugreifen.
Hinzufügen eines Dokuments zur Bibliothek
  1. Wählen Sie den Ordner aus, unter dem Sie das Element hinzufügen möchten.
  2. Klicken Sie auf Dateien hinzufügen.
  3. Wählen Sie die Datei(en) aus, die Sie hinzuzufügen möchten.
  4. Klicken Sie auf Öffnen.
Hinzufügen eines Dokuments zur Bibliothek per Ziehen und Ablegen

  • Möglicherweise haben Sie keine Zugriffsrechte für diese Funktion. Weitere Informationen hierzu erhalten Sie von Ihrem Systemadministrator.
  • Wenn Sie Internet Explorer verwenden, müssen Sie das Desktopintegrationsmodul installieren, um diese Funktion nutzen zu können.
  1. Wechseln Sie zur Datei, die Sie hinzufügen möchten.
  2. Klicken Sie auf der Navigationsleiste auf Bibliothek.
  3. Klicken Sie in der Strukturansicht auf das Pluszeichen (+), um der Ordner bis zum Speicherort einzublenden, an dem Sie die Datei speichern möchten.
  4. Ziehen Sie die Datei, und legen Sie sie auf der Bibliotheksansicht ab.
  5. Geben Sie im Feld Beschreibung eine Beschreibung für die Datei ein, wenn die Standardbeschreibung nicht verständlich ist.
  6. Klicken Sie auf OK.

Sie können einen Ordner samt Inhalt nicht per Ziehen und Ablegen der Bibliothek hinzufügen. Dies ist nur bei einzelnen Dateien möglich.
