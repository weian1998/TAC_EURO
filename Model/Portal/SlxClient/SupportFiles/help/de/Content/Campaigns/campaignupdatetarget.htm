Aktualisieren von Zielinformationen

Über das Dialogfeld Ziel aktualisieren können Sie mehrere Zielkontakte oder -interessenten gleichzeitig aktualisieren. Verwenden Sie die Felder im unteren Bereich des Dialogfelds, um mehrere Ziele mit denselben Informationen zu aktualisieren.

So öffnen Sie den Dialog "Ziele aktualisieren"

  • Klicken Sie in der Kampagnendetailansicht auf die Registerkarte Ziele und dann auf Ausgewählte Ziele aktualisieren .

So aktualisieren Sie mehrere Verkaufschancen mit denselben Informationen

  1. Klicken Sie auf die Kontrollkästchen Auswählen der Ziele, die Sie aktualisieren möchten, oder klicken Sie auf die Schaltfläche Alle auswählen, um alle Ziele zu aktualisieren. Klicken Sie anschließend auf Ziele aktualisieren.
  2. Wählen Sie im Dropdown-Feld im unteren Teil des Dialogfelds die zu aktualisierenden Informationen aus. Wählen Sie unter folgenden Optionen:
  • Phase
    1. Wählen Sie in der Dropdown-Liste am unteren Rand des Dialogs Phase aus.
    2. Klicken Sie auf den Dropdown-Pfeil Zu und wählen Sie einen Eintrag aus der Liste aus.
  • Status
    1. Wählen Sie Status aus der Dropdown-Liste aus.
    2. Klicken Sie auf den Dropdown-Pfeil Zu und wählen Sie einen neuen Status in der Liste.
  • Anfangsziel
    1. Wählen Sie Anfangsziel aus der Dropdown-Liste aus.
    2. Wenn das Ziel der Kampagne bereits beim Start angehört hat, wählen Sie Ja. Wenn das Ziel nach dem Start der Kampagne hinzugefügt wurde, wählen Sie Nein.
  • Antwort hinzufügen
    1. Wählen Sie Antwort hinzufügen aus.
    2. Füllen Sie die Antwortinformationen aus.
  1. Wenn Sie das Aktualisieren der Zielinformationen abgeschlossen haben, klicken Sie auf Aktualisieren.
