Hinzufügen/Bearbeiten/Abschließen von Kampagnenphasen

Kampagnenphasen sind Gruppen von Aufgaben, die Benutzern von Sage SalesLogix Marketing beim Verwalten ihrer Kampagnen helfen können. Sie haben die Möglichkeit, Informationen zu Phasen hinzuzufügen und zu bearbeiten oder den Abschluss einer Phase zu erfassen.

Vorgehensweise

Hinzufügen einer Phase
  1. Klicken Sie in der Detailansicht Kampagne auf die Registerkarte Phasen/Aufgaben, und klicken Sie dann auf Phase hinzufügen.
  2. Geben Sie eine Beschreibung für die Phase ein.
  3. Klicken Sie im Feld Status auf die Dropdown-Pfeil und wählen Sie einen Eintrag in der ListeIn Bearbeitung.
  4. Klicken Sie im Feld Interessentenquelle auf die Schaltfläche Suchen, und wählen Sie eine Interessentenquelle aus.
    Ihre Auswahl wird zum Dialogfeld hinzugefügt.
  5. Ändern Sie bei Bedarf den Code. Klicken Sie im Dialog auf OK.

    Wenn der Kampagnencode geändert wird, entspricht der Kampagnencode für diese Phase nicht mehr der Kampagne oder anderen Kampagnenphasen.

  6. Geben Sie im Feld Kommentare ggf. zusätzliche Informationen ein.
  7. Klicken Sie ggf. auf StartdatumKalender-Schaltfläche, um das Datum auszuwählen.  
  8. Klicken Sie auf EnddatumKalender -Schaltfläche, um das Datum auszuwählen.  
  9. Fügen Sie dem Raster AufgabenAufgaben hinzu.
  10. Klicken Sie auf OK.
Bearbeiten einer Kampagnen-Phase
  1. Klicken Sie in der Detailansicht Kampagne auf die Registerkarte Phasen/Aufgaben, und klicken Sie dann auf Phase bearbeiten.
    • Wählen Sie im Dialog Kampagne einfügen oder Kampagne kopieren die zu bearbeitende Phase aus, und klicken Sie dann auf Bearbeiten.
    • Wählen Sie in der Ansicht Kampagnendetail die Registerkarte Phasen/Aufgaben aus, klicken Sie mit der rechten Maustaste auf die zu bearbeitende Phase, und zeigen Sie dann auf Phase bearbeiten.
  2. Nehmen Sie bei Bedarf Änderungen vor, und klicken Sie dann auf OK.
Abschließen einer Kampagnen-Phase
  1. Klicken Sie in der Detailansicht Kampagne auf die Registerkarte Phasen/Aufgaben, wählen Sie die abzuschließende Phase aus, und klicken Sie dann auf Phase abschließen.
    Der Dialog Phase abschließen wird angezeigt. Darin ist Status auf Abgeschlossen festgelegt.
  2. Nehmen Sie bei Bedarf Änderungen vor, und klicken Sie dann auf OK.
