Registerkarte "Marketing"

Auf der Registerkarte Marketing in den Detailansichten Firma, Kontakt oder Interessent können Sie die Kampagnen mit einem bestimmten Zielkontakt oder -interessenten anzeigen. Sie können außerdem Informationen zu Kampagnenantworten hinzufügen oder bearbeiten sowie einen Kontakt oder Interessenten aus künftigen Phasen der Kampagne löschen.

Auf der Registerkarte Marketing werden die folgenden Informationen zu Kampagnenantworten angezeigt:

  • Kampagnenname
  • Kampagnencode
  • Status
  • Phase
  • Kontakt (nur Firmendetailansicht)
  • Anfangsdatum
  • Enddatum
  • Antwortdatum
  • Antwortmethode

So zeigen Sie die Reaktion auf eine Kampagne für den aktuellen Datensatz an

  • Klicken Sie in der Detailansicht Kontakt, Interessent oder Firma auf die Registerkarte Marketing. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.

So fügen Sie eine Kampagnenantwort hinzu

  • Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie auf die Schaltfläche Kampagnenantwort hinzufügen und führen Sie die Schritte zum Hinzufügen einer Kampagnenantwort aus.
    • (Nur Firmendetailansicht) Um eine Kampagnenantwort für einen Kontakt oder Interessenten hinzuzufügen, der ursprünglich in die Kampagne einbezogen war, klicken Sie neben dem Datensatz auf Hinzufügen und führen die Schritte zum Hinzufügen einer Kampagnenantwort aus.

So löschen Sie eine Antwort aus einer Kampagne

  1. Klicken Sie neben der Antwort, die Sie löschen möchten, auf Löschen.

Die Antwort wird zwar weiterhin in der Registerkarte aufgeführt, ihr Status wird jedoch in Entfernt geändert, das Antwortdatum wird gelöscht und die Antwort wird bei Antwortformeln nicht mehr berücksichtigt.

So entfernen Sie ein Ziel aus weiteren Phasen der Kampagne

  1. Klicken Sie neben der Antwort, die Sie aus weiteren Phasen der Kampagne entfernen möchten, auf Entfernen.
    Die Antwort wird zwar weiterhin in der Registerkarte aufgeführt, ihr Status wird jedoch in Entfernt geändert, das Antwortdatum wird gelöscht und die Antwort wird bei Antwortformeln nicht mehr berücksichtigt.
