Hinzufügen/Bearbeiten/Abschließen von Kampagnenaufgaben

Kampagnenaufgaben können Benutzern von Sage SalesLogix Marketing helfen, ihre Kampagnen zu verwalten. Sie haben die Möglichkeit, Informationen zu Aufgaben hinzuzufügen und zu bearbeiten oder den Abschluss einer Aufgabe zu erfassen.

Vorgehensweise

Hinzufügen von Kampagnenaufgaben
  1. Klicken Sie in der Detailansicht Kampagne auf die Registerkarte Phasen/Aufgaben, suchen Sie die Phase, zu der Sie eine Aufgabe hinzufügen möchten, und klicken Sie dann auf Aufgabe hinzufügen.
  2. Geben Sie eine Beschreibung für die Aufgabe ein.
  3. Klicken Sie im Feld Status auf die Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste.
  4. Klicken Sie im Feld Priorität auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  5. Geben Sie in das Feld % abgeschlossen einen Prozentsatz von 0 bis 100 ein.
  6. Klicken Sie nebenFälligkeitsdatum auf Kalender-Schaltfläche, um das Datum auszuwählen.  
  7. Falls die Aufgabe bereits abgeschlossen wurde, klicken Sie auf AbschlussdatumKalender-Schaltfläche, um das Datum auszuwählen.  .
  8. Geben Sie im Feld Kommentare ggf. zusätzliche Informationen ein.
  9. Geben Sie in das Feld Geschätzte Kosten die geschätzten Kosten der Aufgabe ein.
  10. Wenn Sie die tatsächlichen Kosten der Aufgabe kennen, geben Sie den Betrag in das Feld Ist-Kosten ein.
  11. Geben Sie in das Feld Geschätzte Stunden die geschätzte Anzahl Stunden ein, die zum Abschließen der Aufgabe erforderlich sind.
  12. Wenn Sie wissen, wie viele Stunden tatsächlich zum Abschließen der Aufgabe erforderlich sind, geben Sie den Wert in das Feld Tatsächlich aufgewendete Stunden ein.
  13. Wenn Mehrere Währungen konfiguriert ist, wählen Sie Anzeige unter Verwendung von "Eigene Währung" aus, um Kosten für Aufgaben in dem Währungstyp anzeigen zu lassen, den Sie in Ihren Benutzeroptionen ausgewählt haben.
  14. Wählen Sie die entsprechende Option unter Zuweisen zu aus.
  15. Führen Sie im Feld Zuweisen zu einen der folgenden Schritte aus:
    • Wenn Sie Vorhandener Benutzer/vorhandenes Team gewählt haben, klicken Sie auf die Schaltfläche Suchen und wählen den Benutzer oder das Team aus, dem bzw. der Sie die Aufgabe zuweisen möchten.
    • Wenn Sie Vorhandene Abteilung ausgewählt haben, klicken Sie auf den Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus.
    • Wenn Sie Vorhandener Kontakt gewählt haben, klicken Sie auf die Schaltfläche Suchen und wählen dann die Person aus, der Sie die Aufgabe zuweisen möchten.
    • Wenn Sie Andere Einzelperson gewählt haben, geben Sie den Namen der Person ein, der Sie die Aufgabe zuweisen möchten.
    • Wenn Sie Keine gewählt haben, fahren Sie mit dem nächsten Schritt fort.
  16. Klicken Sie auf Datum der ZuweisungKalender-Schaltfläche, um das Datum auszuwählen.  .
  1. Klicken Sie auf OK.
Bearbeiten von Kampagnenaufgaben
  1. Klicken Sie in der Detailansicht Kampagne auf die Registerkarte Phasen/Aufgaben, und klicken Sie dann auf Bearbeiten.
    • Wählen Sie im Dialog Kampagne einfügen oder Kampagne kopieren die zu bearbeitende Aufgabe aus, und klicken Sie dann auf Bearbeiten.
    • Klicken Sie im Dialog Phase hinzufügen, Phase bearbeiten oder Phase abschließen auf Aufgabe bearbeiten.
    • Wählen Sie in der Ansicht Kampagnendetail die Registerkarte Phasen/Aufgaben aus, klicken Sie mit der rechten Maustaste auf das Raster, und zeigen Sie dann auf Aufgabe bearbeiten.
  2. Nehmen Sie bei Bedarf Änderungen vor, und klicken Sie dann auf OK.

Wenn Sie die Einstellung Fälligkeitsdatum ändern, werden Sie aufgefordert, die Einstellung Fälligkeitsdatum für alle anderen nicht abgeschlossenen Aufgaben um dieselbe Anzahl von Tagen zu verändern. Klicken Sie auf Ja, um die Einstellung Fälligkeitsdatum für die anderen Aufgaben anzupassen.
Abschließen von Kampagnenaufgaben
  1. Klicken Sie in der Detailansicht Kampagne auf die Registerkarte Phasen/Aufgaben, und klicken Sie dann auf Abschließen.
    Der Dialog Phase abschließen wird angezeigt. Darin sind Status auf Abgeschlossen und % abgeschlossen auf den Wert 100 festgelegt. Als Abschlussdatum wird das aktuelle Datum angezeigt.
  2. Nehmen Sie bei Bedarf Änderungen vor und klicken Sie dann auf OK.

Wenn Sie die Einstellung Fälligkeitsdatum ändern, werden Sie aufgefordert, die Einstellung Fälligkeitsdatum für alle anderen nicht abgeschlossenen Aufgaben um dieselbe Anzahl von Tagen zu verändern. Klicken Sie auf Ja, um die Einstellung Fälligkeitsdatum für die anderen Aufgaben anzupassen.
