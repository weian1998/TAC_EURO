Löschen einer Kampagne

Wenn Sie eine Kampagne löschen, löschen Sie auch alle Informationen, die mit dieser Kampagne verknüpft sind, beispielsweise Antwort- und Verkaufschanceninformationen.

So löschen Sie eine Kampagne

  1. Suchen Sie die zu löschende Kampagne.
  2. Klicken Sie in der Detailansicht Kampagne auf die Schaltfläche Löschen.

Aktive Kampagnen können nicht gelöscht werden. Wenn Sie versuchen, eine aktive Kampagne zu löschen, wird ein Meldungsfeld angezeigt. Sie müssen den Kampagnenstatus vor dem Löschen auf einen anderen Wert als Aktiv festlegen.

  1. Klicken Sie auf OK, um den Datensatz unwiderruflich zu löschen.
