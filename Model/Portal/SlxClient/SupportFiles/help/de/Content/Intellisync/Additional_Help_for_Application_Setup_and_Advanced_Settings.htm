Weitere Hilfe zum Einrichten der Anwendung

Weitere Informationen über das Konfigurieren der folgenden Einstellungen finden Sie in den Hilfethemen, die über den Assistenten für erweiterte Einstellungen für Intellisync for SalesLogix zur Verfügung stehen.

  • Erstellen von Filtern
  • Zuordnen von Feldern
  • Angeben von Anmeldeinformationen
  • Empfangen von Bestätigungsmeldungen
  • Auswählen einer Desktop-Anwendung
  • Festlegen von Filterbedingungen und -regeln
  • Festlegen, was synchronisiert werden soll
  • Festlegen, wie synchronisiert werden soll
  • Festlegen, wann synchronisiert werden soll
  • Festlegen von Outlook- oder Lotus Notes-Datenaustauschoptionen
  • Festlegen von Outlook- oder Lotus Notes-Aufgabenoptionen
  • Festlegen von Outlook- oder Lotus Notes-Kalenderoptionen

Die Hilfeschaltflächen der Intellisync-Einrichtungsoberfläche rufen Hilfethemen auf, die von Nokia, dem Hersteller von Intellisync, verfasst wurden. Die Terminologie in diesen Hilfethemen unterscheidet sich eventuell teilweise von den Begriffen in einer Sage SalesLogix-Implementierung.
