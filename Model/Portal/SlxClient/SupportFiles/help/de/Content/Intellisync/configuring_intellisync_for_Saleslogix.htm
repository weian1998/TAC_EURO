Konfigurieren von Intellisync for SalesLogix

Synchronisierungseinstellungen müssen auf dem Computer jedes Benutzers konfiguriert werden. Für die Synchronisierung muss in Sage SalesLogix eine Ad-hoc-Gruppe vorhanden sein, die als Synchronisierungsgruppe konfiguriert werden muss. Für die Konfiguration von Intellisync benötigen Sie den URL für das Intellisync for SalesLogix-Portal. Ausführliche Informationen hierzu erhalten Sie von Ihrem Systemadministrator.

Für unterschiedliche Datensatztypen können verschiedene Einstellungen konfiguriert werden. Sie können beispielsweise die Synchronisierung in eine Richtung für Kontakte und die Synchronisierung in beide Richtungen für Kalenderelemente aktivieren und die Synchronisierung auf unterschiedliche Zeitpunkte festlegen. In den folgenden Schritten werden die empfohlenen Synchronisierungseinstellungen zwischen Microsoft Outlook oder Lotus Notes und Intellisync for SalesLogix beschrieben.

Die Hilfeschaltflächen der Intellisync-Einrichtungsoberfläche rufen Hilfethemen auf, die von Nokia, dem Hersteller von Intellisync, verfasst wurden. Die Terminologie in diesen Hilfethemen unterscheidet sich eventuell teilweise von den Begriffen in einer Sage SalesLogix-Implementierung. Weitere Informationen finden Sie im Thema Weitere Hilfe zu Anwendungssetup.

So gehen Sie zum Konfigurieren vor

  1. Starten Sie Intellisync for SalesLogix.
    • Klicken Sie auf Start, zeigen Sie auf Programme, dann auf Sage SalesLogix und danach auf Intellisync for SalesLogix und klicken Sie anschließend auf Intellisync for SalesLogix.
  1. Klicken Sie auf die Registerkarte Setup und führen Sie folgende Schritte aus:
    1. Geben Sie im Feld Benutzer-ID Ihren SalesLogix-Benutzernamen ein.
    2. Geben Sie im Feld Kennwort Ihr Passwort ein.
    3. Geben Sie in das Feld Verbindungs-URL den Pfad zu Ihrem Intellisync for SalesLogix-Portal ein.
      Den URL zum Portal erfahren Sie von Ihrem Systemadministrator.
  2. Klicken Sie auf Anwendungssetup.
  3. Im Bildschirm Setup können Sie unter Aktiv die Kontrollkästchen für jeden Datentyp aktivieren oder deaktivieren. Aktiv bedeutet, dass die Daten bei jeder Synchronisierung synchronisiert werden.

    Um die Kontakte einzugrenzen, die mit Sage SalesLogix synchronisiert werden sollen,empfiehlt Sage SalesLogix, einen SyncSalesLogix-Filter zu erstellen.

  4. Klicken Sie auf die Schaltfläche Setup-Assistent.
  5. Wählen Sie im Bildschirm Desktop-Anwendung die Option Microsoft Outlook oder Lotus Notes aus und klicken Sie auf Weiter.
  6. Wählen Sie im Bildschirm Zu synchronisierende Daten die Option Alle von Microsoft Outlook unterstützten Intellisync for SalesLogix-Anwendungen oder die entsprechende Option für Lotus Notes aus und klicken Sie auf Weiter.
  7. Wählen Sie im Bildschirm Synchronisieren die Option Bidirektionale Synchronisierung aus und klicken Sie auf Weiter.
  8. Wählen Sie im Bildschirm Zeitpunkt der Synchronisierung aus, wann die Synchronisierung erfolgen soll, und klicken Sie auf Weiter.
    • Alle 10 Minuten – Wählen Sie in der Dropdown-Liste ein Intervall aus.
      Der Benutzer startet Intellisync manuell, PushSync wird gestartet und Daten werden im angegebenen Intervall synchronisiert.
    • Täglich zum angegebenen Zeitpunkt – Wählen Sie in der Dropdown-Liste ein Intervall aus.
      Der Benutzer startet Intellisync manuell, PushSync wird gestartet und Daten werden im angegebenen Intervall synchronisiert.
    • Bei Anmeldung am Computer – Nachdem sich der Benutzer angemeldet hat, wird Intellisync gestartet und in der Windows-Taskleiste ausgeführt.
    • Bei Erkennen einer Änderung in Lotus Notes/Outlook - Intellisync wird gestartet und in der Windows-Taskleiste ausgeführt.
    • Bei Herstellen einer Verbindung mit Intellisync for SalesLogix Server.
  9. Wählen Sie im Bildschirm Intellisync for SalesLogix-Verbindungsinformationen Ihr Benutzerprofil aus, und klicken Sie auf Weiter.
  10. Füllen Sie die weiteren Optionen/Bildschirme mit den entsprechenden Angaben für den Informationstyp aus, den Sie für Outlook oder Lotus Notes synchronisieren.
    1. Wählen Sie im Bildschirm Optionen für Aufgaben für Outlook oder Lotus Notes die Option Alle Einträge übertragen aus und klicken Sie auf Weiter.
      Diese Option synchronisiert alle aktiven, nicht abgeschlossenen und abgeschlossenen Aufgaben.
    2. Wählen Sie im Bildschirm Optionen für Termine die Option Ausschließlich zukünftige Einträge übertragen aus, und klicken Sie auf Weiter.
    3. Wenn Sie Intellisync für Lotus Notes konfigurieren wechseln Sie im Bildschirm Optionen für Kontakte zur Lotus Notes-Datenbank und klicken auf Weiter.
  11. Prüfen Sie im Bildschirm Intellisync for SalesLogix-Verbindungsinformationen Ihre Sage SalesLogix-Benutzer-ID, Ihr Passwort sowie die Verbindung zu Ihrem Intellisync for SalesLogix-Portal, und klicken Sie anschließend auf Weiter.
  12. Klicken Sie auf Fertig stellen.
  13. Klicken Sie auf Schließen.
  14. Klicken Sie auf OK.
  15. Wählen Sie die Gruppe "SyncSalesLogix" aus, falls Sie dies nicht bereits getan haben.
