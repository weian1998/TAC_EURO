Intellisync-Integritätsprüfung

Intellisync for SalesLogix bietet zur Fehlerbehebung die Intellisync-Integritätsprüfung. Sie können zur Fehlerbehebung die Protokollierung im Portal aktivieren und die Version der Assemblys überprüfen. Das Überprüfen der Assemblyversionen erfolgt zumeist auf Anforderung des technischen Supports von Sage.

Der Zugriff auf die Intellisync-Integritätsprüfung erfolgt über den Internetinformationsdienste-Manager (IIS), das Intellisync-Portal und den Sage SalesLogix Personal Web Server.

So erfolgt der Zugriff über den Internetinformationsdienste-Manager

  1. Klicken Sie auf Start, zeigen Sie auf Programme und anschließend auf Verwaltung.
  2. Klicken Sie auf Internetinformationsdienste-Manager (IIS).
  3. Blenden Sie den Computernamen, dann Websites und schließlich SalesLogix ein.
  4. Doppelklicken Sie auf SLXIntellisync.
  5. Führen Sie einen der folgenden Schritte durch:
  • Windows 2003: Klicken Sie im rechten Bereich in der Dateiliste mit der rechten Maustaste auf Default.aspx, und klicken Sie auf Durchsuchen.
  • Windows 2008: Klicken Sie im rechten Bereich in der Dateiliste mit der rechten Maustaste auf Standarddokument, zeigen Sie auf Anwendung verwalten, und klicken Sie dann auf Durchsuchen. Das Intellisync-Portal wird im Browser geöffnet. Falls nicht, vergewissern Sie sich, dass die Einstellungen ordnungsgemäß konfiguriert sind und das Portal bereitgestellt wurde.
  1. Klicken Sie im Browser auf Diagnostics, und führen Sie einen der folgenden Schritte durch:
  • Um Benutzeranmeldedaten zu testen, geben Sie den Benutzernamen und das Passwort des zu testenden Benutzers ein und klicken auf Verbindung herstellen. Es wird eine Anforderung an den Server gesendet, um zu testen, ob die Anmeldeinformationen für das Portal gültig sind. Eine Meldung gibt an, ob der Anmeldeversuch erfolgreich war.

    • Dieser Test ermittelt nur, ob die Anmeldeinformationen für eine erfolgreiche Anmeldung beim Portal gültig sind. Es wird nicht getestet, ob eine funktionierende Datenbankverbindung vorliegt.

  • Um Assemblys und ihre Versionsnummer anzuzeigen, klicken Sie unten in der Liste auf die Schaltflächen Vorwärts und Zurück.

So erfolgt der Zugriff über das Portal

  1. Öffnen Sie das Intellisync-Portal.

Geben Sie z. B. den Portal-URL im folgenden Format in den Browser ein: http://Server:Port/Portalname/default.aspx

  1. Klicken Sie auf Diagnostics.
  1. Führen Sie einen der folgenden Schritte durch:
  • Um Assemblys und ihre Versionsnummer anzuzeigen, klicken Sie unten in der Liste auf die Schaltflächen Vorwärts und Zurück.
  • Um Benutzeranmeldedaten zu testen, geben Sie den Benutzernamen und das Passwort für den zu testenden Benutzer ein und klicken auf Verbindung herstellen.

Es wird eine Anforderung an den Server gesendet, um zu testen, ob sich der Benutzer beim Intellisync-Portal anmelden kann. Eine Meldung gibt an, ob der Test erfolgreich war.

So erfolgt der Zugriff über den Personal Web Server

  • So testen Sie Benutzeranmeldedaten
  1. Klicken Sie auf der Windows-Taskleiste mit der rechten Maustaste auf das Symbol für Sage SalesLogix Personal Web Server, und klicken Sie dann auf Eigenschaften.
  2. Klicken Sie im Dialogfeld Sage SalesLogix Personal Web Server auf Verwalten.
  3. Geben Sie den Benutzernamen und das Passwort für den zu testenden Benutzer ein.
  4. Klicken Sie auf Verbindung testen.

Es wird eine Anforderung an den Server gesendet, um zu testen, ob sich der Benutzer beim Intellisync-Portal anmelden kann. Eine Meldung gibt an, ob der Anmeldeversuch erfolgreich war.

  • So zeigen Sie Assemblys und ihre Versionsnummer an
  1. Klicken Sie auf der Windows-Taskleiste mit der rechten Maustaste auf das Symbol für Sage SalesLogix Personal Web Server, und klicken Sie dann auf Eigenschaften.
  2. Wählen Sie im Dialogfeld Sage SalesLogix Personal Web Server das Intellisync-Portal im Feld Website aus.
  3. Klicken Sie auf Details.

Prüfen Sie die installierten Assemblys für das Intellisync-Portal und ihre Versionsnummern.

  • So vergleichen Sie Portalversionen
  1. Klicken Sie auf der Windows-Taskleiste mit der rechten Maustaste auf das Symbol Sage SalesLogix Personal Web Server, und klicken Sie dann auf Gleiche Versionen.
  2. Prüfen Sie die Version der Datei "sage.saleslogix.dll" für jedes installierte Portal.
