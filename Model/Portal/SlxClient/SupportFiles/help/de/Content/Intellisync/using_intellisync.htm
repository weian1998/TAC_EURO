Arbeiten mit Intellisync for SalesLogix

Nach der Konfiguration synchronisiert Intellisync for SalesLogix Dateien zwischen Sage SalesLogix und Microsoft Outlook oder Lotus Notes gemäß dem Zeitplan, den Sie beim Konfigurieren festgelegt haben. Manchmal ist Folgendes erforderlich:

  • Durchführen der manuellen Synchronisierung

    Sie können eine Änderung an Sage SalesLogix-Daten unmittelbar abrufen oder sie an andere Benutzer sofort nach der Durchführung verteilen, anstatt den nächsten geplanten Synchronisierungszyklus abwarten zu müssen. Weitere Informationen zur erforderlichen Vorgehensweise finden Sie unter Manuelle Synchronisierung des Kalenders und von Kontakten.

    Beispiel: Sie werden vom Hauptsitz Ihres Unternehmens über eine wichtige Datenänderung bei einem der Kontakte in Ihrer SyncSalesLogix-Gruppe informiert. Sie möchten diese Änderung sofort abrufen und nicht auf den nächsten geplanten Synchronisierungszyklus warten. Weitere Informationen zur erforderlichen Vorgehensweise finden Sie im Sales Client-Hilfethema "Manuelle Synchronisierung des Kalenders und von Kontakten".

  • Hinzufügen eines Kontakts zu Ihrer SyncSalesLogix-Gruppe

    Die Gruppe "SyncSalesLogix" ist eine Ad-hoc-Gruppe, das heißt, sie basiert auf den Kontakten, die beim Erstellen der Gruppe ausgewählt wurden. Es ist wahrscheinlich notwendig, in regelmäßigen Abständen neue Kontakte zu der Gruppe hinzuzufügen. Weitere Informationen zur erforderlichen Vorgehensweise finden Sie unter dem Thema Hinzufügen von Kontakten zur Gruppe "SyncSalesLogix".

  • Verstehen, wie abgeschlossene Einträge aktualisiert werden

    Wenn Sie einen Eintrag in Sage SalesLogix abschließen, der in Microsoft Outlookoder Lotus Notes erstellt wurde, wird der Eintrag in Microsoft Outlookoder Lotus Notes erst im nächsten Synchronisierungszyklus aktualisiert.

  • Für in Sage SalesLogix abgeschlossene Aufgaben werden in Microsoft Outlook oder Lotus Notes Alarme entfernt.
  • In Sage SalesLogix abgeschlossene Aufgaben werden in Microsoft Outlook oder Lotus Notes als abgeschlossen gekennzeichnet.
  • Löschen von Aktivitäten-, Kontakt- oder Ereignisdatensätzen
  • Verstehen, wie Konflikte gelöst werden
  • Weitere Hilfe zum Setup
  • Beheben doppelter Kalendereinträge

    Wenn Sie Intellisync for SalesLogix für die Synchronisierung von Aktivitäten und Ereignissen mit Microsoft Outlook oder Lotus Notes verwenden, werden unter folgenden Umständen Duplikate erzeugt:

  • Sie erstellen das Ereignis in Sage SalesLogix und
  • synchronisieren das Ereignis mit Outlook oder Lotus Notes und
  • ändern das Ereignis anschließend in Outlook oder Lotus Notes in ein sich wiederholendes Ereignis.

    • Wird das Ereignis wieder mit Sage SalesLogix synchronisiert, wird es sich wiederholendes Ereignis ohne Zeitangabe angezeigt, doch das ursprüngliche Ereignis bleibt ebenfalls im Kalender. Um Verwirrung zu vermeiden, können Sie das ursprüngliche Ereignis manuell löschen. Weitere Informationen finden Sie unter Löschen von Aktivitäten, Kontakten oder Ereignissen.
