Grundlegendes zu Faktoren mit Auswirkung auf die Synchronisierung

Die Dauer der Synchronisierung von Daten zwischen Sage SalesLogix und Microsoft Outlook oder Lotus Notes hängt von mehreren Faktoren ab. Der wichtigste Faktor ist das Datenvolumen, das zwischen den Systemen übertragen wird. Sie können die Performance verbessern, indem Sie das durch Intellisync for SalesLogix übertragene Datenvolumen steuern. Dazu gibt es mehrere Möglichkeiten.

Letztes Synchronisierungsdatum

Beim Datenaustausch wird das Datum in einer Protokolldatei aufgezeichnet. Bei nachfolgenden Synchronisierungen wird auf Änderungen seit dem letzten Synchronisierungsdatum geprüft und es werden nur Daten übertragen, die sich seit diesem Datum geändert haben. Durch regelmäßige Synchronisierungen können Sie das Datenvolumen verringern, das während der einzelnen Zyklen übertragen werden muss.  

Sicherheit

Die integrierte Datensicherheit, die mit dem bei Sage SalesLogix angemeldeten Benutzer verknüpft ist, wirkt sich ebenfalls auf die Synchronisierung aus. Die Sicherheitsrechte des angemeldeten Benutzers begrenzen die Menge der übertragbaren Daten.

Aufheben der Zuordnung von Feldern

Intellisync for SalesLogix gibt Benutzern die Möglichkeit zu definieren, wie die Daten zwischen Anwendungen zugeordnet werden. Wird die Zuordnung für ein Feld aufgehoben, beendet der Übersetzer die Datenübertragung in dieses Feld. Gleichermaßen gilt, dass das Zuordnen eines bisher nicht zugeordneten Felds die Synchronisierung von Daten zwischen den Anwendungen in dieses Feld startet. Weitere Informationen finden Sie unter Grundlegendes zur Feldzuordnung bei Intellisync for SalesLogix.
