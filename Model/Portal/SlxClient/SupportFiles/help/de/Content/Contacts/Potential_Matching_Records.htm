Potenziell übereinstimmende Datensätze

Wenn Sie einen Kontakt oder eine Firma hinzufügen, können Sie den Kontakt oder die Firma, den bzw. die Sie hinzufügen, mit Kontakten oder Firmen vergleichen, die in der Datenbank enthalten sind. Wenn in der Datenbank ein Datensatz mit identischen Informationen enthalten ist, können Sie dadurch doppelte Datensätze vermeiden. Wenn Sie beschließen, eine übereinstimmende Firma zu verwenden, werden die Firmeninformationen zu den Firmenfeldern im Dialog Kontakt-/Firmeninformationen hinzugefügt. Wenn die Kontakt- oder Firmeninformationen nicht übereinstimmen, können Sie fortfahren, Ihren Kontakt oder Ihre Firma hinzuzufügen.

So suchen Sie nach übereinstimmenden Datensätzen

  1. Geben Sie Kontakt- und Firmeninformationen im Dialog ein, und klicken Sie danach auf die Schaltfläche Nach übereinstimmenden Datensätzen suchen.
  2. Wählen Sie neben Typ die zu suchenden Datensatztypen aus.
  3. Aktivieren Sie unter Mit Filtern abgleichen die Kontrollkästchen der Filter, die Sie anwenden möchten.
  4. Wählen Sie eine der folgenden Optionen aus:
    • Mit allen ausgewählten Filtern abgleichen: Ein Datensatz ist nur ein Duplikat, wenn er allen ausgewählten Filtern entspricht.
    • Mit ausgewählten Filtern abgleichen: Ein Datensatz ist ein Duplikat, wenn er einem der ausgewählten Filter entspricht.
  5. Um eine erweiterte Suche durchzuführen, klicken Sie auf Erweiterte Optionen.
  6. Klicken Sie auf Potenzielle Übereinstimmungen aktualisieren.
  7. Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie im Raster Übereinstimmende Kontakt-/Interessentendatensätze neben einem potenziell übereinstimmenden Datensatz auf Öffnen, um zusätzliche Informationen zu diesem Datensatz anzuzeigen.
    • Klicken Sie im Raster Überstimmende Firmendatensätze auf Firma verwenden, um den Kontakt zu einer vorhandenen Firma hinzuzufügen.
    • Klicken Sie auf Abbrechen, um zu den Kontakt-/Firmeninformationen zurückzukehren und weitere Kontakte und Firmen einzufügen.
