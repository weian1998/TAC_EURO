Kopieren von Informationen in die Zwischenablage

Die Informationen werden in die Zwischenablage kopiert, damit Sie sie in anderen Anwendungen verwenden können. Das ist hilfreich, wenn Sie Informationen in eine E-Mail oder ein Microsoft Word-Dokument kopieren müssen.

Wenn Sie Firefox als Browser verwenden, steht diese Funktion nicht zur Verfügung.

So kopieren Sie Kontaktinformationen aus einem Kontakt

  1. Öffnen Sie die Kontaktdetailansicht.
  2. Klicken Sie, falls verfügbar, auf die Schaltfläche Kontaktinformationen in Zwischenablage kopieren.
  3. Wenn das Dialogfeld Text in Zwischenablage kopieren geöffnet wird, wählen Sie den zu kopierenden Text aus, drücken STRG + C und klicken dann auf Schließen.

So kopieren Sie Kontaktinformationen aus einer Firma

  1. Öffnen Sie die Firmendetailansicht.
  2. Klicken Sie, falls verfügbar, auf die Schaltfläche Kontaktinformationen in Zwischenablage kopieren.
  3. Wenn das Dialogfeld Text in Zwischenablage kopieren geöffnet wird, wählen Sie den zu kopierenden Text aus, drücken STRG + C und klicken dann auf Schließen.
