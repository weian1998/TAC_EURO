Hinzufügen einer neuen Firma

Über das Dialogfeld Firma hinzufügen können Sie eine neue Firma hinzufügen.

Durch Ihren Sicherheitszugriff wird bestimmt, welche Funktionen verfügbar sind. Wenden Sie sich an den Administrator, wenn Sie Ihre Zugriffsrechte ändern möchten. Für WebViewer-Benutzer kann die Funktionalität eingeschränkt sein.

So fügen Sie Informationen hinzu

  1. Klicken Sie im Menü Neu auf Firma.
  2. Füllen Sie die übrigen Firmeninformationsfelder gemäß den Richtlinien Ihres Unternehmens aus.
  3. Klicken Sie auf Speichern.
