Zuweisen einer Firma zu einem Team

Sie können einer Firma einem Benutzerteam zuweisen, wenn mehrere Benutzer Zugriff auf dieselben Firmen benötigen oder mit denselben Firmen arbeiten. Die Bildung eines Teams basiert häufig auf Kriterien wie Gebiet, Region, Firmentyp oder Abteilung.

Wenn ein Benutzer einem Team hinzugefügt wird, wird sein Standardsicherheitsprofil verwendet.

Ein Benutzer kann mehreren Teams angehören.

Ein Team kann über ein eigenes Sicherheitsprofil verfügen, das die Zugriffsrechte für das Team definiert. Nachdem ein Benutzer einem Team hinzugefügt wurde, können Sie das Sicherheitsprofil des Benutzers ändern, damit es mit dem Profil des Teams übereinstimmt. Wenn Sie ein Benutzersicherheitsprofil in einem Team ändern, hat das keine Auswirkungen auf das Standardprofil des Benutzers oder auf das Profil in anderen Teams, in denen der Benutzer Mitglied ist.

So weisen Sie einem Team eine Firma zu

  1. Öffnen Sie den Dialog Zuweisung des Zugriffsberechtigten mit einem der folgenden Verfahren:
    • Wenn Sie eine neue Firma eingeben, öffnen Sie den Dialog Firmeninformationen bearbeiten und doppelklicken auf das Feld Zugriffsberechtigter.
    • Wenn Sie mit einer vorhandenen Firma arbeiten, öffnen Sie die Firmen-Detailansicht und doppelklicken auf das Feld Zugriffsberechtigter.
  2. Klicken Sie auf die Registerkarte Teams.
  3. Wenn nur die Teams angezeigt werden sollen, denen Sie angehören, aktivieren Sie das Kontrollkästchen Eigene Teams.
    Wenn Sie als Administrator angemeldet sind, ist das Kontrollkästchen Eigene Teams
    nicht verfügbar.
  4. Wählen Sie ein Team in der Liste aus, und klicken Sie dann auf OK.
