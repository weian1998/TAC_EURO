Detailansicht "Firmenbeziehung"

Verwenden Sie Beziehungen, um Verbindungen zwischen Firmen herzustellen.

Wenn Sie zum Beispiel in einer Firma mit vielen Geschäftsbereichen arbeiten, können Sie für jeden Geschäftsbereich eine Firma anlegen (Geschäftsbereichsfirmen) und außerdem noch eine Firma, die das Unternehmen als Ganzes repräsentiert (übergeordnete Firma). Anschließend können Sie jede Geschäftsbereichsfirma mit der übergeordneten Firma verknüpfen. Wenn es Kontakte in verschiedenen Geschäftsbereichen der übergeordneten Firma gibt, können Sie die Kontakte verknüpfen.

Für jede Firma können theoretisch mehrere Firmenbeziehungen existieren. In der Detailansicht Firmenbeziehung können Sie detaillierte Informationen zu den Beziehungen einer Firma zu einer anderen Firma anzeigen. Diese Ansicht umfasst die Namen der Firmen, eine kurze Beschreibung sowie etwaige Anmerkungen.

So verknüpfen Sie Firmen

  1. Klicken Sie in der Ansicht Firmendetail auf die Registerkarte Beziehungen. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  2. Klicken Sie auf die Schaltfläche Beziehung hinzufügen.
  3. Klicken Sie auf die Schaltfläche Suchen und suchen Sie die Firma, die Sie mit der ersten Firma verknüpfen möchten.
  4. Klicken Sie auf das Dropdown-Pfeil unter der Firma und wählen Sie ein Element in der Liste aus, das die Beziehung der ersten Firma am besten beschreibt.
  5. Klicken Sie auf das Dropdown-Pfeil unter der zweiten Firma und wählen Sie ein Element in der Liste aus, das die Beziehung der zweiten Firma am besten beschreibt.
  6. Geben Sie in die Felder Beschreibung für jede Firma eine Beschreibung ein.

  7. Klicken Sie auf OK.

So bearbeiten Sie Informationen

  1. Klicken Sie in der Ansicht Firmendetail auf die Registerkarte Beziehungen.
  2. Klicken Sie auf den Hyperlink Bearbeiten für die Beziehung, die Sie anzeigen oder bearbeiten möchten.
  3. Nehmen Sie die Änderungen vor.

  4. Klicken Sie auf OK.
