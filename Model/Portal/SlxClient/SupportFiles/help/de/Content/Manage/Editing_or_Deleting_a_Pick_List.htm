Bearbeiten oder Löschen einer Auswahlliste

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Sie können eine Auswahlliste bearbeiten, indem Sie sie umbenennen, ihre Einträge bearbeiten oder die Auswahlliste löschen.

Bei Ändern des Namens einer Auswahlliste funktioniert benutzerdefinierter Code, der auf die Auswahlliste verweist, nicht mehr.

So bearbeiten Sie Informationen

  1. Öffnen Sie die Auswahllisten-Detailansicht.
  2. Führen Sie einen der folgenden Schritte durch:
  • Geben Sie zum Umbenennen der Auswahlliste in das Feld Name einen neuen Namen ein.
  • Ändern Sie die Auswahllistenattribute den Anforderungen entsprechend.
  • Ergänzen, bearbeiten oder ändern Sie die Auswahllistenattribute den Anforderungen entsprechend.
  1. Klicken Sie auf Speichern.

So gehen Sie zum Löschen vor

  1. Öffnen Sie die Auswahllisten-Detailansicht.
  2. Klicken Sie auf Löschen.
  3. Klicken Sie auf OK, um den Löschvorgang zu bestätigen.
