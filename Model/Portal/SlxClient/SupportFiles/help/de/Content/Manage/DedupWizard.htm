Verwenden von "Nach Duplikaten suchen"

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Der Assistent für die Duplikatsuche vergleicht Datensätze in Sage SalesLogix, um zu bestimmen, ob Ihre Daten Duplikate enthalten. Der Assistent erstellt einen Auftrag, der mögliche Duplikatdaten enthält. Anhand der Auftragsergebnisse können Sie doppelte Daten zusammenführen oder entfernen.

SpeedSearch muss für die Verwendung von Nach Duplikaten suchen konfiguriert werden.

So gehen Sie zur Ausführung vor

  1. Klicken Sie im Menü Extras auf Nach Duplikaten suchen.
  2. Gehen Sie auf dem Bildschirm Wählen Sie eine Quelle aus wie folgt vor:
    1. Wählen Sie in der Dropdown-Liste Wählen Sie den Auftragstyp aus den Entitätstyp aus, den Sie auf Duplikate überprüfen möchten.
    2. Wählen Sie in der Dropdown-Liste Gruppe die Gruppe der Datensätze im ausgewählten Entitätstyp aus.
    3. Klicken Sie auf Weiter.
  1. Auf dem Bildschirm Suchoptionen können Sie die Filter auswählen, die der Assistent beim Abgleichen von Daten verwenden soll.
    Wenn Sie einen Wert auswählen, müssen die Daten in beiden Datensätze exakt übereinstimmen, um als Duplikat betrachtet zu werden.
  2. Klicken Sie auf die Registerkarte Erweiterte Übereinstimmungsoptionen, um weitere Übereinstimmungsoptionen festzulegen. Klicken Sie auf OK, wenn Sie fertig sind.
  3. Klicken Sie auf Weiter.
  4. Überprüfen Sie auf dem Bildschirm Überprüfen die für diesen Auftrag gewählten Einstellungen.
  5. Klicken Sie auf Senden.
  6. Klicken Sie zum Abbrechen eines laufenden Auftrags auf Abbrechen.
  7. Klicken Sie bei Bedarf auf den Hyperlink Auftragsnummer, um die Detailansicht "Historie der Duplikatsuche" zu öffnen und etwaige potenzielle Duplikate anzuzeigen.
