Hinzufügen eines Produktpakets

Sie können Pakete der Paketliste hinzufügen. Diese Liste steht zur Verfügung, wenn Verkaufschancen- oder Verkaufsauftragsprodukte hinzugefügt oder bearbeitet werden.

So fügen Sie Informationen hinzu

  1. Klicken Sie im Im Menü Administration auf Neues Paket.
    Das Dialogfeld Paket einfügen wird geöffnet.
  2. Füllen Sie die Informationsfelder für die Produktpakete aus:
    • Name - Geben Sie den Namen des Produktpakets ein.
    • Beschreibung - Geben Sie eine Beschreibung des Produktpakets ein.
    • Status - Klicken Sie auf die Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  3. Klicken Sie auf Speichern
  4. Führen Sie die folgenden Schritte durch, um dem Paket Produkte hinzuzufügen:
    1. Öffnen Sie in der Paketdetailansicht die Registerkarte Produkte, und klicken Sie auf Produkt verknüpfen.
    2.  Markieren Sie ein oder mehrere Produkte, und klicken Sie auf Auswahl hinzufügen.
    3. Ändern Sie bei Bedarf die Menge. Wählen Sie den Datensatz aus, den Sie bearbeiten möchten, nehmen Sie Ihre Änderungen im Raster vor, und klicken Sie auf Speichern.. Um Ihre Änderungen rückgängig zu machen, klicken Sie auf Zurücksetzen, bevor Sie auf Speichern klicken.
      Es können nicht alle Spalten im Raster bearbeitet werden. Zum Bestimmen, ob eine Spalte bearbeitet werden kann, klicken Sie auf das Raster. Wenn ein weißes Feld angezeigt wird, kann die Spalte bearbeitet werden.
