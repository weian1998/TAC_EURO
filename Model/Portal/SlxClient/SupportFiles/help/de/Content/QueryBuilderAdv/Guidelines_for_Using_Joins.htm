Richtlinien für den Einsatz von Joins

Gehen Sie beim Erstellen von Tabellen-Joins in Sage SalesLogix nach den folgenden Richtlinien vor:

  • Verwenden Sie in einer Ansicht mit Feldern aus mehreren Tabellen "Left Joins". Wenn Sie "Inner Joins" verwenden und sich in einer der Tabellen keine Daten befinden, wird in JEDEM Feld in der Ansicht der Hinweis NO DATA (Keine Daten) angezeigt.
  • Verwenden Sie in einer Gruppe "Inner Joins" für die in Bedingungen verwendeten Tabellen. Auf diese Weise rufen Sie nur Datensätze ab, die den angegebenen Kriterien entsprechen. Sie können jedoch auch "Left Joins" für im Layout verwendete Tabellen verwenden, sodass alle Datensätze, die Ihren Bedingungen entsprechen, eingeschlossen werden, und zwar auch dann, wenn eine der Tabellen keine Daten enthält.

Standardmäßig ist die Tabelle ADRESSE mit der Tabelle FIRMA über einen Inner Join verbunden. Die Eigenschaften dieses Join dürfen nicht geändert werden.
