Verwenden eines Werts als Literal

Die Auswahl Wert literal verwenden im Dialogfeld Bedingung zuweisen platziert den Text genau so in die Abfrage, wie er auch im Feld Wert angezeigt wird. In den meisten Fällen sollte diese Option nicht aktiviert werden, damit der Abfrage-Assistent die richtige SQL-Anweisung bestimmen kann. Erfahrene Benutzer können in Ausnahmefällen jedoch einen bestimmten Text eingeben, der direkt in die SQL-Anweisung geschrieben wird.

Sie können z. B. eine datenbankspezifische Systemfunktion eingeben (z. B. GETDATE in einer MS SQL Server-Datenbank) und Wert nicht für SQL formatieren auswählen. So stellen Sie sicher, dass GETDATE in die SQL-Anweisung ohne Anführungszeichen aufgenommen wird.
