Zuweisen eines Abfragelayouts

Im Dialog Abfragelayout zuweisen können Sie das Design der Datenspalten (oder Datenfelder) in der Abfrage festlegen.

So legen Sie Layoutoptionen fest

  1. Öffnen Sie den Abfrage-Assistenten und klicken Sie auf die Registerkarte Layout.
  2. Fügen Sie ein Layout-Feld hinzu oder bearbeiten Sie ein vorhandenes Feld.
  3. Geben Sie im Feld Beschriftung den Namen ein, der als Spaltenüberschrift angezeigt wird.
  4. Wählen Sie im Feld Breite die Spaltenbreite aus. Der Standardwert ist 100.

Alle Änderungen der Gruppenbreite bleiben nur während der aktuellen Sitzung aktiv.

  1. Wählen Sie im Feld Beschrift.-Ausrich. die Ausrichtung der Spaltenüberschrift aus.
  2. Wählen Sie im Feld Textausrichtung die Ausrichtung der Daten aus.
  3. Wählen Sie im Feld Formattyp das entsprechende Format für den Datentyp in diesem Feld aus.

    Mit dem Formattyp Telefon wird der Wert des Feldes als 10-stellige Telefonnummer nach US-Standard formatiert. Wenn in Ihrem Unternehmen nicht das 10-stellige US-Format verwendet wird, kann der Formattyp für Telefonnummernfelder auf Keine eingestellt werden, damit keine Telefonnummernmaske angewendet wird.

  4. Geben Sie ggf. eine Formatzeichenfolge in das Feld Format ein, um den gewählten Standardformattyp zu ändern.
  5. Um die Informationen in der Spalte mit der Portalansicht einer Entität zu verbinden, wählen Sie Entität - Verbunden aus.
  6. Um die Formatierung eines Cascading Style Sheets (CSS) zu übernehmen, geben Sie in das Feld CSS-Klasse die Klasse ein, die Sie anwenden möchten.
