Hinzufügen und Bearbeiten von berechneten Feldern

 Nur für Systemadministratoren

So öffnen Sie den Dialog "Berechnete Felder verwalten"

  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Schaltfläche Formeln.
So gehen Sie zum Hinzufügen vor
  1. Öffnen Sie den Abfrage-Assistenten, und klicken Sie anschließend auf die Schaltfläche Formeln.

Diese Schaltfläche wird nur angezeigt, wenn Sie als Administrator angemeldet sind.

  1. Klicken Sie auf Hinzufügen.
  2. Klicken Sie auf die Registerkarte Eigenschaften.
  3. Geben Sie in das erste Feld Name/Alias einen Namen ein, der den Zweck der Formel beschreibt.

Der Name der Formel wird nur im Formel-Manager angezeigt.

  1. Geben Sie in das zweite Feld einen Namen ein, der in der Formel verwendet werden soll. Dieser Name darf keine Leerzeichen oder Symbole enthalten. Sie müssen einen Wert in dieses Feld eingeben. Sie können Formelfelder erst erstellen, wenn dieses Feld einen Wert enthält.

Wenn ein Formelfeld beim Erstellen eines Plug-Ins oder einer Abfrage verwendet wird, wird dieser Name in der Liste der verfügbaren Felder angezeigt.

  1. Klicken Sie auf den Dropdown-Pfeil Basistabelle, und wählen Sie die Haupttabelle aus, die Sie zum Erstellen des berechneten Felds verwenden.

Beim Erstellen von einem Tabellen-Join ist dies die übergeordnete Tabelle.

  1. Klicken Sie auf den Dropdown-Pfeil Berechnungstyp, und wählen Sie einen Berechnungstyp aus.
  2. Geben Sie eine Beschreibung ein, wenn Sie eine ausführlichere Beschreibung des verfügbaren Felds wünschen.
  3. Klicken Sie auf die Registerkarte Formel.
  4. Stellen Sie fest, ob die gewünschten Werte für das berechnete Feld in der ausgewählten Basistabelle enthalten sind.

Die in der oberen Listenansicht gezeigte Tabelle ist die Tabelle, die Sie auf der Registerkarte Eigenschaften ausgewählt haben.

  1. Wenn die Basistabelle nicht die benötigten Werte enthält, sollten Sie die Basistabelle ändern, anstatt keinen Join zu erstellen. Klicken Sie auf die Registerkarte Eigenschaften und wählen Sie eine andere Basistabelle aus.
  2. Erstellen Sie die Formel.
  3. Klicken Sie auf das Zeichen + oder -, um die Liste der Tabellen und Felder zu erweitern bzw. zu reduzieren.
  4. Zum Ergänzen eines Feldes klicken Sie im oberen Kästchen auf das Feld, um es an eine beliebige Position innerhalb der Formel einzufügen.

Dies ist das erste Feld in der Formel.

  1. Führen Sie einen der folgenden Schritte durch:
  • Klicken Sie auf einen Operator, wenn Sie eine numerische Berechnung durchführen.

Sie müssen zwischen den Feldern einer numerischen Formel einen Operator einfügen.

  • Bei einer Zeichenfolge ist kein Operator notwendig, Sie sollten aber aus Gründen der Übersichtlichkeit evtl. eine Leerstelle einfügen.
  1. Klicken Sie auf ein weiteres Feld in der Strukturansicht, um es in die untere Registerkartenansicht zu übernehmen, oder tragen Sie eine Ganzzahl ein.
  2. Sie können weitere Felder hinzufügen, um das berechnete Feld zusammenzustellen.
  3. Mit Klammern können Sie Teile Ihrer Formeln gruppieren. Platzieren Sie den Cursor zum Einfügen einer Klammer an der gewünschten Stelle in der Formel, und klicken Sie dann auf eine Klammer-Schaltfläche. Sie müssen sowohl eine öffnende als auch eine schließende Klammer einfügen.
  4. Klicken Sie auf OK.
  5. Klicken Sie auf Schließen.
So gehen Sie zum Bearbeiten vor
  1. Wählen Sie im Dialog Berechnete Felder verwalten die Berechnung aus, und klicken Sie dann auf Bearbeiten
  2. Klicken Sie im Dialog Formel bearbeiten auf die Registerkarte Formel.
  • Zum Ergänzen eines Feldes klicken Sie im oberen Kästchen auf das Feld, um es an eine beliebige Position innerhalb der Formel einzufügen.
  • Platzieren Sie den Cursor zum Hinzufügen eines Operators oder einer Klammer innerhalb der Formel und klicken Sie auf eine Operator- oder Klammer-Schaltfläche. Ein Operator wird zwischen Feldern in einer numerischen Berechnung benötigt und Klammern müssen aus einer öffnenden und einer schließenden Klammer bestehen.
  • Zum Löschen eines Eintrags in der Formel markieren Sie diesen und klicken Sie dann auf Löschen.
  1. Klicken Sie auf OK.
  2. Klicken Sie auf Schließen.
