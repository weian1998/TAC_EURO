Beispiel: Erstellen einer Gruppe mit dem Abfrage-Assistenten

Im folgenden Beispiel wird erläutert, wie eine Gruppe mit allen Hauptkontakten aus den jeweiligen Vertriebsabteilungen der Unternehmen erstellt wird.

  1. Melden Sie sich beim Sage SalesLogix Web Client an.
  2. Klicken Sie auf der Navigationsleiste auf Kontakte.
  3. Klicken Sie im Menü Gruppen auf Gruppe hinzufügen.
  4. Geben Sie im Abfrage-Assistenten auf der Registerkarte Eigenschaften Folgendes ein:
  5. Im Feld Name geben Sie Primäre Vertriebskontakte ein.
  6. Im Feld Beschreibung geben Sie Hauptkontakte aus dem Vertrieb ein.
  7. Füllen Sie auf der Registerkarte Bedingungen Folgendes aus:
  1. Wählen Sie oben links im Abfrage-Assistenten die Option KONTAKT.
  2. Doppelklicken Sie oben rechts im Abfrage-Assistenten auf Ist als "Haupt" festgelegt.

Das Dialogfeld Bedingung zuweisen wird mit Ist als "Haupt" festgelegt im Feld Feldname und Zeichenfolge als Feldtyp geöffnet.

  1. Wählen Sie im Feld Operator den Operator (=) Gleich aus.
  2. Klicken Sie neben dem Feld Wert auf Durchsuchen, wählen Sie T (für "True") und klicken Sie danach auf OK.

Lassen Sie das Kontrollkästchen Groß/Kleinschreibung berücksichtigender Vergleich aktiviert und deaktivieren Sie das Kontrollkästchen Wert nicht für SQL formatieren.

  1. Klicken Sie auf OK.
  2. Doppelklicken Sie oben rechts im Abfrage-Assistenten auf Abteilung.

Das Dialogfeld Zuweisungsbedingung wird mit "Abteilung" im Feld Feldname und "Zeichenfolge" als Feldtyp geöffnet.

  1. Wählen Sie im Feld Operator die Option enthält aus.
  2. Klicken Sie neben dem Feld Wert auf Durchsuchen, wählen Sie Vertrieb aus und klicken Sie dann auf OK.
  3. Deaktivieren Sie das Kontrollkästchen Groß/Kleinschreibung und wählen Sie nicht die Option Wert literal verwenden aus.
  4. Klicken Sie auf OK.
  1. Klicken Sie auf die Registerkarte Layout. Auf der Registerkarte Layout wird dargestellt, welche Felder angezeigt werden, wenn die Abfrage als Gruppe gespeichert wird. Führen Sie folgende Schritte aus:
  2. Wählen Sie auf der rechten Seite des Abfrage-Assistenten das Feld Abteilung und doppelklicken Sie darauf, um den Eintrag in das darunter angezeigte Raster zu übernehmen.
  1. Führen Sie einen Bildlauf mit der Bildlaufleiste ganz nach rechts durch und klicken Sie auf Abteilung.
  2. Klicken Sie so oft auf die Schaltfläche Nach links, bis Abteilung zwischen Firma und Stadt angezeigt wird.
  1. Klicken Sie auf die Registerkarte Sortieren. Standardmäßig werden Kontakte aufsteigend (A-Z) nach Nachname sortiert. Im Bedarfsfall können Sie weitere Sortierkriterien hinzufügen oder die Reihenfolge ändern.
  2. Behalten Sie auf der Registerkarte Standardeinstellungen die Einstellungen bei.
  3. Klicken Sie auf OK.

Die neue Gruppe mit den Hauptvertriebskontakten wird in der Liste Kontaktgruppe aufgeführt. Sie können sie auswählen, um alle Kontakte anzuzeigen, die den von Ihnen festgelegten Kriterien entsprechen.
