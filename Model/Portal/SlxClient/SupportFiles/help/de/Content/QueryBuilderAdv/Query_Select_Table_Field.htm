Auswählen von Tabellen und Feldern

Sie können ein Feld hinzufügen, das in einer Abfrage angezeigt werden soll.

So wählen Sie eine Tabelle aus

  • Wählen Sie auf der linken Seite des Abfrage-Assistenten eine Tabelle aus. Klicken Sie bei Bedarf auf das Pluszeichen, um die Liste der Tabellen zu erweitern.

So wählen Sie ein Feld aus

  • Wählen Sie auf der rechten Seite des Abfrage-Assistenten ein Feld aus. Hier werden alle Felder der ausgewählten Tabelle angezeigt. Doppelklicken Sie auf das Feld, um es in die Tabelle unten im Dialog aufzunehmen.
