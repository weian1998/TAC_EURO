Registerkarte "Client-System"

Auf der Registerkarte Client-System in der Benutzerdetailansicht können Sie die Basisvorlagen eines Benutzers bestimmen und den Standardzugriffsberechtigten von Firmendatensätzen festlegen, die von diesem Benutzer erstellt werden. Benutzer können diese Einstellungen auf der Registerkarte Allgemein in Optionen ändern.

So legen Sie Optionen fest

  1. Klicken Sie in der Benutzerdetailansicht auf die Registerkarte Client-System.
  2. Legen Sie im Abschnitt Standardzugriffsberechtigter den Standardzugriffsberechtigten für neue Firmen fest.
    1. Wählen Sie im Feld Firmen den Standardzugriffsberechtigten für neue Firmen aus, die von diesem Benutzer erstellt werden.
    2. Deaktivieren Sie das Kontrollkästchen Benutzer darf diese Einstellungen ändern, um den Benutzer daran zu hindern, die Einstellung Standardzugriffsberechtigter zu ändern. Weitere Informationen...

    3. Wenn Sie einem Benutzer erlauben, diese Einstellung zu ändern, können die Zugriffsrechte an einer Firma standardmäßig einer Person zufallen, die sich nicht an feststehende Geschäftsregeln hält. Dies kann zur Folge haben, dass manche Firmen in eingerichteten Berichten oder Abfragen nicht berücksichtigt werden.

  3. Wählen Sie die Seriendruckbasisvorlagen aus, die Sie bei der Erstellung neuer Kontakt-, E-Mail-, Fax- oder Briefvorlagen verwenden möchten.
    1. Klicken Sie auf den Dropdown-Pfeil Typ und wählen Sie Kontakt aus.
    2. Klicken Sie neben E-Mail-Basisvorlage auf die Schaltfläche Suchen.

    3. Wählen Sie eine Vorlage aus, und klicken Sie danach auf OK.

    4. Wiederholen Sie die Schritte b bis c für Briefbasisvorlage und Faxbasisvorlage.

  4. Wählen Sie die Seriendruckbasisvorlagen aus, die Sie bei der Erstellung neuer Interessenten-, E-Mail-, Fax- oder Briefvorlagen verwenden möchten.

    Wählen Sie beim Auswählen von Vorlagen für Interessenten nur Vorlagen aus, die mit "(Interessent)" enden.

    1. Klicken Sie auf den Dropdown-Pfeil Typ und wählen Sie Interessent aus.
    2. Click the E-mail Base Template Find button.

    3. Wählen Sie eine Vorlage aus, und klicken Sie danach auf OK.

    4. Wiederholen Sie die Schritte b bis c für Briefbasisvorlage und Faxbasisvorlage.
  5. (Optional) Deaktivieren Sie das Kontrollkästchen Benutzer darf diese Einstellungen ändern, um den Benutzer daran zu hindern, die Standardvorlagen zu ändern.
  6. Klicken Sie auf Speichern.

Beim Konfigurieren von WebViewer-Benutzern ist nur der Abschnitt Standardzugriffsberechtigter verfügbar.
