Zurücksetzen gesperrter Benutzer

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Sage SalesLogix muss für einen Benutzer gesperrt werden, wenn die Anzahl fehlgeschlagener Anmeldeversuche die zulässige Anzahl der Versuche überschreitet. Der Schwellenwert für Anmeldeversuche wird in Administrator auf der Registerkarte Passwort festgelegt. Wenn die Sperre eines Benutzers zurückgesetzt wird, ändert sich die Anzahl der Versuche in 0, sodass der Benutzer wieder versuchen kann, sich anzumelden.

So setzen Sie einen oder mehrere gesperrte Benutzer zurück

  1. Klicken Sie auf der Navigationsleiste auf Administration., und klicken Sie dann auf Benutzer.
  2. Öffnen Sie die Gruppe Benutzer mit fehlgeschlagener Anmeldung.
  3. Klicken Sie im Aufgabenbereich unter Allgemeine Aufgaben auf Benutzer zurücksetzen.
