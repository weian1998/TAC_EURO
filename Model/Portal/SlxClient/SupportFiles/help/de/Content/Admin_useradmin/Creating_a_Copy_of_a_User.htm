Erstellen einer Kopie von Benutzern

Wenn Sie eine Kopie eines vorhandenen Benutzerkontos erstellen, werden alle Daten mit Ausnahme benutzerspezifischer Informationen wie Benutzername, E-Mail-Adresse und anderer persönlicher Informationen kopiert.

Ein inaktiver Benutzer kann nicht kopiert werden. Sie können aber das Profil eines inaktiven Benutzers als Vorlage für einen neuen Benutzer verwenden.

So gehen Sie zum Erstellen vor

  1. Öffnen Sie die Benutzerdetailansicht des Benutzers, den Sie kopieren möchten.
  2. Klicken Sie im Bereich Allgemeine Aufgaben auf Benutzer kopieren.
  3. Klicken Sie auf OK, um den Vorgang zu bestätigen.

Der neue Benutzer wird in der Benutzerliste als <Kopie von Benutzername> angezeigt.

  1. Ändern Sie das Benutzerprofil mit Informationen, die spezifisch für den neuen Benutzer sind.
  2. Klicken Sie auf Speichern.
