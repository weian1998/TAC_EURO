Registerkarte "Rollen"

In der Benutzerdetailansicht können Sie auf der Registerkarte Rollen einem Benutzer eine Rolle zuweisen oder von diesem entfernen. Wenn ein Benutzer einer Rolle zugewiesen ist, hat der Benutzer Zugriff auf die in dieser Rolle definierten Aktionen. Wenn ein Benutzer mehreren Rollen zugewiesen ist, hat er Zugriff auf alle gesicherten Aktionen in diesen Rollen, auch wenn die einzelnen Rollen nicht dieselben Aktionen enthalten.

Vorgehensweise

Öffnen der Registerkarte Rollen

  1. Öffnen Sie die Benutzerdetailansicht.
  2. Klicken Sie auf die Registerkarte Rollen.

Zuweisen eines Benutzers zu einer Rolle

  1. Klicken Sie auf Rolle hinzufügen.
  2. Wählen Sie über die Suchfunktion die Rollen aus, denen Sie den Benutzer zuweisen möchten.
  3. Klicken Sie auf Auswahl hinzufügen.
    Dieselbe Rolle kann nur ein Mal hinzugefügt werden.

Entfernen eines Benutzer aus einer Rolle

  1. Wählen Sie den Datensatz aus, den Sie entfernen möchten, und klicken Sie auf die Schaltfläche Entfernen.
  2. Klicken Sie auf Ja.
