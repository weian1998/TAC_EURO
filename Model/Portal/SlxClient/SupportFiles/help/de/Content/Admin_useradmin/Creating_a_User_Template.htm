Erstellen von Benutzervorlagen

In Benutzervorlagen werden die Einstellungen definiert, die Sie beim Erstellen neuer Benutzerkonten verwenden möchten. Es wird empfohlen, mit Vorlagen zu arbeiten, da sich dadurch bei der Verwaltung viel Zeit sparen lässt.

Sage SalesLogix enthält mehrere Standardbenutzervorlagen. Sie können die Standardbenutzervorlagen anpassen oder neue Benutzervorlagen erstellen. Weitere Informationen...

Sie können beispielsweise eine Benutzervorlage mit dem Namen "Vorgesetzter" erstellen, die als allgemeines Benutzerprofil für alle Vorgesetzten in Ihrem Unternehmen dient. Wenn Sie danach die Benutzer erstellen und die Vorlage übernehmen, ist das Profil bereits mit den gewünschten Einstellungen konfiguriert.

Benutzervorlagen erfordern keine Sage SalesLogix-Benutzerlizenz und eignen sich nicht für eine Anmeldung an Sage SalesLogix.

Wird die Vorlage nicht der Rolle Standardbenutzer zugewiesen, können alle neuen Benutzer, die mithilfe der Vorlage erstellt werden, nicht auf manche der Web Client-Funktionen zugreifen.

So gehen Sie zum Erstellen vor

  1. Klicken Sie im Menü Administration auf Benutzer hinzufügen.
  2. Wählen Sie im Feld Hinzufügen die Option Vorlage aus.
  3. Wählen Sie im Feld Menge die Anzahl der hinzuzufügenden Benutzervorlagen aus.
  4. Wenn Sie zum Konfigurieren der Einstellungen jede Registerkarte im Profil der Vorlage manuell bearbeiten möchten, fahren Sie mit Schritt 7 fort.
  5. Um die Vorlage anhand einer vorhandenen Vorlage oder des Profils eines vorhandenen Benutzers zu erstellen, aktivieren Sie das Kontrollkästchen Profil erstellen von. Wählen Sie danach eine der folgenden Optionen aus:
  • Vorlage - Dient zum Hinzufügen einer Vorlage anhand einer vorhandenen Benutzervorlage. Wählen Sie im Feld Vorlagen den Namen der Vorlage aus.
  • Vorhandener Benutzer -Dient zum Hinzufügen eines neuen Benutzers anhand des Profils eines vorhandenen Benutzers. Wählen Sie im Feld Benutzer den Namen des vorhandenen Benutzers aus.
  1. Wählen Sie die Registerkarten aus, die aus der Vorlage oder dem Benutzerprofil kopiert werden sollen. Weitere Informationen...
  • Allgemein - Allgemeine und Abteilungseinstellungen werden kopiert. Die Felder Benutzername, Name, Titel und E-Mail werden beim Kopieren von Informationen nicht übernommen.
  • Mitarbeiter - Die Felder Fax, Mobil, Pager, Durchwahl, PIN und Buchhaltung sowie die persönlichen Felder werden beim Kopieren von Informationen nicht übernommen.
  • Sicherheit - Alle Sicherheitseinstellungen werden kopiert.
  • Teams - Alle Teameinstellungen werden kopiert.
  • Kalender - Alle Kalendereinstellungen werden kopiert.
  • Service/Support - Alle Service- und Support-Einstellungen werden kopiert.
  • Client-Optionen (Client-System) - Der Standardzugriffsberechtigte der Firma und Basisvorlageneinstellungen werden kopiert.
  1. Klicken Sie auf OK.
  2. Nach dem Erstellen der Vorlage können Sie in der Benutzerdetailansicht das Profil des Benutzers ändern.

Einige der Felder im Benutzerprofil können in Benutzervorlagen nicht verwendet und daher nicht geändert werden.
