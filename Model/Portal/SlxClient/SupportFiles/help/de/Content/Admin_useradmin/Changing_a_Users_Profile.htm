Ändern von Benutzerprofilen

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Benutzer können nicht in Benutzervorlagen umgewandelt werden, und Benutzervorlagen können nicht in andere Benutzertypen konvertiert werden.

So nehmen Sie die Änderung vor

  1. Öffnen Sie die Benutzerdetailansicht des Benutzers, den Sie bearbeiten möchten.
  2. Ändern Sie das Profil des Benutzers den Anforderungen entsprechend.

Füllen Sie für alle Benutzer folgende Registerkarten aus:

  • Mithilfe der Informationsfelder können Sie den Namen, die E-Mail-Adresse, den Titel, die Abteilung, den Manager (Vorgesetzten) und das Passwort des Benutzers ändern. Den Benutzernamen, den der Benutzer zur Anmeldung an Sage SalesLogix eingibt, darf weder Leerzeichen oder noch ein Apostroph enthalten.
  • Auf der Registerkarte Mitarbeiter können Sie Informationen zum Unternehmen, geschäftliche Telefon- und Faxnummern sowie die persönlichen Angaben des Benutzers festlegen.
  • Auf der Registerkarte Teammitgliedschaft können Sie den Benutzer Systemteams hinzufügen.
  • Auf der Registerkarte Benutzerteammitglieder können Sie anderen Benutzern Zugriff auf die Firmen dieses Benutzers gewähren.
  • Auf der Registerkarte Benutzerteammitgliedschaft können Sie diesem Benutzer Zugriff auf die Firmen anderer Benutzer gewähren.
  • Auf der Registerkarte Andere Kalender können Sie den Zugriff auf Kalender anderer Benutzer festlegen.
  • Auf der Registerkarte Kalender des Benutzers können Sie anderen Benutzer Zugriff auf den Kalender dieses Benutzers gewähren.
  • Auf der Registerkarte Sicherheit können Sie ein Sicherheitsprofil auf Feldebene festlegen.
  • Auf der Registerkarte Abteilungsmitgliedschaft können Sie den Benutzer zu Abteilungen hinzufügen.
  • Auf der Registerkarte Client-System können Sie den standardmäßigen Firmenzugriffsberechtigten und Standardvorlagen festlegen.

Füllen Sie für Customer Service- und/oder Support-Benutzer folgende Registerkarte aus:

  • Nutzen Sie die Registerkarte Service/Support, um Optionen für Customer Service-Funktionen, Support-Features und SpeedSearch festzulegen.

Füllen Sie bei Bedarf folgende Registerkarten aus:

  • Nutzen Sie die Registerkarte Notizen, um weitere benutzerspezifische Informationen zu speichern. (Dabei handelt es sich in der Regel um optionale Informationen aus der Personalakte des Benutzers.)
  1. Klicken Sie auf Speichern.
