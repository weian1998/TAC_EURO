Kopieren von Profilinformationen

Sie können Konfigurationsinformationen aus einem vorhandenen Benutzerkonto oder einer Vorlage in ein oder mehrere Benutzerkonten kopieren. Durch das Kopieren der Profilinformationen eines Benutzerkontos in ein anderes werden die vorhandenen Einstellungen überschrieben. Weitere Informationen...

Beim Kopieren von Kalenderinformationen eines Web-Benutzerkontos in ein anderes können beim Zielbenutzer des Kopiervorgangs unerwartete Beschränkungen beim Zugriff auf seinen eigenen Kalender auftreten. Beispiel: Wenn der Kalender von Benutzer A aus dem Konto von Benutzer B kopiert wird, werden die Kalendersicherheitseinstellungen, die Benutzer B in seinem Kalendersicherheitsdatensatz für Benutzer A hat, in die Sicherheitseinstellungen von Benutzer A kopiert. Falls bei Benutzer B Beschränkungen für den Zugriff auf Benutzer A festgelegt sind, wird der Zugriff von Benutzer A ebenfalls beschränkt. Um solche Probleme zu vermeiden, sollten Sie keine Kalenderinformationen kopieren. Legen Sie nach dem Kopieren der Profilinformationen den Kalenderzugriff auf den Registerkarten Andere Kalender und Kalender des Benutzers fest.

So gehen Sie zum Kopieren vor

  1. Öffnen Sie die Benutzerdetailansicht des Benutzers, in dessen Profil Sie Informationen kopieren möchten.
  2. Klicken Sie im Bereich Allgemeine Aufgaben auf Profil kopieren.
  3. Wählen Sie im Dialogfeld Profil kopieren aus, ob Sie das Profil aus Vorhandener Benutzer oder Vorlage kopieren möchten.
  4. Wählen Sie den Benutzer oder die Vorlage aus, aus dem/der Sie das Profil kopieren möchten.
  5. Wählen Sie die zu kopierenden Registerkarten aus. Weitere Informationen...
  • Allgemein - Allgemeine und Abteilungseinstellungen werden kopiert. Die Felder Benutzername, Name, Titel und E-Mail werden beim Kopieren von Informationen nicht übernommen.
  • Mitarbeiter - Die Felder Fax, Mobil, Pager, Durchwahl, PIN und Buchhaltung sowie die persönlichen Felder werden beim Kopieren von Informationen nicht übernommen.
  • Sicherheit - Alle Sicherheitseinstellungen werden kopiert.
  • Teams - Alle Teameinstellungen werden kopiert.
  • Kalender - Alle Kalendereinstellungen werden kopiert.
  • Client-Optionen (Client-System) - Der Standardzugriffsberechtigte der Firma und Basisvorlageneinstellungen werden kopiert.
  • Service/Support - Alle Service- und Support-Einstellungen werden kopiert.
  1. Überprüfen Sie im Abschnitt Zielbenutzer, ob dies der Benutzer ist, dessen Profil Sie überschreiben möchten.
  2. Klicken Sie auf OK.
  3. Klicken Sie auf OK, um den Vorgang zu bestätigen.
