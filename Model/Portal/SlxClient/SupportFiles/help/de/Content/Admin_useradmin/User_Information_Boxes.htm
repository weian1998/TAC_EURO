Benutzerinformationsfelder

In der nachfolgenden Tabelle werden die Standardinformationsfelder beschrieben.

Feld Beschreibung/Aktion
Anmeldung aktiviert Bestimmt, sich der Benutzer bei Sage SalesLogix anmelden kann. Deaktivieren Sie diese Option, falls der Benutzer inaktiv wird und keinen Zugriff auf Sage SalesLogix haben soll.
Benutzername

Der Name des Benutzers, der sich bei Sage SalesLogix anmeldet.

Der Benutzername ist erforderlich und darf weder Leerzeichen noch ein Apostroph enthalten.

Name Der Name des Benutzers. Klicken Sie auf die Schaltfläche Bearbeiten, um das Dialogfeld Namen bearbeiten zu öffnen.
Titel Der Titel des Benutzers.
E-Mail Die E-Mail-Adresse des Benutzers.
Benutzertyp Der Sage SalesLogix-Benutzertyp. Wählen Sie zum Ändern des Benutzertyp einen neuen Typ in der Liste aus. Wenn Sie einen WebViewer-Benutzer ändern, müssen Sie Internetinformationsdienste (IIS) zurücksetzen.
Passwort ändern Dient zum Ändern des Passworts des Benutzers.
Password wurde aktualisiert Zeigt den Zeitpunkt (Datum und Uhrzeit), an dem das Passwort zuletzt geändert wurde.
Verwendete Vorlage Die letzte diesem Benutzer zugeordnete Benutzervorlage.
Bundesland/Kanton Der Standort des Benutzers.
Geschäftsbereich Der Geschäftsbereich des Benutzers.
Manager Der Vorgesetzte bzw. Manager des Benutzers. Nur Manager, die Manager dieses Benutzers sein können, werden in der Liste angezeigt. Manager werden automatisch zur Liste hinzugefügt, sofern sie nicht inaktiv sind.
Benutzer ist ein Manager

Bestimmt, ob der Benutzer ein Manager ist. Sein Name muss in der Liste Manager angezeigt werden. Diese Option ist nicht für den Benutzer admin nicht verfügbar.
