Aktivieren oder Deaktivieren einer Benutzeranmeldung

Sie können den Anmeldestatus eines Benutzers als aktiv oder inaktiv definieren. Neue Benutzer sind standardmäßig aktiv. Wenn eine Benutzeranmeldung inaktiv ist, kann der Benutzer nicht auf die Sage SalesLogix-Datenbank zugreifen. Dennoch wird das Profil eines inaktiven Benutzers nicht gelöscht, dem auch weiterhin eine Lizenz zugewiesen bleibt.

Weisen Sie vor der Deaktivierung eines Benutzers die Firmen dieses Benutzers über eine Gebietsneuausrichtung in Administrator neu zu. Um Firmen neu zuweisen zu können, muss der Benutzer aktiv sein. Ein Benutzer kann unbeschränkt lange deaktiviert bleiben.

Wenn Sie die dem Benutzer zugewiesene Lizenz verfügbar machen müssen, deaktivieren Sie den Benutzer.

So erfolgt die Aktivierung oder Deaktivierung

  1. Öffnen Sie die Benutzerdetailansicht des Benutzers, den Sie aktivieren oder deaktivieren möchten.
  2. Aktivieren oder deaktivieren Sie das Kontrollkästchen Anmeldung aktiviert.
  3. Klicken Sie auf Speichern.
