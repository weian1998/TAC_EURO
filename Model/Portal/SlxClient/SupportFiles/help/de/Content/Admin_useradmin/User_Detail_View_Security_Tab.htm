Registerkarte "Sicherheit"

Diese Funktion ist nur verfügbar, wenn die entsprechende Lizenz installiert ist. Ausführliche Information zur Lizenzierung finden Sie im Sage SalesLogix-Planungshandbuch.

Auf der Registerkarte Sicherheit in der Benutzerdetailansicht können Sie ein Standardprofil für die Sicherheit auf Feldebene eines Benutzers festlegen. Der Systemadministrator hat stets Zugriff auf alle Systemelemente. Seine Zugriffsberechtigungen können nicht geändert werden.

So legen Sie Optionen fest

  1. Klicken Sie in der Benutzerdetailansicht auf die Registerkarte Sicherheit.
  2. Wählen Sie im Feld Profil für Sicherheit auf Feldebene das zu verwendende Standardprofil aus, wenn dieser Benutzer einem Team oder dem Team eines anderen Benutzers hinzugefügt wird. Sicherheitsprofile werden in Administrator erstellt.
    3. Weitere Informationen...

    Wenn Benutzer A dem Team von Benutzer B manuell hinzugefügt wird (auf der Registerkarte Benutzerteammitgliedschaft), kann Benutzer A auf alle Datensätze von Benutzer B zugreifen.  Die Feldsicherheitsberechtigungen von Benutzer A bestimmen jedoch, ob Benutzer A die Datensätze von Benutzer B ändern darf.  Benutzer haben stets Lese- und Schreibzugriff auf alle Firmen, deren individueller Zugriffsberechtigter sie sind oder deren Zugriffsberechtigter Alle sind.

  3. Klicken Sie auf Speichern.
