Was sind Verkaufschancen?

Potenzielle Verkäufe an Firmen und Kontakte werden als Verkaufschancen bezeichnet. Mit der Verkaufschancen-Funktionalität können Sie eine Vielzahl von Informationen zu einer Verkaufschance sammeln und für die Berichterstellung nutzen, z. B. die Wahrscheinlichkeit eines Geschäftsabschlusses, das voraussichtliche Abschlussdatum usw. Sie können Verkaufschancen für jede öffentliche Firma erstellen, auf die Sie Zugriff haben. Eine Verkaufschance kann nur eine Firma beinhalten, jedoch mehrere verknüpfte Kontakte. Ihre Zugriffsrechte auf Firmen bestimmen Ihren Zugriff auf die mit den Firmen verknüpften Verkaufschancen.

Sie können die Verkaufschancen-Detailansicht verwenden, um detaillierte Informationen zu einer einzelnen Verkaufschance aufzuzeichnen und damit zu arbeiten. In der Verkaufschancen-Listenansicht können Sie im Tabellenformat mit Verkaufschancen arbeiten.

Im Folgenden finden Sie einige Funktionen, mit denen Sie Verkaufschancen verwalten können:

  • Erstellen von Gruppen, um mit einer Teilmenge von Verkaufschancen zu arbeiten. Sie können Gruppen anhand bestimmter Bedingungen erstellen, oder indem Sie Verkaufschancen einzeln auswählen. Erstellen Sie Gruppen in der Verkaufschancen-Listenansicht.
  • Zuweisen von verkaufschancenspezifischen Produkten. Sie können beispielsweise die Produktmengen, an denen ein Kunde interessiert ist, und den voraussichtlichen Umsatz aufzeichnen.
  • Einbeziehen wichtiger Kontakte in Bezug auf die Verkaufschance
  • Verfolgen von an Verkaufschancen beteiligten Mitbewerbern zur Vereinfachung Ihrer Strategieplanung
  • Nachverfolgen der Abschlusswahrscheinlichkeit, des voraussichtlichen und tatsächlichen Abschlussdatums und des potenziellen Umsatzes
