Bearbeiten von Verkaufschancenprodukt-Informationen

Mit diesem Dialogfeld können Sie Produktinformationen eingeben, z. B. Menge, Preis oder Rabatt.

So bearbeiten Sie Informationen

  1. Klicken Sie in der Verkaufschancen-Detailansicht auf der Registerkarte Produkte auf den Hyperlink des Produkts, das Sie bearbeiten möchten, oder nehmen Sie die Änderungen im Produktraster vor.
  2. Klicken Sie im Feld Preisstufe auf Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
  3. Geben Sie im Feld Rabatt einen prozentualen Rabatt ein. Die Zahl wird automatisch in eine Prozentangabe geändert. Beispiel: "50" wird zu "50 %". Wenn Sie ",5" eingeben, wird dies zu ",5 %".

Der angepasste Preis wird automatisch berechnet.

  1. Geben Sie im Feld Menge die Menge des Produkts ein, die hinzugefügt werden soll.

Der erweiterte Preis wird automatisch berechnet.

  1. Klicken Sie auf OK.
