Bearbeiten eines Mitbewerbers bei Verkaufschancen

Nachdem Sie einen Mitbewerber zu einer Verkaufschance hinzugefügt haben, können Sie im Dialogfeld Mitbewerber bearbeiten die Informationen zum Mitbewerber bearbeiten.

So bearbeiten Sie Informationen

  1. Öffnen Sie das Dialogfeld Mitbewerber bei Verkaufschance bearbeiten.
  • Klicken Sie in der Ansicht Verkaufschancendetail auf der Registerkarte Mitbewerber auf den Hyperlink Bearbeiten des zu bearbeitenden Mitbewerbers.
  • Wenn Sie einen Mitbewerber bearbeiten möchten, klicken Sie im Dialogfeld Verkaufschance hinzufügen auf den zugehörigen Hyperlink Bearbeiten.
  1. Wenn Sie die Einstufung dieses Mitbewerbers ändern möchten, geben Sie eine Zahl in das Feld Bewertung ein.

Dadurch wird die Bewertung dieses Mitbewerbers nur für diese Verkaufschance geändert, die Gesamtbewertung des Mitbewerbers wird jedoch nicht geändert.

  1. Wählen Sie Marktführer aus, um den Mitbewerber als Marktführer zu kennzeichnen.
  2. Geben Sie die Stärken und Schwächen des Mitbewerbers in die dafür vorgesehenen Felder ein.
  3. Geben Sie im Bereich Strategie ggf. Notizen zur Strategie des Mitbewerbers ein.
  4. Geben Sie im Bereich Notizen ggf. zusätzliche Informationen zum Mitbewerber ein.
  5. Klicken Sie auf OK.
