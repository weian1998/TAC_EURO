Entfernen eines Kontakts aus einer Verkaufschance

Sie können Kontakte jederzeit aus einer Verkaufschance löschen. Sie können Kontakte im Dialogfeld Verkaufschance hinzufügen oder in der Ansicht Verkaufschancendetail löschen. Das Löschen eines Kontakts entfernt lediglich die Verknüpfung des Kontakts mit einer bestimmtem Verkaufschance. Der Kontakt wird nicht aus der Datenbank gelöscht.

So entfernen Sie einen Kontakt aus dem Dialog "Verkaufschance hinzufügen"

  1. Wählen Sie im Raster Kontakte den Kontakt aus, die Sie löschen möchten.
  2. Klicken Sie auf die Schaltfläche Ausgewählte Kontakte entfernen und anschließend auf OK, um den Löschvorgang zu bestätigen.
  3. Klicken Sie auf OK.

So entfernen Sie einen Kontakt aus der Verkaufschancen-Detailansicht

  1. Klicken Sie auf die Registerkarte Kontakte.
  2. Klicken Sie auf die Schaltfläche Entfernen und anschließend auf OK, um den Löschvorgang zu bestätigen.
  3. Klicken Sie auf OK.
