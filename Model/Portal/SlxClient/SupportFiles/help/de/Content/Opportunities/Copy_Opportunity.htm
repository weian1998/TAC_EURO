Kopieren einer Verkaufschance

Sie können eine vorhandene Verkaufschance kopieren, um schnell eine neue Verkaufschance mit denselben Mitbewerbern, Kontakten und Produkten sowie anderen Informationen zu erstellen.

So kopieren Sie eine Verkaufschance

  1. Öffnen Sie die Verkaufschancen-Detailansicht für die Kampagne, die Sie kopieren möchten.
  2. Klicken Sie auf Kopieren.
    Das Dialogfeld Verkaufschance hinzufügen wird mit den folgendenInformation aus der ursprünglichen Verkaufschance geöffnet:
    • Beschreibung
    • Firma
    • Account Manager
    • Typ
    • Interessentenquelle
    • Mitbewerber
    • Produkte
    • Kontakte
  3. Ändern Sie die kopierten Informationen bei Bedarf und füllen Sie dann die verbleibenden Informationsfelder aus.
  4. Klicken Sie auf Speichern
