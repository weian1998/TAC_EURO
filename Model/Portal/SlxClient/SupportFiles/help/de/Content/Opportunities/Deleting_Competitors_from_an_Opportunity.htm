Entfernen eines Mitbewerbers aus einer Verkaufschance

Sie können einen Mitbewerber aus einer Verkaufschance entfernen, wenn die Firma keine Konkurrenz für den Verkauf mehr darstellt oder versehentlich ausgewählt wurde. Sie können Mitbewerber im Dialog Verkaufschance hinzufügen oder in der Verkaufschancen-Detailansicht löschen. Das Löschen eines Mitbewerbers ändert nur die Verknüpfung des Mitbewerbers mit einer bestimmten Verkaufschance. Der Mitbewerber wird nicht aus der Datenbank gelöscht.

So entfernen Sie Mitbewerber im Dialog Verkaufschance hinzufügen

  1. Wählen Sie im Raster Mitbewerber die Mitbewerber aus, die Sie löschen möchten.
  2. Klicken Sie auf die Schaltfläche Mitbewerber entfernen.

So entfernen Sie Mitbewerber aus der Verkaufschancen-Detailansicht

  1. Klicken Sie auf die Registerkarte Mitbewerber. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  2. Klicken Sie neben dem Mitbewerber, den Sie aus der Verkaufschance löschen möchten, auf Löschen.
  3. Klicken Sie auf OK.
