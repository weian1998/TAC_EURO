Abschließen einer Verkaufschance

Wenn Sie bereit sind, eine Verkaufschance abzuschließen, können Sie den Status der Verkaufschance in der Verkaufschancen-Detailansicht in Abgeschlossen - Erfolg oder Abgeschlossen - Verloren ändern. Wir empfehlen, eine Verkaufschance auch dann abzuschließen, wenn kein Geschäft zustande gekommen ist. Für eine künftige Bezugnahme und zum Zwecke der Berichterstellung sollten Sie auch den Grund für das Abschließen der Verkaufschance angeben. Verwenden Sie das Feld Kommentare zur Verkaufschance, um den Erfolg oder den Misserfolg der Verkaufschance ausführlich zu begründen.

So schließen Sie eine gewonnene Verkaufschance ab
  1. Öffnen Sie die Verkaufschancen-Detailansicht der Verkaufschance, die Sie abschließen möchten.
  2. Klicken Sie im Feld Status auf die Dropdown-Pfeilund wählen Sie Abgeschlossen - Erfolg aus der Liste.
    Das Dialogfeld Verkaufschance schließen wird angezeigt.
  3. Ändern Sie ggf. den Eintrag Tatsächlicher Betrag des Verkaufs.
  4. Ändern Sie ggf. das Datum für Tatsächlicher Abschluss. Das Datum ändern Sie, indem Sie auf die Schaltfläche Kalender-Schaltfläche klicken, ein neues Datum auswählen und anschließend auf OK klicken.
  5. Klicken Sie neben dem Feld Grund für Erfolg auf die Schaltfläche mit den Auslassungszeichen, wählen Sie einen oder mehrere Einträge in der Liste aus und klicken Sie anschließend auf OK.
  6. Klicken Sie bei Bedarf auf Ersetzte MitbewerberSuchen, um den Mitbewerber auszuwählen, der die Verkaufschance verloren hat. Wählen Sie einen Mitbewerber aus dem Dialogfeld Mitbewerber suchen aus und klicken Sie auf OK. Wiederholen Sie diesen Prozess, bis alle ersetzten Mitbewerber hinzugefügt wurden.
  7. Geben Sie in das Feld Kommentarezusätzliche Notizen ein.
  8. Klicken Sie auf OK.
So schließen Sie eine verlorene Verkaufschance ab
  1. Öffnen Sie die Verkaufschancen-Detailansicht der Verkaufschance, die Sie abschließen möchten.
  2. Klicken Sie im Feld Status auf die Dropdown-Pfeil, und wählen Sie Abgeschlossen - Verloren in der Liste aus.
  3. Ändern Sie ggf. den Eintrag Tatsächlicher Betrag des Verkaufs.
  4. Ändern Sie ggf. das Datum für Tatsächlicher Abschluss. Das Datum ändern Sie, indem Sie auf die Schaltfläche Kalender-Schaltfläche klicken, ein neues Datum auswählen und anschließend auf OK klicken.
  5. Klicken Sie neben dem Feld Grund für Misserfolg auf die Schaltfläche mit den Auslassungszeichen, wählen Sie einen oder mehrere Einträge in der Liste aus und klicken Sie anschließend auf OK.
  6. Klicken Sie bei Bedarf auf Mitbewerber - Suchen, um den Mitbewerber auszuwählen, der die Verkaufschance verloren hat. Wählen Sie einen Mitbewerber aus dem Dialogfeld Mitbewerber suchen aus und klicken Sie auf OK.
  7. Im Feld Kommentare können Sie beliebige zusätzliche Anmerkungen eingeben.
  8. Klicken Sie auf Speichern.
