Aktualisieren mehrerer Verkaufschancen

Über den Dialog Verkaufschancen aktualisieren können Sie mehrere Verkaufschancen gleichzeitig aktualisieren.

Mehrere Verkaufschancen aktualisieren

  1. Klicken Sie auf der Navigationsleiste auf die Schaltfläche Verkaufschancen.
  2. Öffnen Sie die Gruppe, die die zu aktualisierenden Verkaufschancen enthält.
  3. Wählen Sie die Datensätze aus, die Sie aktualisieren möchten, und führen Sie danach einen der folgenden Schritte aus:
    • Klicken Sie mit der rechtem Maustaste, und wählen Sie Verkaufschancen aktualisieren.
    • Klicken Sie im Aufgabenbereich unter Allgemeine Aufgaben auf Verkaufschancen aktualisieren.
  4. Im Dialogfeld Verkaufschancen aktualisieren können Sie die folgenden Informationen aktualisieren:

    • Account Manager

      1. Wählen Sie in der Dropdown-Liste Aktualisieren den Eintrag Account Manager aus.
      2. Klicken Sie auf die Schaltfläche An:Schaltfläche Suchen, wählen Sie einen neuen Account Manager in der Liste aus und klicken Sie auf OK.
    • Zu Forecast hinzufügen
      1. Wählen Sie in der Dropdown-Liste Aktualisieren den Eintrag Zu Forecast hinzufügen? aus.
      2. Wählen Sie Ja, um Verkaufschancen in Forecast-Berichte einzuschließen, oder wählen Sie Nein, um Verkaufschancen aus Forecast-Berichten auszuschließen.
    • Abschlusswahrscheinlichkeit in %
      1. Wählen Sie in der Dropdown-Liste Aktualisieren den Eintrag Abschlusswahrscheinlichkeit in % aus.
      2. Klicken Sie auf die Dropdown-Pfeil, und wählen Sie danach in der Liste eine neue Abschlusswahrscheinlichkeit aus.

    • Voraussichtlicher Abschluss

      1. Wählen Sie in der Dropdown-Liste Aktualisieren den Eintrag Voraussichtlicher Abschluss aus.
      2. Wählen Sie aus, wie das voraussichtliche Abschlussdatum aktualisiert werden soll.
        • Wenn Sie das voraussichtliche Abschlussdatum auf ein bestimmtes Datum verschieben möchten, klicken Sie auf die Option An: und dann auf die Schaltfläche Kalender, wählen ein neues Datum aus und klicken auf OK.
        • Wenn Sie das geschätzte Abschlussdatum um eine bestimmte Anzahl von Tagen nach hinten verschieben möchten, wählen Sie Verschieben aus, wählen in der Dropdown-Liste die Option Rückwärts aus und geben danach die Anzahl der Tage ein, um die Sie das Datum zurückverschieben möchten.
        • Wenn Sie das geschätzte Abschlussdatum um eine bestimmte Anzahl von Tagen nach vorne verschieben möchten, wählen Sie Verschieben aus, wählen in der Dropdown-Liste die Option Vorwärts aus und geben danach die Anzahl der Tage ein, um die Sie das Datum nach vorne verschieben möchten.
    • Kommentare
      1. Wählen Sie in der Dropdown-Liste Aktualisieren den Eintrag Kommentare aus.
      2. Geben Sie Ihre Notizen in das Feld An: neben dem Dropdown-Feld ein.
  5. Klicken Sie auf OK.
