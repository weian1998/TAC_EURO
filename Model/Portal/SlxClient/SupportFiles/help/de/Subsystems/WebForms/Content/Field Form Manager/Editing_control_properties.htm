Bearbeiten von Steuerelementeigenschaften

Der Formular-Designer zeigt viele der Eigenschaftswerte des derzeit ausgewählten Steuerelements und ermöglicht ihre Bearbeitung.

So bearbeiten Sie Informationen

  1. Öffnen Sie im Formular-Designer ein Formular.
  2. Klicken Sie im Layout auf ein Steuerelement.
    Die Eigenschaften des ausgewählten Steuerelements werden auf den Registerkarten unter dem Bereich mit dem Steuerelementlayout angezeigt.
    Eventuell können Sie mit dem Application Architect auf weitere Steuerelementeigenschaften zugreifen.
  3. Klicken Sie auf eine Registerkarte, und ändern Sie die Einstellungen des Steuerelements, das Sie konfigurieren. Klicken Sie auf einen der folgenden Links, um weitere Informationen den Eigenschaften eines Steuerelements anzuzeigen:
