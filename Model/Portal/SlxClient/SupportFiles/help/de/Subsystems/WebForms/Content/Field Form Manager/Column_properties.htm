Spalteneigenschaften

Sie können die folgenden Eigenschaften dieser Spalte anzeigen und deren Werte ändern:

Standardeigenschaft Beschreibung

Index

Position der Zeile oder Spalte im Formular. Beginnt bei Null.

Diese Informationen sind im Formular-Designer schreibgeschützt.

Breite

Gibt eine Zuweisung für die Breite der einzelnen Spalten an.

Größentyp

Es gilt eine der folgenden Zuweisungsmethoden für die Breite :

 

  • Absolut: Feste Spaltenbreite, unabhängig von der Fenstergröße.
  • Größe autom. festlegen: Das System bestimmt die optimale Breite für jede Spalte automatisch. Dabei ist eine Mindestbreite für jede Spalte vorgesehen, die beachtet werden muss, bevor ein Bildlauf erfolgt. Diese Mindestbreite wird vom System festgelegt und kann nicht vom Formular-Designer angegeben werden.
  • Prozent: Prozentuale Zuweisung auf der Grundlage der verfügbaren Fensterbreite. Die prozentualen Zuweisungen aller Spalten müssen zusammen 100 % ergeben. Liegt die Fenstergröße unter einem bestimmten Grenzwert, behalten die Spalten eine Mindestbreite. Es erfolgt dann ein Bildlauf im Formular. Diese Mindestbreite wird vom System festgelegt und kann nicht mit dem QuickForm-Editor angegeben werden.
