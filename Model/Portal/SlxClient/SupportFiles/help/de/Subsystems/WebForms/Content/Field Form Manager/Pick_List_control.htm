Eigenschaften des Steuerelements "Auswahlliste"

Sie können die folgenden Eigenschaften dieses Steuerelements anzeigen und deren Werte ändern:

Standardeigenschaft Beschreibung

Beschriftung

Die Bezeichnung, die auf dem Formular dieses Steuerelements angezeigt wird.

Beschriftungsausrichtung

Ausrichtung des Bezeichnungstexts. Folgende Optionen sind verfügbar:

  • Links
  • Rechts
  • Zentriert

QuickInfo

Der Text, der als QuickInfo angezeigt wird, wenn die Maus über das Steuerelement bewegt wird.

Schreibgeschützt

Lässt keine Bearbeitungen des Formulars zu.
Erweiterte Eigenschaft Beschreibung

Anzeigemodus

Anzeigemodus des Steuerelements. Die verfügbaren Optionen sind je nach Steuerelementtyp unterschiedlich:

  • Als Steuerelement: Das Steuerelement sieht wie ein Textfeld aus.
  • Als Text: Das Steuerelement wird als normaler Text angezeigt.
  • Als Hyperlink: Das Steuerelement wird als Hyperlink angezeigt.
  • Normal: Das Steuerelement sieht wie eine Bezeichnung aus.

Bezeichnungsplatzierung

Bezeichnungsposition im Verhältnis zum Steuerelement.

Sichtbar

Dient zum Ein- oder Ausblenden dieses Steuerelements auf dem Formular.

Mehrfachauswahl zulassen

Erlaubt Benutzern, mehr als einen Eintrag in der Auswahlliste auszuwählen.

Die Verhaltensweisen der Netzwerk- und Web Client-Auswahllisten können in der Ansicht Auswahllistendetails in Web Client definiert werden. Sie können beispielsweise Erforderlicher Eintrag für eine bestimmte Auswahlliste aktivieren bzw. deaktivieren. Damit Sie Verhaltensweisen auf Web Client-Auswahllisten anwenden können, muss das Attribut Optionen auf Web-Auswahlliste anwenden ausgewählt sein. Andernfalls gelten die Verhaltensweisen, die das QuickForm-Steuerelement definiert, welches die Auswahlliste verwendet.

Vor dem Festlegen von Auswahllistenoptionen müssen Sie mit den Eigenschaften des Auswahllisten-Steuerelements und der Speicherung von Auswahllistendaten in der Datenbank vertraut sein. Sie müssen wissen, wie Daten in der Auswahlliste an die Datenbank gebunden sind, um zu bestimmen, ob eine Option unterstützt wird. Wenn Sie beispielsweise die Auswahlliste Einstellungen in Mehrfachauswahl zulassen ändern, funktioniert die Auswahlliste nicht ordnungsgemäß, da das Datenbankfeld für die Speicherung großer Textmengen zu klein ist.

Darf Text bearbeiten

Aktiviert

Ermöglicht dem Benutzer die Interaktion mit diesem Steuerelement.

Max. Länge

Maximale Anzahl von Zeichen, die der Benutzer eingeben kann.

Erforderlich

Erfordert beim Speichern von Daten einen Wert.

Datenbindungen

Definiert, woher die Daten für das Steuerelement kommen. Diese Informationen sind im Formular-Designer schreibgeschützt.

Auswahllistenname

Name der Liste der Werte, die der Benutzer bei der Dateneingabe auswählen kann. Diese Informationen sind im Formular-Designer schreibgeschützt.

Speichermodus

Bestimmt, wie der Wert gespeichert wird. Folgende Optionen sind verfügbar:

  • Text - Wenn der Wert als Zeichenfolge gespeichert wurde, wird die Zeichenfolge angezeigt.
  • ID - Wenn der Wert als ID gespeichert wurde, wird die entsprechende Zeichenfolge aus der Auswahlliste abgerufen und angezeigt.
  • Code - Wenn ein Code gespeichert wurde, wird die entsprechende Zeichenfolge angezeigt. Zumeist nicht verwendet.

Diese Informationen sind im Formular-Designer schreibgeschützt.

Steuerelement-ID

ID dieses Steuerelements. Diese Informationen sind im Formular-Designer schreibgeschützt.

Steuerelementtyp

Sage SalesLogix-Steuerelementtyp, z. B. Textfeld. Diese Informationen sind im Formular-Designer schreibgeschützt.
Eigenschaft "Layout" Beschreibung

Zeile

Zeilennummer des Steuerelements.

Spalte

Spaltennummer des Steuerelements.

Zeilenbreite

Anzahl der Zellen, die das Steuerelement vertikal einnimmt.

Spaltenweite

Anzahl der Zellen, die das Steuerelement horizontal einnimmt.
