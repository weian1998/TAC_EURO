Eigenschaften des Steuerelements "Numerisch"

Sie können die folgenden Eigenschaften dieses Steuerelements anzeigen und deren Werte ändern:

Standardeigenschaft Beschreibung

Beschriftung

Die Bezeichnung, die auf dem Formular dieses Steuerelements angezeigt wird.

Beschriftungsausrichtung

Ausrichtung des Bezeichnungstexts. Folgende Optionen sind verfügbar:

  • Links
  • Rechts
  • Zentriert

QuickInfo

Der Text, der als QuickInfo angezeigt wird, wenn die Maus über das Steuerelement bewegt wird.

Schreibgeschützt

Lässt keine Bearbeitungen des Formulars zu.
Erweiterte Eigenschaft Beschreibung

Bezeichnungsplatzierung

Bezeichnungsposition im Verhältnis zum Steuerelement.
Formattyp

Wird zum Umwandeln eines numerischen Werts in eine Zeichenfolge verwendet.Für das Steuerelement stehen folgende Optionen zur Verfügung:

  • Zahl
  • Prozent
  • Dezimal
  • Wissenschaftlich

Sichtbar

Dient zum Ein- oder Ausblenden dieses Steuerelements auf dem Formular.

Aktiviert

Ermöglicht dem Benutzer die Interaktion mit diesem Steuerelement.

Max. Länge

Maximale Anzahl von Zeichen, die der Benutzer eingeben kann.
Erforderlich

Erfordert beim Speichern von Daten einen Wert.

Dezimalstellen

Die Anzahl von Stellen hinter dem Dezimalzeichen, die angezeigt wird und für Dateneingaben zulässig ist.

Beim Standardwert -1 wird bei den regionalen Einstellungen des Benutzers festgelegte "Anzahl der Dezimalstellen" verwendet. Geben Sie alternativ eine ganze Zahl von 0 bis 4 ein. Falls Sie einen Wert größer 4 eingeben möchten, stellen Sie in Application Architect sicher, dass das Feld, an das das Steuerelement gebunden ist, mit der entsprechenden Dezimalstellenanzahl definiert wurde.

Gibt der Benutzer weniger Ziffern an, als die angegebene Genauigkeit vorsieht, werden die Stellen bei einem Steuerelement/einer Spalte vom Typ "Währung" mit Nullen aufgefüllt. Wenn bei einem Steuerelement/einer Spalte vom Typ "Währung" Stellen bis zu festgelegten Genauigkeit mit Nullen aufgefüllt werden sollen, legen Sie Streng=Wahr fest.

Streng

 Füllt die Anzeigen von Anzahl bei Bedarf mit nachfolgenden Nullen auf. Beispiel: Wenn Dezimalstelle=4, Streng=Wahr und der numerische Wert 2,3 ist, wird der Wert als 2,3000 angezeigt.

Datenbindungen

Definiert, woher die Daten für das Steuerelement kommen. Diese Informationen sind im Formular-Designer schreibgeschützt.

Steuerelement-ID

ID dieses Steuerelements. Diese Informationen sind im Formular-Designer schreibgeschützt.

Steuerelementtyp

Sage SalesLogix-Steuerelementtyp, z. B. Textfeld. Diese Informationen sind im Formular-Designer schreibgeschützt.
Eigenschaft "Layout" Beschreibung

Zeile

Zeilennummer des Steuerelements.

Spalte

Spaltennummer des Steuerelements.

Zeilenbreite

Anzahl der Zellen, die das Steuerelement vertikal einnimmt.

Spaltenweite

Anzahl der Zellen, die das Steuerelement horizontal einnimmt.
