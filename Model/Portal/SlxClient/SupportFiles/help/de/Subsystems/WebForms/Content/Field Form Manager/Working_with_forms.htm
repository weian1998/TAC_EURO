Formular-Designer - Arbeiten mit Formularen

 Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Sie können einfache Änderungen an einer Ansicht vornehmen, indem Sie mit dem Formular-Designer die dazugehörigen Formulare bearbeiten. Sie können beispielsweise Beschriftungen ändern, Objekte neu positionieren sowie Informationen hinzufügen oder ausblenden.

Wenn Sie die Formularänderungen speichern, werden auch alle Ansichten geändert, die das Formular enthalten.

Ihre Änderungen betreffen alle Benutzer des Systems, da Definitionen von Formularen und Ansichten gemeinsam genutzt werden.

QuickForms können mit dem Formular-Designer im Sage SalesLogix Web Client oder Application Architect bearbeitet werden. Die Bearbeitung benutzerdefinierter Formulare ist mit dem Formular-Designer nicht möglich. Bei jedem Formular, das Sie mit dem Formular-Designer bearbeiten können, wird in der Symbolleiste die Schaltfläche Formular bearbeiten angezeigt.

Vorgehensweise

Öffnen Sie Formular-Designer
  1. Öffnen Sie im Browser eine Web Client-Ansicht, z. B. die Firmendetailansicht.
    Beachten Sie, dass jedes der Formulare in der Ansicht über eine Symbolleiste verfügt.
  2. Klicken Sie auf der Symbolleiste des Formulars, das Sie ändern möchten, bei gedrückter STRG-Taste auf die Schaltfläche Formular bearbeiten.
    Im Formular-Designer wird eine weitere Browser-Registerkarte angezeigt. Sie können zwischen diesen Registerkarten umschalten, um Änderungen nacheinander vorzunehmen und die Ergebnisse anzuzeigen.
  3. Klicken Sie auf die neue Registerkarte im Browser.
    Formular-Designer zeigt das Steuerelementlayout des Formulars an.

Verwenden Sie alternativ die Formular-Manager-Listenansicht, um den Formular-Designer zu öffnen.

Bearbeiten eines Formulars
  1. Öffnen Sie das Formular im Formular-Designer.
  2. Klicken Sie auf die Registerkarte Verwendung, und beachten Sie, welche Ansichten dieses Formular enthalten. Änderungen am Formular wirken sich auf alle diese Ansichten aus.
  3. Nehmen Sie Änderungen am Formular vor. Beispiel:
    • Ändern von Beschriftungen und anderen Steuerelementeigenschaften
    • Verschieben und Ausblenden von Steuerelementen
    • Hinzufügen von Steuerelementen (Hinzufügen von Datenfeldern zu Formularen)
    • Ändern von Formular-, Zeilen- und Spalteneigenschaften
  1. Klicken Sie auf Speichern.
    Änderungen werden sofort wirksam.
Anzeigen der Änderungen
  • Wechseln Sie zur ursprünglichen Browser-Registerkarte, und drücken Sie F5. Sie können zwischen diesen Registerkarten umschalten, um Änderungen nacheinander vorzunehmen und die Ergebnisse anzuzeigen.
  • Wechseln Sie alternativ zu einer Ansicht, die das bearbeitete Formular enthält.

Sie können Änderungen an der Zusammenfassungsansicht von QuickForms anzeigen, nachdem der Erstellungs- und Bereitstellungsvorgang in Application Architect erfolgt ist. Weitere Informationen finden Sie unter "Erstellen der Webplattform" in der Application Architect-Hilfe.
Rückgängigmachen der letzten Änderungen
  • Wechseln Sie zur Browser-Registerkarte für den Formular-Designer und drücken Sie F5, um zur letzten gespeicherten Version des Formulars zurückzukehren.
Freigeben meiner Änderungen für andere Benutzer
  • Wenn Sie Änderungen speichern, sind alle anderen Benutzer im System unmittelbar betroffen. Wenn ein Benutzer die Ansicht bereits geöffnet hat, muss zum Aktualisieren der Seite F5 gedrückt werden.
  • Wenn Sie die Änderungen in einem anderen System freigeben möchten, müssen Sie Application Architect verwenden. Weitere Informationen finden Sie unter "Arbeiten mit dem Bundle-Modell" in der Application Architect-Hilfe.

Bearbeiten der Werte von Steuerelementeigenschaften

Verschieben eines Steuerelements
  1. Klicken Sie auf ein Steuerelement, und ziehen Sie es.
  2. Legen Sie das Element an einer Position ab, die hervorgehoben wird, wenn Sie den Mauszeiger darüber bewegen.
    Vorhandene Steuerelemente werden bei Bedarf automatisch verschoben.
Löschen eines Steuerelements
  • Der Formular-Designer ermöglicht das Ausblenden von Steuerelementen, jedoch nicht das Löschen. Steuerelemente können nur in Sage SalesLogix Application Architect gelöscht werden.
Ausblenden eines Steuerelements
  1. Klicken Sie auf das Steuerelement und dann auf die Registerkarte Erweitert.
  2. Deaktivieren Sie in der Gruppe Design das Eigenschaftenfeld Sichtbar.
  3. Wenn Sie den Platz, den das ausgeblendete Steuerelement belegt, erneut verwenden möchten, verschieben Sie das Steuerelement an eine andere Position, z. B. an das Ende der Seiten. Steuerelemente können nur in Sage SalesLogix Application Architect gelöscht werden.

Hinzufügen eines Datenfelds zu einem Formular (Hinzufügen eines Steuerelements)

Ändern der Größe eines Steuerelements
  1. Klicken Sie auf das Steuerelement und dann auf die Registerkarte Layout.
  2. Ändern Sie in der Gruppe Größe die Eigenschaften Zeilenbreite oder Spaltenweite den Anforderungen entsprechend.
    Auf dem Formular Firmendetails nimmt Unternehmensbeschreibung beispielsweise zwei Zeilen ein. Beachten Sie, dass einige Steuerelementtypen eine konstante Anzeigegröße ungeachtet der Anzahl der Bereiche haben, die sie belegen.
Ändern der Beschriftung eines Steuerelements
  1. Klicken Sie auf das Steuerelement und dann auf die Registerkarte Normal.
  2. Ändern Sie in der Gruppe Design die Eigenschaften Beschriftung den Anforderungen entsprechend.
Daten als schreibgeschützt markieren
  1. Klicken Sie auf das Steuerelement und dann auf die Registerkarte Normal.
  2. Aktivieren Sie in der Gruppe Verhalten das Kontrollkästchen Schreibgeschützt.
Anzeigen und Ändern von Formulareinstellungen
  1. Klicken Sie auf der Benutzeroberfläche von Formular-Designer auf einen nicht belegten Bereich, z. B. auf die linke obere Ecke.
    Die Formulareigenschaften werden in einem Abschnitt mit Registerkarten angezeigt.
  2. Ändern Sie die Eigenschaftenwerte den Anforderungen entsprechend. Aktives Steuerelement ist beispielsweise das Steuerelement, zu dem der Cursor des Benutzers bewegt wird, wenn die Ansicht erstmals angezeigt wird.
Ändern der Breite einer Spalte von Steuerelementen
  1. Klicken Sie über einer Spalte auf das Auswahlelement.
    Die Spalteneigenschaften werden in einem Abschnitt mit Registerkarten angezeigt.
  2. Ändern Sie in der Gruppe Design die Option Breite.
  3. Wiederholen Sie dies für sämtliche Spalten.

    Wenn Größentyp auf Prozent festgelegt ist, stellen Sie sicher, dass die Gesamtbreite aller Spalten auf 100 normalisiert ist.

Hinzufügen oder Löschen einer leeren Zeile oder Spalte
  1. Klicken Sie mit der rechten Maustaste auf das Auswahlrechteck links neben einer Zeile oder über einer Spalte.
  2. Über das Kontextmenü können Sie eine leere Zeile oder Spalte löschen oder einfügen.
Hinzufügen eines Formulars
  • Zum Hinzufügen von Formularen verwenden Sie Application Architect. Weitere Informationen dazu finden Sie im Application Architect-Hilfethema "Erstellen eines neuen Formulars".
