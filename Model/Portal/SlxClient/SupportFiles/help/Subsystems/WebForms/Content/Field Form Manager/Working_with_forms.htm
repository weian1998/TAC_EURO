Form Designer - Working with Forms

 The system administrator and users with the appropriate role can access these features.

You can make simple changes to a view by editing its component forms with the Form Designer. For example, you can change captions, move things around, and add and hide information.

When you save the form changes, all the views that contain the form change as well.

Changes you make affect everyone on the system because form and view definitions are shared.

Quick Forms can be edited with the Form Designer in the Sage SalesLogix Web Client or the Application Architect. Custom forms cannot be edited with the Form Designer. Each form that can be edited with the Form Designer displays the Edit Form button in its toolbar to users with access to the feature.

How Do I?

General

Open the Form Designer

  1. Open a Web Client view such as the Account Details view in your browser.
    Note that each of the forms in the view has a toolbar.
  2. In the toolbar of the form you want to edit, CTRL-click Edit Form .
    Another browser tab opens with Form Designer.
  3. Click the new browser tab.
    The Form Designer displays the control layout of the form. You can switch between the new tab with the Form Designer and the other tab with the Web Client view to repeatedly make changes and view the results.

Alternatively, use the Form Manager List view to open the Form Designer.

Edit a form
  1. Open the form in the Form Designer.
  2. Click the Usage tab and note which views contain this form. Changes to the form will affect all of these views. If you do not see a Usage tab, click the empty space in the upper left corner of the Form Designer surface.
  3. Make changes to the form such as:
    • Change captions and other control properties
    • Move and hide controls
    • Add controls (Add data fields to a form)
    • Change form, row, and column properties
  1. Click Save.
    Changes take effect immediately.
View the changes
  • Switch to the original browser tab and press F5. You can switch between these tabs to repeatedly make changes and view the results.
  • Alternatively, browse to any view that contains the form you edited.

You can view changes to summary view quick forms after a build and deploy is done in Application Architect. Refer to "Building the Web Platform" in the Application Architect help for more information.
Undo my last changes
  • On the browser tab for the Form Designer, press F5 to revert to the last saved version of the form.
Release my changes to other users
  • When you save changes, they affect everyone on your system immediately. If someone has the view open already, they must press F5 to refresh the page.
  • If you want to release the changes to another system, you must use Application Architect. Refer to "Working with the Bundle Model" in the Application Architect help for more information.
Controls

View and edit control property values

Move a control
  1. Click a control and drag it.
  2. Drop it on any location that is highlighted when the cursor hovers over it.
    Existing controls are moved automatically if necessary.
Delete a control
  • The Form Designer permits you to hide controls but not delete them. Use Sage SalesLogix Application Architect if you must delete a control.
Hide a control
  1. Select the control, and click the Advanced tab.
  2. In the Appearance group, clear the Visible property box.
  3. If you want to be able to reuse the space occupied by the hidden control, move it to another location such as the bottom of the page. Use Sage SalesLogix Application Architect if you must delete a control.

Add a control/data field

Change the size of a control
  1. Select the control, and click the Layout tab.
  2. In the Size group, change the Row Span or Column Span properties as necessary.
    For example, on the AccountDetails form, Desc. of Business spans two rows. Note that some types of controls have a constant display size regardless of the number of spaces they occupy.
Change the label on a control
  1. Select the control, and click the Basic tab.
  2. In the Appearance group, change the Caption property as necessary.
Make a control's data read only
  1. Select the control, and click the Basic tab.
  2. In the Behavior group, check the Read Only box.
Forms
View and edit form property values
  1. Click on the Form Designer surface anywhere outside of an occupied space, for example in the upper left corner.
    The form properties display in the tabbed area.
  2. View and change property values as necessary. For example, Active Control is the control you want the user's cursor to start on when the view first displays.
Determine where a form is used
  1. Click on the Form Designer surface anywhere outside of an occupied space, for example in the upper left corner.
    The form properties display in the tabbed area
  2. Click the Usage tab.
    All the views that contain the form are listed.
Change the width of a column of controls
  1. Click the column selector above a column.
    The column properties display in the tabbed area.
  2. In the Appearance group, change Width.
  3. Repeat for each column.

    If Size Type is Percent, the total of all column widths is normalized to 100.

Add or delete an empty row or column
  1. Right-click the selection rectangle to the left of a row or above a column.
  2. Use the context menu to delete or insert an empty row or column.
Add a form
  • Use Application Architect to add forms. For more information, refer to the Application Architect help topic, "Create a New Form".
