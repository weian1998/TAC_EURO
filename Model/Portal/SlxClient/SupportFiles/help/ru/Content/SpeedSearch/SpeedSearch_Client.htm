Быстрый поиск

Быстрый поиск является мощным инструментом, который поможет вам найти информацию, хранящуюся в базе данных Sage SalesLogix или внешние файлы. Например, вы можете осуществлять поиск по заявкам, чтобы помочь в решении проблемы клиенту или использовать его для поиска конкретного документа, например, презентации. Тип информации, по которой возможен поиск, определяется вашим системным администратором.

Перед открытием быстрого поиска вы можете выделить и скопировать текст, который сможете использовать в качестве ключевых слов при поиске. Используйте один из перечисленных методов для открытия быстрого поиска.

Чтобы открыть быстрый поиск из клиента Sage SalesLogix

  • Выполните одно из следующих действий:

    • В меню Поиск, нажмите Быстрый поиск.
    • На панели инструментов, нажмите кнопку Быстрый поиск.
    • При добавлении заявки или просмотре существующей заявки, нажмите кнопку Поиск на закладке Детали, для поиска решения. Вы можете вставить результаты поиска в заявку.
Чтобы открыть быстрый поиск вне клиента Sage SalesLogix
  • В меню Пуск, Программы, откройте Sage SalesLogix, затем нажмите Клиент быстрого поиска.

Чтобы начать быстрый поиск

  • Выполните одно из следующих действий:

    • Введите информацию в панели меню.
    • Нажмите кнопку Быстрый поиск из закладки детали заявки или представления создания заявки.
    • В меню Инструменты, нажмите Расширенный поиск.

Если у вас возникли проблемы с открытием быстрого поиска, свяжитесь с вашим системным администратором. Информация по настройке и решению проблем быстрого поиска находится в справке администратора.
