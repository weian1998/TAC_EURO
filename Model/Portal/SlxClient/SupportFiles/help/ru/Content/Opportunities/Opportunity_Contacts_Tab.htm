Контакты Сделки

Используйте это представление для создания, редактирования и удаления связи контакта со сделкой.

Как выполнить?

Открыть закладку

Закладка отображается на форме Создания сделки и в Детальном представлении Сделки. Если закладка не отображается, то откройте закладку Все закладки.

Добавить контакт к сделке
  1. Нажмите на кнопку Добавить контакт.
  2. Для того чтобы выбрать нужный контакт используйте поиск.
  3. Нажмите ОК.

Редактировать информацию о Контакте Сделки

Удалить Контакт из Сделки

Представление содержит следующие колонки:

  • Контакт - Отображает имя контакта. Щелкните гиперссылку для отображения Детального представления Контакта.
  • Должность - отображает должность контакта.
  • Роль - отображает роль контакта в сделке.
  • Основной - показывает, является ли данный контакт основным для текущего субъекта.
  • Телефон - отображает телефонный номер контакта.
  • E-mail - отображает адрес электронной почты контакта. Щелкните ссылку для открытия нового электронного сообщения с введенным адресом электронной почты контакта в поле Кому.
