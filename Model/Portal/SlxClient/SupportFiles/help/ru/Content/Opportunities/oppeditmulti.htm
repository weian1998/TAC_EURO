Обновление нескольких Сделок

При помощи формы Обновить Сделки вы можете обновить сразу несколько сделок.

Для обновления нескольких сделок

  1. На Панели Навигации нажмите кнопку Сделки.
  2. Откройте группу, которая содержит сделки, которые вы хотите обновить.
  3. Выберите записи, которые вы хотите обновить, и выполните одно из следующих действий:
    • Щелкните правой кнопкой мыши и выберите Обновление Сделок.
    • В Панели задач, под Общими задачами, нажмите Обновление Сделок.
  4. Используйте диалоговое окно Обновление Сделок для обновления любой из следующей информации:

    • Менеджер Субъекта

      1. Из выпадающего списка Обновить выберите Менеджер Субъекта.
      2. В поле На: нажмите кнопку Найти, выберите нового менеджера из списка, и затем нажмите ОК.
    • Добавить в Прогноз
      1. Из выпадающего списка Обновить выберите Добавить в Прогноз?.
      2. Выберите Да для включения сделки в отчеты по прогнозам или выберите Нет для исключения сделок из отчетов по прогнозам.
    • Вероятность Закрытия %
      1. Из выпадающего списка Обновить выберите Вероятность Закрытия %.
      2. Нажмите стрелку, а затем выберите новую вероятность закрытия из списка.

    • Планируемое закрытие

      1. Из выпадающего списка Обновить выберите Планируемое закрытие.

      2. Выберите, как вы хотите изменить ожидаемую дату закрытия:

        • Для переноса ожидаемой даты закрытия на определенную дату, выберите опцию К:, нажмите кнопку Календарь, выберите дату и нажмите ОК.
        • Для переноса ожидаемой даты закрытия назад на несколько дней, выберите опцию Перенести , выберите Назад из выпадающего списка, а затем введите количество дней, на которое вы хотите перенести дату.
        • Для переноса ожидаемой даты закрытия вперед на несколько дней, выберите опцию Перенести, выберите Вперед из выпадающего списка, а затем введите количество дней, на которое вы хотите перенести дату.
    • Комментарии
      1. Из выпадающего списка Обновить выберите Комментарии.
      2. Введите ваши заметки в поле На: рядом с выпадающем списком.
  5. Нажмите ОК.
