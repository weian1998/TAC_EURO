Удаление Конкурента из Сделки

Вы можете удалить конкурента из сделки, если эта компания болше не конкурирует по продажам, или была выбрана ошибочно. Вы можете удалять конкурентов из формы Создать сделку или детального представления Сделки. Удаление конкурента изменяет только связь конкурента с отдельной сделкой. Конкурент не удаляется из базы данных.

Для удаления конкурента из диалога Создать Сделку

  1. В таблице Конкуренты щелкните на конкурента, которого вы хотите удалить.
  2. Нажмите кнопку Удалить Конкурента.

Для удаления конкурента с Детального представления Сделки

  1. Выберите закладку Конкуренты. Если закладка не отображается, то откройте закладку Все закладки.
  2. Нажмите Удалить рядом с конкурентом, которого вы хотите удалить из сделки.
  3. Нажмите ОК.
