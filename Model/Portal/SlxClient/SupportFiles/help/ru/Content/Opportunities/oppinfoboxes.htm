Информационные поля Сделки

В таблице ниже описаны основные поля.

Поле Описание/Действие
Сделка Название, данное сделке при ее добавлении. Например, название сделки может быть: Компания XYZ Управление-Фаза 1.
Субъект Указанный для сделки субъект. Щелкните на ссылку для перехода к детальному представлению Субъекта.
Менеджер Субъекта Сотрудник вашей компании, ответственный за субъект. Нажмите кнопку Найти для поиска пользователя или команды внутри своей организации, отвечающего за взаимоотношения с субъектом.
Посредник Посредник, связанный со сделкой. Нажмите кнопку Найти для выбора посредника.
План. Закрытие Дата, когда ожидается закрытие сделки. Нажмите кнопку кнопкуКалендарь для выбора даты, затем нажмите ОК.  
Вероятность закрытия Вероятность того, что сделка будет закрыта. Нажмите кнопку стрелку и выберите элемент из списка.
Статус Статус сделки. Ваша компания определяет список статусов сделки. Нажмите кнопку стрелку и выберите элемент из списка.
Добавить в Прогноз Показывает, что сделка будет включена в отчеты по прогнозам.
Учетная система Имя соединения с учетным приложением. Нажмите кнопку Найти для выбора учетной системы. (Только для Интеграции учета)
Прайс лист Прайс лист, связанный с заказом. Отображаются прайс листы, связанные с выбранной учетной системой. Поиск прайс листа не доступен пока не выбрана Учетная система. (Только для Интеграции учета)
Комментарии Дополнительная информация о сделке.
