Удаление Продукта из Сделки

Вы можете удалить продукты из сделки если клиент больше не заинтересован в продукте, или продукты были добавлены по ошибке. Вы можете удалять продукты из диалога Добавить Сделку или Детального представления Сделки. Удаление продукта изменяет только связь продукта с отдельной сделкой. Продукт не удаляется из базы данных.

Для удаления продукта из диалога Добавить Сделку

  1. В таблице Продукты выберите нужный продукт, а затем нажмите Удалить.
  2. Нажмите Удалить.
  3. Нажмите ОК.

Для удаления продуктов с Детального представления Сделки

  1. На детальном представлении Сделки выберите закладку Продукты. Если закладка не отображается, то откройте закладку Все закладки.
  2. Нажмите Удалить для продукта, который вы хотите удалить, и нажмите ОК, чтобы подтвердить удаление.
