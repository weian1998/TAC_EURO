Удаление Контакта из Сделки

Вы можете удалять контакты из сделки в любое время. Вы можете удалять контакты из диалога Создать Сделку или Детального представления Сделки. Удаление контакта изменяет только связь контакта с отдельной сделкой. Контакт не удаляется из базы данных.

Для удаления контакта из диалога Создать Сделку

  1. В таблице Контакты щелкните на контакт, который вы хотите удалить.
  2. Нажмите кнопку Удалить Контакт, и затем ОК для подтверждения удаления.
  3. Нажмите ОК.

Для удаления контакта с Детального представления Сделки

  1. Выберите закладку Контакты.
  2. Нажмите кнопку Удалить, и затем ОК для подтверждения удаления.
  3. Нажмите ОК.
