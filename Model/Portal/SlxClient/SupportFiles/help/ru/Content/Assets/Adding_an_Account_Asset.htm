Добавление нового или редактирование Актива для Субъекта

Вы можете добавить новый Актив для Субъекта или изменить список продуктов, связанных с Субъектом. Также список Активов пополняется после того, как выиграна сделка, связанная с Субъектом.

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Для того чтобы добавить

  1. Открыть Детальное представление Субъекта.
  2. На закладке Активы, нажмите кнопку Добавить новый Актив.
  3. Заполнить необходимую информацию по продукту.
  4. Нажмите ОК.
    Актив появится в списке продуктов субъекта.

Для того чтобы редактировать

  1. На Детальном представлении Субъекта, выберите закладку Активы.
  2. На закладке Активы, выполните одно из следующих действий:
    • Нажмите Изменить, измените информацию по активу и нажмите ОК.

    • Выберите запись, которую необходимо изменить, произведите изменения в таблице, а затем нажмите Сохранить. Чтобы отменить изменения, нажмите Сброс до нажатия на кнопку Сохранить.

      Не все столбцы могут быть отредактированы в прямо в таблице. Чтобы определить, может ли столбец быть отредактирован, щелкните на таблице. Если ячейка стала белой, то столбец может быть отредактирован.
