Работа с интерактивными панелями

Интерактивные панели отображаются как закладки на странице Добро пожаловать и содержат настраиваемую, предопределенную аналитику в форме построения диаграмм и виджетов списка. Страница Добро пожаловать может содержать несколько примеров интерактивных панелей, и каждая панель может содержать несколько виджетов.

Существует две закладки интерактивной панели: Моя интерактивная панель и Продажи. Каждая закладка имеет несколько виджетов. Можно добавить или редактировать виджеты для отображения различных данных или отображения одних и тех же данных, но разными способами.

Используйте контекстное меню на закладке интерактивной панели, чтобы добавить новую закладку, копировать, скрыть/показать и удалить закладки, или поделиться закладками с командой или подразделением. Используйте перетаскивание для перегруппировки виджетов на закладке. Виджеты автоматически изменяют размер, учитывая тип диаграммы и данных, также как и колонки.

Можно также отправить все данные для группы из списочного представления к виджету списка группы на интерактивной панели с помощью ссылки Разместить на интерактивной панели.

Если вы не являетесь владельцем интерактивной панели, вы должны сделать копию, чтобы внести изменения.

Как выполнить?

Получить доступ к интерактивной панели
  • На навигационной панели, нажмите Добро пожаловать.
Добавить закладку
  1. Щелкните правой кнопкой на любой закладке интерактивной панели и нажмите Новая закладка.
  2. В поле Заголовок введите название закладки.
  3. При необходимости, выберите шаблон, описываюший макет, который вы хотите использовать для вашей закладки.
  4. Чтобы сделать выбранный шаблон - макетом по умолчанию, установите флаг По умолчанию.
  5. Нажмите ОК.
Копировать закладку
  1. Щелкните правой кнопкой на закладке интерактивной панели и нажмите Копировать закладку.
  2. В поле Заголовок введите название закладки.
  3. Нажмите OK.
    Закладка и все ее содержимое будет скопировано в новую закладку.
Удалить закладку
  1. Щелкните правой кнопкой на закладке интерактивной панели и нажмите Удалить закладку.
  2. Нажмите Да для подтверждения.
    Закладка и все ее содержимое будет удалено.
Скрыть закладку
  • Щелкните правой кнопкой на закладке интерактивной панели и нажмите Скрыть закладку.
    Закладка будет скрыта
Показать закладку
  1. Щелкните правой кнопкой на любой закладке интерактивной панели и нажмите Показать закладку.
  2. Нажмите Показать напротив имени закладки, которую хотите показать.
    Закладка отобразится на интерактивной панели.
Предоставить доступ к закладке другим пользователям
  1. Щелкните правой кнопкой на закладке интерактивной панели и нажмите Доступ.

  2. Выполните одно из следующих действий:

    • Нажмите Все, чтобы предоставить доступ всем пользователям.
    • Нажмите кнопку Добавить, найдите пользователей, команды или подразделения, которым вы хотите предоставить доступ к закладке, затем нажмите OK.
      Если вы добавили пользователия, команду или подразделение по ошибке, выделите элементы которые хотите удалить из списка и нажмите Удалить.

      Если вы предоставили доступ к закладке интерактивной панели, которая содержит группу, вы также должны предоставить доступ к группе, чтобы она могла отобразиться на интерактивной панели других пользователей.

Изменить заголовок закладки, изменить размер столбцов или сделать закладку - закладкой по умолчанию
  1. Щелкните правой кнопкой на закладке интерактивной панели и нажмите Изменить параметры.
  2. Введите новый заголовок для закладки.
  3. Выберите макет, который хотите использовать для закладки:
    • Одна колонка
    • Две равные колонки
    • Две колонки - левая больше
    • Две колонки - правая больше
  4. Установите флаг По умолчанию, чтобы сделать текущую закладку - закладкой по умолчанию.
    Закладка по умолчанию не может быть скрыта или удалена.
  5. Нажмите ОК.
