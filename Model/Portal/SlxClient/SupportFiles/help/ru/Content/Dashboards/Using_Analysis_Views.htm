Использование аналитических представлений

Аналитические представления показывают информацию как и наборы содержимого, однако они предлагают больше параметров для фильтрации данных. Аналитических представления открываются нажатием ссылок в наборах содержимого.

  • Подобно комплекту содержимого,таблица является ключом к аналитическому представлению.Графики представляют данные в таблицах. Вы управляете данными в таблице, используя фильтры. Строка общего итога внизу большинства таблиц вычисляет результаты.
  • Вы можете выбрать индивидуальные значения фильтра из множества групп фильтров, чтобы точно сфокусировать результаты своего поиска.
  • Размер графиков и таблицы может изменятся перемещением их границ.
  • Результаты таблицы аналитического представления могут экпортироваться в Microsoft Excel одни нажатием мышки.
