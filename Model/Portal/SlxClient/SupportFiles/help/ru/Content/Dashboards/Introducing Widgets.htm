Знакомство с виджетами

Виджеты - дополнительные программные модули, добавляемые к интерактивной панели. Существуют несколько типов виджетов, которые можно использовать для отображения данных. Наведите курсор мышки на диаграмму в виджете для отображения разреза и показателей, выбранных для этого виджета.

Разрезы - данные, связанные с сущностью, которую вы хотите просмотреть.

Например, для сущности субъект, вы можете просмотреть следующие разрезы:

  • Менеджер субъекта
  • Город
  • Отрасль
  • Доступ
  • SIC
  • Область
  • Статус
  • Подтип
  • Тип

Показатели - это способы вычисления данных, которые вы хотите просмотреть.

Например, для сущности субъект, вы можете использовать следующие показатели:

  • Дней после последнего дела
  • Средний доход
  • Среднее время в качестве клиента
  • Количество субъектов
  • Суммарный доход

Нажмите на ссылку в нижней части каждого виджета, чтобы просмотреть более подробную информацию по этим данным. Например, если вы настроили виджет списка групп для отображения Всех Открытых Сделок, ссылка Просмотр списка открывает списочное представление сделок. Нажмите Добро пожаловать на панели навигации, чтобы вернуться к интерактивной панели.

Существует несколько типов виджетов:

Графические виджеты

  • Гистограмма - Отображает данные в виде горизонтальной столбчатой диаграммы. Пример При выборе данных для среднего размера заказа для владельца, по оси Y отображаются команды, по оси X - количество заказов. Каждый столбец протягивается горизонтально до числа на оси X, наиболее близко совпадающего с суммой заказов.
  • Столбчатая диаграмма - Отображает данные в виде вертикальных столбцов. Пример При выборе данных для среднего размера заказа для владельца, по оси X отображаются команды, по оси Y - количество заказов. Каждый столбец протягивается вертикально до числа на оси Y, наиболее близко совпадающего с суммой заказов. Можно добавить уровень к диаграмме, который отображается как горизонтальная линия, пересекающая столбцы.
  • Воронка - Отображает данные в увеличивающейся пропорции в виде разноцветной воронки. Воронки часто используются для иллюстрации потенциального дохода или ожидаемого количества сделок. Примеры

Открытые сделки по плану продаж

Используйте воронку для отображения потенциального дохода от сделок для каждого уровня дохода.

  1. Щелкните правой кнопкой по закладке интерактивной панели и нажмите Добавить содержимое.
  2. Намжите кнопку Добавить под Воронкой.
  3. Введите Заголовок для воронки, например Открытые сделки по плану продаж.
  4. В списке Сущность выберите Сделка.
  5. В списке Группа выберите Все открытые.
  6. В списке Разрез выберите План.
  7. В списке Параметр выберите Суммарный план продаж.
  8. Определите, хотите ли вы Отображать метки, Обрезать метки и Длину меток.
  9. Нажмите ОК.
    Каждый Ряд значений плана продаж отображается различным цветом. Размер каждого уровня представляет собой количество сделок с определенным планом продаж.

Открытые сделки по стадии

Используйте воронку для просмотра количества сделок в каждой стадии.

  1. Щелкните правой кнопкой по закладке интерактивной панели и нажмите Добавить содержимое.
  2. Намжите кнопку Добавить под Воронкой.
  3. Введите Заголовок для воронки, например Мои сделки по стадии.
  4. В списке Сущность выберите Сделка.
  5. В списке Группа выберите Все сделки.
  6. В списке Разрез выберите Стадия.
  7. В списке Параметр выберите Количество сделок.
  8. Определите, хотите ли вы Отображать метки, Обрезать метки и Длину меток.
  9. Нажмите ОК.
    Каждая стадия представлена разным цветом. Размер каждого уровня представляет собой количество сделок в определенной стадии.
  • Линейчатая диаграмма - Отображает данные в виде последовательности точек, соединенных линией. Пример При выборе данных для активных маркетинговых кампаний по региону, по оси Х отображаются команды, по оси Y - количество кампаний.
  • Круговая диаграмма - Отображает данные в виде цветной круговой диаграммы с соответствующей легендой значений. Пример При выборе субъекта по региону с пятью сегментами, диаграмма отображает круг с пятью разноцветными сегментами, легенда показывает команды и общее количество субъектов на команду. Можно настроить круговую диаграмму таким образом, чтобы она отображала 5 или 10 сегментов, и выбрать, отображать ли сегмент Другое, включающий в себя все непоказанные данные.

Виджеты - списки

  • Группа - Отображает определенную информацию по группе. Пример Если вы выбрали Мои закрытые сделки. Можно отсортировать столбцы в списке и перейти к субъекту для подробной информации.

    		•
    • Если вы предоставили доступ к закладке интерактивной панели, которая содержит группу, вы также должны предоставить доступ к группе, чтобы она могла отобразиться на интерактивной панели других пользователей.
    • Ссылка просмотреть группы не доступна для истории

  • Ссылки - Отображает определяемые вами ссылки. Пример добавьте ссылки, которые вы часто используете в Sage SalesLogix или интернете.
  • Недавно просмотренные - Отображает недавно просмотренные детальные представления в виде ссылок. Если вы просматривали детальные представления субъекта и контакта, они попадут в список в виде ссылок.

Другие виджеты

  • Дела на сегодня - Отображает список всех ваших дел, запланированных на сегодня. Чтобы открыть детальную информацию по делу, нажмите на ссылку.
  • Добро пожаловать - Отображает знакомство с Web клиентом Sage SalesLogix, со ссылками на справочную информацию по Web клиенту Sage SalesLogix. Краткое описание всех функций предоставляемых Sage SalesLogix, чтобы помочь вам в управлении вашей работой.
