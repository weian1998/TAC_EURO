Использование виджетов

Sage SalesLogix предоставляет несколько стандартных виджетов. Вы можете добавлять, редактировать и удалять виджеты из интерактивной панели, также вы можете изменять порядок виджетов на странице путем перетаскивания.

Используя предопределенные фильтры, вы можете отображать показатели данных через диапазоны или различные значения и отобразить эти данные в одном из графических виджетов. Например, вы можете настроить столбчатую диаграмму на отображение всех закрытых и выигранных сделок по доступу, общему количеству продаж или же настроить круговую диаграмму отражающую выигранные продажи по командам.

Однако, при использовании фильтров различных значений все графические виджеты, за исключением круговой диаграммы, ограниченны 10 точками; первые десять из выборки. Виджет круговой диаграммы, позволяет выбрать настройку +1 - категория Другие, в которую попадают остальные данные. При использовании диапазонных фильтров, данные возвращаются для каждого значения из диапазона. Это позволяет отображать в графиках намного больше, чем 10 сечений или столбцов, но данная настройка может быть изменена вашим системным администратором.

Как выполнить?

Добавить виджет
  1. Щелкните правой кнопкой по закладке Моя интерактивная панель, затем выберите пункт меню Добавить содержимое.
  2. В диалоге Добавить содержимое, нажмите кнопку Добавить на виджете, который хотите добавить.
    Например, чтобы добавить виджет круговой диаграммы, нажмите Добавить под Круговой диаграммой.
    Пустой виджет отобразится на интерактивной панели и откроется редактор виджета.
  3. Настройте виджет, затем нажмите OK.
Изменить существующий виджет
  1. На панели инструментов виджета, нажмите кнопку Настройки .
  2. Произведите необходимые изменения.
  3. Нажмите ОК.
    Виджет отобразится с изменениями.
Настроить графический виджет

    • Гистограмма

      • Заголовок - Заголовок, которые отобразится на панели заголовка виджета.

      • Подпись - Текст, который отобразится ниже заголовка, для предоставления дополнительной информации.

      • Сущность - Сущность из которой вы хотите получить данные. Например, Сделки. Все сущности, имеющие списочное представление доступны для просмотра в виде графиков и списков.

      • Группа - Группа, из которой вы хотите просмотреть данные. Например, Все сделки. Все группы, доступные пользователю, могут быть использованы в графиках и списках.

      • Показатель - Данные, которые вы хотите просмотреть. Например, Вероятность. Все диапазонные фильтры и фильтры различных значений доступны в качестве показателей. 

        Доступные показатели зависят от доступных фильтров сущности и других настроек. Для более подробной информации, обратитесь к вашему системному администратору.

      • Разрез - Способ вычисления данных. Например,Общая фактическая сумма. Все сущности со списочными представлениями имеют количественные разрезы, например, Кол-во наводок и кол-во субъектов. Существуют дополнительные разрезы для сделок и истории.

      • Ось-X - Метка, используемая для описания данных по оси X графика.

      • Ось-Y - Метка, используемая для описания данных по оси Y графика.

      • Отображать метки - Выберите Да или Нет, чтобы отображать метки или не отображать метки на графике.

      • Обрезать метки - Выберите Да или Нет, чтобы обрезать или не обрезать метки так, чтобы они помещались на графике.

      • Длина метки - Максимальная длина метки.

Настроить виджет списка группы
  1. В диалоге Настройка списка группы, введите заголовок виджета.
  2. Выберите Сущность и Группу из выпадающих списков. Например, Сделки и Закрытые и выигранные.
  3. Введите Количество записей , которое будет отображаться в списке. Например, 10, 25, or 50.
  4. Введите количество записей, которые вы хотите видеть в поле Видимые записи; 10, 25 или 50.
    Например, если вы ввели 50 записей на шаге 2, но хотите видеть только десять записей одновременно, введите 10. Виджет изменит размеры, под количество видимых записей.
  5. Нажмите ОК.
    Виджет отобразит список.

  		6.

    Вы можете использовать любое число от 1 и до настроенного максимального количества записей для отображения. Максимум по умолчанию - 50.
Настроить виджет списка ссылок
  1. В диалоге Настройка ссылок, введите заголовок виджета.
  2. Нажмите Добавить ссылку.
  3. Дважды щелкните на поле URL иконки и введите путь до иконки, которая будет отображаться для ссылки. Например, /google.jpg.

    		4.

    Этот URL должен быть относительным путем к изображению на Web сервере.
  4. Дважды щелкните на поле URL ссылки и введите URL для ссылки, которую хотите отобразить. Например, http://localhost/SlxClient/Account.aspx.

    5. Этот путь может быть относительным путем к порталу SLXClient или абсолютной ссылкой на внешний сайт. URL для внешних сайтов, должен начинаться с http://. Например, для ссылки на www.google.com вы должны ввести значение URL http://www.google.com. Иначе, браузер будет искать http://<SLXClient site>/www.google.com.
  5. Дважды щелкните на поле Текст ссылки и введите текст, который будет отображаться в виде ссылки. Например, Страница поиска Google.
  6. Дважды щелкните на поле Цель и введите идентификатор цели. Например, если вы хотите, чтобы ссылка открылась в новом окне браузера, введите Новое окно. Когда вы нажмете на ссылку, откроется новое окно/закладка браузера.

    		7.

    Если вы оставите это поле пустым, при нажатии на ссылку, она откроется в активном окне/закладке браузера.
  7. Нажмите Добавить ссылку, чтобы добавить другую ссылку или нажмите OK, чтобы закрыть редактор.
    Когда вы нажмете OK, виджет обновится и отобразит ссылки.
Минимизировать виджет
  • На панели инструментов виджета, нажмите кнопку Минимизировать .
    Содержание виджета будет свернуто. Нажмите кнопку снова, чтобы развернуть виджет.
Удалить виджет
  • На панели инструментов виджета, нажмите кнопку Закрыть .
    Виджет будет удален из интерактивной панели.
