Редактирование пользовательских настроек

Sage SalesLogix предоставляет инструменты, позволяющие изменять некоторые компоненты интерфейса для того, чтобы вы могли работать как вам удобнее всего. Пользовательские опции позволяют вам устанавливать определенные персональные настройки для таких возможностей как загрузка веб клиента Sage SalesLogix и напоминания ваших дел. Параметры настройки регистрируются только для пользователя, который их установил, и являются невидимыми для других пользователей Sage SalesLogix.

Редактирование пользовательских настроекions

  1. В меню Сервис, выберите Параметры.
  2. Перейдите на закладку параметров, которые вы хотите отредактировать.
  3. Закладка Общие/Поиск для редактирования формы запуска, куратора или команды по умолчанию, основных шаблонов рассылки, количества шаблонов в недавно используемом списке или настроек электронной почты.
  4. Закладка Группа для выбора группы записей, которая будет использоваться в качестве группы по умолчанию для любого типа записей.
  5. Закладка Календарь для настройки параметров календаря. Вы можете определить информацию, которая отображается в календаре, вид по умолчанию или день начала и время завершения.
  6. Закладка Изменить пароль, чтобы изменить пароль.
  7. Закладка Сделки, чтобы настроить параметры для новых сделок.
  8. Закладка Оповещения , чтобы установить значения по умолчанию, для определения времени за кототорое Вы будете предупреждены о назначенном деле.
  9. Закладка Дела, чтобы установить параметры дел основного представления, последующих дел или оповещения.
  10. Нажмите Сохранить.

Необходимо нажать кнопку Сохранить на каждой закладке или выбранные опции не будут сохранены.
