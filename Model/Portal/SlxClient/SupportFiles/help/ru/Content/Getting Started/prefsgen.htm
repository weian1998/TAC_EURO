Параметры Общие

Используйте закладку Общие для определения таких настроек, как название карточки, которая будет открываться по умолчанию при вашем первом входе в , и кого использовать как владельца по умолчанию для записей, которые вы создаете.

Для настройки общих параметров

  1. В меню Сервис, выберите Параметры.
  2. Выберите закладку Общее.
  3. В поле Показывать при Запуске выберите карточку по умолчанию, которую вы хотите показывать при вашем входе в Web Client (например, Что Нового? или Календарь).
  4. В поле Куратор/Команда по Умолчаниюнажмите кнопкуНайти, выберите пользователя или команду, которого вы хотите назначить по умолчанию куратором новых субъектов, а затем нажмите OK .
  5. Если вы хотите быть автоматически выходить из веб-клиента после определенного периода бездействия, выберите пункт Включить автоматический выход и выберите количество минут бездействия из раскрывающегося списка.
  6. Если вы хотите получать уведомление, когда у вас имеются несохраненные изменения на странице, выберите флажок Напомнить перед уходом со страницы с несохраненными данными. Это свойство выбрано по умолчанию. Уберите флажок если не хотите получать уведомления.
  7. Только для Мульти-валюты: Чтобы выбрать тип валюты, нажмите на кнопку Моя валюта Найти и выберите валюту из списка, а затем нажмите кнопкуOK.
  8. В разделе Группа контактов Intellisync выберите Группу контатков в порядке обработке при синхронизации.
    Если Специальная группа контактов, которые вы хотите синхронизировать, не отображается в списке, вам необходимо создать группу SyncSalesLogix.
  9. Выберите вид отображения списка флажков, если вы хотите, чтобы все ваши списочные представления были отображены в виде столбца флажков.
  10. Выберите Базовые шаблоны рассылки, которые вы хотите использовать при создании нового электронного письма контакту, факса или письма.
    1. В поле Тип нажмите стрелку вниз и выберите Контакт.
    2. В поле Базовый шаблон E-mail нажмите кнопку Найти .

    3. Выберите шаблон и нажмите ОК.

    4. Повторите шаги b-c для Базового шаблона письма и Базового шаблона факса.

  11. Выберите Базовые шаблоны рассылки, которые вы хотите использовать при создании нового электронного письма контакту, факса или письма.

    При выборе шаблонов для наводок, выберайте лишь те шаблоны, которые оканчиваются на "(Наводка)".

    1. В поле Тип нажмите стрелку вниз и выберите Наводка.
    2. В поле Базовый шаблон E-mail нажмите кнопку Найти .

    3. Выберите шаблон и нажмите ОК.

    4. Повторите шаги b-c для Базового шаблона письма и Базового шаблона факса.
  12. В поле Меню Написать - Последние Шаблоны введите количество шаблонов, которые должны показываться в списке последних использованных шаблонов.
  13. В поле Поставщик Факсов щелкните указывающую вниз стрелку и выберите поставщика факсов, которого вы хотите использовать для отправки факсов.
  14. В поле Записать в Историю выберите Запросить если вы хотите чтобы диалог Завершить E-mail отображался перед тем, как сообщение электронной почты запишется в историю. Если вы хотите чтобы сообщение было записано в историю без вашего подтверждения, выберите Не Запрашивать.

    Этот параметр применим только к e-mail. Необходимо установить параметры для Отправить SLX и записать в историю в Desktop Manager.

  15. Чтобы установить Desktop Integration module, нажмите Установить улучшения.
  16. Для пользователей Разъединенные пользователи Веб Клиента: Чтобы управлять отчетами offline, убедитесь, что проставлен флажок Использовать Активные Отчеты .
  17. Нажмите Сохранить.
