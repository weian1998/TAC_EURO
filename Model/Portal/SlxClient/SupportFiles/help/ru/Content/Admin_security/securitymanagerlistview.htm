Использование менеджера безопасности

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Используйте списочное представление Менеджера безопасности для управления доступом к полям в профилях безопасности.

Как выполнить?

Открыть списочное представление менеджера безопасности
  • На навигационной панели, нажмите Администрирование, затем нажмите Менеджер безопасности.
Обновить доступ к полям в профиле безопасности
  1. На представлении Менеджера безопасности выберите профиль безопасности, который необходимо обновить.
  2. Для каждого из полей выберите один из следующих вариантов доступа:
  1. Чтение Запись – Пользователи могут просматривать и редактировать данные. Это значение является значением по умолчанию для всех таблиц и полей.
  2. Только чтение – Пользователи могут лишь просмтривать данные.
  3. Нет доступа – Пользователи не могут просматривать и редактировать данные.

Чтобы применить один и тот же вариант доступа ко всем полям профиля безопасности, установите флажок в соответствующем заголовке столбца. Например, установив флажок в заголовке столбца Чтение/Запись вы дадите доступ на просмотр и редактирование всех данных всех полей в профиле безопасности.

  1. По завершению внесения изменений, нажмите Сохранить.
    Для отмены изменений, до нажатия на кнопку сохранить, нажмите Сброс.
Обновить список
  • Нажмите кнопку Обновить.
Использовать обычные задачи
  • На панели задач, в разделе Обычные задачи, вы можете:
  • Добавить профиль
  • Изменить профиль
