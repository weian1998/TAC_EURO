Назначение пользователям роли Стандартный пользователь

По умолчанию, пользователям, созданным в Web Client'е, назначается роль Стандартный пользователь. Эта роль предоставляет права на добавление, редактирование и удаление неадминистративных сущностей Web клиента (субъекты, контакты и сделки).

Пользователям, созданным в Администраторе, не приписывается роль. Если у вас есть существующие пользователи или вы создаете новых пользователей в Администраторе, необходимо назначить пользователям стандартную роль пользователя.

Для назначения роли

  1. Откройте списочное представление Пользователей.
  2. Выберите пользователей, которые должны обладать правами на добавление, редактирование и удаление неадминистративных сущностей в Web клиенте.
  3. На панеле Задачи пользователя, нажмите Добавить к роли.
  4. В диалоге Выбор роли, найдите и выберите роль Стандартный пользователь, и нажмитеOK.

  5. Нажмите ОК.
