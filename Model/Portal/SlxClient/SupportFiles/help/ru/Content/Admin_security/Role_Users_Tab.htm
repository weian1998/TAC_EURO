Закладка Пользователи

Используйте закладку Пользователи, чтобы назначить одному или более пользователям роли, или удалить пользователя из роли. Когда пользователю назначена роль, он получает доступ к действиям опрелеленным в этой роли. Если пользователю назначена более чем одна роль, он/она будет иметь доступ ко всем действиям, связанным с ролями, даже если каждая роль содержит различные действия.

Как выполнить?

Назначить роль пользователю

  1. Откройте детальное представление Роли.
  2. Откройте закладку Пользователи.
  3. Нажмите кнопку Добавить пользователей.
  4. Воспользуйтесь поиском для того, чтобы найти и выбрать всех пользователей, которым вы хотите назначить роль.
  5. Нажмите Добавить.
  6. Нажмите Закрыть по окончанию.

Удалить пользователя из роли

  1. Откройте детальное представление Роли.
  2. Откройте закладку Пользователи.
  3. Выберите запись, которую вы хотите удалить и нажмите кнопку Удалить.

  4. Нажмите Да.
