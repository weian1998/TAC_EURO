Использование списочного представления Ролей

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Списочное представление Ролей отображает роли в виде таблицы. Таблица содержит информацию о каждой роли, например, название роли и описание.

  • Чтобы отсортировать таблицу по конкретному полю, нажмите на заголовк столбца, по которому необходимо произвести сортировку.
  • Чтобы отобразить все роли определенного типа, выбреите необходимую группу.
  • Чтобы отобразить роли, отвечающие заданым критериям, настройте фильтры.

Как выполнить?

Создать роль

Редактировать или удалить роль

Открыть списочное представление ролей
  • На навигационной панели, нажмите Администрирование, затем нажмите Роли.
Открыть детальное представление ролей
  • Нажмите на название роли, чтобы открыть Детальное представление роли.
