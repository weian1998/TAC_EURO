Просмотр выбранных настроек импорта

Используйте это представление для просмора сделанных настроек пред запуском импорта наводок в Sage SalesLogix.

Чтобы выбрать

  1. Во время импорта наводок, после выбора групповы действий после импорта, нажмите Далее.
  2. Просмотрите следующие:
    • Файл со списком наводок - Файл, содержащий наводки для импорта.
    • Доступ по умолчанию - Доступ для всех импортируемых наводок.
    • Источник наводки по умолчанию - Источник наводки для всех импортируемых наводок.
    • Добавить в группу наводок - Будут ли все импортируемые наводки добавлены в группу.
    • Название группы - Все импортируемые наводки будут добавлены в эту группу.
    • Проверить на дубликаты - Указывает, будет ли импорт включать в себя поиск повторяющихся записей в Sage SalesLogix.
    • Действия импорта - Список действий, которые будут применены к импортируемым наводкам.
  3. Нажмите Подвердить для начала процесса импорта.
