Конвертация наводок

Как только наводке присвоена квалификация, наводку можно конвертировать в контакт или субъект. Диалоговое окно Конвертирования Наводки содержит информацию о похожих контактах и субъектах. Наводки с совпадающей информацией можно добавить в существующие контакт, субъект или можно создать новые контакт и субъект на основе наводки. Для поиска дублирующих записей можно воспользоваться функцией Фильтрации.

Sage SalesLogix может осуществлять поиск только среди записей, к которым у вас есть доступ. Возможно существуют дублирующие записи, к которым вы не можете получить доступ.

Для конвертации квалифицированных наводок

  1. Откройте Детальное представление Наводки, которую необходимо конвертировать, а затем нажмите кнопку Преобразовать квалиф. наводку.
    Откроется диалоговое окно Конвертировать Наводку с информацией по наводке в верхней части диалогового окна и информацией по дубликату контакта или субъекта в нижней части диалогового окна.
  2. На закладке Фильтры совпадений, отметьте флажками фильтры, которые вы хотите применить.
  3. Выберите один из следующих Параметров:
    • Соответ. всем выбранным фильтрам - запись будет найдена, только при совпадении всех выбранных параметров фильтрации.
    • Соответ. любому из выбранных фильтров - запись будет найдена, при совпадении хотя бы одного из выбранных параметров фильтрации.
  4. Для выполнения Расширенного поиска нажмите Расширенные параметры.
  5. Нажмите Изменить потенциальные соответствия.
  6. Нажмите на стрелку вниз в поле Конфликты, чтобы выбрать способ рзарешения конфликтов.
    • Выберите пункт Побеждает субъект если необходимо, чтобы осталась информация существующего контакта.
    • Выберите пункт Выигрывает Наводка если необходимо, чтобы спорная информация была заменена на информацию из наводки.
  7. Если необходимо создать сделку, установите флаг Создать сделку после конвертации наводки .
    Вся информация по делам, истории и заметкам Наводки будет связана с созданной Сделкой.
  8. Выполните одно из следующих действий:
    • Объединить наводку с дублирующим контактом
      1. В таблице Совпадающие контакты/наводки нажмите Объединить рядом с записью, к которой нужно добавить информацию из наводки.
      2. Выполните все шаги для объединения записей.
    • Добавить контакт в дублирующий субъект
      • В таблице Совпадающие Субъекты нажмите Добавить контакт рядом с записью, к которой нужно добавить информацию о контакте.
        Наводка будет добавлена как Контакт для выбранного субъекта и откроется Детальное представление нового Контакта.
    • Продолжить конвертацию наводки в новый контакт и субъект
      • Нажмите Конвертация.
