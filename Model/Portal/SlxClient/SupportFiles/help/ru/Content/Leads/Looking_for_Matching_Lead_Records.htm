Поиск Совпадающих Записей Наводки

При добавлении наводки, вы можете сравнить наводку с другими наводками, контактами или субъектами, существующими в базе данных. Если запись со схожей информацией существует в базе данных, то вы можете избежать дублирования записей.

Sage SalesLogix может осуществлять поиск только среди записей, к которым у вас есть доступ. Возможно существуют дублирующие записи, к которым вы не можете получить доступ.

Поиск совпадающих записей

  1. Введите информацию о наводке в диалог Создать Наводку , а затем нажмите Найти похожие записи.
    Откроется диалоговое окно Поиск дубликатов.
  2. Рядом с Тип, выберите какие типы записей следует искать.
  3. На закладке Фильтры совпадений, отметьте флажками фильтры, которые вы хотите применить.
  4. Выберите один из следующих Параметров:
    • Соответствует всем выбранным фильтрам - Запись является дубликатом только, если она совпадает со всеми выбранными фильтрами.
    • Соответствует любому из выбранных фильтров - Запись является дубликатом, если она совпадает с любым из выбранных фильтров.
  5. Для выполнения Расширенного поиска нажмите Расширенные параметры.
  6. Нажмите Изменить потенциальные соответствия.
  7. Выполните одно из следующих действий:
    • Щелкните на Открыть рядом с потенциально совпадающей записью, чтобы перейти в детальное представление записи.
    • В таблице Совпавшие Записи Субъекта, щелкните на Добавить Контакт, чтобы преобразовать наводку в контакт для существующего субъекта.
    • Нажмите Отмена для возвращения в представление Создать Наводку и продолжите добавлять наводку.
