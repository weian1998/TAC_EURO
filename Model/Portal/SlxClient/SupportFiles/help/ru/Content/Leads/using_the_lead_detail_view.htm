Использование Детального представления Наводки

Используйте детальное представление для просмотра, добавления или редактирования информации о выбранной записи. Детальное представление состоит из информационных полей, закладок и определяемой пользователем, средней панели.

Ваша рабочая область может выглядеть по-разному, в зависимости от прав доступа или использования нестандартных настроек .

Как выполнить?

Открыть Детальное представление Наводки.
  • На навигационной панели нажмите Наводки, чтобы открыть Списочное представление Наводок. При необходимости, используйте Поиск, чтобы найти наводку и затем щелкните по имени нужной наводки.

Используйте Поиск для нахождения записей

Просмотреть группы

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.
Использовать общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Электронная почта
  • Почтовая Рассылка
  • Добавить отклик маркетинговой программе
  • Добавить заметку
  • Новая встреча
  • Новый звонок
  • Новое действие

Информационные поля

Информационные поля наводки содержат основную информацию по наводке. В данном разделе также содержатся поля квалификации наводки. Пользователи с соответствующими правами могут редактировать эти поля.

Закладки

Каждая закладка содержит определенную информацию, которую вы можете использовать для связанных с наводкой дел. Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
