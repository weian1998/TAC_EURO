Работа с дублирующимися Наводками, найденными при импорте

При импорте наводок, вы можете просматривать наводки или контакты, которые уже существуют в базе данных. Используйте это представление при ручном импорте или при объединении дублирующихся записей.

Sage SalesLogix может осуществлять поиск только среди записей, к которым у вас есть доступ. Возможно существуют дублирующие записи, к которым вы не можете получить доступ.

Для поиска совпадающих записей

  1. Откройте Детальное представление истории импорта наводок и выберите закладку Дубликаты.
  2. Нажмите на кнопку Решить возле Наводки, которую вы хотите импортировать или объединить.
  3. Выберите Тип записей, которые нужно искать.
  4. На закладке Фильтры совпадений, отметьте флажками фильтры, которые вы хотите применить.
  5. Укажите одину из следующих Настроек:
  • Соответствует всем выбранным фильтрам - запись будет найдена, только при совпадении всех выбранных параметров фильтрации.
  • Соответствует любому из выбранных фильтров - запись будет найдена, при совпадении хотя бы одного из выбранных параметров фильтрации.
  1. Для выполнения Расширенного поиска нажмите Расширенные параметры.
  2. Нажмите Изменить потенциальные соответствия.
  3. Выполните одно из следующих действий:
  4. Объединить наводку с дублирующим контактом
  1. В таблице Совпадающие контакты/наводки выберите контакт и нажмите Объединить рядом с записью, к которой нужно добавить информацию о наводке.
  2. Выполните все шаги для объединения записей.
  1. Импортировать наводку вручную
  • Нажмите на кнопку Создать Наводку.
  1. Нажать кнопку Отмена чтобы оставить наводку как дубликат.

Когда дублирующая запись разрешена, она удаляется из закладки Дубликаты.
