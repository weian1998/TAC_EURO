Управление дубликатами

При импорте наводок, вы можете просматривать наводки или контакты, которые уже существуют в базе данных. Если в базе данных имеется запись с такой же информацией, вы можете избежать дублирования записей.

Sage SalesLogix может осуществлять поиск только среди записей, к которым у вас есть доступ. Возможно существуют дублирующие записи, к которым вы не можете получить доступ.

Для поиска совпадающих записей

  1. Во время импорта наводок, после настройки соответствия полей, нажмите Далее.
  2. Выберите пункт Проверка на дубликаты наводок.
  3. Выберите Тип записей, которые нужно искать.
  4. На закладке Фильтры совпадений, отметьте флажками фильтры, которые вы хотите применить.

Доступные параметры зависят от значений, выбранных при настройке соответсвия полей.

  1. Для Расширенного поиска нажмите Расширенные параметры соответствия.

  2. Для автоматического слияния дублирующихся записей при импорте, выберите пункт Автоматически объединять дублирующиеся записи.
  3. В поле Конфликты выберите одно из следущих значений:
  4. Существующая запись победила - Когда записи сливаются, используется информация по записи, уже существующей в базе данных.
  • Импортированная запись победила - Когда записи сливаются, информация по импортируемой записи перезаписывает информацию, уже существующую в базе данных.

    Для получения дополнительной информации смотрите темуПравила автоматического слияния при импорте.
  1. Нажмите кнопку Выполнить проверку для начала поиска дубликатов для первых 100 импортируемых записей.
  2. Если вы хотите проверить дублирующиеся записи в файле- источнике импорта, установите флажок Проверить дубликаты, содержащиеся в импортируемом исходном файле.

    3. Установка этого флажка может увеличить время, затраченное на выполнение импорта.
  3. Нажмите Далее для перехода к следующему шагу.

Найденные дублирующиеся записи и не объединенные отображаются в Детальном представлении истории импорта наводок на закладке Дубликаты после завершения импорта. Из этой закладки вы можете управлять дублирующимися записями.
