Добавление Отклика в процессе импорта

Используйте это представление для назначения отклика каждой импортированной наводке. При помощи Откликов можно отслеживать взаимодействия с контактом или наводкой. Отклик будет отображаться на закладке Отклики детального представления.

Чтобы добавить

  1. В процессе импорта наводок, создайте групповое действие и нажмите кнопку Определить рядом с полем Добавить отклик.
  2. Рядом с полем Источник наводки нажмите кнопку Найти для поиска основного источника.
  3. Рядом с полем Дата Отклика нажмите кнопкуКалендарь для выбора даты, затем нажмите ОК.  
  4. В поле Статус нажмите стрелку и выберите элемент из списка.
  5. В поле Способ отклика нажмите стрелку и выберите элемент из списка.
  6. Если необходимо, в поле Маркетинг нажмите кнопку Найти для поиска маркетинговой кампании. Ваш выбор будет добавлен в диалоговое окно.
  7. Рядом с полем Стадия нажмите на указывающую вниз стрелку и выберите значение из списка.
  8. В поле Интерес нажмите стрелку и выберите каким образом заинтересованы контакт или наводка.
  9. В поле Уровень интереса нажмите стрелку и выберите уровень интереса.
  10. В поле Комментарии введите любые примечания.
  11. Нажмите Сохранить.
