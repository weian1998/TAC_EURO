Выбор файла для импорта

Используйте это представление, чтобы выбрать файл, содержащий наводки, которых вы хотите импортировать.

Чтобы выбрать файл

  1. Откройте Мастер импорта наводок.
  2. Рядом с полем Выберите файл, содержащий информацию по наводкам нажмите кнопку Выберите файл, выберите файл и нажмите Открыть.
    Файл появится в поле Текущий файл списка наводок.
  3. Рядом с полем Доступ по умолчанию нажмите кнопку Найти для выбора пользователя или команды, которые будут иметь доступ к импортированным наводкам.
  4. Рядом с полем Ист. наводки по умолчанию нажмите кнопку Найти и используйте поиск для выбора источника информации о наводках.
  5. Для добавления наводок в специальную группу выберите пункт Добавить импортированные наводки в спец. группу и выполните одно из следующих действий:
    • Для добавления импортированных наводок в новую группу, выберите пункт Создать новую спец. группу и укажите название группы.
    • Для добавления импортированных наводок в существующую группу, выберите пункт Добавить в существующую спец. группу и выберите название группы из списка.
  6. Нажмите Далее для перехода к следующему шагу.
