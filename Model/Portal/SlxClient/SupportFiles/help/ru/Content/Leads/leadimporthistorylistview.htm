Использование Списочного представления Истории импорта Наводок

Используйте Списочное представление Истории импорта для просмотра списка импортов списков наводок.

Как выполнить?

Открыть Списочное представление Истории импорта Наводок
  • В меню Сервис выберите пункт История импорта.
Открыть Детальное представление Истории импорта Наводок из Списочного представления
  • В Списочном представлении Истории импорта Наводок нажмите на Номер импорта, который вы хотите просмотреть.

Работа со Списочным представлением

Представление содержит следующие колонки:

  • Номер импорта - Уникальный номер, идентифицирующий импорт наводок.
  • Описание - Название списка импортируемых записей.
  • Дата Создания - Дата импорта списка.
  • Статус - Статус импорта. Например, Завершенный.
  • Кол-во импортированных - Количество импортированных записей по Наводкам.
  • Кол-во дубликатов - Количество дубликатов, найденных во время импорта.
  • Всего объединено - Количество записей, объединенных во время импорта.
  • Кол-во ошибок - Количество ошибок, возникших во время импорта.
