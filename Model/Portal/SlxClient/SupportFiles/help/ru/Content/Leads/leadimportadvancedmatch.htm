Использование расширенных параметров соответствия

Используйте настройки расширенного поиска совпадений для определения, каким образом будет осуществляться поиск потенциальных дубликатов.

Для установки расширенных параметров

  1. Используйте поля блока Определение дубликатов по Оценке для определения типа поиска совпадений. Настройки доступны только на шаге Управление дубликатами при импорте наводок.
  1. В поле Дубликат введите минимальное значение (в процентах) совпадения, при котором записи будут считаться дубликатами.
  2. В поле Возможный дубликат введите минимальное значение (в процентах) совпадения, при котором записи будут считаться возможными дубликатами.
  1. Выберите пункт Использовать морфологический поиск чтобы учитывать альтернативные написания слов.
  2. Выберите пункт Использовать звуковой поиск чтобы учитывать похожие по звучанию слова.
  3. Выберите пункт Использовать синоним чтобы учитывать похожие по значению слова.
  4. Выберите пункт Использовать нечеткий поиск чтобы учитывать похожие или слова с опечатками. Затем выберите Уровень нечеткости из списка, чтобы определить при каком уровне различия, слова будут определяться как дубликаты. Укажите количество символов, при котором слова будет считаться разными.

Нечеткое совпадение:

  • Каждое совпадение должно начинаться с первой буквы критериев.
  • После первой буквы, каждое совпадение должно иметь число различий в символах не более уровня нечеткости. Например, уровень нечеткости 3 найдет любое название с тремя или менее различиями в символах.
    Пример

    Если вы используете "Jon" как критерий поиска, уровень нечеткости 1 найдет, что Jon и John - нечеткое совпадение, а Jon и Johnny - нет. Чтобы Johnny стало совпадением, вам нужно увеличить уровень нечеткости до 3, но тогда вы получите также имена Jack и Jeff.

    Используйте следующую таблицу как пример других совпадений, которые будут найдены в зависимости от уровня нечеткости.

    Уровень Нечеткости 1

    (разрешает 1 различный символ)

    Уровень Нечеткости 2

    (разрешает 2 различных символа)

    Уровень Нечеткости 3

    (разрешает 3 различных символа)
    Jon Также будет найдено: Также будет найдено:
    John Jim Jeff
    Joe Jay Jacob
    Joan Josh Joyce
      Jason Jack
      Juan Johnny
      Joel  
  1. Нажмите ОК.
