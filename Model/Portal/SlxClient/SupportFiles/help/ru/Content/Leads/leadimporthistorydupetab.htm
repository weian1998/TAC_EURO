История импорта Наводок - Закладка Дубликаты

Используйте эту закладку для просмотра списка записей, отмеченных как дубликаты, которые должны быть решены прежде, чем их можно будет импортировать.

Для того чтобы открыть

  • В Детальном представлении Истории импорта Наводок выберите закладку Дубликаты. Если закладка не отображается, то откройте закладку Все закладки.

Для того чтобы обработать дубликаты

  • Щелкните по ссылке Решить рядом с необходимой записью.

Откроется диалог Обработка дубликатов при импорте Наводок.
