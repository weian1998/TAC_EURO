Добавление Цели Маркетинговой программы в процессе импорта

Используйте это представление для добавления каждой импортируемой наводки в качестве цели Маркетинговой программы. Каждая наводка появится на Детальном представлении Маркетинговой программы на закладке Цели.

Чтобы добавить

  1. В процессе импорта наводок, создайте групповое действие и нажмите кнопку Определить рядом с полем Добавить цель программы.
  2. В поле Программа нажмите кнопку Найти для поиска маркетинговой программы.
  3. Нажмите Сохранить.
