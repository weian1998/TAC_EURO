Информационные поля Субъекта

В таблице ниже описаны основные поля.

Поле Описание
Субъект Компания, с которой у вас имеются налаженные или потенциально возможные бизнес-отношения.
Web Web адрес субъекта. Щелкните на поле и внесите изменения. Для того чтобы открыть Web-сайт, нажмите кнопку WWW.
Основной телефон Основной телефонный номер субъекта. Щелкните на поле и внесите изменения.
Адрес

Адрес или местоположение субъекта. Нажмите на кнопку Редактировать, чтобы открыть диалоговое окно Редактирования Адреса.

Для просмотра адреса на карте нажмите кнопку MapQuest.
Факс Номер факса субъекта. Щелкните на поле и внесите изменения.
Тип Категория субъекта. Нажмите стрелку и выберите элемент из списка.
Подтип Подкатегория субъекта. Нажмите стрелку и выберите элемент из списка.
Статус Текущее состояние Субъекта. Нажмите стрелку и выберите элемент из списка
Отрасль Отрасль деятельности Субъекта. Нажмите стрелку и выберите элемент из списка.
Описание бизнеса Добавление описания деятельности субъекта.
Менеджер Человек или команда внутри своей организации, которые управляют взаимоотношениями с Субъектом.Нажмите кнопку Найти для того, чтобы найти пользователя или команду внутри своей организации для управления взаимоотношениями с Субъектом.
Доступ Пользователь или команда пользователей, у которых есть права доступа к информации о субъекте. Нажмите кнопку Найти для того, чтобы найти пользователя или команду внутри своей организации, которые обладают правами на доступ к информации о субъекте.
Источник наводки Источник, из которого ваша компания узнала о субъекте (например, из рекламы в журнале или заказа продукции по телефону). Нажмите кнопку Найти рядом с полем, затем выберите Основной Источник, который вы хотите добавить к Субъекту.
