Использование закладки Активы Субъекта

Данная закладка используется для просмотра списка приобретенных продуктов, которые связаны с выигранными сделками, или добавлены для Субъекта.

Как выполнить?

Открыть закладку Активы Субъекта
  1. Откройте детальное представление Субъекта.
  2. Выберите закладку Активы. Если закладка не отображается, то откройте закладку Все закладки.

Редактировать Актив

Удалить Актив

Представление содержит следующие колонки:

  • Продукт - отображается название продукта.
  • Единица учета - отображается Единица учета приобретенного продукта.
  • Серийный номер - отображается уникальный серийный номер продукта.
  • Количество - количество продуктов, приобретенных Субъектом.
  • Контакт - Контакт Субъекта, ответственный за приобретение продуктов.
  • Местоположение - местонахождение приобретенных продуктов.
  • Продукт № - код продукта.
