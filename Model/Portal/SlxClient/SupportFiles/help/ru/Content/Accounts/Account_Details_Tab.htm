Закладка Детали Субъекта

Закладка Детали находится на Детальном представлении Субъекта. Эта закладка содержит дополнительную информацию о Субъекте. Вы можете редактировать информацию в любое время, щелкнув левой кнопкой мыши по необходимому полю и введя значение, также для некоторых полей доступна возможность выбора значения из справочника.

Ваша рабочая область может выглядеть по-разному, в зависимости от прав доступа или использования нестандартных настроек .

Для изменения

  1. На Детальном представлении Субъекта, который требуется изменить, выберите закладку Детали.
  2. Когда ваши изменения и обновления будут завершены, нажмите кнопку Сохранить.
Поле Описание
Регион Регион, в котором расположен субъект. Нажмите стрелку и выберите элемент из списка , а затем нажмите кнопку ОК.
Прочий Номер Дополнительный телефонный номер субъекта. Щелкните на поле и введите изменения.
Биржевой символ Биржевой символ ценных бумаг субъекта. Биржевой символ позволяет просматривать текущие котировки акций для субъекта. Щелкните на поле и введите изменения.
Кредитный рейтинг Оценка кредитоспособности субъекта. Щелкните на поле и введите изменения.
Код SIC Код отрасли субъекта. Щелкните на поле и введите изменения.
Сотрудники Количество сотрудников у субъекта. Щелкните на поле и введите изменения.
Годовой оборот Годовой доход субъекта. Щелкните на поле и введите изменения.
Источник наводки Источник, из которого ваша компания узнала о субъекте (например, из рекламы в журнале или заказа продукции по телефону). Нажмите кнопку Найти рядом с полем, затем выберите Основной Источник, который вы хотите добавить к Субъекту.
Валюта Валюта Субъекта (в Системе учета).
Часовой Пояс

Часовой пояс субъекта. Нажмите на указывающую вниз стрелку для выбора значения из списка.

Нажмите Показать Местное Время для отображения местного времени субъекта.
ИД Субъекта Отображает идентификационный номер субъекта, который создается автоматически при создании субъекта. Вы не можете изменить это поле.
Источник Импорта Отображает значение, если информация о субъекте была импортирована из другого источника.
Создано Пользователь, который создал в системе запись о субъекте.
Дата Создания Дата создания записи о субъекте.
Изменил Пользователь, который последним изменял запись о субъекте.
Дата Изменения Дата последнего изменения записи о субъекте.
Изменил историю Пользователь, создавший последнюю запись в истории субъекта.
История на День создания последней записи в истории.
Дней с последнего открытия Количество дней с момента завершения последнего дела по субъекту.
