Закладка Партнер. сделки

Вы можете использовать закладку Партнер. сделки на Детальном представлении Субъекта для просмотра информации о сделках для посредников. На закладке Партнер. сделки отображается список сделок, в которых текущий субъект является посредником.

Как выполнить?

Открыть закладку Партнер. Сделки

  1. Откройте Детальное представление Субъекта.
  2. Выберите закладку Партнер. сделки. Если закладка не отображается, то откройте закладку Все закладки.
Для того чтобы добавить
  1. Нажмите кнопку Создать новую Сделку.
    Откроется диалог Создать Сделку.
  2. Укажите необходимую информацию по сделке в полях.

  3. Нажмите Сохранить

На закладке Партнер. сделки отображается следующая информация:

  • Открыта - отображается количество открытых сделок и их суммарный потенциал.
  • Закрыта и Выиграна - отображается количество выигранных сделок и их суммарный потенциал.
  • Закрыта и Проиграна - отображает количество проигранных сделок и их суммарный потенциал.
  • Неактивна - отображает количество неактивных сделок и их суммарный потенциал.

В колонках таблицы на закладке Партнер. сделки отображается следующая информация:

  • Название Сделки - краткое описание (наименование) сделки. Для того чтобы открыть карточку сделки, щелкните по соответствующей гиперссылке.
  • Статус - статус работы по сделке.
  • Ожидаемое Закрытие - планируемая дата закрытия сделки.
  • Потенциал - общий доход, который может принести сделка.
  • Вероятность - вероятность того, что сделка будет выиграна.
  • Открыто дней - количество дней, прошедших с момента открытия сделки.
