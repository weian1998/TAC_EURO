Закладка Контакты Детального представления Субъекта

Вы можете использовать закладку Контакты для отображения списка контактов, связанных с текущим субъектом. Вы также можете добавлять контакты к субъекту, используя закладку Контакты.

Как выполнить?

Открыть закладку Контакты Субъекта

  1. Откройте детальное представление Субъекта.
  2. Выберите закладку Контакты. Если закладка не отображается, то откройте закладку Все закладки.
Для того чтобы добавить
  1. На детальном представлении Субъекта перейдите к закладке Контакты.
  2. Нажмите кнопку Добавить Новый Контакт.
    Откроется страница Добавить Контакт/Субъект.

Представление содержит следующие колонки:

  • Имя - Отображает имя контакта. Щелкните гиперссылку для отображения Детального представления Контакта.
  • Основной - Показывает, является ли данный контакт основным для текущего субъекта.
  • Телефон - Отображает телефонный номер контакта.
  • Должность - Отображает должность контакта.
  • Город - Отображает город контакта.
  • Область - Отображает область контакта.
  • Тип - Отображает тип контакта.
  • Департамент - Отображает департамент, к которому относится контакт.
