Закладка Сделки - Детальное представление Субъекта

Вы можете просмотреть информацию о сделках, связанных с выбранным субъектом. Вы также можете создавать новые сделки для субъекта.

Как выполнить?

Открыть закладку Сделки

  1. Откройте Детальное представление Субъекта.
  2. Выберите закладку Сделки. Если закладка не отображается, то откройте закладку Все закладки.
Для того чтобы добавить
  1. Нажмите кнопку Создать Сделку.
    Откроется диалоговое окно Создать Сделку.
  2. Укажите необходимую информацию по сделке в полях.

  3. Нажмите Сохранить

Закладка Сделки отображает следующую информацию:

  • Открытые - отображает количество сделок и суммарный потенциал всех открытых сделок для субъекта.
  • Закрыта и Выиграна - отображает количество сделок и суммарный потенциал всех закрытых выигранных сделок для субъекта.
  • Закрыта и Проиграна - отображает количество сделок и суммарный потенциал всех закрытых проигранных сделок для субъекта.
  • Неактивна - отображает количество сделок и суммарный потенциал всех неактивных сделок для субъекта.

Таблица на закладке Сделки имеет следующие колонки:

  • Название Сделки - Отображает название сделки.
  • Статус - Отображает статус сделки.
  • Потенциал - Отображает общий доход, который может быть получен от сделки.
  • Вероятность - Отображает вероятность того, что эта сделка будет выиграна.
  • Открыта дней - Отображает количество дней, в течении которых сделка была открыта.
