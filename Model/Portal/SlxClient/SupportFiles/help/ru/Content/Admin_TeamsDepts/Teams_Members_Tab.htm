Закладка Участники

Закладка Участники отображает пользователей, отделы и команды, входящие в состав команды. Используйте эту закладку для управления участниками команды.

Чтобы управлять

  1. На Детальном представлении Команды перейдите к закладке Участники.
  2. Отметьте флажок Автоматическое добавление менеджера пользователя в команду, если вы хотите добавить менеджера пользователя к команде при добавлении нового участника команды.

  3. Выполните одно из следующих действий:

    • Нажмите Добавить , чтобы добавить нового участника команды. Используйте Поиск чтобы найти и выбрать участника команды, которого вы хотите добавить.
    • Щелкните ссылку Удалить для участника команды, которого вы хотите удалить.
    • Нажмите ссылку Профиль безопасности чтобы изменить профиль безопасности пользователя в команде.
    • На панели Общие задачи, нажмите Заменить во всех командах, чтобы заменить выбранную команду другим пользователем, отделом или командой.
