Закладка Команды

Закладка Команды отображает команды, в которых департамент является участником. Эта закладка используется для установки или удаления связи с командой.

Чтобы управлять

  1. На Детальном представлении Департамента перейдите к закладке Команды.

  2. Выполните одно из следующих действий:

    • Нажмите Добавить, чтобы добавить отдел к команде. Для того чтобы выбрать необходимую команду используйте форму поиска.
    • Выберите запись, которую вы хотите удалить и нажмите кнопку Удалить.

  3. Нажмите ОК.
