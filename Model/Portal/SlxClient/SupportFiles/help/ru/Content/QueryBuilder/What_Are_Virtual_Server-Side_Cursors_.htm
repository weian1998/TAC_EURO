Что такое Виртуальные серверные курсоры (VSSC)

Виртуальные серверные курсоры (VSSC) разработаны для улучшения выполнения и использования памяти для больших групп в Списочном представлении в клиенте Sage SalesLogix.

Чтобы определить, использует ли группа VSSC

  • Выберите закладку группы из нужного списочного или детального представления, найдите одну из следующих иконок в верхнем правом углу представления, рядом с кнопкой Параметры группы.
Иконка Что она значит?
Группа в виртуальном серверном режиме - группа обрабатывается с помощью виртуальных серверных курсоров.
Группа в клиентском режиме - группа обрабатывается в клиенте. Не используются виртуальные серверные курсоры.

 

Для более подробной информации о виртуальных серверных курсорах, смотрите LAN Developers Reference.
