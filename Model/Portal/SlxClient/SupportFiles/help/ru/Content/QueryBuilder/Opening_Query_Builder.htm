Открытие построителя запросов

Вы можете использовать Построитель Запросов для добавления или изменения ваших групп.

Чтобы открыть Построитель запросов

  • Выполните одно из следующих действий:
  • В соответсвующем списке в меню Группы выберите Показать Группы.
  • Правой кнопкой мыши кликните по группе, которую вы хотите изменить, затем нажмите Редактировать.
