СтрокаФорматирования Полей Валюта и Телефон

Нижепреведенная таблица описывает параметры строк форматирования для полей валюты и телефонного номера.

Тип Поля Параметр Описание
Валюта m еньги. Формат поля должен быть значением с плавающей запятой. Значение преобразуется в строку, которая представляет количество валюты.
Телефонный Номер # Используйте знак # для обозначения, как будет отображаться номер. Вы можете использовать любой символ в качестве разделителя кроме букв и цифр. По умолчанию телефонный номер имеет формат (###) ###-####.

Строка форматирования должна начинаться со знака % и заканчиваться символом типа форматирования:

[%] [тип]
