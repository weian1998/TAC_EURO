Строки Форматирования Логического Поля

Строка форматирования для настройки Логического поля зависит от желаемого результата. Введите Значение Истина/Значение Ложь. Оно должно состоять из буквенно-цифровых символов и включать символ / между значениями истина и ложь. Истина всегда идет перед ложью. Форматом по умолчанию для Типа Формата ftBoolean является Да/Нет.

Например

Тип Формата: ftBoolean

Строка Форматирования: Покупка товара/Не заинтересован

Данные (Неформатированные): T

Результат: Покупка товара
