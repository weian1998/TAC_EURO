Строки Форматирования Целочисленного Поля

Приведенная ниже таблица описывает параметры строки форматирования для целочисленных полей. Допустимые форматы строк зависят от выбранного вами типа формата.

Строка форматирования должна начинаться со знака % и заканчиваться символом типа форматирования:

[%] [тип]
Параметр Описание
d Десятичное. Значение должно быть целым числом. Значение показывается как десятичное число.
x Шестнадцатиричное. Формат поля должен быть целым значением. Значение пребразуется в строку шестнадцатиричных цифр. Если строка форматирования содержит показатель точности, он означает что результирующая строка должна содержать минимум заданное количество цифр; если значение имеет меньшее количество цифр, то результирующая строка дополняется слева нулями.
