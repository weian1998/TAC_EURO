Закладка Свойства Построителя запросов

Используйте закладку Свойства Построителя запросов для заполнения имени и описания запроса.

Построитель Запросов поддерживает много функций. Видимые закладки зависят от функции, которую вы используете.

Чтобы установить свойства построителя запросов

  1. Откройте Построитель Запросов.
  2. Выберите закладку Свойства.
  3. В поле Имя введите название запроса.

Если вы создаете группу, имя, которое вы определите для запроса, появится в Менеджере Групп.

  1. В поле Отображаемое имя введите имя, которое должно отображаться.

Если вы создаете группу, то в качестве названия группы в списочном представлении используется Отображаемое имя. Если поле Отображаемое имя не заполнено, то в качестве названия будет отображаться текст, введенный в поле Имя.

  1. В поле Описание введите описание для определения назначения запроса.
