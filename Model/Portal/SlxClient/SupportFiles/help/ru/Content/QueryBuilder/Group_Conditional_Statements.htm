Группировка Условий

В расширенных запросах к базе данных может понадобиться использовать не только несколько условий, но также группировать вместе эти условия для возможности выполнения более сложного поиска.

Для группировки условий

  1. Откройте Построитель Запросов.
  2. Выберите закладку Условия и добавьте все необходимые условия.
  3. Для первого условия группы нажмите кнопку Добавить (+) в столбце с символом открывающей круглой скобки “(“.
  4. Для последнего условия группы нажмите кнопку Добавить (+) в столбце с символом закрывающей круглой скобки “).
  5. Повторите шаги до тех пор, пока не отметите все необходимые группы. Возможно исопльзование подуровней групп, но это может вызвать путаницу. Каждое последующее условие будет использоваться как подуровень предшествующего ему условия.

Для удаления групп из списка условий

  1. Для первого условия группы нажмите кнопку Удалить (-) в столбце с символом открывающей круглой скобки “(“.
  2. Для последнего условия группы нажмите кнопку Удалить (-) в столбце с символом закрывающей круглой скобки “)”.
  3. При необходимости, измените условные операторы.
