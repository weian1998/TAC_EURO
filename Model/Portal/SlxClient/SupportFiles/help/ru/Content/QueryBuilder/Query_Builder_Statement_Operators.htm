Операторы Условий Построителя Запросов

Когда вы добавляете условное утверждение, вам нужно выбрать оператор. Оператор используется для определения того, удолетворяет ли запись или нет заданным в условном утверждении критериям. Когда вы одбавляете условное утверждение, используемый по умолчанию оператор и другие доступные операторы зависят от типа поля.

В следующей таблице описаны все возможные операторы.

Оператор Выбирает запись, если соответствующее поле содержит значение, которое:
Начинается с Начинается со значения, заданного в поле Значение.
Содержит Содержит строку, заданную в поле Значение.
(=) равно Равно значению, заданному в поле Значение.
(<>) не равно Не равно значению, заданному в поле Значение.
(<) меньше чем Меньше чем значение, заданное в поле Значение.
(<=) меньше или равно Меньше или равно значению, заданному в поле Значение.
(>) больше чем Больше чем значение, заданное в поле Значение.
(>=) больше или равно Больше или равно значению, заданному в поле Значение.
В Содержит одну из строк, заданных в поле Значение, такую как встроенную константу ИЛИ утверждение. Вы должны заключать значения в кавычки для указания того, что следует искать.
не содержит данные Есть null (не имеет значения).
Содержит данные Не есть null (имеет любое значение).
в течение последних ххх дней Дата приходится на последние "xxx" дней. Введите число или два разделенных запятыми числа для обозначения промежутка времени.
в течение следующих ххх дней Дата приходится на последующие "xxx" дней. Введите число или два разделенных запятыми числа для обозначения промежутка времени.
