Что такое Построитель Запросов?

Построитель запросов позволяет вам выбирать данные из базы на основании одного или более условий. Например, вам необходимо посмотреть список основных контактов в банковской индустрии. Построитель Запросов связан с различными областями Sage SalesLogix. Для эффективного использования Построителя запросов, вы должны понимать структуру базы данных Sage SalesLogix и концепцию реляционных баз данных, например, отношение данных и объединения.
