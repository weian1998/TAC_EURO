Закладка Макет Построителя запросов

Используйте закладку Макет Построителя запросов, чтобы определить какие поля и данные будут отображаться. Вы можете добавить, редактировать, удалить или изменять порядок полей на закладке. Вы можете также установить название колонки, ширину или формат каждого поля. На закладке Макет должно быть по крайней мере одно поле или данные не появятся.

Построитель Запросов поддерживает много функций. Видимые закладки зависят от функции, которую вы используете.
Для добавления поля в макет
  1. Откройте Построитель Запросов.
  2. Выберите закладку Макет.

    Макет по умолчанию определяется группой, которую вы выбрали Группой по умолчанию в настройках Групп.

  3. Выберите таблицу.
  4. Выберите поле.

Вы можете пролистать список макета для просмотра добавленных в макет полей.

  1. Выберите поле и нажмите Редактировать.
  2. Заполните поля диалога Назначить шаблон запроса.

Для редактирования поля в макете

  1. Откройте Построитель Запросов.
  2. Выберите закладку Макет.
  3. Выберите поле в таблице.
  4. Нажмите Редактировать.
  5. Внесите изменения в диалоговом окне Назначить шаблон запроса.
Для удаления поля из макета
  1. Откройте Построитель Запросов.
  2. Выберите закладку Макет.
  3. Выберите поле в таблице.
  4. Нажмите Удалить.
Для изменения порядка полей в макете
  1. Откройте Построитель Запросов.
  2. Выберите закладку Макет.
  3. Выберите поле. Выполните одно из следующих действий:
  • Перетащите поле на новое место.
  • Нажимайте Влево или Вправо до тех пор, пока поле не передвинется на нужное место.
