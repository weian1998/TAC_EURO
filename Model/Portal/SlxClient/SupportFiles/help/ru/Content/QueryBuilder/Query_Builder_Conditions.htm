Условия Построителя запросов

Перейдите на закладку Условия Построителя запросов для добавления, изменения или удаления условных операторов, которые накладываются на выборку данных. Условия применяются в том порядке, в котором они представлены в таблице; хотя вы можете изменить порядок, используя кнопки Вверх и Вниз. Вы можете также группировать условные операторы.
  • Построитель Запросов поддерживает много функций. Видимые закладки зависят от функции, которую вы используете.
  • Все группы, которые вы создаете, должны содержать хотя бы одно условие. Группы, которые включают все записи, могут замедлить производительность системы, также может возникнуть ошибка при попытке просмотра пользователем этой группы.
Чтобы добавить условный оператор
  1. Откройте Построитель Запросов.
  2. В Построителе Запросов перейдите на закладку Условия.
  3. Выберите таблицу в левой части Построителя Запросов. При необходимости, нажмите знак плюс для раскрытия списка таблиц.
  4. Выберите поле в верхней правой части Построителя Запросов. Здесь отображены все поля, относящиеся к выделенной таблице. Дважды щелкните на поле для добавления его в таблицу внизу диалога.

Откроется диалог Назначить Условие. Вы можете открыть это окно в любое время при двойном щелчке по оператору в таблице.

  1. Выберите Оператор.
  2. В поле Значение введите значение или нажмите Просмотр для выбора значения из всех существующих записей для этого поля.

Если вы выбрали в качестве оператора "В" значения должны быть заключены в кавычки для каждой записи.

  1. Выберите Сравнение с учетом регистра , если вы хотите, чтобы условие зависило от регистра букв. По умолчанию эта опия выбрана.

Если вы используете MS SQL Server, Сравнение С Учетом Регистра невозможно. Все запросы нечувствительны к регистру.

  1. При необходимости, выберите Использовать значение в качестве константы.
  2. Нажмите ОК.
  3. Чтобы добавить другое условие, повторите шаги с 1-9, затем выберите соотвествующее объединение.
Чтобы изменить условный оператор
  1. Откройте Построитель Запросов.
  2. Выберите закладку Условия.
  3. Выберите оператор из таблицы (в нижней части окна Построителя запросов).
  4. Выберите пункт Изменить.
  5. В окне Назначить условие сделайте изменения.
  6. Нажмите ОК.
Чтобы удалить условный оператор
  1. Откройте Построитель Запросов.
  2. Выберите закладку Условия.
  3. Выберите условие в таблице.
  4. Нажмите Удалить.
