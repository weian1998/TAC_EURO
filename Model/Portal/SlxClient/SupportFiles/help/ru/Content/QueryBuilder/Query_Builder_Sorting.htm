Закладка Сортировка Построителя запросов

Используйте заклаку Сортировка Построителя запросов для указания, по каким полям сортировать данные и данные отображаются в порядке возрастания или убывания. Критерии сортировки применяются в том порядке, в котором они отображаются на вкладке.

  • Построитель Запросов поддерживает много функций. Видимые закладки зависят от функции, которую вы используете.

  • Некоторые отчеты включают предопределенные критерии сортировки, которые могут быть удалены при необходимости..
  • Blob поля не могут быть использованы для сортировки данных.

(Binary Large Object) Тип данных, содержащий большой объем данных. Blobs могут иметь двоичные или текстовые данные. Используйте Blob Viewer в диалоге Результаты SQL, чтобы увидеть текстовые данные blob.

  • Числовые вычисляемые поля не могут быть использованы для сортировки данных, потому что математические вычисления завершаться только после сортировки.
Добавить критерии сортировки
  1. Откройте Построитель Запросов.
  2. Выберите закладку Сортировка.
  3. Выберите таблицу в левой части Построителя Запросов. При необходимости, нажмите знак плюс для раскрытия списка таблиц.
  4. Выберите поле в правой части Построителя Запросов. Здесь отображены все поля, относящиеся к выделенной таблице. Дважды щелкните на поле для добавления его в таблицу внизу диалога.
  5. Выберите Направление из выпадающего списка, По возрастанию (сортирует от А до Я) или По убыванию (сортирует от Я до А).
  6. Чтобы добавить второе поле для сортировки, повторите шаги с 3-5. Каждая последующая сортировка будет приминяться только после предыдущей.

Числовые вычисляемые поля не могут быть отсортированы.

Чтобы изменить порядок сортировки
  1. Откройте Построитель Запросов.
  2. Выберите закладку Сортировка.
  3. Выберите поле в таблице.
  4. Нажмите Переместить Вверх или Переместить Вниз.
Чтобы удалить критерий сортировки
  1. Откройте Построитель Запросов.
  2. Выберите закладку Сортировка.
  3. Выберите поле, которое хотите удалить из таблицы.
  4. Нажмите Удалить.
