Понимание факторов, влияющих на синхронизацию

Время, необходимое для синхронизации данных между Sage SalesLogix и Microsoft Outlook или Lotus Notes зависит от нескольких факторов. Наиболее важным фактором является объем данных, передаваемый между системами. Вы можете улучшить производительность, контролируя объем данных, передаваемых через Intellisync для SalesLogix. Есть несколько способов сделать это.

Дата последней синхронизации

Когда синхронизация завершается, дата записывается в лог-файл. Используйте эту дату, чтобы проверить наличие изменений с момента последней синхронизации и передавайте только те данные, которые были изменены. Если проводить синхронизации регулярно, можно уменьшить объем данных, которые должны быть переданы в течение определенного цикла.  

Безопасность

Безопасность во время синхронизации определяется настройками безопасности текущего пользователя Sage SalesLogix. Во время синхронизации передаются только данные, к которым имеет доступ текущий пользователь.

Удаление сопоставления полей

Intellisync для SalesLogix позволяет пользователям определять, как данные должны передаваться между приложениями. Если вы удалите сопоставление поля, данные больше не будут в него записываться. Если вы снова сопоставите поле, сопоставление с которым ранее удалили, это запустит синхронизацию этих полей между приложениями. Смотрите раздел Понимание сопоставления полей в Intellisync для SalesLogix для более подробной информации.
