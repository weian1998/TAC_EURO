Настройка Intellisync для SalesLogix

Настройка должна быть выполнена на рабочем месте каждого пользователя. Для синхронизации, специальная группа должна существовать в Sage SalesLogix, и должна быть настроена в качестве группы синхронизации. Для настройки Intellisync вам нужна ссылка на портал Intellisync для SalesLogix.  Для получения подробной информации обратитесь к своему администратору.

Можно настроить различные параметры для различных типов записей. Например, можно настроить одностороннюю синхронизацию для контактов и двухстороннюю для календаря, и установить синхронизацию в различное время. Следующие шаги рекомендуется выполнить для настройки синхронизации между Microsoft Outlook или Lotus Notes и Intellisync для SalesLogix.

Кнопка помощи в интерфейсе Intellisync вызывает справку, созданную разработчиком Intellisync - компанией Nokia. Некоторые термины в этой справке могут отличаться от терминов, используемых в SalesLogix. Смотрите раздел дополнительная помощь при установке приложения для дополнительной информации.

Чтобы настроить

  1. Запустите Intellisync для SalesLogix.
  • В меню Пуск выберите Программы -> Sage SalesLogix -> Intellisync для SalesLogix и нажмите на пункт Intellisync для SalesLogix.
  1. Нажмите Установить и сделайте следующее.
  • В поле ИД пользователя введите имя пользователя Sage SalesLogix.
  • В поле Пароль введите пароль.
  • В поле URL для подключения укажите путь к порталу Intellisync для SalesLogix.

Обратитесь к вашему администратору для получения пути к порталу.

  1. Нажмите кнопку Настройка приложения
  2. На экране настройка, можно выбрать или снять флаги Активен для каждого типа данных. активен - означает, что данные синхронизируются при каждой синхронизации.

    Чтобы ограничить количество синхронизуемых контактов, Sage SalesLogix, Sage SalesLogix рекомендует создать фильтр SyncSalesLogix.

  3. Нажмите кнопку Мастер установки.
  4. В окне Приложения выберите Microsoft Outlook или Lotus Notes, и нажмите Далее.
  5. В окне Что синхронизировать выберите Все приложения Intellisync для SalesLogix, поддерживаемые Microsoft Outlook или Lotus Notes и нажмите Далее.
  6. В окне Как синхронизировать выберите Двусторонняя синхронизация и нажмите Далее.
  7. В окне Когда синхронизировать, укажите, когда должна запускаться синхронизация и нажмите Далее.
  • Каждые 10 минут - выберите из выпадающего списка.
    Пользователь начинает синхронизацию вручную, запускается PushSync и данные синхронизируются с выбранной периодичностью.
  • Каждый день в определенное время - выберите из выпадающего списка.
    Пользователь начинает синхронизацию вручную, запускается PushSync и данные синхронизируются с выбранной периодичностью.
  • При включении компьютера - Когда пользователь заходит в систему, запускается Intellisync и синхронизация.
  • При обнаружении изменения в Lotus Notes/Outlook - запускается Intellisync и начинается синхронизация
  1. На экране Информация о соединении Intellisync для SalesLogix , выберите профиль пользователя, нажмите Далее.
  1. Закончите заполнение оставшихся экранов для нужного типа синхронизируемой информации для Outlook или Lotus Notes.
  1. На экране Параметры дел как для Outlook, так и для Lotus Notes, выберите Перенести все пункты и нажмите Далее.

Эта опция позволит синхронизировать все активные, завершенные и незавершенные дела.

  1. На экране Параметры назначений выберите Переносить только будущие элементы, нажмите Далее.
  2. Если вы настраиваете Intellisync для Lotus Notes, на экране Параметры для Контактов, выберите базу данных Lotus Notes и нажмите Далее.
  1. В окне Информация для подключения Intellisync для SalesLogix проверьте ваш логин, пароль и путь к порталу Intellisync для SalesLogix, нажмите Далее.
  2. Нажмите Завершить.
  3. Нажмите Закрыть.
  4. Нажмите ОК.
  5. Если вы еще не выбрали, выберите группу SyncSalesLogix.
