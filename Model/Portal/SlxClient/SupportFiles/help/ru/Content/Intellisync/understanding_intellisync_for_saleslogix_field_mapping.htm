Понимание сопоставления полей в Intellisync для SalesLogix

Intellisync для SalesLogix работает, перемещая данные между коннекторами, не подключая непосредственно одну базу данных к другой. Например, при экспорте данных из Sage SalesLogix в Microsoft Outlook или Lotus Notes, Intellisync использует как коннектор Sage SalesLogix, так и коннектор Outlook или Lotus Notes . Коннектор Sage SalesLogix считывает данные специальной группы, указанной вами при настройке; Коннектор Outlook или Lotus Notes считывает данные из категории SyncSalesLogix в Outlook или Lotus Notes.

При ручной синхронизации, Intellisync сравнивает данные в обоих коннекторах и предоставляет вам возможность просмотреть изменения, принять или отклонить их. Во время автоматической синхронизации, процесс таков же, но нет экранов подтверждения. Intellisync полагается на настройку параметров разрешения конфликтов.

В процессе синхронизации происходит обмен данными только между сопоставленными полями; поля без сопоставления - игнорируются. Стандартный набор сопоставленных полей настроен в Intellisync для SalesLogix по умолчанию. Таблицы, описывающие сопоставленные поля по умолчанию находятся в справке администратора Sage SalesLogix. Вы можете просмотреть и переопределить текущие настройки сопоставления полей, сопоставить дополнительные поля или удалить сопоставление полей. Сопоставление полей зависи от версии используемого коннектора и типа действия.

Создание нового сопоставления полей позволяет синхронизировать данные между приложениями. Удаление сопоставления прекратит синхронизацию данных. Чтобы получить информацию об изменении сопоставления полей, смотрите справку, доступную из мастера настройки приложения Intellisync для SalesLogix.

  • Важно помнить, что изменение сопоставления полей может привести к неожиданным результатам. После изменения сопоставления полей, но перед синхронизацией, сделайте бекап ваших данных. Кроме того, целесообразно запустить синхронизацию на тестовых данных, чтобы проверить как ваши изменения повлияют на критически важные данные.
  • Если вы изменили сопоставление полей для данных, которые уже были синхронизированы, то следующая синхронизация по сути будет первой для этих данных. Поэтому, вы должны запустить синхронизацию Intellisync для SalesLogix до внесения изменений в сопоставление полей.
