Разъединенные пользователи Веб Клиента Sage SalesLogix

Только для удаленных пользователей

Разъединенные пользователи Веб Клиента работает также как и удаленный клиент Sage SalesLogix, за исключением того, что пользователи подключаются к своим локальным базам данных используя Web клиент. Разъединенные пользователи Веб Клиента позволяет вам работать отсоединенно без использования подключения к интернету.

Чтобы перенести изменения между пользовательской базой данных отсоединенного Web клиента и основной базой данных, используется синхронизация. Чтобы синхронизировать изменения между вашей базой данных и основной базой данных, нужно использовать клиента синхронизации. Для получения дополнительной информации смотрите онлайн справку клиента синхронизации.

Для запуска Web отчетов в отсоединенном режиме, the Разъединенные пользователи Веб Клиента должен быть запущен на платформе Window 2003 Server с установленным IIS. SalesLogix Web Reporting Server и Crystal Reports Server также должны быть установленны и настроены. Если эти требования не выполнены, то отсоединенные Web клиенты смогут запускать Web Reporting только в том случае, когда они подключены к основному серверу Web Reporting. Такие пользователи будут видеть данные в отчетах из основной базы данных, а не из их удаленной базы данных.

Каковы особенности отсоединенного Web клиента?

  • Представление Что нового? позволяет пользователям отсоединенного Web клиента подписаться и отписаться от субъектов.
  • (Только для Интеграции учета) Синхронизация между Sage SalesLogix и поддерживаемой системой учета осуществляется на основной базе данных. Таким образом, любые изменение на отсоединенном Web клиенте или в системе учета должны быть синхронизованны через основную базу данных.
