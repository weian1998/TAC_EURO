Синхронизация с базой данных головного офиса

Чтобы поддерживать свою удаленную базу данных в актуальном состоянии в сравнении с базой данных головного офиса, очень важно регулярно выполнять синхронизацию. Если вы не выполняете синхронизацию каждый день или через день, базы данных с общими записями (головной офис и удаленные клиенты) скоро устареют. Частое выполнение синхронизации, также снижает время синхронизации, т.к. обрабатывается меньшее число транзакций.

Автоматическая синхронизация позволяет вам выполнять синхронизацию автоматически при первом подключении к головному офису и в дальнейшем, периодически, пока сохраняется подключение. Автоматическая синхронизация влючена по умолчанию. Чтобы узнать больше, смотрите раздел “Клиент синхронизации Sage SalesLogix” в справке клиента Sage SalesLogix.

Чтобы выполнить синхронизацию с базой данных головного офиса

  1. В меню Файл, нажмите Синхронизировать.
  2. На закладке Автоматическая синхронизация, выберите Включена.
  3. В полях Синхронизация выполняется при первом подключении и каждые, установите время, в которое вы хотите, чтобы выполнялась синхронизация.
  4. Откройте закладку Настройки синхронизации.
  5. Для синхронизации через модемное соединение, установите флаг Подключиться используя RAS, затем выберите профиль из списка RAS профилей. Снимите флаг Подключиться используя RAS, если вы используете сетевое подключение.
  6. Нажмите Синхронизовать сейчас чтобы начать процесс синхронизации.
  7. По завершению синхронизации, нажмите Закрыть.

Если в процессе синхронизации возникли ошибки, свяжитесь с вашим системным администратором для их исправления.
