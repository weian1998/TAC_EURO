Закладка Учет

(Только для Интеграции учета) Закладка Учет на детальном представлении субъекта отображает информацию об учетных системах, связанных с Sage SalesLogix. Можно использовать эту закладку для управления связями между системами. Представление содержит следующие колонки:

  • Система учета - система учета, с которой связан субъект.
  • Статус - статус связи.
  • Доступ - имя системы учета, которая имеет доступ к информации о субъекте.
  • Создано - пользователь, создавший связь.
  • Дата создания - дата создания связи субъекта.

Как выполнить?

Открыть закладку Учет

  1. Открыть Детальное представление Субъекта.
  2. Выберите закладку Учет. Если закладка не отображается, то откройте закладку Все закладки

Что такое Статус связи

Колонка Статус отображает статус связи между субъектом Sage SalesLogix и учетной системой.

  • Связано - субъект связан с коммерческой организацией в учетной системе.
  • Доступ - учетная система имеет доступ к информации о коммерческой организации наряду с другой связанной системой учета. Например, система учета может быть дочерней системой другой связанной системы учета, и обе имеют доступ к одному субъекту. Информация о доступе отображается в колонке Доступ.

Связать Субъект с Системой учета

Только системы учета, настроенные на закладке Системы учета, появятся в списке. Если статус системы учета - Доступ, систему не возможно связать. Можно связать только родительский субъект для общих систем.

  1. Откройте карточку необходимого Субъекта и откройте закладку Учет.
  2. Нажмите Выдвинуть субъект на панели задач Учета.
  3. Заполните информацию на экранах Мастера связи.

Когда вы удаляете связь между записями, никакие записи в Sage SalesLogix или в учетной системе не удаляются. Только связь между записью Sage SalesLogix и записью в учетной системе удаляется. Любые связанные под-сущности (контакты, адреса, телефон, e-mail), которые синхронизировались из-за наличия связи, не будут больше синхронизироваться.

  1. Откройте карточку необходимого Субъекта и откройте закладку Учет.
  2. В таблице кликните Удалить связь для учетной системы, из которой вы хотите удалить связь.
  3. Нажмите ОК для подтверждения.
