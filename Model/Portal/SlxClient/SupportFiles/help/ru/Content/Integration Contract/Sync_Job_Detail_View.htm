Детальное представление Задачи синхронизации

Используйте детальное представление для просмотра, добавления или редактирования информации о выбранной записи. Детальное представление состоит из информационных полей, закладок и определяемой пользователем, средней панели.

Ваша рабочая область может выглядеть по-разному, в зависимости от прав доступа или использования нестандартных настроек .

Как выполнить?

Открыть Детальное представление Задачи Синхронизации
  • На Навигационной Панели нажмите Администрирование, затем Задачи синхронизации, чтобы открыть Список Задач синхронизации. Для того чтобы перейти к задаче, можно использовать Поиск, после чего выберите имя задачи.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Просмотреть группы

Информационные поля

Информационные поля содержат информацию о задаче синхронизации интеграции учета.

  • Название задачи - Тип задачи (ручная или по расписанию) и имя задачи.
  • Конечная точка - приложение, с которым вы синхронизируете информацию.
  • Время начала - дата и время начала задачи синхронизации.
  • Время завершения - дата и время окончания задачи синхронизации.

Закладки

Каждая закладка содержит определенную информацию, которую вы можете использовать для связанных с задачей дел. Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

Лог Элементы

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
