Настройка Интеграции Учета

Выполните следующие задачи для настройки Интеграции учета.

  • Если работает поддержка мультивалютности, убедитесь, что Sage SalesLogix и ваша учетная система имеют одинаковую основную валюту.

  • Запустите Утилиту обновления телефона и e-mail.

  • Проверка подлинности Windows должна быть настроена для пользователя с правами администратора перед запуском Утилиты обновления телефонов и E-mail. Дополнительные сведения см. в разделе «Использование проверки подлинности Windows» в разделе справки Администратора.
    • Интеграция Учета возможна, когда возможны и настроены точки интеграции на стороне Sage SalesLogix и на стороне учетной системы. Используйте Детальное представление Интеграции учета, чтобы установить конечные точки.

    Когда вы включаете или отключаете Интеграцию учета, изменения вступят в силу только после повторного входа в систему.
    • Sage рекомендует сравнивать схемы базы данных в Sage SalesLogix со схемами учетного приложения, чтобы убедиться в целостности данных. Используйте закладку Системы учета, чтобы определить различия схемы.

    Если у вас кастомизированная система Sage SalesLogix, нужно просмотреть Sage CRM ERP Contract и убедиться, что информация будет синхронизироваться между системами. Например, если вы создавали пользовательские сущности, которые не соответствуют Sage SalesLogix, синхронизация может не работать.
    • Используйте закладку Совпадения, чтобы установить глобальные стандартные критерии совпадений для пользователей, создающих связи.
    • Настройте Службу синхронизации Интеграции учета.
    • При необходимости, экпортируйте и объединяйте записи из учетной системы.

    Используйте инструмент экспорта, предложенный вашей учетной системой, для экспорта нужных данных в Sage SalesLogix.

    • Настройте роли пользователя.
