Списочное представление Задач синхронизации

Список Задач Синхронизации отображает информацию о задачах синхронизации между Sage SalesLogix и учетным приложением.

  • Чтобы сортировать таблицу по конкретному полю, нажмите на один из заголовков столбца.
  • Чтобы отобразить задачи, отвечающие специфическим критериям, настройте фильтры.

Как выполнить?

Открыть Списочное представление Задач Синхронизации
  • На Панели Навигации нажмите Администрирование, а затем нажмите Задачи Синхронизации.
Открыть Детальное представление Задачи Синхронизации
  • Для того чтобы открыть карточку Задачи синхронизации, щелкните мышкой по ее названию.
