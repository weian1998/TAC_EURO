Установить Интеграцию учета

Используйте диалог Установить интеграцию учета, чтобы определить детали учетной информации.

Поле Описание/Действие
Снабжение учета

Введите SData URL системы учета. Используйте формат: http://server:port/sdatavirtualdir/ application/contract/operatingcompany /

  • Вместо application введите имя учетного приложения. Например, Sage1000.
  • Вместо contract введите GCRM.
  • Вместо operatingcompany введите компанию в вашем учетном приложении, с которой вы интегрируетесь.
Имя Пользователя Введите имя пользователя, используемое для входа в учетную систему.
Пароль Введите пароль, соответствующий имени пользователя.
Проверка снабжения Нажмите, чтобы проверить и отобразить статус соединения.
Статус Отображает результаты проверки снабжения учета.
Название Введите отображаемое имя для системы учета. Это имя будет видно в Веб клиенте, когда пользователи связывают записи между Sage SalesLogix и учетной системой.
Разрешить синхронизацию, линковку и транзакции Проставьте или снимите флажок. Когда параметр отключен, система учета не доступна для связывания или синхронизации с Sage SalesLogix.
Фильтр по ресурсам

Выберите тип ресурса и нажмите кнопку Выполнить, чтобы отобразить различия схемы в таблице.

Эта функция сравнивает схемы выбранных ресурсов в двух интегрированных системах и возвращает любые различия в длине строк. Если значения строк не совпадают, используйте Database Manager или другое приложение, чтобы изменить схемы в одной из систем.

Если это не первая система учета для связи, необходимо перезапустить IIS на веб-сервере, для того чтобы новые каналы были включены во время синхронизации.
