Закладка Системы учета

Используйте закладку Системы учета для управления связями с вашими учетными системами.

Как выполнить?

Добавить или Изменить Интеграцию учета

  1. Открыть Детальное представление Интеграции учета.
  2. Выберите закладку Системы учета.

  3. Выполните одно из следующих действий:

  • Чтобы добавить снабжение, нажмите Установить интеграцию учета.
  • Для изменения существующего пункта, нажмите Изменить.
  1. Заполните поля в диалоге Установить интеграцию учета.
  2. Нажмите ОК.

Если это не первая система учета для связи, необходимо перезапустить IIS на веб-сервере, для того чтобы новые каналы были включены во время синхронизации.

Синхронизировать Все системы учета

  1. Открыть Детальное представление Интеграции учета.
  2. Выберите закладку Системы учета.
  3. Нажмите кнопку Синхронизировать все учетные системы.
