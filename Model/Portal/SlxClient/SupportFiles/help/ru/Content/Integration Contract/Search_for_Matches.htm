Поиск Совпадений

Экран Поиск совпадений позволяет вам выбрать критерии, которые будут использоваться при поиске в системе учета клиентов, которые являются потенциальными совпадениями с субъектом, который вы связываете из Sage SalesLogix.

Чтобы выбрать

  1. Откройте Мастер связи и пройдите предыдущие экраны.
  1. Заполните экран Поиск совпадений, нажмите Следующий, чтобы перейти к следующему шагу.
Поле Описание/Действие
Свойство Выберите поле для поиска совпадающих данных. Доступные поля - поля, связанные между Sage SalesLogix и системой учета.
Оператор Выберите критерии совпадения, используемые системой для выбранного свойства. Доступные операторы - Равно, Начинается с, Содержит.
Искать значение Введите значение, по которому хотите искать.
Добавить или удалить критерии поикам Чтобы добавить дополнительные критерии соответствия, нажмите . Чтобы удалить критерии соответствия, нажмите .
