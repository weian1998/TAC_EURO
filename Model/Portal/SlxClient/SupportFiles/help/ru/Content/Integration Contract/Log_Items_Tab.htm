Закладка Лог Элементы

Закладка Лог Элементы отображает сообщения, сформированные при работе задачи синхронизации. Используйте фильтры на этой закладке для выделения информации логирования, которую вы хотите просмотреть.

Для того чтобы просмотреть

  1. Откройте Детальное представление задачи синхронизации для задачи, которая содержит необходимые элементы.
  2. В списке Фильтр по типу Лог выберите тип сообщения об ошибке, который вы хотите просмотреть.
  • Отладка - отображает детализированные ошибки, для помощи в устранении проблем.
  • Ошибки - отображает информацию, если случилась серьезная проблема, такая как потеря данных или потеря функциональности.
  • Информационные - отображает действия, предпринимаемые службой.
  • Предупреждения - отображает событие, не обязательно значительное, но отражающее возможную проблему в будущем.
  1. В списке Фильтр по типу ресурса, выберите сущность для выбранного типа лога.
  2. Нажмите кнопку Обновить Лог элементы.
  3. Нажмите ссылку Выбрать для лога в таблице. Детальная информация отображается в поле Стек вызовов.
