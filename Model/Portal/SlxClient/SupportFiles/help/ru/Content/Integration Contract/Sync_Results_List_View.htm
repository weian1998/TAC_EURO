Списочное представление Результатов синхронизации

Список Результатов Синхронизации отображает информацию о циклах синхронизации между Sage SalesLogix и учетным приложением. Отображаются только данные, связанные с применением записей в Sage SalesLogix или в конечной точке учетной системы.

  • Чтобы сортировать таблицу по конкретному полю, нажмите на один из заголовков столбца.
  • Чтобы отобразить задачи, отвечающие специфическим критериям, настройте фильтры.

Как выполнить?

Открыть Списочное представление Результатов Синхронизации
  • На Панели Навигации нажмите Администрирование, а затем нажмите Результаты Синхронизации.
Открыть Детальное представление Результата Синхронизации
  • Для того чтобы открыть карточку Результата синхронизации, щелкните мышкой по времени начала.
