Запуск Утилиты Обновления Записей E-mail и Телефона Субъекта.

До установки конечных точек вашей Учетной Интеграции, вы должны запустить утилиту обновления записей E-mail и Телефона Субъекта. Данная утилита создает запись в таблице ERPPHONENUMBER или ERPEMAILADDRESS для каждого телефонного номера и столбца email в таблицах СУБЪЕКТ и КОНТАКТ. Данные записи соответствуют оригинальному значению столбца сущности, в которой они инициировались.

Проверка Подлинности Windows должна настраиваться для пользователя с правами администратора до запуска утилиты обновления E-mail и Телефона Субъекта. Для получения более подробных действий смотрите тему "Использование Проверки Подлинности Windows" в Справке Администратора.

Для запуска утилиты обновления записей E-mail и телефона субъекта.

  1. На вашей рабочей станции администратора, найдите файлы ICLegacyPhoneEmailUpdater.exe и ICLegacyPhoneEmailUpdater.config.
    Конфигурационные файлы по умолчанию находятся в ...\Program Files\SalesLogix
  2. Дважды щелкните на ICLegacyPhoneEmailUpdater.config , чтобы открыть файл в текстовом редакторе по вашему выбору.
  3. Прокрутите до раздела <connectionStrings> и измените значение Initial Catalog с SALESLOGIX_EVAL на псевдоним вашей базы данных.
    Например:
    <connectionStrings> <clear/> <add name="Default" connectionString="Provider=SLXOLEDB.1;Persist Security Info=True;Initial Catalog=MyDatabaseAlias;Data Source=localhost;Extended Properties=&quot;PORT=1706;LOG=ON;CASEINSENSITIVEFIND=ON;AUTOINCBATCHSIZE=1;SVRCERT=12345;ACTIVITYSECURITY=OFF;TIMEZONE=NONE&quot;"/> </connectionStrings>
  4. Дважды щелкните на ICLegacyPhoneEmailUpdater.exe для запуска утилиты.
  5. Щелкните Создать Телефонные Записи ERP для создания телефонных записей.
  6. Щелкните Создать E-mail Записи ERP для создания записей e-mail.
  7. Нажмите кнопку Закрыть.
