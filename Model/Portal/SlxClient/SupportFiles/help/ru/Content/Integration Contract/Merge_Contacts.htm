Объединение контактов

Экран Объединение контактов отображает контакты для субъекта Sage SalesLogix, который вы связываете с клиентом, выбранным в учетной системе. Используйте этот экран для создания связей и объединения записей контакта Sage SalesLogix и учетной системы.

Чтобы выбрать

  1. Откройте Мастер связи и пройдите предыдущие экраны.
  1. Заполните экран Объединить контакты, нажмите Следующий, чтобы перейти к следующему шагу.
Поле Описание/Действие
Несвязанные контакты из Sage SalesLogix Отображает контакты в Sage SalesLogix, которые доступны для связывания с контактом учетной системы. Нажмите Выбрать для выбора контакта.
Несвязанные контакты из [конечная точка вашей учетной системы] Отображает контакты в системе учета, которые доступны для связывания с контактом Sage SalesLogix. Нажмите Выбрать для выбора контакта.
Кнопка Связать

Выберите несвязанный контакт в таблицах Несвязанных контактах из ERP и Несвязанных контактах из SLX, затем нажмите Связать для создания связи.

Чтобы выбрать Контакт, нажмите ссылку Выбрать.
Связанные контакты

Отображает совпадения, которые были идентифицированы в качестве потенциальных связей, используя определенные критерии совпадения.

  • Нажмите Удалить Связь, чтобы удалить предложенную связь и создать новую связь.

  • Нажмите Выбрать для отображения полей контакта в источнике и целевой системе.
