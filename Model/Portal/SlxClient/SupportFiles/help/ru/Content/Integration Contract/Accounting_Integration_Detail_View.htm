Детальное представление Интеграции Учета

Используйте детальное представление Интеграции Учета для настройки связей между Sage SalesLogix и вашими учетными системами.

Интеграция Учета возможна, когда возможны и настроены точки интеграции на стороне Sage SalesLogix и на стороне учетной системы. Если ничего не указано на закладке Системы учета, тогда интеграция не возможна.

Когда вы включаете или отключаете Интеграцию учета, изменения вступят в силу только после повторного входа в систему.

Установите свойства подаваемой информации для Sage SalesLogix, затем настройте свойства информации для каждой интегрируемой учетной системы.

Чтобы установить

  1. В меню Администрирование выберите Установка интеграции
  2. В поле SalesLogix Снабжение введите SData URL для учетной системы.
    Используйте формат: http://server:port/sdatavirtualdir/slx/gcrm/-/ 
    • Замените sdatavirtualdir на каталог, в который развернут ваш портал SData.
      По умолчанию каталог "SData".
      Например, http://localhost:3333/sdata/slx/gcrm/-/
  3. В поле Имя пользователя, введите имя пользователя для базы данных Sage SalesLogix, к которой подсоединяется учетная система.
  4. В поле Пароль введите пароль для имени пользователя.
  5. В поле Название введите отображаемое название для снабжения.
  6. Нажмите Сохранить.
  7. При необходимости, выберите Ограничить выдвижение Субъекта в единственную систему учета, чтобы ограничить интеграцию одной учетной системой.
  8. При необходимости, установите флажок Разрешить системе учета управлять Заказами для удаления возможности создавать и редактировать заказы в Sage SalesLogix. Когда вы выставите этот флажок, ваша система учета должна быть использована для управления заказами.

    9. Эта опция необходима для некоторых интеграции, например, с X3.
  9. Используйте закладку Системы учета для настройки деталей снабжения для вашей учетной системы.
  10. Используйте закладку Синхронизация для настройки интервалов синхронизации.
  11. Используйте закладку Совпадения, чтобы установить глобальные стандартные критерии совпадений для пользователей, создающих связи.
