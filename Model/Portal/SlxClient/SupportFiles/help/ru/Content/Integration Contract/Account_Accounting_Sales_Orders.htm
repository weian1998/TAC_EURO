Закладка Заказы учета

(Только для Интеграции учета) Закладка Заказы учета на детальном представлении Субъекта отображает заказы субъекта из учетной системы. Можно использовать эту закладку для просмотра такой информации, как дата заказа, статус, платежи и так далее. Эта закладка только для чтения.

Для того чтобы просмотреть

  1. Откройте Детальное представление Субъекта для связанного субъекта.
  2. Выберите закладку Заказы учета.
  3. Чтобы просмотреть дополнительные детали счета, нажмите номер заказа в колонке Номер заказа. Детальное представление заказа отображает дополнительную информацию (только чтение).

Представление содержит следующие колонки:

Колонка Описание
Номер заказа Номер заказа.
Дата заказа Дата размещения заказа.
Статус Статус заказа.
Статус хранения Статус хранения заказа. Например, В ожидании.
Тип Тип заказа.
Номер заказа Номер заказа на покупку.
Чистый итог Итоговая сумма заказа, включающая скидки, и исключая налог.
Сумма скидки Любая скидка, примененная к заказу.
Сумма издержек Общая сумма любых издержек, примененных к заказу (например, финансовые издержки).
Сумма налога Сумма налога для заказа.
Валовая сумма Итоговая сумма заказа, включающая скидки, издержки и налог.
Валюта Валюта заказа.
