Настройка Службы синхронизации Интеграции учета

Служба синхронизации Интеграции учета (SalesLogix System Service) делает возможным обмен данными между Sage SalesLogix и вашей системой учета. Служба должна начать работу автоматически.

Для настройки

  1. Убедитесь, что вы настроили и подключили конечные точки перед началом работы службы.

  2. Просмотрите файл Sage.SalesLogix.SyncEngineService.exe.config, убедитесь, что соединение определено верным образом в конце файла. Информация о соединении должна соответствовать connection.config на вашем Веб Сайте.

По умолчанию файл конфигурации расположен в ...\Program Files\SalesLogix\Sage.SalesLogix.SyncEngineService.exe.config и файл с информацией по соединению в ...\inetpub\wwwroot\slxclient.

  1. На административной рабочей станции откройте Службы.
  2. Правой кнопкой мыши кликните SalesLogix System Service, затем нажмите Свойства.
  3. В поле Тип старта выберите Автоматически.

  4. Запустите службу под пользователем WebDLL.

  5. Убедитесь, что вы включили Windows Authentication для пользователя WebDLL.

  6. Нажмите Начать.
  7. Нажмите ОК.
