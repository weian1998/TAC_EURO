Использование Интеграции учета

Обычно, пользователь Sage SalesLogix управляет интеграцией субъектов и заказов на продажи из Sage SalesLogix и вашей системой с помощью:

  • Создания субъекта в Sage SalesLogix.
  • Связывания субъекта Sage SalesLogix с субъектом в учетной системе.
  • Создания сделки для субъекта.
  • Создания заказа на продажи для сделки.
  • Проверки цен на продукты.
  • Подтверждения заказа на продажи в учетной системе.

После подтверждения заказа на продажи, Учетное подразделение управляет процессом. Информация о транзакциях отображается в Sage SalesLogix для отслеживания целей.
