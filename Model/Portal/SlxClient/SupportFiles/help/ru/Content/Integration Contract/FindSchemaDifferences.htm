Сравнение и Настройка Различий схемы

Sage рекомендует сравнивать схемы базы данных в Sage SalesLogix со схемами учетного приложения, чтобы убедиться в целостности данных. Можно сравнить данные для различных ресурсов, чтобы определить, существуют ли различия в длине строк между системами.

Эта функция сравнивает схемы выбранных ресурсов в двух интегрированных системах и возвращает любые различия в длине строк.

Узнать больше о том, как Sage SalesLogix управляет различиями схем...

Максимальная Длина - Если синхронизируются данные, превосходящие максимальную длину поля, вся строчка будет отклонена.

Цифры дробной части - Если синхронизируются данные, превосходящие количество десятичных знаков, значение будет округлено до максимального знака после запятой. Например, если поле валюты определено как 17.4, тогда значение 123.12348 сохранится как 123.1235, и значение 123.12343 сохранится как 123.1234.

Чтобы сравнить

  1. В меню Администрирование выберите Установка интеграции.
  2. На закладке Системы учета нажмите Изменить для учетной системы.
  3. В диалоге Установка интеграции учета используйте раздел Различия схемы, чтобы найти различия между системами.
  1. В поле Фильтр по ресурсам выберите тип ресурса.
  2. Нажмите Выполнить.
  3. Просмотрите различия в таблице.
  4. Отметьте любую информацию, которая может нарушить целостность данных. Например, поле учетной системы может принять только три символа, в то время как то же поле в Sage SalesLogix принимает любой текст.
  5. Повторите шаг 3 для всех типов ресурсов.
  1. Нажмите ОК.
  2. Используйте Database Manager или другое приложение, чтобы изменить схемы в одной из систем.
