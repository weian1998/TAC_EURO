Краткий обзор шаблонов

Вы можете использовать шаблоны для создания писем, предложений, факсов, или сообщений электронной почты, которые будут использоваться для более чем одного контакта. Шаблоны содержат стандартный текст, который не будет меняться, в то время как поля почтовой рассылки содержат информацию Sage SalesLogix, которая меняется в зависимости от контакта, для которого выполняется рассылка.

Пример: Вы можете ответить на запрос информации о компании используя шаблон письма, содержащий стандартный текст, такой как "Благодарим Вас за Ваш вопрос...", и поля рассылки, такие как дата, имя контакта, название субъекта, адресс, приветствие, а также ваше имя и должность. Вы можете создавать и изменять содержимое письма для любого контакта.Sage SalesLogix поддерживает следующее программное обеспечение и типы шаблонов:

Поддерживаемые Sage SalesLogix Программные Продукты и Шаблоны

  • Sage SalesLogix поддерживает Microsoft Word для создания шаблонов.
  • Редактор шаблонов Sage SalesLogix связан с Word, что позволяет вам использовать возможности Word при создании или изменении шаблонов.
  • Демонстрационная база данных Sage SalesLogix содержит примеры шаблонов, называющиеся основные шаблоны, для писем, сообщений электронной почты и бланков факсов. Когда вы хотите создать новый шаблон, используйте основные шаблоны Писем, Факсов или Электронной Почты в качестве заготовки, поскольку легче изменить существующий шаблон, чем создавать новый.

Общие и Личные Шаблоны Sage SalesLogix

В диалоговом окне Управление шаблонами Sage SalesLogix классифицирует шаблоны как Общие и Личные.

  • Общие шаблоны создаются и распространяются системным администратором, и хранятся в папке Общие Шаблоны. Общие шаблоны обычно доступны всем пользователям; если в Sale SalesLogix не отображаются Общие Шаблоны, вы можете создать новый шаблон или скопировать Общий шаблон и создать Личный шаблон для собственного использования. Для упрощения нахождения соответствующих шаблонов у них должны быть уникальные названия; однако, шаблоны учитываются в Sage SalesLogix по их идентификационному номеру, а не по названию. Владелец Личного Шаблона может открыть к нему доступ другому пользователю.
  • Личные шаблоны создаются пользователем и автоматически сохраняются в папке Личные Шаблоны. дереве отображены только семейства, для которых существуют Личные Шаблоны. Если в Sage SalesLogix не отображены Личные шаблоны, вы можете создать новый шаблон или скопировать Общий шаблон, и сделать из него Личный шаблон для собственного использования. Для упрощения нахождения соответствующих шаблонов у них должны быть уникальные названия; однако, шаблоны учитываются в Sage SalesLogix по их идентификационному номеру, а не по названию. Владелец Личного Шаблона может открыть к нему доступ другому пользователю.
