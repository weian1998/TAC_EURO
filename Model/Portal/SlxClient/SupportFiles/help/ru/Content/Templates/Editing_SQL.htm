Редактирование SQL

Редактирование SQL рекомендуется проводить только опытным пользователям.

Можно использовать диалог Редактировать SQL, чтобы создать и редактировать запросы SQL для полей почтовой рассылки. Можно использовать меню Редактировать или Панель инструментов, чтобы вырезать, копировать, вставить или удалить текст. Также можно вставить параметры в SQL запрос.

Чтобы изменить SQL запрос

  1. Введите или редактируйте запрос SELECT, который вы хотите добавить к полю для рассылки.
  2. Нажмите в диалоге Редактировать SQL, где вы хотите вставить параметр.
  3. Нажмите кнопку Вставить параметр или из меню Редактировать выберите Вставить параметр.
    Отобразится меню со списком параметров, которые можно добавить.
    • AccountID - Вставляет :AccountID SQL параметр. Этот параметр будет заменен на AccountID для записи контакта.
    • AccountManager ID - Вставляет :AccountManagerID SQL параметр. Этот параметр будет заменен на AccountManagerID для записи контакта.
    • AddressID - Вставляет :AddressID SQL параметр. Этот параметр будет заменен на AddressID для записи контакта.
    • ContactID - Вставляет :ContactID SQL параметр. Этот параметр будет заменен на ContactID для записи контакта.
    • OpportunityID - Вставляет :OpportunityID SQL параметр. Этот параметр будет заменен на OpportunityID для записи контакта.
    • UserID - Вставляет :UserID SQL параметр. Этот параметр будет заменен на User ID для пользователя.
  4. Выберите параметр, который вы хотите вставить.
  5. Нажмите ОК.
