Редактирование Полей для рассылки

Можно использовать Редактор полей для почтовой рассылки, чтобы создавать новые и существующие поля для рассылки. Можно просмотреть поля для рассылки в списочном представлении и параметры редактирования на панели редактирования. Также можно удалить поля для рассылки из шаблона, выбрав кнопку Удалить из шаблона в панели инструментов.

Чтобы изменить поля для рассылки

  1. Откройте Редактор шаблонов.
  2. Выберите пункт Изменить поля.
    Открывается диалог Редактор полей почтовой рассылки SalesLogix. Поля для рассылки шаблона отображаются в списке. Списочное представление состоит из следующих колонок:
    • Поле - Отображает название поля для рассылки, появляющееся в шаблоне и иконку, связанную с типом поляe.
    • Тип - Отображает тип поля для рассылки. Возможные типы: Стандартный, подпись Outlook, таблица продуктов по сделкам, SQL, пользовательский.
    • Позиция - Отображает номер позиции поля, под которым оно появляется в шаблоне.
  3. Выделите поле таблицы, которое вы хотите изменить в Списочном представлении.
    В зависимости от типа поля для рассылки, будут отображаться различные параметры редактирования.
  • Редактирование стандартного Поля для рассылки
  • Идентификатор поля - Позволяет вам изменить текст, который появляется в шаблоне. Идентификатор поля для Стандартного поля для рассылки состоит из имени таблицы и отображаемого имени. Вы можете изменить только отображаемое имя.
  • Тип формата - Позволяет вам изменить тип форматирования, примененный к выбранному полю для рассылки. Например, фиксированный, целый, процент или валюта.
  • Строка форматирования - Позволяет вам форматировать значение, отображаемое в поле для рассылки. Нужно установить тип формата перед добавлением строки форматирования. Допустимые форматы строк зависят от выбранного вами типа формата.
  • Текст Перед - позволяет вам ввести текст, который появится перед выбранным полем для рассылки.
  • TТекст После - позволяет вам ввести текст, который появится после выбранного поля для рассылки.
  • Специальный формат - позволяет вам применить специальное форматирование к полю для рассылки (включая текст, который появляется до и после поля для рассылки).
  • Таблица - Отображает таблицу, связанную с полем базы данных. Если поле базы данных является результатом присоединения, тогда заголовок этого параметра будет "Основная таблица".
  • Поле - Отображает поле базы данных. Если поле базы данных является результатом присоединения, тогда заголовок этого параметра будет "Ключ".
  • Вычисляемое - Отображает, является ли поле вычисляемым.
  • Присоединение - Отображает, является ли поле результатом присоединения.
  • Защитить Форматирование символов: Позволяет защитить форматирование символов во время рассылки.
  • Редактирование Поля подписи Outlook
  • Идентификатор поля - Позволяет вам изменить текст, который появляется в шаблоне.
  • Редактирование Поля продукта по сделке
  • Идентификатор поля - Позволяет вам изменить текст, который появляется в шаблоне.
  • Формат таблицы - Позволяет применить форматирование к полю таблицы.
  • Параметры - Позволяет применить свойства к формату таблицы.
  • Auto Fit - Позволяет установить свойства автоматического изменения размеров таблицы.
  • Sum Text - Позволяет ввести текст, который будет использоваться как лэйбл для суммы цены
  • продукты в таблице.
  • Изменить Колонки Таблицы - позволяет изменить названия столбцов, ширину, выравнивание.
  • Редактирование Пользовательского Поля
  • Идентификатор поля - Позволяет вам изменить текст, который появляется в шаблоне.
  • Редактирование Поля Таблицы
  • Идентификатор поля - Позволяет вам изменить текст, который появляется в шаблоне.
  • SQL - AПозволяет вам ввести запрос SELECT для получения данных из OLE DB provider.
  • Все колонки и строки используются для создания таблицы.
  • Форматировать Как - Позволяет изменить тип поля для рассылки.
  • Формат таблицы - Позволяет применить форматирование к полю таблицы.
  • Параметры - Позволяет применить свойства к формату таблицы.
  • Auto Fit - Позволяет установить свойства автоматического изменения размеров таблицы.
  • Изменить Колонки Таблицы - позволяет изменить названия столбцов, ширину, выравнивание.
  • Редактирование Списка с Буллитами
  • Идентификатор поля - Позволяет вам изменить текст, который появляется в шаблоне.
  • SQL - Позволяет вам ввести запрос SELECT для получения данных из OLE DB provider. Только записи первой колонки будут отображены.

Нажмите кнопку Редактировать, чтобы открыть диалог Редактировать SQL.

  • Форматировать как - Отображает тип поля для рассылки.
  • Редактирование Нумерованного Списка
  • Идентификатор поля - Позволяет вам изменить текст, который появляется в шаблоне.
  • SQL - Позволяет вам ввести запрос SELECT для получения данных из OLE DB provider. Только записи первой колонки будут отображены.

Нажмите кнопку Редактировать, чтобы открыть диалог Редактировать SQL.

  • Форматировать как - Отображает тип поля для рассылки.
  • Редактирование Текстового Поля
  • Редактирование Поля изображения
  1. Нажмите Применить после внесения изменений.
  2. Нажмите Да для сохранения изменений.
