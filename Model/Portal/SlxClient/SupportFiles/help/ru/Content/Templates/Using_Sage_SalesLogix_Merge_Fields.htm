Использование полей для рассылки Sage SalesLogix

Меню Вставить поля SLX содержит большинство часто используемых полей для рассылки. При создании или редактировании шаблона, когда вам необходимо поле Sage SalesLogix, которого нет в меню, можно получить доступ ко всем полям для рассылки Sage SalesLogix. В дополнение, можно добавить поле к меню и кастомизировать меню.
  • Поля для рассылки составляются на основе XML, учитывающего регистр. Если регистр в поле для рассылки меняется, он не будет отослан.
  • При создании или изменении шаблона, нельзя помещать поля для рассылки в текстовое поле, поскольку эта функция не поддерживается Microsoft Word.

Диалог Вставить поле для рассылки имеет две панели:

  • Левая панель содержит закладки с таблицами Контактов или Наводок, Адресов и Информации о пользователях. Нажмите плюсик, чтобы раскрыть иерархическое представление таблиц. Выберите Показать все связи, чтобы увидеть все присоединенные таблицы.
  • Правая панель показывает все поля для выбранной таблицы. Вы можете кликнуть правой кнопкой мыши по полю, чтобы создать локальную связь.
    • Таблица Поле
    Контакт или наводка Содержит информацию о получателе письма, включая имя и компанию. Если Контакт выбран в качестве Основной таблицы в свойствах шаблона, поля из таблицы Контакт будут отображаться. Если Наводка выбрана в качестве Основной таблицы, поля из таблицы Наводки будут отображаться.
    Адрес Содержит информацию об адресе получателя. Используйте эту закладку при выборе полей адреса контакта или полей адреса наводок.
    Информация о пользователе Содержит поля, включающие информацию об адресе, об отправителе.

Чтобы получить доступ к дополнительным полям рассылки

  1. Из меню Вставить поля SLX нажмите Все поля SLX.
  2. Выберите одну из закладок.
  3. На левой панели, выберите название таблицы.
  4. На правой панели, выберите поле, затем нажмите Вставить.
  5. Продолжайте добавлять поля в шаблон, затем нажмите Завершить.

Чтобы добавить поля в меню Вставить поле SLX

  1. Из меню Вставить поля SLX нажмите Кастомизировать
  2. Нажмите Вставить поле.
  3. Нажмите закладку, содержащую таблицы и поля, которые вы хотите вставить.
  4. На правой панели, выберите поле.
  5. Если необходимо поменять тип формата поля, в списке Тип формата выберите тип формата.
  6. В поле Строка форматирования введите строку форматирования.
  7. В разделе Показать как, поле появится в меню как есть, но вы можете ввести отображаемое имя для поля.
  8. Нажмите кнопку Добавить к меню.
  9. Продолжайте добавлять поля в меню, или нажмите Завершить.
