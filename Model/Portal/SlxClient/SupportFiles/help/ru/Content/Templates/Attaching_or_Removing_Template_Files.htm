Прикрепление или удаление файлов Шаблона

Вы можете вложить файлы Windows или файлы Библиотеки Sage SalesLogix в шаблон. Когда шаблон используется в почтовой рассылке, прикрепленные файлы автоматически отправляются контакту. Если шаблон, содержащий вложение, копируется, вложение копируется также. Вы можете удалить вложение из шаблона.

Для вложения файла в шаблон

  1. Откройте Редактор шаблонов.
  2. В зависимости от типа файла, который вы хотите вложить, выберите одно из нижеследующих действий:
    • Файл Windows: Нажмите кнопку Добавить файл. Выберите путь к файлу, нажмите Добавить файл.
    • Файл библиотеки: Нажмите кнопку обавить файл библиотеки. В диалоге Библиотека SalesLogix выберите файлы и нажмите Вставить.
      Имя файла появляется в поле Вложение.
  3. Вы можете добавить другой файл или нажать кнопку Сохранить и Закрыть.

Для удаления файла из шаблона

  1. Откройте Редактор шаблонов.
  2. В поле Вложения выберите вложение, которое вы хотите удалить.
  3. Выберите одно из следующих действий, чтобы удалить вложение:
    • После щелчка правой кнопкой мыши выберите Удалить.
    • Нажмите клавишу Delete.
  4. Вы можете удалить еще один файл или нажать Сохранить и закрыть, чтобы закрыть редактор шаблонов.
