Параметры Email

При создании e-mail шаблона, можно установить параметры для сообщений, которые вы хотите отправить. Например, как администратору, вам может быть нужно отправить e-mail от лица вашей команды продавцов, с общего адреса компании (например, salesteam@company.com). Возможно, вам потребуется, чтобы поле Reply To в ответном сообщении заполнялось динамически во время почтовой рассылки, таким образом, чтобы когда контакты отвечают на полученное сообщение, оно автоматически адресуется менеджеру приписанного им субъекта.

Кнопка параметры E-mail доступна только в редакторе E-mail.

Чтобы установить параметры E-mail

  1. Выберите шаблон E-mail, откройте Редактор шаблонов.
  2. Нажмите Параметры E-mail на панели инструментов.
  3. В поле Ответить, выберите кто должен отобразиться в поле Кому в ответном сообщении. Нажмите кнопку Адресная книга, чтобы выбрать имя из адресной книги Outlook, или из выпадающего меню, выберите один из следующих параметров:
  • <<Текущий пользователь>> - этот параметр вставляет строку <<Текущий пользователь>> в поле Ответить и показывает текущего пользователя в качестве пользователя почтовой рассылки. Это опция по умолчанию.
  • <<Менеджер Субъекта>> - этот параметр вставляет строку <<Менеджер субъекта>> в поле Ответить и показывает пользователя в качестве менеджера субъекта соответствующего контакта. Эта опция доступна только для Outlook.
  • Специальный пользователь - Этот параметр открывает адресную книгу Outlook и отображает выбранное имя в поле Ответить, показывая пользователя как выбранного из адресной книги. Эта опция доступна только для Outlook.
  1. Из списка важность выберите Низкая, Нормальная (По умолчанию) и Высокая.
  2. Чтобы определить чувствительность сообщения, выберите Нормальная (по умолчанию), Личная, Частная, или Конфиденциально из списка Чувствительность.
  3. Чтобы получить уведомление о том, что сообщение было отправлено, выберите Запрос отчета о доставке.
  4. Чтобы получить уведомление о том, что сообщение было прочитано, выберите Запрос отчета о прочтении.
  5. Нажмите ОК.
