Использование редактора E-mail

С помощью окна Редактора E-mail можно содавать новый шаблон e-mail или редактировать существующий шаблон. Так же как Редакторы Писем и Факсов, Редактор e-mail содержит панель инструментов Sage SalesLogix, которая предоставляет доступ к специальным функциям наряду с функциями Microsoft Word.Редактор E-mail отображает следующие опции при редактировании или создании шаблона:

Поле Описание/действие
Вставить поле SLX Нажмите, чтобы отобразить меню с наиболее часто используемыми полями для рассылки.
Вставить Файл Нажмите, чтобы вставить файл Windows как e-mail вложение. Эта опция доступна только для шаблона типа e-mail.
Вставить файл Библиотеки Нажмите, чтобы вставить файл из библиотеки как e-mail вложение. Эта опция доступна только для шаблона типа e-mail.
Сохранить и Закрыть Нажмите, чтобы сохранить изменения, сделанные в шаблоне, и закрыть редактор шаблонов.
Сохранить Нажмите, чтобы сохранить изменения, сделанные в шаблоне, и оставить окно редактора открытым.
Отобразить Позволяет выбрать, какие из следующих e-mail полей отображаются в редакторе: От, Кому, Тема и Вложения. Эта опция доступна только для шаблона типа e-mail
Формат сообщения Нажмите стрелку, чтобы выбрать формат сообщения из списка. Эта опция доступна только для шаблона типа e-mail.
Просмотр Нажмите, чтобы просмотреть шаблон, как он будет отправлен с текущими записями.
Почтовая Рассылка SLX Нажмите, чтобы начать почтовую рассылку.
Кому: Это поле автоматически заполняется адресом e-mail контакта и не может быть отредактировано из этого окна.
Тема Нажмите для ввода темы шаблона e-mail. При выполнении почтовой рассылки по этому шаблону, тема Почтовой рассылки автоматически заполнится текстом из строки Тема шаблона.
Показать меню Word Нажмите, чтобы показать меню Microsoft Word.
Редактировать поля Нажмите, чтобы открыть окно Редактор полей для рассылки .
Вставка специальных полей Нажмите, чтобы вставить новую таблицу продуктов по сделке, SQL запрос, или пользовательское поле для рассылки в шаблон.
Параметры E-mail Нажмите, чтобы изменить такие параметры E-mail как Ответить, Важность и Чувствительность.

При выборе формата сообщения для шаблона, помните, что хотя Microsoft Word не отображает информацию заголовков и колонтитулов или графику на странице HTML, он хранит данные.
