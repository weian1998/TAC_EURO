Установка формата полей для рассылки

Используйте диалог Формат полей SalesLogix, чтобы установить тип формата и строку форматирования для полей для рассылки.

Формат поля по умолчанию для большинства текстовых полей None. Если поле содержит численные значения, номер телефона, валюту, логический тип данных, или другие данные, измените формат поля, чтобы соответствовать типу информации. Посмотрите на параметры в списке форматов поля, чтобы выбрать, какой тип данных вы используете.

Показать примеры
  • Выберите Дату в качестве типа формата, после чего вы будете вводить строку в формате mm/dd/yyyy.
  • TПользовательский тип формата по умолчанию отформатирует USERID как “LastName, FirstName”, при использовании строки форматирования “FL”, USERID отформатируется как “FirstName LastName”.

Чтобы установить формат поля для рассылки

  1. В диалоге Формат поля SalesLogixвыберите тип из списка Тип формата.
  2. В поле Строка форматирования введите строку, необходимую для типа формата.
  3. Нажмите ОК.
