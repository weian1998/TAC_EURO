Создание Предложения

Можно использовать Почтовую Рассылку для создания торговых предложений.

Для создания предложения

  1. В меню Написать выберите Шаблоны.
  2. В диалоговом окне Управление шаблонами нажмите Добавить.
  3. В диалоге свойств установите свойства шаблона нажмите OK.
    Основной шаблон, соответствующий вашему выбору, появляется в редакторе. Например, если вы создаете шаблон письма, основной шаблон письма появляется в редакторе.
    Можно добавить или удалить содержимое, или вставить текст из другого документа.
  4. Чтобы добавить поля для рассылки Sage SalesLogix в шаблон, установите курсор там, где вы хотите видеть поле, нажмите Вставить поле SLX и затем из списка полей выберите поле.
    Новое поле появляется в шаблоне.
  5. Продолжайте добавлять или удалять поля при необходимости.
  6. Чтобы добавить таблицу продуктов по сделке, поместите курсор туда, где вы хотите чтобы отображалась таблица, затем на панели инструментов, нажмите Вставить специальное поле.
    1. Из меню выберите Таблица продуктов по сделке.
      Открывается диалог в SalesLogix Mail Merge Field Editor.
    2. Выделите поле таблицы, которое вы хотите изменить.
    3. Используйте изменить поля, чтобы произвести изменения, затем нажмите Принять.

      • Идентификатор поля - Позволяет вам изменить текст, который появляется в шаблоне.

      • Формат таблицы - Позволяет применить форматирование к полю таблицы.

      • Параметры - Позволяет применить свойства к формату таблицы.

      • Auto Fit - Позволяет установить свойства автоматического изменения размеров таблицы.
      • Sum Text - Позволяет ввести текст, который будет использоваться как лэйбл для суммы цены продуктов в таблице.
      • Изменить Колонки Таблицы - позволяет изменить названия столбцов, ширину, выравнивание.

  7. Нажмите ОК.
