Изменение Колонок

Изменение колонок рекомендуется проводить только опытным пользователям.

При изменении колонок в диалоге Изменить Колонки, информация о колонках должна вводиться вручную на основе SQL. Нет автоматической синхронизации между SQL и информацией в колонках. Если не обеспечивается никакая информация о колонках, колонки будут форматироваться в соответствии с параметрами по умолчанию, названия полей будут использоваться в качестве названий колонок. Если определено больше колонок, чем содержится в SQL, тогда лишняя информация о колонках будет отброшена. Если колонок определено меньше, чем содержится в SQL, тогда оставшиеся колонки будут отформатированы в соответствии с параметрами по умолчанию, но не будут иметь названия.

Диалог Изменить колонки содержит следующие параметры:

  • Нажмите кнопку Добавить колонки, чтобы добавить новую колонку в конец списка со следующими параметрами по умолчанию:

    • Название = “Column%d” где %d номер колонки

    • Колонка = %d где %d номер колонки

    • Ширина = 1.00

    • Выравнивание = (по умолчанию)

    • Тип формата= (none)

    • Строка Форматирования = пусто

  • Кнопка Удалить колонку удаляет выбранную колонку из списка.
  • Кнопка Сдвинуть вверх сдвигает выбранную колонку на одну позицию вверх. Если колонка уже на первой позиции, это действие не имеет эффекта.
  • Кнопка Сдвинуть вниз сдвигает выбранную колонку на одну позицию вниз. Если колонка уже на последней позиции, это действие не имеет эффекта.
  • Таблица колонок используется для отображения и редактирования колонок. Название, Ширина, Выравнивание, Тип формата, Строка форматирования могут быть изменены с помощью выделения колонки и нажатия клавиши F2.
    • Заголовок: Отображает текст, который будет показываться в первой ячейке колонки.
    • Колонка: Отображает порядковый номер позиции колонки. Нельзя редактировать эту колонку.
    • Ширина: Отображает ширину колонки в дюймах. Ширина по умолчанию 1,00.
    • Выравнивание: Представляет собой выравнивание текста в ячейке.
    • Тип Формата: Отображает тип форматирования, применяемый к этому полю.
    • Вы можете выбрать из списка: (none), Фиксированный, Целый, Дата/Время,
    • Проценты, Валюта, Пользователь, Телефон, Куратор, Логический.
    • Строка Форматирования: Отображает строку, которая будет использоваться для выбранного Типа формата.

    		•

    Поля Таблицы продуктов по сделке фиксированы, колонки нельзя перемещать, добавлять или удалять.
