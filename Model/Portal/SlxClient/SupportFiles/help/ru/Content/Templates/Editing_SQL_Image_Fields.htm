Редактирование полей изображений SQL

Можно использовать Редактор полей для почтовой рассылки, чтобы создавать новые и существующие поля изображений для рассылки. Когда вы выбираете SQL Image field из списка, параметры редактирования будут отображаться на панели редактирования.

  • Идентификатор поля - Позволяет вам изменить текст, который появляется в шаблоне.
  • SSQL - Позволяет вам ввести запрос SELECT для получения данных из OLE DB provider. Только записи первой колонки будут отображены.
    Нажмите кнопку Редактировать, чтобы открыть диалогe Редактировать SQL .
  • Форматировать как - Отображает тип поля для рассылки.
  • Возвращение запроса - Позволяет выбрать, каким образом хранится изображение. Вы можете выбрать из:
  • Данные изображения

    Изображение хранится как поле в базе данных. Необходимо также выбрать изображение из выпадающего списка. Тип изображения используется для создания временного имени файла с расширением типа файла.

  • Полный путь и имя файла изображения

    Используйте этот параметр, если запись, возвращенная в результате SQL запроса содержит полностью определенный путь и имя изображения.

    Если изображение начинается с HTTP, оно будет вставлено как гиперссылка на этот файл, иначе, копия изображения будет храниться в документе.

  • Адрес гиперссылки - Позволяет ввести URL, который будет открываться из поля для рассылки.
  • Примеры Адресов гиперссылок
  • http://www.saleslogix.com/
  • mailto:jabbott@abbott.demo?Subject=Special Offer&Body=Yes, please send me info on this special offer!
  • mailto::SLXUserName <:SLXUserEmail>?Subject=Special Offer&Body=Yes, please send me info on this special offer!
  • ftp://ftp.winzip.com/winzip81.exe
  • news://msnews.microsoft.com/microsoft.public.word.mailmerge.fields
  • Предмет гиперссылки - Позволяет выбрать, каким образом будет открываться URL. Выберите Предмет из выпадающего списка.
    • Параметр предмета Описание
    Страница по умолчанию (none) Открывает ссылку в том же окне.
    Тот же фрейм

    Открывает ссылку в том же фрейме.

    Относится к HTML цели "_self".
    Целая страница

    Открывает ссылку полностью в текущем окне.

    Относится к HTML цели "_top".
    Новое окно

    Открывает ссылку в новом окне без названия.

    Относится к HTML цели "_blank".

    Родительский

    Фрейм

    Открывает ссылку в родительском окне документа с ссылкой.

    Относится к HTML цели "_parent".
  • Подсказка гиперссылки - Позволяет ввести текст, который появляется когда указатель мыши устанавливается на гиперссылке.
    Параметры могут быть включены в адрес гиперссылки и подсказку гиперссылки, что может быть полезно при создании mailto:гиперссылки.
    • Параметр Sage SalesLogix Описание
    SLXAccountManagerEmail Параметр :SLXAccountManagerEmail будет заменен на e-mail адрес менеджера субъекта для записи контакта.
    SLXAccountManagerName Параметр :SLXAccountManagerName будет заменен на имя менеджера субъекта для записи контакта.
    SLXAccountManagerTitle Параметр :SLXAccountManagerTitle будет заменен на должность менеджера субъекта для записи контакта.
    SLXAccountManagerPhone Параметр :SLXAccountManagerPhone будет заменен на номер телефона менеджера субъекта для записи контакта.
    SLXAccountManagerFax Параметр :SLXAccountManagerFax будет заменен на номер факса менеджера субъекта для записи контакта.
    SLXUserEmail Параметр :SLXUserEmail будет заменен на e-mail пользователя почтовой рассылки.
    SLXUserName Параметр :SLXUserName будет заменен на имя пользователя почтовой рассылки.
    SLXUserTitle Параметр :SLXUserTitle будет заменен на должность пользователя почтовой рассылки.
    SLXUserPhone Параметр :SLXUserPhone будет заменен на номер телефона пользователя почтовой рассылки
    SLXUserFax Параметр :SLXUserFax будет заменен на номер факса пользователя почтовой рассылки
