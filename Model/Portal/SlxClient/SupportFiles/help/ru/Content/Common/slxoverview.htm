Знакомство с Sage SalesLogix для Веб

При помощи Sage SalesLogix Web вы можете управлять всеми аспектами цикла продаж. Кроме возможности управления информацией о контактах, субъектах, сделках, заявках, делах, пользователям предоставляется доступ к инструментам для формирования отчетности, автоматизации некоторых процессов, а также для создания электронных писем, факсов и почтовых рассылок. С помощью дополнительных прав доступа, можно иметь возможность использования административных инструментов для управления пользователями, командами, подразделениями, справочниками.

В зависимости от того как администратор настроил систему вы можете получить доступ к Web клиенту Sage SalesLogix через сеть или телефон.

В данном руководстве приводится описание того, как эффективно использовать клиент Sage SalesLogix Web. Ваша рабочая область может выглядеть по-разному, в зависимости от прав доступа или использования нестандартно настроенной установки Web клиента Sage SalesLogix.

Ваш системный администратор может предоставить дополнительную информацию.
