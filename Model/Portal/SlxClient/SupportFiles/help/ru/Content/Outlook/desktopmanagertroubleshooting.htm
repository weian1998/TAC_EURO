Устранение неисправностей с Desktop Integration Module

Вы можете столкнуться с следующими проблемами с Desktop Integration module.

Проблема:

Как мне узнать, установлена ли у меня корректная версия Desktop Integration module?

Решение:

  • Посмотрите на начальную страницу Веб Клиента Sage SalesLogix или в Параметрах выберите Общее.

Если рядом с кнопкой Расширить SalesLogix есть текст "Расширения установлены", значит, установлена корректная версия. Если вы видите текст "Нажмите, чтобы загрузить расширения продукта", тогда нужно установить последнюю версию Desktop Integration module.

Проблема:

Иконка Sage SalesLogix Desktop Manager не появляется в SysTray.

Возможные причины и решения:

  • Не установлен Desktop Integration Module.
  • Необходимо установить Desktop Integration Module.
  • Desktop Integration Module был установлен до того, как был установлен браузер.
  • Если вы установили текущий браузер после установки Desktop Integration Module, нужно переустановить Desktop Integration Module, используя текущий браузер.

    • Это может случиться, если вы установили Firefox после установки Desktop Integration Module с помощью Internet Explorer.
  • Откройте вручную SLXDesktopManager.exe. Местоположение этого файла зависит от операционной системы:

Местолположение по умолчанию:

\\Documents and Settings\[username]\Local Settings\Application Data\ (XP/2000/2003) или

\\Users\<user.domain>\appdata\local (Vista/7/2008).

Проблема:

Как отключить Desktop Manager?

Решение:

  • На панели задач Windows, в разделе System Tray, щелкните правой кнопкой мыши Sage SalesLogix Desktop Manager icon и выберите Выход.

Перед удалением Sage SalesLogix Desktop integration Module необходимо сначала выйти из Sage SalesLogix Desktop Manager.

Проблема:

Мне было предложено загрузить новую версию Desktop Integration Module.

Решение:

  • Загрузите новую версию Desktop Integration Module.

    • Если вы используете Firefox, нужно сначала деинсталлировать Дополнение Sage SalesLogix Desktop Integration Module в Firefox.

    Чтобы деинсталлировать дополнение Desktop Integration Module, откройте Firefox. В меню Tools, нажмите Add-ons, выберите Sage SalesLogix Desktop Integration Module, нажмите Uninstall. Нажмите Uninstall для подтверждения. Нажмите Restart Firefox. Нажмите Restart.

Проблема:

Как деинсталлировать Desktop Integration Module?

Решение:

  1. Откройте Добавить или удалить программы в Windows.
  2. Выберите из списка Sage SalesLogix Desktop Integration Module и нажмите Удалить.
  3. Нажмите Да.

    		4.

    Если вы используете Firefox, нужно сначала деинсталлировать Дополнение Sage SalesLogix Desktop Integration Module в Firefox.

    Чтобы деинсталлировать дополнение Desktop Integration Module, откройте Firefox. В меню Tools, нажмите Add-ons, выберите Sage SalesLogix Desktop Integration Module, нажмите Uninstall. Нажмите Uninstall для подтверждения. Нажмите Restart Firefox. Нажмите Restart.
