Изменение Пароля (Администратор)

Можно изменить пароль пользователя на форме Пользователи или через Параметры пользователя.

Узнать больше...

Новые пользователи создаются с паролем password по умолчанию. Системный администратор, административный пользователь или пользователь Sage SalesLogix могут изменить этот пароль в любой момент.
  • Пароли должны начинаться и заканчиваться на цифру (0-9) или на букву алфавита (A-Z или a-z).
  • Точка с запятой - не допустимый символ для пароля.

Чтобы изменить пароль в Настройках Пользователя

  1. В меню Сервис, выберите Параметры.
  2. Выберите закладку Пароль, а затем выберите необходимого пользователя из соответствующего списка.
  3. Введите новый пароль.
  4. Введите снова новый пароль в поле Подтверждение Пароля.
  5. Нажмите Сохранить. Появится сообщение об удачной смене пароля.

    		6.

    Вы должны выйти из Web клиента Sage SalesLogix, а затем зайти снова, чтобы изменения вступили в силу.

Чтобы изменить пароль в форме Пользователи

  1. На Панели Навигации нажмите Администрирование.
  2. Нажмите Пользователи и выберите нужное имя пользователя.
  3. В Детальном представлении Пользователя, нажмите кнопку Изменить пароль.
