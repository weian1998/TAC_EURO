Что такое группы?

Группы - это совокупность субъектов, контактов, сделок, заявок, объединенных по определенному признаку или условию. Группы помогают управлять технологическим процессом. Вы можете использовать группы для работы с подмножеством записей, выбирать соотвествующие данные для вывода в отчет или для отправки в качестве сообщений, электронных писем и факсов, используя возможности Mail Merge. Одна запись может отображаться в нескольких группах. Вы можете экспортировать записи группы в Excel или распределить права доступа к своим группам между другими пользователями.

Для просмотра дополнительных групп, нажмите на стрелки.

Существует три типа групп:

  • Специальные группы - группы созданные из выбранных записей
  • Группы поиска - группы, основанные на поиске
  • Группы расширенного запроса - группы, основанные на условиях построителя запросов

Чтобы открыть группу

  • Выполните одно из следующих действий:
    • В меню Группы, выберите группу, которую хотите просмотреть.
    • Нажмите на закладку группы, которую хотите открыть. Например, Все субъекты.

Чтобы посмотреть условия группы

  1. Щелкните правой кнопкой по закладке группы.
  2. Нажмите Изменить.
    Откроется построитель запросов SalesLogix.
  3. Используйте закладку условия построителя запросов, чтобы посмотреть условия, которым должны соответствовать записи в группе. Данная закладка недоступна для специальных групп.
