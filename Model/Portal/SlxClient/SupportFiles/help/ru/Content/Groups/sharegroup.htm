Совместное использование групп

Вы можете дать доступ к вашим группам другим пользователям Sage SalesLogix, командам или подразделениям.

  • Если вы выпускаете группу для других пользователей и затем удаляете ее, группа по-прежнему остается доступной для других пользователей.
  • Вы не можете дать доступ к группе, к которой вам предоставил доступ другой пользователь.
  • При внесении изменений в группу совместного использования вы должны обновить группу, чтобы другие пользователи видели последнюю версию.
  • Вы можете просматривать информацию о правах доступа к существующим группам, определять владельца и пользователей, для которых открыт доступ на просмотр.
  • Пользователи могут просматривать только те субъекты, контакты, сделки и заявки, к которым у них есть доступ.
  • Если специальную группу выпустили для других пользователей, то они могут добавить или удалить записи группы.
  • Если вы предоставляете доступ к закладке интерактивной панели, которая содержит группу, к группе также будет предоставлен доступ для тех же пользователей, которым был предоставлен доступ к закладке интерактивной панели.
Чтобы предоставить доступ к группе другим пользователям
  1. Щелкните правой кнопкой по закладке группы.
  2. Нажмите Доступ.
    Если у вас нет прав на предоставление доступа к группе данный пункт меню будет недоступен.
  3. Нажмите кнопку Добавить.
  4. В диалоге Назначения доступа, откройте закладку Пользователи.
  5. Чтобы ограничить список пользователей, выберите тип пользователей из выпадающего списка Тип пользователия (например, Удаленный).
  6. Выберите пользователя(ей), которым вы хотите предоставить доступ, затем нажмите OK.
Чтобы предоставить доступ к группе другим командам
  1. Щелкните правой кнопкой по закладке группы.
  2. Нажмите Доступ.
    Если у вас нет прав на предоставление доступа к группе данный пункт меню будет недоступен.
  3. Нажмите кнопку Добавить.
  4. В диалоге Назначения доступа, откройте закладку Команды.
Чтобы предоставить доступ к группе другим подразделениям
  1. Щелкните правой кнопкой по закладке группы.
  2. Нажмите Доступ.
    Если у вас нет прав на предоставление доступа к группе данный пункт меню будет недоступен.
  3. Нажмите кнопку Добавить.
  4. В диалоге Назначения доступа, откройте закладку Подразделения.
  5. Выберите подразделение(я), которым вы хотите предоставить доступ, затем нажмите OK.
Чтобы удалить доступ к группе
  1. Щелкните правой кнопкой по закладке группы.
  2. Нажмите Доступ.
    Если у вас нет прав на предоставление доступа к группе данный пункт меню будет недоступен.
  3. Выберите пользователя, подразделение или команду, которым вы больше не хотите предоставлять доступ к группе.
  4. Нажмите кнопку Удалить, затем нажмите OK.
Чтобы обновить гуппу общего использования

Если вы внесли изменения в группу и хотите предоставить доступ к данным изменениям, вы должны обновить группу.

  1. Щелкните правой кнопкой по закладке группы.
  2. Нажмите Доступ.
    Если у вас нет прав на предоставление доступа к группе данный пункт меню будет недоступен.
  3. В диалоге Досутп к группе, нажмите OK, чтобы выпустить изменения для пользователей из списка.
