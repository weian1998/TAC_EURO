Добавление специальной группы

Вы можете создавать специальные грппы для хранения списка выбранных записей из любого списочного представления.

Чтобы добавить специальную группу

  1. Откройте списочное представление
  2. Откройте группу или воспользуйтесь поиском, чтобы найти необходимые записи, для добавления в группу.
  3. Выберите одну или несколько записей удерживая клавиши CTRL или Shift при нажатии на записи, затем выполните одно из следующих действий:
    • В панели задач, выберите Сохранить записи как группу.
    • Щелкните правой кнопкой в таблице списочного представления и выберите пункт меню Сохранить записи как группу.

    		4.

    Если вы не выбрали ни одной записи, вам будет предложено использовать все записи из группы. При нажатии OK, все записи группы будут выбраны.
  4. В диалоге Добавить записи в новую группу введите название в поле Название группы.

  5. Нажмите ОК.
