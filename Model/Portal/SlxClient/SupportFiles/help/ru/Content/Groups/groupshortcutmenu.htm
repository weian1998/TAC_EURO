Использование контекстного меню в группах

Контекстное меню позвляет вам создавать и управлять группами.

Не все пункты меню доступны во всех списочных представлениях.

Чтобы открыть контекстное меню

  1. В списочном или детальном представлении, щелкните правой кнопкой по закладке группы.
  2. Выберите из следующих пунктов меню:
    • Скопировать
    • Изменить
    • Скрыть
    • Удалить
    • Доступ
    • Экспорт в Excel
    • Экспорт в файл
    • По умолчанию
    • Восстановить макет

    Отменяет все настройки, сделанные в группе и возвращает внешний вид группы к макету определенному в закладке макет построителя запросов.

    • Добавить группу

Чтобы открыть контекстное меню результатов поиска

  1. После выполнения поиска, щелкните правой кнопкой по закладке Результаты поиска.
  2. Выберите из следующих пунктов меню:
    • Сохранить поиск как группу
    • Экспорт в Excel
    • Экспорт в файл
    • Восстановить макет

    Отменяет все настройки, сделанные в группе и возвращает внешний вид группы к макету определенному в закладке макет построителя запросов.

    • Добавить группу

Чтобы открыть контекстное меню списочного представления

  1. Выберите одну или несколько записей
  2. Щелкните правой кнопкой в списочном представлении группы.
  3. Выберите из следующих пунктов меню:
    • Сохранить записи как группу
    • Удалить из группы
    • Добавить в существующую группу

    • Обновить наводки (только для списочного представления наводок)

    Позволяет изменить доступ к выбранным записям.

    • Удалить наводки (только для списочного представления наводок)
    • Обновить сделки (только для списочного представления сделок)
    • Статистика сделки (только для списочного представления сделок)
    • Слияние записей (только для списочных представлений контактов и субъектов)
    • E-mail (только для списочного представления контактов)
