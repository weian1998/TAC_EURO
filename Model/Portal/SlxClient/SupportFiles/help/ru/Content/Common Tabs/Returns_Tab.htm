Использование закладки возвраты

Закладка возвраты на детальных представлениях субъекта и заявки отображает список всех возвратов для выбранной записи.

Как выполнить?

Открыть закладку возвраты
  1. Откройте детальное представление субъекта или заявки.
  2. Откройте закладку Возвраты.Если закладка не отображается, то откройте закладку Все закладки.
Добавить возврат (только для детального представления заявки)
  1. Нажмите Создать возврат.
  2. Следуйте инструкции по добавлению возврата.
Посмотреть более подробную информацию о возврате
  • Нажмите на ИД возврата, чтобы открыть детальное представление возврата.
