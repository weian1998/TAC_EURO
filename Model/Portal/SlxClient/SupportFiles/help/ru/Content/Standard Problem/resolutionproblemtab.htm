Закладка стандартные проблемы стандартного решения

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Используйте это представление для просмотра и связи стандартных проблем с решением.

Как выполнить?

Открыть это представление
  • На детальном представлении стандартного решения, откройте закладку Стандартные проблемы.
Связать с проблемой
  1. Нажмите кнопку Связать с проблемой.
  2. Воспользуйтесь поиском для нахождения стандартной проблемы.
Удалить проблему
  1. Выберите запись, которую вы хотите удалить и нажмите кнопку Удалить.
  2. Нажмите ОК.

Представление содержит следующие колонки:

  • Описание - Отображает описание проблемы.
  • Область - Отображает, где возникает проблема.
  • Категория - Отображает тип проблемы.
  • Проблема - Отображает проблему.
  • Заметки - Отображает краткие заметки о проблеме.
