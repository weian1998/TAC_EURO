Информационные поля стандартной проблемы

Поля информации о стандартной проблеме содержат информацию о выбранной проблеме. При необходимости, вы можете редактировать или удалять информацию.

 

Поле Описание/Действие
Описание Введите описание решения, которое будет понятно другим пользователям.

Проблема

Эти три поля кратко описывают суть проблемы. Нажмите рядом с каждым из полей кнопку найти, выберите соответствующие значения из списка, а затем нажмите OK.

Набор элементов списка Категория зависит от пунктов, выбранных для Области, а набор элементов списка Проблема зависит от пунктов, выбранных для Категории.
Заметки Введите информацию, которая поможет другим пользователям понять решение.
