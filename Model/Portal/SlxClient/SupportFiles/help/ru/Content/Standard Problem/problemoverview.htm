Что такое стандартные проблемы и решения?

Стандартная проблема - это проблема в вашем продукте, которая влечет за собой множество обращений клиентов. Ваш центр обслуживания клиентов может часто получать звонки о похожих проблемах. Используйте Стандартные проблемы для фиксирования описаний этих проблем, и Стандартные решения - для хранения способов их решения. Когда стандартные проблемы и решения установлены, их можно использовать чтобы дополнить описание заявки или использовать в качестве решения заявки.

Стандартная проблема может быть связана с одним стандартным решением или несколькими решениями. В дополнение, стандартное решение может быть связано с несколькими проблемами. Например, проблема с печатью может иметь три возможных решения. Также, решение для сообщения об ошибке может относиться к нескольким проблемам.
