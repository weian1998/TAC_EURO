Информационные поля контракта

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

 

Поле Описание/Действие
Номер При добавлении контракта, введите его номер. По умолчанию, номер добавляется автоматически, но его можно вести вручную. Если указан номер, который уже используется в системе, то появится предупреждающее сообщение.
Активный Установите данный флаг, чтобы указать, что контракт является активным.
Субъект Автоматически отображается название субъекта, который связан с контрактом.Для изменения субъекта, нажмите кнопку Найти. This cannot be changed once the contract is created.
Контакт Автоматически отображается имя основного контакта, связанного с контрактом. Для изменения основного контакта, нажмите кнопку Найти.
Комментарии Дополнительная информация по контракту.
Сервис Тип услуг по контракту. Для изменения, нажмите кнопку стрелку и выберите элемент из списка.
Тип контракта Для изменения, стрелку и выберите элемент из списка. Значение в данном поле влияет на остальные информационные поля. Например, Если Тип контракта Дни, тогда значение Полных дней становится доступным. После того, как контракт был создан, это поле редактировать нельзя.
Дата начала Дата начала предоставления услуг по контракту. Для изменения, нажмите кнопкуКалендарь, выберите дату, и нажмите OK.  
Начать с первой Заявки Установите данный флаг, если необходимо, чтобы действие контракта началось с момента заведения в системе первой заявки. Данное поле доступно только при Типе контракта - Дни. Если данный флаг установлен, поля Дата начала и Дата окончания будут пустыми до тех пор, пока первая заявка имеет статус Открытая. После того, как контракт был создан, это поле редактировать нельзя.
Дата окончания Дата окончания действия контракта. Для изменения, нажмите кнопкуКалендарь, выберите дату, и нажмите OK.  
Полных дней Введите количество дней, которые Сервисная служба или Служба поддержки предлагает клиенту сверх контракта. Параметр доступен для любого типа контракта.
Количество

Длительность в единицах времени, указанных в поле Тип контракта. Поле может заполняться автоматически, на основании информации полей Тип контракта, Полных дней и Начало с первой заявки, либо вводиться вручную.

Если Тип контракта - Дни, и в поле Количество указано значение, Дата окончания вычисляется автоматически с учетом Даты начала.
Остаток Даное поле содержит информацию о том, сколько осталось до окончания контракта. Система вычисляет данный параметр на основании полей тип контракта, Полных дней, Дата начала и Дата окончания. Если Заявки и дела по Заявкам уже завершены, остаток времени будет уменьшаться. Поле недоступно для редактирования.
Номер заказа При необходимости, можно ввести номер заказа.
Дата покупки Дата покупки. Для изменения, нажмите кнопкуКалендарь, выберите дату, и нажмите OK.  
Цена Сумма по контракту. Если цена не указана, система вычисляет это поле автоматически: Сумма контракта минус Скидка.
Скидка Если необходимо, введите значение скидки.
Сумма Контракта Сумма по контракту. Введите сумму контракта. Если сумма не указана, система вычисляет это поле автоматически: Сумма контракта минус Скидка.
Изменено В данном поле указывается имя пользователя, который внес последние изменения в информацию по контракту. Поле недоступно для редактирования.
Дата изменения В данном поле указывается дата последних изменений информации по контракту. Поле недоступно для редактирования.
