Закладка пакеты продукта

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Для управления пакетами продуктов, связанными с продуктом, можно использовать закладку Пакеты Продукта.

Как выполнить?

Добавить пакет продуктов

Изменить пакет продуктов

Закладка Пакеты продуктов отображает следующую информацию:

  • Имя - отображается название пакета продуктов..
  • Описание - введите описание пакета продуктов.
  • Статус - статус пакета
