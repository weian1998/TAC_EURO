Добавление Пакета Продуктов

Можно добавить пакеты в список пакетов. Этот список доступен при добавлении или редактировании продуктов сделки или продуктов заказа.

Для того чтобы добавить

  1. В меню Администрирование, выберите Новый пакет
    Откроется диалог Добавить Пакет.

  2. Заполните информационные поля пакета:

    • Имя - отображается название продукта.
    • Описание - введите описание пакета продуктов..
    • Статус - нажмите кнопку стрелку и выберите элемент из списка.
  3. Нажмите Сохранить

  4. Чтобы добавить продукты к пакету, сделайте следующее:

    1.  На детальном представлении Пакета откройте закладку Продукты и нажмите Найти.
    2.  После выбора продукта нажмите OK.
    3. При необходимости измените количестов. Выберите запись, которую необходимо изменить, произведите изменения в таблице, а затем нажмите Сохранить. Чтобы отменить изменения, нажмите Сброс до нажатия на кнопку Сохранить.
      Не все столбцы могут быть отредактированы в прямо в таблице. Чтобы определить, может ли столбец быть отредактирован, щелкните на таблице. Если ячейка стала белой, то столбец может быть отредактирован.
