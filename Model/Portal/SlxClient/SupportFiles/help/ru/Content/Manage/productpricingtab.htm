Закладка Уровни Цен Продукта

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Для просмотра или управления информацией о цене продукта можно использовать закладку Уровень Цены на детальном представлении Продукта.

Если закладка Уровень Цены не появляется в детальном представлении, нажмите Все закладки для просмотра всех доступных закладок. Выберите закладку Уровень Цены и перетащите ее в область закладок или в настраиваемую пользователем область Детального представления.

Как выполнить?

Добавить цену продукта

Изменить цену продукта

Удалить цену продукта
  1. Щелкните по ссылке Удалить рядом с необходимой записью.

  2. Нажмите ОК.

Закладка Цены отображает следующую информацию:

  • Описание - отображает описание цены.
  • Цена - Отображает цену на продукт. Эта колонка основана на типе валюты по умолчанию, определенному вашим администратором.
  • Стандартно - отображает цену на продукт по умолчанию.
