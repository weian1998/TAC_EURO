Добавление справочника

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Чтобы добавить

  1. На навигационной панели нажмите Администрирование, а затем нажмите Справочник.
  2. На Панели Общие задачи, нажмите Добавить справочник.В поле Название справочника введите название справочника, затем нажмите OK
  3. В детальном представлении Справочника установите Атрибуты справочника.
  4. Добавьте пункты к справочнику с помощью закладки Элементы.
  5. Если необходимо, нажмите кнопку Тестировать списокстрелку для просмотра справочника.
  6. Нажмите Сохранить.
