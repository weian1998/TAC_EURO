Добавление конкурента в список

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Можно добавить конкурентов в список. Этот список доступен всем пользователям при добавлении или редактировании конкурентов по сделке.

Для того чтобы добавить

  1. В меню Администрирование,выберите Добавить Конкурента.

Откроется диалог Добавить Конкурента.

  1. Заполните поля по конкуренту:
  • Конкурент - введите имя конкурента.
  • Субъект - Нажмите кнопку Найти выберите субъект с помощью поиска.
  • Контак - Нажмите кнопку Найти выберите контакт с помощью поиска. Контакт по умолчанию - основной контакт выбранного субъекта.
  • Телефон - отображает телефонный номер конкурента.
  • Адрес Web - отображает Веб Адрес конкурента.
  • Рейтинг - введите рейтинг конкурента.
  • Слабые стороны - введите слабые стороны конкурента.
  • Достоинства - введите сильные стороны конкурента.
  • Стратегия - Введите или редактируйте используемую конкурентом стратегию.
  • Заметки - Введите или редактируйте заметки о конкуренте.
  1. Нажмите Сохранить.
