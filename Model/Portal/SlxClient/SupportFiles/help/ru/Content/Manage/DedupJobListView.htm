Списочное представление истории проверки на дубликатов

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Списочное представление истории проверки на дубликатов отображает все запланированные и завершенные задачи по проверке на дубликаты. Чтобы просмотреть результаты задачи нажмите на номер задачи в списке.

Представление содержит следующие колонки:

  • Номер задачи - номер задачи.
  • Статус - статус работы по сделке.
  • Описание - описание задачи.
  • Название сущности - сущность, к которой относятся записи.
  • Описание источника - Группа, которую обрабатывала задача.
  • Стартовано - Дата и время начала задачи.
  • Пользователь - пользователь, запланировавший задачу.
  • Завершено - Дата и время завершения задачи.
  • Число записей - число записей в задаче.
  • Кол-во обработанных - число обработанных записей в задаче.
  • Кол-во дубликатов - количество дублей.

Как выполнить?

Открыть Списочное представление Проверить на историю дубликатов
  • В меню Инструменты выберите пункт Дублировать историю задачи

Используйте Поиск для нахождения записей

Открыть детальное представление Истории проверки на дубликаты
  • Нажмите номер задачи, чтобы открыть детальное представление Истории проверки на дубликаты .
