Закладка Элементы

Закладка Элементы используется для управления пунктами в списке. Эта закладка отображает Элементы, которые видят пользователи, когда открывают справочник.

Если вы используете другие приложения, использующие информацию из базы данных вместе с Sage SalesLogix, нужно использовать одни и те же пункты справочника в обоих приложениях.

 

Для управления элеменами

  1. На детальном представлении Справочника нажмите закладку Элементы.

  2. Выполните одно из следующих действий:

    • Чтобы добавить пункт, нажмите Добавить. Заполните поля в диалоговом окне Добавить элемент, затем нажмите Сохранить и Новый для продолжения добавления пунктов или нажмите OK, если вы закончили добавление пунктов.
    • Для изменения существующего пункта, нажмите Изменить. Заполните поля диалога Изменить элемент,затем нажмите OK.
    • Чтобы удалить элемент, нажмите Удалить.

  3. Нажмите ОК.
