Добавление или редактирование цен продукта

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Можно добавить или изменить информацию о цене продукта.

Как выполнить?

Добавить уровень цены

  1. Выполните одно из следующих действий:

    • Откройтеe детальное представление продукта и нажмите закладку Уровни цен.
    • Откройте форму Добавить продукт.
  2. Нажмите кнопку Добавить Цену .

  3. Добавьте следующую информацию:

    • Описание - Введите описание цены.
    • Цена - Введите цену на продукт.
    • Стандартно - определите цену на продукт по умолчанию. Можно выбрать только одну цену по умолчанию. Если вы выберите более одной, первая цена больше не будет выбрана.

  4. Нажмите ОК.

Изменить уровень цены

  1. Выполните одно из следующих действий:

    • Откройте детальное представление продукта и нажмите закладку Уровни цен.
    • Откройте форму Добавить продукт.
  2. Щелкните по ссылке Изменить рядом с необходимой записью.
  4. Измените следующую информацию:
    • Описание - Введите описание цены.
    • Цена - Введите цену на продукт.
    • Стандартно - определите цену на продукт по умолчанию. Можно выбрать только одну цену по умолчанию. Если вы выберите более одной, первая цена больше не будет выбрана.

  5. Нажмите ОК.
