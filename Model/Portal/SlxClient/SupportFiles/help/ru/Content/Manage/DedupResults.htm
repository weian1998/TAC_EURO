Детальное представление истории проверки на дубликатов

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Детальное представление истории проверки на дубликатов отображает детали задачи Мастера дубликатов. Можно просмотреть информацию о задаче и разрешить любые потенциальные дубликаты.

Как выполнить?

Открыть детальное представление Проверить на историю дубликатов
  1. В меню Инструменты выберите пункт Дублировать историю задачи.
  2. Нажмите на номер задачи в списке, чтобы открыть детальное представление.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Использовать список групп Панели Задач или общие задачи
