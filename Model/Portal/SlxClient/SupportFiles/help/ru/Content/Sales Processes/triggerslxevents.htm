Дела, которые Запускают События в Sage SalesLogix

Администратор системы управляет действиями, которые запускают события.

Чтобы использовать все свойства процессов продаж, вы должны загрузить Sage SalesLogix Desktop Integration. Вы можете установить Sage SalesLogix Desktop Integration из формы входа в Web клиент.. 
Действие Событие
Нет Этот шаг не содержит действия.
Почтовая Рассылка Запускает окна, позволяющие вам создать документ из шаблона. Это позволяет вам создавать письма, факсы и электронные сообщения. Спектр доступных для вас вариантов выбора зависит от опций, выбранных лицом, создавшим процесс продаж.
Основной Скрипт Выполняется выбранный скрипт из семейства Процессов Продаж.
Форма Запускается форма Sage SalesLogix.
Телефонный Звонок Открывает диалог Планировать Телефонный Звонок. Спектр доступных для вас вариантов выбора зависит от опций, выбранных лицом, создавшим процесс продаж.
Действие Открывает диалог Планировать Действие. Спектр доступных для вас вариантов выбора зависит от опций, выбранных лицом, создавшим процесс продаж.
Встреча Открывает диалог Планировать Встречу. Спектр доступных для вас вариантов выбора зависит от опций, выбранных лицом, создавшим процесс продаж.
Запрос Литературы Открывает диалог Запрос Литературы, повзоляющий вам удовлетворить просьбу о предоставлении литературы.
Процесс Контакта Открывает диалог Планировать Процесс.

Системный администратор может изменять справочник Типы Дел для добавления или удаления действий, которые запускают события SalesLogix.
