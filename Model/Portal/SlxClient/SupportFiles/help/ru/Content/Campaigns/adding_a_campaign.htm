Добавление маркетинговой программы

Используйте диалоговое окно Создание маркетинговой программы, чтобы добавить информацию о маркетинговой программе, выбрать цели маркетинговой программы и определить стадии и задачи маркетинговой программы.

Чтобы добавить маркетинговую программу

  1. В меню Новая, нажмите Маркетинговая программа.
    Появится диалоговое окно Создание маркетинговой программы.
  2. В поле Источник наводки нажмите на кнопку с многоточием, выберите Реклама-Общая, нажмите Добавить, затем нажмите OK.
  3. В поле Товары нажмите на кнопку с многоточием, нажмите на кнопку найти, выберите PocketPal50, нажмите Добавить, затем нажмите OK.
  4. Заполните информационные поля маркетинговой программы.
  5. Нажмите ОК.
    Вы можете добавить цели, стадии и задачи, используя диалоговое окно Создание маркетинговой программы. Однако, действия в последующих разделах описывают каким образом добавлять данные записи, после того как кампания была добавлена в Sage SalesLogix.
