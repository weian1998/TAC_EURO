Закладка цели маркетинговой программы

Закладка цели маркетинговой программы отображает списокT всех целей маркетинговой программы. При помощи данной закладки можно управлять списком целей, которые созданы для Маркетинговой Кампании (добавлять и удалять записи, выгружать их в Excel или создавать группы). Эти функции не доступны при использовании внешнего списка целей, но можно включить информацию о внешнем списке.

Как выполнить?

Открыть закладку цели маркетинговой программы
  • На Детальном представлении маркетинговой программы, откройте закладку Цели. Если записи не отобразились, нажмите Обновить.
Использовать цели из внешнего списка
  1. Установите флаг Использовать внешний список.

Если вы используете внешний список, все связанные цели будут удалены из маркетинговой программы.

  1. Нажмите OK, чтобы продолжить.
  2. Используйте следующие поля, чтобы дать информацию о списке:
  • Описание - введите описание или название списка целей используемого для маркетинговой программы.
  • Размещение - введите местоположение хранения списка. Например, C:\My Documents\Campaign Target List
  • Количество целей - введите количество целей
  1. Нажмите Сохранить.
Просмотреть записи
  • Установите флаг Отображать результат при открытии.

Удалить цель
  1. На закладке Цели Детального представления маркетинговой программы, установите флаги напротив записей, которые вы хотите удалить из маркетинговой программы.
  2. Нажмите Удалить выбранные цели.
  3. Нажмите OK.
