Фильтры целей и откликов маркетинговой программы

Используйте параметр Показать фильтры, для того чтобы ограничить целевые контакты и наводки, отображаемые в списке.

Чтобы использовать фильтр

  1. Нажмите Показать фильтры, чтобы открыть список фильтров.
  2. Установите флаги на фильтах, которые вы хотите применить.
    • Контакты - Отображает отклики контактов, которые отвечают требованиям фильтра.
    • Наводки - Отображает отклики наводок, которые отвечают требованиям фильтра.
    • Откликнувшиеся - Выберите Да, чтобы просмотреть цели, которые откликнулись или Нет, чтобы просмотреть цели, которые не откликнулись.
    • Статус - Выберите статус из выпадающего списка. Отобразятся только цели с выбранным статусом.
    • Группа - Выберите группу из выпадающего списка. Отобразятся только цели из выбранной группы.
    • Приоритет - Выберите приоритет из выпадающего списка. Отобразятся только цели с выбранным приоритетом.
    • Последняя стадия - Выберите стадию из выпадающего списка. Отобразятся только цели в рамках выбранной стадии.

    Вы можете выбрать больше одного фильтра. Например, контакты и статус, чтобы просмотреть отклики контактов с одинаковым статусом.
  3. При необходимости, нажмите кнопку Обновить, чтобы обновить таблицу.
