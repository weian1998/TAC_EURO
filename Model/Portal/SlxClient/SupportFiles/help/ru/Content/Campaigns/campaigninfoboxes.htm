Информационные поля маркетинговой программы

В таблице ниже описаны основные поля.

Поле Описание/Действие
Название маркетинговой программы Название маркетинговой программы (краткое наименование). Щелкните по полю и введите название.
Описание Краткое описание маркетинговой программы. Щелкните на поле и введите описание.
Цель Основная цель запуска маркетинговой программы (например, "Увеличение продаж в центральном регионе"). Щелкните на поле и введите цель маркетинговой программы.
Рекламный лозунг Основной промо-девиз (например, "Скидки до 50% процентов"). Щелкните на поле и введите рекламный лозунг.
Источник наводки Источник наводки маркетинговой программы. Нажмите кнопку Найти и нажмите Связать напротив источника наводки для связывания.
Статус Статус маркетинговой программы. Нажмите стрелку вниз и выберите значение из списка.
Код Уникальный идентификационный номер маркетинговой программы (например, PM12345). Можно указать любую ранее не использованную комбинацию.
Дата начала Дата начала маркетинговой программы. Нажмите кнопкуКалендарь для выбора даты, затем нажмите ОК.  
Дата окончания Дата окончания маркетинговой программы. Нажмите кнопкуКалендарь для выбора даты, затем нажмите ОК.  
Менеджер Пользователь или команда из вашей организации, которые будут отвечать за работу по маркетинговой программе. Нажмите кнопку Найти, чтобы найти пользователя или команду из вашей организации.
Доступ Пользователь или команда, которые получат доступ к маркетинговой программе. , чтобы найти пользователя или команду из вашей организации.
