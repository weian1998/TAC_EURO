Добавление/Редактирование/Завершение стадий маркетинговой программы

Стадии маркетинговой программы являются группами задач, которые могут помочь пользователям-маркетологам Sage SalesLogix управлять их маркетинговыми программами. Вы можете добавить и редактировать информацию о стадии или завершить стадию, как только она закончилась.

Как выполнить?

Добавить стадию
  1. В детальном представлении Маркетинговой программы, откройте закладку Стадии/Задачи, затем нажмите Добавить стадию. Щелкните на закладку Стадии/Задачи.
  2. Щелкните правой кнопкой мыши внутри закладки Стадии/Задачи, затем выберите пункт меню Добавить стадию.
    Появляется диалоговое окно Добавить стадию.
  3. Введите Описание для стадии.
  4. В поле Статус нажмите стрелку и выберите пункт из списка
  5. В поле Источник Наводки нажмите кнопку Найти для выбора источника наводки.
    Ваш выбор добавлен в диалоговое окно.
  6. При необходимости, измените маркетинговой программы. Нажмите OK в диалоговом окне.

    Изменение кода кампании означает, что код маркетинговой программы для данной стадии не будет больше совпадать с маркетинговой программой и другими стадиями маркетинговой программы.

  7. В поле Комментарии введите любую дополнительную информацию.
  8. При необходимости, в поле Дата начала, нажмите на кнопкуКалендарь для выбора даты, затем нажмите ОК.  
  9. В поле Дата окончания, нажмите на кнопкуКалендарь для выбора даты, затем нажмите ОК.  
  10. Добавьте задачи в таблицу Задачи.
  11. Нажмите ОК.
Редактировать стадию маркетинговой программы
  1. В детальном представлении Маркетинговой программы, откройте закладку Стадии/Задачи, затем нажмите Редактировать стадию.
    • Из диалогов Создание маркетинговой программы или Копирование маркетинговой программы выберите стадию, которую вы хотите изменить, затем нажмите Изменить.
    • В Детальном представлении маркетинговой программы, откройте закладку Стадии/Задачи, щелкните правой кнопкой мыши на стадию, которую хотите изменить, затем выберите пункт меню Изменить стадию.
  2. При необходимости, произведите изменения, затем нажмите OK.
Завершить стадию кампании
  1. Из диалогов Создание маркетинговой программы или Копирование маркетинговой программы выберите стадию, которую вы хотите завершить, затем нажмите Завершить стадию.

Откроется диалог Завершение стадии со значением статуса - "Завершенная".
