Обновление информации о целях

При помощи диалогового окна Обновление целей, можно обновить несколько целевых контактов или наводок сразу. Для обновления нескольких целей с одинаковой информацией используйте поля в нижней части диалогового окна.

Чтобы открыть диалог Обновление целей

  • На Детальном представлении маркетинговой программы, откройте закладку Цели, затем нажмите Обновить выбранные цели .

Чтобы обновить несколько целей с одинаковой информацией

  1. Установите флаги Выбрать у целей, которые вы хотите обновить, или нажмите кнопку Выбрать все для обновления всего списка целей, затем нажмите Обновить цели.
  2. Из выпадающего списка в нижней части диалога выберите информацию, которую вы бы хотели обновить. Выберите следующие варианты:
  • Стадия
    1. Выберите Стадию из выпадающего списка в нижней части диалога.
    2. Выберите значение поля Сменить на из выпадающего списка.
  • Статус
    1. Выберите Статус из выпадающего списка.
    2. Выберите значение поля Сменить на из выпадающего списка.
  • Начальная цель
    1. Выберите Начальную цель из выпадающего списка.
    2. Выберите Да, если цель была частью маркетинговой программы, при ее запуске или Нет, если цель была добавлена после запуска маркетинговой программы.
  • Добавить отклик
    1. Выберите Добавить отклик.
    2. Заполните информацию об отклике.
  1. По завершению, нажмите Обновить.
