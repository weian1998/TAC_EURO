Управление целями маркетинговой программы

Используйте диалоговое окно Поиск целей, чтобы определить какие контакты и наводки следует нацелить на маркетинговую программу.

Как выполнить?

Открыть диалоговое окно Поиск целей
  • В Детальном представлении маркетинговой программы, откройте закладку Цели, затем нажмите Добавить цели. Если закладка не отображается, то откройте закладку Все закладки.
Использовать фильтры для добавления целей в маркетинговую программу
  1. Откройте закладку Поиск целей.
  2. Нажмите кнопку Управление списком.
  3. Используйте фильтры, чтобы сузить список записей.
  4. Чтобы выяснить сколько целей совпадают с критериями фильтра перед поиском, нажмите кнопку Сколько?.
  5. Нажмите Поиск.
  6. Нажмите кнопку Добавить цели, чтобы добавить все цели из списка.
    Появятся в таблице Цели.
Использовать группы, чтобы добавить цели в маркетинговую программу
  1. Откройте закладку Добавить из группы.
  2. Выберите параметр Группа наводок или Группа контактов.
  3. Нажмите стрелку вниз и выберите значение из списка.
  4. Нажмите Поиск.

Отобразится список записей группы.

  1. Нажмите Добавить цели.

Цели из группы добавятся в маркетинговую программу.
