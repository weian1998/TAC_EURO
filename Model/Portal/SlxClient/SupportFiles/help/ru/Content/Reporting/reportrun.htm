Запуск отчета

Чтобы запустить отчет

  1. На навигационной панели, нажмите Отчеты.
  2. Из списка Выберите тип отчета, выберите отчет.
  3. Из списка Выберите название отчета, выберите отчетt. Описание отчета появится в поле ниже.
    (После выбора, Администраторы могут посмотреть для кого был выпущен выбранный отчет.)
  4. Из списка Показать записи, которые соответствуют, выберите одно из следующего:
    • Все записи - включает все записи для выбранного типа отчета. Для фильтрации нажмите Изменить.
    • Текущая группа - (Для типов отчетов Контактов, Сделок, Заявок, Дефектов и Возвратов) Включает данные только из текущей группы на главной форме. Для фильтрации нажмите Изменить.
  5. Нажмите Показать отчет. Если фильтр группы включает данные Unicode, вам будет подсказано удалить группу из фильтра либо выбрать другую группу.
    Отчет отобразится в просмотрщике Sage SalesLogix Crystal Reports.
  6. В поле Вывести отчет на, выберите Просмотр.
  7. Нажмите Сгенерировать отчет.

Для более подробной информации смотрите раздел "Фильтрация данных в отчетах" Help.
