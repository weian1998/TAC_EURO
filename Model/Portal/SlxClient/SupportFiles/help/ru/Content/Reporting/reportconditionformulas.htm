Просмотр формул условий

Администраторы, знакомые с Языком Расширенной Разметки (XML) или SQL могут просмотреть формулы условий отчета для помощи в устранении проблем. Эта опция может быть полезной для решения проблем.

Для просмотра формул условия

  1. На представлении Отчета, выберите фильтр отчета и нажмите Изменить.
  2. В разделе Условия выполните одно из следующих действий:
  • Нажмите Просмотреть XML для отображения XML-кода отчета.
  • Нажмите Просмотреть SQL для отображения SQL-кода отчета.
  1. Надмите OK.
