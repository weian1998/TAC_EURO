Что такое отчеты?

Представление Отчеты предоставляет доступ к набору предопределенных запросов к базе данных, которые представляют собой наиболее часто используемые бизнес отчеты. Доступность отчетов определяется вашим системным администратором и, как правило, соотносится с потребностями вашего бизнеса.

Dashboard представляет собой набор инструментов, который позволяет отображать и взаимодействовать с различного рода информацией. Вы можете использовать полученные данные, чтобы получить представление об организованной и индивидуальной производительности, обнаружить основные причины снижения производительности и принять корректирующие меры.

  • Поймете, что такое Dashboard
  • Регулировать содержание набора данных
  • Получить доступ к представлению Аналитика и набору фильтров

Sage SalesLogix Web отчетность позволяет вам:

  • Делиться фильтрами отчетов.
  • Использовать фильтры для ограничения данных.
  • Экспортировать и печатать отчеты.

Для открытия представления Отчеты

  • На навигационной панели нажмите Отчеты.

Для запуска Web отчетов в отсоединенном режиме, Разъединенные пользователи Веб Клиента должен быть запущен на платформе Window 2003 Server с установленным IIS. SalesLogix Web Reporting Server и Crystal Reports Server также должны быть установленны и настроены. Если эти условия не выполняются, то пользователи отсоединенных Web клиентов смогут запустить Web отчетность только, если у них есть подключение к хост-машине на которой установлен Web Reporting Server. Такие пользователи будут видеть данные в отчетах из базы данных хост-машины, а не из своей удаленной базы данных.
