Добавление или Редактирование Продукта заказчика

Можно добавить или редактировать продукт заказчика из закладки Продукты Заказа.

Нельзя добавить продукт заказчика из диалога Добавить Заказ. Используйте детальное представление заказа для добавления продуктов заказчика.

Для добавления или изменения

  1. Откройте закладку Продукты Заказа.
  2. Выполните одно из следующих действий:
    • Нажмите кнопку Добавить продукт заказчика.
    • Нажмите на кнопку Редактировать для продукта заказчика, который вы хотите изменить.
  3. В диалоге Добавить/Изменить продукт заказчика сделайте следующее:
    • Продукт - Введите название продукта заказчика.
    • Описание - Введите описание продукта.
    • Единица учета - Введите единицу учета продукта.
    • Семейство - Выберите семейство, с которым будет связан продукт.
    • Цена - Отображает цену на продукт.
    • Скидка - Введите скидку на продукт.
    • Догов.цена - Отображает базовую цену продукта, минус скидка.
    • Количество - Введите количество продукта для заказа.
    • Общая сумма - Отображает произведение цены продукта на его количество без учета скидок. Эта колонка основана на типе валюты по умолчанию, определенному вашиим администратором.
    • Фиксированная цена - Выберите, чтобы заблокировать цену на продукт. Нужно иметь соответствующую роль, чтобы иметь возможность заблокировать цену. (Только для Интеграции учета)

  4. Нажмите ОК.
