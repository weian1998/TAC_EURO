Добавление продуктов Заказа

Можно добавить продукты, используя форму Добавить продукт или можно добавить продукты заказчика используя закладку Продукты. Используйте Поиск, чтобы сузить список продуктов и пакетов. После добавления продукта, вы можете редактировать детали продукта, например, скидку, количество и установленную цену.

(Только для Интеграции учета)

  • Только продукты из выбранного прайс листа отображаются в диалоге Добавить продукт. Только один прайс лист может быть приписан к заказу на продажи, и прайс лист не может быть изменен, если продукты добавлены в заказ.
  • Нужно выбрать рабочую компанию для субъекта, перед тем как добавлять продукты к сделке или заказу.

Для того чтобы добавить

  1. Откройте форму Добавить продукт.
    • В диалоге Создать Заказ в разделе Продукты нажмите кнопку Включить Продукт.
    • На Детальном представлении Заказа выберите закладку Продукты, а затем нажмите кнопку Включить Продукт.
  2. Используйте Поиск чтобы найти и выбрать любую продукты которую требуется добавить.
  3. Нажмите Добавить выбранные, чтобы добавить продукты и остаются в диалоговом окне.
  4. После завершения добавления продуктов, нажмите кнопку Закрыть, чтобы закрыть диалоговое окно.
