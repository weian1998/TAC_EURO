Добавление Заказа

Используйте эту форму, чтобы добавить заказ, включая адрес выставления счета, адрес доставки или информацию по продукту и другую информацию по заказу.

Нельзя распечатать или отправить по электронной почте котировку, пока заказ на продажи не будет подтвержден учетной системой. (Только для Интеграции учета)

Для того чтобы добавить

  1. Выполните одно из следующих действий:
    • В меню Создать выберите Заказ.
    • В Детальном представлении Сделки, на панели задач, нажмите Добавить Заказ или на закладке Заказы, нажмите кнопку Добавить Заказ.
      Субъект сделки, продукты и другая информация автоматически копируются в новый заказ.
  2. Выберите Заказ.
  3. Добавить информацию по заказу.

    Необходимо связать субъект с заказом.
  4. (Необязательно) Редактируйте информацию об Адресе счетов. По умолчанию, поле заполняется связанным с субъектом основным контактом и основным адресом, если заказ был создан из детального представления Контакта, Субъекта или Сделки.
    1. Чтобы выбрать Имя, нажмите кнопку Найти и используйте поиск чтобы найти контакт.
    2. Рядом с полем Адрес нажмите на указывающую вниз стрелку и выберите значение из списка.
    3. Чтобы редактировать Адрес, нажмите кнопку Редактировать. Для просмотра адреса на карте нажмите кнопку MapQuest.
  5. (Необязательно) Редактируйте информацию об Адресе доставки. По умолчанию, поле заполняется связанным с субъектом основным контактом и адресом доставки, если заказ был создан из детального представления Контакта, Субъекта или Сделки.
    1. Чтобы выбрать Имя, нажмите кнопку Найти и используйте поиск чтобы найти контакт.
    2. Рядом с полем Адрес нажмите на указывающую вниз стрелку и выберите значение из списка.
    3. Чтобы редактировать Адрес, нажмите кнопку Редактировать. Для просмотра адреса на карте нажмите кнопку MapQuest.
    4. В поле Доставка нажмите стрелку и выберите элемент из списка.
  6. Добавление Продуктов.
    Раздел Продукты отображает все продукты, связанные с заказом. Если вам нужно удалить продукт, выберите запись в таблице Продукты, затем нажмите кнопку Удалить.
    Общая сумма заказа пересчитывается, когда добавляются или удаляются продукты.
  7. Нажмите Сохранить.
