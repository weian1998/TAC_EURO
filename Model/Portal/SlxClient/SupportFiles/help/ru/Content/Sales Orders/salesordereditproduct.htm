Редактирование Информации о Продукте

Используйте диалог Изменить продукт заказа, чтобы изменить информацию о продукте заказа.

Для того чтобы изменить

  1. На детальном представлении Заказа выберите закладку Продукты.
  2. Щелкните ссылку Изменить для продукта, который вы хотите изменить.

Чтобы изменить продукт в таблице, выберите продукт, который необходимо изменить, и введите информацию в нужную колонку.

  1. В диалоге Изменить продукт введите нужную информацию.
  • В поле Линия введите информацию о линии продукта.
  • В поле Описание редактируйте описание продукта.
  • В поле Семейство выберите семейство продукта.
  • В поле Уровень цены нажмите стрелку вниз и выберите используемый уровень цены.
  • В поле Скидка введите значение скидки. Число автоматически переводится в процент. Например, "50" будет отображено как "50%". Если вы введете ".5", будет отображаться ".5%".

После чего Уточненная цена продукта будет вычислена автоматически.

  • В поле Количество введите количество продукта.
  • Когда интеграция учета включена, можно изменить следующую информацию:
  • В поле Склад выберите склад продукта. (Только для Интеграции учета)
  • В поле Единица измерения окне выберите единицу для измерения количества товара. (Только для Интеграции учета)
  • Установие флаг Фиксированная цена, чтобы зафиксировать цену на продукт. (Только для Интеграции учета)
  • В поле Комментарии можно внести какие-либо замечания. (Только для Интеграции учета)
  • Используйте нижнюю часть диалога для просмотра информации о складе и хранении. (Только для Интеграции учета)
  1. Нажмите ОК.
