Закладка Заказы

Закладка Заказы отображается на Детальных представлениях Сделок и Субъектов. Вы можете использовать эту закладку для отслеживания заказов по сделке и субъекту. Вы можете добавлять, изменять и удалять заказы, связанные со сделкой или субъектом на закладке Заказы.

Представление содержит следующие колонки:

  • Номер Заказа - Отображает номер заказа.
  • Статус - Отображает статус заказа.
  • Субъект - Отображает субъект, связанный с заказом. Отображается только в Детальном представлении Сделки.
  • Контакт для отправки счетов - Отображает контакта, которому должны отсылаться счета по заказу.
  • Контакт для доставки - Отображает контакт, которому должна осуществляться доставка заказа.
  • Доставка - Отображает способ доставки.
  • Номер заказа РО - Отображает номер заказа у клиента.

Для добавления Заказа

  1. Нажмите кнопку Добавить заказ.
    Только для Детального представления Сделки. Продукты сделки и другая информация автоматически копируются в новый заказ.
  2. Следуйте инструкции по добавлению заказа.

Чтобы перейти к заказу

  • Нажмите Номер заказа который вы хотите просмотреть.

Чтобы удалить заказ

  1. Щелкните по ссылке Удалить рядом с необходимой записью.

  2. Щелкните правой кнопкой мыши заказ, который требуется удалить, а затем выберите Удалить.

  3. Нажмите ОК.
    Заказ удаляется навсегда.
