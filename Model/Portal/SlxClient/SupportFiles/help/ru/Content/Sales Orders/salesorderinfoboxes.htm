Информационные поля Заказа

В таблице ниже описаны основные поля.

Поле Описание
Заказ на продажу или Котировка Тип заказа. Чтобы перевести котировку в заказ, скопируйте котировку и поменяйте статус на заказ.
Субъект Субъект, связанный с заказом. Для поиска субъекта нажмите кнопку Поиск. Это обязательная информация.
Менеджер Субъекта Сотрудник вашей компании, ответственный за субъект. Нажмите кнопку Найти для поиска лица или команды внутри своей организации, управляющего взаимоотношениями с субъектом.
Сделка Сделка, связанная с заказом. Для того чтобы найти сделку, нажмите на кнопку Поиск.
Дата создания

Дата создания заказа.

Это поле отображается только на Детальном представлении Заказа и не может быть изменено.
Запрошено Человек, сделавший заказ. Нажмите кнопку Найти для выбора контакта.
Требуемая дата Дата, которую клиент разместил в заказе. Для изменения, нажмите кнопкуКалендарь, выберите дату, и нажмите OK.  
Обещанная Дата Обещанная дата доставки клиенту. Для изменения, нажмите кнопкуКалендарь, выберите дату, и нажмите OK.  
Дата срока платежа Дата, к которой заказ должен быть доставлен. Для изменения, нажмите кнопкуКалендарь, выберите дату, и нажмите OK.  
Номер заказа Номер заказа у клиента.
Тип Тип заказа. Тип, выбранный здесь, и тип заказа (заказ или котировка) могут быть различными значениями. Тип - это справочник, редактируемый пользователем, может содержать или не содержать значения, соответствующие типу заказа. Нажмите стрелку вниз и выберите значение из списка.
Статус Статус заказа. Нажмите стрелку и выберите элемент из списка. Если статус Заказа изменить на Закрытый, все связанные продукты будут добавлены на закладку активы субъекта.
Стоимость доставки Стоимость доставки заказа. Введите сумму в это поле.
Валюта Валюта заказа. Нажмите кнопку Найти чтобы открыть поиск и выбрать валюту.
Курс валюты Отображает курс валюты, если вы выбрали валюту.
Налог Ставка налога для применения к заказу. Введите сумму в поле.
Учетная система

Имя соединения с учетным приложением. Нажмите кнопку Найти для выбора учетной системы.

(Только для Интеграции учета)

Прайс лист

Прайс лист, связанный с заказом. Отображаются прайс листы, связанные с выбранной учетной системой. Поиск прайс листа не доступен пока не выбрана Учетная система.

Только один прайс лист может быть приписан к заказу на продажи, и прайс лист не может быть изменен, если продукты добавлены в заказ. (Только для Интеграции учета)
