Удаление Продукта из Заказа

Вы можете удалить продукты из заказа если клиент больше не заинтересован в продукте, или продукты были добавлены по ошибке. Вы можете удалять продукты из диалога Добавить Заказ или Детального представления Заказа. Удаление продукта изменяет только связь продукта с отдельным заказом. Продукт не удаляется из базы данных.

Для удаления продукта из диалога Добавить Заказ

  • В таблице Продукты нажмите Удалить рядом с продуктом, который вы хотите удалить, а затем нажмите ОК для подтверждения удаления.

Для удаления продуктов из Детального представления Заказа

  1. На детальном представлении Заказа выберите закладку Продукты. Если закладка не отображается, то откройте закладку Все закладки.
  2. Выберите запись, которую вы хотите удалить и нажмите кнопку Удалить..
