Закладка Продукты Заказа

Используйте эту закладку для добавления, редактирования и удаления продуктов из заказа.

Как выполнить?

Открыть закладку

  • Выполните одно из следующих действий:

  • На Детальном представлении Заказа нажмите закладку Продукты. Если закладка не отображается, то откройте закладку Все закладки.
  • Добавление Заказа

Добавление продукта

Добавление или редактирование продукта заказчика

Редактирование Информации о Продукте

Редактирование информации о продукте в таблице
  • Выберите запись, которую необходимо изменить, внесите изменения в таблицу, а затем нажмите Сохранить. Чтобы отменить изменения, нажмите кнопку Сброс до нажатия кнопки Сохранить.

    Не все поля могут быть отредактированы в таблице. Чтобы определить, может ли столбец быть отредактирован, щелкните таблицу. Если поле белое, то столбец может быть отредактирован.

Удаление продукта

Изменение порядка в списке
  • Нажмите кнопку Перенумировать все элементы линии.

Представление содержит следующие колонки:

  • Изменить - нажмите, чтобы изменить информацию о продукте.
    Это доступно только из Детального представления Заказа.
  • Используйте поле Заблокировано или нажмите кнопку Блокировать все пункты или Разблокировать все пункты, чтобы блокировать или разблокировать цену продукта. (Только для Интеграции учета)
  • Линия # - отображает номер линии, приписанный каждому элементу заказа.
  • Продукт - отображает название продукта.
  • Семейство - отображает семейство продуктов.
  • Уровень Цены - отображает используемый для этого продукта уровень цен.
  • Цена (Основная Валюта) - отображает цену продукта. Эта колонка основана на типе валюты по умолчанию, определенному вашиим администратором.
  • Скидка - отображает процент скидки на продукт.
  • Догов. Цена - отображает базовую цену продукта, минус скидка. Эта колонка основана на типе валюты по умолчанию, определенному вашиим администратором.
  • Количество - отображает количество продукта, добавленного в сделку. Позволяет ввести новое количество.
  • Единица - отображает единицы, используемые для измерения количества продукта. Например, "Штуки" или "Коробка из 10 шт". (Только для Интеграции учета)
  • Расширенный (Валюта Сделки) - отображает произведение цены продукта на его количество без учета скидок, в пересчете на валюту сделки. (только для режима поддержки Нескольких Валют)
  • Расширенный (Основная Валюта) - отображает произведение цены продукта на его количество без учета скидок. Эта колонка основана на типе валюты по умолчанию, определенному вашиим администратором.
