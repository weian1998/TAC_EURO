Использование списочного представлия ресурсов

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

В списочном представлении ресурсов отображаются все ресурсы, к которым вы имеете доступ. Вы можете использовать группы или поиск, чтобы сократить список отображаемых записей.

Как выполнить?

Открыть списочное представление ресурсов
  • На навигационной панели нажмите Администрирование, затем нажмите Ресурсы.
Обновить список
  • Нажмите кнопку Обновить.

Работа с записями в таблице

Отфильтровать список

Открыть детальное представление ресурсов
  • Для открытия детального представления ресурса нажмите на название ресурса.

Используйте Поиск для нахождения записей Просмотреть группы

Использовать обычные задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Сохранить записи как группу
  • Экспортировать в файл
