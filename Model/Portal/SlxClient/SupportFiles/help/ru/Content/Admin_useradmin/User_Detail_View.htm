Детальное представление пользователя

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Используйте детальное представление для просмотра, добавления или редактирования информации о выбранной записи. Детальное представление состоит из информационных полей, закладок и определяемой пользователем, средней панели.

Как выполнить?

Откройте детальное представление пользователя
  1. На навигационной панели нажмите Администрированиеа затем Пользователи 4ля открытия Списка пользователей.
  2. Нажмите на пользователя в списке, или используйте , Поиск, чтобы найти пользователя.

Информация о пользователе открывается в Детальном представлении.

Используйте Поиск для поиска записей

Открыть группу

Переключиться в режим списочного представления
  • На инструментальной панели нажмите кнопку Представление в виде списка.

Использовать общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы
  • Копировать пользователя
  • Копировать профиль
  • Переместить во Все команды

Информационные поля

Параметры Пользователя содержат основную информацию по Пользователю.

Закладки детального представления

Каждая закладка отображает специфическую информацию, связанную с профилем пользователя. Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

Клиентская система Другой календарь Календарь пользователя
Участник отдела Безопасность Члены команды пользователя
Сотрудник Сервис/Поддержка Участники команды пользователя
Заметки Участники команды  

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
