Добавление пользователя

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Перед тем как добавлять одного или более пользователей, просмотрите Новому сотруднику необходим доступ к Sage SalesLogix и проставьте первые четыре флажка. 

Вы не можете добавлять администраторов Sage SalesLogix (пользователя с именем “admin").

Чтобы добавить

  1. В меню Администратор выберите Добавить пользователей.
  2. В поле Добавить выберите тип пользователя, который вы хотите добавить.
  3. Нажмите Обновить лицензии чтобы обновить доступные номера лицензий.
  4. В окне Количество выберите число пользователей, которых вы хотите добавить.

Число пользователей, доступных для данного типа лицензии, отображается в поле Количество.

  1. Если вы хотите вручную изменить каждую вкладку в профиле пользователя, перейдите к шагу 8.
  2. Чтобы создать пользователя из шаблона или с помощью профиля существующего пользователя, выберите свойство Создать из профиля.Выберите одно из следующих действий:

    		3.

    Если шаблон или пользователь не приписаны к Стандартной роли пользователя, любые новые пользователи, созданные из профиля, не будут иметь доступ к некоторым возможностям Веб Клиента.
  • Шаблон добавляет пользователя на основе существующего шаблона пользователя. В полеШаблоны выберите имя шаблона пользователя.
  • Существующий пользователь добавляет нового пользователя на основе существующего профиля пользователя. В поле Пользователь выберите имя существующего пользователя.
  1. Выберите закладки, которые нужно скопировать из шаблона в профиль пользователя. Узнать больше...
  • Общий - Копируются общие настройки и настройки отдела. Поля Имя пользователя, Имя, Должность и E-Mail не копируются при копировании информации.
  • Сотрудник - Поля Факс, Телефон, Пейджер, PIN#, все личные поля не копируются при копировании информации.
  • Безопасность - все настройки защиты копируются.
  • Команды - все командные настройки копируются.
  • Календарь - Все настройки календаря копируются.
  • Сервис/Поддержка - Все настройки сервиса и поддержки копируются.
  • Параметры Клиента (Клиентская система) - копируются настройки владельца субъектов и основные настройки шаблона.
  1. Нажмите ОК.

При добавлении нового пользователя, автоматически приписывается код позиции.

  1. После создания нового пользователя, используйте детальное представление Пользователя для редактирования профиля пользователя.

    Вам необходимо будет предоставить имя пользователя и пароль для новых пользователей. Если вы не хотите оставлять пароль по умолчанию, измените пароль на общий пароль и проинструктируйте пользователей, как им изменить свой пароль при входе в систему Sage SalesLogix в первый раз.

  2. Нажмите Сохранить.
