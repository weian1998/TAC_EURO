Создание шаблона пользователя

Шаблоны пользователей определяют параметры, которые вы хотите использовать при создании новых пользователей. Шаблоны рекомендуются к использованию, поскольку существенно сокращают время на администрирование.

Sage SalesLogix включает несколько стандартных шаблонов пользователя. Вы можете изменить стандартный шаблон пользователя, или создать новый. Узнать больше...

Например, вы можете создать шаблон под названием Менеджер, который будет функционировать как начальный профиль для всех менеджеров вашей компании. Затем, когда при создании пользователей вы применяете шаблон, профиль уже составлен с нужными параметрами.Шаблонам пользователей не нужна пользовательская лицензия Sage SalesLogix, шаблоны не могут войти в систему.

Если шаблон не приписан к Стандартной роли пользователя, любые новые пользователи, созданные из шаблона, не будут иметь доступ к некоторым возможностям Веб Клиента.

Для создания

  1. В меню Администрирование выберите Добавить пользователей.
  2. В поле Добавить выберите Шаблон.
  3. В окне Количество выберите число шаблонов пользователей, которых вы хотите добавить.
  4. Если вы хотите вручную изменить каждую вкладку в профиле шаблона, перейдите к шагу 7.
  5. Чтобы создать шаблон из существующего шаблона или с помощью профиля существующего пользователя, выберите свойство Создать из профиля. Выберите одно из следующих действий:
  • Шаблон добавляет шаблон на основе существующего шаблона пользователя. В поле Шаблон, выберите имя шаблона пользователя.
  • Существующий пользователь добавляет нового пользователя на основе существующего профиля пользователя. В поле Пользователь, выберите имя существующего пользователя.
  1. Выберите закладки, которые нужно скопировать из шаблона или профиля пользователя. Узнать больше...
  • Общий - Копируются общие настройки и настройки отдела. Поля Имя пользователя, Имя, Должность и E-Mail не копируются при копировании информации.
  • Сотрудник - Поля Факс, Телефон, Пейджер, PIN#, все личные поля не копируются при копировании информации.
  • Безопасность - все настройки защиты копируются.
  • Команды - все командные настройки копируются.
  • Календарь - Все настройки календаря копируются.
  • Параметры Клиента (Клиентская система) - копируются настройки владельца субъектов и основные настройки шаблона.
  • Сервис/Поддержка - Все настройки сервиса и поддержки копируются.
  1. Нажмите ОК.
  2. После создания шаблона, используйте детальное представление Пользователя для редактирования профиля пользователя.

Некоторые из полей в профиле пользователя не применяются в шаблонах пользователя и будут не доступны для редактирования.
