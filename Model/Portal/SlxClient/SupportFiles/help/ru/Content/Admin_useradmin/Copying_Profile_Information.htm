Копирование информации Профиля

Можно скопировать информацию по конфигурации из существующего пользователя или шаблона другому пользователю. Копирование информации профиля от одного пользователя к другому переписывает существующие параметры. Узнать больше...

При копировании информации Календаря от одного пользователя Web к другому, пользователь, которому вы копируете информацию, может получить непредвиденные ограничения, связанные с его собственным календарем. Например, при копировании календаря пользователя А из пользователя B, безопасность календаря, которая указана в настройках пользователя B, копируется в безопасность пользователя А. Доступ пользователя А будет ограничен, если пользователь B имеет ограничения относительно календаря пользователя А. Чтобы избежать проблем, не копируйте информацию календаря. Определите доступ к календарям на закладках Другой календарь и Календарь пользователя после копирования информации пользователя.

Для копирования

  1. Откройте Детальное представление субъекта для пользователя, в чей профиль вы хотите скопировать информацию.
  2. На панели Общие задача выберите Копировать профиль.
  3. В диалоговом окне Копировать профиль выберите, хотите ли вы скопировать профиль из Существующего пользователя или Шаблона.
  4. Выберите Пользователя или Шаблон из которого вы хотите скопировать профиль.
  5. Выберите закладки, которые вы хотите копировать. Узнать больше...
  • Общий - Копируются общие настройки и настройки отдела. Поля Имя пользователя, Имя, Должность и E-Mail не копируются при копировании информации.
  • Сотрудник - Поля Факс, Телефон, Пейджер, PIN#, все личные поля не копируются при копировании информации.
  • Безопасность - все настройки защиты копируются.
  • Команды - все командные настройки копируются.
  • Календарь - Все настройки календаря копируются.
  • Параметры Клиента (Клиентская система) - копируются настройки владельца субъектов и основные настройки шаблона.
  • Сервис/Поддержка - Все настройки сервиса и поддержки копируются.
  1. В секции Целевой пользователь убедитесь, что это пользователь, чей профиль вы хотите переписать.
  2. Нажмите ОК.
  3. Нажмите OK для подтверждения.
