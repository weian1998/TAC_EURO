Использование списочного представления пользователей

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Список пользователей отображает пользователей в таблице. Таблица обеспечивает информацию по каждому пользователю, такую как должность и тип пользователя.

  • Чтобы сортировать таблицу по конкретному полю, нажмите на один из заголовков столбца.
  • Чтобы отобразить все отделы определенного типа, нажмите tab в таблице.
