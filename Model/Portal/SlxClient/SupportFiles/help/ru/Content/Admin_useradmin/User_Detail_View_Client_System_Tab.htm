Закладка Клиентская система

Используйте закладку Клиентская система на детальном представлении Пользователя, чтобы определить основные шаблоны пользователя и установить куратора субъектов по умолчанию, созданных этим пользователем. Пользователи могут изменить эти параметры на закладке Общее в Настройках .

Чтобы настроить параметры

  1. В Детальном представлении пользователя откройте закладку Клиентская система.
  2. Используйте секцию Доступ по умолчанию, чтобы установить доступ по умолчанию для новых субъектов.
    1. В поле Субъекты выберите доступ по умолчанию для новых субъектов, созданных этим пользователем.
    2. Снимите флажок Разрешить пользователю изменять эти установки чтобы запретить пользователю изменение доступа по умолчанию. Узнать больше...

    3. Если вы позволяете пользователю изменять этот параметр, владельцем субъекта может оказаться куратор, не соответствующий установленным бизнес правилам. Это может оставить субъекты за пределами установленных отчетов и запросов.

  3. Выберите Базовые шаблоны рассылки, которые вы хотите использовать при создании нового электронного письма контакту, факса или письма.
    1. В поле Тип нажмите стрелку вниз и выберите Контакт.
    2. В поле Базовый шаблон E-mail нажмите кнопку Найти .

    3. Выберите шаблон и нажмите ОК.

    4. Повторите шаги b-c для Базового шаблона письма и Базового шаблона факса.

  4. Выберите Базовые шаблоны рассылки, которые вы хотите использовать при создании нового электронного письма контакту, факса или письма.

    При выборе шаблонов для наводок, выберайте лишь те шаблоны, которые оканчиваются на "(Наводка)".

    1. В поле Тип нажмите стрелку вниз и выберите Наводка.
    2. В поле Базовый шаблон E-mail нажмите кнопку Найти .

    3. Выберите шаблон и нажмите ОК.

    4. Повторите шаги b-c для Базового шаблона письма и Базового шаблона факса.

  5. (Не обязательно) Снимите флажок Разрешить пользователю изменять эти параметры, чтобы запретить пользователю изменять шаблоны по умолчанию.

  6. Нажмите Сохранить.

При конфигурировании пользователей WebViewer, доступна только секция Куратор по умолчанию.
