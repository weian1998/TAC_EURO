Закладка Другой календарь

Используйте закладку Другой Календарь на детальном представлении Пользователя, чтобы установить права доступа для просмотра и изменения календарей других пользователей. По умолчанию, пользователи не имеют доступа к другим календарям пользователей. Когда календарь пользователя добавляется к профилю другого пользователя, доступ на просмотр устанавливается автоматически.

При определении доступа к календарю придерживайтесь следующих правил:

  • Пользователь появляется в своем списке календарей. Не удаляйте пользователя из его собственного календаря. Удаление пользователя приводит к удалению доступа к его собственному календарю.
  • Если пользователь не появляется в списке календарей, пользователь, чей профиль вы редактируете, не имеет доступа к календарю этого пользователя.
  • Если пользователь появляется в списке, пользователь, чей профиль вы редактируете, имеет доступ на просмотр календаря этого пользователя (Когда флаги Добавить, Изменить, Удалить и Синхронизировать не выбраны, пользователь имеет доступ только на чтение).
  • Если пользователь организовывает дело, другой пользователь с доступом к его календарю изменяет его, человек, который организовал дело, получает автоматическое уведомление об изменении.

Как выполнить?

Открыть закладку Другой календарь
  • На Детальном представлении пользователя перейдите к закладке Другой календарь. Если закладка не отображается, то откройте закладку Все закладки.
Разрешить доступ к календарю другого пользователя
  1. Нажмите Добавить пользователя для добавления пользователей.
  2. Используйте поиск для поиска и выбора пользователей, которых вы хотите добавить.
  3. Нажмите ОК.
  4. Используйте диалоговое окно Доступ к календарю для установки прав доступа.
  5. Нажмите ОК.
Изменение доступа к календарю другого пользователя
  1. Нажмите Изменитьрядом с пользователем, календарь которого вы хотите изменить.
  2. Используйте диалоговое окно Доступ к календарю для установки прав доступа.

  3. Нажмите ОК.

Удалить доступ к календарю другого пользователя
  1. Нажмите Удалить рядом с пользователем, календарь которого вы хотите удалить.
    Если вы удаляете пользователя, выбранного в качестве календаря по умолчанию для этого пользователя в Веб клиенте, календарь этого пользователя все еще доступен до тех пор пока новый пользователь по умолчанию не будет выбран.
  2. Нажмите ОК.
