Изменение типа пользователя

Используйте Детальное представление Пользователя для перевода пользователя из одного типа в другой. Например, можно перевести пользователя Только Веб в Конкурентного пользователя. Также можно удалить пользователя или снова активировать удаленного пользователя.

Для изменения типа пользователя, необходима доступная лицензия, за исключением увольнения пользователя. Увольнение пользователя освобождает одну лицензию, тип которой был приписан к этому пользователю.

ПРавила

Перед переводом пользователя в Уволенные пользователи, сделайте следующее:

  • Если пользователю принадлежат любые записи, проведите перегруппировку данных (в Администрировании).
  • Если пользователь является менеджером субъекта для любых субъектов, проведите перегруппировку данных (в Администрировании), чтобы приписать нового менеджера субъекта.
  • Если пользователь является менеджером, перепишите всех подчиненных ему пользователей другому менеджеру или Администратору.

Перед тем как менять тип пользователя на Пользователя Только Веб, сделайте следующее:

  • Если пользователь является менеджером, удалите эту настройку в Детальном представлении Пользователя.
  • Если другие пользователи записаны как члены команды этого пользователя удалите их.

Чтобы изменить

  1. Откройте Детальное представление пользователя которого необходимо изменить.
  2. В поле Тип пользователя выберите новый тип пользователя.
  3. Нажмите Сохранить.
