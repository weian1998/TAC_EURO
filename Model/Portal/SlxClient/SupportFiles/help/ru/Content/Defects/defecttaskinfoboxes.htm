Создание и редактирование Задач по Дефекту

Используйте поля карточки Задачи Дефекта для указания подробностей нового Дела или редактирования существующего.

Важно правильно заполнять даты Получения дефекта и завершения работ по Дефекту. Эти даты используются для вычисления стоимости работ по Дефекту для вашего клиента в соответсвии с указанным типом Дела.

В таблице ниже описаны основные поля.

Поле Описание/Действие
Дело Для изменения, нажмите стрелку и выберите элемент из списка. Click OK
Пользователь Имя пользователя, назначенного ответственным за задачу. Нажмите кнопку Найти для выбора другого пользователя.
Тип ставки Имя пользователя, назначенного ответственным за задачу. Нажмите кнопку Найти для выбора друго типа ставки.
Дата получения Дата и время когда Дело по Дефекту было добавлено. Нажмите кнопкуКалендарь, выберите дату и нажмите OK.
Дата завершения Дата когда Дело было завершено. Нажмите кнопкуКалендарь, выберите дату и нажмите OK.
Единицы времени Единицы времени, по которым рассчитывается стоимость работ по контракту Клиента.
Затрачено часов Количество минут между Датой получения и Датой завершения.
Ставка по Делам Ставка определяется из контракта Клиента на основании Типа ставки.
Всего вознаграждения Общая стоимость этого Дела. Рассчитывается автоматически на основании затраченного времени и типа ставки.
Комментарии Введите информацию в поле Комментарии.
