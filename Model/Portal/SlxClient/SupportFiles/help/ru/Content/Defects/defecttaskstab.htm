Использование закладки Задачи Дефекта

Используйте эту вкладку, чтобы просмотреть все Дела, связанные с дефектом, для добавления или изменения времени, затраченного на устранение Дефекта или изменения Дела. В качестве Дела может быть учтена любая деятельность, необходимая для обработки или разрешения Дефекта. Например, если вам нужно исследовать Дефект, добавьте это Дело здесь.

Представление содержит следующие колонки:

  • Тип Дела

  • Пользователь

  • Дата получения

  • Дата завершения

  • Тип ставки

  • Единицы времени

  • Затрачено часов

  • Всего вознаграждения

Для того чтобы открыть

  1. Открыть детальное представление Дефекта.
  2. Откройте закладку Задачи Дефекта. Если закладка не отображается, то откройте закладку Все закладки.

Для добавления Задачи

  1. Нажмите кнопку Добавить Задачу.
  2. Укажите всю необходимую информацию на форме Добавления Задачи по Дефекту.
  3. Нажмите ОК.

Для редактирования Задачи

  1. Щелкните по ссылке Редактировать рядом с необходимой записью.
  2. Внесите изменения в параметры Дефекта.
  3. Нажмите ОК.

Для удаления Задачи

  1. Выберите запись, которую вы хотите удалить, и нажмите кнопку Удалить..
  2. Нажмите Да.
