Использование списочного представления Дефектов

В списочном представлении дефектов отображаются все дефекты, к которым вы имеете доступ. Вы можете использовать группы или поиск, чтобы сократить список отображаемых записей.

Как выполнить?

Открыть списочное представление Дефектов
  • На Панели Навигации нажмите Дефекты.
Обновить список
  • Нажмите кнопку Обновить.

Используйте сводное представление

Работа с записями в таблице

Отфильтровать список

Используйте Поиск для нахождения записей

Открыть Детальное представление Дефекта
  • Для того тобы открыть детальное представление Дефекта щелкните по полю ИД дефекта.
Использовать общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Сохранить записи как группу
  • Выгрузить в файл
  • Перевести на Интерактивную панель
