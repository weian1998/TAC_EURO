Использование закладки Комментарии по Дефекту

Закладка Комментарии по Дефекту представляют собой текстовое поле для всех служебных (внутренних) заметок, связанных с дефектом.

Для того чтобы добавить

  1. Выполните одно из следующих действий:

  • Открыть детальное представление Дефекта.
  • Откройте детальное представление Контракта.
  1. Выберите закладку Комментарии. Если закладка не отображается, то откройте закладку Все закладки.
  2. Введите информацию в поле Комментарии.
