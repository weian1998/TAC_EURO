Использование закладки Продукты Дефекта

Закладка Продукты Дефекта в Детальном представлении Дефекта содержит информацию по всем продуктам, которые может затронуть данный дефект. Эта закладка используется для установки или удаления связи продукта и дефекта. Пользователи могут также использовать данную закладку для определения серьезности дефекта и действий, необходимых для его устранения.

Для того чтобы открыть

  1. Открыть детальное представление Дефекта.
  2. Открыть закладку Продукты. Если закладка не отображается, то откройте закладку Все закладки.

Чтобы связать

  1. Выберите пункт Связать.
  2. Для того чтобы выбрать нужный продукт используйте поиск.
  3. Нажмите Добавить выбранные.
     

Чтобы разорвать связь между продуктом и дефектом

  1. Выберите запись, которую вы хотите удалить, и нажмите кнопку Удалить.

  2. Нажмите Да.
