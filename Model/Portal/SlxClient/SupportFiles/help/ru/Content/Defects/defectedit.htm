Редактирование данных по Дефекту

Используйте Детальное представление для редактирования информации по дефекту. Например, можно изменить статус или ввести дополнительные комментарии с описанием действий по исправлению дефекта.

Для того чтобы изменить

  1. Открыть детальное представление Дефекта.
  2. Внесите изменения в параметры Дефекта.
  3. Нажмите Сохранить.
