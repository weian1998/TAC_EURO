Использование Детального представления Дефекта

Используйте детальное представление для просмотра, добавления или редактирования информации о выбранной записи. Детальное представление состоит из информационных полей, закладок и определяемой пользователем, средней панели.

Новый дефект можно создать на основе существующего, скопировав информацию введенную ранее. При нажатии кнопки Копировать SalesLogix копирует информацию из оригинала.

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Как выполнить?

Открыть Детальное представление Дефекта
  • Чтобы открыть Список Дефектов, на Навигационной Панели нажмите кнопку Дефекты. Для того чтобы перейти к дефекту, используйте Поиск, далее выберите ID дефекта.

Используйте Поиск для нахождения записей

Просмотреть группы

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Просмотреть или напечатать подробный отчет

Использовать общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Подробный Отчет
  • E-mail
  • Почтовая Рассылка
  • Добавить заметку
  • Новая встреча
  • Новый звонок
  • Новое действие

Информационные поля

Информационные поля содержат основную информацию по дефекту. Пользователи с соответствующими правами могут редактировать эти поля.

Закладки

Каждая закладка содержит определенную информацию, которую вы можете использовать для связанных с дефектом дел. Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
