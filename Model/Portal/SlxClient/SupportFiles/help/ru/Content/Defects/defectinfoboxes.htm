Параметры Дефекта

В таблице ниже описаны основные поля.

 

Поле Описание/Действие

ИД дефекта

(Только для Детального представления Заказов)

 Автоматически добавляет следующее значение ID при создании дефекта.
Проект Проект, с которым связан Дефект. Нажмите стрелку и выберите элемент из списка.
Имеющаяся версия Версия продукта или проекта, в котором был обнаружен дефект. Нажмите стрелку и выберите элемент из списка
Целевая версия Версия продукта, в которой ожидается исправление дефекта. Нажмите стрелку и выберите элемент из списка.
Текущий прогресс Описание процесса обработки дефекта, например, В процессе или Не обнаружен. Нажмите стрелку и выберите элемент из списка.
Исправлено в версии Версия, в которой дефект был исправлен.
Область/Категория/Проблема

ти три поля кратко описывают суть проблемы. Нажмите рядом с каждым из полей кнопку Найти, выберите соответствующие значения из списка, а затем нажмите OK.

Набор элементов списка Категория зависит от пунктов, выбранных для Области; а набор элементов списка Проблема зависит от пунктов, выбранных для Категории.

Тема

(Только для Создать Дефект)

 Введите краткое описание дефекта.

Описание

(Только для Создания Дефекта)

 Введите подробное описание проблемы.

Решение

(Только для Создания Дефекта)

 Введите шаги, необходимые для исправления дефекта.
Статус По умолчанию указывается статус Открыта. Для изменения, нажмите стрелку и выберите элемент из списка.
Источник Описание того, как была получена информация о дефекте: при помощи Заявки или отзыва продукции. Для изменения, нажмите стрелку и выберите элемент из списка.
Тип Укажите тип дефекта. Нажмите стрелку и выберите элемент из списка.
Приоритет Приоритет определяет порядок, в котором будут исправляться дефекты и ресурсы для их исправления. Дефекты с высоким приоритетом исправляются в первую очередь. Нажмите стрелку и выберите элемент из списка.
Важность Степень серьезности дефекта. Нажмитестрелку и выберите элемент из списка.
Частота Выберите частоту из справочника.
Передать для быстрого поиска

Установите этот флажок, если вы хотите чтобы информация о дефекте была доступна для поиска другими пользователями.

В зависимости от того, как ваш администратор настроил быстрый поиск, выбор этой опции может автоматически позволить информации стать доступной, или же информация требует подтверждения для использования в быстром поиске другим пользователем.
Утверждено для быстрого поиска Установите этот флажок для подтверждения того, что дефект был отправлен в быстрый поиск. Эта опция доступна если ваш администратор назначил вас лицом, уполномоченным подтверждать заявки для быстрого поиска.
Общий Доступ

Данное поле появляется в том случае, если администратор дал Вам разрешение подтверждать запросы Быстрого поиска. Выберите одно из следующих действий:

  • 5-внутренний - Дефекты включаются в индексы Быстрого поиска только для пользователей системы.

  • 1-заказчик - Дефекты включаются в индексы Быстрого поиска для доступа как пользователей системы, так и клиентов.
Поручено SalesLogix автоматически вставляет имя пользователя на основании настроек Автоназначения, выбранных в разделе Область. Для переназначения дефекта нажмите Find и осуществите поиск соответствующего пользователя и нажмите OK.
Телефон В поле указывается номер телефона назначенного пользователя.
Дата поручения Время и дата назначения дефекта для текущего назначенного пользователя. Для изменения Даты поручения нажмите кнопкуКалендарь, выберите дату и нажмите OK.
Доступ Пользователь или команда пользователей, обладающие правами доступа к информации о дефекте. Для того чтобы выбрать пользователя или команду пользователей используйте поиск.
