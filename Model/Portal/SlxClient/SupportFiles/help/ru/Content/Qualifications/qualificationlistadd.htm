Добавление или редактирование элемента квалификации

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Элементы квалификации - это критерии для проверки информации о наводках или определения, когда наводка готова перейти в сделку.

Как выполнить?

Добавить элемент квалификации
  1. На детальном представлении квалификации, откройте закладку Элементы и нажмите Добавить элемент квалификации.
  2. Внесите следующую информацию:
    • Видимый - Выставьте флаг, если вы хотите отображать элемент на детальном представлении Наводки.
    • Позиция сортировки - Введите число, представляющее порядок, в котором элемент должен появляться. Например, введите 2, если элемент должен быть вторым в списке.
    • Описание - Введите описание для элемента квалификации, которое будет отображено на детальном представлении Наводки.
    • Показать заметки - Выставьте флаг, если вы хотите, чтобы пользователи могли ввести информацию в поле заметок на детальном представлении Наводки.

  3. Нажмите ОК.

Изменить элемент квалификации
  1. На детальном представлении квалификации, откройте закладку Элементы и нажмите Изменить напротив элемента квалификации, который хотите изменить.
  2. Измените следующую информацию:
    • Видимый - Выставьте флаг, если вы хотите отображать элемент на детальном представлении Наводки.
    • Позиция сортировки - Введите число, представляющее порядок, в котором элемент должен появляться. Например, введите 2, если элемент должен быть вторым в списке.
    • Описание - Введите описание для элемента квалификации, которое будет отображено на детальном представлении Наводки.
    • Показать заметки - Выставьте флаг, если вы хотите, чтобы пользователи могли ввести информацию в поле заметок на детальном представлении Наводки.

  3. Нажмите ОК.
