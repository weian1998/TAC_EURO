Информационнные поля дела по заявке

В таблице ниже описаны основные поля.

Поле Описание/Действие
Субъект Отображает субъект, связанный с делом по заявке. Нажмите на имя субъекта, чтобы открыть детальное представление субъекта.
Контакт Отображает контакт, связанный с делом по заявке. Нажмите на имя контакта, чтобы открыть детальное представление контакта.
Заявка Отображает заявку, связанную с делом. Нажмите на название заявки, чтобы открыть детальное представление заявки.
Тип Виды работ, выполненных по заявке. Тип "Синхронизированный" отображается автоматически, если это синхронизированное дело по заявке. Чтобы изменить, нажмитестрелку и выберите элемент из списка. Нажмите OK.
Общи доступ Доступ к просмотру дела по заявке. Определяет, может ли клиент просмотреть дело по заявке в клиентском портале. Чтобы изменить, нажмитестрелку и выберите элемент из списка. Нажмите OK.
Пользователь Имя текущего пользователя. Нажмите кнопку Найти и введите значение при помощи поиска.
Дата начала Дата и время начала дела. необходимо для измерения затраченного времени по контракту с вашим кллиентом. Для изменения, нажмите кнопкуКалендарь, выберите дату, и нажмите OK.  
Дата окончания Дата и время завершения дела. Необходимо для измерения затраченного времени по контракту с вашим клиентом. Для изменения, нажмите кнопкуКалендарь, выберите дату, и нажмите OK.  
Завершенный Определяет, была ли заявка отмечена как следующее действие по делу. Указание этого флажка добавит Заявку в группу Завершенные в списочном представлении заявок.
Единицы времени Единицы времени показывают количество единиц времени, затраченных по контракту клиента. Это поле обновляется когда вы вводите дату окончания или завершаете отсчет в заявке.
Затрачено часов Затрачено часов отображает время, прошедшее между датами начала и окончания или между временем начала отсчета и временем окончания отсчета. Поле обновляется автоматически.
Тип ставки Цена, которую заплатит клиент за выполнение этого дела. Нажмите кнопку Найти и введите значение при помощи поиска. 
Курс Отображает курс по работам для выбранного способа расчета. Система отображает это число на основе выбранного типа ставки.
Всего трудозатрат Отображает полную трудовую стоимость работ по делу. Поле рассчитывается автоматически на основе значений ставки и затраченных часов.
Всего компонент Отображает полную стоимость компонент по делу. Значение рассчитывается автоматически на основе стоимости всех компонент дела.
Полная стоимость Отображает полную стоимость работ по делу. Значение поля рассчитывается автоматически на основании значений полей трудовые ресурсы и всего компонент.
Комментарии Содержат дополнительную информацию по делу. Заполните поле вручную или скопируйте текст из другого источника.
