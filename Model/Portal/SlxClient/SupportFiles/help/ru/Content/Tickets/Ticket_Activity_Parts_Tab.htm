Компоненты дела по заявке

Используйте это представление для управления компонентами дела по заявке.

Как выполнить?

Открыть закладку компоненты дела по заявке
  1. Откройте детальное представление заявки.
  2. Откройте закладку Компоненты дела по заявке. Если закладка не отображается, то откройте закладку Все закладки.

Добавить компонент

Изменить компонент

Удалить компонент
  1. Выберите запись, которую вы хотите удалить и нажмите кнопку Удалить..
Скопировать компонент
  • Выберите запись, которую хотите скопировать и нажмите кнопку Копировать компонент.

Копия выбранного компонента отобразится в таблице.

Открыть или изменить дело по заявке
  • Нажмите изменить дело по заявке напротив компонента дела по заявке, связанного с делом по заявке, которое вы хотите изменить.
    Откроется представление редактирование дела по заявке.
