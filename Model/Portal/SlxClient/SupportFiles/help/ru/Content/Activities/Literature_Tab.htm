Закладка Информационные материалы

Закладка Информационные материалы в представлении Дела отображает запросы документации по выбранному пользователю или пользователям. Литература может включать в себя каталоги, брошюры, рекламные объявления и т.д.

Как выполнить?

Просмотреть Информационные материалы
  1. Откройте представление Дела.
  2. Откройте закладку Информационные материалыа.
    Отобразится список всех запросов на инф. материалы, к которым у вас есть доступ.
  3. При необходимости, используйте фильтр, чтобы сузить список.
  4. Нажмите Просмотреть для открытие запроса на инф. материал или правой кнопкой мыши щелкните по запросу инф. материала и выберите Открыть запрос.
Показать или скрыть панель деталей
  1. Нажмите Детали.
  2. Выберите запись.
    Панель Детали отобразится в нижней части списка с подробной информацией о выбранной записи.
  1. Для того, чтобы скрыть панель Детали нажмите Скрыть детали.
Просмотреть панель Сводно
  1. Нажмите Сводно.
  2. Для возврата к списочному представлению нажмите Списокt.
Планировать запрос инф. материала

  1. Выполните одно из следующих действий:

    • На Панели задач под Общими задачами выберите Планировать Запрос.
    • Щелкните правой кнопкой мыши в гриде и нажмите Планировать запрос.
  2. Выполните следующие шаги для планирования запроса на инф. материал.
