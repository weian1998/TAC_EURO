Закладка Дела

Используйте закладку Дела для управления делами и событиями, связанными с текущим субъектом, контактом, сделкой или заявкой. Используйте закладку Дела для управления Делами и событиями связанными с текущей записью. Отображаемая на закладке Дела информация включает в себя тип дела, дату и время, продолжительность, ответственного сотрудника, пользователя, который создал запись, категорию и отметку о рассмотрении.

Первые два столбца в таблице могут содержать графические символы, которые представляют

тип деятельности и любые другие установленные параметры.

Как выполнить?

Открыть закладку Дела
  • В детальном представлении Контакта, Субъекта, Сделки, Наводки и Заявки нажмите на закладку Детали. Если закладка не отображается, то откройте закладку Все закладки.

Фильтр по моему списку

  1. Нажмите кнопку Фильтр .
  2. Выполнте одно или несколько действий:
    • Нажмите Тип на указывающую вниз стрелку и выберите значение из списка.
    • Используйте поле Диапазон дат для выбора диапазона дат в период которых происходило дело.
    • Нажмите Пользователь на указывающую вниз стрелку и выберите значение из списка.
    • В поле Тема введите один или несколько символов, содержащихся в теме. Чем больше символов вы введете, тем более короткий список данных будет отображен.
  3. Для запуска фильтрации по списку нажмите кнопку Применить или кнопку Сброс, чтобы очистить ваши параметры фильтра.
    Для того чтобы скрыть фильтрацию, повторно нажмите кнопку Фильтр.
Просмотреть дело
  • Длдя просмотра информации по делу нажмите поле Тип дела.

Планировать звонокl

  • Нажмите кнопку Планировать звонокl и выполните шаги по планированию звонка.

Планировать встречу

  • Нажмите кнопку Планировать встречу и выполните шаги для планирования встречи.

Планировать дело

  • Нажмите кнопку Планировать дело и выполните шаги для планирования дела.
Завершить дело
  1. Рядом с делом, которое вы хотите завершить, нажмите Завершить.
  2. Выполните шаги для завершения дела.

Для дел, связанных с заявками, используйте закладку Дела по заявке.
