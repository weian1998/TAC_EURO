Закладка События

Используйте закладку События в представлении Дела для просмотра запланированных событий.

Как выполнить?

Просмотреть события
  1. Откройте представление Дела и перейдите на закладку События.
  2. При необходимости, используйте фильтры, чтобы сузить список.
  3. Нажмите по типу события, чтобы открыть событие.
Просмотреть панель сводно
  1. Нажмите Сводно.
  2. Для возврата к списочному представлению нажмите Список.
Показать или скрыть панель деталей
  1. Нажмите Детали.
  2. Выберите запись.
    Панель Детали отобразится в нижней части списка с подробной информацией о выбранной записи.
  1. Для того, чтобы скрыть панель Детали нажмите Скрыть детали.
Открыть событие
  • Правой кнопкой мыши щелкните по событию, которое вы хотите открыть, и выберите Открыть событие.

Удалить событие

Планировать событие

  1. Выполните одно из следующих действий:

    • На панели задач в области Общие задачи под пунктом Планировать выберите Событие.
    • Щелкните правой кновкой мыши в гриде и выберите Планировать событие.
  2. Выполните следующие шаги для планирования события.
