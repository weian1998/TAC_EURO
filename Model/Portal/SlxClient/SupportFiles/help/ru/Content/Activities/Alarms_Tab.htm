Закладка Напоминания

Закладка Напоминания на Делах отображает список дел с заданным заренее оповещением. В этой закладке вы пожете отложить или отклонить оповещения по делам.

Для автоматической настройки оповещений для любых запланированных дел выберите меню Сервис, пункт Параметры и нажмите закладка Дела. Перейдите в секцию Тип дел по умолчанию и выберите параметры оповещения по умолчанию для каждого типа дел.

Как выполнить?

Просмотреть мои оповещения
  1. В представлении Дел нажмите Оповещения.
  2. По необходимости, используйте Фильтр, чтобы сузить список.
Показать или скрыть панель деталей
  1. Нажмите Детали.
  2. Выберите запись.
    Панель Детали отобразится в нижней части списка с подробной информацией о выбранной записи.
  1. Для того, чтобы скрыть панель Детали нажмите Скрыть детали.
Показать панель Сводной информации
  1. Нажмите Сводно.
  2. Для возврата к списочному представлению нажмите Список.
Отложить одно или несколько оповещений
  1. Выберите оповещение или оповещения, которые вы хотите отложить. Чтобы выбрать несколько записей, нажмите и удерживайте [Ctrl] или [Shift] и выделите записи..
  2. Нажмите кнопку Отложить оповещение.
  3. (Не обязательно) Для изменения времени, на которое вы хотите отложить оповещение, нажмите Отложить на, прокрутите стрелку вниз и выберите значение из списка. По умолчанию значение поля Отложить на устанавливается в параметре Отложить оповещение по умолчанию.
  4. Нажмите Отложить.
Отклонить одно или несколько оповещений
  1. Выберите оповещение или оповещения, которые вы хотите отклонить. Чтобы выбрать несколько записей, нажмите и удерживайте [Ctrl] или [Shift] и выделите записи.
  2. Нажмите кнопку Отклонить оповещение.

Использование контекстного меню

Завершить дело
  1. Выполните одно из следующих действий:
    • В Сводном представлении или на панели Детали нажмите кнопку Завершить для дела, которое вы хотите завершить.
    • Выберите дело, которое вы хотите завершить, и на Панели задач в области Общие задачи нажмите Завершить дело.
    • Щелкните правой кнопкой мыши по делу, которое вы хотите завершить и нажмите Завершить дело.
  2. Следуйте инструкции для завершения дела.

Завершение нескольких дел

Удалить дело
  1. Выберите дело, которое вы хотите удалить и на Панели задач в области Общие задачи нажмите Удалить дело.
  2. Нажмите ОК.
Удалить несколько дел
  1. Выберите дела, которые вы хотите удалить.
  2. В Панели задач в области Общие задачи нажмите Удалить дело.
  3. Нажмите ОК.
Планировать звонок

  1. Выполните одно из следующих действий:

    • На Панели задач под Общими задачами выберите Планировать,а затем нажмите Звонок.
    • Щелкните правой кнопкой мыши и выберите пункт Планировать, а затем нажмите Звонок.
  2. Следуйте инструкции для планирования звонка.
Планировать встречу

  1. Выполните одно из следующих действий:

    • На Панели задач под Общими задачами выберите Планировать,а затем нажмите Встреча.
    • Щелкните правой кнопкой мыши в гриде и выберите пункт Планировать, а затем нажмите Встреча.
  2. Следуйте инструкции для планирования встречи.
Планировать дело

  1. Выполните одно из следующих действий:

    • На Панели задач под Общими задачами выберите Планировать,а затем нажмите Дело.
    • Щелкните правой кнопкой мыши в гриде и выберите пункт Планировать, а затем нажмите Дело.
  2. Следуйте инструкции для планирования дела.
Планировать личное дело

  1. Выполните одно из следующих действий:

    • На Панели задач под Общими задачами выберите Планировать,а затем нажмите Личное дело.
    • Щелкните правой кнопкой мыши в гриде и выберите пункт Планировать, а затем нажмите Личное дело.
  2. Следуйте инструкции для планирования личного дела.
