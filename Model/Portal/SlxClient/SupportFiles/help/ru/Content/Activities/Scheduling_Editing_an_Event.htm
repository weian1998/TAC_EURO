Планирование или изменение события

События можно запланировать, изменить, удалить и просмотреть в Веб Клиенте.

Если вы используете Intellisync для SalesLogix для синхронизации дел и событий с Microsoft Outlook или Lotus Notes, дубликаты будут созданы при следующих обстоятельствах:

  • вы создаете событие в Sage SalesLogix, и
  • синхронизируете событие с Outlook или Lotus Notes, а затем
  • в Outlook или Lotus Notes, меняете событие для повторяющегося события.

Когда событие синхронизируется обратно в SalesLogix, оно появится как повторяющиеся в течение дня дело, но исходное событие также будет в календаре. Чтобы избежать путаницы, вы можете вручную удалить исходные события. Для более подробной информации смотрите тему Удаление события.

Для планирования события

  1. В меню Планировать выберите Событие.
  2. В поле Дата начала нажмитекнопкуКалендарь для выбора даты, на которую вы хотите запланировать событие, а хатем нажмитеOK.

Если вы не используете дату начала по умолчанию, как только вы установите новую дата начала, дата окончания автоматически обновится, чтобы соответствовать дате начала.

  1. В поле Дата окончания нажмите кнопкуКалендарь для выбора даты окончания события, а затем нажмите OK.
  2. В поле Тип нажмите стрелку и выберите тип события.
  3. В поле Местоположение введите расположение события.
  4. В поле Описание введите информацию о событии.
  5. В поле Пользователь нажмите кнопку Найти и используйте поиск для выберите пользователя, для которого будет запланировано событие.

  6. Нажмите ОК.

Для просмотра или изменения события из представления Дела

  1. Откройте списочное представление дел , а затем откройте События.
  2. При необходимости, используйте фильтры, чтобы сузить список.
  3. Нажмите Дата началасобытия для открытия события.
  4. При необходимости, измените информацию по событию.

  5. Нажмите ОК.
