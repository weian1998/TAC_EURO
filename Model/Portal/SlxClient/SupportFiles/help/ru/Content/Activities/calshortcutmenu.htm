Работа с контекстным меню представления Календарь

При помощи данного меню можно назначить новое дело (например, встречу) или обработать запланированные дела. Пункты меню зависят от того, какая страница загружена в данный момент.

Для планирования дела

  • В представлении Календарь откройте контекстное меню (щелчок правой кнопкой мыши).
Опции меню Описание
Планировать звонок Открывается диалоговое окно Планировать звонок .
Планировать встречу Открывается диалоговое окно Планировать встречу .
Планировать дело Открывается диалоговое окно Планировать дело.
Планировать личное дело Открывается диалоговое окно Планировать личное дело .
Планировать событие Открывается диалоговое окно Планировать событие .

Для работы с запланированным делом

  • В представлении Календарь щелкните правой кнопкой мыши по необходимому делу.
Опция меню Описание
Открыть дело Открывается диалоговое окно Изменить дело.
Завершить дело Открывается диалоговое окно Завершить дело.
Удалить дело Дело удаляется.
Перейти Позволяет открывать детальное представление любой записи, связанной с делом.

Для работы с завершенным делом

  • В представлении Календарь щелкните правой кнопкой мыши по необходимому делу.
Опция меню Описание
Изменить историю Открывается диалоговое окно детали истории.

Для работы с событием

Опция меню Описание
Открыть событие Открывается диалоговое окно Изменить дело.
Удалить событие Событие удаляется.
