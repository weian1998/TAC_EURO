Завершение нескольких дел

Вы можете завершить сразу несколько дел в представлении Дела. Можно либо завершить несколько дел сразу, либо завершать каждое дело отдельно. Если вы выбрали завершение каждого дела отдельно, откроется окно Завершить дело, позволяющее вам редактировать информацию о деле перед завершением. После того, как вы закончили дело, окно Завершить дело будет открываться для следующего выбранного дела до тех пор пока каждое из выбранных дел не будет завершено.

Ваш календарь безопасности определяет дела, которые вы можете планировать, просматривать, редактировать, завершать или удалять.

Для завершения нескольких дел одновременно

  1. В представлении Дела выберите дела, которые вы хотите завершить и на Панели задач в области Общих задач нажмите Завершить дело.
  2. Нажмите Результат стрелку и выберите элемент из списка.
  3. В поле Заметка введите любую дополнительную информацию по всем делам, которые вы хотите завершить.
  4. Нажмите Как запланировано для завершения дела, используя дату и время, введенные в поле Запланировано. Или нажмите Сейчас для завершения дела, используя текущие дату и время.

    Если любое из выбранных дел является повторяющимся событием, то будет завершено только выделенное событие или события повторяющегося дела.

Дела завершатся автоматически указанной датой.

Для завершения нескольких дел по отдельности

  1. В представлении Дела выберите дела, которые вы хотите завершить и на Панели задач в области Общих задач нажмите Завершить дело.
  2. Нажмите Результат стрелку и выберите элемент из списка.
  3. В поле Заметка введите любую дополнительную информацию по всем делам, которые вы хотите завершить.
  4. Нажмите По отдельности.
  5. Если вы выбрали повторяющееся дело, выберите Завершить это событие или Завершить все события, а затем нажмте Далее.
  6. Выполните оставшиеся шаги для завершения каждого дела, до тех пор, пока каждое дело не будет завершено.
