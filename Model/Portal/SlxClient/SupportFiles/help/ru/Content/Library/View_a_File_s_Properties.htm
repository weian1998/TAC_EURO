Свойства документов Библиотеки

Используйте диалоговое окно Свойства Документа для того чтобы добавить или изменить свойства файлов в Библиотеке. В дополнение к редактируемым полям, это диалоговое окно содержит следующую информацию только для чтения:

  • Директория: Местоположение сохраненного документа.
  • Статус: Только для удаленных пользователей. Статус может иметь следующие значения: Доступен, Заказан, Доставлен, Не прочитан, Прочитан или Исправлен.
  • Создан: Дата добавления файла к библиотеке.
  • Размер: размер файла.

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Чтобы настроить свойства документа Библиотеки

  1. На Панели Навигации нажмите Библиотека.
  2. В списке выберите документ Библиотеки, который вы хотите редактировать, нажмите кнопку Свойства файла.

  3. Выберите папку, содержащую файл, который вы хотите просмотреть.

  4. Щелкните правой кнопкой мыши на значке файла, свойства которого вы хотите просмотреть.

  5. Используйте Исправлен для изменения даты и времени, когда файл был редактирован последний раз, таким образом, при синхронизации, файл будет отправлен к Удаленным Клиентам.
    Это поле не обновляется автоматически.
  6. Уберите флажок Срок действия не ограничен чтобы активировать поле Срок действия.
    Это свойство выбрано по умолчанию.
  7. Установите дату валидации в поле Срок действия. Это свойство можно использовать для таких чувствительных ко времени элементов, как счета.
  8. Используйте Описаниедля прикрепления комментария (255 символов) к файлу.
    Описание файла появляется в детальном представлении Библиотеки.
  9. Используйте поле Обзордля прикрепления комментария (1024 символов) к файлу.
    Обзор будет доступен к просмотру только в этом диалоговом окнее.
  10. Установите свойство Принудительное распространение чтобы автоматически отправлять файлы и исправления всем Удаленным пользователям.
  11. Нажмите ОК.
