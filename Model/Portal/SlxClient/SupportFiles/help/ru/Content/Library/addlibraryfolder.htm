Добавление, Переименование и Удаление Папки Библиотеки

Для оптимизации организации можно использовать папки Библиотеки для группировки документов.

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.Только системный администратор и пользователи с ролью администратора могут иметь доступ к этим функциям.

Как выполнить?

Добавить папку библиотекиr
  1. На Панели Навигации нажмите Библиотека.
  2. Выберите папку, в которую вы хотите добавить папку, нажмите Добавить папку.
  3. В диалоговом окне Новая папка введите название новой папки.

  4. Нажмите ОК.

Переименовать папку
  1. На Панели Навигации нажмите Библиотека.
  2. Выберите папку, которую Вы хотите переименовать, и нажмите Переименовать папку.
  3. В диалоговом окне введите имя новой папки.

  4. Нажмите ОК.

Удалить папку
  1. На Панели Навигации нажмите Библиотека.
  2. Выберите папку, которую Вы хотите удалить, и нажмите Удалить папку.
  3. Нажмите Да для подтверждения удаления.

Все папки и файлы в папке будут также удалены.
