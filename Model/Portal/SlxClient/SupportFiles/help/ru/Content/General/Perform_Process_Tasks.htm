Выполнение Процессных Задач

Если вы сконфигурировали сочетание процессов, то вы можете выполнить Процессные Задачи из области Общих Задач в некоторых списочных представлениях Панели Задач.

Процессные Задачи доступны лишь, если ваш администратор сконфигурировал сочетание процессов.

Для выполнения процессных задач

  1. Откройте списочное представление.
  2. В Панеле Задач области Общие Задачи, если Процессные Задачидоступны, сделайте одно из нижеследующего:
    • Щелкните Щелкнуть для Завершениярядом с элементом для открытия и завершения.
    • Щелкните Отменить для отмены действия.
