Установка и Использование Sage SalesLogix Desktop Integration

Sage SalesLogix Desktop Integration Module - это набор функций для улучшения и упрощения работы с Sage SalesLogix.

Эти функции включают в себя:

  • Перетаскивание Вложений
  • Перетаскивание E-mail из Microsoft Outlook
  • Outlook Integration - Отправить SLX и Записать в Историю
  • Почтовая рассылка
  • Экспорт в Excel
  • Перетаскивание файлов Библиотеки(только для администарота)

Как выполнить?

Установить Desktop Integration Module, если браузером является Internet Explorer
  1. Выполните одно из следующих действий: .
    На кнопке будет отображаться надпись Улучшения установлены, если уже установлена последняя версия Desktop Integration.
    • На стартовой странице Sage SalesLogix Web Client нажмите нажмите Установить улучшения.
    • В меню Сервис выберите Параметры, откройте закладку Общее, а затемнажмите Установить улучшения.
  2. Нажмите Далее.
  3. Закройте Microsoft Outlook и любые открытые браузеры.
  4. Для установки Desktop Integration Module нажмите Да.
  5. Если появляется сообщение, что используется Microsoft Outlook или Internet Explorer, закройте обе программы и нажмите Повторить.
  6. На экране Добро пожаловать нажмите Далее.
  7. На экране Лицензионное Соглашение прочитайте информацию, примите соглашение и нажмите Далее.
  8. Выполните одно из следующих действий:
    • Выберите Complete для установки всех функций Desktop Integration по умолчанию.
    • Выберите Custom, чтобы выбрать, какие фукции Desktop Integration вы будете устанавливать или изменять при установке.
  9. Нажмите Далее.
  10. Если вы выбралиустановку Custom, то выполните одно из следующий действий и нажмите Далее:
    • Если вы не хотите устанавливатьl Office Integration (Почтовая рассылка, Экспорт в Excel, перетаскивание), нажмите стрелку вниз рядом Office Integration и выберите Эта функция будет недоступна.
    • Если вы не хотите устанавливать Microsoft Outlook Integration (Отправить SLX/Запись Истории и Desktop Manager),нажмите стрелку вниз рядом Outlook Integration и выберите Эта функция будет недоступна.
    • Чтобы изменить местоположение установки, нажмите Изменить найдите или введите нужное место установки, а затем нажмите кнопку OK.
  11. Нажмите Установить.
  12. Нажмите Готово.

Открывается Desktop Manager, позволяющий вам установить параметры Desktop Integration для Отправить SLX и Записать в историю.

Если вы не используете Outlook Integration, вам не нужно настраивать параметры в Desktop Manager.

    Чтобы вручную открыть Sage SalesLogix Desktop Manager, в разделе System Tray панели задач Windows кликните дважды иконку Sage SalesLogix Desktop Manager.
