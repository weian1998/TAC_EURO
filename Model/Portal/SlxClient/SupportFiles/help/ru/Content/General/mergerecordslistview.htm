Объединение записей

Используйте объединение записей, чтобы избежать дублей в базе данных. Вы можете объединить только те дублирующиеся субъекты или контакты, к которым имеете доступ.

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.
Для объединения из списочного представления
  1. В списке Контактов или Субъектов выберите записи, которые вы хотите объединить.
    Чтобы выбрать много записей одновременно, зажмите клавишу CTRL и щелкните каждую запись. Выбранные записи будут подсвечены.
  2. После щелчка правой кнопкой мыши выберите Объединить записи.
    Откроется диалог Объединение Записей.
  3. В верхней части таблицы, выберите запись, которая должна стать основной.
    • Источник - запись, которая конвертируется или импортируется.
    • Цель - запись SalesLogix, которая была найдена как дубликат.

      • Любые элементы, которых нет в списке, будут связаны с целью.

  4. Если необходимо, выберите вариант использования свойств неосновной записи на шаге 3.
    Например, если вы выбрали Целевую запись в качестве основной, но хотите использовать информацию о Компании из записи-источника, выберить пункт Источник рядом с полем Компания.
  5. Просмотрите все колонки Свойств чтобы убедиться в том, что правильная информация будет использована при объединении записей.
    Объединная запись будет использовать информацию только выбранной записи.
  6. Нажмите Объединить.
Чтобы объединить из окна Проверка дубликатов
  1. В окне История проверки дубликатов закладки Результаты поиска нажмите ссылку Разрешить.
  2. В форме Разрешить дубликат нажмите ссылку Объединить.

Откроется диалог Объединение Записей.

  1. В верхней части таблицы, выберите запись, которая должна стать основной.
    • Источник - запись, которая конвертируется или импортируется.
    • Цель - запись SalesLogix, которая была найдена как дубликат.

      • Любые элементы, которых нет в списке, будут связаны с целью.

  2. Если необходимо, выберите вариант использования свойств неосновной записи на шаге 3.
    Например, если вы выбрали Целевую запись в качестве основной, но хотите использовать информацию о Компании из записи-источника, выберить пункт Источник рядом с полем Компания.
  3. Просмотрите все колонки Свойств чтобы убедиться в том, что правильная информация будет использована при объединении записей.
    TОбъединная запись будет использовать информацию только выбранной записи.
  4. Нажмите Объединить.
  5. Нажмите ОК.
