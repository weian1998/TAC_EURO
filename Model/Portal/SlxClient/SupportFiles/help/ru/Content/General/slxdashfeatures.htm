Настройка вашей Персональной рабочей области

При помощи данной страницы можно просмотреть запланированные на сегодня дела, а также недавно просмотренные карточки, разделы справки и проч. В зависимости от ваших настроек пользователя, Мое рабочее пространство может быть настроено как первая страница, которая должна отображаться пользователю при регистрации в Sage SalesLogix Web Client.

Вы можете настраивать вашу персональную рабочую область добавляя и удаляя содержимое, перемещая и скрывая закладки и устанавливая закладку по-умолчанию.

Как выполнить?

Обновление содержимого в персональном рабочем пространстве
  • Нажмите для обновления нужного содержимого.
Открыть меню персонального рабочего пространства
  • На закладке рабочее пространство нажмите .
Переместить содержимое
  • Используйте метод перетаскивания для перемещения содержимое.
Удалить содержимое
  • Для удаления содержимого из закладки Рабочее пространство нажмите .
Добавить закладку
  1. Откройте меню персонального рабочего пространства.
  • На закладке Мое рабочее пространство нажмите
  1. В меню нажмите Добавить закладку.
  2. В поле Имя введите название закладки.
  3. Нажмите OK.
  4. Добавьте содержимое закладки.
  5. По необходимости, измените новую закладку Персонального рабочего пространства.
Удалить закладку
  1. В вашем персональном рабочем пространстве выберите закладку, которую вы хотите удалить.
  2. Откройте меню персонального рабочего пространства и нажмите Удалить закладку.
  3. Нажмите Да.
Добавить содержимое закладки
  1. Нажмите на закладку Персональное рабочее пространство, где нужно добавить новое содержимое.
  2. В меню Персонального рабочего пространства выберите пункт Добавить содержимое.
  3. Отметьте галочками все содержимое, которое вы хотите добавить.
  4. Нажмите OK.
Изменить внешний вид закладки
  1. Нажимите на закладку, которую хотите изменить.
  2. В меню Персонального рабочего пространства нажмите Изменить настройки.
  3. По необходимости, в поле Имя введите имя закладки.
  4. Чтобы сделать эту закладку закладкой по-умолчанию, укажите признак Сделать страницу по-умолчанию.
  5. Под полем Выберите шаблон отображенияукажите настройки использования закладки. Вы можете выбрать:
  • Разбить на две колонки
  • Разбить на три колонки
  • Две колонки, левая больше
  • Две колонки, правая больше
  • Одна колонка
  1. Нажмите OK.
