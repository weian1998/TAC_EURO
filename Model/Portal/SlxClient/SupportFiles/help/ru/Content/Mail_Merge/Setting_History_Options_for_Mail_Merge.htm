Установка параметров истории для Почтовой рассылки

Закладка Параметры истории позволяет вам создавать элементы истории для почтовой рассылки и планировать последующее дело.

Чтобы настроить параметры истории

  1. На закладке Параметры истории, в области История сделайте следующее:
    1. To attach the correspondence to each record's Чтобы прикрепить корреспонденцию к закладке Вложения контакта, выберите Вложить <e-mail/факс/письмо> для каждого <Контакта/Наводки>.
      Файл рассылки прилагается к каждой наводке или контакту, субъекту или сделке в зависимости от выбранной опции в Параметрах рассылки.
      Для Веб пользователей Sage SalesLogix, при простановке этого параметра, будет отображаться предупреждающее сообщение в течение рассылки. Эта процедура может занять длительное время.
    2. Чтобы добавить элемент истории, выберите Добавить запись истории для каждой записи <Контакта/Наводки>.
    3. В поле Результатнажмите многоточие, чтобы выбрать тип результата из списка доступных результатов.
    4. В поле Тема нажмите кнопку многоточие и выберите значение из списка .
    5. В поле Категория нажмите кнопку многоточие и выберите значение из списка.
    6. В поле Заметки введите заметки или важные элементы, которые вы хотите включить в историю.
  2. В области Планировать последующее дело сделайте следующее:
    1. Если Вы получили подтвержающее сообщения дел, нажмите OK.
    2. В списке выберите тип последующей деятельности.
    3. Для переноса заметок на последующее дело нажмите Перенос заметок.

    4. В поле Время начала, нажмите кнопку календаря, чтобы выбрать день, на который запланировано дело.

    5. Выполните одно из следующих действий:

      • Выберите флажок В течение дня, чтобы создать дело, которое произойдет на конкретную дату, но ни в определенное время и без определенноой продолжительности.

      • Нажмите Продолжительность,на указывающую вниз стрелку и выберите значение из списка.
    6. Чтобы установить оповещение для дела, нажмите Напоминание. Используйте раскрывающиеся списки, чтобы выбрать количество минут, часов или дней. Вы должны ввести значение от 1 до 240.
    7. В поле Результатнажмите многоточие, чтобы выбрать тип результата из списка доступных результатов.
    8. В области Заметки выполните одно из следующих действий:
      • Введите заметки в поле.
      • Нажмите Заметкичтобы открыть большее поле для ввода заметок, введите их и нажмите OK.
    9. В поле Приоритет выберите значение важности дела.
    10. В поле Категория нажмите кнопку многоточие и выберите значение из списка.
    11. В поле Руководитель используйте Поиск для выбора руководителя.
  3. После завершения настройки параметров Рассылки и Истоии, нажмите Рассылка.
