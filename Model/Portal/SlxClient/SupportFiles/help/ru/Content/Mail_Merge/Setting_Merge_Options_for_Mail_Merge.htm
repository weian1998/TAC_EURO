Установка параметров Почтовой рассылки

Закладка Параметры рассылки позволяет установить параметры Почтовой рассылки.

Чтобы настроить параметры рассылки

  1. На закладке Параметры рассылки, в поле Шаблон, нажмите на папку, чтобы выбрать шаблон.
    При выборе шаблонов для наводок, выберайте лишь те шаблоны, которые оканчиваются на "(Наводка)".
  2. В области Рассылка выберите параметры источника Sage SalesLogix.
      • Текущая запись - Выберите, чтобы использовать текущую запись. Информация источника может содержать контакт для субъекта, контакт, или сделку или информацию о контакте для наводки.
      • Текущая группа - Выберите, чтобы использовать группу текущего контакта, субъекта, сделки или группу наводок. Информация источника содержит имена контакта, имена субъекта, имена наводок и адреса.
      • Специальная группа - Выберите группу из списка. Информация источника содержит имена контакта, имена субъекта, имена наводок и адреса.
      • Специфика - Выберите субъект или сделку из списка и используйте поиск, чтобы найти запись. Субъект - информация источника - имена контакта, субъекта и адреса. Сделка - информация источника - имена контакта, имя субъекта и адреса, связанные со сделкой.
  3. В списке Выход выберите один из параметров и заполните информационные поля для выбранного параметра. Вывести на Принтер позволяет вам напечатать лейблы или конверты.

    • E-mail

      1. Из списка Вывод выберите E-mail.
        В поле От появляется e-mail адрес пользователя, вошедшего в систему.
      2. (Опционально) Нажмите Адресную книгу, чтобы выбрать адрес e-mail другого пользователя в качестве отправителя.
      3. (Опционально) Нажмите Адресную книгу в полях CC и BCC чтобы выбрать других получателей .
      4. В поле Тема введите информацию, которую хотите видеть в поле Тема e-mail.
      5. (Опционально) в поле Вложить нажмите стрелку рядом с папкой, затем выберите тип вложения.
      6. В поле Формат показывается формат по умолчанию выбранного вами шаблона, но вы можете выбрать другой формат.
      7. Параметр Сохранить копию e-mail в "Отправленные" выбран по умолчанию, но вы можете убрать этот параметр.
        Этот параметр сохраняет копию всех отправленных сообщений в Outlook или в Mail Client в папке Отправленные.

    • Факс

      1. Из списка Вывод выберите Факс.
      2. В поле Тема введите тему.
      3. В списке Обложка выберите одно из следующих действий:
        • Оставить появившуюся обложку по умолчанию.
        • Нажмите стрелку и выберите другую обложку.
        • Нажмите стрелку и выберите Без обложки.
      4. Если вы хотите вставить сообщение на страницу обложки, выберите одно из следующих действий:
        • Для короткого сообщения, введите сообщение в поле Сообщение.
        • Для длинного сообщения нажмите Сообщение. В диалоговом окне Изменить сообщение введите ваше сообщение, а затем нажмите OK.

    • Принтер

      1. В списке Вывод выберите Принтер. В поле появляется имя принтера по умолчанию, но вы можете изменить принтер.
      2. (Опционально) Чтобы настроить принтер, нажмите Свойства принтера.
      3. Чтобы напечатать лейблы или конверты, нажмите Создать лейблы адреса.
      4. Чтобы форматировать лейблы или конверты, нажмите Настроить лейблы.

    • Файл

      1. В списке Вывод выберите Файл.
      2. В поле Директория нажмите папку, чтобы выбрать местоположение файла.
        Файл рассылки будет иметь название в соответствии с следующим правилом: Имя шаблона (ИМЯ СУЩНОСТИ)ДАТАВРЕМЯ.DOC. В случае контактов или наводок с одинаковым именем, Sage SalesLogix добавит число к имени, если файл с таким именем уже существует.
        Пример:
        Default Letter (Keith Abbott) 07-13-2005 14.07.17PM.doc
        Default Letter (Keith Abbott2) 09-14-2005 12.11.43PM.doc
  4. Чтобы оценить, сколько бумаги или лейблов загрузить в принтер, нажмите Подсчет, чтобы просмотреть информацию о количестве рассылки.
    С помощью этой информации вы также можете выяснить, существуют ли пропущенные e-mail адреса или номера факсов перед отправкой.
  5. В области Параметры рассылкивыберите один или более параметров. Доступные параметры зависят от типа выбранного вами вывода.
    • Редактировать отдельное <e-mail/письмо/факс> перед отправкой
      • Вам нужно редактировать отдельный элемент рассылки перед отправкой. Например, можно добавить личное обращение к одному письму. При выборе этого параметра, отображается диалог Изменить документы для отправки во время процесса рассылки, позволяющий вам выбрать документы и изменить содержимое
    • Изменить шаблон для всех записей
      • Вам нужно изменить шаблон для этой рассылки. Например, вы выбрали базовый шаблон письма, но хотите добавить дополнительные поля для рассылки к шаблону. При выборе этого параметра, во время процесса рассылки откроется Редактор Шаблонов, позволяющий редактировать шаблон
    • НЕ отсылать контактам, помеченным "Не беспокоить"
      • Этот параметр выбран по умолчанию, чтобы не отправлять элементы рассылки контактам, которые попросили не присылать им рассылки. Флаг Не запрашивать есть на детальном представлении контакта.

      • Можно удалить этот параметр, чтобы отправить элементы рассылки всем контактам

  6. После завершения настройки параметров Рассылки и Истоии, нажмите Рассылка.
