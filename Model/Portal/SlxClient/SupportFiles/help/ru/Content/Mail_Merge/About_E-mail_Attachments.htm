О вложениях E-mail

Можно прикрепить файл при отправке сообщения электронной почты контакту или наводке. Если выбраны корректные параметры, файл хранится на закладке Вложения Контакта или Субъекта, и в папке Документы. Для веб пользователей Sage SalesLogix, файл хранится в папке Документы Веб сервера. На закладке Вложения, имя файла в точности совпадает с именем документа. В папке Документы, имя файла включает внутренний код, приписанный Sage SalesLogix и имя документа.

Вы можете вложить файлы Windows или файлы Библиотеки Sage SalesLogix в шаблон. Когда вы используете шаблон для рассылки электронной почты, вложенные файлы пересылаются контактам Если вы копируете шаблон, то все вложения также копируются в новый шаблон. Вы можете удалить вложение из шаблона.

Для вложения файла в шаблон
  1. Откройте Редактор шаблонов.
  2. Выполните одно из следующих действий:
  • Для вложения файла Windows, нажмите кнопку Вставить Файл, найдите местоположение файл, а затем нажмите Вставить файл.
  • Для вложения файла Библиотеки Sage SalesLogix нажмите кнопку Вставить Файл Библиотеки Sage SalesLogix. В диалоге Библиотека SalesLogix выберите файлы и нажмите Вставить.
  1. Вы можете добавить другой файл или нажать кнопку Сохранить и Закрыть.
Для удаления файла из шаблона
  1. Откройте Редактор шаблонов.
  2. В поле Вложения выберите вложение, которое вы хотите удалить.
  3. Выполните одно из следующих действий:
  • После щелчка правой кнопкой мыши выберите Удалить.
  • Нажмите клавишу Delete.
  1. Вы можете удалить другой файл или нажать кнопку Сохранить и Закрыть.
