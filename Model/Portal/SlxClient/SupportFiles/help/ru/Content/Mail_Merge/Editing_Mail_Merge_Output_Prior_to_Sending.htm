Редактирование почтовой рассылки до отправления

Можно редактировать отправляемый документ отдельным получателям перед отправкой. Например, можно добавить личное обращение к одному письму. Этот параметр находится на закладке Параметры рассылки для почтовой рассылки.

Чтобы редактировать почтовую рассылку перед отправкой

  1. В меню Написать выберите Почтовая рассылка.
  2. На закладке Параметры рассылки заполните поля Шаблон, Отправить с и Вывод.
  3. В области Параметры рассылки выберите первый параметр с названием Редактировать отдельный документ перед отправкой.
  4. Заполните остальные информационные поля, нажмите Рассылка.

Механизм рассылки начнет обрабатывать, и вам может понадобиться выбрать адрес контакта.

  1. В диалоговом окне Изменить отправляемые документы выберите документ для изменения, нажмите Открыть.
  2. В редакторе внесите изменения или добавьте обращение в документ.
  3. На панели инструментов, нажмите Сохранить и Закрыть.

  4. Опционально) Выберите другой документ, нажмите Открытьи следуйте шагам 6 и 7 для каждого документа

  5. После внесения изменений нажмите Завершить.

Отправляемые документы обрабатываются в соответствии с вашим выбором.
