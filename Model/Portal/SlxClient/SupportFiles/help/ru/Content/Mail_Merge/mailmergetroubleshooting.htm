Устранение неисправностей Desktop Integration - Почтовая Рассылка

Вы можете столкнуться с следующими проблемами Desktop Integration и почтовой рассылки.

Проблема:

“Служба почтовой рассылки не может быть загружена. Эта ошибка может возникнуть, если Sage SalesLogix Desktop Integration не установлен, либо установлен неверно.”

Возможные причины и решения:

  • Не установлен Desktop Integration Module.
  • Необходимо установить Desktop Integration Module.
  • Desktop Integration Module был установлен до того, как был установлен браузер.
  • Если вы установили текущий браузер после установки Desktop Integration Module, нужно переустановить Desktop Integration Module, используя текущий браузер.

    • Это может случиться, если вы установили Firefox после установки Desktop Integration Module с помощью Internet Explorer.

Проблема:

“Произошла ошибка в GetMailMergeService(). Эта ошибка может возникнуть, если Sage SalesLogix Desktop Integration не установлен, либо установлен неверно Определенное имя класса не может быть отнесено к COM объекту. Подробности: name=Error; line=257; source=sage-mailmerge-loader-debug.js”

Возможные причины и решения:

  • Один из файлов .dlls Почтовой Рассылки установлен неверно.

  • Необходимо переустановить Desktop Integration Module.

Проблема:

“Произошла ошибка в Sage.MailMergeService.PopulateWriteMenu(). Эта ошибка может возникнуть, если Sage SalesLogix Desktop Integration не установлен, либо установлен неверно Определенное имя класса не может быть отнесено к COM объекту. Подробности: name=SyntaxError; number=-2146827287.”

Возможные причины и решения:

  • Один из файлов .dlls Почтовой Рассылки установлен неверно
  • Необходимо переустановить Desktop Integration Module .

Проблема:

"Произошла ошибка в ExecuteWriteAction(). Объект не поддерживает это свойство или метод."

Возможные причины и решения:

  • Сайт Sage SalesLogix Web Client не относится к списку надежных сайтов.
  • Добавьте Sage SalesLogix Web Client site в список надежных сайтов браузера. Для дополнительной информации смотрите Справку Браузера.
  • Ваш браузер работает в защищенном режиме.
  • Если вы используете Internet Explorer v. 8.0, нужно отключить Защищенный Режим. Для дополнительной информации смотрите Справку Браузера.

Проблема:

"Возникла ошибка со скриптом на этой странице.Ошибка:'Ext.ux.data.Buffered.Json.Reader' is null or not an object."

Возможные причины и решения:

  • Сайт Sage SalesLogix Web Client не относится к списку надежных сайтов.
  • Добавьте Sage SalesLogix Web Client site в список надежных сайтов браузера. Для дополнительной информации смотрите Справку Браузера.
  • Ваш браузер работает в защищенном режиме.
  • Если вы используете Internet Explorer v. 8.0, нужно отключить Защищенный Режим. Для дополнительной информации смотрите Справку Браузера.
