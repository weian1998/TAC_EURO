Выбор Типа адреса

Диалог Выбрать тип адреса отображается при выполнении почтовой рассылки контакту, имеющему более одного адреса, или группе контактов, и при использовании шаблона, который содержит любые поля для рассылки из таблицы адресов.

Чтобы выбрать тип адреса

  1. В диалоге Выбрать тип адреса выберите один из следующих параметров:
    • Основной адрес контакта - Этот параметр будет использовать адрес, отмеченный “основным” для всех контактов, обрабатываемых при почтовой рассылке.
    • Адрес для доставки контакта - Этот параметр будет использовать адрес, отмеченный как “Адрес для доставки” для всех контактов, обрабатываемых при почтовой рассылке.
    • Другой адрес– Используйте стрелку чтобы выбрать список других типов адресов, приписанных контактам. Любые контакты, обрабатываемые во время почтовой рассылки, которые не содержатвыбранный тип адреса, по умолчанию будут использовать основной адрес.
  2. Нажмите ОК.

Если вы не хотите, чтобы появлялся диалог Выбрать тип адреса, но вам нужна информация об адресе в шаблоне, выберите поле адреса, связанное либо с таблицей Контакта, либо с таблицей информации о пользователе. Несмотря на тип рассылки, при использовании полей адреса, связанных либо с таблицей Контакта, либо с таблицей информации о пользователе, рассылка будет совершаться по основному адресу контакта, и диалог Выбрать тип адреса не будет отображаться.
