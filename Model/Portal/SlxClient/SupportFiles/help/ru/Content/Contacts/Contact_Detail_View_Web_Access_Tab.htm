Закладка Web-доступ в карточке детального представления Контакта

Закладка Web-доступ в карточке детального представления Контакта доступна пользователям с правами администратора и тем пользователям, у которых есть разрешение на определение прав доступа к контактам на Customer Portal. Когда контакту предоставляется доступ в систему, он может регистрироваться в Customer Portal, просматривать заявки по своему Субъекту, добавлять новые, изменять статус существующих, а также закрывать заявки. Клиенты могут просматривать и добавлять информацию только по собственным субъектам.

Перед тем, как регистрироваться в Customer Portal, клиент должен быть заведен как контакт в системе SalesLogix Client. Необходимо помнить, что имя пользователя и пароль являются чувствительными к регистру; каждый контакт должен иметь уникальное имя пользователя.

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Для получения доступа к Customer Portal

  1. Открыть детальное представление контакта.
    • На навигационной панели нажмите Контакты и используйте поиск для поиска контакта, а затем откройте детальное представление Контакта.
  2. Откройте закладку Web-Доступ. Если закладка не отображается, то откройте закладку Все закладки.
  3. Для установки доступа к Порталу заказчика, установите флаг в поле Web-доступ .
  4. В поле Имя пользователя введите Имя пользователя для данного контакта. Это Имя пользователя будет использоваться при регистрации в Портале заказчика.
  5. В поле Новый пароль введите пароль. Пароль не должен содержать более 8 символов. Клиент будет использовать данный пароль для входа в Портал заказчика.
  6. В поле Повторите новый пароль введите новый пароль еще раз.
  7. В поле Подсказка пароля введите подсказку на тот случай, если контакт забудет пароль. Подсказка должна содержать не более 64 символов.
  8. Нажмите Сохранить

Для удаления доступа к Порталу заказчика

  1. OОткрыть детальное представление контакта.
    • На навигационной панели нажмите Контакты и используйте поиск для поиска контакта, а затем откройте детальное представление Контакта.
  2. Снимите флаг Web-доступ. Удаление доступа к Порталу Заказчика не удалит контакт из базы данных, а лишь отключит возможность контакта для входа на Портал Заказчика. При снятии доступа у контакта к Порталу заказчика, введенные контактом заявки удалены не будут.
