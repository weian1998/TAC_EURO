Добавление нового Контакта в существующий Субъект

Для добавления нового контакта в существующий субъект воспользуйтесь диалоговым окном Добавить Контакт/Субъект. Поля, относящиеся к субъекту содержат данные существующего субъекта и недоступны для редактирования.

Для добавления

  1. В меню Новый нажмите Субъект/Контакт.
  2. Кликните по ссылке рядом с Использовать Существующий Субъект и найдите субъект, связанный с добавляемым контактом.
    Выбранная информация по субъекту будет отображаться в секции Информация по субъекту.

    Любая информация по субъекту, введенная до выбора существующего субъекта, заменяется.

  3. Заполните поля контактной информации, как того требует политика вашей компании.
  4. Нажмите Найти похожие записи для поиска записей с похожей информацией .
  5. Нажмите Сохранить.
