Установка контакту признака Основной

Основной контакт - это главный контакт по субъекту или по сделке. Как правило, основной контакт в организации - это контакт, который чаще всего к вам обращается.

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Для назначение основного контакта по субъекту

  1. Откройте детальное представление контакта для контакта, которого вы хотите назначить в качестве основного.
  2. Установите флаг Основной контакт субъекта.
  3. Нажмите Сохранить.

Для назначение основного контакта по сделке

  1. Откройте детальное представление сделки Для соответствующей сделки и откройте закладку Контакты.
  2. В списке контактов найдите контакта, которому вы хотите установить признак основной, и нажмите Изменить рядом с именем контакта.
  3. Установите флаг Основной и нажмите Сохранить.
