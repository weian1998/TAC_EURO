Поля информации о Контакте

В таблице ниже описаны основные поля.

Поле Описание

Имя контакта

 Полное имя контакта. Нажмите кнопку Изменить для открытия диалогового окна Изменить имя .
Субъект Субъект, к которому привязан контакт. Щелкните на имя для перехода к Детальному представлению Субъекта.
Должность Должность контакта в компании субъекта. Нажмитестрелку и выберите элемент из списка.
Помощник введите имя помощника контакта.
Уважаемый

Имя контакта для использования в приветствии в письмах, факсах и электронных письмах. Щелкните по полю и внесите изменения.

По умолчанию, поле заполняется именем контакта. Если имя контакта изменилось, информация не будет обновлена автоматически.
Адрес

Адрес местоположения контакта. Для того чтобы открыть форму Редактирования Адреса, нажмите на кнопку Изменить.

Для просмотра адреса на карте нажмите кнопку MapQuest.
Рабочий Номер рабочего телефона контакта. Щелкните по полю и внесите изменения.
Факс Номер факса контакта. Щелкните по полю и внесите изменения.
Мобильный телефон Мобильный телефон контакта. Щелкните по полю и внесите изменения.
Домашний телефон Домашний телефон контакта. Щелкните по полю и внесите изменения.
Другое AВведите какой-либо еще номер, который мог вам оставить контакт. Щелкните по полю и внесите изменения..
Предпочитаемый контакт Предпочитаемый способ связи с контактом/субъектом. Нажмите на указывающую вниз стрелку и выберите Электронная Почта, Телефон, Факс или Письмо.
E-mail Адрес электронной почты контакта. Щелкните по полю и внесите изменения. Для отправки сообщения электронной почты контакту нажмите кнопку E-mail.
Веб URL Web-адрес субъекта. Щелкните на поле и внесите изменения. Для того чтобы открыть Web-сайт контакта нажмите кнопку WWW.
Основнойконтакт для субъекта Если отмечено, то контакт назначен основным контактом для субъекта.
Контакт авторизованной службы Контакты, помеченные "Контакт авторизованной службы " могут обращаться в службу поддержки субъекта.
Не беспокоить Если выбрано, то у контакт не беспокоить. Контакты с пометкой "Не Запрашивать" исключаются из процессов Почтовой Рассылки и Маркетинговых Программ.
Не по Email, Не Звонить, Не по почте, Не по факсу Если у контакта не выбрано Не Запрашивать, то может быть выбрана любая комбинация упомянутых полей.
Куратор Лицо или команда внутри вашей организации, у которой есть доступ к субъекту.
Менеджер Лицо или команда внутри вашей организации, которая управляет взаимоотношениями с субъектом. Нажмите кнопку Найти для поиска лица или команды внутри своей организации, управляющего взаимоотношениями с субъектом
Тип контакта Категория контакта. Нажмите стрелку и выберите элемент из списка
Статус контакта Статус контакта внутри вашей организации. Нажмите стрелку и выберите элемент из списка.
