Добавление и редактирование вычисляемых полей

 Только для Системного Администратора

Для открытия диалогового окна Управление вычисляемыми полями

  1. Откройте Построитель Запросов.
  2. Нажмите кнопку Вычисления.
Для добавления
  1. Откройте Построитель запросов, нажмите на кнопку Вычисления.

Эта кнопка появляется только в том случае, если вы вошли как Администратор.

  1. Нажмите Добавить.
  2. Выберите закладку Свойства.
  3. В поле Название/Алиас введите название, которое будет определять назначение вычисления.

Имя вычисления появится только в окне Управление вычислениями.

  1. Во втором поле введите имя, используемое при вычислении. Оно не должно включать пробелы и некоторые символы. Является обязательным полем. Вы не сможете создать вычисляемое поле до тех пор, пока имя не будет определено.

Когда вычисляемое поле используют в плагинах или запросах, его имя появится в списке доступных полей.

  1. Нажмите стрелку вниз в поле Базовая таблица и выберите из списка основную таблицу, которую вы будете использовать при создании вычисляемого поля.

Если объединяются две таблицы, то это будет родительская.

  1. Нажмите стрелку раскрывающегося списка Тип вычислений и выберите тип вычислени.
  2. Введите Описание если хотите иметь подробное описание.
  3. Выбрать закладку Вычисление.
  4. Определите значения, которые вы хотите использовать, из выбранной основной таблицы. 

Таблица, выбранная на закладке Свойства, отображается в верхнем списке представления.

  1. Если основная таблица не содержит нужные значения, предпочтительнее поменять основную таблицу, чем создавать связь. Перейдите на закладку Свойства и выберите другую основную таблицу.
  2. Создать вычисление.
  3. Нажмите + или - чтобы развернуть или свернуть список таблиц и полей.
  4. Выберите поле из списка выше для добавления его в любое место вычисления.

Это первое поле в вычислении.

  1. Выполните одно из следующих действий:
  • Выберите операцию, если создаете числовое вычисление.

Вам нужно добавить операцию между полями числового вычисления.

  • Для строки, вам не нужно выбирать оператора, но вы, возможно, захотите, вставить пробел для внешнего вида.
  1. Нажмите другое поле из списка или введите целое число.
  2. Продолжайте добавлять необходимые поля, чтобы построить вычисляемое поле.
  3. Используйте скобки для группировки вычислений. Для вставки скобок, переместите курсор, где хотите поставить скобку и нажмите на кнопку с изображением скобки. Должны быть открывающаяся и закрывающаяся скобки.
  4. Нажмите ОК.
  5. Нажмите Закрыть.
Для изменения
  1. В диалоговом окне Управление Вычисляемыми полями выберите вычисляемое поле, и затем нажмите Изменить.
  2. В диалоговом окне Редактировать вычисление перейдите на закладку Вычисление.
  • Чтобы добавить поле, выберите поле в верхней части окна, чтобы добавить его в любую позицию в вычислении.
  • Чтобы добавить оператор или скобку, курсор должен находится в поле вычисления, нажмите на соответствующий оператор или скобку. Оператор должен быть между полями вычисления, а скобки должны быть открывающимися и закрывающимися.
  • Для удаления любого элемента вычисления, выделите его и нажмите на Удалить.
  1. Нажмите ОК.
  2. Нажмите Закрыть.
