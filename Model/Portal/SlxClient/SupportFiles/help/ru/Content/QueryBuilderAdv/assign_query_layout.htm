Назначение Макета Запроса

Используйте диалог Назначение Макета Запроса для задания вида каждой колонки данных (или поля) в запросе.

Для задания параметров макета

  1. Откройте Построитель Запросов и выберите закладку Макет.
  2. Добавьте или измените поле Макета.
  3. В поле Заголовок введите название для отображения в заголовке столбца.
  4. В поле Ширина укажите размер ширины столбца. По умолчанию 100.

Любые изменения ширины для группы, которые вы сделали, останутся только для текущего сеанса работы.

  1. В поле Выравнивание Заголовка выберите выравнивание заголовка столбца.
  2. В поле Выравнивание Текста выберите выравнивание данных.
  3. В поле Тип формата выберите соответствующий формат для типа данных, показываемых в этом поле.

    Тип формата Телефон форматирует значение поля в стандартный 10-значный телефонный номер. Если ваша компания не использует стандартный 10-значный формат, нужно установить тип формата на Нет, чтобы предотвратить применение любой телефонной маски.

  4. При необходимости, введите строку форматирования в поле Формат для изменения Типа Формата по умолчанию, который вы выбрали.
  5. Для связывания информации в столбце с сущностью, выберите Связанная Сущность.
  6. Для применения форматирования при помощи вложенных таблиц стиля (CSS), введите класс, который вы хотите использовать в поле Класс CSS.
