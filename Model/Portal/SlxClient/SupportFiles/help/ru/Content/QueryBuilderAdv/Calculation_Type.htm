Типы Вычисляемых Полей

Существует два типа вычислений: строковые и числовые. При задании вычисляемых полей при помощи строк (алфавитно-цифровые значения), вы объединяете или связываете значения вместе для создания одного значения. Числовые поля не могут быть использованы в строках. При построении вычисляемых полей при помощи одного или более числовых полей, вы выполняете математическую операцию (суммирование, вычитаение, умножение или деление) над значениями каждого поля для создания нового значения. Вы также можете использовать константы в числовых вычислениях. Не числовые поля (такие как строки) не могут быть использованы в числовых вычислениях.
