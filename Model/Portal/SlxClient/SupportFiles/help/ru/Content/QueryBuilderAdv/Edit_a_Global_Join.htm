Изменение Глобальных Связей

 Только для Системного Администратора

Глобальная связь является постоянной и существует для каждого сеанса в структуре дерева данных Построителя Запросов. Для предопределенных глобальных связей (системных связей) вы можете изменить только параметры Тип Связи, Видимость и Тип Каскада.

Для того чтобы изменить

  1. Откройте Построитель Запросов, а затем нажмите кнопку Глобальные Связи.

Эта кнопка появляется только в том случае, если вы вошли как Администратор.

  1. В диалоге Управление глобальными связями выберите редактируемую связь и нажмите Изменить.
  2. Измените Тип Связи, Видимость или Тип Каскада.
  3. Нажмите ОК..
