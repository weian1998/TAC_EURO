Пример: Использование Построителя Запросов для Создания Группы

Следующий пример показывает как создать группу для всех основных контактов, работающих в Отделе Продаж свих компаний.

  1. Войдите в Web клиент Sage SalesLogix.
  2. На Панели Навигации нажмите Контакты.
  3. В меню Группы выберите пункт Добавить Новую Группу.
  4. В Построителе Запросов выполните следующее на закладке Свойства:
  5. В поле Имя введите Основные Контакты Продаж.
  6. В поле Описание введите Основные контакты, которые работают в отделе Продаж.
  7. На закладке Условия выполните следующее:
  1. В левой верхней части Построителя Запросов выберите КОНТАКТ.
  2. В правой верхней части Построителя Запросов дважды щелкните Основной.

Появится диалог Назначение Условия с введенным в поле Название Поля текстом Основной , и значением Строка в поле Тип Поля.

  1. В поле Оператор выберите (=) равно.
  2. Рядом с полем Значение нажмите Просмотр, выберите T (для Истины), а затем нажмите ОК.

Оставьте выбранным Сравнение С Учетом Регистра и не выбранным Использовать Значение В Качестве Константы.

  1. Нажмите ОК.
  2. В правой верхней части Построителя Запросов дважды щелкните Департамент.

Появится диалог Назначение Условия с введенным в поле Название Поля текстом Департамент, и значением Строка в поле Тип Поля.

  1. В поле Оператор выберите содержит.
  2. Рядом с полем Значение нажмите Просмотр, выберите Продажи, а затем нажмите ОК.
  3. Снимите флажки Сравнение С Учетом Регистра и Использовать Значение В Качестве Константы.
  4. Нажмите ОК.
  1. Выберите закладку Макет. Закладка Макет показывает, какие поля будут появляться после того, как запрос будет сохранен в качестве группы. Выполните следующее:
  2. В правой части Построителя Запросов выберите поле Департамент и дважды щелкните на нем для добавления в таблицу снизу.
  1. Используйте полосу прокрутки для прокрутки вправо и щелкните Департамент.
  2. Нажимайте кнопку Влеводо тех пор, пока Департамент не появится между Субъект и Город.
  1. Выберите закладку Сортировка. По умолчанию контакты отсортированы по фамилии в алфавитном порядке (А-Я). При необходимости, вы можете добавить дополнительное условие сортировки для изменения порядка отображения.
  2. На закладке Стандартно оставьте параметры такими как есть.
  3. Нажмите ОК.

Новая группа Основные Контакты Продаж появится в списке Группа Контакты и вы сможете выбрать ее для отображения всех контактов, которые удовлетворяют всем заданным вами условиям.
