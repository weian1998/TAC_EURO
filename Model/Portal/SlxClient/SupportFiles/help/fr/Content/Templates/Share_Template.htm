Partage de modèles avec d'autres utilisateurs

Vous pouvez partager vos modèles avec d'autres utilisateurs de Sage Saleslogix, équipes ou services.

Cette fonctionnalité est uniquement disponible dans le client Web Sage SalesLogix.

Procédure

Partager un modèle
  1. Dans le menu Rédiger, sélectionnez Modèles.
  2. Ouvrez la boîte de dialogue de gestion des modèles ou de sélection d'un modèle.
    • Utilisateurs de Sage SalesLogix Network et Web : Dans le menu Rédiger, sélectionnez Modèles.
    • Utilisateurs d'Architect : Dans le menu Gestion, sélectionnez Modèles.
  3. Dans la boîte de dialogue Gestion des modèles, sélectionnez le modèle que vous souhaitez partager dans la liste.
  4. Cliquez sur Partager.
  5. Cliquez sur le bouton Ajouter.

  6. Effectuez l'une des opérations suivantes :

    • Cliquez sur l'onglet Utilisateurs. Pour limiter le nombre d'utilisateurs répertoriés, sélectionnez un type d'utilisateur dans la liste déroulante Type d'utilisateur (A distance, par exemple).
    • Cliquez sur l'onglet Services.
    • Cliquez sur l'onglet Equipes. Pour limiter le nombre d'équipes répertoriées, cochez la case Mes équipes.

    • Si vous êtes connecté en tant qu'administrateur, la case Mes équipes ne sera pas disponible.
  7. Sélectionnez dans la liste et cliquez sur OK.
Supprimer l'accès à un modèle partagé
  1. Dans le menu Rédiger, sélectionnez Modèles.
  2. Dans la boîte de dialogue Gestion des modèles, sélectionnez le modèle auquel vous souhaitez supprimer l'accès dans la liste.
  3. Cliquez sur Partager.
  4. Sélectionnez l'utilisateur, le service ou l'équipe avec qui vous ne souhaitez plus partager votre modèle.
  5. Cliquez sur le bouton Supprimer, puis sur OK.
Mettre à jour un modèle partagé

Si vous modifiez un modèle et souhaitez partager les modifications, vous devez partager de nouveau le modèle.

  1. Dans le menu Rédiger, sélectionnez Modèles.
  2. Dans la boîte de dialogue Gestion des modèles, sélectionnez le modèle que vous souhaitez partager dans la liste.
  3. Cliquez sur Partager.
  4. Dans la boîte de dialogue Partager le modèle, cliquez sur OK pour diffuser les modifications aux utilisateurs répertoriés.
