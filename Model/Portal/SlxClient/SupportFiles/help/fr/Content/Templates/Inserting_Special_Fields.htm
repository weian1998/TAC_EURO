Insertion de champs spéciaux

L'option d'insertion de champs spéciaux n'est disponible que pour les utilisateurs expérimentés.

Pour insérer une nouvelle table des produits de l'opportunité, une instruction SQL ou un champ de publipostage personnalisé, vous pouvez cliquer sur le bouton Insérer un champ spécial dans l'éditeur de modèle d'e-mail, de fax ou de lettre.

Pour insérer un champ spécial

  1. Ouvrez l'Editeur de modèles.
  2. Dans la barre d'outils, cliquez sur Insérer un champ spécial.
  3. Dans le menu, sélectionnez le type de champ à ajouter au modèle.
    • Table des produits de l'opportunité

    Ajoute une table avec des informations sur les produits de l'opportunité et sur les prix.

    • SQL

      • Vous pouvez insérer les champs SQL suivants :

      • Texte
      • Image
      • Table
      • Liste à puces
      • Liste numérotée
    • Personnalisé

      • Si un champ personnalisé est ajouté au modèle et que les scripts n'ont pas été modifiés correctement, le publipostage n'est pas exécuté tant que les scripts ne sont pas corrigés ou que le champ personnalisé n'est pas supprimé.

    Avant d'insérer un champ personnalisé, ce dernier doit être traité grâce à l'insertion d'un code dans le script SLX Mail Merge OnCustomFieldName (LAN) ou dans la fonction de classe Sage.MailMergeService.prototype.HandleOnCustomFieldName located in jscript\sage-mailmerge\sage-mailmerge-service.js.

    La boîte de dialogue Editeur des champs de publipostage SalesLogix s'ouvre.

  4. Si nécessaire, utilisez l'option de modification des champs pour effectuer vos modifications, puis cliquez sur Appliquer.
  5. Cliquez sur OK.
