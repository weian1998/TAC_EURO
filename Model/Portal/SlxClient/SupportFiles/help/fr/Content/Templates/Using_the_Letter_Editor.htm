Utilisation de l'éditeur de lettre

La fenêtre de l'Editeur d'e-mail vous permet de créer un nouveau modèle de lettre ou de modifier un modèle existant. Tout comme les éditeurs d'e-mail et de fax, l'éditeur de lettre contient une barre d'outils Sage SalesLogix qui vous permet d'accéder aux fonctions spéciales de modèles ainsi qu'à de nombreuses fonctions Microsoft Word.

La vue de l'éditeur de lettre affiche les options suivantes lors de la modification ou de la création d'un modèle :

Zone Description/Action
Insérer le champ SLX Cliquez pour afficher le menu avec les champs de fusion Sage SalesLogix les plus courants.
Insérer le fichier Cette option est uniquement disponible pour un modèle de type e-mail.
Insérer un fichier de la bibliothèque Cette option est uniquement disponible pour un modèle de type e-mail.
Enregistrer et fermer Cliquez pour enregistrer les modifications apportées au modèle et fermez l'éditeur de modèle.
Enregistrer Cliquez pour enregistrer les modifications apportées au modèle mais ne fermez pas la fenêtre de l'éditeur de modèle.
Afficher Cette option est uniquement disponible pour un modèle d'e-mail.
Format de message Cette option est uniquement disponible pour un modèle d'e-mail.
Aperçu Cliquez pour visualiser le modèle tel qu'il sera fusionné avec l'enregistrement en cours.
Publipostage SLX Cliquez pour lancer un publipostage.
Afficher le menu Word Cliquez pour afficher le menu Microsoft Word.
Modifier les champs Cliquez pour ouvrir la fenêtre de l'Editeur des champs de publipostage SalesLogix .
Insérer un champ spécial Cliquez pour ajouter une nouvelle table des produits de l'opportunité, une nouvelle instruction SQL ou un nouveau champ de publipostage personnalisé au modèle.
Options de messagerie Cette option est uniquement disponible pour un modèle d'e-mail.
