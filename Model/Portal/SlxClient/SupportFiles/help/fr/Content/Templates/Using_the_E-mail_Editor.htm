Utilisation de l'éditeur d'e-mail

La fenêtre Editeur d'e-mail vous permet de créer un nouveau modèle d'e-mail ou de modifier un modèle existant. Tout comme les éditeurs de fax et de lettre, l'éditeur d'e-mail contient une barre d'outils Sage SalesLogix qui vous permet d'accéder aux fonctions spéciales de modèles ainsi qu'à de nombreuses fonctions Microsoft Word.

La vue de l'éditeur d'e-mail affiche les options suivantes lors de la modification ou de la création d'un modèle :

Zone Description/Action
Insérer le champ SLX Cliquez pour afficher le menu avec les champs de fusion Sage SalesLogix les plus courants.
Insérer le fichier Cliquez pour ajouter un fichier Windows comme pièce jointe à cet e-mail. Cette option est uniquement disponible pour un modèle de type e-mail.
Insérer un fichier de la bibliothèque Cliquez pour ajouter des fichiers de la bibliothèque comme pièces jointes à cet e-mail. Cette option est uniquement disponible pour un modèle de type e-mail.
Enregistrer et fermer Cliquez pour enregistrer les modifications apportées au modèle et fermez l'éditeur de modèle.
Enregistrer Cliquez pour enregistrer les modifications apportées au modèle mais ne fermez pas la fenêtre de l'éditeur de modèle.
Afficher Vous permet de sélectionner les champs d'e-mail que vous souhaitez afficher dans l'éditeur : De, A, Objet et Pièces jointes. Cette option est uniquement disponible pour un modèle de type e-mail.
Format de message Cliquez sur la flèche pour choisir un format de message dans la liste. Cette option est uniquement disponible pour un modèle de type e-mail.
Aperçu Cliquez pour visualiser le modèle tel qu'il sera fusionné avec l'enregistrement en cours.
Publipostage SLX Cliquez pour lancer un publipostage.
À : Ce champ est rempli automatiquement avec l'adresse e-mail du contact et ne peut être modifié depuis cette fenêtre.
Objet Cliquez pour saisir un objet pour le modèle d' e-mail. Lorsque vous effectuez un publipostage vers ce modèle et que vous sélectionnez e-mail comme Sortie, la ligne objet du publipostage est automatiquement renseignée avec le texte de la ligne objet du modèle.
Afficher le menu Word Cliquez pour afficher le menu Microsoft Word.
Modifier les champs Cliquez pour ouvrir la fenêtre de l'Editeur des champs de publipostage SalesLogix .
Insérer un champ spécial Cliquez pour ajouter une nouvelle table des produits de l'opportunité, une nouvelle instruction SQL ou un nouveau champ de publipostage personnalisé au modèle.
Options de messagerie Cliquez pour modifier les options de messagerie telles que Répondre à, Importance et Critère de diffusion.

Lorsque vous sélectionnez un format de message pour un modèle, veuillez noter que Microsoft Word n'affiche pas les informations d'en-tête et de pied de page ou les graphiques dans une page HTML, mais qu'il enregistre les données de ceux -ci.
