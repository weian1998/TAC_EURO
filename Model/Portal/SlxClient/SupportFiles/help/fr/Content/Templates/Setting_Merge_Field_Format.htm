Définition du format du champ de fusion

Utilisez la boîte de dialogue Formater un champ SalesLogix pour définir le type de format et la chaîne de formatage pour un champ de fusion Sage SalesLogix.

Le format de champ par défaut pour la plupart des champs de texte est Aucun. Cependant, si le champ contient des données numériques, un numéro de téléphone, une devise, une valeur vrai/faux ou d'autres données, choisissez un format de champ approprié. Recherchez dans la liste Format de champ l'option correspondant à votre type de données.

Afficher des exemples
  • Si vous sélectionnez le type de format Date, vous devez taper la chaîne au format mm/jj/aaaa.
  • Le type de format Utilisateur formate un IDUTILISATEUR sous la forme "Nom, Prénom" par défaut et si vous utilisez la ChaîneFormat "FL", l'IDUTILISATEUR est au format "Prénom, Nom".

Pour définir le format de champ de fusion

  1. Dans la boîte de dialogue Formater un champ SalesLogix, sélectionnez un type dans la liste Type de format.
  2. Dans la zone Chaîne de formatage, tapez la chaîne souhaitée pour le type de format.
  3. Cliquez sur OK.
