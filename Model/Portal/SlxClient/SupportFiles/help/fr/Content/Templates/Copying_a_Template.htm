Copie d'un modèle

Vous pouvez copier des modèles publics et privés ; cependant, lorsque vous copiez un modèle public, il est enregistré en tant que modèle privé.

Si vous diffusez un modèle privé à un autre utilisateur, il est recommandé de commencer par copier ce modèle, puis de diffuser la copie du modèle à l'autre utilisateur. Cela permet de maintenir le modèle privé dans votre liste.

Pour copier un modèle

  1. Dans le menu Rédiger, sélectionnez Modèles.
  2. Dans la boîte de dialogue Gérer les modèles, sélectionnez un modèle dans la liste, puis cliquez sur Copier.
  3. Dans la boîte de dialogue de propriétés, définissez les propriétés du modèle, puis cliquez sur OK.
    Le modèle s'ouvre dans l'éditeur.
  4. Effectuez l'une des opérations suivantes :
    • Si aucune modification n'est nécessaire, sur la barre d'outils Sage SalesLogix, cliquez sur Enregistrer et fermer.
    • Si des modifications sont nécessaires, utilisez la barre d'outils Sage SalesLogix pour ajouter des champs, afficher un aperçu du modèle ou effectuer d'autres opérations.
