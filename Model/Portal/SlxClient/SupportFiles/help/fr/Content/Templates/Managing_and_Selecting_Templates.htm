Gestion et sélection des modèles

La boîte de dialogue Gestion des modèles ou Sélectionner un modèle vous permet d'accéder aux modèles publics et privés disponibles.

Lors de la sélection de modèles de leads, sélectionnez uniquement ceux qui se terminent par "(Lead)".

Procédure

Ouvrez la boîte de dialogue de gestion des modèles ou de sélection d'un modèle.
  • Dans le menu Rédiger, sélectionnez Modèles.

Modifier le type de modèle d'un modèle privé
  1. Dans la liste Modèles privés, cliquez sur le dossier Lettre, Fax ou E-mail pour développer la liste des modèles.
  2. Cliquez avec le bouton droit de la souris sur le modèle à modifier. Dans le menu contextuel, pointez sur Type de modèle, puis cliquez sur l'un des types proposés.
    Les types par défaut sont les suivants : Lettre, Fax, E-mail.

Prévisualiser un modèle

  1. Sélectionnez un modèle dans la liste, puis cliquez sur Afficher aperçu.
    Une fenêtre secondaire s'ouvre avec le texte Objet, les informations de pièces jointes et le corps du modèle.
  2. Pour afficher un aperçu du modèle avec les données fusionnées, cochez la case Afficher le modèle avec les données fusionnées.
  3. Pour masquer l'aperçu, cliquez sur Masquer l'aperçu.

Pour accéder au menu contextuel (Réduire tout, Développer tout, Nouveau, Modifier, Propriétés, Copier, Supprimer et Type de modèle), cliquez avec le bouton droit de la souris sur le nom du modèle.

Ajouter un nouveau dossier de modèles
  • Un administrateur système peut ajouter un dossier de modèles prsonnalisé en ajoutant un nouvel élément à la liste de sélection Types de modèles. Pour plus d'informations, veuillez contacter votre administrateur.
  • Administrateurs système - Consultez « Ajouter un élément à une liste de sélection » dans l'aide du module Administrateur ou dans le client Web Sage SalesLogix, consultez la rubrique « Onglet Eléments » de la liste de sélection pour des étapes détaillées d'ajout d'un élément dans une liste de sélection.
