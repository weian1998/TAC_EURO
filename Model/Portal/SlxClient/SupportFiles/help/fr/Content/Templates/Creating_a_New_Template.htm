Création d'un modèle

L'éditeur de modèle vous permet de créer un modèle privé.

Pour créer un nouveau modèle

  1. Dans le menu Rédiger, sélectionnez Gérer les modèles.
  2. Dans le menu Rédiger, sélectionnez Modèles.
  3. Dans la boîte de dialogue Gestion des modèles, cliquez sur Nouveau.
  4. Dans la boîte de dialogue de propriétés, définissez les propriétés du modèle, puis cliquez sur OK.
    Un exemple de modèle type est ouvert dans l'éditeur. Par exemple, si vous créez un modèle de lettre, un exemple de lettre type apparaît dans l'éditeur.
    Vous pouvez taper ou supprimer un contenu, ou encore coller du texte provenant d'un autre document.
  5. Pour ajouter des champs de fusion SalesLogix au modèle, placez le curseur dans le modèle à l'endroit voulu pour le champ de fusion, cliquez sur Insérer le champ SLX, puis sélectionnez un champ dans la liste des champs de fusion.
    Le nouveau champ de fusion apparaît dans le modèle.
  6. Vous pouvez ajouter ou supprimer des champs de fusion selon vos besoins.
  7. Pour enregistrer le modèle sans fermer l'éditeur, cliquez sur Enregistrer.
  8. Dans l'éditeur, utilisez la barre d'outils Sage SalesLogix pour prévisualiser le modèle fusionné, joindre des fichiers ou effectuer d'autres opérations.
  9. Lorsque vous avez terminé, cliquez sur Enregistrer et fermer pour enregistrer le modèle et fermer l'éditeur.
