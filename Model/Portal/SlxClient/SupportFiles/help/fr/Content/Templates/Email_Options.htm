Options d'e-mail

Lorsque vous créez un modèle d'e-mail, vous pouvez définir les options des messages e-mail à envoyer. Par exemple, en tant qu'administrateur, il se peut que vous ayez besoin d'envoyer un e-mail en masse au nom de votre équipe commerciale à partir d'une adresse de société générique (c'est-à-dire, equipecommerciale@societe.com). Le champ Répondre à dans l'e-mail sortant peut être complété de manière dynamique au cours du publipostage : ainsi, lorsque les destinataires répondent à cet e-mail reçu, leur réponse est automatiquement adressée à leur responsable de compte respectif.

Le bouton Options de messagerie n'est disponible que dans l'éditeur d'e-mail.

Pour définir les options de messagerie

  1. Sélectionnez un modèle d'e-mail, puis ouvrez l'éditeur de modèle.
  2. Cliquez sur Options de messagerie dans la barre d'outils.
  3. Dans la zone Répondre à, sélectionnez le nom de la personne à afficher dans le champ A de l'e-mail de réponse. Cliquez sur le bouton du Carnet d'adresses pour sélectionner un nom spécifique dans le Carnet d'adresses MS Outlook ou, dans le menu déroulant du champ, sélectionnez l'une des options suivantes :
  • <<Utilisateur actuel>> : cette option insère la chaîne <<Utilisateur actuel>> dans le champ Répondre à et désigne l'utilisateur actuel comme l'utilisateur du publipostage. Il s'agit de l'option par défaut.
  • <<Responsable du compte>> : cette option insère la chaîne <<Responsable du compte>> dans le champ Répondre à et désigne l'utilisateur en tant que responsable du compte du contact correspondant. Cette option est disponible pour MS Outlook uniquement.
  • Utilisateur spécifique : cette option ouvre le Carnet d'adresses MS Outlook et affiche le nom d'affichage MS Outlook sélectionné dans le champ Répondre à, désignant l'utilisateur comme l'utilisateur sélectionné dans le Carnet d'adresses. Cette option est disponible pour MS Outlook uniquement.
  1. Dans la liste Importance, sélectionnez Faible, Normale (par défaut) ou Haute.
  2. Pour communiquer le critère de diffusion d'un e-mail, sélectionnez Normal (par défaut), Personnel, Privé ou Confidentiel dans la liste Critère de diffusion.
  3. Pour recevoir un avis de confirmation d'envoi de l'e-mail, sélectionnez Demander un accusé de réception.
  4. Pour recevoir un avis indiquant que l'e-mail a été lu, sélectionnez Demander un accusé de lecture.
  5. Cliquez sur OK.
