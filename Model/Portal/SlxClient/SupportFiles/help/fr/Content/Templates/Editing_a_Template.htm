Modification d'un modèle

Vous pouvez modifier des modèles publics et privés ; cependant, pour modifier un modèle public, copiez-le, puis enregistrez-le en tant que modèle privé.

Si vous modifiez le modèle lors du publipostage, lorsque vous avez terminé vos modifications, vous devez enregistrer et fermer l'éditeur pour continuer la fusion.

Pour modifier un modèle

  1. Dans le menu Rédiger, sélectionnez Modèles.
  2. Dans la boîte de dialogue Gestion des modèles, sélectionnez un modèle dans la liste Modèles privés ou publics, puis cliquez sur Modifier.
  3. Si vous modifiez un modèle public, à l'affichage du message Avertissement, cliquez sur Oui pour copier ce modèle.
  4. Dans la boîte de dialogue de propriétés, effectuez l'une des opérations suivantes :
    • Définissez les propriétés du modèle, puis cliquez sur OK.
    • Laissez les propriétés telles quelles, puis cliquez sur Fermer.
      L'éditeur s'affiche avec le texte standard du modèle et les champs de fusion visibles.
  5. Pour modifier le modèle, vous pouvez taper un complément de texte dans le modèle ou utiliser la barre d'outils Sage SalesLogix pour :
    • insérer des champs de fusion Sage SalesLogix dans le modèle ;
    • prévisualiser le modèle.
    • Attacher des fichiers au modèle si c'est un modèle de type e-mail
  6. Une fois que toutes les modifications sont effectuées, sur la barre d'outils Sage SalesLogix, cliquez sur Enregistrer et fermer.
