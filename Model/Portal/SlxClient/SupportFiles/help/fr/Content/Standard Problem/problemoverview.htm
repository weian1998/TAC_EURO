Que sont les problèmes et résolutions standard ?

Un problème standard est un problème sur un produit qui génère des messages de nombreux clients. Votre centre d'appel peut fréquement recevoir des appels pour des problèmes similaires. Utilisez les Problèmes standard pour enregistrer les descriptions de ces problèmes, et les Résolutions standard pour enregistrer les moyens démontrés par lesquels ils peuvent être résolus. Une fois que les problèmes standard et les résolutions standard sont définis, ils peuvent être utilisés pour compléter une description de ticket ou utilisés comme ticket de résolution.

Un problème standard peut être associé à une résolution standard, ou à plusieurs résolutions. En outre, une résolution standard peut être associée à plusieurs problèmes. Par exemple, un problème d'impression peut être associé à trois résolutions possibles. De même, la résolution pour un message d'erreur peut s'appliquer à différents problèmes.
