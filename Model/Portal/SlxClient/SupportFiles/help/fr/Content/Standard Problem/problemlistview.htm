Utilisation de la vue Liste des problèmes standard

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

La vue Liste des problèmes standard affiche tous les problèmes standard auxquels vous avez accès. Utilisez les options de groupes et de recherche pour restreindre la liste des enregistrements qui s'affiche.

Procédure

Ouvrez la vue Liste des problèmes standard
  • Dans la barre de navigation, cliquez sur Service, puis sur Problèmes standard.
Actualiser les informations de la liste
  • Cliquez sur le bouton Actualiser.

Travailler avec des enregistrements dans une grille

Filtrer ma liste

Utiliser la Recherche pour rechercher les enregistrements

Ouvrez la vue Détails du problème standard
  • Cliquez sur l'ID du problème standard pour ouvrir la vue Détails du problème standard.

Afficher un groupe

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Enregistrer les enregistrements en tant que groupe
  • Exporter vers un fichier
