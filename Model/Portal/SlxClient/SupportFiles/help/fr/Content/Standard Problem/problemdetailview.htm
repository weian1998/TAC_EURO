Utilisation de la vue Détails du problème standard

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné.

Procédure

Ouvrez la vue Détails du problème standard
  • Dans la barre de navigation, cliquez sur Service, puis sur Problèmes standard pour ouvrir la vueListe des problèmes standard. Si nécessaire, utilisez la Recherche pour localiser un problème, puis cliquez sur le N° du problème standard.

Utiliser la Recherche pour rechercher les enregistrements

Passez en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.

Utiliser la liste des groupes du Volet des tâches ou les tâches courantes

Afficher un groupe

Zones d'informations

Les zones d'informations sur le problème standard contiennent des informations sur le problème standard. Les utilisateurs disposant des autorisations appropriées peuvent modifier ces zones.

Onglets des vues Détails

Chaque onglet affiche des informations spécifiques concernant le rôle. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.

section définie par l'utilisateur

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
