Onglet Résolutions standard des problèmes standard

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez cette vue pour afficher ou associer les résolutions standard à un problème.

Procédure

Ouvrez cette vue
  • Dans la vue Détails du problème standard, cliquez sur l'onglet Résolutions standard.
Associer une résolution
  1. Cliquez sur le bouton Associer une résolution.
  2. Utilisez la fonction Rechercher pour trouver une résolution standard.
  3. Cliquez sur Ajouter sélection.
Supprimer une résolution
  1. Sélectionnez l'enregistrement à supprimer, puis cliquez sur le bouton Supprimer.
  2. Cliquez sur OK.

Cette vue contient les colonnes suivantes :

  • Description - Indique la résolution du problème.
  • Notes - Indique de brèves notes à propos de la résolution.
