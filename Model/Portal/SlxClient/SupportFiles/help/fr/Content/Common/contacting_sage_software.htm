Support technique

Les clients ayant souscrit à un contrat d’assistance ont accès au support technique directement auprès de Sage Software et de ses partenaires commerciaux agréés. Nos services sont joignables par téléphone aux heures de bureau du lundi au vendredi, hors jours fériés. La liste des personnes à contacter est accessible sur le site Web de Sage SalesLogix : http://www.sagesaleslogix.com

Les partenaires disposant d'un contrat de support technique valide et d'un code d'accès Internet peuvent adresser leurs demandes d'assistance par voie électronique sur le portail Web d'assistance partenaire Sage SalesLogix : partners.sagenorthamerica.com.

Les clients disposant d'un contrat de support technique valide et d'un code d'accès Internet peuvent adresser leurs demandes d'assistance par voie électronique sur le portail Web d'assistance client Sage SalesLogix : customers.sagenorthamerica.com.
