Nouveautés de cette version

Le voir cible Sage SalesLogix contient les mises à jour suivantes :

Version voir cible

  • Les groupes de la vue Liste se souviennent de vos modifications de tri ou de largeur de colonne.
  • Les améliorations apportées aux recherches comprennent la capacité d'effectuer une recherche à partir de la barre de navigation de gauche afin de rechercher tout type d'enregistrement, ainsi qu'une nouvelle option permettant de définir une condition de recherche par défaut.
  • La fonctionnalité d'alertes vous avertit lorsqu'une alarme d'activité se déclenche ou qu'une notification d'activité n'a pas été confirmée. Vous pouvez définir les options des alertes afin de les personnaliser.
  • Améliorations des activités :
    • L'onglet Disponibilité vous permet de voir les agendas des autres utilisateurs et de déterminer la disponibilité des autres utilisateurs et des ressources.
    • Une zone Emplacement vous permet de définir l'emplacement d'un appel téléphonique ou d'une réunion et d'ajouter automatiquement l'emplacement si un autre emplacement est sélectionné dans l'onglet Disponibilité.
    • Les confirmations d'activité offrent désormais plus d'informations afin de vous aider à accepter ou à refuser une invitation.
    • La liste des activités mises à jour possèdent de nouveaux onglets, des filtres améliorés et des tâches courantes. Utilisez l'onglet Mes activités pour voir les activités dont vous êtes le responsable ou membre.
    • Vous pouvez voir les informations de contact, de compte ou de lead, par exemple un numéro de téléphone, en plaçant la souris sur le nom correspondant dans la vue Détails des activités ou Alertes.
  • Nouvelles options utilisateur et options mises à jour relatives aux activités et à l'agenda :
    • Les options sont désormais partagées entre le client Sage SalesLogix et le client Web Sage SalesLogix. Certains utilisateurs peuvent avoir besoin de réinitialiser leurs options.
    • Les nouvelles options utilisateurs relatives aux activités vous permettent de définir des valeurs par défaut en fonction du type d'activité.
    • Les nouvelles options utilisateur relatives à l'agenda vous permettent de déterminer quelles informations d'activité afficher sur votre agenda, les jours de votre semaine de travail ainsi que le premier jour de la semaine. Il est aussi possible de définir une option qui rappelle aux utilisateurs quels agendas ont été sélectionnées pour l'affichage.
  • L'agenda a été mis à jour et comprend de nouvelles fonctionnalités :
    • La capacité de personnaliser les jours de votre semaine de travail
    • La capacité de voir jusqu'à quatre utilisateurs sur chaque vue d'agenda
  • L'onglet Pièces jointes vous permet d'ajouter des documents Google.
  • Les options de déplacement des contacts et de gestion des contacts en double sont désormais gérées séparement. Les nouvelles méthodes vous permettent de déterminer quelles informations sont déplacées et lesquelles sont réaffectées.
  • Nouveaux modèles de publipostage pour les leads.
  • Capacité de partager les modèles de publipostage avec les autres utilisateurs.
  • Le menu Opportunité a été supprimé. Les options Mettre à jour plusieurs opportunités et Statistiques de l'opportunité sont désormais disponibles sur le menu contextuel de la vue Liste des opportunités.
  • Fonctionnalités pour les administrateurs et les utilisateurs avec un rôle approprié :
    • Le Gestionnaire de sécurité vous permet de gérer l'accès au niveau des champs dans les profils de sécurité.
    • La gestion des ressources vous permet de déterminer si une ressource est un emplacement.

    • Le concepteur de formulaire Web vous permet d'effectuer des modifications simples sur une vue. Par exemple, vous pouvez modifier les légendes, déplacer des éléments, ajouter ou masquer des informations. Pour des informations supplémentaires, consultez la rubrique d'aide « Utilisation des formulaires ».
