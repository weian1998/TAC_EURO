Modification d'un lead

Utilisez la vue Détails du lead pour modifier les informations d'un lead.

Pour modifier

  1. Dans la barre de navigation, cliquez sur Leads.
  2. Utilisez la Recherche pour trouver le lead.
  3. Modifiez les informations du lead.
  4. Cliquez sur Enregistrer.
