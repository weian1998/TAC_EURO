Gestion des doublons

Lorsque vous importez des leads, vous pouvez rechercher d'autres leads ou contacts qui existent dans la base de données. Si un enregistrement de la base de données contient les mêmes informations, vous pouvez éviter la création d'un doublon.

Sage SalesLogix ne peut rechercher que les doublons que vous avez le droit d'afficher. Il se peut qu'il existe également un doublon auquel vous n'avez pas accès.

Pour rechercher des enregistrements correspondants

  1. Lors de l'importation de leads, après le mappage des champs, cliquer sur Suivant.
  2. Sélectionnez Rechercher les leads en double.
  3. En regard de Type, sélectionnez les types d'enregistrements à rechercher.
  4. Sous Filtres de correspondance, cochez les cases des filtres à appliquer.

Les options disponibles dépendent des valeurs que vous avez sélectionnées lors du mappage des champs.

  1. Pour effectuer une Recherche avancée cliquez sur Options de correspondance avancée.

  2. Pour fusionner automatiquement les doublons lors de l'importation, sélectionnez Fusionner automatiquement les doublons.
  3. Dans la zone Conflits, sélectionnez l'une des options suivantes :
  4. l'enregistrement existant l'emporte : lorsque les doublons sont fusionnés, ce sont les informations de l'enregistrement existant qui sont utilisées.
  • l'enregistrement importé l'emporte : lorsque les doublons sont fusionnés, les informations de l'enregistrement importé remplacent celles de l'enregistrement existant.

    Reportez-vous à la rubrique Règles de fusion automatique des enregistrements lors de l'importation pour plus d'informations.
  1. Cliquez sur Exécuter le test pour exécuter le Test de recherche des doublons sur les 100 premiers enregistrements de l'importation.
  2. Si vous souhaitez vérifier l'existence de doublons dans le fichier source de l'importation, sélectionnez l'option Verifier les doublons contenus dans le fichier source d'importation.

    3. Le fait de sélectionner cette option peut accroître la durée de l'importation.
  3. Cliquez sur Suivant pour passer à l' étape suivante.

Les doublons détectés et non fusionnés apparaissent sur l'onglet Doublons de la vue Détails de l'historique des imports de leads une fois l'importation terminée. De là vous pouvez gérer les doublons.
