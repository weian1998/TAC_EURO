Importation de leads à partir d'une liste

Utilisez la vue Importer des leads pour importer des listes de leads dans Sage SalesLogix.

Les fichiers texte doivent être de type ANSI ou ASCII.

Pour importer des leads

  1. Dans le menu Outils, cliquez sur Importer des leads.
  2. Dans le menu Insérer, cliquez sur Importer des leads.
  3. Remplissez les informations de Sélection d'un fichier puis cliquez sur Suivant.
  4. Remplissez les informations de Définition d'un séparateur puis cliquez sur Suivant.
  5. Remplissez les informations de Mappage de champs puis cliquez sur Suivant.
  6. Remplissez les informations de Gestion des doublons et cliquez sur Suivant.
  7. Remplissez les informations de Sélection d'actions d'importation et cliquez sur Suivant.
  8. Repassez en revue les sélections d'importation et cliquez sur Soumettre.
  9. Contrôlez la demande de processus d'importation.
  10. Quand l'importation est terminée, cliquez sur le lien Numéro d'importation pour afficher les résultats de l'historique des imports.
