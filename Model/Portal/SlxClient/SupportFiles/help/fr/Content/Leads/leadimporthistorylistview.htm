Utilisation de la vue Liste de l'historique d'importation des leads

Utilisez la vue Liste de l'historique d'importation pour voir une liste des listes de leads importés.

Procédure

Ouvrir la vue Liste de l'historique d'importation des leads
  • Dans le menu Outils, cliquez sur Historique d'importation.
Ouvrez la vue Détails de l'historique d'importation des leads depuis la vue Liste
  • Depuis la vue Liste de l'historique d'importation des leads, cliquez sur le Numéro d'importation de l'importation que vous souhaitez afficher.

Travailler sur une vue Liste

Cette vue contient les colonnes suivantes :

  • Numéro d'importation : numéro unique qui identifie l'importation de lead.
  • Description : nom de la liste qui a été importée.
  • Date de création : date à laquelle la liste a été importée.
  • État : état de l'importation. Par exemple, Terminée.
  • Nombre d'enregistrements importés : nombre de leads qui ont été importés.
  • Nombre de doublons : nombre de doublons dans l'importation.
  • Nombre des fusions : nombre des enregistrements qui ont été fusionnés lors de l'importation.
  • Nombre des erreurs : nombre des erreurs survenues lors de l'importation.
