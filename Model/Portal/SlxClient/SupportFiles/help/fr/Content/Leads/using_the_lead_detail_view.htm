Utilisation de la vue Détails du lead

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Votre espace de travail peut être différent selon vos droits d'accès ou si votre installation utilise un personnalisé.

Procédure

Ouvrez la vue Détails du lead
  • Dans la barre de navigation, cliquez sur Leads pour ouvrir la vue Liste des leads. Si nécessaire, utilisez la Recherche pour localiser un lead, puis cliquez sur son nom.

Utiliser la Recherche pour rechercher les enregistrements

Afficher un groupe

Passez en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.
Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • E-mail
  • Publipostage
  • Ajouter une réponse à la campagne
  • Ajouter une note
  • Nouvelle réunion
  • Nouvel appel téléphonique
  • Nouvelle tâche à faire

zones d'informations

Les zones d'informations du lead contiennent des informations sur le lead. Cette partie contient également les zones de qualification du lead. Les utilisateurs disposant des autorisations appropriées peuvent modifier ces zones.

onglets des vues Détails

Chacun d'eux affiche des informations spécifiques que vous pouvez utiliser pour les activités relatives au lead sélectionné. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.

section définie par l'utilisateur

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
