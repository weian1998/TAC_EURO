Réinitialisation des utilisateurs verrouillés

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Si le nombre de tentatives de connexion d'un utilisateur dépasse le nombre autorisé, l'utilisateur peut être bloqué à la connexion. Le seuil de tentatives de connexion est défini dans l'Administrateur, sur l'onglet Options système. La réinitialisation d'un utilisateur bloqué ramène le nombre de tentatives à zéro, et permet de nouveau à l'utilisateur de se connecter.

Pour réinitialiser un ou plusieurs utilisateurs verrouillés

  1. Dans la barre de navigation, cliquez sur Administration,, puis cliquez sur Utilisateurs.
  2. Ouvrez le groupe Utilisateurs avec échec de connexion.
  3. Dans le Volet des tâches, sous Tâches Utilisateurs, cliquez sur Réinitialiser les utilisateurs.
