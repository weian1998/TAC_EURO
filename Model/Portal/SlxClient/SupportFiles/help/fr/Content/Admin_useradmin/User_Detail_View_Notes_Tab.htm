Onglet Notes

Utilisez l'onglet Notes de la vue Détails de l'utilisateur pour ajouter des notes relatives à l'utilisateur. Cet onglet vous permet d'ajouter des informations ne figurant sur aucun autre onglet.

Pour gérer

  1. Dans la vue Détails de l'utilisateur, cliquez sur l'onglet Notes.
  2. Pour ajouter une date / un horodatage, cliquez sur .
  3. Saisissez les notes relatives à l'utilisateur.
  4. Cliquez sur Enregistrer.
