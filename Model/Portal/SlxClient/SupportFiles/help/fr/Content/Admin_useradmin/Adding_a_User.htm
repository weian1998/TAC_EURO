Ajout d'un utilisateur

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Avant d'ajouter un ou plusieurs utilisateurs, consultez la rubrique Nouvel employé ayant besoin d'accéder à Sage SalesLogix et exécutez les quatre premières étapes.

Vous ne pouvez pas ajouter d'administrateurs Sage SalesLogix (l'utilisateur portant le nom d'utilisateur « admin »).

Pour ajouter

  1. Dans le menu Administration, cliquez sur Ajouter des utilisateurs.
  2. Dans le champ Ajouter, sélectionnez le type d'utilisateur à ajouter.
  3. Cliquez sur Actualiser les licences pour mettre à jour le nombre de licences disponibles.
  4. Dans le champ Quantité, sélectionnez le nombre d'utilisateurs à ajouter.

Le nombre d'utilisateurs disponibles pour ce type de licence est affiché en regard du champ Quantité.

  1. Si vous souhaitez modifier manuellement chaque onglet du profil de l'utilisateur pour configurer les paramètres, passez à l'étape 8.
  2. Pour créer l'utilisateur à partir d'un modèle ou à l'aide du profil d'un utilisateur existant, cochez la case Créer un profil à partir de. Puis, sélectionnez l'une des options suivantes :

    		3.

    Si le modèle ou l'utilisateur n'est pas attribué au Rôle Utilisateur Standard, les nouveaux utilisateurs créés à partir du profil n'auront pas accès à certaines fonctionnalités du client Web.
  • L'option Modèle ajoute un utilisateur à partir d'un utilisateur modèle existant. Dans la zone Modèles, sélectionnez le nom du modèle d'utilisateur.
  • L'option Utilisateur existant ajoute un nouvel utilisateur à partir du profil d'un utilisateur existant. Dans le champ Utilisateur, sélectionnez le nom de cet utilisateur existant.
  1. Sélectionnez les onglets à copier du modèle ou du profil utilisateur. En savoir plus...
  • Général - les paramètres généraux et les paramètres du service sont copiés. les champs NomUtilisateur, Nom, Fonction et E-mail ne s'appliquent pas lorsque vous copiez des informations.
  • Employé - les champs Fax, Portable, Pager, Direct, N° ID, Comptabilité et tous les champs de données personnelles ne s'appliquent pas lorsque vous copiez des informations.
  • Sécurité - tous les paramètres de sécurité sont copiés.
  • Equipes -tous les paramètres d'équipes sont copiés.
  • Agenda - tous les paramètres d'agenda sont copiés.
  • Service/Assistance technique - tous les paramètres service et support technique sont copiés.
  • Options client (système client) - les paramètres du propriétaire du compte par défaut et du modèle de base sont copiés.
  1. Cliquez sur OK.

Lorsque vous ajoutez un nouvel utilisateur, un code site est automatiquement associé à cet individu.

  1. Une fois le nouvel utilisateur créé, utilisez la vue Détails de l'utilisateur pour modifier le profil de l'utilisateur.

    Vous devrez fournir le nom d'utilisateur et le mot de passe aux nouveaux utilisateurs. Si vous ne souhaitez pas partager le mot de passe par défaut, modifiez le mot de pase en un mot de passe générique et demandez aux utilisateurs de modifier leur mot de passe la première fois qu'ils se connectent à Sage SalesLogix.

  2. Cliquez sur Enregistrer.
