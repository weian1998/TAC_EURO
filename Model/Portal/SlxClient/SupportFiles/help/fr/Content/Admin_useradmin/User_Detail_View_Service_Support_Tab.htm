Onglet Service/Assistance technique

Utilisez l'onglet Service/Assistance technique de la vue Détails de l'utilisateur pour définir les options relatives aux fonctions du service client, de l'assistance technique et de SpeedSearch. Lorsque vous définissez les droits d'accès d'un utilisateur, n'oubliez pas ce qui suit :

  • Lorsque vous ajoutez un nouvel utilisateur, « Disponible pour prendre des appels » est la seule option sélectionnée par défaut.
  • Les modifications prennent effet au chargement de l'application. Lorsqu'un utilisateur actualise cette interface ou qu'il se connecte à cette application, le système applique les modifications.
  • Lorsque vous affichez l'onglet Service/Support dans le profil d'un utilisateur distant, le préfixe du N° de Ticket de l'utilisateur distant est également affiché (en lecture seule)

Pour plus d'informations sur les N° de Ticket

Pour prendre en charge la synchronisation entre les bases de données distantes et la base de données (hôte) du bureau principal, le système Sage SalesLogix doit garantir que des N° de Ticket uniques sont créés lorsque les utilisateurs ajoutent des tickets depuis le bureau principal, les bureaux distants ou les ordinateurs des utilisateurs distants. Le système affecte un préfixe de ticket à chaque base de données. Le préfixe se divise en deux éléments :

  • Identifiant de site – Les trois premiers chiffres du préfixe identifient le site. L'identifiant de site de la base de données hôte est 001. Des identifiants de site compris entre 100 et 199 sont attribués aux bases de données des bureaux distants. Les identifiants de site des bases de données d'utilisateurs distants vont de 200 à l'infini. (Notez toutefois que les identifiants de site ne sont pas forcément assignés dans l'ordre.)
  • Base de clé – Les deux derniers chiffres du préfixe représentent le numéro de révision de la base de données. Si vous créez une nouvelle base de données pour un bureau ou un utilisateur distant, ce nombre est incrémenté.

Lorsqu'un utilisateur crée un ticket, le système attribue le préfixe correct suivi par un suffixe à six chiffres généré automatiquement (par exemple, 001-01-000015).

Pour définir les options

  1. Dans la vue Détails de l'utilisateur, cliquez sur l'onglet Service/Assistance technique.
  2. Sélectionnez une ou plusieurs des options suivantes :
  • Disponible pour prendre des appels - permet à d'autres utilisateurs d'attribuer des tickets à cet utilisateur.

Les tickets attribués à un utilisateur non disponible sont paramétrés par défaut sur « Non attribué ».

  • Notifier l'utilisateur en cas de nouveau ticket - envoie un message e-mail lorsqu'un ticket est attribué à l'utilisateur.

Le système envoie un message e-mail lors de l'affectation d'un nouveau ticket ou de la réaffectation d'un ticket existant à cet utilisateur.

  • L'utilisateur peut autoriser l'accès au portail client - permet à l'utilisateur d'autoriser des clients à accéder aux tickets via le portail client.
  • L'utilisateur est défini par défaut sur « Soumettre à SpeedSearch » - définit Soumettre à SpeedSearch comme l'option par défaut lorsque l'utilisateur crée un nouveau ticket.
  • L'utilisateur est autorisé à approuver les soumissions SpeedSearch - permet à l'utilisateur de déterminer les tickets et les défauts disponibles dans SpeedSearch.

Cette option n'est disponible que si le processus d'approbation SpeedSearch est activé pour le bureau correspondant. Cette option est activée dans l'Administrateur.

  • Notifier l'utilisateur en cas de nouveau défaut - envoie un message e-mail lorsqu'un défaut est attribué à l'utilisateur.

Un e-mail est envoyé lors de l'affectation d'un nouveau défaut ou de la réaffectation d'un défaut existant à cet utilisateur.

  1. Cliquez sur Enregistrer.
