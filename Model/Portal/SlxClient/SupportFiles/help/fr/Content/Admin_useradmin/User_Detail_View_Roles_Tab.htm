Onglet Rôles

Utilisez l'onglet Rôle dans la vue Détails de l'utilisateur pour affecter un utilisateur à un rôle, ou pour l'en supprimer. Lorsqu'un utilisateur est affecté à un rôle, l'utilisateur a accès aux actions définies dans ce rôle. Si un utilisateur est affecté à plus d'un rôle, il aura accès à toutes les actions sécurisées contenues dans ces rôles, même si chaque rôle ne contient pas les mêmes actions.

Procédure

Ouvrez l'onglet Rôles.

  1. Ouvrez la vue Détails de l'utilisateur.
  2. Cliquez sur l'onglet Rôles.

Affecter un utilisateur à un rôle

  1. Cliquez sur Ajouter un rôle.
  2. Utilisez la recherche pour localiser et sélectionner les rôles que vous souhaitez ajouter à l'utilisateur.
  3. Cliquez sur Ajouter les rôles sélectionnés.
    Vous ne pouvez pas ajouter le même rôle plusieurs fois.

Supprimer un utilisateur d'un rôle

  1. Sélectionnez l'enregistrement à supprimer, puis cliquez sur le bouton Supprimer.
  2. Cliquez sur Oui.
