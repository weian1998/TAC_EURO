Configuration d'Intellisync for SalesLogix

Les préférences de synchronisation doivent être configurées sur l'ordinateur de chaque utilisateur. Pour la synchronisation, un groupe ad hoc doit exister dans Sage SalesLogix et être configuré comme groupe de synchronisation. Pour configurer Intellisync, vous aurez besoin de l'URL du portail Intellisync for SalesLogix . Contactez l'administrateur système pour en savoir plus.

Vous pouvez configurer différents paramètres pour différents types d'enregistrements. Par exemple, vous pouvez activer la synchronisation unilatérale pour les contacts et la synchronisation bilatérale pour les éléments de l'agenda et les configurer pour qu'ils soient synchronisés à des moments différents. Les étapes suivantes indiquent les paramètres de synchronisation recommandés entre Microsoft Outlook ou Lotus Notes et Intellisync for SalesLogix.

Les boutons Aide de l'interface de configuration d'Intellisync appellent des rubriques créées par le créateur d'Intellisync, Nokia. Il se peut que la terminologie employée dans ces rubriques d'aide diffère parfois des termes utilisés dans la mise en œuvre de Sage SalesLogix. Reportez-vous à la rubrique Aide supplémentaire pour la configuration d'applications pour de plus amples informations.

Pour configurer

  1. Démarrez Intellisync for SalesLogix.
  • Cliquez sur Démarrer, pointez sur Programmes, puis sur Sage SalesLogix et sur Intellisync for SalesLogix et cliquez sur Intellisync for SalesLogix.
  1. Cliquez sur Configuration et procédez comme suit.
  • Dans la zone ID utilisateur, entrez votre nom d'utilisateur Sage SalesLogix.
  • Dans la zone Mot de passe, entrez votre mot de passe.
  • Dans la zone URL de connexion, entrez le chemin d'accès à votre portail Intellisync for SalesLogix.

Contactez votre administrateur pour obtenir l'URL du portail.

  1. Cliquez sur Configurer l'application.
  2. Sur l'écran Configuration, vous pouvez cocher ou décocher les cases dans Actif pour chaque type de données. Actif signifie que les données sont synchronisées lors de chaque synchronisation.

    Afin de limiter les contacts à synchroniser avec Sage SalesLogix, Sage SalesLogix vous recommande de créer un filtre SyncSalesLogix.

  3. Cliquez sur le bouton Assistant de configuration.
  4. Sur l'écran Application de bureau, sélectionnez Microsoft Outlook ou Lotus Notes, puis cliquez sur Suivant.
  5. Sur l'écran Que synchroniser ?, sélectionnez Toutes les applications Intellisync for SalesLogix prises en charge par Microsoft Outlook ou Lotus Notes, puis cliquez sur Suivant.
  6. Sur l'écran Comment synchroniser ?, sélectionnez Synchronisation bidirectionnelle, puis cliquez sur Suivant.
  7. Sur l'écran Quand synchroniser ?, choisissez à quel moment effectuer la synchronisation et cliquez sur Suivant.
  • Toutes les 10 minutes – sélectionnez un incrément dans la liste déroulante.
    L'utilisateur démarre Intellisync manuellement, PushSync démarre et synchronise les données à l'intervalle spécifié.
  • Tous les jours à un intervalle spécifié – sélectionnez un incrément dans la liste déroulante.
    L'utilisateur démarre Intellisync manuellement, PushSync démarre et synchronise les données à l'intervalle spécifié.
  • Quand je me connecte à mon ordinateur - Quand l'utilisateur se connecte, Intellisync démarre et s'exécute dans la barre d'état du système.
  • Quand un changement est détecté dans Lotus Notes/MS Outlook - Intellisync démarre et s'exécute dans la barre d'état du système.
  • Quand je me connecte au serveur Intellisync for SalesLogix.
  1. Sur l'écran Informations de connexion Intellisync for SalesLogix, sélectionnez votre profil d'utilisateur et cliquez sur Suivant.
  1. Renseignez les autres écrans de façon appropriée selon le type d'informations à synchroniser, pour MS Outlook ou Lotus Notes.
  1. Sur l'écran Options pour les tâches à faire de MS Outlook ou de Lotus Notes, sélectionnez Transférer tous les éléments et cliquez sur Suivant.

Cette option synchronise toutes les tâches à faire actives, non terminées et terminées.

  1. Sur l'écran Options pour les rendez-vous, sélectionnez Transférer uniquement les éléments à venir et cliquez sur Suivant.
  2. Si vous configurez Intellisync for Lotus Notes, sur l'écran Options des contacts, accédez à la base de données Lotus Notes et cliquez sur Suivant.
  1. Sur l'écran Informations de connexion Intellisync for SalesLogix, vérifiez votre ID utilisateur Sage SalesLogix, le mot de passe et l'URL de connexion, puis cliquez sur Suivant.
  2. Cliquez sur Terminer.
  3. Cliquez sur Fermer.
  4. Cliquez sur OK.
  5. Si vous ne l'avez pas encore fait, vous devez sélectionner le groupe SyncSalesLogix.
