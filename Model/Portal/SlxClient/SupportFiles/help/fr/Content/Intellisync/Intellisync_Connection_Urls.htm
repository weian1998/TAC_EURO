URL de connexion à Intellisync

Avant de configurer Intellisync for SalesLogix, vérifiez que vous avez la bonne URL de connexion. L'URL de connexion est le chemin d'accès au portail Intellisync for SalesLogix Chaque base de données requiert son propre portail. Contactez votre administrateur système Sage SalesLogix pour l'URL.

  • Pour les utilisateurs du réseau, le chemin d'accès est celui du portail Intellisync for SalesLogix sur l'hôte.
  • Pour les utilisateurs du bureau distant, le portail est généralement situé sur l'ordinateur du bureau distant.
  • Pour les utilisateurs distants, le portail se trouve sur leur propre ordinateur local.

Pour les paramètres de connexion avancés, cliquez sur le bouton Avancé sur l'écran principal Intellisync.
