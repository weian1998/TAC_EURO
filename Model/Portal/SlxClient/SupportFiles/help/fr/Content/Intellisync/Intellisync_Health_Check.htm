Vérification de l'intégrité d'Intellisync

Intellisync for SalesLogix comprend une option de dépannage nommée Vérification de l'intégrité d'Intellisync. Vous pouvez dépanner la connexion au portail et vérifier la version des assemblages. Dans la plupart des cas, le contrôle des versions d'assemblages se fait uniquement à la demande du support technique Sage.

Vous pouvez accéder à la Vérification de l'intégrité d'Intellisync à partir de IIS Manager, du portail d'Intellisync et du serveur Web personnel Sage SalesLogix.

Pour accéder à partir de IIS Manager

  1. Cliquez sur Démarrer, pointez sur Programmes, puis sur Outils d'administration.
  2. Cliquez sur Internet Information Services (IIS) Manager.
  3. Développez le nom de l'appareil, les sites Web, puis SalesLogix.
  4. Double-cliquez sur SLXIntellisync.
  5. Effectuez l'une des opérations suivantes :
  • MS Windows 2003 : dans la liste de fichiers du volet droit, cliquez avec le bouton droit de la souris sur Default.aspx et cliquez sur Parcourir.
  • MS Windows 2008 : Dans la liste de fichiers du volet droit, cliquez avec le bouton droit de la souris sur Documents par défaut, pointez sur Gestion de l'application puis cliquez sur Parcourir. Le navigateur affiche le portail Intellisync. S'il ne s'affiche pas, assurez-vous que les paramètres sont configurés correctement et que le portail a été déployé.
  1. Dans le navigateur, cliquez sur Diagnostics et effectuez l'une des opérations suivantes :
  • Pour tester des informations de connexion d'utilisateur, saisissez le nom d'utilisateur et le mot de passe de l'utilisateur que vous souhaitez tester puis cliquez sur Essayer de se connecter. Une demande est soumise au serveur pour tester la validité des informations de connexion pour le portail. Un message indique si le test a été réussi.

    • Ce test détermine uniquement si les informations de connexion permettent de se connecter au portail. Il ne teste pas si une connexion à la base de données fonctionne.

  • Pour afficher les assemblages et leur numéro de version, cliquez sur les boutons Suivant et Précédent au bas de la liste.

Pour accéder à partir du portail

  1. Ouvrez le portail Intellisync.

Saisissez par exemple l'URL du portail dans le navigateur au format : http://server:port/portal name/default.aspx

  1. Cliquez sur Diagnostics.
  1. Effectuez l'une des opérations suivantes :
  • Pour afficher les assemblages et leur numéro de version, cliquez sur les boutons Suivant et Précédent au bas de la liste.
  • Pour tester des informations de connexion d'utilisateur, saisissez le nom d'utilisateur et le mot de passe de l'utilisateur que vous souhaitez tester puis cliquez sur Essayer de se connecter.

Une requête est soumise au serveur pour tester la connexion de l'utilisateur au portail Intellisync. Un message indique si le test a été réussi.

Pour accéder depuis le serveur Web personnel

  • Pour tester des informations de connexion d'utilisateur
  1. Cliquez avec le bouton droit sur l'icône Serveur Web personnel Sage SalesLogix dans la barre d'état système, puis cliquez sur Propriétés.
  2. Dans la boîte de dialogue Serveur Web personnel Sage SalesLogix, cliquez sur Gérer.
  3. Saisissez le nom d’utilisateur et le mot de passe de l'utilisateur que vous souhaitez tester.
  4. Cliquez sur Tester la connexion.

Une requête est soumise au serveur pour tester la connexion de l'utilisateur au portail Intellisync. Un message indique si le test a été réussi.

  • Pour afficher les assemblages et leur numéro de version
  1. Cliquez avec le bouton droit sur l'icône Serveur Web personnel Sage SalesLogix dans la barre d'état système, puis cliquez sur Propriétés.
  2. Dans la boîte de dialogue Serveur Web personnel Sage SalesLogix, sélectionnez le portail Intellisync dans la zone Site Web.
  3. Cliquez sur Détails.

Affichez les assemblages installés pour le portail Intellisync et leur numéro de version.

  • Pour comparer des versions de portail
  1. Cliquez avec le bouton droit sur l'icône Serveur Web personnel Sage SalesLogix dans la barre d'état système, puis cliquez sur Versions côte à côte.
  2. Afficher la version du fichier sage.saleslogix.dll pour chaque portail installé.
