Suppression d'activités, de contacts, ou d'événements

Intellisync for SalesLogix n'autorise pas MS Outlook ou Lotus Notes à supprimer des activités, des contacts ou des événements de la base de données Sage SalesLogix. Vous devez supprimer ces enregistrements manuellement après la synchronisation.

Marquage d'éléments supprimés

Lorsqu'un élément est supprimé de , MS Outlook ou Lotus Notes , la balise ISYNC_DEL est ajoutée à l'enregistrement de l'activité, du contact ou de l'événement Sage SalesLogix comme suit :

  • Pour les activités, la balise ISYNC_DEL est insérée dans le champ Priorité.
  • Pour les contacts, la balise ISYNC_DEL est insérée dans le champ Description.
  • Pour les événements, la balise ISYNC_DEL est insérée dans le champ Type.

La demande d'obtention de liste Activités/tâches à faire filtre les enregistrements ayant la valeur de champ ISYNC_DEL afin qu'ils ne soient pas synchronisés, mais les enregistrements doivent être supprimés manuellement dans SalesLogix pour une suppression permanente.

Affichage des éléments supprimés

  • Pour voir toutes les activités qui ont été supprimées de MS Outlookou de Lotus Notes , triez la liste des activités par code de priorité et recherchez ISYNC DEL.
  • Pour voir tous les contacts qui ont été supprimés de MS Outlookou de Lotus Notes, triez la liste des contacts par le champ Description et recherchez ISYNC DEL.
  • Pour voir tous les événement qui ont été supprimés de MS Outlookou de Lotus Notes, triez la liste des événement par le champ Type et recherchez ISYNC DEL.

Suppression manuelle d'un élément marqué

Vous pouvez supprimer les activités, contacts ou événements de Sage SalesLogix si vous le souhaitez.

  1. Localisez l'élément comme décrit dans Affichage des éléments supprimés.
  2. Sélectionnez l'élément, cliquez avec le bouton droit puis cliquez sur Supprimer.
