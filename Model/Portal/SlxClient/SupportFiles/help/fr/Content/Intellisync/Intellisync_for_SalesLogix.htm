Intellisync for SalesLogix

Intellisync for SalesLogix est un programme distinct. Il est installé et intégré à Sage SalesLogix. Intellisync for SalesLogix vous permet de partager vos activités et contacts Sage SalesLogix avec Microsoft Outlook et Lotus Notes.

Pour utiliser Intellisync for SalesLogix afin de partager des données entre deux applications, vous devez :

  • être un utilisateur sous licence à la fois de Sage SalesLogix et de MS Outlookou de Lotus Notes,
  • vous assurer qu'Intellisync for SalesLogix a été installé.

Cliquez sur Démarrer, pointez sur Programmes, puis sur Sage SalesLogix. Intellisync for SalesLogix est l'une des options de la liste. Si vous ne voyez pas Intellisync for SalesLogix dans la liste, contactez votre administrateur.

Intellisync Client est déjà installé pour les utilisateurs de Client Web hors ligne.
  • Effectuez les tâches suivantes dans le client Web Sage SalesLogix :
  • Créez un groupe ad hoc pour identifier les enregistrements de contact qui seront synchronisés avec MS Outlook, ou Lotus Notes. Nommez ce groupe SyncSalesLogix.
  • Sélectionnez SyncSalesLogix comme votre groupe de synchronisation de contacts.
  • Configurez Intellisync for SalesLogix.
