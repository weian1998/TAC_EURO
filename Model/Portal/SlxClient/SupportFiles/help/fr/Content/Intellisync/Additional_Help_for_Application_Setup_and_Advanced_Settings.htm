Aide supplémentaire pour la configuration d'applications

Vous trouverez des informations supplémentaires sur la configuration des paramètres suivants dans les rubriques d'aide accessibles via l'Assistant de configuration avancée d'Intellisync for SalesLogix.

  • Création de filtres
  • Mappage des champs
  • Obtention d'informations de connexion
  • Réception de confirmations
  • Sélection d'une application de bureau
  • Spécification de conditions et règles de filtrage
  • Spécification des éléments à synchroniser
  • Spécification du mode de synchronisation
  • Spécification du moment de la synchronisation
  • Spécification des options d'échange de données MS Outlook (ou Lotus Notes)
  • Spécification des options des tâches MS Outlook (ou Lotus Notes)
  • Spécification des options de calendrier MS Outlook (ou Lotus Notes)

Les boutons Aide de l'interface de configuration d'Intellisync appellent des rubriques créées par le créateur d'Intellisync, Nokia. Il se peut que la terminologie employée dans ces rubriques d'aide diffère parfois des termes utilisés dans la mise en œuvre de Sage SalesLogix.
