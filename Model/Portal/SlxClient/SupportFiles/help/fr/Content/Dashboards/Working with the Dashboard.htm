Utilisation des tableaux de bord

Les tableaux de bords s'affichent sous forme d'onglets sur la page d'accueil et contiennent des outils d'analyse prédéfinis personnalisables prenant la forme de widgets graphiques et de widgets de liste. La page d'accueil peut contenir plusieurs instances de tableau de bord et chaque tableau de bord peut contenir plusieurs widgets.

Il existe deux onglets de tableau de bord prêt à l'emploi : Mon tableau de bord et Ventes. Chaque onglet a plusieurs widgets prêts à l'emploi. Vous pouvez ajouter et modifier des widgets de façon à ce qu'ils affichent des données différentes ou bien les mêmes données de façon différente.

Utilisez le menu contextuel d'un onglet de tableau de bord pour ajouter un nouvel onglet, pour copier, pour masquer/afficher et pour supprimer des onglets, ou pour partager des onglets avec votre équipe ou service. Utilisez l'outil glisser-déposer pour réorganiser les widgets sur l'onglet. Les widgets se redimensionnent automatiquement en fonction du type de données et de graphique, ainsi que de la taille des colonnes.

Vous pouvez également envoyer toutes les données d'un groupe à partir d'une vue Liste vers un widget Liste des groupes sur le tableau de bord via le lien Promouvoir vers le tableau de bord.

Si vous n'êtes pas le propriétaire du tableau de bord, vous devez d'abord effectuer une copie pour le partager ou le modifier.

Procédure

Accéder à un tableau de bord
  • Dans la barre de navigation, cliquez sur Accueil.
Ajouter un nouvel onglet
  1. Cliquez avec le bouton droit de la souris sur un onglet du tableau de bord puis cliquez sur Nouvel onglet.
  2. Dans le champ Titre, tapez le nom de l'onglet.
  3. Si nécessaire, sélectionnez le modèle qui décrit le type de présentation que vous voulez utiliser pour votre onglet.
  4. Pour définir le modèle sélectionné comme la présentation par défaut, sélectionnez Définir par défaut.
  5. Cliquez sur OK.
Copier un onglet
  1. Cliquez avec le bouton droit de la souris sur l'onglet de tableau de bord, puis cliquez sur Copier l'onglet.
  2. Dans le champ Titre, tapez le nom de l'onglet.
  3. Cliquez sur OK.
    L'onglet et son contenu seront copiés vers un nouvel onglet.
Supprimer un onglet
  1. Cliquez avec le bouton droit de la souris sur l'onglet de tableau de bord, puis cliquez sur Supprimer l'onglet.
  2. Cliquez sur Oui pour confirmer.
    Le tableau de bord et tout son contenu sont supprimés de l'espace de travail.
Masquer un onglet
  • Cliquez avec le bouton droit de la souris sur l'onglet du tableau de bord, puis cliquez sur Masquer l'onglet.
    L'onglet est caché à la vue.
Afficher un onglet
  1. Cliquez avec le bouton droit de la souris sur l'onglet du tableau de bord, puis cliquez sur Afficher l'onglet.
  2. Cliquez sur Afficher à côté du nom de l'onglet que vous voulez afficher .
    L'onglet s'affiche dans le tableau de bord.
Partager un onglet avec d'autres utilisateurs
  1. Cliquez avec le bouton droit de la souris sur l'onglet de tableau de bord, puis cliquez sur Partager l'onglet.

  2. Effectuez l'une des opérations suivantes :

    • Cliquez sur Tout le monde pour partager l'onglet avec tous les utilisateurs.
    • Cliquez sur le bouton Ajouter, recherchez les utilisateurs, équipes ou services avec lesquels vous souhaitez partager l'onglet, puis cliquez sur OK.
      Si vous avez ajouté un utilisateur, une équipe ou un service par erreur, sélectionnez les éléments que vous souhaitez supprimer de la liste et cliquez sur Supprimer.

      Si vous partagez un onglet Tableau de bord contenant un groupe, le groupe doit aussi être partagé avec les utilisateurs afin que le groupe s'affiche dans l'onglet Tableau de bord.

Modifier l'étiquette de l'onglet, redimensionner les colonnes ou faire de l'onglet l'onglet par défaut
  1. Cliquez avec le bouton droit de la souris sur l'onglet de tableau de bord, puis cliquez sur Options de modification.
  2. Tapez un nouveau titre pour modifier l'étiquette de l'onglet.
  3. Sélectionnez la présentation de colonnes souhaitée pour cet onglet :
    • Une seule colonne
    • Deux colonnes égales
    • Deux colonnes avec la gauche plus large
    • Deux colonnes avec la droite plus large
  4. Sélectionnez l'option Définir comme onglet par défaut pour définir l'onglet actuel comme onglet par défaut.
    L'onglet par défaut ne peut pas être masqué ou supprimé.
  5. Cliquez sur OK.
