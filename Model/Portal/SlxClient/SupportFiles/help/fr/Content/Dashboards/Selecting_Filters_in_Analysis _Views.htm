Sélection de filtres dans les vues Analyse

Changer les filtres des vues Analyse

Le filtrage vous permet de cibler des données spécifiques. Par exemple, si vous sélectionnez 30 derniers jours dans la zone Période, la base de données donne les résultats des 30 derniers jours uniquement. Vous pouvez affiner davantage vos résultats en effectuant un drill pour plus d'informations.

Notre scénario :
Vous voulez un aperçu détaillé des opportunités ouvertes dans votre entreprise.

Pour ouvrir une vue d'analyse

  1. Dans la barre de navigation, cliquez sur Tableaux de bord, puis sur Tableau de bord des ventes.
  2. Dans la section État du pipeline, cliquez sur l'onglet Ouvrir les opportunités.
  3. Cliquez sur le premier lien hypertexte dans la colonne Responsable du compte.
    Ceci ouvre la fenêtre de la vue Analyse des opportunités ouvertes, avec le filtre Responsable du compte défini sur le paramètre sélectionné dans l'ensemble de contenu Opportunités ouvertes.

    Vue Analyse

  4. Cliquez sur Options de filtres.
    Le volet des filtres s'ouvre avec une liste d'éléments de filtre. Notez que le filtre Responsable du compte affiche le responsable du compte que vous avez sélectionné à l'étape précédente.
  5. Cliquez sur Effacer les filtres dans le volet des options de filtre à gauche de la vue Analyse.
    Avec la restriction du filtre supprimée, la grille et les graphiques sont actualisés pour afficher toutes les données disponibles relatives aux opportunités ouvertes. Trier par est toujours défini sur Responsable du compte.
  6. Dans la colonne Valeur, cliquez sur TOUS à côté du filtre Responsable du compte.
  7. Cliquez sur les points de suspension situés sur la droite dans la colonne Valeur.
    La boîte de dialogue Ajouter ou modifier des valeurs de filtre s'affiche.

    Boîte de dialogue Ajouter ou modifier des valeurs de filtre

  8. Sélectionnez trois valeurs de responsable du compte. Vous pouvez rechercher des valeurs à l'aide de la barre de défilement ou en écrivant dans la zone Rechercher pour affiner votre sélection.
  9. Quand vous avez terminé, cliquez sur OK.
    Les valeurs des opportunités ouvertes des responsables du compte que vous avez sélectionnés s'affichent dans la grille et les graphiques.

    Vous pouvez cliquer sur Options de filtre pour masquer le volet de filtre, ce qui augmente la zone d'affichage de la grille et des graphiques dans la vue Analyse.

  10. Pour imprimer la vue Analyse, dans le menu Fichier, cliquez sur Impression écran.
  11. Pour exporter les données de la grille vers Microsoft Excel, cliquez sur Exporter vers MS Excel.
    Une fiche de travail Excel s'ouvre avec toutes les colonnes et données de la grille représentées. (nécessite MS Excel)

    Seul le nombre de lignes affiché dans la grille est exporté. Utilisez le sélecteur Afficher les lignes pour afficher des lignes supplémentaires au besoin.

Pour plus d'informations, consultez la rubrique "Planification ou modification d'une activité " dans l'aide du Tableau de bord .
