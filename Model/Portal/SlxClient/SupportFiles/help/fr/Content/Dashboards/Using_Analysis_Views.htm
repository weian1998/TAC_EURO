Utilisation des vues Analyse

Les vues d'analyse affichent des informations semblables aux ensembles de contenu, mais elles offrent davantage d'options pour filtrer les données. Les vues Analyse peuvent être ouvertes en cliquant sur les liens dans les ensembles de contenu.

  • Comme un ensemble de contenu, la grille est la clé d'une vue d'analyse. Les graphiques représentent les données dans les grilles. Vous contrôlez les données dans la grille avec les filtres. Une ligne Total général au bas de la plupart des grilles calcule les résultats.
  • Vous pouvez sélectionner des valeurs de filtres individuelles parmi divers groupes de filtres afin de cibler précisément les résultats de votre recherche.
  • La taille des graphiques et de la grille peut être modifiée en déplaçant leurs bords.
  • Les résultats de la grille de la vue d'analyse peut être exportée vers MS Excel en un seul clic.
