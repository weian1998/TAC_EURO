Promouvoir un groupe vers le tableau de bord

Promouvoir un groupe vers le tableau de bord envoie toutes les données du groupe vers un widget de liste de groupes sur le tableau de bord.

Pour promouvoir un groupe vers le tableau de bord

  1. A partir d'une vue Liste, cliquez sur l'onglet du groupe que vous souhaitez vers le tableau de bord.
  2. Dans le Volet des tâches, cliquez sur Promouvoir vers le tableau de bord.
  3. Cliquez sur le nom du tableau de bord vers lequel vous souhaitez envoyer les données, puis cliquez sur OK.
  4. Cliquez sur Accueil dans la barre de navigation pour afficher le tableau de bord.
    Les données apparaissent sur le tableau de bord dans un widget Liste des groupes. La configuration par défaut de la Liste des groupes est 10 lignes ; toutefois, vous pouvez la modifier dans l' éditeur de widget de façon à ce qu'elle affiche jusqu'à 50 lignes.
