Utilisation des widgets

Sage SalesLogix fournit plusieurs widgets prêts à l'emploi. Vous pouvez ajouter, modifier et supprimer des widgets dans le tableau de bord et réorganiser les widgets sur la page via l'outil glisser-déposer.

A l'aide de filtres prédéfinis, vous pouvez afficher les métriques pour vos données sur des plages ou à partir de valeurs distinctes et afficher ces données dans l'un des widgets graphiques. Par exemple, vous pouvez configurer un widget graphique à colonne de façon à ce qu'il affiche toutes les opportunités clôturées et gagnées, par propriétaire et montant total de la vente, ou un graphique circulaire représentant les ventes gagnées par une équipe.

Toutefois, avec les filtres distincts, tous les widgets graphiques, sauf le graphique circulaire, sont limités à 10 points de données à la fois ; les dix premiers de la sélection. Le widget de graphique circulaire vous permet de sélectionner une option supplémentaire qui représente le reste (en plus des 5 ou 10 points sélectionnés) regroupés dans une catégorie Autre. Avec les filtres par plage, un point de données pour chaque plage de données définie est renvoyé. Cela peut entraîner un graphique affichant bien plus de 10 secteurs ou colonnes, mais ce paramètre peut être modifié par votre administrateur système.

Procédure

Ajouter un widget
  1. Cliquez avec le bouton droit sur l'onglet Mon tableau de bord, puis cliquez sur Ajouter du contenu.
  2. Dans la boîte de dialogue Ajout de nouveau contenu, cliquez sur le bouton Ajouter du widget que vous souhaitez ajouter.
    Par exemple, pour ajouter un widget graphique circulaire, cliquez sur Ajouter sous Graphique circulaire.
    Le widget vide apparaît sur le tableau de bord et l'éditeur de widget s'ouvre.
  3. Configurez les paramètres du widget et cliquez sur OK.
Modifier un widget existant
  1. Sur la barre d'outils du widget, cliquez sur le bouton Paramètres .
  2. Effectuez les modifications nécessaires.
  3. Cliquez sur OK.
    Le widget affiche les modifications.

    • Barres

      • Titre - Le titre qui s'affiche dans la barre de titre du widget.

      • Légende : le texte qui s'affiche sous le titre et donne des informations supplémentaires.

      • Entité - L'entité dont vous souhaitez extraire les données. Par exemple, Opportunités. Toutes les entités avec des vues Liste peuvent être visualisées sous forme de graphiques et de listes.

      • Groupe - Le groupe pour lequel vous voulez afficher les données. Par exemple, Toutes les opportunités. Tous les groupes accessibles à un utilisateur peuvent être utilisés dans des graphiques et des listes.

      • Dimension - Les données que vous souhaitez afficher. Par exemple, Probabilité. Tous les filtres Plage et Distinct actuels sont disponibles sous forme de dimensions.

        Les dimensions disponibles dépendent des filtres d'entité disponibles et d'autres paramètres de configuration. Pour plus d'informations, adressez-vous à votre administrateur système.

      • Métrique - La façon dont vous calculez les données. Par exemple, Montant réel total. Toutes les entités avec des vues Liste ont une métrique Count associée, par exemple, CountLeads et CountAccounts. Il existe des métriques supplémentaires pour les opportunités et l'historique.

      • Axe X - L'étiquette utilisée pour décrire les données sur l'axe X du graphique.

      • Axe Y - L'étiquette utilisée pour décrire les données sur l'axe Y du graphique.

      • Tronquer les étiquettes - Sélectionnez Oui ou Non pour tronquer les étiquettes afin qu'elles apparaissent sur le graphique.

      • Longueur d'étiquette - La longueur maximale d'une étiquette avant qu'elle soit tronquée.

Configurer le widget Liste des groupes
  1. Dans la boîte de dialogue Paramètres de la liste Groupe, saisissez un titre pour le widget.
  2. Sélectionnez une Entité et un Groupe dans les listes déroulantes. Par exemple, Opportunités et Clôturées et gagnées.
  3. Saisissez le Nombre d'enregistrementsque vous souhaitez dans la liste. Par exemple, 10, 25 ou 50.
  4. Saisissez le nombre d'enregistrements que vous souhaitez voir dans la zone Enregistrements visibles ; 10, 25 ou 50.
    Par exemple, si vous avez saisi 50 enregistrements à l'étape 2, mais ne souhaitez en voir que 10 à la fois, tapez 10. Le widget se redimensionne en fonction du nombre d'enregistrements visibles.
  5. Cliquez sur OK.
    Le widget affiche la liste.

  		6.

    Vous pouvez utiliser n'importe quel nombre entre 1 et le nombre maximum d'enregistrements devant s'afficher. Le maximum par défaut est 50.
Configurer le widget Liste des liens
  1. Dans la boîte de dialogue Paramètres de lien, saisissez un titre pour le widget.
  2. Cliquez sur Ajouter un lien.
  3. Double-cliquez sur la zone URL de l'icône et saisissez le chemin d'accès de l'icône que vous souhaitez afficher pour le lien. Par exemple, /google.jpg.

    		4.

    Cette URL doit être un chemin relatif vers une image sur le serveur Web.
  4. Double-cliquez sur la zone URL du lien et saisissez le chemin d'accès du lien que vous souhaitez afficher. Par exemple, http://localhost/SlxClient/Account.aspx.

    5. Ce chemin peut être un chemin relatif à l'intérieur du portail SLXClient, ou un lien absolu vers un site externe. L'URL d'un site externe doit inclure l'indicatif http://. Par exemple, pour accéder à www.google.com, l'URL doit être http://www.google.com. Sinon, le navigateur recherchera http://<SLXClient site>/www.google.com.
  5. Double-cliquez sur la zone Texte du lien et saisissez le texte que vous souhaitez afficher comme lien. Par exemple, Page de recherche Google.
  6. Double-cliquez sur la zone Zone cible et saisissez un identifiant pour la cible. Par exemple, si vous souhaitez que le lien s'ouvre dans une nouvelle fenêtre de navigateur, saisissez Nouvelle fenêtre. Lorsque vous cliquerez sur le lien, une nouvelle fenêtre/un nouvel onglet de navigateur s'ouvrira.

    		7.

    Si vous laissez ce champ vide, le lien s'ouvre dans la fenêtre de navigateur active / dans l'onglet actif lorsque vous cliquez dessus. Si plusieurs liens ont la même désignation cible, chaque lien remplace l'autre au fur et à mesure que vous cliquez dessus.
  7. Cliquez sur Ajouter un lien pour ajouter un autre lien ou sur OK pour fermer l'éditeur.
    Lorsque vous cliquez sur OK, le widget est actualisé et affiche les liens.
Réduire un widget
  • Sur la barre d'outils du widget, cliquez sur le bouton Réduire .
    Le contenu du widget est réduit dans le tableau de bord. Cliquez de nouveau sur le bouton agrandir le widget.
Supprimer un widget
  • Sur la barre d'outils du widget, cliquez sur le bouton Fermer .
    Le widget est définitivement supprimé du tableau de bord.
