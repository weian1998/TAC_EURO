Affichage des résultats de SpeedSearch

L'onglet Résultats affiche la liste des éléments correspondant à vos critères de recherche. Lorsque vous sélectionnez un élément, il s'affiche dans l'onglet Aperçu. En cliquant sur un lien dans l'onglet Résultats, vous affichez la vue Détails de l'enregistrement ou vous ouvrez la boîte de dialogue Détails de l'historique ou Détails de l'activité.

Si plus d'une page de résultats de recherche est affichée, cliquez sur les boutons Précédent ou Suivant pour afficher plus de résultats.

La zone Résultats de SpeedSearch affiche la liste des éléments correspondant à vos critères de recherche. Vous pouvez utiliser les boutons de déplacement à la droite de l'écran ou les boutons de déplacement de votre clavier pour parcourir les résultats. Lorsque vous sélectionnez un élément, il s'affiche dans la zone Aperçu.

Pour trier les résultats, cliquez sur l'un des noms de colonne.

Nom de colonne Description
Objet Une brève description de l'élément. Votre administrateur système détermine les informations utilisées comme objet.
Source Le type d'élément où la correspondance a été trouvée (par exemple Ticket, Défaut ou Document).
Créé La date de création de l'élément. Cette date s'affiche au format année-mois-jour pour le tri.
Dernière mise à jour La date de la dernière révision de l'élément.
Mots-clés Le nombre d'occurrences de chaque mot-clé dans l'élément.
Résultats Le nombre total d'occurrences des mots-clés dans l'élément. Ce nombre est rempli de zéros à gauche pour le tri.
Pertinence Le classement selon le nombre d'éléments dans lesquels chaque mot-clé apparaît dans votre recherche, le nombre d'occurrences de chaque mot-clé dans la recherche et la densité de résultats dans chaque élément.

Ouverture des éléments

Le type de fichiers ou enregistrements, votre accès à eux et la manière dont vous avez ouvert SpeedSearch, contrôlant les éléments que vous pouvez ouvrir. (Par exemple, vous pouvez uniquement ouvrir les tickets si vous avez ouvert SpeedSearch à partir du client Sage SalesLogix.)

Pour ouvrir les éléments

  • Sélectionnez l'élément dans la liste Résultats, puis cliquez sur Ouvrir résultat.
    Si le bouton Ouvrir résultat n'est pas disponible, cet élément ne peut pas être ouvert ou vous n'y avez pas accès. Votre administrateur système doit activer cette fonction. Consultez le sujet d'aide du module Administrateur « Ajouter ou modifier un index de base de données » pour plus de détails.

Insérer des résultats

Lorsque vous ouvrez SpeedSearch dans la vue Détails du ticket ou Détail du défaut, vous pouvez insérer du texte à partir des résultats de la recherche dans le ticket ou défaut actif.

Pour insérer une résolution dans le ticket ou défaut actif

  1. Dans la liste Résultats, sélectionnez le défaut ou ticket contenant la résolution que vous souhaitez utiliser.
  2. Si vous ne souhaitez insérer qu'une partie de la résolution, sélectionnez le texte dans la zone Aperçu. Sans cela, toute la résolution sera copiée dans le ticket ou défaut actif.
  3. Cliquez sur Insérer un résultat.

Marquer un résultat comme étant utilisé

Si vous trouvez un élément utile mais que vous n'insérez pas les résultats, vous pouvez indiquer qu'un résultat a été utilisé. Cela permet au système de garder une trace des éléments fréquemment utilisés.

Pour marquer un résultat comme étant utilisé

  • Sélectionnez l'élément et cliquez sur Résultat utilisé.

Pour effectuer une recherche et insérer une résolution dans le ticket actif

  1. Dans l'onglet Détails de la vue Détails du ticket, cliquez sur le bouton SpeedSearch.
  2. Effectuez une recherche.
  3. Dans l'onglet Résultat, sélectionnez l'élément contenant la résolution que vous souhaitez utiliser.
  4. Si vous ne souhaitez insérer qu'une partie de la résolution, sélectionnez le texte dans la zone Aperçu. Sans cela, toute la résolution sera copiée dans le ticket actif.
  5. Cliquez sur le bouton Copier et coller le texte sélectionné dans un enregistrement.
    SpeedSearch ferme et vous retournez au ticket actif. La résolution a été collée dans la zone Résolution.
