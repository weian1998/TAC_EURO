SpeedSearch Client

SpeedSearch est un outil puissant qui permet aux utilisateurs de rechercher des informations stockées dans la base de données Sage SalesLogix ou dans des fichiers externes. Par exemple, vous pouvez rechercher parmi des tickets ou des procédures existantes pour aider un client à résoudre son problème ou rechercher un document spécifique, tel qu'un argumentaire de vente. Le type d'informations que vous pouvez rechercher dépend des index que votre administrateur système a définis dans le module Administrateur.

Avant d'ouvrir SpeedSearch, vous pouvez sélectionner et copier le texte à utiliser comme mots-clés pour la recherche. Utilisez ensuite l'une des méthodes suivantes pour ouvrir SpeedSearch.

Pour ouvrir SpeedSearch dans le client Sage SalesLogix

  • Effectuez l'une des opérations suivantes :

    • Dans le menu Recherche, cliquez sur SpeedSearch.
    • Dans la barre d'outils, cliquez sur le bouton SpeedSearch.
    • Quand vous ajoutez un ticket ou affichez un ticket existant, cliquez sur le bouton Recherche dans l'onglet Détails pour rechercher une résolution. Vous pouvez insérer les résultats de la recherche dans le ticket actif.
Pour ouvrir SpeedSearch hors de Sage SalesLogix
  • Dans le menu Démarrer, pointez sur Programmes, pointez sur Sage SalesLogix, puis cliquez sur client SpeedSearch.

Pour réaliser une recherche SpeedSearch

  • Effectuez l'une des opérations suivantes :

    • Entrez les informations relatives à la recherche dans la barre de menus.
    • Cliquez sur le bouton SpeedSearch dans l'onglet Détails du ticket ou la vue Insérer un ticket.
    • Dans le menu Outils, cliquez sur Advanced SpeedSearch.

Si vous rencontrez des problèmes pour ouvrir ou utiliser SpeedSearch, contactez votre administrateur Sage SalesLogix. Les informations de configuration et de dépannage de SpeedSearch figure dans l'aide du module Administrateur.
