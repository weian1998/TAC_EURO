Copie d'une opportunité

Vous pouvez copier une opportunité existante pour en créer rapidement une nouvelle comprenant les mêmes concurrents, contacts et produits, ou autres informations identiques.

Pour copier une opportunité

  1. Ouvrez la vue Détails de l'opportunité de la campagne à copier.
  2. Cliquez sur Copier.
    La boîte de dialogue Insérer une opportunité apparaît avec les informations suivantes issues de l'opportunité d'origine :
    • Description
    • Compte
    • Responsable du compte
    • Type
    • Source de lead
    • Concurrents
    • Produits
    • Contacts
  3. Vous pouvez modifier, au besoin, les informations copiées, puis renseigner les zones d'informations restantes.
  4. Cliquez sur Enregistrer
