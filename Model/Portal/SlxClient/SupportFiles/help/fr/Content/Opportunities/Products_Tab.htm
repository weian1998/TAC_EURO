Produits d'une opportunité

Utilisez cette vue pour ajouter, modifier ou supprimer des produits associés à l'opportunité. Vous pouvez également modifier l'ordre de la liste des produits.

Procédure

Ouvrez cette vue
  • Cette vue apparaît dans la vue Insérer une opportunité et sous forme d'onglet dans la vue Détails de l'opportunité. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.

Supprimer un produit d'une opportunité

Modifier les informations relatives aux produits d'une opportunité

Modifier les informations de produit dans la grille
  • Sélectionnez l'enregistrement à modifier, faites vos modifications dans la grille, puis Cliquez sur Enregistrer. Pour annuler vos modifications, cliquez sur Réinitialiser avant de cliquer sur Enregistrer.

    Les colonnes ne peuvent pas toutes être modifiées dans la grille. Pour savoir si une colonne peut être modifiée, cliquez dans la grille. Si une zone blanche apparaît, la colonne est modifiable.

Cette vue contient les colonnes suivantes :

  • Ligne n° : affiche le numéro de ligne du produit. Cette information est utile lorsque vous imprimez la commande car vous pouvez vous assurer que les produits figurent dans la commande appropriée. (Intégration de la comptabilité uniquement)
  • Produit - Affiche le nom du produit.
  • Famille - Affiche la famille du produit.
  • Niveau de prix - Affiche le niveau de prix utilisé pour ce produit. Cliquez dans la grille pour apporter des modifications.
  • Prix (devise de base) - Affiche le prix du produit. Cette colonne est basée sur le type de devise par défaut défini par votre administrateur.
  • Remise - Vous permet de saisir un pourcentage de remise. Cliquez dans la grille pour apporter des modifications.
  • Prix Ajusté - Affiche le prix de base du produit moins la remise. Cliquez dans la grille pour apporter des modifications.  Vous permet de saisir un nouveau prix ajusté. Si vous saisissez un nouveau prix ajusté, la remise est recalculée pour refléter ce nouveau montant. Cette colonne est basée sur le type de devise par défaut défini par votre administrateur.
  • Prix (Devise de l'opportunité) : affiche le prix du produit converti dans le type de devise de l'opportunité. (Devises multiples uniquement).
  • Quantité - Affiche la quantité numérique du produit ajoutée à l'opportunité. Cliquez dans la grille pour apporter des modifications. Vous permet de saisir une nouvelle quantité.
  • Unité : affiche l'échelle utilisée pour mesurer la quantité du produit. Par exemple, « à l'unité » ou « Pack de 10 ». (Intégration de la comptabilité uniquement)
  • Prix étendu (Devise de base) : affiche le total calculé du prix du produit multiplié par la quantité, moins toute remise éventuelle. Cette colonne est basée sur le type de devise par défaut défini par votre administrateur.
  • Etendu (Devise de l'opportunité) : affiche le total calculé du prix du produit multiplié par la quantité, moins toute remise éventuelle, et converti dans le type de devise de l'opportunité (Devises multiples uniquement).
