Modification d'un concurrent associé à une opportunité

Après avoir ajouté un concurrent à une opportunité, vous pouvez modifier les informations de ce concurrent dans la boîte de dialogue Modifier le concurrent.

Pour modifier

  1. Ouvrez la boîte de dialogue Modifier le concurrent de l'opportunité.
  • Dans l'onglet Concurrent de la vue Détails de l'opportunité, cliquez sur le lien hypertexte Modifier du concurrent à modifier.
  • Dans la boîte de dialogue Insérer une opportunité, cliquez sur le lien hypertexte Modifier du concurrent que vous souhaitez modifier.
  1. Pour modifier l'évaluation de ce concurrent, entrez un chiffre dans la zone Evaluation.

Seule l'évaluation du concurrent pour cette opportunité est modifiée. L'évaluation globale du concurrent demeure inchangée.

  1. Sélectionnez Titulaire pour identifier le concurrent comme étant l'actuel titulaire du compte.
  2. Renseignez les zones Points forts et Faiblesses pour le concurrent.
  3. Dans la zone Stratégie, saisissez des notes sur les stratégies du concurrent.
  4. Dans la zone Notes, saisissez des informations supplémentaires concernant le concurrent.
  5. Cliquez sur OK.
