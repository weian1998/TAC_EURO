Résumé de l'opportunité

Vous pouvez afficher la valeur et les informations récapitulatives de l'opportunité sélectionnée dans le résumé de l'opportunité situé dans la section supérieure de la vue Détails de l'opportunité. Vous pouvez modifier les informations le cas échéant.

Le lien hypertexte Copier les informations de l'opp. dans un e-mail vous permet d'envoyer un e-mail dont le corps du message contient déjà les informations sur l'opportunité. Les informations copiées dans l'e-mail comprennent la description de l'opportunité, le nom du compte, le nom du responsable du compte, le nom du revendeur, la date de clôture estimée, la probabilité de clôture et des commentaires. De plus, elles comprennent les informations sur les produits, les concurrents et les contacts, ainsi que les informations récapitulatives du résumé de l'opportunité.

Pour copier les informations d'opportunité dans un e-mail

  1. Dans la vue de la liste des Opportunités, cliquez sur une description de l'opportunité.

La vue Détails des opportunités s'affiche.

  1. Dans la zone Résumé de l'opportunité, cliquez sur Copier les informations de l'opp. dans un e-mail.

Microsoft Outlook s'ouvre et une copie des informations actuelles de l'opportunité apparaît dans l'e-mail.

Les informations qui s'affichent dans le résumé de l'opportunité dépendent de l'état de cette dernière.

Résumé de l'opportunité si celle-ci a un état Clôturée - Gagnée ou Clôturée - Perdue.
