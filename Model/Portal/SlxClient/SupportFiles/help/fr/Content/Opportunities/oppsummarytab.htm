Onglet Résumé

L'onglet Synthèse de l'opportunité sur la vue Détails de l'opportunité affiche la liste des activités et des interactions avec l'opportunité sélectionnée. Les lignes Dernier se rapportent à la dernière activité réalisée. Les lignes Suivant affichent les activités de suivi planifiées.

Pour afficher l'onglet Synthèse de l'opportunité

  1. Dans la barre de navigation, cliquez sur Opportunités, puis sélectionnez une description pour l'opportunité.
  2. Cliquez sur l'onglet Synthèse de l'opportunité. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
