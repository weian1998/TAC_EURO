Onglet Membres

L'onglet Membres affiche les utilisateurs, services et équipes faisant partie d'une équipe. Utilisez cet onglet pour gérer les membres d'une équipe.

Pour gérer

  1. Dans la vue Détails de l'équipe, cliquez sur l'onglet Membres.
  2. Cochez la case Ajouter automatiquement le responsable de l'utilisateur à l'équipe si vous souhaitez ajouter le responsable d'un utilisateur lorsque vous ajoutez des membres à une équipe.
  3. Effectuez l'une des opérations suivantes :
    • Cliquez sur Ajouter un membre d'équipe pour ajouter un membre d'équipe. Utilisez la recherche pour trouver et sélectionner le membre de l'équipe à ajouter puis cliquez sur OK.
    • Cliquez sur le lien Supprimer du membre de l'équipe que vous souhaitez supprimer de l'équipe puis cliquez sur OK.
    • Cliquez sur le lien Profil de sécurité pour modifier le profil de sécurité d'un membre de l'équipe à l'intérieur de l'équipe.
    • Dans le volet Tâches courantes, cliquez sur Remplacer dans toutes les équipes pour remplacer l'équipe sélectionnée par un autre utilisateur, un autre service ou une autre équipe, dans toutes les équipes.
