Modification du propriétaire d'une équipe

Vous pouvez utiliser la vue Equipes ou la vue Utilisateurs pour modifier le propriétaire d'une équipe. Une équipe peut avoir plusieurs propriétaires.

Procédure

La vue Equipes permet de gérer le propriétaire d'une équipe

  1. Ouvrez la vue Détails de l'équipe.
  2. Sur l'onglet Membres, cliquez sur le lien Profil de sécurité de l'utilisateur que vous souhaitez ajouter ou supprimer comme propriétaire de l'équipe.
  3. Dans la zone Profil de sécurité, sélectionnez un nouveau profil de sécurité pour l'utilisateur.
  4. Cliquez sur OK.
  5. Cliquez sur Enregistrer.

Utiliser la vue Utilisateurs pour gérer le propriétaire d'une équipe

  1. Ouvrez la vue Détails de l'utilisateur de l'utilisateur que vous souhaitez ajouter ou supprimer comme propriétaire de l'équipe.
  2. Cliquez sur l'onglet Appartenance à une équipe.
  3. Cliquez sur le lien Profil de sécurité de l'équipe appropriée.
  4. Dans la zone Profil de sécurité, sélectionnez un nouveau profil de sécurité pour l'utilisateur.
  5. Cliquez sur OK.
  6. Cliquez sur Enregistrer.
