Suppression ou remplacement de membres d'une équipe

Vous pouvez supprimer des membres d'une équipe ou remplacer un utilisateur par un autre au sein d'une équipe.

Procédure

Supprimer un membre d'une équipe

  1. Ouvrez la vue Détails de l'équipe dont vous voulez supprimer des membres.
  2. Cliquez sur l'onglet Membres.
  3. Cliquez sur le lien Supprimer du membre de l'équipe que vous souhaitez supprimer.
  4. Pour confirmer la suppression, cliquez sur OK.
  5. Cliquez sur Enregistrer.

Supprimer un utilisateur de toutes les équipes

  1. Ouvrez la vue Liste des utilisateurs et sélectionnez l'utilisateur que vous souhaitez supprimer de toutes les équipes.
  2. Dans le volet Tâches courantes, cliquez sur Supprimer de toutes les équipes.
  3. Pour confirmer la suppression, cliquez sur OK.

Remplacer un membre d'une équipe par un autre dans toutes les équipes

  1. Ouvrez la vue Détails de l'utilisateur, Détails du service ou Détails de l'équipe du membre de l'équipe que vous souhaitez remplacer.
  2. Dans le volet Tâches courantes, cliquez sur Remplacer dans toutes les équipes.
  3. Dans la boîte de dialogue Remplacer, confirmez l'utilisateur, le service ou l'équipe à supprimer de toutes les équipes.
  4. Dans la zone Par, utilisez la fonction rechercher pour localiser l'utilisateur, le service ou l'équipe qui remplacera l'utilisateur, le service ou l'équipe que vous supprimez.
  5. Cliquez sur OK.
