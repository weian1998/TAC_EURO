Copie d'un service

Lorsque vous copiez un service existant, tous ses membres sont inclus dans le nouveau service.

Pour copier

  1. Ouvrez la vue Détails du service du service que vous souhaitez copier.
  2. Dans le volet Tâches courantes, cliquez sur Copier le service.
  3. Pour confirmer, cliquez sur OK.
  4. Dans la zone Nom, entrez un nouveau nom pour le service.
  5. Si nécessaire, modifiez les membres de l'équipe et les équipes dont le service est membre.
  6. Cliquez sur Enregistrer.
