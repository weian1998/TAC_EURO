Vue Détails de l'équipe

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Procédure

Ouvrez la vue Détails de l'équipe
  1. Sur la barre de navigation, cliquez sur Administration, puis sur Equipes pour ouvrir la vue Liste des équipes.
  2. Cliquez sur une équipe dans la liste, ou utilisez la fonction Rechercher pour localiser une équipe.

Les informations sur l'équipe s'ouvrent dans la vue Détails.

Utilisez la fonction Rechercher pour rechercher des enregistrements d'équipe

Ouvrez un groupe

Passez en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste.

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe.
  • Supprimer du groupe.
  • Enregistrer les enregistrements en tant que groupe
  • Exporter vers un fichier
  • Remplacer sur toutes les équipes
  • Copier l'équipe

Zones d'informations

Les zones d'informations sur l'équipe contiennent des informations sur l'équipe.

Onglets des vues Détails

Chaque onglet affiche des informations spécifiques sur le profil de l'équipe. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.

Membres
Accès au compte

section définie par l'utilisateur

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
