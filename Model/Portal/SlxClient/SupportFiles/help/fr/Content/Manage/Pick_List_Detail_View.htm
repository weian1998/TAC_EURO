Vue Détails de la liste de sélection

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Procédure

Ouvrez la vue Détails de la liste de sélection
  1. Sur la barre de navigation, cliquez sur Administration, puis sur Listes de sélection pour ouvrir la vue Liste des listes de sélection.
  2. Cliquez sur une liste de sélection dans la liste, ou utilisez la fonction Rechercher pour localiser une liste de sélection.

Les informations de la liste de sélection s'ouvrent dans la vue Détails.

Utilisez la fonction Rechercher pour rechercher des enregistrements de liste de sélection

Ouvrez un groupe

Passez en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste.
Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe.
  • Supprimer du groupe.
  • Ajouter une liste de sélection

Zones d'informations

Les zones d'informations de la liste de sélection contiennent les informations essentielles sur la liste de sélection.

Onglets des vues Détails

L'onglet Eléments permet de gérer les éléments d'une liste de sélection.

section définie par l'utilisateur

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
