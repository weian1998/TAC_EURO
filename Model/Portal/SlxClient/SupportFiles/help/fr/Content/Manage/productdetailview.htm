Utilisation de la vue Détails du produit

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Les produits sont en lecture seule lorsque l'intégration de la comptabilité est activée. Vous devez gérer les produits dans votre système de comptabilité et synchroniser les modifications dans Sage SalesLogix.

Procédure

Ouvrez la vue Détails du produit
  • Dans la barre de navigation, cliquez sur Ventes, Support, ou Administration, puis cliquez sur Produits pour ouvrir la vue Liste des produits Si nécessaire, utilisez la Recherche pour localiser un produit, puis cliquez sur le nom du produit.

Utiliser la Recherche pour rechercher les enregistrements

Passez en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.

Utiliser la liste des groupes du Volet des tâches ou les tâches courantes

Afficher un groupe

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe.
  • Supprimer du groupe.

Zones d'informations

  • Nom - Le nom du produit.
  • Description - La description du produit.
  • Code article - Le code article du produit.
  • Famille - La famille du produit. Cliquez sur bouton en forme de flèche et sélectionnez un élément dans la liste.
  • Etat - L'état du produit. Cliquez sur bouton en forme de flèche et sélectionnez un élément dans la liste.
  • Coût - Le coût du produit.
  • Unité de mesure : l'échelle utilisée pour mesurer la quantité du produit. Par exemple, « A l'unité » ou « Pack de 10».

onglets des vues Détails

Chacun d'eux affiche des informations spécifiques qui servent dans le cadre des activités relatives à l'opportunité. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.
