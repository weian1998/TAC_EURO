Zones d'information des listes de sélection

Le tableau suivant décrit les zones d'informations par défaut. En savoir plus sur les options des listes de sélection...

Avant de définir les options des listes de sélection, assurez-vous de comprendre les propriétés de contrôle des listes de sélection et la façon dont les données des listes de sélection sont stockées dans la base de données. Le fait de comprendre comment les données sont liées entre la liste de sélection et la base de données déterminera si une option est prise en charge. Par exemple, si vous modifiez la liste de sélection Préférences en « Autoriser les sélections multiples », la liste de sélection ne fonctionnera pas correctement parce que le champ de la base de données est trop petit pour stocker de grandes quantités de texte.

Zone Description/Action Applicable au Web Applicable au réseau
Nom Nom de la liste de sélection.
Entrée obligatoire Oblige les utilisateurs à saisir une valeur avant de passer au champ suivant. Lorsqu'elle est sélectionnée, cette option s'applique automatiquement aux listes de sélection réseau. Elle s'applique aux listes de sélection Web lorsque l'option Appliquer les options aux listes de sélection Web est elle aussi sélectionnée. *
Autoriser les sélections multiples Autorise les utilisateurs à sélectionner plusieurs éléments de la liste. Lorsqu'elle est sélectionnée, cette option s'applique automatiquement aux listes de sélection réseau. Elle s'applique aux listes de sélection Web lorsque l'option Appliquer les options aux listes de sélection Web est elle aussi sélectionnée. *
Le texte doit correspondre à un élément de la liste Veille à ce que la valeur saisie corresponde à un élément de la liste. Lorsqu'elle est sélectionnée, cette option s'applique automatiquement aux listes de sélection réseau. Elle s'applique aux listes de sélection Web lorsque l'option Appliquer les options aux listes de sélection Web est elle aussi sélectionnée. *
Trié par ordre alphabétique Affiche les éléments de la liste par ordre alphabétique. Si cette option est sélectionnée, les valeurs d'ordre sont ignorées. Lorsqu'elle est sélectionnée, cette option s'applique automatiquement aux listes de sélection réseau. Elle s'applique aux listes de sélection Web lorsque l'option Appliquer les options aux listes de sélection Web est elle aussi sélectionnée. *
Les utilisateurs sont autorisés à modifier les éléments Autorise les utilisateurs à ajouter ou à modifier des éléments dans la liste de sélection. Cette option ne s'applique qu'aux listes de sélection réseau. Les utilisateurs ne sont pas autorisés à modifier les éléments des listes de sélection dans le client Web.  
Appliquer les options aux listes de sélection Web Cochez cette case pour utiliser les options sélectionnées sur cette vue pour les listes de sélection Web. Si cette option est décochée, les listes de sélection Web sont contrôlées par les attributs définis sur le contrôle dans l'application Architect.  
Valeur par défaut actuelle Affiche la valeur par défaut de la liste de sélection.
Tester cette liste Cliquez sur bouton en forme de flèche pour afficher la liste de sélection et les éléments qu'elle contient.

* L'option « Appliquer les options aux listes de sélection Web » doit aussi être sélectionnée.
