Modification d'un produit dans la liste

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails du produit pour modifier les informations du produit.Les produits sont en lecture seule lorsque l'intégration du système de comptabilité est activée. Vous devez gérer les produits dans votre système de comptabilité et synchroniser les modifications dans Sage SalesLogix.

Pour modifier

  1. Dans la barre de navigation, cliquez sur Ventes, Support, ou Administration, puis cliquez sur Produits pour ouvrir la vue Liste des produits Si nécessaire, utilisez la Recherche pour localiser un produit, puis cliquez sur le nom du produit.
  2. Si nécessaire, modifiez les informations du produit.
  3. Cliquez sur Enregistrer.
