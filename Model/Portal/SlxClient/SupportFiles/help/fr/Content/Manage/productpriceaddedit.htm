Ajout ou modification de prix de produits

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Vous pouvez ajouter ou modifier des informations de prix de produit.

Procédure

Ajouter un niveau de prix

  1. Effectuez l'une des opérations suivantes :

    • Ouvrez la vue Détails du produit et cliquez sur l'onglet Niveaux de prix.
    • Ouvre la vue Insérer un produit.
  2. Cliquez sur le bouton Ajouter un prix.
  3. Ajoutez les informations suivantes :
    • Description : tapez une description de prix.
    • Prix : tapez un prix pour le produit.
    • Défaut : identifiez le prix par défaut du produit. Il est impossible de choisir plusieurs tarifs par défaut. Si vous en sélectionnez plusieurs, le premier prix ne sera plus sélectionné.

  4. Cliquez sur OK.

Modifier un niveau de prix

  1. Effectuez l'une des opérations suivantes :

    • Ouvrez la vue Détails du produit et cliquez sur l'onglet Niveaux de prix.
    • Ouvre la vue Insérer un produit.
  2. Cliquez sur Modifier en regard du prix que vous souhaitez modifier.
  3. Modifiez l'une des informations suivantes :
    • Description - Modifiez la description du prix.
    • Prix - Modifiez le prix du produit.
    • Défaut - Identifiez le prix par défaut du produit. Il est impossible de choisir plusieurs tarifs par défaut. Si vous en sélectionnez plusieurs, le premier prix ne sera plus sélectionné.

  4. Cliquez sur OK.
