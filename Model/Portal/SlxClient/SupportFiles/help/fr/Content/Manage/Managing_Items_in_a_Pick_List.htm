Gestion des éléments d'une liste de sélection

Vous pouvez modifier une liste de sélection en ajoutant un nouvel élément, en modifiant un élément existant ou en supprimant un élément.

Si vous ajoutez un élément personnalisé à la liste de sélection Etat de l'opportunité et qu'un utilisateur définit l'état de l'opportunité sur cet élément, le résumé de la vue Détails de l'opportunité se comporte comme si l'état de l'opportunité était défini sur « Ouvert ».

Pour gérer

  1. Ouvrez la vue Détails de la liste de sélection.
  2. Sur l'onglet Eléments, effectuez l'une des opérations suivantes :
  • Cliquez sur Ajouter pour ajouter un nouvel élément.
  • Cliquez sur le lien Modifier de l'élément que vous souhaitez modifier.
  • Cliquez sur le lien Supprimer de l'élément que vous souhaitez supprimer.

Si vous utilisez une autre application qui partage les informations de base de données avec Sage SalesLogix, vous devez supprimer les mêmes éléments dans les deux applications.

  1. Dans la boîte de dialogue Ajouter/Modifier un élément :
  • Dans la zone Texte, saisissez le texte que vous souhaitez afficher dans la liste de sélection.
  • (Facultatif) Dans la zone Code, saisissez un code pour l'élément.

Si vous utilisez une autre application qui partage les informations de base de données avec Sage SalesLogix, vous devez utiliser les mêmes codes de liste de sélection dans les deux applications.

  • Dans la zone Ordre, spécifiez l'ordre d'apparition de l'élément dans la liste. En savoir plus...

Par exemple, si vous entrez 0, l'élément apparaît en première position dans la liste. Si vous avez plusieurs éléments avec le même numéro d'ordre, ils sont regroupés selon ce numéro et le groupe entier apparaît dans la liste avant le numéro d'ordre venant juste après. Notez que si l'attribut Trié par ordre alphabétique est sélectionné, les valeurs d'ordre sont ignorées.

  • Cochez la case Est l'élément par défaut pour définir l'élément comme l'élément par défaut de la liste.
  • Cliquez sur Enregistrer et Nouveau pour continuer à ajouter des éléments, ou cliquez sur OK lorsque vous avez terminé.
  1. Cliquez sur Enregistrer.
