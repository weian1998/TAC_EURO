Onglet Lots de produits

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Vous pouvez utiliser l'onglet Lots de produits pour afficher et gérer les lots de produits associés à un produit.

Ajouter un lot de produits

Modifier un lot de produits

L'onglet Lots de produits contient les informations suivantes :

  • Nom - Le nom du lot de produits. Cliquez sur le nom pour ouvrir la vue Détails du lot.
  • Description - La description du lot de produits.
  • État - L'état du lot.
