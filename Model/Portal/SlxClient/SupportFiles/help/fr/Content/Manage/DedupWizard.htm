Utilisation de la recherche des doublons

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

L'Assistant de recherche des doublons compare les enregistrements dans Sage SalesLogix pour déterminer si les données contiennent des doublons. L'assistant crée une tâche contenant des données potentiellement en double. Vous pouvez fusionner ou supprimer les données en double à l'aide des résultats de la tâche.

L'option SpeedSearch doit être configurée pour utiliser la recherche des doublons.

Pour exécuter

  1. Dans le menu Outils, cliquez sur Recherche des doublons.
  2. Dans l'écran Sélectionner une source :
    1. Sélectionnez le type d'entité pour lequel vous souhaitez vérifier l'existence de doublons dans la liste déroulante Sélectionner le type de tâche.
    2. Dans la liste déroulante Groupe, sélectionnez un groupe d'enregistrements au sein du type d'entité sélectionné.
    3. Cliquez sur Suivant.
  1. Dans l'écran Options de recherche, sélectionnez les filtres que l'assistant doit utiliser pour la mise en correspondance des données.
    Si vous sélectionnez une valeur, les données des deux enregistrements doivent concorder de manière exacte pour être considérés comme des doublons.
  2. Cliquez sur Options de correspondance avancées pour définir des options de correspondance supplémentaires. Lorsque vous avez terminé, cliquez sur OK.
  3. Cliquez sur Suivant.
  4. Dans l'écran Révision, vérifiez vos sélections pour cette tâche.
  5. Cliquez sur Envoyer.
  6. Pour annuler une tâche en cours de traitement, cliquez sur Abandonner.
  7. Au besoin, cliquez sur le lien Numéro de tâche pour ouvrir la vue Détails de l'historique de la recherche des doublons.
