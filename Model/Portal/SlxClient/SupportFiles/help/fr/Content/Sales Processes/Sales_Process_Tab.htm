Onglet Processus de vente

Cette fonction est uniquement disponible lorsque la licence appropriée est installée. Pour plus d'informations sur les licences, reportez-vous au Guide de planification de Sage SalesLogix.

Vous pouvez utiliser l'onglet Processus de vente pour lier un processus de vente à une opportunité, modifier le processus de vente actuellement lié à une opportunité ou exécuter les étapes au sein d'un processus de vente.

Dans le processus de ventes, vous pouvez planifier, mener à bien ou sauter des étapes. En temps normal, vous effectuerez les étapes dans l'ordre indiqué. Vous avez toutefois la possibilité de passer d'une phase à une autre pour travailler sur des étapes spécifiques.

Vous pouvez fermer une étape en cochant la case de la colonne située en regard du nom de cette étape. Une fois toutes les étapes d'une phase terminées, le pourcentage de probabilité (dans la zone Résumé) se met à jour.

  • Pour que la probabilité de l'opportunité se mette à jour automatiquement, les phases doivent être effectuées dans l'ordre. De plus, certains processus de vente comprennent des étapes obligatoires qui ne permettent pas d'effectuer les étapes dans un mauvais ordre.
  • Il ne se met pas à jour si la phase contient des étapes ouvertes ou si la phase précédente est encore ouverte.
  • Si une phase a été sautée, vous pouvez la terminer manuellement. La probabilité sera alors proprement mise à jour.
  • Si vous réouvrez une étape, en décochant la case correspondante, le pourcentage de probabilité ne diminue pas.

Procédure

Lier un processus de vente à une opportunité et modification du processus lié

Tant que l'opportunité est ouverte, vous pouvez joindre un processus de vente à une opportunité ouverte ou remplacer le processus de vente affecté.

Pour joindre un processus de vente à une opportunité

  1. Dans la vue Détails de l'opportunité, cliquez sur l'onglet Processus de vente.
  2. Dans la zone Processus de vente, cliquez sur la flèche du menu déroulant, puis sélectionnez un processus de vente.

Les phases et les étapes du processus sélectionné s'affichent dans la grille.

Supprimez ou modifiez un processus de vente lié à une opportunité
  1. Dans la vue Détails de l'opportunité, cliquez sur l'onglet Processus de vente.

  2. Effectuez l'une des opérations suivantes :

    • Sélectionnez Aucun dans la liste Processus de vente pour supprimer un processus lié.
    • Sélectionnez un autre processus dans la liste Processus de vente pour modifier les processus.
      Vous pouvez modifier le processus de vente lié à une opportunité même si certaines étapes du processus sont terminées.
Utilisation de phases et d'étapes dans un processus de vente

La grille de l'onglet Processus de vente affiche les phases ainsi que les étapes qui les composent. Les phases regroupent des étapes. Par exemple, les étapes d'une phase nommée Qualifier pourraient inclure Démarchage par téléphone, Envoi des informations et Suivi. Chaque phase affiche le nom de la phase et la probabilité (en pourcentage) que vous remportiez l'opportunité une fois la phase terminée. La phase s'affiche comme étant terminée lorsque l'utilisateur coche la case en regard de la dernière étape de cette phase.

Pour terminer une étape

  1. Dans l'onglet Processus de vente, localisez la phase sur laquelle vous voulez travailler.

La colonne Terminé indique si l'étape est terminée, la colonne Etape affiche le nom de l'étape et la colonne Description affiche l'action contenue dans l'étape. Si les étapes ne sont pas visibles, cliquez sur le bouton de chevron situé à droite du nom de la phase.

  1. Dans la colonne Etape, cliquez sur le nom de l'étape.

Ce nom ressemble à un hyperlien (en bleu et souligné).

  1. Effectuez l'une des opérations suivantes :
  • Si la colonne Description contient la valeur Aucun, suivez les instructions de la colonne Etape.
  • Si la colonne Description contient une action (par exemple Rendez-vous ou Publipostage), cliquez sur le lien hypertexte et saisissez les informations dans les boîtes de dialogue d'activité. (si disponibles).
  1. Une fois que vous avez terminé l'action détaillée dans l'étape, cochez la case située dans la colonne adjacente.

La date de fin est automatiquement saisie dans la colonne Date d'achèvement.
