Utilisation de l'onglet Détails de l'expédition du retour

Utilisez cet onglet pour ajouter des informations sur les produits de remplacement expédiés au client dans le cadre du retour sélectionné. Les informations sur le client sont ajoutées à partir des informations sur le retour mais vous pouvez sélectionner un autre contact pour recevoir les pièces de remplacement expédiées.

Procédure

Ouvrir l'onglet Détails de l'expédition
  1. Ouvrez la vue Détails des retours.
  2. Cliquez sur l'onglet Détails de l'expédition.Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets
Modifier les informations du contact de l'expédition
  1. Dans la zone Contact, cliquez sur Bouton Rechercher et utilisez la recherche pour sélectionner un autre contact auquel expédier le(s) produit(s) de remplacement.
  2. Utilisez les zones Expédier par et N° de Suivi pour inclure des informations supplémentaires sur l'expédition.
  3. Dans la zone A l'attention de, saisissez le nom du client ou les autres informations nécessaires à l'expédition du/des produit(s) de remplacement.
  4. Cliquez sur le bouton AdresseModifier pour ajouter ou changer l'adresse d'expédition.
  5. Entrez toute instruction d'expédition spéciale dans la zone Instructions spéciales.
  6. Cliquez sur Enregistrer.
Ajouter un produit en cours d'expédition
  1. Sous Actifs à expédier, cliquez sur Ajouter Renvoyer le produit expédié.
  2. Remplissez les zones d'informationsAjouter un produit de retour expédié.
  3. Cliquez sur OK.
Modifier un produit en cours d'expédition
  1. Sous Actifs à expédier, cliquez sur Modifier en regard du produit que vous souhaitez modifier.
  2. Modifiez les zones d'information du produit de retour expédié.
  3. Cliquez sur OK.
Supprimer un produit en cours d'expédition
  1. Sous Actifs à expédier, cliquez sur Supprimer en regard de l'élément que vous souhaitez supprimer.
  2. Cliquez sur OK.
