Utilisation des retours

Les retours ont valeur d'accord d'acceptation de produits retournés par un client pour réparation, remplacement, remboursement ou avoir La vue Détails du retour permet de travailler sur un retour donné, tandis que la vue Liste des retours affiche tous les retours dans un format de type feuille de calcul.

Certaines fonctions vous aident dans la gestion de vos retours :

  • Utilisez la vueDétails du retour pour ajouter ou modifier les informations sur un retour.
  • Utilisez la vueListe des retours pour afficher et gérer les groupes de retours.
