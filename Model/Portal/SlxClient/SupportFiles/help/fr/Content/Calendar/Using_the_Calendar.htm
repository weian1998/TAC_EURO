Utilisation de l'agenda

Utilisez l'agenda pour gérer votre temps et assurer le suivi des activités et des événements que vous avez planifiés. L'agenda Sage SalesLogix affiche à la fois les activités et événements ouverts et terminés en fonction des options de l'agenda. Vous pouvez afficher votre planification par vue journalière, hebdomadaire ou mensuelle. Vous pouvez utiliser le menu contextuel de l'agenda pour planifier, modifier, terminer ou supprimer des activités et des événements.

Les événements apparaissent en haut de chaque jour pour lequel ils sont planifiés et s'étendent de la date de début à la date de fin. Les activités non confirmées apparaissent dans l'agenda avec des lignes diagonales blanches dans l'en-tête de l'activité. Utilisez la section Activités sans horaire défini pour afficher les activités devant commencer pendant la période sélectionnée mais qui n'ont pas d'horaire de début spécifique.

En fonction de vos droits d'accès, vous pouvez afficher les agendas et les activités des autres utilisateurs Sage SalesLogix. Une couleur est attribuée à chaque utilisateur pour l'aider à identifier les activités correspondant à chaque utilisateur. Si une activité est planifiée pour plusieurs utilisateurs et que vous êtes le responsable ou un des membres, elle affichera votre couleur. Si vous n'êtes ni membre ni responsable de l'activité, alors la couleur du responsable de l'activité apparaîtra.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Procédure

Afficher l'agenda
  1. Dans la barre de navigation, cliquez sur Agenda.
  2. Sélectionnez le type d'agenda à afficher.
    • Vue Jour - Affiche une vue temporelle dynamique pour la journée. Répertorie toutes les activités et événements planifiés pour cette journée.
    • Vue Semaine de travail - Affiche uniquement les jours de votre semaine de travail comme défini dans vos options utilisateur. Répertorie les activités et événements planifiés pour chaque jour.
    • Vue Semaine - Affiche chaque jour de la semaine sélectionnée. Répertorie les activités et événements planifiés pour chaque jour.
    • Vue Mois - Affiche l'agenda au format mensuel, avec la liste des activités quotidiennes planifiées.
  3. Pour afficher l'agenda d'un autre utilisateur, sélectionnez le nom de cet utilisateur dans la liste déroulante.
Afficher les activités sans horaire défini
  1. Dans le volet Tâches, affichez la section Activités sans horaire défini. La section Activités sans horaire défini affiche toutes les activités sans horaire défini planifiées pendant la période sélectionnée de l'agenda. Par exemple, jour, semaine ou mois.
    Pour trier une colonne, cliquez sur l'en-tête de la colonne. Pour développer la colonne, cliquez sur le séparateur de colonne et maintenez le bouton de la souris enfoncé tout en faisant glisser le séparateur jusqu'à la taille voulue.
  2. Cliquez sur le lien Type pour ouvrir une activité.
Afficher les activités sans horaire défini terminées
  1. Vérifiez que l'option de l'agendaAfficher l'historique sur l'agenda est définie sur Oui.
  2. Dans le volet Tâches, affichez la section Activités sans horaire défini terminées. La section Activités sans horaire défini terminées affiche toutes les activités sans horaire avec une date de fin dans la période sélectionnée de l'agenda. Par exemple, jour, semaine ou mois.
  3. Cliquez sur le lien Type pour ouvrir une activité.
Afficher les détails de l’activité
  • Faites passer votre souris sur une activité pour afficher les détails importants.
  • Cliquez avec le bouton droit de la souris sur l'activité, puis cliquez sur Ouvrir l'activité pour ouvrir la vue des détails.
Afficher les détails de l’événement
  • Faites passer votre souris sur un événement pour afficher les détails importants.
  • Cliquez avec le bouton droit de la souris sur l'événement, puis cliquez sur Ouvrir l'événement pour ouvrir la vue des détails.
Passer d'une date à une autre
  • Pour revenir à la date actuelle depuis n'importe quelle autre date, cliquez sur Aujourd'hui.
  • Utilisez l'agenda de navigation dans la section Agenda du volet Tâches pour atteindre une autre date.
Visualiser les agendas d'autres utilisateurs
  • Dans le volet Tâches, sous Utilisateurs de l'agenda, sélectionnez l'utilisateur ou les utilisateurs que vous souhaitez afficher.

    Vous pouvez afficher jusqu'à quatre agendas, le vôtre compris.

Planifier une activité ou un événement

  1. Effectuez l'une des opérations suivantes :

    • Double-cliquez sur la vue Agenda.
      Vos options d'agenda déterminent quel type d'activité s'ouvre par défaut.
    • Cliquez avec le bouton droit de la souris sur l'agenda, puis sélectionnez Planifier un appel téléphonique, Planifier une réunion, Planifier une tâche à faire, Planifier une activité personnelle, ou Planifier un événement.
  2. Suivez les étapes pour planifier une activité ou un événement.
Modifier l'heure de début ou la durée d'une activité
  • Pour changer l'heure d'une activité, faites glisser l'activité vers sa nouvelle heure sur l'agenda.
  • Pour modifier la durée d'une activité, pointez votre souris sur le bord d'une activité et déplacez le bord de l'activité vers le haut ou le bas.
