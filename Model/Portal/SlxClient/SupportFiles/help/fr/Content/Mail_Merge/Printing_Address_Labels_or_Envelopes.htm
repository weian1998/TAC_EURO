Impression Etiquettes d'adresse ou enveloppes

Vous pouvez utiliser la fonction Publipostage pour créer et imprimer des étiquettes d'adresse et des enveloppes destinées à un seul ou à plusieurs contacts ou leads.

Si le menu Rédiger n'est pas disponible, vous devez télécharger l'intégration du bureau Sage SalesLogix. Vous pouvez installer l'intégration du bureau Sage SalesLogix, à partir de l'écran de connexion du client Web.. 
Pour créer et imprimer des étiquettes d'adresse ou des enveloppes
  1. Dans le menu Rédiger, pointez sur Etiquettes d'adresse.
  2. Dans la zone Fusionner avec, sélectionnez une des options de source.
      • Enregistrement actif : sélectionnez cette option pour utiliser l'enregistrement actif. L'information source peut contenir un contact pour l'enregistrement de compte, de contact, ou d'opportunité actif, ou pour un lead.
      • Groupe actif : sélectionnez cette option pour utiliser le groupe de contacts, de comptes, de leads ou d'opportunités actuellement actif. Les informations sur la source sont les noms des contacts, les noms des comptes; les noms des leads et les adresses.
      • Groupe spécifique : sélectionnez un groupe dans la liste. Les informations de source sont les noms de contacts, les noms de comptes, les noms de leads et les adresses pour le groupe.
      • Spécifique : sélectionnez Compte ou Opportunité dans la liste puis utilisez la recherche pour trouver un enregistrement. Compte - Les informations sur la source sont les noms des contacts, le nom du compte et les adresses. Opportunité - Les informations sur la source sont les noms des contacts, le nom du compte et les adresses associées à l'opportunité.
  3. Si ceci est applicable, sélectionnez Ne PAS envoyer aux contacts portant la mention « Ne pas démarcher ».
  4. Pour obtenir une estimation du nombre d'étiquettes nécessaires, cliquez sur Compter pour afficher les informations correspondantes.
  5. Dans la zone Type d'étiquette, sélectionnez une étiquette ou une enveloppe.
  6. Dans l'afficheur de rapports, cliquez sur le bouton Imprimer.
  7. Configurez les paramètres de votre imprimante, insérez des étiquettes ou des enveloppes dans l'imprimante sélectionnée, puis cliquez sur OK.
Pour imprimer une étiquette d'adresse ou une enveloppe pour une lettre
  1. Suivez les étapes de rédaction d'une lettre.
  2. Clique sur Oui dans la boîte de dialogue Créer une étiquette d'adresse.
  3. Cliquez sur la flèche du menu déroulant Type d'étiquette, et sélectionnez une étiquette ou une enveloppe dans la liste.
  4. Dans l'afficheur de rapports, cliquez sur le bouton Imprimer.
  5. Configurez les paramètres de votre imprimante, insérez des étiquettes ou des enveloppes dans l'imprimante sélectionnée, puis cliquez sur OK.
Pour imprimer des étiquettes ou des enveloppes pour le publipostage
  1. Suivez les étapes d'utilisation du publipostage.
  2. Dans la zone Sortie de l'onglet Options de fusion, sélectionnez Imprimante.
  3. Sélectionnez Créer des étiquettes d'adresse, puis cliquez sur Configurer les étiquettes.
  4. Dans la zone Type d'étiquette, sélectionnez une étiquette ou une enveloppe.
  5. Cliquez sur OK.
  6. Pour sélectionner une autre imprimante, cliquez sur Propriétés de l'imprimante, sélectionnez une autre imprimante dans la liste, puis cliquez sur OK.
  7. Une fois que vous avez configuré vos options de fusion et d'historique, cliquez sur Fusionner.
  8. S'il est nécessaire de sélectionner une adresse pour le modèle ou si le contact a plusieurs adresses, choisissez une adresse dans la boîte de dialogue Sélectionner le type d'adresse, puis cliquez sur OK.
  9. A l'invite du système, insérez des étiquettes ou des enveloppes dans l'imprimante, puis cliquez sur OK.
