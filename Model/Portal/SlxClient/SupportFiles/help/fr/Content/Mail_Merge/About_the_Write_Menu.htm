Utilisation du menu Rédiger

Vous pouvez utiliser le menu Rédiger de Sage SalesLogix pour effectuer un publipostage ou pour envoyer un e-mail. Vous pouvez communiquer avec un seul contact ou lead ou avec un groupe de contacts ou de leads.

Si le menu Rédiger n'est pas disponible, vous devez télécharger l'intégration du bureau Sage SalesLogix. Vous pouvez installer l'intégration du bureau Sage SalesLogix, à partir de l'écran de connexion du client Web.. 

Utilisez le menu Rédiger pour :

  • rédiger un e-mail à un contact ou à un lead ;
  • Ecrire un e-mail, une lettre, ou un fax à l'aide d'un modèle
  • utiliser le moteur de publipostage pour envoyer une correspondance à un groupe de contacts, de leads, de comptes, etc.
  • créer et imprimer des étiquettes d'adresse ou des enveloppes ;
  • créer, modifier et supprimer des modèles.

Les options de menu Rédiger un E-mail/Lettre/Fax à partir d'un modèle contiennent les modèles utilisés le plus récemment . Lors de la configuration de vos options SalesLogix, vous ou votre administrateur système devez définir le nombre d'éléments à afficher.

Si une liste de modèles est grisée, vous risquez de ne pas pouvoir accéder à l'élément de menu. Si vous avez besoin d'y accéder, contactez votre administrateur système.

Remarques sur le publipostage

  • Microsoft Word est déjà ouvert : Si MS Word est déjà ouvert, un message apparaît vous demandant de le fermer avant que le publipostage ne puisse se poursuivre.
  • Sortie d'e-mail HTML : Le publipostage utilise le format HTML de Microsoft Internet Explorer pour les e-mails. Lorsque certains destinataires ne peuvent pas lire les messages HTML, les messages doivent être envoyés sous forme de message MIME. Quant aux graphiques et aux informations d'en-tête et de pied de page, MS Word ne les affiche pas dans une page HTML, mais il enregistre les données correspondantes.
  • Fusionner avec une table des matières: Il est impossible de fusionner avec un modèle qui contient une table des matières. Il est conseillé de créer la table des matières une fois le processus de fusion terminé. Utilisez la fonction de modification de la sortie du publipostage avant l'envoi, puis insérez la table des matières.
  • Placer des champs de fusion dans des zones de texte: ll est impossible de placer des champs de fusion Sage SalesLogix dans une zone de texte. La fusion ne sera pas exécutée correctement.
