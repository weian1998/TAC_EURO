Ajout/modification d'une autre adresse

Vous pouvez utiliser la boîte de dialogue Adresse pour modifier les informations d'adresse. Vous pouvez ajouter autant d'adresses personnalisées que nécessaire pour un même contact ou compte. Cependant, vous ne pouvez sélectionner qu'une seule adresse de facturation principale et qu'une seule adresse d'expédition principale.

Pour ouvrir la boîte de dialogue Adresse

  • Effectuez l'une des opérations suivantes :
  • Dans la vue Détails du contact, Détails du compte ou Détails du lead, cliquez sur Adresses. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets, puis sélectionnez l'onglet dans la liste affichée. Cliquez sur le lien Modifier de l'adresse à modifier ou cliquez sur le bouton Ajouter une nouvelle adresse pour ajouter une nouvelle adresse.
  • Cliquez sur le bouton AdresseModifier dans la vue Informations sur les contacts/adresses ou Insérer un lead.
  • Cliquez sur le bouton AdresseModifier dans la vue Détails du contact, Détails du compte ou Détails du lead.
  • Cliquez sur le bouton AdresseModifier de la vue Détails du bon de commande dans l'onglet Adresse de facturation/expédition.

Pour ajouter ou modifier

  1. Dans la zone Description, cliquez sur bouton en forme de flèche, sélectionnez une description pour l'adresse, puis cliquez sur OK.
  2. Sélectionnez ou désélectionnez les cases Principale,Facturation principale, ou Expédition principale selon vos besoins.
  3. Saisissez ou modifiez l'adresse dans les zones prévues à cet effet.
  4. Dans les zones Ville, Etat, ou Pays, cliquez sur le bouton en forme de flèche et sélectionnez un élément dans la liste, puis cliquez sur OK.
  5. Cliquez dans la zone Code postal et saisissez les modifications éventuelles.
  6. Cliquez dans la zone Région et saisissez les modifications éventuelles.
  7. Cliquez dans la zone A l'attention de puis saisissez la formule d'appel à utiliser dans la correspondance postale.
  8. Cliquez sur OK.
