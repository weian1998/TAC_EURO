Affichage et modification des éléments et notes d'historique

Utilisez la vue Détails de l'historique pour afficher des informations détaillées sur une activité, un e-mail, une demande de documentation, une note, des modifications de la base de données, un script Visual Basic ou une question terminés.

Vous pouvez modifier uniquement les détails des éléments d'historique que vous avez créés. Toutefois, vous pouvez planifier une activité de suivi et y reporter des notes ou des pièces jointes.

Pour afficher ou modifier

  1. Ouvrez les vues Détails des Contact, Lead, Compte ou Opportunité avec la note ou l'élément d'historique associé.
  2. Cliquez sur Onglet Notes/Historique, puis cliquez sur le lien du Type d'historique de l'élément d'historique à modifier. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  3. Selon le type de l'élément d'historique, vous pouvez modifier les informations suivantes :
    • Dans les zones Résultat et Catégorie, cliquez sur le bouton en forme de flèche et sélectionnez un élément dans la liste puis cliquez sur OK.

      La vue Détail des notes ne comporte pas de champ Résultat.

  4. Dans la zone Notes, saisissez les informations supplémentaires éventuelles.
  5. Le cas échéant, planifiez une activité de suivi.
    1. Dans la zone Suivi, sélectionnez le type d'activité à planifier.
    2. Cochez la case Reporter les notes pour copier des notes de cette activité vers celle de suivi.
    3. Cochez la case Reporter les notes pour copier des notes de cette activité vers celle de suivi.
  6. Si cette option est disponible, cliquez sur Ouvrir l'e-mail pour ouvrir l'e-mail d'origine.
    Cette option n'est disponible que pour les e-mails envoyés à l'aide du bouton Envoyer SLX ou comme partie intégrante d'un publipostage où l'option Joindre l'e-mail à chaque contact a été sélectionnée.
  7. Cliquez sur l'onglet Pièces jointes pour ajouter ou afficher un document ou une URL.

    Le bouton Envoyer SLX de MS Outlook enregistre les pièces jointes à l'aide d'une des méthodes suivantes :

    • Les pièces jointes sont enregistrées en tant que fichiers individuels
    • Une seule pièce jointe qui contient le message électronique et toutes les pièces jointes

    La méthode utilisée est déterminée par votre administrateur.

  8. Cliquez sur OK.
    Si vous avez planifié une activité de suivi, la boîte de dialogue Planifier une activité apparaît afin que vous la complétiez.
