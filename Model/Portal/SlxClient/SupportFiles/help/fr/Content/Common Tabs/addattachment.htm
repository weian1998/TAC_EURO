Ajout d'une pièce jointe

Vous pouvez ajouter une pièce jointe à un enregistrement. Tous les fichiers sont stockés dans le répertoire des pièces jointes partagées et synchronisés avec les bases de données distantes.

Par défaut, il est impossible de joindre des fichiers dont la taille dépasse 4 Mo. Si vous joignez un fichier ou un groupe de fichiers dont la taille dépasse 4 Mo, vous risquez de recevoir un message d'erreur ou d'avertissement. Pour plus d'informations, contactez votre administrateur.

Procédure

Joindre un fichier
  1. Ouvrez l'onglet Pièces jointes.
  2. Cliquez sur le bouton Ajouter un fichier.
  3. Si la boîte de dialogue Joindre un fichier s'ouvre, cliquez sur Sélectionner un fichier.
  4. Sélectionnez le fichier que vous souhaitez ajouter, puis cliquez sur Ouvrir.
  5. Dans la zone Description, saisissez une description de la pièce jointe si le nom de fichier n'est pas explicite.
  6. Cliquez sur OK.
Joindre un fichier à l'aide du copier-coller

  • Si vous utilisez Internet Explorer, cette fonctionnalité nécessite l'installation du module d'intégration du bureau.
  • L'outil glisser-déposer n'est pas disponible sur les onglets Pièces jointes de l'historique ou de l'activité.
  1. Naviguez jusqu'au fichier que vous souhaitez ajouter.
  2. Faites glisser le fichier vers la vue Détails de l'enregistrement approprié.

La boîte de dialogue Ajouter des pièces jointes s'ouvre.

    Si vous êtes invité à installer l'intégration du bureau Sage SalesLogix et que vous souhaitez utiliser cette fonction, cliquez sur Oui et suivez les étapes pour installer l'intégration du bureau Sage SalesLogix. Pour effectuer l'installation, vous serez invité à fermer toutes les fenêtres de navigateur et ainsi que Microsoft Outlook.
  1. Dans la zone Description, saisissez une description du fichier si la description par défaut n'est pas claire.

  2. Cliquez sur OK.

Joindre un document Google
  1. Ouvrez l'onglet Pièces jointes.
  2. Cliquez sur le bouton Ajouter un document Google.
  3. En fonction de vos paramètres de sécurité, il peut être nécessaire de :
    • autoriser les fenêtres contextuelles dans la session de votre navigateur.
    • Vous connecter à votre compte Google et autoriser l'accès.
  4. Dans la liste, double-cliquez sur le document à ajouter.
  5. Lorsque vous avez terminé, fermez la boîte de dialogue.
Joindre une URL
  1. Ouvrez l'onglet Pièces jointes.
  2. Cliquez sur le bouton Ajouter une URL.
  3. Dans la zone URL, saisissez l'adresse URL complète.
  4. Dans la zone Description, saisissez une description.
  5. Cliquez sur OK.
    Il faut à la fois une description et une URL.
