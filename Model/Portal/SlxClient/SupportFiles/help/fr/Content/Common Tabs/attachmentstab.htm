Onglet Pièce jointe

Les pièces jointes sont des fichiers qui sont associés à un enregistrement. L'onglet Pièces jointes figure sur de nombreuses vues détaillées, notamment la vue Détails de l'activité, et affiche la liste des pièces jointes de l'enregistrement sélectionné.  

Le bouton Envoyer SLX de MS Outlook enregistre les pièces jointes à l'aide d'une des méthodes suivantes :

  • Les pièces jointes sont enregistrées en tant que fichiers individuels
  • Une seule pièce jointe qui contient le message électronique et toutes les pièces jointes

La méthode utilisée est déterminée par votre administrateur.

Lorsque vous modifiez un fichier joint, vous devez mettre à jour la pièce jointe pour que les utilisateurs distants puissent accéder à la dernière version.

Les colonnes sont les suivantes : Pièce jointe, Modifié par, Date/Heure de modification, Taille et Extension de fichier. Cliquez sur un en-tête de colonne pour trier la liste en fonction de celle-ci.

  • Si vous modifiez une pièce jointe, vous devez télécharger le nouveau fichier afin que ces modifications soient disponibles dans SalesLogix. Toute modification effectuée se fait sur une copie locale du fichier et non sur le fichier joint.
  • Si un participant à une activité a accès à l'agenda du responsable de l'activité, il peut ajouter ou modifier des pièces jointes. S'il n'a pas accès à l'agenda, il peut seulement afficher, copier ou enregistrer les pièces jointes.

Procédure

Ouvrir l'onglet Pièces jointes
  • Dans une vue détaillée ou dans la vue Détails de l'activité, cliquez sur l'onglet Pièces jointes. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
Filtrez la liste des pièces jointes
  1. Cliquez sur le bouton Filtrer.
  2. Effectuez une ou plusieurs des actions suivantes :
    • dans la zone Pièce jointe, saisissez un ou plusieurs caractères contenus dans le nom de la pièce jointe. Plus vous saisirez de caractères, plus la liste renvoyée sera courte.
    • Utilisez les zones Modifier la plage de dates pour sélectionner une plage de dates pendant laquelle la pièce jointe a été mise à jour pour la dernière fois.
  3. Cliquez sur Appliquer pour filtrer la liste ou Réinitialiser pour effacer vos options de filtrage.
    Cliquez de nouveau sur le bouton Filtrer pour masquer les filtres.
Ouvrir une pièce jointe
  1. Cliquez sur le nom du fichier que vous souhaitez ouvrir.
  2. Si cela vous est demandé, sélectionnez l'option Ouvrir et si nécessaire, sélectionnez un programme dans la liste.

  3. Si nécessaire, cliquez sur OK.

Enregistrement d'une pièce jointe
  1. Cliquez sur le nom du fichier que vous souhaitez enregistrer.
  2. Si cela vous est demandé, sélectionnez l'option Enregistrer.
  3. Si nécessaire, cliquez sur OK.
