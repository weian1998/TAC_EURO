Utilisation de la bibliothèque

La bibliothèque est le lieu de stockage central des informations de votre entreprise (documents, URL, etc.) et permet de distribuer aisément des informations aux utilisateurs distants. Les fichiers sont organisés en dossiers et classés par nom. Cliquez sur les en-têtes de colonne pour trier la liste des fichiers de la bibliothèque.

A l'aide de client Web de Sage SalesLogix, vous pouvez demander puis télécharger un fichier de la bibliothèque lorsque vous en avez besoin. La bibliothèque affiche le nom du fichier, sa taille, ses dates de création, de révision et de péremption, ainsi qu'une brève description du fichier.

La bibliothèque de votre société peut contenir des informations sur les produits, les règles et les procédures, des présentations, des adresses de sites Web, etc.

Seul l'administrateur ou un utilisateur administratif peut ajouter des fichiers à la bibliothèque.

Procédure

Ouvrez la bibliothèque
  • Dans la barre de navigation, cliquez sur Bibliothèque.
Télécharger un fichier de bibliothèque
  1. Dans l'arborescence, cliquez sur les signes « + » pour développer le dossier contenant le fichier à ouvrir.
  2. Dans la liste, cliquez sur le nom du fichier à télécharger.

  3. Effectuez l'une des opérations suivantes :

    • Ouvrez le fichier
    • Enregistrez le fichier

Si votre navigateur ouvre le fichier au lieu de l'enregistrer sur votre ordinateur, modifiez les paramètres par défaut du navigateur afin qu'il enregistre automatiquement les fichiers au lieu de les ouvrir. Vous pouvez également maintenir la touche [Maj] enfoncée, puis cliquer sur le lien pour télécharger sans les ouvrir tous les fichiers à l'exception de ceux comportant une extension. Les fichiers HTML sont ouverts dans une autre fenêtre.

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.L'administrateur système et les utilisateurs dotés d'un rôle administrateur peuvent accéder à ces fonctionnalités.
Ajouter un document à la bibliothèque
  1. Sélectionnez le dossier dans lequel vous souhaitez ajouter l'élément.
  2. Cliquez sur Ajouter des fichiers.
  3. Sélectionnez le ou les fichiers à ajouter.
  4. Cliquez sur Ouvrir.
Ajouter un document à la bibliothèque à l'aide de l'outil glisser-déposer

  • Il est possible que vous n'ayez pas les droits d'accès à cette fonctionnalité. Pour plus d'informations, veuillez contacter votre administrateur.
  • Si vous utilisez Internet Explorer, cette fonctionnalité nécessite l'installation du module d'intégration du bureau.
  1. Naviguez jusqu'au fichier que vous souhaitez ajouter.
  2. Dans la barre de navigation, cliquez sur Bibliothèque.
  3. Dans l'arborescence, cliquez sur les signes « + » pour développer le dossier jusqu'à l'emplacement où vous souhaitez ajouter le fichier.
  4. Faites glisser le fichier dans la vue Bibliothèque.
  5. Dans la zone Description, saisissez une description du fichier si la description par défaut n'est pas claire.
  6. Cliquez sur OK.

Vous ne pouvez pas faire glisser un dossier et son contenu vers la bibliothèque. Vous ne pouvez faire glisser que des fichiers.
