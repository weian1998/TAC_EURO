Ajouter, renommer ou supprimer un dossier de bibliothèque

Pour une meilleure organisation, vous pouvez regrouper des documents dans des dossiers de bibliothèque.

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.L'administrateur système et les utilisateurs dotés d'un rôle administrateur peuvent accéder à ces fonctionnalités.

Procédure

Ajouter un dossier de bibliothèque
  1. Dans la barre de navigation, cliquez sur Bibliothèque.
  2. Sélectionnez le dossier sous lequel vous souhaitez ajouter l'élément et cliquez sur Ajouter un dossier.
  3. Dans la boîte de dialogue Nouveau dossier, entrez un nom pour le nouveau dossier.

  4. Cliquez sur OK.

Renommer un dossier
  1. Dans la barre de navigation, cliquez sur Bibliothèque.
  2. Sélectionnez le dossier que vous souhaitez renommer et cliquez sur Modifier le nom du dossier.
  3. Dans la boîte de dialogue, entrez un nouveau nom pour le dossier.

  4. Cliquez sur OK.

Supprimer un dossier
  1. Dans la barre de navigation, cliquez sur Bibliothèque.
  2. Sélectionnez le dossier à supprimer, puis cliquez sur Supprimer le dossier.
  3. Cliquez sur Oui pour confirmer cette suppression.

Tous les dossiers et fichiers contenus dans ce dossier seront également supprimés.
