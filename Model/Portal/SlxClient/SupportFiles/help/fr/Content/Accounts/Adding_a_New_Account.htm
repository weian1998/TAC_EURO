Ajout d'un nouveau compte

Vous pouvez utiliser la boîte de dialogue Insérer un nouveau compte pour ajouter un nouveau compte.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Pour ajouter

  1. Dans le menu Nouveau, cliquez sur Compte.
  2. Renseignez les zones d'informations du compte restantes, conformément aux règles commerciales de votre société.
  3. Cliquez sur Enregistrer.
