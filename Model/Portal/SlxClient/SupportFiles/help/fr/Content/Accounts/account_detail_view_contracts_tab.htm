Onglet Contrats de la vue Détails du compte

L'onglet Contrats de la vue Détails du compte vous permet de voir les informations spécifiques d'un contrat pour le compte dans un emplacement. Vous pouvez utiliser cet onglet pour vérifier les informations du contrat plutôt que d'ouvrir chaque contrat séparément.

En outre, pour visualiser les informations relatives à un contrat, vous pouvez ouvrir un enregistrement de contrat pour le modifier ou insérer un nouveau contrat au compte.

Pour utiliser l'onglet Contrats

  1. Ouvrez la vue Détails du compte.
  2. Cliquez sur l'onglet Contrats. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  3. Effectuez l'une des actions suivantes :
    • Pour voir les notes de contrat, sélectionnez le contrat à prévisualiser.
    • Pour ouvrir le contrat cliquez sur le numéro de référence du contrat à ouvrir.
      La vue Détails du contrat s'ouvre.
    • Pour créer un nouveau contrat, cliquez sur Créer un contrat.
