Utilisation de l'onglet Actifs de compte

Utilisez cet onglet pour voir une liste des actifs associés à des opportunités gagnées ou ajoutés à un compte.

Procédure

Ouvrir l'onglet Actifs de compte
  1. Ouvrez la vue Détails du compte.
  2. Cliquez sur l'onglet Actifs. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.

Ajouter un actif

Modifier un actif

Supprimer un actif

Cette vue contient les colonnes suivantes :

  • Nom du produit : affiche le nom du produit.
  • Code article : affiche le Code article de l'actif.
  • Numéro de série : affiche un numéro de série unique.
  • Qté : affiche le numéro de cet actif détenu par le compte.
  • Contact : affiche le contact du compte propriétaire de l'actif.
  • Emplacement : affiche l'emplacement de l'actif.
  • Actif n° : affiche le code de l'actif.
