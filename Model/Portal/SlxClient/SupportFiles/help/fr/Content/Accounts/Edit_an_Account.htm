Modification d'un compte

Utilisez la vue Détails du compte pour modifier les informations d'un compte. Pour annuler vos modifications, cliquez sur Réinitialiser avant de cliquer sur le bouton Enregistrer.

Pour modifier

  1. Cliquez sur Comptes dans la barre de navigation.
  2. Recherchez le compte à modifier puis cliquez sur le nom du compte.
  3. Dans la vue Détails du compte, modifiez les informations sur le compte.
  4. Cliquez sur Enregistrer.

Vous serez peut-être invité à mettre à jour les contacts ayant des informations concordantes. Sélectionnez les boîtes de dialogue appropriées, puis cliquez sur OK pour mettre à jour ces informations dans les contacts associés.
