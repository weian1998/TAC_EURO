Vue Détails d'association de comptes

Utilisez les associations pour créer une relation entre les comptes

Par exemple, si vous travaillez avec une entreprise ayant de nombreuses divisions, vous pouvez créer un compte pour chaque division (comptes divisionnaires) et créer un compte pour représenter l'entreprise en son entier (compte parent). Vous pouvez ensuite associer chaque compte divisionnaire au compte parent. S'il existe des contacts dans différentes divisions de la maison mère, vous pouvez alors associer les contacts.

Un compte particulier peut comporter plus d'une association de compte. Utilisez la vue Détail des associations de comptes pour afficher des informations détaillées sur les relations de ce compte avec un autre compte. La vue Association contient les noms des comptes, ainsi qu'une brève description et toutes les notes.

Pour associer les comptes

  1. Dans une vue Détails du Compte, cliquez sur l'onglet Associations. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  2. Cliquez sur le bouton Ajouter une association.
  3. Cliquez sur le bouton Rechercher et cherchez le compte à associer au premier compte. Dans la zone Compte, saisissez abbott, puis cliquez sur le Bouton Rechercher.
  4. Cliquez sur bouton en forme de flèche sous le compte puis sélectionnez l'élément de la liste qui correspond le mieux à la relation du premier compte.
  5. Cliquez sur bouton en forme de flèche sous le deuxième compte puis sélectionnez l'élément de la liste qui correspond le mieux à la relation du deuxième compte.
  6. Dans les zones Description, tapez la description de chaque compte.

  7. Cliquez sur OK.

Pour modifier

  1. Dans une vue Détails du Compte, cliquez sur l'onglet Associations.
  2. Cliquez sur le lien Modifier de l'association que vous souhaitez afficher ou modifier.
  3. Effectuez vos modifications.

  4. Cliquez sur OK.
