Onglet Opportunités des revendeurs

Dans l'onglet Opportunités des revendeurs de la vue Détails du compte, vous pouvez afficher les informations relatives aux opportunités des revendeurs. L'onglet Opportunités des revendeurs contient la liste des opportunités pour lesquelles le compte est un revendeur.

Procédure

Ouvrez l'onglet Opportunités des revendeurs

  1. Ouvrez la vue Détails du compte.
  2. Cliquez sur l'onglet Opportunités des revendeurs. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
Ajouter une opportunité
  1. Cliquez sur le bouton Insérer une nouvelle opportunité.
    La boîte de dialogue Insérer une opportunité s'ouvre.
  2. Saisissez les informations sur les opportunités dans les zones appropriées.

  3. Cliquez sur Enregistrer

L'onglet Opportunités des revendeurs contient les informations suivantes :

  • Ouvert : le nombre d'opportunités et le potentiel total de toutes les opportunités ouvertes pour le compte.
  • Clôturée - Gagnée - Le nombre d'opportunités et le potentiel total de toutes les opportunités fermées et gagnées pour le compte.
  • Clôturée - Perdue - Le nombre d'opportunités et le potentiel total de toutes les opportunités fermées et perdues pour le compte.
  • Inactive - Le nombre d'opportunités et le potentiel total de toutes les opportunités inactives pour le compte.

Les colonnes de la grille de l'onglet Opportunités des revendeurs contiennent les informations suivantes :

  • Nom de l'opportunité - La description de l'opportunité. Cliquez sur le lien hypertexte pour afficher la vue Détails de l'opportunité.
  • Etat : état de l'opportunité.
  • Période de clôture est. : date de clôture estimée de l'opportunité.
  • Potentiel : chiffre d'affaires global que l'opportunité est susceptible de générer.
  • Probabilité : la probabilité de gagner l'opportunité.
  • Jours d'ouverture : le nombre de jours pendant lesquels l'opportunité a été ouverte.
