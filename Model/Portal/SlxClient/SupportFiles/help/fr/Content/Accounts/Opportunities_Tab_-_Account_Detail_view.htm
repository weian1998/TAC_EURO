Onglet Opportunités - Vue Détails du compte

Vous pouvez afficher les informations sur les opportunités associées au compte sélectionné. Vous pouvez aussi créer une nouvelle opportunité pour le compte.

Procédure

Ouvrez l'onglet des opportunités

  1. Ouvrez la vue Détails du compte.
  2. Cliquez sur l'onglet Opportunités. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
Ajouter une opportunité
  1. Cliquez sur le bouton Insérer une nouvelle opportunité.
    La boîte de dialogue Insérer une opportunité s'ouvre.
  2. Saisissez les informations sur les opportunités dans les zones appropriées.

  3. Cliquez sur Enregistrer

L'onglet Opportunités contient les informations suivantes :

  • Ouvert : affiche le nombre d'opportunités et le potentiel total de toutes les opportunités ouvertes du compte.
  • Clôturée - Gagnée : affiche le nombre d'opportunités et le potentiel total de toutes les opportunités clôturées et gagnées du compte.
  • Clôturée - Perdue : affiche le nombre d'opportunités et le potentiel total de toutes les opportunités clôturées et perdues du compte.
  • Inactif : affiche le nombre d'opportunités et le potentiel total de toutes les opportunités inactives du compte.

Les colonnes de la grille de l'onglet Opportunités sont les suivantes :

  • Nom de l'opportunité : affiche la description de l'opportunité.
  • Etat : affiche l'état de l'opportunité.
  • Potentiel : affiche le chiffre d'affaires total pouvant être généré par l'opportunité.
  • Probabilité : affiche la probabilité que l'opportunité soit gagnée.
  • Ouvert depuis : affiche le nombre de jours s'étant écoulés depuis l'ouverture de l'opportunité.
