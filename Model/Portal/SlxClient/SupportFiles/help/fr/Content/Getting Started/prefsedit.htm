Modification des options utilisateur

Sage SalesLogix vous apporte les outils vous permettant de modifier certaines parties de l'interface afin que vous puissiez travailler de la manière qui vous convient le mieux Les Options utilisateur vous permettent de configurer vos préférences pour des fonctionalités telles que la vue qui s'affiche lorsque vous démarrez le client Web Sage SalesLogix et les alertes. Les paramètres que vous saisissez sont enregistrés comme étant ceux de l'utilisateur qui est connecté et ne sont pas visibles pour les autres utilisateurs de Sage SalesLogix.

Pour modifier les options utilisateur

  1. Dans le menu Outils, cliquez sur Options.
  2. Cliquez sur l'onglet des options que vous souhaitez modifier.
  3. Cliquez sur l'onglet Général pour modifier la vue de démarrage initiale, le propriétaire ou l'équipe par défaut, les modèles de publipostage de base, le nombre de modèles dans la liste des éléments récemment utilisés ou les options de messagerie.
  4. Cliquez sur l'onglet Groupe pour sélectionner le groupe d'enregistrement à utiliser comme votre groupe par défaut pour chaque type d'enregistrement.
  5. Cliquez sur l'onglet Agenda pour définir les options de la vue Agenda. Vous pouvez définir les informations qui s'affichent sur votre agenda, la vue par défaut ou les heures de début et de fin de la journée.
  6. Cliquez sur l'onglet Modifier le mot de passe pour changer votre mot de passe.
  7. Cliquez sur l'onglet Opportunités pour définir les valeurs par défaut des nouvelles opportunités.
  8. Cliquez sur l'onglet Alertes pour définir les paramètres par défaut afin de déterminer la manière dont vous serez alerté.
  9. Cliquez sur l'onglet Activités pour définir les options de la vue principale Activité, les activités de suivi ou les alarmes des activités.
  10. Cliquez sur Enregistrer.

Vous devez cliquer sur Enregistrer dans chaque onglet, faute de quoi les options sélectionnées ne seront pas enregistrées.
