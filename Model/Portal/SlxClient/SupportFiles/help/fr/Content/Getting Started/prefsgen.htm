Options générales

Utilisez l'onglet Général pour définir la vue par défaut lors de votre première connexion au ainsi que le propriétaire par défaut de tous les nouveaux enregistrements créés.

Pour définir les options générales

  1. Dans le menu Outils, cliquez sur Options.
  2. Cliquez sur l'onglet Général.
  3. Dans la zone Afficher au démarrage, sélectionnez la vue par défaut à afficher lorsque vous vous connectez au client Web (Nouveautés ou Agenda, par exemple).
  4. Dans la zone Equipe/Propriétaire par défaut, cliquez sur Rechercher, sélectionnez l'utilisateur ou l'équipe qui doit être le propriétaire par défaut des nouveaux comptes puis cliquez sur OK.
  5. Si vous souhaitez être déconnecté automatiquement du client Web après une période d'inactivité, sélectionnez l'option Activer la déconnexion automatique puis saisissez le nombre de minutes d'inactivité.
  6. Si vous souhaitez recevoir une notification lorsque des modifications n'ont pas été enregistrées sur une page, cochez la case Avertissement de données non enregistrées. Cette option est sélectionnée par défaut. Décochez la case si vous ne souhaitez pas recevoir de notification.
  7. Pour les devises multiples uniquement : Pour sélectionner un type de devise, cliquez sur le bouton de Ma devise, Rechercher, sélectionnez une devise dans la liste puis cliquez sur OK.
  8. Dans la zone Groupe de contacts Intellisync, sélectionnez un groupe de contacts ad hoc pour que le processus de synchronisation puisse avoir lieu.
    Si le groupe ad hoc auquel vous souhaitez synchroniser vos contacts n'apparaît pas dans la liste, vous devez créer le groupe SyncSalesLogix.
  9. Sélectionnez l'option Afficher les cases de la vue Liste si vous voulez que toutes vos vues de liste intègrent une colonne de cases à cocher.
  10. Sélectionnez les modèles de publipostage de base à utiliser lors de la création d'un modèle d'e-mail, de fax ou de lettre pour un nouveau contact.
    1. Cliquez sur la flèche de la liste déroulante Type, et sélectionnez Contact.
    2. Cliquez sur le bouton Modèle d'e-mail de base Rechercher .

    3. Sélectionnez un modèle et cliquez sur OK.

    4. Répétez les étapes b et c pour le Modèle de lettre de base et le Modèle de fax de base.

  11. Sélectionnez les modèles de publipostage de base à utiliser lors de la création d'un modèle d'e-mail, de fax ou de lettre pour un nouveau lead.

    Lors de la sélection de modèles de leads, sélectionnez uniquement ceux qui se terminent par "(Lead)".

    1. Cliquez sur la flèche de la liste déroulante Type, et sélectionnez Lead.
    2. Cliquez sur le bouton Modèle d'e-mail de base Rechercher .

    3. Sélectionnez un modèle et cliquez sur OK.

    4. Répétez les étapes b et c pour le Modèle de lettre de base et le Modèle de fax de base.
  12. Dans la zone Menu Composition - Modèles récents, saisissez le nombre de modèles à afficher dans la liste des derniers modèles utilisés.
  13. Dans la zone Fournisseur de fax, cliquez sur la flèche du menu déroulant et sélectionnez le fournisseur à utiliser pour les fax.
  14. Dans la zone Consigner dans l'historique, sélectionnez Invite si vous souhaitez que la boîte de dialogue Terminer l'e-mail s'affiche une fois que vous avez fait glisser et déposé un e-mail. Si vous souhaitez que le message soit enregistré dans l'historique sans que vous en soyez averti, sélectionnez Ne pas demander.

    Cette option ne concerne que le déplacement par glisser-déplacer des e-mails. Vous devez définir des options pour Envoyer SLX et Enregistrer dans l'historique dans le Gestionnaire de bureau.

  15. Pour installer le module d'intégration du bureau, cliquez sur Installer les améliorations.
  16. Pour les utilisateurs de Client Web hors ligne uniquement : Pour exécuter des rapports en étant hors-ligne, vérifiez que la case Utiliser ActiveReporting est cochée.
  17. Cliquez sur Enregistrer.
