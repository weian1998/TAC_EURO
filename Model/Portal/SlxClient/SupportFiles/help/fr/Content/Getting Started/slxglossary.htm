Glossaire

Droits d'accès

Capacité de l'utilisateur à accéder aux données et à utiliser les fonctionnalités du client réseau Sage SalesLogix. L'administrateur système crée un profil de sécurité pour chaque utilisateur dans l'Administrateur Sage SalesLogix, ce qui détermine les droits d'accès de chaque utilisateur dans le client Sage SalesLogix.

Compte

Société avec laquelle il existe une relation commerciale actuelle ou potentielle. Un compte appartient à un utilisateur individuel, à une équipe d'utilisateurs ou à tous les utilisateurs (en lecture seule).

Responsable de compte

Utilisateur responsable du compte. Le responsable du compte n'a pas besoin d'être le propriétaire du compte ou un membre de l'équipe à laquelle appartient le compte.

Activité

Un appel téléphonique, une réunion ou toute autre tâche (à effectuer ou activité personnelle) planifiées et exécutée dans Sage SalesLogix.

Ad hoc

Groupes basés sur les enregistrement sélectionnés. A la différence des groupes fondés sur des instructions spécifiques, les groupes ad hoc ne changent pas. Les enregistrements restent identiques sauf si vous ajoutez des enregistrements au groupe ou en supprimez.

Actif

Un produit vendu à votre client. Les actifs sont des produits physiques que le client possède et qui peuvent ou non porter un numéro de série

Campagne

Action de marketing, telle qu'une campagne de publicité directe. Vous pouvez consulter les contacts ou leads qui ont été inclus dans la campagne et ceux qui y ont répondu.

Confirmation

Notification d'une activité dont vous êtes membre ou dirigeant. Vous recevez une notification lorsqu'un autre utilisateur planifie une activité pour vous ou lorsqu'une activité à laquelle vous participez est mise à jour. Si vous êtes ajouté en tant que participant, vous devez confirmer ou infirmer votre présence à l'activité. Si vous avez planifié une activité à laquelle vous avez convié d'autres participants, vous recevez une notification à chaque fois qu'un participant confirme ou infirme sa présence.

Contact

Un client associé à un compte avec lequel les utilisateurs interagissent. Les comptes peuvent contenir plusieurs contacts.

Contrat

Un contrat qui précisent les termes spécifiques de l'assistance après la vente du produit.

Cookie

Un cookie est un fichier d'information codée, stocké sur l'ordinateur d'un utilisateur et permettant d'identifier cet ordinateur lors de la visite en cours ou des prochaines visites d'un site Web.

Vue Détails

Affiche des informations plus précises sur un enregistrement spécifique.

Web déconnecté

Reportez-vous à la rubrique Web hors ligne.

Défaut

Un enregistrement décrivant un problème dans un produit ou un processus. Il est similaire à un ticket, mais il est généralement résolu par les personnes impliquées dans la conception ou la fabrication du produit.

Intellisync

Un programme distinct qui est installé et intégré à Sage SalesLogix permettant de partager vos activités et contacts Sage SalesLogix avec Microsoft Outlooket Lotus Notes

Lead

Les leads sont des clients potentiels non qualifiés. Vous pouvez planifier des activités pour les leads, les inclure dans un publipostage ou les ajouter à une campagne. Vous pouvez également importer des leads depuis une liste.

Source du lead

Méthode permettant de déterminer à quelle occasion votre société a trouvé un contact ou une opportunité (au cours d'un salon professionnel ou grâce à une publicité, par exemple).

Bibliothèque

Dossier contenant l'intégralité de la bibliothèque, qui est un référentiel d'informations contenant notamment les présentations, politiques et procédures.

Vue Liste

Affiche tous les enregistrements d'un certain type en format feuille de calcul. Les enregistrements peuvent être organisés en groupes et affichés en utilisant les onglets situés en haut de la vue.

Recherche

Outil vous permettant de rechercher un enregistrement spécifique ou plusieurs enregistrements ayant une caractéristique en commun (par exemple, comptes commençant par « Ca »).

Barre des menus

La barre de menus contient les menus Sage SalesLogix.

Mon Espace de travail

Voir Espace de travail Personnel.

Barre de navigation

Barre d'outils située à gauche de l'espace de travail contenant les boutons contrôlant l'ouverture de la vue principale.

Utilisateur réseau

Utilisateurs avec accès aux groupes de travail se connectant directement à la base de données du groupe de travail.

Client Web hors ligne

Permet aux utilisateurs de travailler sans connexion active à Internet. Les utilisateurs se connectent à leur base de données locale utilisant le client Web et emploient la synchronisation pour transférer les modifications entre la base de données de l'utilisateur du client Web déconnecté et la base de données du bureau principal.

Opportunité

Vente potentielle à un compte ou un contact.

Propriétaire

Utilisateur ou équipe d'utilisateurs avec des droits d'accès pour un compte. Les deux équipes système sont Tous les utilisateurs et Tous les utilisateurs (affichage uniquement).

En retard

Une activité est considérée comme échue lorsque la date de début programmée a expiré.

Activité personnelle

Engagement d'ordre privé, comme un rendez-vous chez le dentiste, que seul l'utilisateur l'ayant créé peut visualiser ou modifier.

Espace de travail personnel

Faisant également référence à Mon Espace de travail Mon espace de travail sera peut-être la première page que vous verrez lors de votre connexion au client Web Sage SalesLogix. Vous pouvez personnaliser votre espace de travail personnel et l'ouvrir quand vous le voulez à l'aide du bouton Acceuil de la barre de navigation.

Liste de sélection

Une liste de sélection contient des valeurs que vous pouvez sélectionner pour renseigner un champ.

Bureau distant

Groupe d'utilisateurs réseau dans la base de données locale d'une succursale. La base de données se synchronise avec le bureau principal, exactement comme un utilisateur distant.

Utilisateur distant

Utilisateur avec une base de données locale qui envoie et reçoit les modifications de la base de données par synchronisation. Un utilisateur distant peut travailler à partir d'un ordinateur portable ou de bureau. L'ordinateur de chaque utilisateur distant est considéré comme un client SalesLogix distant et un code site lui est attribué.

Sage SalesLogix offre des fonctionnalités adaptées aux utilisateurs distants. Les utilisateurs distants peuvent visualiser les modifications apportées à leur base de données à chaque synchronisation en utilisant l'option Nouveautés. Grâce à l'abonnement, les utilisateurs distants peuvent limiter la taille de leur base de données, ainsi que le temps nécessaire à la transmission des données synchronisées, en sélectionnant les comptes et contacts spécifiques qu'ils souhaitent stocker localement. Les planificateurs de tâches permettent aux utilisateurs distants de demander des rapports comportant les données qui ne sont pas stockées dans leur base de données locale.

Rapports

Un rapport est une présentation de données organisée. Un rapport est un outil de gestion utilisé pour fournir à l'équipe dirigeante les informations nécessaires à la gestion de la société.

Retour

Les retours ont valeur d'accord d'acceptation de produits retournés par un client pour réparation, remplacement, remboursement ou avoir.

Client réseau Sage SalesLogix

Version réseau du client Sage SalesLogix prenant en charge les utilisateurs réseau.

Client distant Sage SalesLogix

Version utilisateur distant du client réseau SalesLogix incluant la synchronisation, l'abonnement et un sous-ensemble de la base de données du bureau principal. Le terme « client distant » désigne aussi l'ordinateur utilisé par les utilisateurs distants.

Bon de commande

Reprend les produits achetés par l'intermédiaire de vos comptes. Les bons de commande incluent des informations telles que leur date d'édition, leur date de livraison au client et l'état du produit.

Code site

Code d'identification unique représentant le matériel (ordinateur ou serveur) utilisé par les applications Sage SalesLogix. Chaque ordinateur ou serveur dispose d'un code site. Les utilisateurs peuvent avoir plusieurs codes site s'ils utilisent plusieurs ordinateurs. Les codes site des clients Web sont stockés sur le serveur mais sont associés aux modifications de la base de données comme si un utilisateur Web était un utilisateur réseau.

Vue Résumé

Une solution alternative pour consulter les informations dans une vue Liste. La vue résumé est constituée de trois colonnes de données avec des liens et des fenêtres contextuelles contenant des informations supplémentaires.

Synchronisation

Processus permettant de détecter les différences entre les informations contenues dans les bases de données distantes et la base de données réseau. La synchronisation détecte ces modifications, les applique à la base de données réseau et crée des fichiers que les clients distants pourront récupérer pour mettre à jour leurs bases de données distantes. Volet des tâches

Volet situé à droite de l'espace de travail contenant les tâches et fonctionnalités courantes destinées à vous aider à accomplir votre travail. Les options disponibles dépendent du type d'enregistrement et du type de vue dans lequel vous travaillez.

Ticket

Question ou plainte d'un client. Un ticket peut contenir pratiquement toutes les informations nécessaires pour faire part d'une question ou d'un problème, effectuer des recherches s'y rapportant et clôturer la question ou le problème mentionné.

Activité non confirmée

Une confirmation d'activité que vous devez confirmer ou refuser. Les activités non confirmées n'apparaissent pas sur votre agenda et vous ne recevrez pas de rappel lorsqu'une activité non confirmée est sur le point de commencer. Vous pouvez gérer des activités non confirmées dans les vues Alertes et Liste des activités.
