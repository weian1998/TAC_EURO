Exécution d'un rapport

Pour exécuter un rapport

  1. Dans la barre de navigation, cliquez sur Rapports.
  2. Dans la liste Sélectionner le type de rapport, sélectionnez un rapport.
  3. Dans la liste Sélectionner le nom du rapport, sélectionnez un rapport. Une description du rapport apparaît dans la zone ci-dessous.
    (Une fois que ce rapport est sélectionné, les administrateurs peuvent visualiser à qui ce rapport est diffusé.)
  4. Dans la liste Afficher les enregistrements correspondants, sélectionnez l'une des options suivantes :
    • Tous les enregistrements : inclut tous les enregistrements pour le type de rapport sélectionné. Pour affiner votre sélection, cliquez sur Modifier.
    • Groupe actuel : (pour les rapports de type contacts, opportunités, tickets, défauts et RMA) inclut les données relatives à l'enregistrement de groupe actuellement sélectionné dans la vue principale. Pour affiner votre sélection, cliquez sur Modifier.
  5. Cliquez sur Afficher le rapport. Si le filtre du groupe comprend des données Unicode, il vous sera demandé de supprimer le groupe du filtre ou de sélectionner un autre groupe.
    Le rapport apparaît dans le visualiseur de rapports CR de Sage SalesLogix.
  6. Dans la zone Envoyer la sortie du rapport à, sélectionnez Prévisualiser.
  7. Cliquez sur Générer le rapport.

Pour plus d'informations, consultez la rubrique « Filtrage des données dans les rapports » dans l'aide .
