Filtrage des données dans les rapports

Lorsque vous générez un rapport, vous pouvez filtrer les données que celui-ci contient. Les filtres sont constitués d'une ou de plusieurs conditions. Par exemple, vous pouvez définir des conditions afin d'inclure uniquement certains comptes ou groupes dans un rapport. Vous pouvez enregistrer les filtres afin de les appliquer ensuite rapidement à chaque génération d'un rapport. Si nécessaire, les administrateurs peuvent visualiser les formules de conditions du rapport pour effectuer des dépannages.

Pour créer un filtre

  1. Sélectionnez le rapport à générer puis cliquez sur Modifier.

Les conditions de filtre s'affichent sur le côté droit de l'écran.

  1. Dans la liste Faire correspondre, sélectionnez l'une des options suivantes :
  • Toutes : pour associer toutes les conditions spécifiées.
  • N'importe laquelle : pour associer une ou plusieurs des conditions indiquées.
  1. Sélectionnez une condition dans la liste Faire correspondre par (comme indiqué ci-dessous) et remplissez toutes les informations nécessaires pour la condition. Cliquez sur le bouton plus (+) pour ajouter la condition au filtre. Vous pouvez rajouter autant de conditions que nécessaire.

    		2.

    L'administrateur sélectionne les champs utilisés pour calculer les conditions de filtres de données et d'utilisateur. Si vous utilisez ces filtres et que votre rapport ne contient pas les informations souhaitées, cela peut être dû aux champs utilisés pour les calculs. Pour plus d'informations, adressez-vous à votre administrateur.
    Faire correspondre par Commentaires
    Groupe

    Filtre les enregistrements en fonction d'un regroupement prédéfini. Pour certains rapports, le filtrage par type de groupe est impossible. Par exemple, l'option Groupe n'apparaît pas dans la liste si vous sélectionnez le type de rapport « Historique ».

    Pour utiliser ce filtre, sélectionnez un groupe dans la liste Nom de groupe.
    Plage de dates

    Filtre le rapport en fonction de la période ou de la plage de dates sélectionnée.

    Pour utiliser ce filtre, dans la liste Plage, effectuez une des actions suivantes :

    • Sélectionnez une durée.
    • Cliquez sur les boutons Agenda pour sélectionner une plage de dates.
    Utilisateur Filtre les enregistrements en fonction du type d'utilisateur sélectionné ou du nom de l'utilisateur.
    Requête Filtre les enregistrements en fonction de la table, du champ, de l'opérateur et de la valeur sélectionnés ou saisis.
  1. (Facultatif) Pour enregistrer le filtre, saisissez un nom dans la zone Enregistrer la condition sous, puis cliquez sur Enregistrer.

Pour supprimer des conditions dans un filtre

  • Effectuez l'une des opérations suivantes :
  • Cliquez sur Supprimer la condition (croix rouge) à côté de la condition à supprimer.
  • Pour supprimer toutes les conditions, cliquez sur Effacer tout.
