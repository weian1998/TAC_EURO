Présentation des rapports

La vue Rapports vous permet d'accéder à un ensemble de requêtes de base de données prédéfinies qui génèrent des rapports de gestion standard. Les rapports spécifiques disponibles sont déterminés par votre administrateur système et sont généralement personnalisés en fonction des besoins de votre entreprise.

Le tableau de bord est un ensemble d'outils qui vous permet d'afficher et d'interagir avec de nombreuses informations sur les performances et les analyses au format tableau de bord. Vous pouvez utiliser les données fournies pour mieux connaître les performances de l'entreprise et les performances individuelles, pour découvrir les causes des problèmes de performance et prendre les mesures nécessaires ou des décisions stratégiques.

  • Comprendre le tableau de bord des ventes
  • Ajuster les données des ensembles de contenu
  • Accéder aux vues Analyse et définir les filtres

Sage SalesLogix Web Reporting vous permet de :

  • partager des filtres de rapport,
  • filtrer les rapports pour limiter les données qui seront incluses,
  • exporter et imprimer les rapports.

Pour ouvrir la vue Rapports

  • Dans la barre de navigation, cliquez sur Rapports.

Pour exécuter des rapports dans un état déconnecté, le Client Web hors ligne doit être exécuté sur le serveur Windows 2003 avec IIS.Le serveur SalesLogix Web Reporting et le serveur Crystal Reports doivent également être installés et configurés. Si ces conditions ne sont pas remplies, les utilisateurs du client Web déconnectés ne peuvent exécuter Web Reporting que s'ils sont connectés au serveur Web Reporting hôte. Ces utilisateurs verront des rapports qui représentent les données de la base de données hôte et non de leur base de données distante.
