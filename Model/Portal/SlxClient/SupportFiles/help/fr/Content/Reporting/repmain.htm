Aperçu de rapport

Quand vous générez un rapport, vous pouvez décider d'afficher un aperçu dans le visualiseur Crystal Report de Sage SalesLogix avant de l'imprimer. Dans le Visualiseur, vous pouvez imprimer ou exporter votre rapport, régler sa taille d'affichage à l'écran et rechercher du texte dans le rapport.

Vous pouvez uniquement visualiser les rapports ayant été diffusés à votre attention ou à celle d'une équipe à laquelle vous appartenez. Vous ne pouvez pas enregistrer de rapports dans le client Web Sage SalesLogix.

Procédure

Afficher ou imprimer un rapport détaillé depuis une vue Détails

Prévisualiser un rapport dans le Visualiseur
  1. Sélectionnez le rapport à visualiser, puis cliquez sur Afficher le rapport. Le rapport apparaît dans le Visualiseur de rapports. Selon le rapport sélectionné et le nombre de résultats obtenus, une arborescence peut s'afficher sur la partie gauche de la fenêtre. Elle donne la liste des pages du rapport par ordre alphanumérique ou chronologique.
  2. Pour afficher une page spécifique du rapport, cliquez sur le signe plus pour développer la liste, puis cliquez sur un nom dans l'arborescence.
Exporter un rapport dans le Visualiseur
  1. Cliquez sur le bouton Exporter dans la barre d'outils.

La boîte de dialogue Exporter le rapport s'affiche.

  1. Dans la liste déroulante Format de fichier, sélectionnez le format et la destination du rapport.
  2. Sélectionnez l'une des options d'intervalle de pages pour le nombre de pages à exporter. Sélectionnez Tout pour imprimer la totalité du rapport, ou sélectionnez l'option Pages et entrez le numéro de la première page dans la case De, et le numéro de la dernière page dans la zone A.
  3. Cliquez sur OK.
Imprimer un rapport dans le Visualiseur
  1. Cliquez sur le bouton Imprimer dans la barre d'outils.
  2. Sélectionnez l’une des options d'intervalle de pages pour le nombre de pages que vous souhaitez imprimer. Sélectionnez Tout pour imprimer la totalité du rapport, ou sélectionnez l'option Pages et entrez le numéro de la première page dans la case De, et le numéro de la dernière page dans la zone A.
  3. Cliquez sur OK. Acrobat Reader s'ouvre et affiche le rapport que vous avez sélectionné.
  4. Cliquez sur Imprimer. Si le rapport ne s'imprime pas correctement, vous devrez peut-être le reformater pour votre imprimante. L'orientation du rapport aura la priorité sur l'orientation définie dans les paramètres de l'imprimante.
Afficher/masquer l'arborescence des groupes

Selon le rapport sélectionné et le nombre de résultats obtenus, une arborescence peut s'afficher sur la partie gauche de la fenêtre. Elle donne la liste des pages du rapport par ordre alphanumérique ou chronologique.

  • Dans le visualiseur de rapports, cliquez sur le bouton Afficher/masquer l'arborescence des groupes.
Parcourir un rapport
  • Dans l’Afficheur de rapport, cliquez sur les boutons Premier, Précédent, Suivant ou Dernier pour vous déplacer dans les différentes pages d’un rapport.
Rechercher du texte dans un rapport
  1. Dans le Visualiseur de rapports, tapez les critères de recherche dans la zone Rechercher.
  2. Cliquez sur le bouton Rechercher (jumelles).
