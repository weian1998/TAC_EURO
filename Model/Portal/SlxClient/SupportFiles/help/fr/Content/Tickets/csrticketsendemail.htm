Envoi d'un message e-mail depuis un ticket

Vous pouvez envoyer un message e-mail à un client ou à un autre utilisateur Sage SalesLogix tout en ajoutant un nouveau ticket ou depuis la vue Détails du ticket. L'e-mail peut fournir des informations sur le ticket, informer d'autres utilisateurs dont le client, de l'ouverture du ticket ou poser des questions pour faciliter la résolution du problème par exemple.

Pour envoyer un e-mail depuis un ticket

  1. Dans la barre de navigation, cliquez sur Tickets.
  1. Dans la vue de liste Tickets, utilisez la fonction de recherche pour localiser le ticket et ouvrez la vue Détails du ticket.
  2. Dans la vue Détails du Ticket, dans la barre d'outils, cliquez sur le bouton Détails de l'e-mail.
  3. Dans la zone A de la boîte de dialogue Envoyer un e-mail du ticket, cliquez sur une ou plusieurs des options de destinataire.
  4. Sous Type d'e-mail, sélectionnez l'un des types d'informations à envoyer. Les options sont les suivantes :
    5. Option Action
    Informations sur le ticket Le message comprend les valeurs suivantes Date de réception, Description, Résolution (si disponible), tous les Commentaires internes et N° de Ticket.
    Aucun Le N° de Ticket renseigne la ligne objet de l'e-mail. Aucune autre information concernant le ticket n'est comprise dans le message.

Les commentaires internes ne sont disponbiles que pour les autres utilisateurs Sage SalesLogix. Si l'un des destinataires de l'e-mail est un contact, les commentaires internes ne seront alors pas inclus dans l'e-mail.
  1. Cliquez sur OK.
  1. Envoyez le message via le système de messagerie que vous avez sélectionné.
