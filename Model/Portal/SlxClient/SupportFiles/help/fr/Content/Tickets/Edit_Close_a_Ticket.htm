Modification ou clôture d'un ticket

Quand vous devez modifier l'état ou les informations d'un ticket, utilisez la vue Détails du ticket. Pour annuler vos modifications, cliquez sur Réinitialiser avant de cliquer sur le bouton Enregistrer.

Procédure

Ouvrez la vue Détails du ticket
  • Dans la barre de navigation, cliquez sur Tickets pour ouvrir la vue Liste des tickets. Si nécessaire, utilisez la Recherche pour localiser un ticket, puis cliquez sur le nom du numéro ticket.
Modification d'un ticket
  1. Ouvrez la vue Détails du ticket.
  2. Cliquez, si nécessaire, sur Démarrage chrono pour démarrer l'enregistrement du temps consacré à l'activité.
  3. Modifiez les informations du ticket.

    Si vous modifiez cette zone, vous devez également modifier l'utilisateur Affecté à. Les informations de Affecté à ne sont pas automatiquement mises à jour par Sage SalesLogix.
  4. (En option) Cliquez sur l'onglet Détails ou, dans la zone Détails, procédez comme suit :
    • Si la description du problème doit être modifiée, modifiez-la dans la zone Description.
    • Si vous devez ajouter la résolution du problème, saisissez la résolution dans la zoneSolution.
  5. Effectuez l'une des actions suivantes si toutes les modifications ou mises à jour sont terminées :
    • Si le démarrage chrono est lancé, cliquez sur Arrêt chrono et remplissez les zones d'information de l'activité du ticket.
    • Si le démarrage chrono n'est pas lancé et que vous n'avez pas besoin d'enregistrer une activité, cliquez sur OK.
  6. Cliquez sur Enregistrer.Cliquez sur Enregistrer.
Clôturer un ticket
  1. Ouvrez la vue Détails du ticket.

  2. Assurez-vous que toutes les informations du ticket sont complètes et exactes.

    Si vous modifiez cette zone, vous devez également modifier l'utilisateur Affecté à. Les informations de Affecté à ne sont pas automatiquement mises à jour par Sage SalesLogix.
  3. Dans la zone État, cliquez sur le bouton en forme de flèche, puis sélectionnez Clôturé.

    Si le démarrage chrono est lancé et que vous avez des activités de ticket ouvertes, un message vous demande de fermer l'activité de ticket ouverte ou de cliquer sur le bouton Arrêt chrono.

  4. Cliquez sur l'onglet Détails et mettez à jour laSolution.
  5. Si vous avez résolu le problème du client dès le premier appel, cochez la case Résolu au 1er appel ?
  6. Si vous souhaitez que les informations saisies concernant la résolution soient intégrées aux index SpeedSearch, cochez la case Soumettre à SpeedSearch.
  7. Si votre administrateur vous a accordé l'autorisation d'approuver des tickets, cochez la case Approuvé pour SpeedSearch. Dans le cas contraire, cette option ne s'affichera pas.
  8. Cliquez sur Enregistrer.
