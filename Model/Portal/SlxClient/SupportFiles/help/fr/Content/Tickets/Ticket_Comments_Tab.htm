Utilisation de l'onglet Commentaires sur le ticket

Utilisez les zones de texte Commentaires pour toutes les remarques internes, les étapes du dépannage ou tout autre information importante en rapport avec le ticket.

Pour ajouter des commentaires internes à un ticket

  1. Dans la vue Détails du ticket, cliquez sur l'onglet Commentaires. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  2. Saisissez vos commentaires dans la zone commentaires internes.
  3. Cliquez sur le bouton Horodatage utilisateur pour saisir votre nom, ainsi que la date et l'heure (en fonction de vos paramètres régionaux) des commentaires.
  4. Cliquez sur Enregistrer.
