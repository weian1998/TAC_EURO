Exemple : utilisation de Query Builder pour créer un groupe

L'exemple suivant explique comment créer un groupe de tous les contacts principaux travaillant dans le service des ventes de leur société.

  1. Connectez-vous au Client Web Sage SalesLogix.
  2. Dans la barre de navigation, cliquez sur Contacts.
  3. Dans le menu Groupes, cliquez sur Ajouter un groupe.
  4. Dans Query Builder, remplissez les zones suivantes de l'onglet Propriétés  :
  5. Dans la zone Nom, saisissez Principaux contacts des ventes.
  6. Dans la zone Description, saisissez Principaux contacts travaillant dans le service Ventes.
  7. Sous l'onglet Conditions, procédez comme suit :
  1. Dans le coin supérieur gauche de Query Builder, sélectionnez CONTACT.
  2. Dans le coin supérieur droit de Query Builder, double-cliquez sur EstPrincipal.

La boîte de dialogue Affecter la condition apparaît avec EstPrincipal dans la zone Nom du champ, et Chaîne en tant que Type de champ.

  1. Dans la zone Opérateur, sélectionnez (=) égal à.
  2. A côté de la zone Valeur, cliquez sur Parcourir et sélectionnez T (pour Vrai), puis cliquez sur OK.

Laissez la case Comparaison sensible à la casse cochée et ne sélectionnez pas Utiliser la valeur comme une valeur littérale.

  1. Cliquez sur OK.
  2. Dans le coin supérieur droit de Query Builder, double-cliquez sur Service.

La boîte de dialogue Affecter la condition apparaît avec Service dans la zone Nom du champ, et Chaîne en tant que Type de champ.

  1. Dans la zone Opérateur, sélectionnez contient.
  2. A côté de la zone Valeur, cliquez sur Parcourir, sélectionnez Ventes, puis cliquez sur OK.
  3. Désactivez l'option Comparaison sensible à la casse et ne sélectionnez pas Utiliser la valeur comme littéral.
  4. Cliquez sur OK.
  1. Cliquez sur l'onglet Présentation. L'onglet Présentation affiche les champs qui apparaîtront lorsque la requête sera enregistrée en tant que groupe. Remplissez les zones suivantes :
  2. Dans le coin supérieur droit de Query Builder, sélectionnez le champ Service et double-cliquez pour l'ajouter à la grille ci-dessous.
  1. Utilisez la barre de défilement pour aller tout à droite et cliquez sur Service.
  2. Cliquez sur le bouton Dépl. à gauche plusieurs fois jusqu'à ce que Département apparaisse entre Compte et Ville.
  1. Cliquez sur l'onglet Tri. Par défaut, les contacts sont triés selon leur nom de famille dans l'ordre croissant (A à Z). Si besoin est, vous pouvez ajouter un critère de tri supplémentaire ou modifier l'ordre.
  2. Sous l'onglet Paramètres par défaut, ne modifiez aucune option.
  3. Cliquez sur OK.

Le nouveau groupe Principaux contacts des ventes apparaît dans la liste Groupe de contacts et vous pouvez le sélectionner pour afficher tous les contacts répondant aux critères définis.
