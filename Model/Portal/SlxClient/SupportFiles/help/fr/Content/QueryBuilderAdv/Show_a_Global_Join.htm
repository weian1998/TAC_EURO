Affichage d'une jointure globale

Sage SalesLogix dispose d'un certain nombre de jointures globales prédéfinies. Toutes ne sont cependant pas visibles à l'ouverture de Query Builder du fait des contraintes d'espace. Si une de ces jointures existe déjà lorsque vous utilisez Query Builder pour sélectionner des données, utilisez-la plutôt que d'en créer une nouvelle.

Pour ajouter une jointure locale

  1. Ouvrez Query Builder.
  2. Si vous le souhaitez, utilisez le signe plus pour développer la liste de tables dans le coin supérieur gauche de Query Builder. Sélectionner une table.
  3. Cliquez sur Jointure globale.

La boîte de dialogue Gestionnaire de jointures globales apparaît et affiche les jointures globales existantes.

Si une jointure globale n'est pas disponible, vous pouvez ajouter une jointure locale.
