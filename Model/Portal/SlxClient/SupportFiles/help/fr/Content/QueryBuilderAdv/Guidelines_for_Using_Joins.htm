Instructions pour l'utilisation des jointures

Utilisez les instructions suivantes pour créer des jointures de table dans Sage SalesLogix :

  • Dans une vue affichant les champs de plusieurs tables, utilisez les jointures gauches. Si vous utilisez les jointures internes et qu'une table ne contient aucune donnée, AUCUNE DONNEE n'apparaîtra dans AUCUN des champs de la vue.
  • Dans un groupe, utilisez les jointures internes pour les tables utilisées dans les conditions. Ainsi, vous n'obtiendrez que les enregistrements répondant à vos critères. Vous pouvez toutefois vouloir des jointures gauches pour les tables utilisées dans la présentation pour que tous les enregistrements répondant à vos conditions soient inclus, même si une des tables ne contient pas de données.

Par défaut, la table ADDRESS est liée à la table ACCOUNT grâce à une jointure interne. Ne modifiez pas les propriétés de cette jointure.
