Ajout d'une jointure globale

 Administrateur système uniquement

Une jointure globale est permanente et existe de session en session dans la structure de l'arborescence des données de Query Builder. Soyez vigilant au moment de créer des jointures globales, vous pourriez en créer une par inadvertance ; ce qui provoquerait de sérieux problèmes de performances.

Pour ajouter

  1. Ouvrez Query Builder, puis cliquez sur le bouton Jointures globales.

Ce bouton n'apparaît que si vous êtes connecté en tant qu'administrateur.

  1. Dans la boîte de dialogue Gestionnaire de jointures globales, cliquez sur Ajouter.
  2. Dans la boîte de dialogue Ajouter une jointure globale, sous Table parent et Champ Jointure, sélectionnez la table et le champ pour la jointure globale.
  3. Sous Table enfant et champ Jointure, sélectionnez la table et le champ à joindre au champ et à la table parent.
  4. Sous Type de jointure, sélectionnez le type de jointure.

L'option par défaut est Gauche.

  1. Sous Options de jointure :
  1. Sélectionnez une option de visibilité pour indiquer comment afficher la jointure dans Query Builder.

L'option par défaut est Autoriser.

  1. sélectionnez un Type de cascade pour indiquer la façon dont les enregistrements sont supprimés dans la jointure.

L'option par défaut est Ignorer (ne pas utiliser en cascade).

  1. Cliquez sur OK.
