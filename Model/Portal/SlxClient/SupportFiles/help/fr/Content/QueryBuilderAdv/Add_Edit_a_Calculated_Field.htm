Ajout et modification de champs calculés

 Administrateur système uniquement

Pour ouvrir la boîte de dialogue Gérer des champs calculés

  1. Ouvrez Query Builder.
  2. Cliquez sur le bouton Calculs.
Pour ajouter
  1. Ouvrez Query Builder, puis cliquez sur le bouton Calculs.

Ce bouton n'apparaît que si vous êtes connecté en tant qu'administrateur.

  1. Cliquez sur Ajouter.
  2. Cliquez sur l'onglet Propriétés.
  3. Dans la première zone Nom/Alias, saisissez un nom qui décrit l'objet du calcul.

Le nom du calcul apparaît uniquement dans le gestionnaire des calculs.

  1. Dans la deuxième zone, entrez le nom à utiliser dans le calcul. Il ne peut pas contenir d'espaces ni de symboles. Il s'agit d'un champ obligatoire. Vous ne pourrez pas créer des champs calculés avant que celui-ci n'ait été défini.

Lorsque le champ calculé est utilisé pour la création d'un plugin ou d'une requête, il s'affiche dans la liste des champs disponibles.

  1. Cliquez sur la flèche de la liste déroulante Table de base et sélectionnez la table principale utilisée pour créer le champ calculé.

Si vous associez deux tables, il s'agit de la table parent.

  1. Cliquez sur la flèche de la liste déroulante Type de calc. et sélectionnez un Type de calcul.
  2. Saisissez une description si vous souhaitez avoir une description plus détaillée du champ disponible.
  3. Cliquez sur l'onglet Calcul.
  4. Vérifiez que les valeurs que vous souhaitez appliquer au champ calculé se trouvent dans la Table de base sélectionnée.

La table affichée dans la vue de liste supérieure est la table que vous avez sélectionnée sous l'onglet Propriétés.

  1. Si la table de base ne contient pas les valeurs dont vous avez besoin, vous pouvez changer la table de base plutôt que de créer une jointure. Cliquez sur l'onglet Propriétés et sélectionnez une autre table de base.
  2. Créez le calcul.
  3. Cliquez sur le signe + ou - pour développer ou réduire la liste des tables et des champs afin de trouver le premier champ pour le calcul.
  4. Pour ajouter un champ, cliquez dessus dans la zone supérieure pour l'ajouter à un quelconque endroit du calcul.

Il s'agit du premier champ du calcul.

  1. Effectuez l'une des opérations suivantes :
  • Cliquez sur un opérateur si vous souhaitez créer un calcul numérique.

Vous devez alors insérer un opérateur entre les champs.

  • Dans le cas d'une chaîne, vous n'avez pas besoin de choisir un opérateur mais vous pouvez insérer un espace pour une question d'apparence.
  1. Cliquez sur un autre champ dans la vue de l'arborescence pour l'ajouter à la vue de l'onglet inférieur ou saisissez un nombre entier.
  2. Continuez à ajouter autant de champs que vous souhaitez pour créer le champ calculé.
  3. Utilisez les parenthèses pour regrouper les calculs. Pour insérer une parenthèse, placez votre curseur au point d'insertion dans le calcul et cliquez sur un bouton de parenthèse. Il est impératif d'insérer une parenthèse d'ouverture et une parenthèse de fermeture.
  4. Cliquez sur OK.
  5. Cliquez sur Fermer.
Pour modifier
  1. Dans la boîte de dialogue Gérer les champs calculés, sélectionnez le calcul puis cliquez sur Modifier.
  2. Dans la boîte de dialogue Modifier le calcul, choisissez l'onglet Calcul.
  • Pour ajouter un champ, cliquez dessus dans la zone supérieure pour l'ajouter à un quelconque endroit du calcul.
  • Pour ajouter un opérateur ou une parenthèse, placez le curseur dans le calcul et cliquez sur un bouton d'opérateur ou de parenthèse. Un opérateur est nécessaire entre des champs dans un calcul numérique et il doit y avoir impérativement une parenthèse ouvrante et une parenthèse fermante.
  • Pour supprimer un quelconque élément du calcul, sélectionnez-le et appuyez sur Supprimer.
  1. Cliquez sur OK.
  2. Cliquez sur Fermer.
