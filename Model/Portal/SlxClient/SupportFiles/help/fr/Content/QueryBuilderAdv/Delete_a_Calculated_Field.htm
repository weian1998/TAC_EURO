Suppression des champs calculés

 Administrateur système uniquement

Vous ne pouvez ouvrir la boîte de dialogue Gérer les champs calculés uniquement si vous êtes connecté en tant qu'administrateur.

Pour supprimer un champ calculé

  1. Ouvrez Query Builder, puis cliquez sur le bouton Calculs.

Ce bouton n'apparaît que si vous êtes connecté en tant qu'administrateur.

  1. Dans la boîte de dialogue Gérer les champs calculés, sélectionnez un champ calculé dans la grille.
  2. Cliquez sur Supprimer.
  3. Pour confirmer la suppression, cliquez sur OK.
  4. Cliquez sur Fermer.
