Jointures internes

Type de jointures locales par défaut. Utilisez une jointure interne pour inclure uniquement les enregistrements dont les valeurs pour les champs que vous êtes en train de lier sont les mêmes dans les tables parent et enfant.

Exemple : Vous souhaitez consulter toutes les opportunités avec des activités associées en écartant toutes celles qui n'en ont pas. Les enregistrements affichés dans la colonne Résultats de jointure seraient renvoyés par cette requête. L'opportunité spécifique, associée à une activité, est indiquée entre parenthèses, à droite de l'activité.
