Désactivation d'un contrat

Vous pouvez désactiver un contrat lorsque aucun ticket n'est associé à ce dernier. Les contrats désactivés continuent d'apparaître dans la boîte de dialogue Informations sur les services du compte.

Pour désactiver un contrat

  1. Ouvrez la vue Détails du contrat.
  2. Cliquez sur l'onglet Tickets.
  3. Vérifiez qu'aucun ticket n'est répertorié en tant que lié au contrat.
  4. Dans la partie supérieure de la vue Détails du contrat, décochez la case Actif.
  5. Cliquez sur Enregistrer.
