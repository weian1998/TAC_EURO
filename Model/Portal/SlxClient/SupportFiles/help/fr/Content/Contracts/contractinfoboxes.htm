Zones d'informations sur le contrat

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

 

Zone Description/Action
N° de référence Par défaut, le système attribue un numéro de référence unique à des fins de suivi interne, mais vous pouvez entrer un numéro différent. S'il existe déjà un numéro de référence, la boîte de dialogue Doublon du n° de référence du contrat s'ouvre.
Actif Sélectionnez cette case pour activer le contrat ou désélectionnez-la pour désactiver le contrat.
Compte Affiche le nom du compte associé au contrat. Pour modifier le compte, cliquez sur le bouton Rechercher. Une fois le contrat créé, cela ne peut pas être modifié.
Contact Affiche le nom du contact principal associé au contrat. Pour modifier le contact principal, cliquez sur le bouton Rechercher.
Commentaires Vous pouvez ajouter des informations supplémentaires sur le contrat dans cette zone.
Service Le type de service couvert par le contrat. Pour modifier le service, cliquez sur bouton en forme de flèche et sélectionnez un élément dans la liste.
Type de contract Pour modifier le service, bouton en forme de flèche et sélectionnez un élément dans la liste. La sélection d'un Type de contrat affecte les options des autres zones d'informations sur le contrat. Par exemple, si le Type de contrat est défini sur Jours, la zone Jours de garantie devient disponible. Une fois le contrat créé, il est impossible de modifier la zone Type de contrat.
Date de début La date à laquelle le contrat devrait démarrer. Pour effectuer une modification, cliquez sur le Bouton Agenda, sélectionnez une date, puis cliquez sur OK.  
Commencer par le premier ticket Si vous souhaitez que le contrat commence après la création du premier ticket, sélectionnez cette case. Cette option n'est disponible que lorsque le Type de contrat est défini sur Jours. Lorsque cette option est sélectionnée, les zones Date de début et Date de fin sont vierges jusqu'à l'ouverture du premier ticket. Une fois le contrat créé, cela ne peut pas être modifié.
Expiration La date à laquelle le contrat devrait expirer. Pour effectuer une modification, cliquez sur le Bouton Agenda, sélectionnez une date, puis cliquez sur OK.  
Jours de garantie Saisissez un nombre représentant le nombre de jours après la date d'expiration pendant lesquels le service ou le support technique sera encore offert au client sous contrat. S'applique à tous les types de contrat.
Quantité

Nombre d'unités du type indiqué dans la zone Type de contrat. Le nombre peut s'afficher automatiquement, en fonction des informations figurant dans les zones Type de contrat, Jours de garantie et Commencer par le premier ticket. Vous pouvez également saisir le nombre manuellement.

Si le type de contrat est défini sur Jours et qu'il y a une quantité, la date d'expiration est calculée à partir de la date de début.
Restant Affiche le temps restant sur le contrat actuel. Le système calcule la quantité restante sur ce contrat à l'aide des zones Type de contrat, Jours de garantie, Date d'achat, Date de début et Date d'expiration. Lorsque les tickets et les activités de ticket sont créés et exécutés, la valeur restante diminue. Elle apparaît lors de l'affichage d'un contrat existant et ne peut être modifiée.
Boîte postale Si nécessaire, vous pouvez saisir un numéro de bon de commande pour le contrat.
Date d'achat La date d'achat. Pour effectuer une modification, cliquez sur le Bouton Agenda, sélectionnez une date, puis cliquez sur OK.  
Tarif Saisissez une valeur pour le contrat. Si aucune valeur n'est entrée, le système indique par défaut la valeur du montant du contrat.
Remise Si nécessaire, saisissez une valeur de remise.
Montant du contrat Saisissez la valeur du contrat. Si aucun montant n'est saisi, le système indique par défaut la valeur Prix de la liste moins la valeur de la remise.
Mis à jour par Affiche l'auteur de la dernière mise à jour des informations sur le contrat. Elle apparaît lors de l'affichage d'un contrat existant et ne peut être modifiée.
Dernière mise à jour Affiche la date et l'heure de la dernière mise à jour des informations sur le contrat. Elle apparaît lors de l'affichage d'un contrat existant et ne peut être modifiée.
