Ajout d'un contrat

Utilisez la vue Insérer un nouveau contrat pour créer un enregistrement de contrat. Utilisez le bouton copier pour créer rapidement un nouvel enregistrement avec des informations similaires.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Pour ajouter un contrat

  1. Ouvrez la vue Insérer un nouveau contrat.

    • Dans le menu Nouveau/, cliquez sur Contrat.
    • Dans la vue Détails du compte, cliquez sur l'onglet Contrats, puis sur Créer un contrat.
  2. Renseignez les autres zones d'informations sur le contrat.
  3. Cliquez sur Enregistrer.
  4. Si nécessaire, utilisez l'onglet Actifs couverts pour associer des actifs au contrat.
