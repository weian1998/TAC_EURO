Onglet Utilisateurs

Utilisez l'onglet Utilisateurs pour affecter un ou plusieurs utilisateurs à un rôle ou pour supprimer un utilisateur d'un rôle. Lorsqu'un utilisateur est affecté à un rôle, l'utilisateur a accès aux actions définies dans ce rôle. Si un utilisateur est affecté à plus d'un rôle, il aura accès à toutes les actions sécurisées contenues dans ces rôles, même si chaque rôle ne contient pas les mêmes actions.

Procédure

Affecter un utilisateur à un rôle

  1. Ouvrez la vue Détails du rôle.
  2. Cliquez sur l'onglet Utilisateurs.
  3. Cliquez sur Ajouter des utilisateurs.
  4. Utilisez la recherche pour localiser et sélectionner les utilisateurs que vous souhaitez affecter au rôle.
  5. Cliquez sur Ajouter les actions sélectionnées.
  6. Cliquez sur Fermer lorsque vous avez terminé.

Supprimer un utilisateur d'un rôle

  1. Ouvrez la vue Détails du rôle.
  2. Cliquez sur l'onglet Utilisateurs.
  3. Sélectionnez l'enregistrement à supprimer, puis cliquez sur le bouton Supprimer.

  4. Cliquez sur Oui.
