Modifier ou supprimer une action sécurisée

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.L'administrateur système et les utilisateurs dotés d'un rôle administrateur peuvent accéder à ces fonctionnalités.

Si vous effacez ou renommez une action sécurisée, celle-ci est découplée de tous les éléments auxquels elle était affectée dans l'application Architect. Bien que la relation n'existe plus, l'élément reste toujours sécurisé jusqu'à ce que vous supprimiez manuellement l'action sécurisée de la propriété Sécurité appliquée du contrôle dans l'application Architect. En savoir plus...

Par exemple, vous avez créé une action sécurisée nommée Impression/Rapports et ajouté cette action au bouton Imprimer de la vue Rapports. Si vous supprimez l'action sécurisée Impression/Rapports, Impression/Rapports s'affiche encore dans la propriété Sécurité appliquée du contrôle du bouton Imprimer dans l'application Architect, bien que l'action n'existe plus. Lorsqu'un utilisateur accède à la vue Rapports, il ne peut pas accéder au bouton Imprimer car il n'a pas été affecté à un rôle qui contient l'action sécurisée Impression/Rapports. L'utilisateur admin peut accéder à tous les éléments de l'interfacce utilisateur, pour toutes les actions sécurisées.

Si vous souhaitez supprimer la sécurité de l'élément, supprimez l'action sécurisée, puis supprimez manuellement l'action de la propriété Sécurité appliquée du contrôle dans l'application Architect.

Sage vous recommande d'enregistrer les informations concernant l'onglet Emplacements de l'action sécurisée avant de renommer ou de supprimer une action.

Procédure

Modifier une action sécurisée

  1. Ouvrez la vue Détails des actions sécurisées pour l'action que vous souhaitez modifier.
  2. Utilisez l'onglet Rôles pour ajouter l'action sécurisée à un rôle, ou pour l'en supprimer.
  3. Cliquez sur Enregistrer.

Supprimer une action sécurisée

  1. Ouvrez la vue Détails des actions sécurisées pour l'action que vous souhaitez supprimer.
  2. Cliquez sur Supprimer.
  3. Pour confirmer la suppression, cliquez sur OK.

Récupérer une action sécurisée

Si vous supprimez (ou renommez) accidentellement une action sécurisée, vous pouvez créer une nouvelle action sécurisée, portant le même nom, qui liera de nouveau l'action à tous les contrôles qui l'utilisent dans l'application Architect. Si vous avez supprimé manuellement l'action de la propriété Sécurité appliquée sur le contrôle dans l'application Architect, l'action ne pourra plus être liée au contrôle. Le nom de la nouvelle action sécurisée doit être EXACTEMENT le même que celui de l'action sécurisée précédente.
