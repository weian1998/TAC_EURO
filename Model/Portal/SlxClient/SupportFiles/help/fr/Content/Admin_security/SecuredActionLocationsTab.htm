Onglet Emplacements

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.L'administrateur système et les utilisateurs dotés d'un rôle administrateur peuvent accéder à ces fonctionnalités.

L'onglet Emplacements de la vue Détails des actions sécurisées montre où l'action sécurisée est utilisée. Le nom du fichier et son chemin d'accès à partir du site web s'affichent pour chaque élément. Ces informations sont en lecture seule.

Lorsque vous effacez ou renommez une action sécurisée, celle-ci est découplée de l'élément dans l'application Architect. Avant de modifier l'action sécurisée, Sage vous recommande d'enregistrer les informations trouvées sur cet onglet dans un enregistrement d'historique. Vous pouvez copier et coller les informations dans un fichier texte.

Si nécessaire, vous pouvez utiliser les informations de cet onglet pour recréer une action sécurisée ou pour supprimer l'action de certains éléments dans l'application Architect.
  • La première fois que vous accédez à cet onglet, les informations peuvent être longues à charger.
  • Si vous modifiez des actions sécurisées, les informations de cet onglet s'actualiseront lorsque vous reconstruirez et déploierez votre site Web.
