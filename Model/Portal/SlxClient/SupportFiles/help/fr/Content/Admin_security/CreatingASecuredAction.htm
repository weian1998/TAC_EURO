Création d'une action sécurisée

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.L'administrateur système et les utilisateurs dotés d'un rôle administrateur peuvent accéder à ces fonctionnalités.

Vous pouvez créer de nouvelles actions sécurisées pour sécuriser des éléments dans le client Web.

Lorsque vous créez une nouvelle action sécurisée, assurez-vous que cette action porte un nom unique, n'ayant pas été utilisé précédemment (à moins que vous ne récupériez une action sécurisée accidentellement supprimée). Si vous avez supprimé une action sécurisée sans la supprimer manuellement des éléments dans l'application Architect, la nouvelle action peut accorder des permissions imprévues. En savoir plus...

Par exemple, vous avez créé une action sécurisée nommée Impression/Rapports et ajouté cette action à plusieurs boutons Imprimer (à l'aide de la propriété Sécurité appliquée dans l'application Architect). Vous avez ensuite supprimé l'action sécurisée Impression/Rapports, mais sans modifier manuellement la propriété Sécurité appliquée pour les boutons Imprimer. Si vous avez créé une nouvelle action sécurisée nommée Impression/Rapports, tous les boutons Imprimer liés précédemment à l'action Impression/Rapports seront liés à la nouvelle action Impression/Rapports.

Sage vous recommande d'utiliser des noms descriptifs pour vos actions sécurisées, qui pourront être utilisés pour garder une trace du but recherché pour chaque action. Par exemple, l'action sécurisée Entité/Compte/Supprimer sert à contrôler des éléments qui permettent aux utilisateurs de supprimer des comptes. Cette structure de nomenclature est recommandée, mais pas obligatoire. Le nom de l'action sécurisée est une valeur de type chaîne et n'importe quelle convention de nomenclature peut être utilisée.

Pour créer

  1. Dans le menu Administration, cliquez sur Nouvelle action sécurisée.
  2. Dans le champ Action sécurisée, saisissez le nom de l'action sécurisée.
  3. Dans le champ Description, entrez des informations descriptives sur cette action sécurisée.
  4. Cliquez sur Enregistrer pour enregistrer l'action sécurisée ou Enregistrer et nouveau pour enregistrer l'action sécurisée et en créer une autre.
  5. Utilisez l'onglet Rôles pour ajouter l'action sécurisée à un rôle.
  6. Utilisez l'onglet Emplacements pour voir où l'action sécurisée est utilisée.
