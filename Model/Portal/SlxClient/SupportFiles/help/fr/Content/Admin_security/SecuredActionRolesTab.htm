Onglet Rôles

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.L'administrateur système et les utilisateurs dotés d'un rôle administrateur peuvent accéder à ces fonctionnalités.

Utilisez l'onglet Rôles dans la vue Détails des actions sécurisées pour gérer les actions sécurisées affectées à un rôle. Un rôle determine les fonctionnalités auxquelles un utilisateur peut accéder dans le client Web. L'ajout d'actions sécurisées à un rôle détermine les fonctionnalités auxquelles le rôle permet d'accéder. L'ajout d'utilisateurs au rôle détermine à qui l'accès est autorisé.

La relation entre les rôles et les actions sécurisées est affichée sur cet onglet et sur l'onglet Actions dans la vue Détails du rôle.

  • L'onglet Rôles affiche les actions sécurisées du rôle.
  • L'onglet Actions affiche les rôles qui contiennent l'action sécurisée.

Lorsque vous ajoutez une action sécurisée à un rôle ou si vous supprimez une action sécurisée d'un rôle, la modification s'affiche sur les deux onglets.

Procédure

Ajout d'une action sécurisée à un rôle

  1. Ouvrez la vue Détails de l'action sécurisée.
  2. Cliquez sur l'onglet Rôles.
  3. Cliquez sur Ajouter un rôle.
  4. Utilisez la recherche pour localiser et sélectionner le rôle auquel vous souhaitez affecter l'action sécurisée.
  5. Pour ajouter d'autres rôles, répétez les étapes 3 à 4.
  6. Cliquez sur OK.

Supprimer une action sécurisée d'un rôle

  1. Ouvrez la vue Détails de l'action sécurisée.
  2. Cliquez sur l'onglet Rôles.
  3. Cliquez sur le lien Supprimer pour le rôle duquel vous souhaitez supprimer l'action sécurisée.

  4. Cliquez sur OK.
