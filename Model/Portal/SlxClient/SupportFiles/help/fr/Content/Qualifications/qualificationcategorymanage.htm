Gestion des qualifications

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Les qualifications sont un ensemble d'éléments de qualification utilisés pour valider les informations concernant un lead ou pour déterminer si celui-ci est prêt à devenir une opportunité de vente, et à quel moment.

Procédure

Ajout d'une qualification
  1. Dans le menu Administration, cliquez sur Nouvelle qualification.
  2. Dans la zone Qualification pour, entrez un nom descriptif.
  3. Cliquez sur Enregistrer.
  4. Dans la vue Détails de la qualification, utilisez l'onglet Éléments pour ajouter des éléments de qualification.
Modification d'une qualification
  1. Ouvrez la vue Détails de la qualification.
  2. Dans la zone Qualification pour, modifiez le nom descriptif.
  3. Cliquez sur Enregistrer.
Suppression d'une qualification
  1. Dans la barre de navigation, cliquez sur Administration, puis cliquez sur Qualifications pour ouvrir la vue Liste des qualifications. Si nécessaire, utilisez la Recherche pour localiser une qualification, puis cliquez sur le nom de la qualification.
  2. Dans la vue Détails de la qualification, cliquez sur le bouton Supprimer.

  3. Cliquez sur OK.

Quand vous supprimez une qualification, vous supprimez également tous les éléments de qualification associés.
