Ajout ou modification d'un élément de qualification

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Les éléments de qualification sont les critères utilisés pour valider les informations concernant un lead ou pour déterminer si celui-ci est prêt à devenir une opportunité de vente, et à quel moment.

Procédure

Ajout d'un élément de qualification
  1. Dans la vue Détails de la qualification, ouvrez l'onglet Éléments et cliquez sur Ajout d'un élément de qualification.
  2. Ajoutez les informations suivantes :
    • Visible - Cochez cette case si vous souhaitez que l'élément s'affiche dans la vue Détails du lead.
    • Position de tri - Saisissez le nombre représentant l'ordre dans lequel l'élément doit s'afficher. Par exemple, entrez la valeur 2 si l'élément doit s'afficher en deuxième position dans la liste.
    • Description - Saisissez la description de l'élément de qualification telle que vous souhaitez qu'elle s'affiche dans la vue Détails du lead.
    • Afficher les notes - Cochez cette case si vous voulez que les utilisateurs puissent saisir des commentaires dans une zone de notes dans la vue Détails du lead.

  3. Cliquez sur OK.

Modification d'un élément de qualification
  1. Dans la vue Détails de la qualification, ouvrez l'onglet Éléments et cliquez sur Modifier en regard de l'élément de qualification que vous souhaitez modifier.
  2. Modifiez l'une des informations suivantes :
    • Visible - Cochez cette case si vous souhaitez que l'élément s'affiche dans la vue Détails du lead.
    • Position de tri - Saisissez le nombre représentant l'ordre dans lequel l'élément doit s'afficher. Par exemple, entrez la valeur 2 si l'élément doit s'afficher en deuxième position dans la liste.
    • Description - Saisissez la description de l'élément de qualification telle que vous souhaitez qu'elle s'affiche dans la vue Détails du lead.
    • Afficher les notes - Cochez cette case si vous voulez que les utilisateurs puissent saisir des commentaires dans une zone de notes dans la vue Détails du lead.

  3. Cliquez sur OK.
