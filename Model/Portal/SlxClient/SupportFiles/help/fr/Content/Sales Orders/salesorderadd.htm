Ajout d'un bon de commande

Utilisez cette vue pour ajouter un bon de commande, incluant l'adresse de facturation, l'adresse d'expédition ou des informations sur le produit et d'autres informations sur le bon de commande.

Vous ne pouvez pas imprimer ou envoyer par e-mail un devis avant que le bon de commande soit approuvé par le système de comptabilité. (Intégration de la comptabilité uniquement)

Pour ajouter

  1. Effectuez l'une des opérations suivantes :
    • Dans le menu Nouveau, cliquez sur Bon de commande.
    • Dans la vue Détails de l'opportunité, dans le volet Tâches, cliquez sur Ajouter un bon de commande ou sur l'onglet Bons de commande, cliquez sur le bouton Insérer un bon de commande.
      Le compte, les produits et les autres informations de l'opportunité sont automatiquement copiés dans le nouveau bon de commande.
  2. Sélectionnez Bon de commande.
  3. Ajoutez des informations sur le bon de commande.

    Vous devez associer un compte au bon de commande.
  4. (Optionnel) Modifiez les informations de l'Adresse de facturation. Par défaut, ce champ est renseigné avec le nom du contact principal du compte, et les informations de l'adresse principale si le bon de commande a été créé depuis les vues Détails du contact, du compte ou de l'opportunité.
    1. Pour sélectionner un nom, cliquez sur le bouton Rechercher et utiliser la recherche pour rechercher un contact.
    2. Cliquez sur l'adresse la flèche du menu déroulant et effectuez votre sélection dans la liste.
    3. Pour modifier l'adresse, cliquez sur le bouton Modifier. Pour afficher une carte de l'adresse, cliquez sur le bouton MapQuest.
  5. (Optionnel) Modifiez les informations de l'Adresse d'expédition. Par défaut, ce champ est renseigné avec le nom du contact principal du compte correspondant, et avec les informations de l'adresse d'expédition si le bon de commande a été créé depuis les vues Détails du contact, du compte ou de l'opportunité.
    1. Pour sélectionner un nom, cliquez sur le bouton Rechercher et utiliser la recherche pour rechercher un contact.
    2. Cliquez sur l'adresse la flèche du menu déroulant et effectuez votre sélection dans la liste.
    3. Pour modifier l'adresse, cliquez sur le bouton Modifier. Pour afficher une carte de l'adresse, cliquez sur le bouton MapQuest.
    4. Cliquez sur Expédier par bouton en forme de flèche et sélectionnez un élément dans la liste.
  6. Ajouter des produits.
    La section Produit affiche tous les produits associés au bon de commande. Si vous souhaitez supprimer un produit, sélectionnez l'enregistrement dans la grille Produits puis cliquez sur le bouton Supprimer.
    Le total de la commande est recalculé lorsque des produits sont ajoutés ou supprimés.
  7. Cliquez sur Enregistrer.
