Utilisation de la vue Détails du bon de commande

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur. 

Procédure

Ouvrez la vue Détails du bon de commande.
  • Dans la barre de navigation, cliquez sur Bons de commande pour ouvrir la vue Liste des bons de commande. Si nécessaire, utilisez la Recherche pour localiser un bon de commande, puis cliquez sur le numéro du bon de commande.

Utiliser la Recherche pour rechercher les enregistrements

Afficher un groupe

Passez en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.

Envoyer les informations du bon de commande par e-mail

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter une note
  • Nouvelle réunion
  • Nouvel appel téléphonique
  • Nouvelle tâche à faire

Dans le Volet des tâches, sous Tâches de comptabilité, vous pouvez :

  • Vérifier les prix
  • Soumettre à un Système de comptabilité
  • Afficher l'État du bon de commande

zones d'informations

Les zones d'informations sur le bon de commande contiennent les principales informations relatives au bon de commande. Le résumé du bon de commande s'affiche également dans la partie supérieure de la vue Détails. Les utilisateurs disposant des autorisations appropriées peuvent modifier ces zones.

onglets des vues détails

Chacun d'eux affiche des informations spécifiques que vous pouvez utiliser pour les activités relatives au bon de commande. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
