Résumé du bon de commande

Vous pouvez afficher des informations sur la valeur du bon de commande sélectionné dans le résumé du bon de commande situé dans la section supérieure de la vue Détails du bon de commande. Vous pouvez modifier les informations le cas échéant.

Le lien hypertexte Copier dans un e-mail vous permet d'envoyer un e-mail dont le corps du message contient déjà les informations sur le bon de commande. Les informations copiées dans l'e-mail comprennent le compte, l'opportunité, la date de création, la date de livraison garantie, l'ID du bon de commande, le type, l'état et des commentaires. De plus, elles comprennent les informations sur les produits, la facturation et l'expédition, ainsi que la valeur et les informations récapitulatives du résumé du bon de commande.

Les informations qui s'affichent dans le résumé dépendent de l'état du bon de commande.

Pour copier les informations du bon de commande dans un e-mail

  • Ouvrez la vue Détails du bon de commande.

  • Dans la barre de navigation, cliquez sur Bons de commande pour ouvrir la vue Liste des bons de commande. Si nécessaire, utilisez la Recherche pour localiser un bon de commande, puis cliquez sur le nom du bon de commande.
  1. Dans la zone Résumé du bon de commande, cliquez sur Copier dans un e-mail.

Un e-mail s'ouvre avec les informations du bon de commande dans le corps de l'e-mail, les contacts de facturation et d'expédition dans le champ A et le contact ayant demandé le bon de commande dans le champ Cc. Le champ Objet contient l'ID du bon de commande et le compte d'expédition.

Résumé du bon de commande

Si l'option Devises multiples est configurée, le résumé affichera les informations sur la devise de base, la devise du bon de commande et votre propre devise.

 

Elément Description Modifiable par l'utilisateur ?
Sous-total Le coût total de tous les produits associés au bon de commande, moins les remises. Non
Remise Toute remise devant être appliquée au bon de commande. Cliquez sur Remise pour modifier. Oui
Expédition Le coût d'expédition de la commande au client. Cliquez sur Expédition pour modifier. Oui
Taxe Le taux d'imposition à appliquer au bon de commande. Cliquez sur Taxe pour modifier. Oui
Total général Le montant total du bon de commande, incluant les remises, le coût d'expédition et la taxe. Non

Devise

(Devises multiples uniquement)

 Affiche le type de devise si la commande a été créée à l'aide d'un type de devise différent. Pour modifier, cliquez sur le type de devise pour rechercher la nouvelle devise. Oui

Taux de change

(Devises multiples uniquement)

 Affiche le taux de change si la commande a été créée à l'aide d'un type de devise différent. Non

Type de taux de change

(Devises multiples uniquement)

 Affiche la date de définition de la devise. Non
