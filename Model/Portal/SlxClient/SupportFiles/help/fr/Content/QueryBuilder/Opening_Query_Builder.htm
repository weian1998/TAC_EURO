Ouverture de Query Builder

Vous pouvez utiliser Query Builder pour ajouter ou modifier des groupes.

Pour ouvrir Query Builder

  • Effectuez l'une des opérations suivantes :
  • Dans la liste de vue appropriée, sur le menu Groupes, cliquez sur Ajouter un groupe pour ajouter un groupe.
  • Cliquez avec le bouton droit de la souris sur le groupe à modifier, puis cliquez sur Modifier.
