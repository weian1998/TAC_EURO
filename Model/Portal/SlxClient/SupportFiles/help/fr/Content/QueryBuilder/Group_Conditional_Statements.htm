Groupement d'instructions conditionnelles

Dans une requête de base de données avancée, vous devrez peut-être utiliser plusieurs conditions et également grouper ces conditions afin que des recherches plus complexes puissent être réalisées.

Pour grouper des instructions conditionnelles

  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Conditions et ajoutez toutes les instructions conditionnelles.
  3. Pour la première condition du groupe, cliquez sur le bouton Ajouter (+) dans la colonne symbole de parenthèse d'ouverture « ( ».
  4. Pour la dernière condition du groupe, cliquez sur le bouton Ajouter (+) dans la colonne symbole de parenthèse de fermeture « ) ».
  5. Continuez ainsi jusqu'à avoir créer tous les groupes que vous souhaitez. Il est possible de réaliser des groupements par sous-niveaux, mais cela peut entraîner des confusions. Chaque condition suivante sera appliquée en tant que sous-niveau de la condition la précédant.

Pour supprimer le groupe des instructions conditionnelles

  1. Pour la première condition du groupe, cliquez sur le bouton Supprimer (-) situé à côté du symbole de parenthèse d'ouverture « ( ».
  2. Pour la dernière condition du groupe, cliquez sur le bouton Supprimer (-) situé à côté du symbole de parenthèse de fermeture « ) ».
  3. Modifiez les connecteurs au besoin.
