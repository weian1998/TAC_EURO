Chaînes de format de champ fixe

Le tableau suivant dresse la liste des options de chaîne de formatage pour les champs fixes. La validité d'un format de chaîne dépend du type de format sélectionné.

Option Description
e Scientifique. Le format de ce champ doit être du type valeur à virgule flottante. La valeur est convertie en chaîne du type « -d.ddd...E+ddd ». La chaîne résultante commence par un signe moins si le nombre est négatif. La virgule est toujours précédée d'un chiffre. Le nombre total de chiffres dans la chaîne résultante (y compris celui avant la virgule) est donné par le spécificateur de précision dans la chaîne de format ; une précision par défaut de 15 est employée s'il n'existe aucun spécificateur de précision. L'exposant « E » dans la chaîne résultante est toujours suivi d'un signe plus ou moins et d'au moins trois chiffres.
f Fixe. La valeur est affichée en tant que champ fixe. Le format de ce champ doit être de type valeur à virgule flottante. La valeur est convertie en chaîne du type « -ddd.ddd... ». La chaîne résultante commence par un signe moins si le nombre est négatif. Le nombre de chiffres après la virgule est déterminé par le spécificateur de précision dans la chaîne de formatage ; la valeur par défaut est de 2 chiffres s'il n'existe aucun spécificateur de précision.
g Général. Le format de ce champ doit être de type valeur à virgule flottante. La valeur est convertie à la chaîne décimale la plus courte possible sous la forme scientifique ou fixe. Le nombre de chiffres pertinents dans la chaîne résultante est déterminé par le spécificateur de précision ; une précision par défaut de 15 est employée s'il n'existe aucun spécificateur de précision. Les zéros de fin sont supprimés de la chaîne et une virgule n'apparaît que si elle est nécessaire. La chaîne résultante utilise le format de virgule fixe si, dans la valeur, le nombre de chiffres après la virgule est inférieur ou égal à la précision définie et si la valeur est supérieure ou égale à 0,00001. Si ce n'est pas le cas, la chaîne résultante utilise le format scientifique.
n Nombre. Le format de ce champ doit être de type valeur à virgule flottante. La valeur est convertie en chaîne du type « -d,ddd,ddd.ddd... ». Le format « n » correspond au format « f », sauf si la chaîne résultante contient un séparateur de milliers.
