Présentation de Query Builder

Utilisez l'onglet Présentation de Query Builder pour indiquer les champs à afficher et l'apparence des données. Vous pouvez ajouter, modifier, supprimer ou réorganiser les champs dans la présentation. Vous pouvez également définir le nom de la colonne, sa largeur et le format de chaque champ. L'onglet Présentation doit comprendre au moins un champ pour que des données apparaissent.

Le Query Builder prend en charge de nombreuses fonctionnalités. Les onglets visibles dépendent de la fonctionnalité que vous êtes en train d'utiliser.
Pour ajouter un champ à une présentation
  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Présentation.

    La présentation par défaut des nouveaux groupes est définie d'après le groupe sélectionné comme groupe par défaut.

  3. Sélectionner une table.
  4. Sélectionnez un champ.

Vous pouvez faire défiler la liste pour afficher le champ ajouté à la présentation.

  1. Sélectionnez le champ, puis cliquez sur Modifier.
  2. Suivez les instructions de la boîte de dialogue Affecter une présentation à la requête.

Pour modifier un champ dans une présentation

  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Présentation.
  3. Sélectionnez un champ dans la grille.
  4. Cliquez sur Modifier.
  5. Effectuez vos changements dans la boîte de dialogue Présentation de la requête.
Pour supprimer un champ dans une présentation
  1. Ouvrez Query Builder
  2. Cliquez sur l'onglet Présentation.
  3. Sélectionnez un champ dans la grille.
  4. Cliquez sur Supprimer.
Pour réorganiser des champs dans une présentation
  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Présentation.
  3. Sélectionnez un champ. Effectuez l'une des opérations suivantes :
  • Faites glisser le champ vers son nouvel emplacement.
  • Cliquez sur Déplacer vers la gauche ou Déplacer vers la droite jusqu'à ce que le champ atteigne son nouvel emplacement.
