Tri de Query Builder

Utilisez l'onglet Tri de Query Builder pour indiquer le ou les champs par lesquels trier les données et si les données doivent apparaître dans l'ordre croissant ou décroissant. Les critères de tri sont appliqués dans l'ordre dans lequel ils apparaissent sous l'onglet.

  • Le Query Builder prend en charge de nombreuses fonctionnalités. Les onglets visibles dépendent de la fonctionnalité que vous êtes en train d'utiliser.

  • Certains rapports incluent des critères de tri prédéfinis qui peuvent être supprimés, si besoin est.
  • Les champs Blob peuvent être utilisés pour trier les données.

(Binary Large OBject) Type de données contenant un grand volume variable de données. Les blobs peuvent comporter des données binaires ou de texte. Utilisez l'Afficheur de blob dans la boîte de dialogue Jeu de résultats SQL pour afficher les données des blobs de texte.

  • Les champs calculés numériques ne peuvent pas être utilisés pour trier des données car le calcul mathématique ne s'achève que lorsque le tri a été appliqué.
Pour ajouter des critères de tri
  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Tri.
  3. Sélectionnez une table dans le côté gauche de Query Builder. Si besoin, cliquez sur le signe plus pour développer la liste des tables.
  4. Sélectionnez une table du côté droit de Query Builder. Cette zone affiche tous les champs appartenant à la table choisie. Double-cliquez sur le champ pour l'ajouter à la grille au bas de la boîte de dialogue.
  5. Cliquez sur la flèche du menu déroulant Direction, puis sélectionnez Croissant (de A à Z) ou Décroissant (de Z à A).
  6. Pour ajouter un champ de tri secondaire, répétez les étapes 3 à 5. Chaque option de tri suivante sera appliquée en tant que sous-niveau de l'option de tri la précédant.

Les champs calculés numériques ne peuvent pas être triés.

Pour réorganiser les critères de tri
  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Tri.
  3. Sélectionnez un champ dans la grille.
  4. Cliquez sur Monter ou Descendre.
Pour supprimer des critères de tri
  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Tri.
  3. Sélectionnez le champ à supprimer de la grille.
  4. Cliquez sur Supprimer.
