Que sont les curseurs côté serveur virtuels (VSSC)

Les curseurs côté serveur virtuels (VSSC, Virtual Server-Side Cursors) sont conçus pour améliorer les performances et optimiser l'utilisation de la mémoire pour les groupes de taille importante dans les vues Groupes et Liste du client Sage SalesLogix.

Pour déterminer si un groupe utilise les curseurs côté serveur virtuels :

  • Sélectionnez l'onglet du groupe dans la vue Liste ou Détails appropriée, et cherchez une des icônes suivantes dans le coin supérieur droit de la vue, à côté du bouton Options du groupe.
Icône Signification
Groupe en mode serveur virtuel : le groupe est traité à l'aide de curseurs côté serveur virtuels.
Groupe en mode client : le groupe est traité sur le client. Il ne fait pas appel aux curseurs côté serveur virtuels.

 

Pour plus d'informations sur les curseurs côté serveur virtuels, reportez-vous au Guide de référence des Développeurs LAN.
