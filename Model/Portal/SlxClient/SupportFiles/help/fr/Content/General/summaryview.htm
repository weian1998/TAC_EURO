Utilisation de la vue Résumé

Utilisez la vue Résumé pour voir les informations couramment utilisées pour chaque type d'enregistrement. Les informations et les fonctionnalités disponibles dépendent du type d'enregistrement.

Procédure

Ouvrez la vue Résumé
  • Sur une vue Liste, cliquez sur Résumé.

La vue Résumé affiche les informations dans trois colonnes.

Colonne Description/Action
Gauche

Affiche les informations principales de l'enregistrement. En fonction du type d'enregistrement, vous pourrez peut-être :

  • cliquer sur le nom de l'enregistrement pour ouvrir la vue Détails ;
  • cliquer sur une adresse e-mail pour envoyer un e-mail ;
  • cliquer sur une adresse Web pour ouvrir un site Web dans une nouvelle fenêtre de navigateur ;
  • Si vous consultez des activités, cliquez sur Terminer pour terminer l'activité.
Milieu Affiche des informations supplémentaires sur l'enregistrement pouvant inclure des liens vers des enregistrements associés ou des informations en lecture seule.
Droite

Affiche les types d'enregistrement associés. En fonction du type d'enregistrement, vous pourrez peut-être :

  • cliquer sur des liens hypertexte pour ouvrir la vue Détails avec l'onglet approprié sélectionné ;
  • cliquez sur des valeurs numériques pour afficher une liste des enregistrements associés. Par exemple, des contacts ou des tickets. La liste qui apparaît peut contenir des liens hypertexte pour ouvrir la vue Détails de l'enregistrement associé.
