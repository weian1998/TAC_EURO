Exportation d'enregistrements vers un fichier

Vous pouvez exporter des enregistrements vers un fichier. Les en-têtes de colonnes et la mise en page des groupes seront aussi exportées.

Si l'option Exporter vers un fichier n'est pas disponible, vous devez télécharger l'intégration du bureau Sage SalesLogix. Vous pouvez installer l'intégration du bureau Sage SalesLogix, à partir de l'écran de connexion du client Web.. 

Pour exporter des enregistrements vers un fichier

  1. Affichez le groupe ou les résultats de la recherche contenant les enregistrements que vous voulez exporter.
  2. Effectuez l'une des opérations suivantes :
    • Sélectionnez un ou plusieurs enregistrements en appuyant sur la touche CTRL ou Shift en cliquant sur chaque élément. Dans le Volet des tâches, sous Tâches courantes, sélectionnez Exporter vers un fichier.

      Si vous ne sélectionnez aucun enregistrement, on vous demandera si tous les enregistrements du groupe doivent être utilisés. Si vous cliquez sur OK, tous les enregistrements du groupe seront sélectionnés.

    • Pour exporter le groupe complet, cliquez avec le bouton droit de la souris sur l'onglet du groupe et cliquez sur Exporter vers un fichier.

    		3.

    Si vous ne sélectionnez aucun enregistrement, on vous demandera si tous les enregistrements du groupe doivent être utilisés. Si vous cliquez sur OK, tous les enregistrements du groupe seront sélectionnés.
  3. Sélectionnez le format de fichier que vous souhaitez utiliser.
    • Standard CSV - les enregistrements individuels sont séparés par une virgule
    • Délimité par des tabulations - les enregistrements individuels sont séparés par une tabulation

    Si vous souhaitez que Sage SalesLogix mémorise votre sélection et que vous ne souhaitez plus recevoir d'invite, sélectionnez Se souvenir de mon choix et ne plus me le demander.

  4. Cliquez sur OK.

Il se peut que vous receviez un message d'erreur si vous essayez d'exporter un groupe qui inclut un champ de données comportant un grand volume de données.
