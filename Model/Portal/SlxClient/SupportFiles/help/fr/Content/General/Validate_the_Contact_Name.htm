Ajouter ou modifier des informations sur des noms

Utilisez la boîte de dialogue Modifier le nom pour ajouter ou modifier des informations sur les noms.Ceci garantit qu'un nom comprenne un prénom et un nom et soit correctement stocké dans la base de données. Les fonctionnalités de modèle et de publipostage requièrent le nom, le titre et le suffixe corrects.

Pour ajouter ou modifier

  1. Ouvrez la boîte de dialogue Modifier le nom
    • Dans les vues Informations sur le compte/contact ou Détails du contact, cliquez sur le bouton Modifier le contact.
    • Dans les vues Insérer un lead, Détails du lead ou Détails sur l'utilisateur, cliquez sur le bouton Modifier le nom.
  2. Cliquez sur le Préfixe bouton en forme de flèche et sélectionnez un élément dans la liste.
  3. Dans le champ Prénom tapez le prénom de l'utilisateur.
  4. Dans le champ Deuxième tapez le deuxième prénom de l'utilisateur.
  5. Dans le champ Nom tapez le nom de l'utilisateur.
  6. Le cas échéant, cliquez sur le Suffixe bouton en forme de flèche et sélectionnez un élément dans la liste.

  7. Cliquez sur OK.
