Utilisation des menus contextuels de la barre de navigation

Les menus contextuels de la barre de navigation vous permettent d'ouvrir une vue de listes, de créer un nouvel enregistrement ou, le cas échéant, de créer un nouveau groupe. Les options disponibles des menus contextuels varient selon le type d'enregistrement.

Le menu contextuel de la barre de navigation n'est pas disponible pour les rubriques Nouveautés, Bibliothèque, Rapports ou Processus.

Pour ouvrir le menu contextuel de la barre de navigation

  • Dans la barre de navigation, cliquez avec le bouton droit sur Contacts, Comptes, Opportunités, Leads, Contrats, Retours, Défauts ou Tickets.
Elément de menu Description
Recherche... Ouvre la Recherche.
Nouveau... Ouvre la vue Nouveau contact/compte, Nouvelle opportunité, Nouveau lead, Insérer un nouveau contrat, Insérer un nouveau défaut, Insérer un retour, ou Nouveau Ticket.
Enregistrer en tant que groupe Cette option n'est disponible que dans une vue de liste après avoir effectué une recherche. Crée un nouveau groupe à l'aide des critères de recherche.
Créer un nouveau groupe Ouvre la boîte de dialogue Nouveau groupe qui vous permet de définir les critères du groupe.

Pour ouvrir le menu contextuel des agendas et des activités

  • Dans la barre de navigation, cliquez avec le bouton droit sur Agenda ou sur Activités.
    • Elément de menu Description
    Planifier un appel Ouvre la vue Planifier un appel téléphonique.
    Planifier une réunion Ouvre la vue Planifier une réunion.
    Planifier une tâche à faire Ouvre la vue Planifier une tâche à faire.
    Planifier une activité personnelle Ouvre la vue Planifier une activité personnelle.
    Planifier un événement Ouvre la vue Planifier un évènement.
