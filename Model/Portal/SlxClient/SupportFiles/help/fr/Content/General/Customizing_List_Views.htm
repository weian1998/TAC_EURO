Personnalisation de vos groupes de Vue de liste

Vous pouvez personnaliser vos onglets de vue de liste en ajoutant ou en supprimant des colonnes, en modifiant l'ordre des colonnes et la largeur des colonnes. Sage SalesLogix se souvient de ces modifications à chaque fois que vous vous connectez avec le même navigateur et ordinateur. Vous pouvez aussi masquer ou afficher les cases des vues de liste.

Ces modifications ne seront pas disponibles sur un autre ordinateur et seront perdues si vous effacez le cache de votre navigateur. Pour modifier un groupe de manière permanente, utilisez l'onglet Présentation de Query Builder.

Procédure

Ajouter ou supprimer des colonnes
  1. Dans une vue de liste, ouvrez le groupe que vous voulez modifier.
  2. Cliquez avec le bouton droit de la souris sur un en-tête de colonne, sélectionnez les colonnes que vous voulez ajouter et effacez celles que vous voulez supprimer.
  3. Lorsque vous avez terminé, cliquez hors du champ.
Modifier l'ordre des colonnes
  1. Dans une vue de liste, ouvrez le groupe que vous voulez modifier.
  2. Cliquez sur l'en-tête de la colonne que vous voulez déplacer et faites-la glisser à l'endroit voulu.
Modifier la largeur des colonnes
  1. Dans une vue de liste, ouvrez le groupe que vous voulez modifier.
  2. Placez votre souris sur le bord de la colonne jusqu'à ce que le curseur change, puis cliquez et faites glisser le bord de la colonne à la largeur souhaitée.
Masquer ou afficher la colonne à cocher
  1. Dans une vue de liste, ouvrez le groupe que vous voulez modifier.
  2. Cliquez avec le bouton droit de la souris sur un en-tête de colonne, puis sélectionnez la première option pour afficher les cases ou supprimez la première option pour masquer les cases.
  3. Lorsque vous avez terminé, cliquez hors du champ.
Masquer ou afficher les cases pour tous les groupes
  1. Dans le menu Outils, cliquez sur Options.
  2. Dans l'onglet Groupes
Annuler mes personnalisations
  • Dans une vue de liste, cliquez avec le bouton droit de la souris sur l'onglet de groupe pour lequel vous voulez annuler vos modifications et cliquez sur Rétablir la présentation.
    Annule toutes les personnalisations effectuées au groupe et rétablit l'apparence du groupe à la présentation définie par l'onglet Présentation de Query Builder.
