Ajout ou suppression de filtres

Les filtres vous permettent de limiter la liste des enregistrements s'affichant dans une vue Liste. Les filtres disponibles sont basés sur les colonnes de la vue Liste sélectionnée et sur les informations contenues dans ces colonnes. Cependant, vous pouvez personnaliser la liste des filtres pour inclure uniquement les filtres que vous utilisez pour chaque groupe.

Pour ajouter ou supprimer des filtres affichés dans le Volet des tâches

  1. Dans une vue de liste, dans le Volet des tâches, cliquez sur Modifier les filtres.
    La boîte de dialogue Modifier les filtres s'affiche.
  2. Cochez les cases des filtres que vous voulez afficher dans la liste Filtres et décochez les cases des filtres que vous voulez masquer. Cliquez sur Sélectionner tout pour sélectionner tous les filtres.

  3. Cliquez sur OK.
