Ajout ou suppression d'éléments de filtre

Les filtres vous permettent de limiter la liste des enregistrements s'affichant dans une vue Liste. Vous pouvez personnaliser vos filtres pour inclure uniquement les éléments de filtre que vous voulez utiliser pour chaque groupe.

Pour ajouter ou supprimer des éléments de filtre

  1. A partir d'une vue Liste, dans le Volet des tâches, sous Filtres, cliquez sur un nom de filtre. Par exemple, cliquez sur Propriétaire.
  2. Cliquez sur Modifier des éléments.
    La boîte de dialogue Modifier des éléments de filtre s'ouvre.
  3. Si nécessaire, saisissez les premières lettres dans la zone Rechercher un élément et cliquez sur Rechercher pour rechercher un élément de filtre.
  4. Cochez les cases des éléments de filtre que vous voulez afficher dans la liste Filtres et décochez les cases des éléments que vous voulez masquer. Cliquez sur Sélectionner tout pour sélectionner tous les éléments de filtre.
  5. Cliquez sur OK.
    Le filtre inclut uniquement les éléments sélectionnés.
