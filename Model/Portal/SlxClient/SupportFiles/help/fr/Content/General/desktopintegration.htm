Installation et utilisation du module d'intégration du bureau Sage SalesLogix

Le module d'intégration du bureau Sage SalesLogix est un ensemble de fonctionnalités vous permettant d'améliorer votre expérience Sage SalesLogix.

Les nouvelles fonctions incluent les éléments suivants :

  • Glisser-déposer des pièces jointes
  • Glisser-déposer des e-mails depuis Microsoft Outlook
  • Intégration de MS Outlook - Envoyer à SLX et Enregistrer dans l'historique
  • Publipostage
  • Exporter vers MS Excel
  • Glisser-déposer des fichiers de bibliothèque (Administrateur uniquement)

Procédure

Installer le module d'intégration du bureau si Internet Explorer est mon navigateur
  1. Effectuez l'une des opérations suivantes : .
    L'étiquette du bouton affichera Améliorations installées si la version la plus récente de l'intégration du bureau est déjà installée.
    • Sur l'écran de connexion du client Web Sage SalesLogix, cliquez sur Installer les améliorations
    • Dans le menu Outils, cliquez sur Options puis sur l'onglet Général puis cliquez sur cliquez sur Installer les améliorations.
  2. Cliquez sur Exécuter.
  3. Fermez Microsoft Outlook et toutes les fenêtres de navigateur ouvertes.
  4. Pour installer le module d'intégration du bureau, cliquez sur Oui.
  5. Si vous êtes informé que Microsoft Outlook ou Internet Explorer est en cours d'utilisation, fermez chaque programme et cliquez sur Réessayer.
  6. Dans l'écran Bienvenue, cliquez sur Suivant.
  7. Sur l’écran Contrat de licence, lisez les informations, acceptez l'accord, puis cliquez sur Suivant.
  8. Effectuez l'une des opérations suivantes :
    • Sélectionnez Terminer pour installer toutes les fonctionnalités de l'intégration du bureau dans l'emplacement d'installation par défaut.
    • Sélectionnez Personnaliser pour sélectionner les fonctionnalités de l'intégration du bureau que vous voulez installer ou modifier l'emplacement d'installation.
  9. Cliquez sur Suivant.
  10. Si vous avez sélectionné l'installation personnalisée, effectuez l'une des actions suivantes puis cliquez sur Suivant :
    • Si vous ne voulez pas installer l'intégration de bureau (publipostage, export vers MS Excel, glisser et déposer), cliquez sur la flèche du menu déroulant en regard de l'Intégration de bureau et sélectionnez Cette fonctionnalité ne sera pas disponible.
    • Si vous ne voulez pas installer l'intégration de MS Outlook (envoyer SLX/enregistrer dans l'historique et gestionnaire de bureau), cliquez sur la flèche du menu déroulant en regard de l'Intégration de MS Outlook et sélectionnez Cette fonctionnalité ne sera pas disponible.
    • Pour modifier l'emplacement d'installation, cliquez sur Modifier, accédez à ou saisissez l'emplacement d'installation souhaité puis cliquez sur OK.
  11. Cliquez sur Installer.
  12. Cliquez sur Terminer.

Le Gestionnaire de bureau s'ouvre, ce qui vous permet de définir vos options d'intégration de bureau pour Envoyer SLX et Enregistrer dans l'historique.

Si vous n'utilisez pas la fonction d'intégration de MS Outlook, vous n'avez pas besoin de configurer les options dans le Gestionnaire de bureau.

    Pour ouvrir le Gestionnaire de bureau Sage SalesLogix manuellement, dans la section Barre d'état de la barre des tâches Windows, double-cliquez sur l'icône du Gestionnaire de bureau Sage SalesLogix.
Installer le module d'intégration du bureau si Firefox est mon navigateur

Pour installer à l'aide de Firefox

  1. Effectuez l'une des opérations suivantes : .
    • Sur l'écran de connexion du client Web Sage SalesLogix, cliquez sur Installer les améliorations
    • Dans Options, cliquez sur l'onglet Général et cliquez sur Installer les améliorations.
  2. Cliquez sur Enregistrer le fichier.
  3. Dans la boîte de dialogue Téléchargements, double-cliquez sur SLXDesktopIntegrationSetup.exe.
  4. Cliquez sur Oui.
  5. Fermez Microsoft Outlook et toutes les fenêtres de navigateur ouvertes.
  6. Si vous êtes informé que Microsoft Outlook ou Firefox est en cours d'utilisation, fermez chaque programme et cliquez sur Réessayer.
  7. Dans l'écran Bienvenue, cliquez sur Suivant.
  8. Sur l’écran Contrat de licence, lisez les informations, acceptez l'accord, puis cliquez sur Suivant.
  9. Effectuez l'une des opérations suivantes :
    • Sélectionnez Terminer pour installer toutes les fonctionnalités de l'intégration du bureau dans l'emplacement d'installation par défaut.
    • Sélectionnez Personnaliser pour sélectionner les fonctionnalités de l'intégration du bureau que vous voulez installer ou modifier l'emplacement d'installation.
  10. Cliquez sur Suivant.
  11. Si vous avez sélectionné l'installation personnalisée, effectuez l'une des actions suivantes puis cliquez sur Suivant :
    • Si vous ne voulez pas installer l'intégration de bureau (publipostage, export vers MS Excel, glisser et déposer), cliquez sur la flèche du menu déroulant en regard de l'Intégration de bureau et sélectionnez Cette fonctionnalité ne sera pas disponible.
    • Si vous ne voulez pas installer l'intégration de MS Outlook (envoyer SLX/enregistrer dans l'historique et gestionnaire de bureau), cliquez sur la flèche du menu déroulant en regard de l'Intégration de MS Outlook et sélectionnez Cette fonctionnalité ne sera pas disponible.

    Si vous n'utilisez pas l'intégration de MS Outlook, il est inutile de configurer les options dans le gestionnaire du bureau.
