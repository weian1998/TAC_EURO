Ajout d'un élément de documentation

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails des éléments de documentation pour modifier les informations d'un élément de documentation.

Pour modifier

  1. Dans la barre de navigation, cliquez sur Administration, puis cliquez sur Eléments de documentation pour ouvrir la vue Liste des éléments de documentation. Si nécessaire, utilisez la fonction Rechercher pour localiser un élément de documentation, puis cliquez sur le numéro de l'élément.
  2. Si nécessaire, modifiez les informations des éléments de documentation.
  3. Cliquez sur Enregistrer.
