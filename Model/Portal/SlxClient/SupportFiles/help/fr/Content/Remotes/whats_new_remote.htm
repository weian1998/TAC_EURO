Nouveautés pour les Client Web hors ligne utilisateurs

Examen des modifications de synchronisation

Après la synchronisation, vous pouvez utiliser Nouveautés pour afficher les modifications effectuées dans votre base de données.

Utilisateurs distants uniquement

Utilisez la vue Nouveautés pour assurer le suivi des ajouts, des mises à jour et des suppressions dans la base de données locale, et pour vous abonner à des comptes. Les informations de la vue Nouveautés sont organisées sur plusieurs onglets.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Pas un utilisateur Client Web hors ligne ?

Procédure

Ouvrir la vue Nouveautés

Pour afficher les modifications de la synchronisation avec Nouveautés

  • Dans la barre de navigation, cliquez sur Nouveautés, puis sur les options au bas de la fenêtre.
    ClicPour rechercher les
    NouveauNouveaux comptes, contacts, opportunités, activités, remarques, historique et documents.
    Mis à jourComptes, contacts, opportunités, activités, remarques et historique auxquels vous êtes abonné et qui ont été modifiés par d'autres utilisateurs.
    SupprimésComptes, contacts ou opportunités supprimés par d'autres utilisateurs.
Filtrer les enregistrements
  1. Sélectionnez le type d'enregistrements à afficher. Sélectionnez Nouveau, Mis à jour ou Supprimé.
  2. Pour filtrer les enregistrements Nouveautés par date, cliquez sur le bouton Afficher les transactions à partir duBouton Agenda, puis sélectionnez une date, cliquez sur OK, et sur Rechercher.
Supprimer les transactions
  • Cliquez sur le bouton Supprimer les transactions à la date du ou avant leBouton Agenda, sélectionnez une date, puis cliquez sur OK et sur Supprimer.
Afficher les nouveaux enregistrements ou les enregistrements modifiés
  • Cliquez sur l'onglet du type d'éléments que vous souhaitez afficher. Les différentes options sont : Contacts, Historique, Notes, Comptes, Activités, Documents, et Opportunités.
