Synchronisation avec la base de données du bureau principal

Pour maintenir votre base de données Client distant à jour avec le bureau principal, il est important d'effectuer des synchronisations régulières. Si vous n'effectuez pas de synchronisation chaque jour ou tous les deux jours, les bases de données ayant des enregistrements partagés (bureau principal et Clients distants) seront obsolètes. Une synchronisation fréquente aide aussi à réduire le temps de synchronisation, car un plus petit nombre de transactions est traité.

La synchronisation automatique vous permet d'effectuer une synchronisation automatique lors de votre première connexion au bureau principal, puis de manière périodique tant que la connexion se poursuit. La synchronisation automatique est activée par défaut. Pour en savoir plus, consultez la rubrique « Client de synchronisation Sage SalesLogix » dans l'aide du client Sage SalesLogix.

Pour effectuer une synchronisation avec la base de données du bureau principal

  1. Dans le menu Fichier, cliquez sur Synchroniser.
  2. Dans l'onglet Synchronisation automatique, sélectionnez Activée.
  3. Dans les zones Synchronisation exécutée lors de la première connexion puis toutes les, définissez l'heure à laquelle vous voulez que la synchronisation se produise.
  4. Cliquez sur l'onglet Options de synchronisation.
  5. Pour synchroniser avec une connexion à distance, sélectionnez l'option Connexion avec RAS puis sélectionnez votre profil dans la liste Profil RAS. Décochez la case Connexion avec RAS si vous vous connectez avec une connexion par réseau.
  6. Cliquez sur Syncroniser maintenant pour démarrer le processus de synchronisation.
  7. Lorsque la synchronisation est terminée, cliquez sur Fermer.

Si des erreurs se produisent pendant la synchronisation, contactez votre administrateur système ou résolvez-les.
