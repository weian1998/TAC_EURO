Nouveautés

Utilisez l'option Nouveautés pour visualiser la liste de toutes les modifications récentes des informations auxquelles vous pouvez accéder. Cette vue présente une liste de l'ensemble des comptes, contacts, leads, opportunités, activités, notes et historiques, contrat, défauts, nouveaux ou modifiés. Vous pouvez uniquement voir les informations auxquelles vos droits de sécurité vous permettent d'accéder.

L'écran Nouveautés est composé de deux parties : un groupe d'onglets dans la partie supérieure qui permet d'accéder à un type particulier d'éléments modifiés (des comptes, par exemple) et la zone de liste qui permet de visualiser les éléments nouveaux ou modifiés.

Êtes-vous un utilisateur Client Web hors ligne ?

Pour afficher les Nouveautés

  1. Dans la barre de navigation, cliquez sur Nouveautés.
  2. Pour filtrer les enregistrements Nouveautés par date, cliquez sur le bouton Modifications de la base de données le ou à partir duBouton Agenda, puis sélectionnez une date, cliquez sur OK, et sur Rechercher.

Pour atteindre une liste spécifique d'éléments nouveaux ou modifiés

  • Cliquez sur l'onglet du type d'éléments que vous souhaitez afficher. Les différentes options sont : Comptes, Contacts, Leads, Opportunités, Activités, Notes,Contrats, Défauts, et Historique .

Pour afficher des informations détaillées

  1. Cliquez sur l'onglet correspondant au type d'élément, tel que Comptes.
  2. Recherchez l'élément que vous souhaitez afficher dans la liste.
  3. Cliquez sur le nom de l'élément.
