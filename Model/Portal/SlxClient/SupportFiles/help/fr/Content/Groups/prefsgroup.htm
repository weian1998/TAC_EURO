Options de groupe

Utilisez l'onglet Options de groupe pour déterminer la façon dont vos groupes, vues de listes et recherches apparaîtront et se comporteront. Ces options ne s'appliquent que lorsque l'utilisateur actuel est connecté à .

Pour définir des options de groupe

  1. Dans le menu Outils, cliquez sur Options.
  2. Cliquez sur l'onglet Groupe.
  3. Définissez vos Options de groupe.
    1. Cliquez sur la flèche de la liste déroulante Vue principale, puis sélectionnez une vue dans la liste.
    2. Cliquez sur la flèche de liste déroulante Groupe par défaut et sélectionnez un groupe par défaut dans la liste (Contacts récents, par exemple).

      3. La présentation par défaut des nouveaux groupes est définie d'après le groupe sélectionné comme groupe par défaut.

    3. Cliquez sur la flèche de liste déroulante Présentation par défaut des recherches et sélectionnez le nom du groupe avec la présentation que vous souhaitez voir appliquer pour toutes les recherches dans la vue principale. Par exemple, dans les vues Contact, toutes les recherches doivent utiliser la présentation de groupe Contacts récents.
      Le groupe que vous sélectionnez détermine également les éléments de recherche disponibles dans la Recherche. Les options de recherche dépendent des colonnes incluses dans la présentation de recherche. Vous devez choisir un groupe incluant toutes les colonnes dans lesquelles vous voudriez effectuer une recherche dans une recherche.
    4. Répétez les étapes a-c pour chaque vue principale.
  4. Dans Options générales de recherche, cliquez sur le bouton en forme de flècheCondition de recherche par défaut et sélectionnez la condition de recherche que vous voulez utiliser pour toutes les recherches de vues de liste. Les options comprennent : Commenant par, Contient, Égal à et Non égal à.
  5. Sélectionnez Ajustement automatique des colonnes pour redimensionner les colonnes en fonction de l'espace disponible, ou décochez la case pour conserver la taille des colonnes et permettre le défilement afin de voir les colonnes ne rentrant pas dans l'espace disponible.
  6. Cliquez sur Enregistrer.
