Affichage des groupes masqués

Vous pouvez afficher un groupe qui a été masqué.

Pour afficher un groupe masqué

  1. Dans la liste de vue appropriée, cliquez sur le menu Groupe.
  2. Assurez-vous que l'option Afficher les groupes masqués est sélectionnée
  3. Cochez les cases Visible des groupes que vous ne souhaitez plus masquer
