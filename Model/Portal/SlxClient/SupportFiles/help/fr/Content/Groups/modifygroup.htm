Modification de groupes

Vous pouvez utiliser Query Builder pour modifier vos groupes.

Pour modifier un groupe

  1. Ouvrez une vue Liste ou Détails.
  2. Cliquez avec le bouton droit de la souris sur un onglet de groupe.
  3. Cliquez sur Modifier.

Query Builder s'ouvre.

  1. Utilisez les onglets de Query Builder pour modifier le groupe.
  • Onglet Propriétés - Utilisez cet onglet pour définir le nom et la description du groupe.
  • Onglet Conditions - Utilisez cet onglet pour définir les instructions conditionnelles servant à sélectionner les enregistrements. Cet onglet n'est pas disponible pour les groupes ad hoc.
  • Onglet Présentation - Utilisez cet onglet pour définir les données qui apparaîtront dans la vue de liste.
  • Onglet Tri - Utilisez cet onglet pour définir le mode de tri des enregistrements.
  1. Une fois toutes les modifications effectuées, cliquez sur OK.
