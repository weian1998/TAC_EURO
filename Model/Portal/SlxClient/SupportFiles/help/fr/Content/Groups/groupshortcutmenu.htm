Utilisation du menu contextuel des groupes

Le menu contextuel des groupes vous permet de créer et de gérer des groupes.

Toutes les options de menu ne sont pas disponibles sur toutes les vues de liste.

Pour ouvrir le menu contextuel des groupes

  1. Dans une vue de liste ou détails, cliquez avec le bouton droit sur l'onglet d'un groupe.
  2. Choisissez parmi les options suivantes :
    • Copier le
    • Modifier
    • Masquer
    • Supprimer
    • Partager
    • Exporter vers MS Excel
    • Exporter vers un fichier
    • Définir par défaut
    • Rétablir la présentation

    Annule toute personnalisation apportée au groupe et rétablit l'apparence du groupe sur la présentation déterminée par l'onglet Présentation de Query Builder.

    • Ajouter un groupe

Pour ouvrir le menu contextuel Résultats de la recherche

  1. Après avoir effectué une Recherche, cliquez avec le bouton droit de la souris sur l'onglet Résultats de la recherche.
  2. Choisissez parmi les options suivantes :
    • Enregistrer la recherche en tant que groupe
    • Exporter vers MS Excel
    • Exporter vers un fichier
    • Rétablir la présentation

    Annule toute personnalisation apportée au groupe et rétablit l'apparence du groupe sur la présentation déterminée par l'onglet Présentation de Query Builder.

    • Ajouter un groupe

Pour ouvrir le menu contextuel des vues de liste Groupe,

  1. Sélectionnez un ou plusieurs enregistrements.
  2. Cliquez avec le bouton droit de la souris dans la vue de liste Groupe.
  3. Choisissez parmi les options suivantes :
    • Enregistrer les enregistrements en tant que groupe
    • Supprimer du groupe.
    • Ajouter au groupe existant

    • Mettre à jour les leads (vue de liste Lead uniquement)

    Vous permet d'affecter un propriétaire aux enregistrements sélectionnés.

    • Supprimer les leads (vue de liste Lead uniquement)
    • Mettre à jour les opportunités (vue de liste Opportunité)
    • Statistiques de l'opportunité (vue de liste Opportunité)
    • Fusionner les enregistrements (vues de liste Contact et Compte uniquement)
    • E-mail (vues de liste Contact et Lead uniquement)
