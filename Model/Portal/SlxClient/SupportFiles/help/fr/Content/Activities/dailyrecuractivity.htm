Activités périodiques, Tous les jours

Vous pouvez choisir de planifier une activité pour qu'elle ait lieu une fois, tous les jours, toutes les semaines, tous les mois ou tous les ans.

Pour planifier une activité avec une périodicité quotidienne

  1. Planifiez ou modifiez une activité.
  2. Cliquez sur l'onglet Périodicité.
  3. Dans Fréquence, sélectionnez Quotidienne.
  4. Sélectionnez Chaque X Jour(s) ou, si vous souhaitez que l'activité ne se répète qu'une fois les occurrences précédentes terminées, sélectionnez Chaque X Jour(s) après l'achèvement.
  5. Saisissez le nombre de jours entre les répétitions.
  6. Dans la zone Démarrer la périodicité, cliquez sur le Bouton Agenda, sélectionnez une date puis cliquez sur OK.

  7. Effectuez l'une des opérations suivantes :

    • Dans la zone Terminer après, saisissez le nombre de fois que l'activité a lieu.
    • Cliquez sur Terminer la périodicitéBouton Agenda, sélectionnez une autre date puis cliquez sur OK.
  8. Lorsque vous avez terminé toutes les modifications d'activité, cliquez sur OK.
