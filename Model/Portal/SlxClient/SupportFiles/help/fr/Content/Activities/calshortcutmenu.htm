Utilisation des menus contextuels des agendas

Les menus contextuels des agendas permettent de planifier une nouvelle activité ou de travailler avec des activités planifiées. Les options des menus contextuels disponibles dépendent d'où l'on accède au menu.

Pour planifier une activité

  • Dans n'importe quelle vue Agenda, cliquez avec le bouton droit de la souris à n'importe quel endroit sauf sur l'activité.
Option de menu Description
Planifier un appel Ouvre la boîte de dialoguePlanifier un appel téléphonique.
Planifier une réunion Ouvre la boîte de dialogue Planifier une réunion.
Planifier une tâche à faire Ouvre la boîte de dialogue Planifier une tâche à faire.
Planifier une activité personnelle Ouvre la boîte de dialogue Planifier une activité personnelle.
Planifier un événement Ouvre la boîte de dialogue Planifier un évènement.

Pour travailler avec une activité planifiée

  • Dans une des vues Agenda, cliquez avec le bouton droit sur une activité.
Option de menu Description
Ouvrir une activité Ouvre la boîte de dialogue Modifier l'activité.
Terminer l'activité Ouvre la boîte de dialogue Terminer l'activité.
Supprimer l'activité Supprime l'activité.
Aller à Vous permet d'ouvrir la vue détaillée de tout enregistrement associé à l'activité.

Pour travailler avec une activité terminée

  • Dans une des vues Agenda, cliquez avec le bouton droit sur une activité terminée.
Option de menu Description
Modifier l'historique Ouvre la boîte de dialogue Détails d'historique.

Pour travailler avec un événement

Option de menu Description
Ouvrir un événement Ouvre la boîte de dialogue Modifier l'activité.
Supprimer un événement Supprime l'événement.
