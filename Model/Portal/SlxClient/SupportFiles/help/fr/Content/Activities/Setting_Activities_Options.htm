Définition des options des Activités

Utilisez l'onglet Activités pour définir les valeurs par défaut telles que le type d'activité

Pour définir les options des activités

  1. Dans le menu Outils, cliquez sur Options.
  2. Cliquez sur l'onglet Activités.
  3. Dans la liste déroulante Vue par défaut, sélectionnez l'onglet du type d'activités que vous souhaitez afficher dans la vue principale des activités (Toutes, par exemple).
  4. Dans la liste déroulante Activité de suivi par défaut, sélectionnez le type d'activité à planifier comme suivi une fois l'activité terminée.
  5. Dans la liste déroulante Reporter les notes, sélectionnez Oui si vous souhaitez reporter des notes existantes lorsque vous planifiez une activité de suivi.
  6. Dans la liste déroulante Reporter les pièces jointes, sélectionnez Oui si vous souhaitez reporter des pièces jointes existantes lorsque vous planifiez une activité de suivi.
  7. Sélectionnez Afficher les activités échues dans la barre d'outils si vous souhaitez être averti des activités échues.
  8. Dans la section Valeurs par défaut par type d'activité, définissez l'alarme par défaut et la durée par défaut pour chaque type d'activité :
    1. Cliquez sur l'alarme par défautla flèche du menu déroulant et effectuez votre sélection dans la liste pour définir l'alarme par défaut quand vous planifiez une nouvelle réunion.
    2. Cliquez sur la durée par défautla flèche du menu déroulant et effectuez votre sélection dans la liste pour définir la durée par défaut quand vous planifiez une nouvelle réunion.
    3. Sélectionnez l'option Activité sans horaire par défaut si vous souhaitez qu'une activité soit sans horaire par défaut.
    4. Si vous avez sélectionné l'option Activité sans horaire par défaut et que vous souhaitez que les activités soient automatiquement reportées au jour suivant si elles ne sont pas terminées à la date prévue, alors sélectionnez l'option Report automatique.

      Vous pouvez uniquement reporter les activités sans horaire.

    5. Répétez les étapes a à d pour les appels téléphoniques, les tâches à faire et les activités personnelles.
  9. Cliquez sur Enregistrer.
    Cliquez sur Enregistrer dans chaque onglet ou les sélections de vos options ne seront pas enregistrées.
