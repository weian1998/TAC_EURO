Suppression d'une activité

Vous avez la possibilité de supprimer n'importe quelle activité que vous créez.

Pour supprimer une activité unique

  1. Effectuez l'une des opérations suivantes :

    • Ouvrez l'activité à terminer et cliquez sur Supprimer.
    • Dans n'importe quel onglet Activités, cliquez sur Terminer.
    • Dans la vue Liste des activités, faites un clic droit sur une activité et sélectionnez Supprimer l'activité.
  2. Si vous avez sélectionné une activité récurrente et que cela vous est demandé, sélectionnez Supprimer cette occurrence ou Supprimer toutes les occurrences, puis cliquez sur Continuer.
  3. Cliquez sur OK pour supprimer l'enregistrement de façon définitive ou sur Annuler pour annuler cette suppression.

Si vous supprimez une activité dont vous êtes membre, le responsable de l'activité reçoit une notification l'informant que vous avez refusé l'activité.

Pour supprimer plusieurs activités

  1. Dans la vue Activités, sélectionnez les activités à supprimer, et dans la zone Tâches communes du volet Tâches, cliquez sur Supprimer l'activité, ou faites un clic droit sur la grille puis sélectionnez Supprimer l'activité.

    Si vous sélectionnez plusieurs activités et que certaines sont récurrentes, alors seules la ou les occurrences sélectionnées de l'activité récurrente seront supprimées.
  2. Cliquez sur OK pour supprimer l'enregistrement de façon définitive ou sur Annuler pour annuler cette suppression.
