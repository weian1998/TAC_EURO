Confirmations d'activité

Les confirmations d'activité vous envoient des notifications sur les activités si vous êtes un participant à une activité ou le responsable d'une activité. Vous recevez une notification lorsqu'un autre utilisateur vous ajoute comme participant ou responsable d'une activité ou lorsqu'une activité à laquelle vous participez est mise à jour ou effacée. Si vous êtes ajouté en tant que participant, vous devez confirmer ou infirmer votre présence à l'activité. Si vous avez planifié une activité à laquelle vous avez convié d'autres participants, vous recevez une notification à chaque fois qu'un participant confirme ou infirme sa présence.

Utilisez l'onglet Confirmations de la vue Liste des activités pour afficher et gérer les confirmations des activités. Si vous avez défini vos options d'alertes pour qu'elles comprennent les confirmations, vous recevrez une Alerte quand un autre utilisateur vous ajoute à une activité.

Le tableau suivant répertorie les types de notifications concernant les activités :

Type Description
Modifiée L'activité a été mise à jour
Confirmée Un participant à une activité que vous avez programmée a confirmé.
Refusée Un participant à une activité que vous avez programmée a décliné.
Supprimée Vous notifie qu'une activité à laquelle vous êtes convié a été supprimée.
Responsable Vous avez été nommé responsable d'une activité planifiée.
Nouveau Une nouvelle activité a été planifiée et vous y êtes convié en tant que membre

Procédure

Ouvrir une confirmation
  1. Dans la vue Activités, cliquez sur Confirmations.
  2. Cliquez sur le lien dans la colonne Notification.
    La vue Détails de la confirmation s'ouvre. Le type de confirmation déterminera les actions disponibles.
Accepter ou refuser une activité

  1. Effectuez l'une des opérations suivantes :

    • Dans la vue Activités, cliquez sur Confirmations, puis sur le lien Nouveau dans la colonne Notification de la confirmation à accepter ou à refuser.
    • Dans la vue Alertes, cliquez sur Non confirmée, puis cliquez sur les informations relatives à l'activité qui apparaissent en bleu à côté de l'icône de l'activité.
  2. Consultez les détails de l'activité.
  3. Saisissez une réponse dans la zone Réponse.
  4. Cliquez sur Accepter ou sur Refuser.

    Vous pouvez également accepter ou refuser une nouvelle confirmation sans consulter les détails de l'onglet Confirmations de la vue Activités en cliquant avec le bouton droit de la souris sur la confirmation, puis en sélectionnant Accepter la confirmation ou Refuser la confirmation, ou bien dans l'onglet Non confirmées des alertes, en sélectionnant la confirmation et en cliquant sur Accepter ou Refuser.
Afficher une confirmation sans la supprimer
  1. Dans la vue Activités, ouvrez l'onglet Confirmations.
  2. Cliquez sur le lien Notification de la confirmation à supprimer.
  3. Cliquez sur Fermer.
Supprimer une confirmation
  1. Dans la vue Activités, ouvrez l'onglet Confirmations.
  2. Cliquez sur le lien Notification de la confirmation à supprimer.

    Il est impossible de supprimer les confirmations signalées comme Nouvelles.
  3. Cliquez sur le bouton Supprimer la confirmation.
