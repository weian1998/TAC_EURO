Utilisation de la vue Activités

Utilisez la vue Activités pour gérer vos activités, événements, demandes de documentation et confirmations planifiés. Si vous disposez de droits d'accès supplémentaires, vous pouvez également afficher d'autres activités d'utilisateur Sage SalesLogix. Utilisez la vue Résumé ou le volet des détails pour afficher des informations supplémentaires pour chaque activité. Vous pouvez utiliser les filtres pour affiner la liste des enregistrements dans la liste. Par exemple, si la période est le mois suivant, utilisez le filtre Période pour afficher uniquement les activités planifiées pour le mois suivant.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Les onglets qui apparaissent dans la vue Activités sont :

Mes activités Toutes Activités échues
Evénements Confirmations
Documentation

Procédure

Ouvrez la vue Liste des activités

Pour ouvrir la vue Liste des activités

  • Dans la barre de navigation, cliquez sur Activités.
Actualiser les informations de la liste
  • Cliquez sur le bouton Actualiser.

Travailler avec des enregistrements dans une grille

Utilisez le menu contextuel des activités

Afficher ou masquer le volet des détails
  1. Cliquez sur Détails.
  2. Sélectionnez un enregistrement.
    Le volet des détails apparaît en bas de la vue Liste et affiche des informations détaillées sur l'enregistrement sélectionné.
  1. Pour masquer le volet des détails, cliquez sur Masquer les détails.
Afficher le volet de résumé
  1. Cliquez sur Résumé.
  2. Pour retourner vers la vue Liste, cliquez sur Liste.

Utilisez les filtres du Volet des tâches pour réduire la liste

Les filtres disponibles sont basés sur les colonnes de la vue Liste sélectionnée et sur les informations contenues dans ces colonnes.
Exemple de filtres
  1. Cliquez sur l'onglet Mes activités.
  2. Utilisez les filtres du Volet des tâches pour réduire la liste :
    • Développez le filtre Type d'activité et sélectionnez Appel téléphonique.
    • Développez le filtre Période et sélectionnez Demain.
      Tous les appels planifiés pour demain apparaissent dans l'onglet.
