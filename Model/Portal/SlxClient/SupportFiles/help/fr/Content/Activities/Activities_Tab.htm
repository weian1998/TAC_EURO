Onglet Activités

L'onglet Activités apparaît dans les vues Détails du compte, du contact, du ticket, du lead, de l'opportunité. Utilisez l'onglet Activités pour gérer les activités et événements associés à l'enregistrement actuel. Les informations affichées dans l'onglet Activités comprennent le type, la date et l'heure, la durée, l'utilisateur, la catégorie et l'objet de l'activité.

Procédure

Ouvrir l'onglet Activités
  • Dans la vue Détails du contact, Détails du compte, Détails de l'opportunité, Détails du lead ou Détails du ticket, cliquez sur l'onglet Activités. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.

Filtrer ma liste

  1. Cliquez sur le bouton Filtrer.
  2. Effectuez une ou plusieurs des actions suivantes :
    • Cliquez surTypela flèche du menu déroulant et effectuez votre sélection dans la liste.
    • Utilisez les zones Plage de dates pour sélectionner une plage de dates pendant laquelle la pièce jointe a été mise à jour pour la dernière fois.
    • Cliquez sur la flèche du menu déroulant et effectuez votre sélection dans la listeUtilisateur.
    • Dans la zone Objet, saisissez un ou plusieurs caractères contenus dans les informations sur l'objet. Plus vous saisirez de caractères, plus la liste renvoyée sera courte.
  3. Cliquez sur Appliquer pour filtrer la liste ou Réinitialiser pour effacer vos options de filtrage.
    Cliquez de nouveau sur le bouton Filtrer pour masquer les filtres.
Afficher les informations sur les activités
  • Cliquez sur le type de l'activité à afficher.

Planifier un appel téléphonique

  • Cliquez sur le bouton Planifier un appel téléphonique puis suivez les étapes pour planifier un appel téléphonique.

Planifier une réunion

  • Cliquez sur le bouton Planifier une réunion puis suivez les étapes pour planifier une réunion.

Planifier une tâche à faire

  • Cliquez sur le bouton Planifier une tâche à faire puis suivez les étapes pour planifier une tâche.
Terminer une activité
  1. Cliquez sur Terminer en regard de l'activité que vous souhaitez terminer.
  2. Suivez les étapes pour terminer une activité.

Pour une activité minutée associée à un ticket, utilisez l'onglet Activité du ticket.
