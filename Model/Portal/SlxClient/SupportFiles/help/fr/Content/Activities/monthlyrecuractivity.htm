Activités périodiques, Tous les mois

Vous pouvez choisir de planifier une activité pour qu'elle ait lieu une fois, tous les jours, toutes les semaines, tous les mois ou tous les ans.

Pour planifier une activité avec une périodicité mensuelle

  1. Planifiez ou modifiez une activité.
  2. Cliquez sur l'onglet Périodicité.
  3. Sous Fréquence, sélectionnez Mensuelle.
  4. Sélectionnez le type de fréquence de l'activité.
    • Si vous souhaitez que l'activité se répète un jour particulier dans le mois, sélectionnez l'option Chaque X Mois le, saisissez le nombre de mois entre chaque répétition, puis cliquez sur la flèche du menu déroulant pour sélectionner le jour du mois où vous souhaitez voir l'activité se répéter.
      Par exemple, une activité se répète tous les deux mois, le 15.
    • Si vous souhaitez que l'activité se répète un jour de la semaine particulier tous les mois, sélectionnez l'option Chaque X Moisle, saisissez le nombre de mois entre les répétitions, cliquez sur la flèche du menu déroulant pour sélectionner la semaine du mois, puis cliquez sur la flèche du menu déroulant pour sélectionner le jour de la semaine où vous souhaitez que l'activité se répète.
      Par exemple, une activité se répète le troisième jeudi de chaque mois.
    • Si vous souhaitez que l'activité ne se répète qu'une fois les occurrences précédentes terminées, sélectionnez l'option Chaque X Mois après la fin, puis saisissez le nombre de mois entre les répétitions.
      Par exemple une activité se répète un mois après qu'elle est été terminée.
  5. Dans la zone Démarrer la périodicité, cliquez sur le Bouton Agenda, sélectionnez une date puis cliquez sur OK.

  6. Effectuez l'une des opérations suivantes :

    • Dans la zone Terminer après, saisissez le nombre de fois que l'activité a lieu.
    • Cliquez sur Terminer la périodicitéBouton Agenda, sélectionnez une autre date puis cliquez sur OK.
  7. Lorsque vous avez terminé toutes les modifications d'activité, cliquez sur OK.
