Activités périodiques, Tous les ans

Vous pouvez choisir de planifier une activité pour qu'elle ait lieu une fois, tous les jours, toutes les semaines, tous les mois ou tous les ans.

Pour planifier une activité avec une périodicité annuelle

  1. Planifiez ou modifiez une activité.

  2. Cliquez sur l'onglet Périodicité.

  3. Sous Fréquence, sélectionnez Annuelle.
  4. Sélectionnez le type de fréquence de l'activité.
    • Si vous souhaitez que l'activité se répète des jours particuliers de l'année, sélectionnez l'option Chaque X Année(s) le, puis cliquez sur les flèches de menu déroulant pour sélectionner le mois et le jour où vous souhaitez que l'activité se répète.
      Par exemple, une activité se répète tous les deux ans, le 15 octobre.
    • Si vous souhaitez que l'activité se répète un jour particulier de la semaine tous les ans, sélectionnez la deuxième option, puis cliquez sur les flèches de menu déroulant pour sélectionner la semaine du mois, le jour de la semaine et le mois où vous souhaitez que l'activité se répète.
      Par exemple, une activité se répète chaque premier mardi de janvier.
    • Si vous souhaitez que l'activité ne se répète qu'une fois les occurrences précédentes terminées, sélectionnez l'option Chaque X Année(s) après la fin, puis saisissez le nombre d'années entre les répétitions.
      Par exemple une activité se répète deux ans après qu'elle est été terminée.
  5. Dans la zone Démarrer la périodicité, cliquez sur le Bouton Agenda, sélectionnez une date puis cliquez sur OK.

  6. Effectuez l'une des opérations suivantes :

    • Dans la zone Terminer après, saisissez le nombre de fois que l'activité a lieu.
    • Cliquez sur Terminer la périodicitéBouton Agenda, sélectionnez une autre date puis cliquez sur OK.
  7. Lorsque vous avez terminé toutes les modifications d'activité, cliquez sur OK.
