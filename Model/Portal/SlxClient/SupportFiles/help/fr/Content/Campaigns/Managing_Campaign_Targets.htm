Gestion des cibles de la campagne

Utilisez la boîte de dialogue Rechercher les cibles pour déterminer quels contacts et quels leads peuvent être les cibles d'une campagne.

Procédure

Ouvrez la boîte de dialogue Rechercher les cibles
  • Dans la vue Détails de campagne, cliquez sur l'onglet Cibles, puis sélectionnez Ajouter des cibles. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
Utilisez les filtres pour ajouter des cibles à une campagne
  1. Cliquez sur l'onglet Rechercher les cibles.
  2. Cliquez sur le bouton Gérer la liste.
  3. Utilisez les filtres pour réduire la liste des enregistrements.
  4. Pour savoir combien de cibles répondent aux critères de filtre avant de commencer la recherche, cliquez sur le bouton Combien ?.
  5. Cliquez sur Rechercher.
  6. Cliquez sur le bouton Ajouter des cibles pour ajouter toutes les cibles de la liste.
    Les cibles apparaissent dans la grille.
Utilisez les groupes pour ajouter des cibles à une campagne
  1. Cliquez sur l'onglet Ajouter à partir d'un groupe.
  2. Sélectionnez l'option Groupe de leads ou Groupe de contacts .
  3. Cliquez sur la flèche du menu déroulant et effectuez votre sélection dans la liste qui apparaît.
  4. Cliquez sur Rechercher.

La liste des enregistrements de groupe s'ouvre.

  1. Cliquez sur Ajouter des cibles.

Les cibles à partir du groupe sont ajoutées à la campagne.
