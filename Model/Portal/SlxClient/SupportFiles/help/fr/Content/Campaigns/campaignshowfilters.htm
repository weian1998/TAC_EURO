Filtres de réponses et de cibles de campagne

Utiliser les options d'Afficher les filtres pour limiter les contacts et les leads de cible qui apparaissent dans la liste.

Pour filtrer

  1. Cliquez sur Afficher les filtres pour développer la liste des filtres.
  2. Cochez les cases correspondant aux filtres à appliquer.
    • Contacts : affiche les réponses des contacts qui correspondent aux critères du filtre.
    • Leads : affiche les réponses des leads qui correspondent aux critères du filtre.
    • Répondu : sélectionnez Oui pour afficher les cibles ayant répondu ou Non pour afficher les cibles n'ayant pas répondu.
    • Etat : cliquez sur la flèche du menu déroulant puis sélectionnez un état dans la liste. La liste affichera uniquement les cibles avec l'état sélectionné.
    • Groupe : cliquez sur la flèche du menu déroulant et sélectionnez un groupe dans la liste. La liste affichera uniquement les cibles dans le groupe sélectionné.
    • Priorité : cliquez sur la flèche du menu déroulant et sélectionnez une priorité dans la liste. La liste affichera uniquement les cibles avec la priorité sélectionnée.
    • Dernière phase : cliquez sur la flèche du menu déroulant puis sélectionnez un phase dans la liste. La liste affichera uniquement les cibles dans la phase sélectionnée.

    Vous pouvez sélectionner plusieurs options de filtrage. Par exemple, vous pouvez sélectionner les options Afficher les contacts ou Méthodes de réponse pour visualiser les réponses des contacts ayant la même méthode de réponse.
  3. Si nécessaire, cliquez sur le bouton Actualiser pour mettre la grille à jour.
