Gestion des ressources

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Vous pouvez ajouter une ressource, telle qu'une salle de conférence, à la liste des ressources disponibles qui apparaît lorsque vous planifiez une activité.

Procédure

Ajout d'une ressource
  1. Dans le menu Administration, cliquez sur Nouvelle ressource.
  2. Renseignez les zones d'informations sur la ressource.
  1. Cliquez sur Enregistrer.
Modification d'une ressource
  1. Dans la barre de navigation, cliquez sur Administration, puis cliquez sur Ressources pour ouvrir la vue Liste des ressources. Si nécessaire, utilisez la Recherche pour localiser une ressource, puis cliquez sur le nom de la ressource.
  2. Si nécessaire, modifiez les zones d'information de la ressource.
  3. Cliquez sur Enregistrer.
Suppression d'une ressource
  1. Dans la barre de navigation, cliquez sur Administration, puis cliquez sur Ressources pour ouvrir la vue Liste des ressources. Si nécessaire, utilisez la Recherche pour localiser une ressource, puis cliquez sur le nom de la ressource.
  2. Dans la vue Détails de la ressource, cliquez sur le bouton Supprimer.

  3. Cliquez sur OK.
