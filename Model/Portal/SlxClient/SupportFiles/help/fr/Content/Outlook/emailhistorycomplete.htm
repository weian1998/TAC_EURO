Terminer un e-mail

Utilisez la boîte de dialogue Terminer - E-mail pour enregistrer des informations sur un e-mail terminé.

Pour saisir les détails d'un enregistrement d'e-mail dans l'historique

  1. Effectuez l'une des opérations suivantes :
    • Dans MS Outlook, sélectionnez un ou plusieurs e-mails et cliquez sur Enregistrer dans l'historique.
    • Dans MS Outlook, après avoir composé un e-mail, cliquez sur Envoyer à SLX.
    • Faites glisser un ou plusieurs e-mails de MS Outlook vers une vue Détails de contact, Détails de compte, Détails de lead, Détails d'opportunité ou Détails de ticket, de Sage SalesLogix.

      • Si l'on vous demande de rechercher les enregistrements en double ou de choisir quels enregistrements ajouter à l'historique, vous devez d'abord sélectionner le bon enregistrement avant que l'e-mail puisse être enregistré dans l'historique.
      • Si l'on vous informe que le contact ou le lead est introuvable, vous ne pourrez pas enregistrer l'e-mail dans l'historique.
  2. Si la boîte de dialogue Saisie rapide s'ouvre, effectuez l'une des opérations suivantes :
    • Cliquez sur Individuellement pour effectuer séparément chaque enregistrement d'e-mail dans l'historique. Passez à l'étape 3.
    • Utilisez la boîte de dialogue Saisie rapide pour saisir des informations d'historique pour plusieurs e-mails.

      • Cette boîte de dialogue s'affiche uniquement si vous avez fait glissé et déposé un e-mail et que l'option Consigner dans l'historique dans Options est définie de façon à faire apparaître une invite. Si vous avez demandé à ne pas recevoir d'invite d'historique, l'historique d'e-mail est enregistré automatiquement et vous pouvez ignorer les étapes restantes.

  3. Si la boîte de dialogue Terminer un e-mail s'ouvre, utilisez-la pour consigner les informations d'historique.
    1. Dans la zone Objet, cliquez sur bouton en forme de flèche et sélectionnez un élément dans la liste.
    2. Dans la zone Emplacement, saisissez un emplacement, le cas échéant.
    3. Le cas échéant, modifiez la date de fin. Pour modifier la date, cliquez sur le Bouton Agenda pour sélectionner une date, puis cliquez sur OK.  
    4. Le cas échéant, modifiez la date Planifiée. Pour modifier la date, cliquez sur le Bouton Agenda pour sélectionner une date, puis cliquez sur OK.  
    5. Effectuez l'une des opérations suivantes :
      • Cliquez sur la la flèche du menu déroulant et effectuez votre sélection dans la listeDurée.
      • Sélectionnez la case Sans horaire pour créer une activité qui a lieu à une date spécifique, mais sans horaire spécifique et sans durée particulière.
    6. Dans la zone Résultat, cliquez sur la bouton en forme de flèche et sélectionnez un élément dans la liste.
    7. Si nécessaire, sélectionnez Contact ou Lead et utilisez la fonction Rechercher ou Supprimer pour modifier ou supprimer un lead, un contact, un compte, une opportunité ou un ticket associé.
    8. Dans la zone Priorité, sélectionnez l'importance de l'e-mail.
    9. Dans la zone Catégorie, sélectionnez la catégorie de l'e-mail.
    10. Dans la zone Responsable, sélectionnez un responsable si une personne autre que vous a terminé l'activité. Les utilisateurs n'apparaissent pas tous dans la liste de sélection Responsable. Vous ne pouvez choisir un utilisateur comme responsable d'activité que si vous avez accès à son agenda.
    11. Dans la zone Notes, saisissez des informations supplémentaires concernant cette activité.
    12. Si nécessaire, spécifiez une activité de suivi dans la zone Suivi. Cochez la case Reporter les notes ou Reporter les pièces jointes pour copier les notes ou pièces jointes de cette activité vers celle de suivi.
    13. Cliquez sur Ouvrir l'e-mail pour afficher l'e-mail.
    14. Cliquez sur l'onglet Pièces jointes pour ajouter ou afficher un document ou une URL.

      15. Le bouton Envoyer SLX de MS Outlook enregistre les pièces jointes à l'aide d'une des méthodes suivantes :

      • Les pièces jointes sont enregistrées en tant que fichiers individuels
      • Une seule pièce jointe qui contient le message électronique et toutes les pièces jointes

      La méthode utilisée est déterminée par votre administrateur.

    15. Cliquez sur OK.

      Si vous avez planifié une activité de suivi, la boîte de dialogue Planifier une activité apparaît afin que vous la complétiez.
