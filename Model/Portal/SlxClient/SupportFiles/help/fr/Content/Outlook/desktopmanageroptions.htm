Définition des options d'historique de messagerie du gestionnaire de bureau

Utilisez le Gestionnaire de bureau Sage SalesLogix pour entrer des informations de connexion et définir les options d'historique des e-mails.

Pour définir les options

  1. Dans la section Barre d'état de la barre des tâches Windows, double-cliquez sur l'icône du Gestionnaire de bureau Sage SalesLogix.
  2. Effectuez l'une des opérations suivantes :
  • Utilisateurs du client Web - Dans la zone URL du portail, tapez l'URL à laquelle se connecter. Utilisez le format suivant :

http://servername:port

  • Utilisateurs de Client Web hors ligne - Dans la zone URL du portail, tapez l'URL à laquelle se connecter. Utilisez le format suivant :

http://localhost

Si vous utilisez un nom totalement qualifié, vous devrez peut-être ajouter domain name.com après le nom du serveur.

  1. Effectuez l'une des opérations suivantes :
  • Si vous utilisez les informations de votre compte Windows pour vous connecter à Sage SalesLogix, sélectionnez l'option Utiliser l'authentification de Windows.
  • Saisissez votre nom d'utilisateur et votre mot de passe.
  1. Dans la zone Utilisateur, entrez le nom d'utilisateur dont vous vous servez pour vous connecter au client Web Sage SalesLogix.
  2. Dans la zone Mot de passe, entrez le mot de passe Sage SalesLogix du nom d'utilisateur entré ci-dessus.
  1. Cliquez sur Tester.
  2. Dans Options d'historique, sélectionnez dans quel cas vous souhaitez recevoir une notification lorsque vous enregistrez un e-mail MS Outlook en tant qu'élément d'historique Sage SalesLogix. Sélectionnez l'une des options suivantes :
  • Rechercher les contacts en double - Vous informe lorsqu'un contact ou un lead en double est détecté.
  • Rechercher les contacts - Vous invite à sélectionner les contacts ou leads auxquels l'élément d'historique doit être associé.
  • Rechercher les contacts non résolus - Vous informe lorsqu'un contact ou un lead est introuvable dans la base de données.
  • Afficher les détails de l'historique avant d'enregistrer - Vous affiche la boîte de dialogue complète de l'e-mail avant qu'un message e-mail ne soit enregistré dans l'historique.
  1. Cochez la case Afficher les notifications de la barre si vous souhaitez voir les messages contextuels décrivant les activités du gestionnaire de bureau lorsqu'elles se produisent.

Ces options ne concernent pas la fonction de déplacement par glisser-déposer des messages. Utilisez l'option Consigner dans l'historique dans Options pour faire glisser et déposer des messages.

Pour de plus amples informations sur le Gestionnaire de bureau, reportez-vous au système d'aide en ligne sur le Gestionnaire de bureau.
