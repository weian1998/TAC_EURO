Utilisation de la vue Liste des défauts

La vue Liste des défauts affiche tous les défauts auxquels vous avez accès. Utilisez les options de groupes et de recherche pour restreindre la liste des enregistrements qui s'affiche.

Procédure

Ouvrir la vue Liste des défauts
  • Dans la barre de navigation, cliquez sur Défauts.
Actualiser les informations de la liste
  • Cliquez sur le bouton Actualiser.

Utilisation de la vue Résumé

Travailler avec des enregistrements dans une grille

Filtrer ma liste

Utiliser la Recherche pour rechercher les enregistrements

Ouvrir la vue Détails du défaut
  • Cliquez sur le N° de défaut pour accéder à la vue Détails du défaut.
Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Enregistrer les enregistrements en tant que groupe
  • Exporter vers un fichier
  • Promouvoir vers le tableau de bord
