Utilisation de la vue Détails du défaut

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Vous pouvez copier des informations à partir d'un défaut existant pour en créer un nouveau. Si vous utilisez le bouton Copier, Sage SalesLogix copie les informations de l'enregistrement d'origine.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Procédure

Ouvrir la vue Détails du défaut
  • Dans la barre de navigation, cliquez sur Défauts pour ouvrir la vue Liste des défauts. Si nécessaire, utilisez la Recherche pour localiser un défaut, puis cliquez sur le N° de défaut.

Utiliser la Recherche pour rechercher les enregistrements

Afficher un groupe

Passez en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.

Afficher ou imprimer un rapport détaillé

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Rapport détaillé
  • E-mail
  • Publipostage
  • Ajouter une note
  • Nouvelle réunion
  • Nouvel appel téléphonique
  • Nouvelle tâche à faire

zones d'informations

Les zones d'informations du défaut contiennent les informations principales sur le défaut. Les utilisateurs disposant des autorisations appropriées peuvent modifier ces zones.

onglets des vues Détails

Chacun d'eux affiche des informations spécifiques que vous pouvez utiliser dans le cadre des activités relatives au défaut. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.

section définie par l'utilisateur

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
