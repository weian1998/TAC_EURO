Utilisation de l'onglet Tâches du défaut

Utilisez cet onglet pour voir toutes les activités associées au défaut, pour ajouter ou modifier le temps de travail sur un défaut ou pour modifier une activité. Les Activités désignent toute action impliquée dans le traitement ou la résolution d'un défaut. Par exemple, si vous avez besoin d'effectuer des recherches sur le défaut, ajoutez cette activité ici.

Cette vue contient les colonnes suivantes :

  • Type d'activité
  • Utilisateur
  • Date de réception
  • Date de réalisation
  • Type de taux
  • Unités de temps
  • Heures écoulées
  • Frais totaux

Pour ouvrir

  1. Ouvrir la vue Détails du défaut.
  2. Cliquez sur l'onglet Tâches du défaut. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.

Pour ajouter une tâche

  1. Cliquez sur Ajouter une tâche.
  2. Complétez les informations dans la zone d'informations Insérer une tâche du défaut.
  3. Cliquez sur OK.

Pour modifier une tâche

  1. Cliquez sur Modifier en regard de la tâche que vous souhaitez modifier.
  2. Modifiez les zones d'informations.
  3. Cliquez sur OK.

Pour supprimer une tâche

  1. Sélectionnez l'enregistrement à supprimer, puis cliquez sur le bouton Supprimer..
  2. Cliquez sur Oui.
