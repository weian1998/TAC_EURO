Modification d'un défaut

Utilisez la vue Détails du défaut pour modifier les informations sur un défaut existant. Par exemple, vous pouvez mettre à jour l'état, modifier l'affectation pour une autre personne ou saisir les détails sur la manière dont un problème a été résolu.

Pour modifier

  1. Ouvrir la vue Détails du défaut.
  2. Modifiez les zones d'informations sur le défaut.
  3. Cliquez sur Enregistrer.
