Sélectionner un système de comptabilité

L'écran Sélectionner un système de comptabilité dans l'Assistant d'association affiche le système de comptabilité disponible pour être associé avec le compte sélectionné.

Pour sélectionner

  1. Ouvrez l'Assistant d'association.
  1. Complétez l'écran Sélectionner un système de comptabilité et cliquez sur Suivant pour passer à l'étape suivante.
Zone Description/Action
Système Sélectionnez le système de comptabilité avec lequel associer le compte.
