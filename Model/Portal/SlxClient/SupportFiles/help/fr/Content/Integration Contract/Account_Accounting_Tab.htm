Onglet Comptabilité

(Intégration de la comptabilité uniquement) L'onglet Comptabilité de la vue Détails du compte affiche les informations de lien pour tous les systèmes de comptabilité actuellement associés à Sage SalesLogix. Vous pouvez utiliser cet onglet pour gérer les liens entre les systèmes. Cette vue contient les colonnes suivantes :

  • Système de comptabilité : le système de comptabilité auquel est associé le compte.
  • Etat : l'état du lien.
  • Partagé par : le nom du système de comptabilité qui partage les informations de compte d'exploitation.
  • Promu par : l'utilisateur qui a créé le lien.
  • Date de promotion : la date à laquelle le compte a été associé.

Procédure

Ouvrez l'onglet Comptabilité.

  1. Ouvrez la vue Détails du compte.
  2. Cliquez sur l'onglet Comptabilité. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets

Comprendre l'état du lien

La colonne Etat affiche l'état du lien entre le compte Sage SalesLogix et le système de comptabilité.

  • Associé - Le compte est associé à un compte d'exploitation dans le système de comptabilité.
  • Partagé - Le système de comptabilité partage les informations du compte d'exploitation avec un système de comptabilité associé différent. Par exemple, le système de comptabilité peut être une filiale d'un autre système de comptabilité associé et tous deux partagent le même compte d'exploitation. Les informations du compte partagé s'affichent dans la colonne Partagé par.

Associer un compte à un système de comptabilité

Seuls les systèmes de comptabilité configurés sur l'onglet Systèmes de comptabilité apparaissent dans la liste. Si l'état d'un système de comptabilité est Partagé, le système ne peut pas être associé. Vous ne pouvez associer que le compte parent des systèmes partagés.

  1. Dans la vue Détails du compte du compte que vous souhaitez associer, cliquez sur l'onglet Comptabilité.
  2. Dans le volet des Tâches Comptabilité, cliquez sur Promouvoir le compte.
  3. Renseignez les écrans de l'Assistant d'association.
  1. Dans la vue Détails du compte du compte que vous souhaitez associer, cliquez sur l'onglet Comptabilité.
  2. Dans la grille, cliquez sur Dissocier pour le système de comptabilité duquel vous souhaitez supprimer le lien.
  3. Pour confirmer la suppression, cliquez sur OK.
