Création d'un nouveau compte à partir d'un bon de commande

Vous pouvez créer un bon de commande avant de créer un compte ou un contact. Si le prospect décide de conclure l'affaire avec votre société, vous pouvez alors créer un nouveau compte et un contact à partir du bon de commande, contenant des informations limitées sur la société et le contact.

Pour créer

  1. Créer un nouveau bon de commande.
  2. Cliquez sur le bouton Ajouter un nouveau compte.
  3. Dans la boîte de dialogue Ajouter un nouveau contact de compte, ajoutez les informations suivantes :
  1. Dans le champ Compte entrez le nom du compte.
  2. Dans le champ Type, sélectionnez le type de compte.
  3. Dans le champ Principal, tapez le numéro de téléphone du bureau principal.
  4. Dans le champ Adresse, entrez l'adresse professionnelle du compte.
  5. Dans le champ Contact tapez le nom du contact.
  6. Dans le champ Fonction, tapez ou sélectionnez la fonction du contact.
  7. Dans le champ Principal, tapez le numéro de téléphone du contact.
  8. Dans le champ Mobile, tapez le numéro de téléphone mobile du contact.
  9. Dans le champ E-mail, tapez l'adresse électronique du contact.
  10. Cliquez sur OK.
  1. Cliquez sur Enregistrer.
