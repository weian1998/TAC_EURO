Onglet Systèmes de comptabilité

Utiliser l'onglet Systèmes de comptabilité pour gérer les flux de connexion de vos systèmes de comptabililté.

Procédure

Ajouter ou modifier un flux

  1. Ouvrez la vue Détails de l'intégration du système de comptabilité.
  2. Cliquez sur l'onglet Systèmes de comptabilité.
  3. Effectuez l'une des opérations suivantes :
  • Pour ajouter un flux, cliquez sur Configuration de l'intégration du système de comptabilité.
  • Pour modifier un flux existant, cliquez sur le lien Modifier du flux que vous souhaitez modifier.
  1. Suivez les instructions de la boîte de dialogue Configuration de l'intégration du système de comptabilité.
  2. Cliquez sur OK.

S'il ne s'agit pas du premier flux de système de comptabilité à associer, vous devez réinitialiser IIS sur le serveur Web afin que tous les nouveaux flux soient inclus lors de la synchronisation.

Synchroniser tous les systèmes de comptabilité

  1. Ouvrez la vue Détails de l'intégration du système de comptabilité.
  2. Cliquez sur l'onglet Systèmes de comptabilité.
  3. Cliquez sur le bouton Synchroniser tous les systèmes de comptabilité.
