Comparaison et configuration des différences de schéma

Sage recommande la comparaison du schéma de base de données de Sage SalesLogix à celui de votre application de comptabilité pour garantir l'intégrité des données. Vous pouvez comparer les données de plusieurs ressources pour savoir si des différences de longueurs existent entre les données des systèmes.

Cette fonctionnalité compare le schéma de la ressource sélectionné dans les deux systèmes intégrés et renvoie les différences de longueur entre les données.

En savoir plus sur la manière dont Sage SalesLogix gère les différences de schéma...

Longueur Max : si la longueur des donnés synchronisées vers Sage SalesLogix dépasse la LongueurMax d'un champ, la ligne entière est rejetée.

Décimales : si les donnés synchronisées vers Sage SalesLogix contiennent un nombre de décimales supérieur à celui défini, la valeur est arrondie en fonction du nombre de décimales autorisé. Par exemple, si un champ de devise a la valeur 17,4, alors une valeur de 123,12348 est enregistrée sous la forme 123,1235 et une valeur de 123.12343 est enregistrée sous la forme 123,1234.

Pour comparer

  1. Dans le menu Administration, cliquez sur Configuration de l'intégration.
  2. Sur l'onglet Systèmes de comptabilité, cliquez sur Modifier pour le système de comptabilité activé.
  3. Dans la boîte de dialogue Configuration de l'intégration de la comptabilité, utilisez la section Différences de schéma pour rechercher des différences entre les systèmes.
  1. Dans le champ Filtrer par ressource, sélectionnez le type de ressource.
  2. Cliquez sur Exécuter.
  3. Vérifiez les différences dans la grille.
  4. Notez toute information qui pourrait causer des problèmes d'intégrité des données. Par exemple, un champ du système de comptabilité peut n'accepter que trois caractères alors que le même champ dans Sage SalesLogix accepte du texte libre.
  5. Répétez l'étape 3 pour tous les types de ressources.
  1. Cliquez sur OK.
  2. Utilisez le Gestionnaire de base de données ou une autre application appropriée pour modifier le schéma de l'un des systèmes.
