Vue Détails de paiement

La vue Détails de paiement affiche des informations en lecture seule de votre système de comptabilité.

Pour afficher

  1. Ouvrez la vue Détails du compte et cliquez sur l'onglet Paiements des clients. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  2. Cliquez sur le numéro du reçu que vous souhaitez afficher. Les informations suivantes s'affichent :
    3. Zone Description
    Numéro de reçu Le numéro du reçu.
    Date de paiement La date du reçu.
    Date de traitement La date à laquelle le reçu a été traité.
    Nom Le nom du reçu.
    Type Le type de reçu. Par exemple, pré-paiement, acompte, etc.
    Etat L'état du reçu. Par exemple, réglé, en attente, autorisé, etc.
    Devise La devise du reçu.
    Type de règlement La méthode de règlement utilisée pour le reçu. Par exemple, liquide, chèque, carte de crédit, etc.
    Référence du règlement La référence du reçu associée au type de règlement. Par exemple, un numéro de chèque, de carte de crédit, etc.
    Facture La référence à un autre document si le reçu est pour ce document. Par exemple : une facture.
    Total net La valeur totale de la facture, incluant les remises, taxes exclues.
    Remises Toute remise pouvant s'appliquer à la facture.
    Frais Le montant total des frais appliqués à la ligne de facture (par exemple, les charges financières ou les intérêts).
    Taxes Les taxes qui s'appliquent à la ligne de facture.
    Montant du paiement La valeur totale du reçu, incluant les taxes, les remises et les frais.
    Source La source du reçu. Par exemple, chèque, liquide, prélèvement automatique, etc.
    Compte Le compte (client) depuis lequel le reçu a été obtenu.
    Code de taxe Le code de taxe du reçu.
