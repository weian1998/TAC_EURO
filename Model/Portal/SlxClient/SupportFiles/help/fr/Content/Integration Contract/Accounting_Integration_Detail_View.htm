Vue Détails de l'intégration du système de comptabilité

Utilisez la vue Détails de l'intégration du système de comptabilité pour configurer les connexions entre Sage SalesLogix et votre(vos) système(s) de comptabilité.

L'intégration du système de comptabilité est activée lorsque le point de terminaison Sage SalesLogix et au moins un point de terminaison cible sont configurés et activés. Si aucun flux n'est activé sur l'onglet Systèmes de comptabilité, alors l'intégration est désactivée.

Lorsque vous activez ou désactivez l'intégration du système de comptabilité, vous devez vous déconnecter puis vous reconnecter à Sage SalesLogix pour que ces modifications prennent effet.

Définissez les détails de vos flux pour Sage SalesLogix, puis configurez le flux pour chaque système de comptabilité intégré.

Pour définir

  1. Dans le menu Administration, cliquez sur Configuration de l'intégration.
  2. Dans la zone Flux SalesLogix, tapez l'URL Sdata du flux GCRM.
    Utilisez le format : http://server:port/sdatavirtualdir/slx/gcrm/-/ 
    • Définissez sdatavirtualdir sur le répertoire dans lequel vous avez déployé votre portail SData.
      Le répertoire par défaut est « SData ».
      Par exemple, http://localhost:3333/sdata/slx/gcrm/-/
  3. Dans la zone Nom d'utilisateur, saisissez le nom d'utilisateur de la base de données SalesLogix à laquelle le système de comptabilité se connecte.
  4. Dans le champ Mot de passe, saisissez le mot de passe correspondant à ce nom d'utilisateur.
  5. Dans la zone Nom d'affichage, saisissez un nom d'affichage pour le flux.
  6. Cliquez sur Enregistrer.
  7. Si nécessaire, sélectionnez l'option Limiter la promotion de compte à un seul système de comptabilité pour limiter votre intégration à Sage SalesLogix et à un seul système de comptabilité.
  8. Si nécessaire, sélectionnez l'option Autoriser le système de comptabilité à gèrer les bons de commande pour supprimer la capacité de créer et modifier des bons de commande dans Sage SalesLogix. Lorsque vous sélectionnez cette option, votre système de comptabilité doit être utilisé pour gérer les bons de commande.

    9. Cette option est nécessaire pour certaines intégrations, par exemple X3.
  9. Utilisez l'onglet Systèmes de comptabilité pour configurer les détails du flux pour votre(vos) système(s) de comptabilité.
  10. Utilisez l'onglet Synchronisation pour configurer les intervalles de synchronisation.
  11. Utilisez l'onglet Correspondance pour définir les critères de correspondance généraux par défaut pour les utilisateurs créant des liens.
