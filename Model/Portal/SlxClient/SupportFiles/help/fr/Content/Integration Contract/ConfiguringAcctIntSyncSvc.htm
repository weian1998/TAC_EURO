Configuration du service de synchronisation d'intégration du système de comptabilité

Le service de synchronisation d'intégration du système de comptabilité (Service système SalesLogix) permet de transférer des données entre Sage SalesLogix et votre système de comptabilité. Le service doit être démarré et configuré pour s'exécuter de manière automatique.

Pour configurer

  1. Veillez à configurer et activer des points d'extrémité avant de démarrer le service.
  2. Consultez le fichier Sage.SalesLogix.SyncEngineService.exe.config pour vérifier que la connexion est définie correctement à la fin du fichier. Les informations de connexion doivent correspondre au fichier de configuration connection.config deployé vers votre site Web.

Par défaut, le fichier de configuration est ...\Program Files\SalesLogix\Sage.SalesLogix.SyncEngineService.exe.config et le fichier de connexion est déployé vers ...\inetpub\wwwroot\slxclient

  1. Sur la station de travail administrative, ouvrez les Services.
  2. Cliquez avec le bouton droit de la souris sur le Service système SalesLogix, puis cliquez sur Propriétés.
  3. Sélectionnez l'option Automatique dans la zone Type de démarrage.
  4. Configurez le service pour se connecter et s'exécuter en tant qu'utilisateur WebDLL.
  5. Vérifiez que l'authentification Windows a été activée pour l'utilisateur WebDLL.
  6. Cliquez sur Démarrer.
  7. Cliquez sur OK.
