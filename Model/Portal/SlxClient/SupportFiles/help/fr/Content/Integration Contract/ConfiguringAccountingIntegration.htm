Configuration de l'intégration du système de comptabilité

Complétez les tâches suivantes pour configurer l'intégration du système de comptabilité.

  • Si la fonctionnalité de devises multiples est activée, vérifiez que Sage SalesLogix et votre système de comptabilité utilise la même devise de base.
  • Exécutez le service de mise à jour de téléphone et e-mail.

  • L'authentification Windows doit être configurée pour l'utilisateur administratif avant d'exécuter le service de mise à jour téléphonique et e-mail de l'intégration du compte. Pour les étapes détaillées, consultez la rubrique « Utilisation de l'authentification Windows » dans l'aide du module Administrateur.
    • L'intégration du système de comptabilité est activée lorsque le point de terminaison Sage SalesLogix et au moins un point de terminaison cible sont configurés et activés. Utilisez la vue Détails de l'intégration du système de comptabilité pour activer des points d'extrémité.

    Lorsque vous activez ou désactivez l'intégration du système de comptabilité, vous devez vous déconnecter puis vous reconnecter à Sage SalesLogix pour que ces modifications prennent effet.
    • Comparez le schéma de base de données de Sage SalesLogix à votre schéma d'application de comptabilité pour garantir l'intégrité des données. Utilisez l'onglet Systèmes de comptabilité pour déterminer les différences de schémas.

    Si vous avez personnalisé Sage SalesLogix, vous devez consulter le contrat ERP de Sage CRM et vérifier que les informations vont être synchronisées entre les différents systèmes. Par exemple, si vous avez créé des entités personnalisées qui ne concordent pas avec celles de sage SalesLogix, la synchronisation peut ne pas fonctionner.
    • Utilisez l'onglet Correspondance pour définir les critères de correspondance généraux par défaut pour les utilisateurs créant des liens.
    • Configurez le service de synchronisation d'intégration du système de comptabilité.
    • Si nécessaire, exportez et fusionnez les enregistrements de votre système de comptabilité.

    Utilisez l'outil d'exportation fourni par votre système de comptabilité pour exporter les données appropriées vers Sage SalesLogix.

    • Configurez les rôles utilisateur.
