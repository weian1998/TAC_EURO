Propriétés du contrôle Pick List

Vous pouvez afficher et modifier les valeurs des propriétés suivantes de ce contrôle :

Propriété Basique Description

Légende

L'étiquette à afficher sur le formulaire de ce contrôle.

Alignement de la légende

Justification du texte de l'étiquette. Les options disponibles sont :

  • Gauche
  • Droite
  • Centre

Info-bulle

Le texte qui s'affiche comme info-bulle lorsque la souris de l'utilisateur passe sur le contrôle.

Lecture seule

Ne permet pas les modifications sur le formulaire.
Propriété Avancée Description

Mode d'affichage

Mode d'affichage du contrôle. Les options disponibles dépendent du type de contrôle :

  • Comme contrôle : ressemble à un champ textuel.
  • Comme texte : s'affiche en tant que texte brut.
  • Comme lien hypertexte : s'affiche en tant que lien hypertexte.
  • Normal : ressemble à une étiquette.

Placement des étiquettes

Position de l'étiquette relative au contrôle.

Visible

Permet d'afficher ou de masquer ce contrôle sur le formulaire.

Autoriser les sélections multiples

Autorise l'utilisateur à sélectionner plusieurs éléments de la liste.

Le comportement des listes de sélection du client Web et du client réseau peut être défini dans la vue Détails de la liste de sélection du client Web. Par exemple, vous pouvez définir que l'entrée est obligatoire ou non pour une liste de sélection individuelle. Afin d'appliquer des comportements aux listes de sélection du client Web, l'attribut Apply options to Web pick list doit être sélectionné. Sinon, les comportements définis par le contrôle Quick Form qui utilise les listes de sélection sont appliqués à la place.

Avant de définir les options des listes de sélection, assurez-vous de comprendre les propriétés de contrôle des listes de sélection et la façon dont les données des listes de sélection sont stockées dans la base de données. Le fait de comprendre comment les données sont liées entre la liste de sélection et la base de données déterminera si une option est prise en charge. Par exemple, si vous modifiez la liste de sélection Préférences en « Autoriser les sélections multiples », la liste de sélection ne fonctionnera pas correctement parce que le champ de la base de données est trop petit pour stocker de grandes quantités de texte.

Longueur Max

Nombre maximal de caractères que l'utilisateur peut saisir.

Obligatoire

Nécessite une valeur lors de l'enregistrement des données.

Associations de données

Définit l'endroit d'où sont issues les données du contrôle. Ces informations sont en lecture seule dans Concepteur de formulaire.

Nom de la liste de sélection

Nom de la liste des valeurs que l'utilisateur peut sélectionner lors de la saisie de données. Ces informations sont en lecture seule dans Concepteur de formulaire.

Mode de stockage

Mode de stockage de la valeur. Les options disponibles sont :

  • Texte - Si la valeur a été stockée sous forme de chaîne, affiche la chaîne.
  • ID - Si la valeur a été stockée sous forme d'ID, récupère la chaîne correspondante de la liste de sélection et l'affiche.
  • Code - Si la valeur a été stockée sous forme de code, affiche la chaîne correspondante. Option peu utilisée.

Ces informations sont en lecture seule dans Concepteur de formulaire.

ID de contrôle

Identifiant du contrôle. Ces informations sont en lecture seule dans Concepteur de formulaire.

Type de contrôle

Type de contrôle Sage SalesLogix tel que TextBox. Ces informations sont en lecture seule dans Concepteur de formulaire.
