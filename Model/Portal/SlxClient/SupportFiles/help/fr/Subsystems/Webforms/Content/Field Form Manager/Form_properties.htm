Propriétés des formulaires

Vous pouvez afficher et modifier les valeurs des propriétés suivantes de ce formulaire :

Propriété Basique Description
Contrôle actif L'identifiant du contrôle que vous souhaitez activer à l'aide du curseur utilisateur lorsque la première page s'affiche. Par exemple, NomCompte sur le formulaire InsérerCompte.
Nom Le nom du formulaire, par exemple DétailsCompte. Ces informations sont en lecture seule dans Concepteur de formulaire.
Description Description du formulaire.
Propriété Basique Description
Utiliser le nom de l'entité comme titre Lorsque cette propriété est définie sur Vrai (true), le nom du formulaire est déterminé par le type d'entité sur lequel est basé le formulaire. Par exemple, si le formulaire est basé sur Compte, le titre du formulaire est le nom du compte actuel.
Propriété Basique Description
Portail Le nom du site Web dont la vue fait partie. Par exemple, le portail du Client Web Sage SalesLogix et le portail Client. Ces informations sont en lecture seule dans Concepteur de formulaire.
Vue Le nom d'une vue qui contient le formulaire. Par exemple, Détails du compte. Ces informations sont en lecture seule dans Concepteur de formulaire.
Modes Pour cette vue, les modes dans lesquels le formulaire est généré : Détails, Insertion, et Liste. Ces informations sont en lecture seule dans Concepteur de formulaire.
Description Description de la vue. Ces informations sont en lecture seule dans Concepteur de formulaire.
