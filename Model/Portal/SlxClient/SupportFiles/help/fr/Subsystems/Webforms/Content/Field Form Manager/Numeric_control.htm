Propriétés du contrôle Numeric

Vous pouvez afficher et modifier les valeurs des propriétés suivantes de ce contrôle :

Propriété Basique Description

Légende

L'étiquette à afficher sur le formulaire de ce contrôle.

Alignement de la légende

Justification du texte de l'étiquette. Les options disponibles sont :

  • Gauche
  • Droite
  • Centre

Info-bulle

Le texte qui s'affiche comme info-bulle lorsque la souris de l'utilisateur passe sur le contrôle.

Lecture seule

Ne permet pas les modifications sur le formulaire.
Propriété Avancée Description

Placement des étiquettes

Position de l'étiquette relative au contrôle.
Type de format

Sert à convertir une valeur numérique en chaîne. Les options disponibles pour le contrôle sont les suivantes :

  • Nombre
  • Pourcentage
  • Decimal
  • Scientifique

Visible

Permet d'afficher ou de masquer ce contrôle sur le formulaire.

Activé

Permet à l'utilisateur d'interagir avec ce contrôle.

Longueur Max

Nombre maximal de caractères que l'utilisateur peut saisir.
Obligatoire

Nécessite une valeur lors de l'enregistrement des données.

Décimales

Le nombre de décimales à afficher et autoriser lors de la saisie de données.

Une valeur par défaut de -1 signifie Utiliser le nombre de décimales spécifié dans les paramètres régionaux de l'utilisateur. Vous pouvez aussi entrez un entier compris entre 0 et 4. Si vous souhaitez entrer un chiffre supérieur à 4, assurez-vous dans l'application Architect que le champ lié au contrôle est défini avec au moins autant de décimales.

Lorsqu'un utilisateur entre moins de décimales que la valeur de précision spécifiée, une colonne ou un contrôle Currency ajoute des zéros. Si vous voulez qu'une colonne ou qu'un contrôle Numeric renseigne la valeur en fonction de la précision spécifiée, définissez Strict=Vrai.

Strict

 Affichage des nombres avec un certain nombre de zéros à la fin, selon les besoins. Par exemple, si Décimales = 4, Strict = Vrai, et que la valeur numérique est 2,3, la valeur s'affiche sous la forme 2,3000.

Associations de données

Définit l'endroit d'où sont issues les données du contrôle. Ces informations sont en lecture seule dans Concepteur de formulaire.

ID de contrôle

Identifiant du contrôle. Ces informations sont en lecture seule dans Concepteur de formulaire.

Type de contrôle

Type de contrôle Sage SalesLogix tel que TextBox. Ces informations sont en lecture seule dans Concepteur de formulaire.
Propriété Présentation Description

Ligne

Numéro de ligne du contrôle.

Column

Numéro de colonne du contrôle.

Largeur de ligne

Nombre de cellules que le contrôle occupe verticalement.

Largeur de colonne

Nombre de cellules que le contrôle occupe horizontalelement.
