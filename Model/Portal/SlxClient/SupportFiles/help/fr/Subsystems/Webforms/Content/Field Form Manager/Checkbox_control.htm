Propriétés du contrôle CheckBox

Vous pouvez afficher et modifier les valeurs des propriétés suivantes de ce contrôle :

Propriété Basique Description

Légende

L'étiquette à afficher sur le formulaire de ce contrôle.

Alignement de la légende

Justification du texte de l'étiquette. Les options disponibles sont :

  • Gauche
  • Droite
  • Centre

Info-bulle

Le texte qui s'affiche comme info-bulle lorsque la souris de l'utilisateur passe sur le contrôle.

Lecture seule

Ne permet pas les modifications sur le formulaire.
Propriété Avancée Description

Placement des étiquettes

Position de l'étiquette relative au contrôle.

Visible

Permet d'afficher ou de masquer ce contrôle sur le formulaire.

Activé

Permet à l'utilisateur d'interagir avec ce contrôle.

Associations de données

Définit l'endroit d'où sont issues les données du contrôle. Ces informations sont en lecture seule dans Concepteur de formulaire.

ID de contrôle

Identifiant du contrôle. Ces informations sont en lecture seule dans Concepteur de formulaire.

Type de contrôle

Type de contrôle Sage SalesLogix tel que TextBox. Ces informations sont en lecture seule dans Concepteur de formulaire.
Propriété Présentation Description

Ligne

Numéro de ligne du contrôle.

Column

Numéro de colonne du contrôle.

Largeur de ligne

Nombre de cellules que le contrôle occupe verticalement.

Largeur de colonne

Nombre de cellules que le contrôle occupe horizontalelement.
