Prise en charge d'un environnement multilingue

Les chaînes de texte utilisées sur les contrôles, telles que les légendes ou les info-bulles, sont rassemblées dans un fichier de ressource afin d'en faciliter la localisation. Les installations prenant en change plusieurs langues comprennent un fichier de ressource contenant les chaînes traduites dans chaque langue. Par exemple, une installation Sage SalesLogix prenant en charge la culture de base et l'allemand (en Allemagne) comprend un fichier de ressource pour la culture Invariable (de base) et un fichier pour l'allemand, 'de-DE'. Un utilisateur du client Web peut ainsi afficher les chaînes en allemand en définissant la langue de son navigateur sur 'de-DE' dans les préférences de langue de celui-ci.

Si votre site est configuré pour prendre en charge plusieurs langues, vous pouvez modifier les légendes et autres chaînes textuelles dans chaque langue prise en charge.

Pour localiser des chaînes textuelles

  1. Modifier un formulaire dans le Concepteur de formulaire.
  2. Sélectionnez la Culture souhaitée dans la liste déroulante.
    Différentes cultures peuvent être ajoutées à cette liste. Pour plus d'informations, consultez la rubrique « Ajouter une culture à la liste dans Concepteur de formulaire » dans l'aide de l'application Architect.
  3. Modifiez les valeurs des propriétés de contrôle pouvant être localisées telles que Légende et Info-bulle.
  4. Cliquez sur Enregistrer.
    Les légendes et autres chaînes pouvant être localisées sont enregistrées dans le fichier ressource de la culture utilisée.
  5. Affichez le formulaire. Pour voir les légendes localisées, modifiez les paramètres dans les préférences de langue de votre navigateur.
