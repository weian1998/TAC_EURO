Concepteur de formulaire - Utilisation des formulaires

 L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Des modifications simples peuvent être apportées à une vue en modifiant les formulaires la composant à l'aide du Concepteur de formulaire. Par exemple, vous pouvez modifier les légendes, déplacer des éléments, ajouter ou masquer des informations.

Lors de l'enregistrement des modifications apportées à un formulaire, toutes les vues le contenant sont également modifiées.

Les modifications que vous effectuez affectent tous les utilisateurs du système, car les définitions de formulaire et de vue sont partagées.

Les Quick Forms peuvent être modifiés avec le Concepteur de formulaire dans le client Web Sage SalesLogix ou l'application Architect. Les formulaires personnalisés ne peuvent pas être modifiés avec le Concepteur de formulaire. Chaque formulaire pouvant être modifié avec le Concepteur de formulaire affiche le bouton Modifier le formulaire dans sa barre d'outils.

Procédure

Ouvrez le Concepteur de formulaire
  1. Ouvrez une vue du client Web comme par exemple la vue Détails du compte dans votre navigateur.
    Remarque : chaque formulaire contenu dans la vue possède une barre d'outils.
  2. Dans la barre d'outils du formulaire que vous souhaitez modifier, cliquez sur Modifier le formulaire en maintenant la touche Ctrl enfoncée.
    Un autre onglet de navigateur s'ouvre avec Concepteur de formulaire. Vous avez la possibilité de passer d'un onglet à l'autre afin d'effectuer des modifications et de voir les résultats, puis de répéter l'opération plusieurs fois.
  3. Cliquez sur l'onglet du nouveau navigateur.
    Le Concepteur de formulaire affiche la présentation des contrôles du formulaire.

Vous pouvez également utiliser la vue Liste du gestionnaire de formulaire pour ouvrir le Concepteur de formulaire.

Modifier un formulaire
  1. Ouvrez le formulaire dans le Concepteur de formulaire.
  2. Cliquez sur l'onglet Utilisation et vérifier quelles vues contiennent ce formulaire. Les modifications apportées au formulaire affecteront l'ensemble de ces vues.
  3. Apportez les modifications nécessaires au formulaire, par exemple :
    • Modifiez les légendes et les autres propriétés de contrôle
    • Déplacer et masquer les contrôles
    • Ajouter des contrôles (ajouter des champs de données à un formulaire)
    • Modifiez les propriétés du formulaire, des lignes et des colonnes
  1. Cliquez sur Enregistrer
    Les modifications prennent effet immédiatement.
Afficher les modifications
  • Passez sur l'onglet du navigateur d'origine et appuyez sur F5. Vous avez la possibilité de passer d'un onglet à l'autre afin d'effectuer des modifications et de voir les résultats, puis de répéter l'opération plusieurs fois.
  • Vous pouvez aussi accéder à n'importe laquelle des vues qui contient le formulaire que vous avez modifié.

Vous pouvez voir les modifications apportées aux quick forms de vue Résumé une fois que la création et le déploiement ont été faits dans l'application Architect. Consultez « Création de la plate-forme Web » dans l'aide de l'application Architect pour plus d'informations.
Annuler mes dernières modifications
  • Sur l'onglet du navigateur du Concepteur de formulaire, appuyez sur F5 pour revenir à la dernière version enregistrée du formulaire.
Diffuser mes modifications aux autres utilisateurs
  • Une fois les modifications enregistrées, elles affectent tous les utilisateurs du système de façon immédiate. Si un utilisateur a déjà ouvert la vue, il doit appuyer sur F5 pour la réactualiser.
  • Si vous souhaitez diffuser les modifications vers un système différent, l'application Architect doit être utilisée. Consultez la rubrique « Utilisation du modèle de bundle » dans l'aide de l'application Architect pour obtenir davantage d'informations.

Modifier les valeurs des propriétés des contrôles

Déplacer un contrôle
  1. Cliquez sur un contrôle et faites le glisser.
  2. Déposez-le à n'importe quel emplacement en surbrillance lorsque le curseur le survole.
    Les contrôles existants sont automatiquement déplacés si nécessaire.
Supprimer un contrôle
  • Le Concepteur de formulaire vous permet de cacher les contrôles mais pas de les supprimer. Pour supprimer un contrôle, utilisez l'application Architect Sage SalesLogix.
Masquer un contrôle
  1. Sélectionnez le contrôle et cliquez sur l'onglet Avancé.
  2. Dans le groupe Apparence, décochez la case de la propriété Visible.
  3. Si vous voulez pouvoir réutiliser l'espace occupé par le contrôle masqué, déplacez-le vers un autre emplacement, le bas de la page par exemple. Pour supprimer un contrôle, utilisez l'application Architect Sage SalesLogix.

Ajouter un champ de données à un formulaire (Ajouter un contrôle)

Changer la taille d'un contrôle
  1. Sélectionnez le contrôle et cliquez sur l'onglet Présentation.
  2. Dans le groupe Taille, modifiez les propriétés Largeur de ligne ou Largeur de colonne selon vos besoins.
    Par exemple, sur le formulaire DétailsCompte, Desc. de l'activité s'étend sur deux lignes. Remarque : certains types de contrôle ont une taille d'affichage constante quel que soit l'espace qu'ils occupent.
Changer l'étiquette d'un contrôle
  1. Sélectionnez le contrôle et cliquez sur l'onglet Basique.
  2. Dans le groupe Apparence, modifiez la propriété Légende selon vos besoins.
Rendre les données en lecture seule
  1. Sélectionnez le contrôle et cliquez sur l'onglet Basique.
  2. Dans le groupe Comportement, cochez la case Lecture seule.
Afficher et modifier les paramètres de formulaire
  1. Cliquez sur le Concepteur de formulaire, n'importe où en dehors d'un espace occupé, par exemple, dans le coin gauche supérieur.
    Les propriétés de formulaire s'affichent dans la zone des onglets.
  2. Consultez et modifiez les valeurs des propriétés, selon vos besoins. Par exemple, Active Control est le contrôle que vous souhaitez activer à l'aide du curseur utilisateur lorsque la première vue s'affiche.
Changer la largeur d'une colonne de contrôles
  1. Cliquez sur le sélecteur de colonne au dessus d'une colonne.
    Les propriétés de la colonne s'affichent dans la zone des onglets.
  2. Dans le groupe Apparence , modifier la Largeur.
  3. Répétez cette procédure pour chaque colonne.

    Si Type de taille est en pourcentage, le total de toutes les largeurs de colonne est normalisé sur 100.

Ajouter ou supprimer une ligne ou colonne vide
  1. Cliquez avec le bouton droit sur le rectangle de sélection situé à gauche d'une ligne ou au dessus d'une colonne.
  2. Utilisez le menu contextuel pour supprimer ou insérer une ligne ou une colonne vide.
Ajout d'un formulaire
  • Utilisez l'application Architect pour ajouter des formulaires. Pour plus d'informations, référez-vous à la rubrique d'aide « Création d'un nouveau formulaire » de l'application Architect.
