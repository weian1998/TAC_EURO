Ajout d'un champ de données à un formulaire (ajout d'un contrôle)

Vous pouvez ajouter des champs de données existants à un formulaire à l'aide du volet Ajouter un champ du Concepteur de formulaire. Celui-ci affiche une arborescence de champs de données qui peuvent être ajoutés au formulaire en cours. Développez les éléments dans l'arborescence qui ont un signe plus afin d'utiliser leurs champs de composants tels que l'adresse/ville.

Si le champ que vous souhaitez ajouter à votre formulaire n'existe pas encore dans la base de données, il doit être ajouté. Consultez « Ajout d'un champ à une table » dans l'aide de l'Administrateur ou « Modification d'une entité » et « Insertion d'une nouvelle propriété d'entité » dans l'aide de l'application Architect pour plus d'informations.

Pour ajouter

  1. Vous pouvez aussi commencer par cliquer sur Enregistrer au cas où vous auriez besoin d'annuler les dernières étapes (facultatif).
  2. Cliquez sur un élément dans le volet Ajouter un champ et faites-le glisser.

  3. Déposez-le à n'importe quel emplacement en surbrillance lorsque le curseur le survole.

    Un contrôle est créé et est lié au champ de données dans la base de données. Le type de contrôle est défini automatiquement en fonction du type de données. Par exemple, si vous déposez un numéro de téléphone sur un formulaire, un contrôle Phone est créé.

  4. Vous pouvez également modifier les valeurs de propriété du nouveau contrôle.
  5. Si vous êtes certain de vouloir conserver le champ ajouté, cliquez sur Enregistrer. Sinon, appuyez sur F5 pour revenir à la dernière version enregistrée du formulaire.
